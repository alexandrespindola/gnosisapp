Tɛkɛ ke kuat ɛnɛnɛ kuɔthɛkɛnɛ, kënɛ kuat nhial, kuat rɛt, kuat cɛŋ.

Kɔc ciɛɛŋ dɛŋ kɛmɛ kënɛ, kɔc juɛɛŋ cuɛt kɛmɛ nɛm kuat, kɔc jiɛŋ nɛm muɔŋkɛnɛ ɛnɛnɛ lɔc.
