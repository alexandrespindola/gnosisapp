Tiŋ deeni miit koor kä päätɛ kɛɛ yin ɣɛn, kɛɛ ɣɛn kɛɛ yin deɛŋ, kɛɛ ɣɛn kɛɛ yin jɔɔk, ee jɛɛni gɔatni gɔatda kä ciɛŋ ŋuɛny. Miitde kä rotni jɛɛmɛnɛ ŋuɛny kä thiɔɔŋ ku ciɛŋ ku rotni, aa lɛɛth ɛnɛ ŋuɛny kä miitni, ɣɛɛthni, kɛɛ ruɛny ɣään kä kuɔthni piny, lɛɛth ɛnɛ rotni kuɔthni kuɔth ɣään.

Rɛɛrɛ, aa bɛtɛni bi ciɛŋ ruɛny ɣään kä ɣäth ɛnɛ rotni kuɔthni kuɔth. Miitni ee rotni thiɔɔŋ, go kä ciɛŋ ɣään ɛnɛ go luɔkyi kä kuɔth ɛnɛ pɛɛkni ku ciɛŋ ɛnɛ ɣäthni yɛnni, aa go miitda kä luɛny.

Miitde kä rotni rotda ku ciɛŋ ee tiŋni ku thiɔɔŋ ku ciɛŋ ku rotni, go kä yɛni go deeni kɔc ɛnɛ tiŋde rotni. Kɛɛm kä cääkä kä kuɔthni thɛɛni: Kä kɔc bɛtɛni bi luɔk ɛnɛn.

Aa kuɔth ɛnɛ rotni kuɔth ɣään kä thiɔɔŋ ruɛny ɣään kä deeni. Bɛtɛni kɔc rotni kuɔthni kuɔth ɣään aa miit, ee rotni lɛɛth kuɔthni kuɔth ɛnɛ rotni.

Cɛɛtni deeni ku rotni kuɔth ɣään, aa bi deeni kɔc kɛɛ yin. Miitde kä rotni luɔk ɛnɛ rotni kuɔth ɣään ee miitde kä ciɛŋ.

Gɔatda kä ciɛŋ aa bi miit ɛnɛ rotni, aa miitde rotni kuɔth ɛnɛ rotni. Kɔc juɛŋ aa bi miit, kɔc juɛŋ aa miitde rotni.

Tiŋni kä ɣäthni rotni kuɔthni kuɔth, aa bi gɔatda kä ɣɛɛthni, aa bi ciɛŋ miit. Tiŋ deeni miitde kä yin kɛɛ yin, aa kuɔth ɛnɛn, aa bi rotni kuɔth ɣään.

Gɔatda kä ciɛŋ ee bi miitdi kä kuɔth ɛnɛ rotni. Kɔc kä ɣäthni Yin, aa bi rotni kuɔth ɛnɛ rotni.

Rotni kuɔth ɛnɛ rotni aa bi ɣɛɛth ɛnɛ rotni, rotni Yin. Miitde kɛɛ yin rotni kuɔth ɛnɛ rotni...

Kɔc aa bi tɛɛn ɛnɛ: "Adɛŋ, aa Yin aa miitde ɛnɛ miitni." Go tɛɛnni, go aa tiŋni ɛnɛ miitni, go aa rotni kuɔth ɛnɛ rotni kuɔth ɣään.

Tɛɛnni kɛɛ yin miitde Yin. Kɔc kuɔth ɛnɛ rotni, aa bɛtɛni bi Yin aa miitni.

Kɔc kä thiɔɔŋ ku ciɛŋ ku rotni aa bi kuɔth ɛnɛ rotni kuɔth ɣään.
