Nɛy jiɛŋ cien jiɔk rɛŋ kuɔth diaal buk, jiɛŋ thɔɔrdiɛt kuɔth diaal lɔc.

Ciɛŋ piel aan nhiam, ran ee päl; ciɛŋ thɔɔrdiɛt aan, ran ee lɛɔŋ; ciɛŋ kuɔth diaal aan, ran ee gamɛl; ciɛŋ biɛc aan, ran ee thiɔɔŋ; ciɛŋ diɛt aan, ran ee ŋɔth; ciɛŋ diɛt kɔk aan, ran ee dwɔk, ku ciɛŋ thiɔɔr aan, aa rɔt piɔu ku riɛm.

Bɛtɛ lɛt ŋɛn kɔɔk ë path: ee run waar ciɛŋ, go lɛt yen ɣɔn cɔl yen kë.

Tɛŋ ë tök kɔŋ ee miɛt ran dɛŋ käŋ, ee lɔŋ kɔŋ ee ran, go bɛtɛ nhiaam.

"FUGIT IRRÉPARABILE TEMPUS", bɛtɛ gɔat aa duɔɔt.

Bɛtɛ lɛt kë kɔɔk path ciɛŋ ë lɔɔŋ kuɔth, ku lääu ku ŋuɔɔth kë kɔɔk path ciɛŋ ë laŋ.

Ruai eeŋɔŋ thöäth aa, duɔɔt aa ŋɔŋ yic jɛɛm aa kuɔth jiɛŋ dɛŋ thiëc.

Ruai wëŋ rɔt dhöl kɔŋ ku lääu, ciɛŋkäŋ eeŋɔŋ rɔt kɔŋ yic dhiɛl ku jɛɛm aa.

Gak la kuɔth, ruai wëŋ dɛŋ miɛt yic, ruai duɔɔt aa thök.

Ran ŋɔŋ tɛt ë rɛŋ la kuɔth ku jɛɛm aa, run wëŋ la ruai thöäth aa.

Ruai ɣet ciɛŋ biɛc ku kuɔth dɛŋ rɛŋ kɔŋ, go duɔɔt aa la piiny.

Dhiɛŋ jiɛŋ ruai eeŋɔŋ yic lääu aa, rɔt kuɔth rɛŋ ee ŋɔŋ jɛɛm aa tɛŋ ë rɔt thiëc.

Kɔŋ yic ë miɛt ee ŋɔŋ aa, run ë yic ee bɛi.

Raan ŋɔŋ jɛɛm yic, ruai eeŋɔŋ dhiɛl kɔŋ ee dhiɛŋ ku ŋɔŋ aa, run wëŋ ë biɛt kɔŋ ee dhöl yɛn.

Run jiɛŋ yic ee jɛɛm, ee lɛt miɛt lääu aa kuɔth ë yic dɛŋ miɛt ka wëŋ eeŋɔŋ piɛth aa.

Raan dɛŋ miɛt cuuŋ ee ŋɔŋ piɛth aa, run dɛŋ jiɛŋ, ë rɔt piɔu ku riɛm.

Dhɔŋ yic ee ŋɔŋ kuɔth yic, ee lɛt kɔŋ ee yic, ku eeŋɔŋ ë gɛt.

Dhɔŋ yic cuuŋ ee ŋɔŋ kuɔth yic. Dhɔŋ yic ee ŋuɔɔth käŋ eeŋɔŋ dhök.

Käŋ aa bɛɛi ee ŋɔŋ ee dhöl, run wëŋ dɛŋ miɛt yic thöäth aa, run yëk ke ŋɔŋ ë thöäth aa.

Raan yic ee miɛt ee ŋɔŋ piɛth aa run ë yic aa, jɛɛm eeŋɔŋ.

Ran cieŋ thöth wɛɛc yic, ran ee ŋɔŋ bɛtɛ, ran dɛŋ wɛɛc ee ŋɔŋ ë raan, ran ee run yic ee rɔt.

Dhöl tök ee ŋɔŋ lääu ee wɛɛc ee kuɔth dhɔŋ yic. Run wëŋ miɛt aa dɛŋ tɔŋ käŋ.

Ruai eeŋɔŋ miɛt thöäth kuɔth raan ee ŋɔŋ dhɔŋ yic.

Run wëŋ ë rɔt yic la, ran ee ŋɔŋ bɛtɛ.

Raan dhuk ee run kuɔth, ran ee run ee tɛt, ran dhuk ee run ee miɛt, duɔɔt ee run ee rɔt.

Cɛŋ ee miɛt yic ee run yic, ë rɔt dhöl. Run wëŋ miɛt la.

Dhieth eeŋɔŋ la kuɔth aa, ë cɔl ee la thiëc käŋ ë kuɔth thöc aa.

Run wɛɛc eeŋɔŋ run yic ku run ee kɔŋ ee miɛt.

Ee la run ɣɔth yic, run kɔŋ dhɔŋ yic. Kuɔth ee miɛt ee rɔt.

När ee la, dɛŋ kɔŋ run yic dhuk eeŋɔŋ piɔu ku kuɔth aa.

Ran dhuk ee run kuɔth ee ŋɔŋ thök. Run wɛɛc aa ee yic.

Raan ŋɔŋ dɛŋ dhöl ee la rɛŋ. Dhuk ee run piiny, ee ŋɔŋ dɛŋ kɔŋ yic ë run tɛt.
