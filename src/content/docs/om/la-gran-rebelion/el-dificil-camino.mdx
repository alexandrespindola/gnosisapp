Dhugumatti, gama dukkanaa kan ofii keenyaa kan hin beekne yookiin hin fudhanneetu jira; ifa hubannoo gama dukkanaa ofii keenyaatti fiduu qabna.

Galmi qorannoo keenya Gnosticism hundi beekumsa ofii keenyaa caalaatti akka hubannu gochuudha.

Yeroo waan hedduu of keessaa qabnu kan hin beeknee fi hin fudhanne, wantoonni akkasii jireenya keenya garmalee walxaxaa godhuu fi dhugumatti haalawwan gosa hundumaa kan beekumsa ofiin furaa ta'an ni kakaasu.

Hunda caalaa kan nama gaddisiisu, gama hin beekamnee fi ofii keenyaa kan hin beeknetti nama biraatti mul'isuu fi achitti arginuu dha.

Fakkeenyaaf: akka waan sobdoota, amanamaa hin taane, dhiphoota ta'anitti isaan ilaalla, wantoota keessa keenyaatti fe'anne waliin walqabatee.

Gnosis waa'ee kanaa yeroo dubbatu, kutaa xiqqoo ofii keenyaa keessa jiraanna jedha.

Kunis hubannoon keenya kutaa baay'ee xiqqoo ofii keenyaa qofatti bal'ata jechuudha.

Yaadni hojii esoterica Gnostic hubannoo ofii keenyaa bal'inaan babal'isuudha.

Hanga ofii keenyaa waliin walitti dhufeenya gaarii hin qabaanneetti, warra kaanii waliin walitti dhufeenya gaarii hin qabaannu, bu'aan isaas walitti bu'iinsa gosa hundumaa ni ta'a.

Karaa kallattiin of ilaaluu dhaan ofii keenyaaf baay'ee hubannaa qabaachuun barbaachisaadha.

Seera Gnostic waliigalaa hojii esoterica Gnostic keessatti, yeroo nama tokko waliin wal hin galle, kun waan of irratti hojjechuu qabnu isa mataa isaa ta'uu isaa mirkaneeffachuu dandeenya.

Wanti warra kaan keessatti baay'ee qeeqamu wanta gama dukkanaa ofii keenyaa keessa boqotuufi kan hin beeknee fi beekuu hin barbaadnedha.

Yeroo haala akkasii keessa jirru gama dukkanaa ofii keenyaa baay'ee guddaadha, garuu yeroo ifni of ilaaluu gama dukkanaa kana ifa godhu, hubannoon beekumsa ofiin ni dabala.

Kun Karaa Qara Billaa ti, hadhaa irra hadhaa'aadha, namoonni hedduun ni jalqabu, kan galma ga'an baay'ee muraasadha.

Akkuma Ji'i gama dhokataa kan hin mul'anne qabdu, gama hin beekamne, akkasumas Ji'a Psikoloji kan keessa keenyatti fe'anne waliinis ta'a.

Ifaa ta'e Ji'i Psikoloji kun Ego, Ani, Ofii mataa kootii, Ofiif irraa kan ijaarameedha.

Ji'a psikoloji kana keessatti wantoota hamaa kan nama sodaachisan, kan nama rifachiisan, kan qabaachuuf gonkumaa hin fudhannetti fe'anna.

Karaa gara jabeenyaa kan DHUGAA DHUNFAA DHUNFAA DHUNFAA TA'UU KEENYAA ti, Gaarreen gurguddaan hammamtu jira!, Tarkaanfiin hammamtu ulfaataa dha!, Mana maree hammamtu suukanneessaadha!.

Yeroo tokko tokko karaa keessaa marsaa fi jijjiirraa hedduu booda, ol ba'iinsa rifaasisaa fi gadi bu'iinsa balaafamaa booda, gammoojjii cirrachaa keessatti ni bada, eessaa akka itti fufu hin beekamu, ifa tokkolleen si hin ibsu.

Karaa balaadhaan guutame kan keessaa fi alaa; karaa iccitii hin himamnee, bakka afuura duuti qofatu bubbisu.

Karaa keessaa kana keessatti yeroo tokko tokko baay'ee gaarii deemna jennee yoo yaadne, dhuguma baay'ee hamaa deemna.

Karaa keessaa kana keessatti yeroo tokko tokko baay'ee hamaa deemna jennee yoo yaadne, baay'ee gaarii deemaa jirra.

Karaa iccitii kana keessatti yeroo tokko tokko wanti gaariin maal akka ta'ee fi wanti hamaan maal akka ta'e hin beeknu.

Wanti yeroo baay'ee dhoorkamu, yeroo tokko tokko sirrii ta'uu ni danda'a; karaa keessaa akkasuma.

Seerri naamusaa hundi karaa keessaa keessatti ni daddarbu; jechi gaarii yookiin barumsi naamusaa bareedaan, yeroo murtaa'etti gufuu cimaa Dhugaa Dhunfaa Dhunfaa Dhunfaa ta'uu keessatti ta'uu danda'a.

Carraan gaariin Kiristoos Dhunfaa gad fagoo Ta'uu keenya keessaa hojii cimaa hojjeta, ni gadda, ni booya, wantoota balaafamaa kan keessa keenyatti fe'anne ni diiga.

Kiristoos akka daa'imaatti garaa namaa keessatti ni dhalata, garuu hanga inni wantoota hin barbaachisne kan keessa keenyatti fe'anne balleessutti, suuta suuta guddatee hanga nama guutuutti ta'a.
