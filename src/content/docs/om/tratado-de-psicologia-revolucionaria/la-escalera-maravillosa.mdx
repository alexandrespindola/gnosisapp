Dhugaa jijjiirama barbaaduu qabna, aadaa nuffisiisaa kana keessaa, jireenya mekaanikaawaa ta'e kana keessaa ba'uu qabna... Wanti duraan ifatti hubachuu qabnu, tokkoon tokkoon keenya, burjuwaa yookaan hojjetaa, qabeenya qabu yookaan kan giddu galeessaa, dureessa yookaan hiyyeessa taanus, dhugumatti Sadarkaa Jiraataa addaa addaa keessa argamna...

Sadarkaan Jiraataa machaa kan nama alkoolii hin dhugnee irraa adda, kan sagaagaltootaas kan durbaa irraa baay'ee adda. Wanti nuti jedhaa jirru kun kan hin falamsiisne, kan hin deebisne dha... Kutaa keenya keessa gara bakka kanaa yeroo geenyu, masaanuu gadii irraa olitti diriiree, dhaabbiin baay'ee dheeraa ta'e kan qabu tokko yaaduun homaa nu hin balleessu...

Shakkiin tokko illee hin jiru, masaanuuwwan kana keessaa tokko irratti argamna; masaanuuwwan gadii irratti namoonni nu caalaa gadhee ta'an ni argamu; masaanuuwwan olitti immoo namoonni nu caalaa gaarii ta'an ni argamu... Daarektara addaa kana keessatti, masaanuu ajaa'ibsiisaa kana keessatti, Sadarkaawwan Jiraataa hunda argachuun ni danda'ama... namni hundi adda adda, kana namni kamiyyuu hin deebisu...

Shakkii malee fuula fokkisaa yookaan miidhagaa irraa yookaan umrii irraa hin dubbannu. Dargaggootaa fi maanguddoota, maanguddoota du'uuf qophaa'anii fi daa'imman dhalatan jira... Dhimmi yeroo fi waggaa; dhalachuu, guddatuu, guddachuu, fuudhuu, horuu, dulloomuufi du'uu, kun daarektaraaf adda dha...

"Masaanuu Ajaa'ibsiisaa" keessatti, Daarektara keessatti yaanni yeroo hin seenu. Sadarkaa masaanuu akkasii keessatti "Sadarkaawwan Jiraataa" qofa argachuu dandeenya... Abdii mekaanikaawaa ta'e kan ummataa homaa hin fayyadu; yeroo wajjin wantoonni ni fooyya'u jedhanii yaadu; akkasitti abbootiin keenyaa fi akaakayyuun keenya yaadan; wantoonni dhugaa akkuma jiranitti faallaa isaa agarsiisaniiru...

"Sadarkaan Jiraataa" wanta lakkaa'amu dha, kun immoo daarektara dha; sadarkaa tokko irra jirra garuu gara sadarkaa biraatti ol bahuu dandeenya... "Masaanuu Ajaa'ibsiisaa" kan nuti irraa haasa'aa jirruu fi kan "Sadarkaawwan Jiraataa" adda addaa ilaalchisee, dhugumatti yeroo daarektaraawaa wajjin walitti dhufeenya hin qabu... "Sadarkaan Jiraataa" ol aanaa ta'e daqiiqaa daqiiqaan olitti nu jira...

Fuuldura daarektaraawaa fagoo keessa hin jiru, garuu asii fi amma; ofuma keenya keessa; daarektara keessa... Iftoomina qaba kan kamiyyuu hubachuu danda'u, sararoonni lamaan - Daarektarii fi Daarektarri - yeroo irraa gara yerootti keessa keenya Xiin-sammuu keessatti argamanii Fannoo uumu...

Namummaan daarektara Jireenyaa irratti guddataa fi babal'ataa. Yeroo daarektaraawaa keessatti dhalataaf du'a; kan badu dha; bor namummaa nama du'eef hin jiru; Jiraataa miti... Sadarkaawwan Jiraataa; Jiraataa mataan isaa, kan yeroo miti, sarara Daarektaraa wajjin walitti dhufeenya hin qabu; ofuma keenya keessa argama. Amma, Daarektara keessa...

Jiraataa ofii ofii keessaa ala barbaaduun ifatti kan hin yaadamne ta'a... Akka qabxii bu'aa isaatti wantoota armaan gadii kaa'uun homaa hin balleessu: Maqaa, digrii, ol guddisuu, kkf., addunyaa qaamaa alaa keessatti, karaa kamiyyuu ol guddisuu dhugaa kan hin uumnen, irra deebiin madaallii Jiraataa, gara sadarkaa ol aanaatti ce'uu "Sadarkaawwan Jiraataa" keessatti...
