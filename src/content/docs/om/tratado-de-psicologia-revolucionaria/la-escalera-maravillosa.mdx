Dhuguma jijjiirama hawwuu qabna, aadaa nuffisiisaa kana keessaa, jireenya akka maashinii qofaatti hojjennu, kan nama nuffisiisu kana keessaa ba'uu qabna...

Wanti dura hubachuu qabnu ifatti, tokkoon tokkoon keenya burjaasiis ta'e hojjataa, qabeenya qabaataanis ta'e giddu galeessaa, dureessa ykn hiyyeessa, dhuguma sadarkaa Jiraachuu keessa jirra...

Sadarkaan Jiraachuu nama machooftuu irraa adda, kan nama hin machoofnee irraa adda, sadarkaan jiraachuu dubartii sagaagalummaa hojjettuu immoo kan durbaa irraa baay'ee adda. Wanti nuti amma jennu kun kan hin falmisiifamnee fi kan hin mormisiifamne dha...

Boqonnaa keenya kana irra yommuu geenyu, manneen dhaabbii gadi irraa olitti diriiree, dhaabbataa fi manneen dhaabbii hedduu qabu yoo yaadne homaa hin dhabnu...

Hin shakkisiisu, manneen dhaabbii kana keessaa isa tokko irratti argamna; manneen dhaabbii gaditti namoota nu caalaa badaa ta'antu jira; manneen dhaabbii olitti namoota nu caalaa gaarii ta'antu argama...

Dhaabbii addaa kana keessatti, manneen dhaabbii dinqisiisaa kana keessatti, sadarkaalee Jiraachuu hundaa argachuun ni danda'ama... namni hunduu adda adda, kana immoo eenyuyyuu hin mormuu danda'u...

Dhuguma amma fuula fokkotaa ykn miidhagaa hin dubbannu, ykn dhimma umrii hin ta'u. Dargaggootaa fi maanguddoota, jaarsolii du'aaf qophaa'anii fi daa'imman dhalatanitu jira...

Dhimmi yeroo fi waggaa; dhalachuu, guddachuu, dagaaguu, fuudhuu, wal horuu, dullummaa fi du'uu, kan daarii qofaati...

"Manneen Dhaabbii Dinqisiisaa" keessatti, Dhaabbii keessatti yaanni yeroo hin galu. Manneen dhaabbii iskeelii akkasii keessatti "Sadarkaalee Jiraachuu" qofa argachuu dandeenya...

Abdii mekaanikaawaa ummataa homaa hin fayyadu; yeroo wajjin wanti caalaa gaarii ta'a jedhanii yaadu; akkasitti akaakayyuu fi abaabileen keenya yaadaniiru; dhugaan immoo faallaa ta'uu isaa agarsiiseera...

"Sadarkaan Jiraachuu" wanta lakkaa'amuudha kana immoo Dhaabbataa dha; manneen dhaabbii keessa jirra garuu manneen dhaabbii biraatti ol bahuu dandeenya...

"Manneen Dhaabbii Dinqisiisaa" kan dubbachaa jirru, kan "Sadarkaalee Jiraachuu" adda addaa ibsu, dhuguma yeroo daarii wajjin walitti dhufeenya hin qabu...

"Sadarkaan Jiraachuu" ol aanaa ta'e, battalumatti olitti kan argamu dha...

Fuuldura fagoo daarii keessa hin jiru, garuu asii fi amma; ofii keenya keessa; Dhaabbii keessatti...

Ifadha, eenyuyyuu hubachuu ni danda'a, sararri lamaan – Daarii fi Dhaabbii – yeroo hundumaa keessa keenya keessatti walitti dhufuun Fannoo uumu...

Amalli daarii jireenyaa irratti guddachaa fi babal'ataa deema. Yeroo daarii isaa keessatti dhalata fi du'a; kan baduudha; fuuldurri kamiyyuu amala nama du'eef hin jiru; Jiraachuu miti...

Sadarkaaleen Jiraachuu; Jiraachuu mataan isaa yeroo irraa miti, daarii wajjin walitti dhufeenya hin qabu; ofii keenya keessa argama. Amma, Dhaabbii keessatti...

Jiraachuu keenya ofii keenya keessaa alatti barbaaduun gatii dhabeessa ta'a...

Wanta armaan gadii akka murteessaatti kaayuun hin miidhu: maqaa, digirii, ol guddina, kkf., addunyaa qaamaa alaa keessatti, karaa kamiinuu ol guddina dhugaa, madaallii haaraa Jiraachuu, gara manneen dhaabbii ol aanaatti "Sadarkaalee Jiraachuu" keessatti hin fidu...
