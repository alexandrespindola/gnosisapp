Wanti daa'ima dhalattee cuunfaa miidhagduu fi jaalatamtuu taasisu hundinuu isa keessa jira; kun immoo dhugaa isaa dha...

Guddina cuunfaa uumama hunda keessatti argamu baay'ee xiqqaadha, kan jalqabaa dha...

Qaamni namaa akka seera baayooloojii sanyichaatti guddataa fi ni dagaaga, garuu dandeettiin akkasii cuunfaadhaaf baay'ee daangaa qabeessa dha...

Cuunfaan gargaarsa malee ofiin guddata jechuun hin falamsiisu, baay'ee xiqqoo...

Dubbii iftoominaan fi ifa ta'een yoo dubbanne, guddinni ofiin ofii fi uumamaa cuunfaa, umrii waggaa sadii, afurii fi shanii keessatti qofa danda'ama, jechuunis, sadarkaa jireenyaa isa jalqabaa keessatti...

Namoonni guddinni fi dagaaginni cuunfaa yeroo hundumaa bifa itti fufiinsa qabuun akka raawwatamu, akka makaanizimii jijjiiramaatti yaadu, garuu Gnoostisizimiin Idil-addunyaa kun akkas akka hin taane ifatti barsiisa...

Cuunfaan caalaatti akka guddatuuf, wanti addaa ta'e tokko raawwatamuu qaba, wanti haaraan raawwatamuu qaba.

Ani ofirratti hojjechuu ilaalchisee cimsee dubbachuu barbaada. Dagaaginni cuunfaa hojii of beekanii fi dhiphina fedhiin argamuu qofa irratti hundaa'a...

Hojiin kun dhimmoota ogummaa, baankii, mukaa hojjechuu, gamoo ijaaruu, daandii baaburaa sirreessuu ykn dhimmoota waajjiraa akka hin ilaallanne hubachuun barbaachisaa dha...

Hojiin kun nama dhuunfaa dagaagfateef nama hundumaaf kan oolu yoo ta'u, kan sammuu ti...

Hundi keenyaa keessa keenya keessatti waan EGO, AN, OFII KO, OFII JAALATUU jedhamu qabna...

Malaanqaan cuunfaan EGO keessatti hidhamee, cufamee argamuun isaa kan nama gaddisiisu dha.

AN Sammuu keessaa balleessuu, elementoota hin barbaachifne diiguu, waan ariifachiisaa, yeroo hin kennine, yeroo boodatti hin deebifamne dha... Kun akkaataa ofirratti hojjechuuti.

AN Sammuu diiguu malee cuunfaa gonkumaa bilisa baasuu hin dandeenyu...

Amantiin, BUUDDHAn, Ogummaan, firraa gaddaa Abbaa keenyaa kan samiirra jiru fi odeeffannoo dhumaa dhuunfaatti nama dhugoomsuuf nu barbaachisan cuunfaa keessa jiru.

Nama keessa elementoota nama hin taane qabnu balleessuu malee AN Sammuu balleessuu eenyuyyuu hin danda'u...

Balleessa suukanneessaa bara kanaa awwaalatti jijjiiruu nu barbaachisa: hinaaffaa kan malaanqaan daandii dhoksaa gochaa ta'ee argame; sassata hin dandamatamne kan jireenya baay'ee hadhaa taasise; hamii hadhaa'aa; maqaa balleessii balaa hedduu uumu; machi; fedhii foonii xurii'aa kan baay'ee foolii hamaa qabu, fi kkf., kkf., kkf.

Yeroo jibbisiisaan kun hundi awwaala koosmikaatti jijjiiramu, cuunfaan bilisa ba'uu malee, ni guddataa fi ni dagaagaa...

Yeroo AN Sammuu du'u, cuunfaan nu keessatti ifa akka qabu hin falamsiisu...

Cuunfaan bilisa ta'e miidhagina dhuunfaa nuuf kenna; miidhagina akkasii keessaa gammachuun guutuun fi Jaalalli dhugaan ni burqa...

Cuunfaan hiika guutuu fi humnoota uumamaa addaa qaba...

Yeroo "Ofiin Duunu", yeroo AN Sammuu diignu, hiikaa fi humna cuunfaa isa gati jabeessaa ni arganna...
