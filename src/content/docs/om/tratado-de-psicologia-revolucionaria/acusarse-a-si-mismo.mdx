Wanti hundumtuu keenya keessa jiru ol irraa, samii irraa, urjiilee irraa dhufa... Dhugumaan wanti ajaa'ibaa kun yaadannoo "LA" (Karaa Albaadiyaa, Galaksii keessa jiraannu) irraa dhufa.

Wanti bareedaan kun yaadannoo "SOL" (Biiftuu) keessa darbee, itti aansuun yaadannoo "FA" (Naannoo Pilaanetootaa) keessa darbee gara addunyaa kanaa gala, keessa keenyas seena. Warri keenya qaama urjiilee irraa dhufu kana simachuuf mijataa ta'e uumaniiru...

Of irratti ciminaan hojjechuu fi namoota keenyaaf aarsaa kennuudhaan, mo'ataa taanee gara garaa Uraniyaatti deebina... Nuti sababa tokkoof, waan tokkoof, sababa addaatiif addunyaa kana keessa jiraanna...

Ifatti, waan ilaaluu, barachuu fi hubachuu qabnu hedduun nu keessa jira, dhugumaan ofii keenya, jireenya keenya ilaalchisee waan tokko beekuu yoo barbaanne... Jiraachuun nama sababa jireenya isaa osoo hin beekin du'u gaddisiisaadha...

Hunduu keenya hiika jireenya isaa dhuunfaa, waan mana hidhaa dhiphinaa keessatti isa hidhee jiru ofumaaf argachuu qaba... Ifatti, hunduma keenya keessa wanti jireenya nu hadheessu fi kan irratti ciminaan qabsaa'uu qabnu jira... Gadadoon itti fufuun dirqama miti, waan nu laaffisuu fi nu gammachiisu sana gara awwaara kosmiikitti hir'isuun baay'ee barbaachisaadha.

Maqaa, kabaja, dippiloomaa, maallaqa, yaada nama gowwoomsan, amala beekamoo fi kkf qabaachuun homaa hin fayyadu. Fakkeessummaa fi boonuun gowwaa kan namummaa sobaa, nama wallaalaa, moofa'aa, boodatti hafoo, haaraa arguu hin dandeenye nu taasisuu hin dagannu.

Duuti hiika hedduu qaba, kan gaarii fi kan hamaa. Yaada ajaa'ibsiisaa "Kabir Yesus Kiristos isa Guddaa" haa ilaallu: "Warri du'an warra du'an haa awwaalan." Namoonni baay'een yoo jiraatan illee, hojii of irratti hojjetamuuf, jijjiirama gara keessaatiif du'aniiru.

Namoonni kun seera amantii fi amantaa isaaniitiin hidhamanii jiru; yaadannoo bara darbeetin dhaga'aniiru; namoota yaada dogoggoraa abbaa fi akaakayyuu isaaniitiin guutaman; namoota waan namoonni jedhanitti gabroomanii, sodaachisaa kan ta'an, dantaa hin qabne, yeroo tokko tokko "beektoota" dhugaa keessa akka jiran kan amanan, sababiin isaas akkasitti itti himamanii jiru, fi kkf.

Namoonni kun addunyaan kun "Gimnaasiyemii Saayikooloojii" akka taate hubachuu hin barbaadan, kana keessatti fokkisaa dhoksaa hundumtuu keenya keessa qabnu balleessuun ni danda'ama... Yoo namoonni hiyyeessaa ta'an kun haala gaddisiisaa keessa akka jiran hubatan, sodaachisaa ta'aniiru.

Haa ta'u malee, namoonni akkasii yeroo hundumaa waa'ee ofii isaanii waan gaarii yaadu; waa'ee amala isaanii ni dhaadatu, ofii isaanii guutuu, arjaa, nama tajaajilu, kabajamoo, nama arjoomu, ogeessa, dirqama isaa kan bahu, fi kkf ta'uutti of amanu. Jireenyi qabatamaan akka mana barumsaatti dinqisiisaadha, garuu akka dhumaatti ofumaaf fudhachuun ifatti gowwummaadha.

Warri jireenya akkuma guyyaa guyyaan jiraatutti fudhatan, barbaachisummaa "Jijjiirama Bu'uuraa" galmaan ga'uuf of irratti hojjechuu hin hubatan. Kan nama gaddisiisu namoonni karaa mekaanikaawaatiin jiraatu, waa'ee hojii keessaa homaa hin dhageenye...

Jijjiiramuun barbaachisaadha, garuu namoonni akkamitti akka jijjiiraman hin beekan; hedduu dhiphatu, sababa dhiphatanis hin beekan... Maallaqa qabaachuun hundumaa miti. Jireenyi namoota badhaadhoo baay'ee dhugumaan gaddisiisaa ta'uu danda'a...
