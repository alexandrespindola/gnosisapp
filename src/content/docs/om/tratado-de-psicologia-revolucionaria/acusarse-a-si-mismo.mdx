Namni keenya tokkoon tokkoon keessa isaanii keessatti kan qabatanii jiran hundeen isaa ol, samii, urjiilee irraa kan dhufeedha...

Dhugumatti hundeen dinqisiisaa ta'e qabxii "LA" irraa kan argamuudha (Karaa Aannannoo, Galaksii keessa jiraannu).

Hundeen bareedaan qabxii "SOL" (Adduu) keessa darba, sana booda qabxii "FA" (Naannoo Pilaaneetii) keessa darbee gara addunyaa kanaatti seenuun keessa keenya seenu.

Abbootiin keenya qaama hundee kana urjiilee irraa dhufuuf oolu uumaniiru...

Namarra keenyarratti ciminaan hojjennee namoota nu waliin jiraniif aarsaa yoo kennine, gara garaa Uraniaatti injifannoon ni deebina...

Nuti sababaaf, waan tokkoof, sababa addaatiif addunyaa kana keessa jiraachaa jirra...

Ifatti baasuuf, wantoota hedduu kan ilaalluu, barannuuf hubannu qabna, dhugumatti ofii keenyaa, jireenya keenya irratti waan tokko beekuu yoo barbaanne...

Jireenyi nama sababa jireenya isaa osoo hin beekiin du'u baay'ee nama gaddisiisa...

Tokkoon tokkoon keenya hiika jireenya isaa, wanta isa dhiphina keessatti hidhee jiru ofiisaatiin argachuu qaba...

Ifatti baasuuf, tokkoon tokkoon keenya keessa wanti jireenya nu hadheessuufi kan itti ciminnee loollu qabnu jira...

Gadadoon itti fufuun dirqama miti, wanta nu laafaa fi gaddisiisaa nu taasisu gara awwaara kosmikitti gabaabsuun hin oolamu.

maqaa, kabaja, dippiloomaa, maallaqa, sirna yaadaa gadi fageenya hin qabnee fi faayidaa hin qabnee, amala beekamaa, kkf, kkf, kkf ittiin of guddifachuun faayidaa hin qabu.

Munaafiqummaan fi boona gowwaa kan namummaa sobaa, akka nama wallaalaa, moofa’aa, duubatti harkisaa, nama jijjiiramuu hin dandeenyee nu taasisu gonkumaa hin dagannu...

Duuti hiika hedduu kan qabu yoo ta'u, hiikni isaa tokko tokkoo gaarii yoo ta'u, tokko tokkoon ammoo badaadha.

Ilaalcha dinqisiisaa "KABIR Guddaa Yesus Kiristos" irraa argamu haa ilaallu. "Warri du'an warra du'an haa awwaalan". Namootni hedduun utuu jiraatanii dalagaa dhuunfaarratti hojjennuuf waan hin oollee fi jijjiirama keessaa hin oolan.

Namootni kun dogmii fi amantaa isaanii keessatti kuulamaniiru; namootni kun yaada bara durii keessatti jajjabaataniiru; dhuunfaan kun ejjennoo durii irraa dhufeen kan guutaman yoo ta'u, namootni kun maal akka jedhan sodaataniiru, baay'ee ho'aa, tuffataa, yeroo tokko tokko "beektoota" dhugaa keessa akka jiran kan amanan sababni isaas akkasitti itti himamaniiru, kkf, kkf, kkf.

Namootni kun addunyaan kun "Gymnasium Psicológico" akka ta'e hubachuu hin barbaadan innis fokkisaa iccitii kan keessa keenyatti qabnu balleessuuf nu gargaara.

Osoo namootni hiyyeeyyiin kun haala gaddisiisaa keessa jiran kana hubataniiru ta'ee, sodaadhaan ni hollatu turan...

Haa ta'u malee, namootni akkasii yeroo hunda ofiisaaniif wanta gaarii yaadu; amala isaaniitiin ni boonu, ofii isaanii guutuu, arjaa, gargaaraa, ulfina qabeessa, nama tola oolu, dandeetti qabu, dirqama isaa kan bahu, kkf akka ta'an ni dhaga'amu.

Jireenyi hojiirra oolchuun akka mana barumsaatti dinqisiisaadha, garuu akka xumuraatti fudhachuun ifatti hin ta'u.

Warri jireenya akka jireenyatti fudhatan, akkuma guyyaa guyyaan itti jiraatan, barbaachisummaa of irratti hojjechuun "Jijjiirama Hundee" argamsiisuuf barbaachisaa ta'e hin hubanne.

Gaddisiisaadha, namootni karaa mekaanikaawaatiin jiraatu, waa'ee hojii keessaa waan tokko hin dhageenyee...

Jijjiiramuun barbaachisaadha garuu namootni akkamitti akka jijjiiraman hin beekan; baay'ee dhiphatu garuu sababa itti dhiphatanis hin beekan...

Maallaqa qabaachuun waan hundumaa miti. Jireenyi namoota badhaadhoo hedduu dhugumaan nama gaddisiisaadha...
