Haalli yeroo malee raawwatamuu qabu, kan bakka bu'iinsa hin qabnee fi kan tursiisuun hin danda'amne, marii keessaa fi iddoo sirriitti inni irraa maddetti xiyyeeffannaa kennuudha.

Marii keessaa dogoggora ta'eef sababa guddaa haala sammuu wal hin simnee fi kan hin jaallatamne yeroo ammaa fi fuulduraaf ta'uun isaa hin shakkisiisu.

Dubbiin gatii hin qabne fi wanti miidhaa geessisu, kan hin fayyadne, kan gowwaa fi kan alaan mul'atu hunduu madda isaa marii keessaa dogoggora ta'eerraa akka qabu ifaadha.

Gnosiis keessatti shaakala dhoksaa callisa keessaa kan jedhamu akka jiru ni beekama; kun barattoota keenya "Mana Hirmaataa Sadaffaa" keessaa ni beeku.

Callisi keessaa wanta murtaa'aa fi ifa ta'eef xiyyeeffannaa kennuu akka qabu ifatti dubbachuun faayidaa qaba.

Yeroo shaakala sammuu keessaa gadi fagoo keessatti adeemsi yaaduu fedhiidhaan yoo dhumate, callisi keessaa ni argama; garuu kana miti wanti boqonnaa kana keessatti ibsuu barbaannu.

"Sammuu duwwaa gochuu" ykn "adii gochuu" jechuun callisa keessaa argachuuf yeroo ammaa barreeffama kana keessatti ibsuu hin yaallu.

Callisa keessaa kan dubbachaa jirru shaakaluun, wanti sammuu keessa seenu dhowwu jechuu miti.

Dhugumaan yeroo ammaa callisa keessaa gosa addaa tokkoo irratti haasa'aa jirra. Wanta walii galaa hin taane...

Callisa keessaa wanta sammuu keessa jiru waliin wal qabsiisnee shaakaluu barbaadna, nama, taatee, dhimma mataa keenyaa ykn kan nama biraa, wanta nutti himan, wanta abalu raawwate, kkf., garuu arraba keessaa hin tuqin, haasawa dhuunfaa hin barbaachisu...

Arraba alaa qofa osoo hin taane, arraba iccitii, keessaa keessaan callisuu barachuun waan ajaa'ibsiisaa fi dinqisiisaadha.

Baay'een isaanii alaan ni callisu, garuu arraba isaaniitiin nama dhiiga isaanii dhangalaasu. Mariin keessaa summii qabu fi hamaa ta'e, wal dhabdee keessaa fida.

Yoo marii keessaa dogoggora ta'e xiinxallannee arginee, dhugaa walakkaa, ykn dhugaa walitti hidhaman haala sirrii hin taaneen, ykn wanta dabalee ykn hanqateen kan hojjetamedha.

Gaddaan jireenyi keenya miiraa "of-jaallachuu" irratti qofa kan hundaa'udha.

Dabalataan hammina kanaaf hunda keenyaaf of jaallachuu qofa, "Egoo" keenyaaf jaalala guddaa qabna, jaalala hin qabneef immoo jibbaa qabna.

Of jaallanna, dhibbeentaa dhibbaan warra Narcissistii dha, kun hin mormamu, hin haalamu.

Hanga "of-jaallachuu" keessaatti cuuphamnee itti fufnutti, guddina Uumaa kamiyyuu caalaa waan hin danda'amne ta'a.

Ilaalcha nama biraa arguu barachuu qabna. Iddoo warra kaanii keessa of galchuu beekuun barbaachisaadha.

"Kanaafuu, wanta namoonni isiniif godhan akka barbaaddan hundumaa, isinis isaaniif akkasuma godhaa." (Maatewos: VII, 12)

Wantii dhugaan barannoo kana keessatti barbaachisu, haalli namoonni walii isaaniif keessaa fi dhoksaatti itti jiraataniidha.

Gaddaan osoo kabaja qabnuu, yeroo tokko tokkoo immoo dhugaa yoo dubbanne, dhoksaatti fi keessatti walitti akka hin galle shakkii hin qabnu.

Namootni akka waan gaarii ta'aniitti mul'atan, guyyaa guyyaan namoota of fakkaatan gara holqa iccitii ofiitti harkisuun waan isaanitti tolu hundaa isaanirratti hojjetu. (Dhiibbaa, qoosuu, arrabsuu, kkf.)
