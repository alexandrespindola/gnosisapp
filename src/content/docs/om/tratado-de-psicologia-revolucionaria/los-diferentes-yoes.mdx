Bineensa harma hoosiftuu hubataa ta’e kan dogoggoraan nama jedhamu, dhugumatti dhuunfaa ifa ta’e hin qabu. Rakkoo fi hadhaa hedduu kan uumuun tokkummaa sammuu kan bineensa fakkaataa namaa keessatti argamuu dhabuudha.

Qaamni qaamaa tokkummaa guutuu dha, yoo dhukkubsate malee akka guututti hojjeta. Haa ta’u malee, jireenyi keessaa kan bineensa fakkaataa namaa gonkumaa tokkummaa sammuu hin qabu. Kan hundarra baayyee hamaa ta’e, wanta manneen barnootaa seexaa-esoterikii fi seexaa-okkultistii garagaraa jedhan faallaa, dhabamuun gurmaa’insa sammuu gadi fageenya onnee nama tokkoo keessatti.

Dhugumatti, haala akkasii keessatti hojiin walta’iinsaa akka guututti jireenya keessaa namaa keessatti hin jiru. Bineensi fakkaataan namaa, haala keessaa isaatiin walqabatee, baay’ina sammuu, walitti qabama "Ana" hedduuti.

Wallalaan hubataa bara dukkanaa kanaa "Ana" waaqeffatu, waaqessa, iddoo aarsaa irra kaa’a, "ALTER EGO", "ANA OL'AANAA", "ANA WAQAYYOO", kkf jedhee waama. "Ogeessonni" bara gurraacha keessa jiraannu "Ana Ol'aanaan" ykn "Ana Gadi aanaan" kutaa Ego baay'inaan qabu tokko keessaa akka ta’an hubachuu hin barbaadan...

Bineensi fakkaataan namaa dhugumatti "ANA Dhaabbataa" hin qabu garuu "Ana" namaa gad aanaa fi hiika hin qabne hedduu qaba. Bineensi yaadaan socho’u kan dogoggoraan nama jedhamu mannaa akka mana jeequmsaa keessa jirutti fakkaata, keessa isaas abbaan manaa tokkoo mannaa hojjetaan hedduu abboomuu fi waan isaanitti tole raawwachuu barbaadan...

Dogoggora guddaan Seexaa-Esoterizimii fi Seexaa-Okultizimii gatii hin qabnee kan biroo akka qaban ykn "ANA Dhaabbataa fi Hin Jijjiiramne" jalqabaa fi dhuma hin qabne akka qaban yaaduu dha... Namootni akkas yaadan sun yeroo muraasaaf illee yoo hubannoo dammaqanii, bineensi fakkaataan namaa hubataa yeroo dheeraaf gonkumaa tokko akka hin taane ifatti ofiisaaniitiin mirkaneessuu danda’u turan...

Bineensi harma hoosiftuu yaadaan socho’u, ilaalcha sammuutiin, itti fufiinsaan jijjiiramaa jira... Namni tokko Luis jedhama yoo ta’e Luisuma yeroo hundaa jechuun dhiichisa baay’ee hamaa ta’etti fakkaata... Dhuunfaan sun Luis jedhamu, "Ana" biraa of keessaa qaba, Egoo biraa qaba, kunis yeroo garaagaraatti amala isaa keessatti ibsama. Luis sassata jaallachuu baatus, "Ana" biraa kan isaa keessa jiru – Pepe haa jennu – sassata ni jaallata, itti fufa...

Namni tokko itti fufiinsaan tokko miti; dhugumatti jijjiiramaa fi wal faallaa hin qabne dhuunfaa tokkoo hubachuuf ogeessa ta’uun hin barbaachisu... Namni tokko "ANA Dhaabbataa fi Hin Jijjiiramne" akka qabu yaaduun, nama biraa fi ofii wajjin gocha miidhaa dha.

Nama tokko keessa namoota hedduu, "Ana" hedduutu jira, kunis nama dammaqaa, hubataa ta’e kamiyyuu ofuma isaatiin fi kallattiin mirkaneessuu danda’a.
