Bineensichi harma-hoonsisu kan yaada qabu jedhamee dogoggoraan namni jedhamu, dhugumatti dhuunfaa ifatti qabu hin qabu.

Malaammaltummaan dhabamuun tokkummaa xiinsammuu nama keessatti mul’atu kun, sababa rakkoo fi gaddaa hedduu ta’uu isaa hin shakkisiisu.

Qaamni qaamaa guutuu dha, yoo dhukkubsate malee akka guutuu orgaanizimitti hojjeta.

Haa ta’u malee, jireenyi keessaa namaa karaa kamiyyuu tokkummaa xiinsammuu miti.

Kan hunda caalaa ulfaataa ta’e, manneen barnootaa gosa Seeyidoo-Eesooteriikii fi Seeyidoo-Okkultiistii gosa garaagaraa waan jedhan irraa haala adda ta’een, dhabamuun gurmaa’insa Xiinsammuu gad fagoo keessa dhuunfaa tokkoo tokkoo keessatti mul’ata.

Dhugumatti haala akkanaa keessatti, jireenya keessaa namootaa keessatti akka guututti hojiin walsimsiisu hin jiru.

Namni, haala isaa keessaa ilaalchisee, baay’ina xiinsammuuti, walitti qabama “Ani” hedduuti.

Wallalaan barate bara dukkanaa kanaa, "Ani"tti waaqeffannaa kennu, ni waaqessu, iddoo aarsaa irra kaa'u, "ALTER EGO", "ANI OL'AANAA", "ANI WAQAA", fi kkf, kkf jedhanii yaamu.

"Ogeessonni" bara dukkanaa keessa jiraannu kanaa, "Ani Ol'aanaan" ykn "Ani Gad Aanaan" kutaa Ego isa baay'ate sana keessaa lama ta'uu isaa hubachuu hin barbaadan...

Namni dhugumatti "Ani dhaabbataa" hin qabu, garuu baay'ina "Ani" kanneen gad aanaa fi kanneen hin barbaachisne qaba.

Bineensichi sammuu qabu kan dogoggoraan namni jedhamu, manatti walitti makametti fakkaata, bakka abbaan tokko jiraachuu danda'uutti, garbichi hedduun abboomanii waan isaan barbaadan hojjachuu barbaadu...

Dogoggora guddaan Seeyidoo-Eesooteriziimii fi Seeyidoo-Okkultiiziimii gabaabaa, warri kaan qabu ykn "Ani Dhaabbataa fi Hin Jijjiiramne" jalqabaa fi dhumni hin qabne qabu jedhanii yaaduu dha...

Warri akkas yaadan kun yoo daqiiqaa tokkoofiyyuu dammaqan, namni sammuu qabu gonkumaa yeroo dheeraaf nama isa tokko akka hin taane ifatti mirkaneessuu danda’u...

Bineensichi sammuu qabu ilaalcha xiinsammuutiin yeroo hundumaa jijjiiramaa jira...

Namni tokko Luis jedhama yoo ta’e yeroo hundumaa Luis ta’uun isaa, akkuma tapha suukaneessaa tokkootti ilaalamuu danda’a...

Dhuunfaa Luis jedhamu sun, keessa isaa keessatti "Ani" kan biraa, Egoo kan biraa, kanneen adda addaa keessa isaa keessatti ifa ta'anii mul'atan, yeroo adda addaatti, Luis hammaartummaatti kan hin jaallanne yoo ta'eyyuu, "Ani" kan biraan keessa isaa jiru —Pepe haa jennu— hammaartummaatti ni jaallata, akkasitti itti fufa...

Namni tokko illee itti fufiinsaan nama isa tokko miti, dhugumatti jijjiiramaa fi walfaallaa lakkoofsaan hin daangeffamne dhuunfaa tokkoo tokkoo hubachuuf ogeessa ta'uun hin barbaachisu...

Namni tokko "Ani Dhaabbataa fi Hin Jijjiiramne" qaba jechuun, akka tasaa nama biraatti fi ofii isaatti dhiittaa raawwachuu waliin walqixa dha...

Keessa nama tokkoo keessatti namni hedduun, "Ani" hedduun ni jiraatu, kunis namni dammaqe tokko, ofiisaatiin fi kallattiidhaan mirkaneessuu danda'a...
