Namoonni HUBANNOO sammuu (CONCIENCIA) dandeetti sammuu (INTELIGENCIA) yookiin beekumsa (INTELECTO) wajjin walitti naqu, nama dandeettii sammuu guddaa qabu yookiin beekumsa guddaa qabu immoo hubannoo sammuu guddaa qaba jedhanii yaadu.

Nuti immoo HUBANNOON SAMMUU nama keessatti shakkii tokko malee, gosa addaa qabiyyee beekumsa keessoo (APREHENSIÓN DE CONOCIMIENTO INTERIOR) dalagaa sammuu irraa bilisa ta’e dha jenna.

Dandeettiin HUBANNOO SAMMUU of beekuu nu dandeessisa.

HUBANNOON SAMMUU waan jiru, bakka jiru, waan beeknu, waan hin beekne sirriitti nu hubachiisa.

PSIICOLOJII RIIVOLUUUSHINARII nama mataa isaa qofaatu of beekuu danda’a jedhee barsiisa.

Yeroo murtaa’e keessatti hubannoo sammuu qabnuu ykn hin qabnu kan beekuu danda’u nu qofa.

Nama tokkicha qofaatu hubannoo sammuu isaa fi yeroo murtaa’e keessatti jiraachuu isaa ykn dhabuu isaa beekuu danda’a.

Nama mataa isaa fi isa malee eenyu illee, yeroo tokkoof, daqiiqaa tokkoof daqiiqaa sana dura, yeroo sana dura dhugumaan hubannoo sammuu akka hin qabne, hubannoon sammuu isaa baay’ee akka rafe, booddee muuxannoo sana akka yaadan yookiin akka yaadannoo, yaadannoo muuxannoo cimaa ta’eetti akka eeggatu hubachuu danda’a.

HUBANNOON SAMMUU BEEYILADAA YAADDUU (ANIMAL RACIONAL) keessatti waan itti fufiinsa qabu, kan dhaabbataa ta’e akka hin taane beekuun ariifachiisaa dha.

Yeroo baay’ee HUBANNOON SAMMUU BEEYILADAA SAMMUU (ANIMAL INTELECTUAL) nama jedhamu keessatti gadi fageenyaan rafa.

Yeroowwan HUBANNOON SAMMUU itti dammaqu baay’ee muraasa; beeyiladaan sammuu ni hojjeta, konkolaataa ni oofaa, ni fuudhaa, ni du’aa, kkf hubannoon sammuu guutummaatti rafee fi yeroowwan addaa keessatti qofa dammaqa:

Jireenyi ilma namaa jireenya abjuuti, garuu inni dammaqaa jira jedhee yaada malee abjootaa akka jiru, hubannoon sammuu isaa rafee akka jiru gonkumaa hin amanu.

Namni tokko yoo dammaqe, ofuma isaatti baay’ee qaana’a, yeruma sana tapha isaa, safuu dhabuu isaa ni hubata.

Jireenyi kun baay’ee safuu kan hin qabne, baay’ee gaddisiisaa fi yeroo muraasa qofa kan ulfina qabu dha.

Dorgomaan boqsaa tokko lola gidduutti yeruma sana yoo dammaqe, qaana’ee ummata ulfina qabu hunda ilaalee, dinqisiifannaa tuuta rafaa fi hubannoo hin qabne duratti, agarsiisa suukanneessaa irraa ni baqata.

Ilmi namaa HUBANNOO SAMMUU RAFE qaba jedhee yeroo amanu, ammas dammaquu akka jalqabe mirkaneeffachuu dandeessu.

Manneen barnootaa ri’aakshinariittiin psiikoloojii moofa’aa jiruu fi jiraachuu HUBANNOO SAMMUU fi faayidaa dhabuu jecha sanaa kan morman, haala rafaa gadi fagoo ta’e himatu. Hordoftoonni manneen barnootaa akkasii haala gadi fagoo ta’een, haala dhiphoo ta’een rafu.

Warri hubannoo sammuu dalagaalee Psiikoloojii wajjin walitti naqan; yaadonni, miirri, kaka’umsi socho’uu fi miiraan hubachuun dhugumaan hubannoo hin qaban, gadi fageenyaan rafu.

Warri jiraachuu HUBANNOO SAMMUU kan amanan garuu sadarkaalee hubannaa addaa addaa dhabuu isaanii mormuun, muuxannoo hubannaa dhabuu, rafuu hubannaa himatu.

Namni yeroo tokkoof yoo dammaqe, sadarkaalee hubannaa addaa addaa ofuma isaatti ilaaluun akka danda’amu muuxannoo mataa isaatiin sirriitti beeka.

Yeroo duraa. Yeroo meeqaaf hubannoo sammuu qabna turre?

Yeroo lammaffaa. Baay’ina. Yeroo meeqa hubannoo sammuu dammaqnee jirra?

Sadaffaa. BAL’INA FI GADI FAGEENYA. Waa’ee maal hubannaa qabna?

PSIICOLOJII RIIVOLUUUSHINARII fi PHILOKALIA durii jabina gurguddaa addaa ta’een hubannoo sammuu dammaqsuun akka danda’amu fi itti fufiinsaa fi to’annaa jala akka oolchu ni mirkaneessu.

BARNOONNI BU’UURAA HUBANNOO SAMMUU dammaqsuu irratti xiyyeeffata. Barnootni waggaa kudhan ykn kudha shan Mana Barumsaa, Kolleejjii fi Yuunibarsiitii keessatti barannu, yeroo kutaa barumsaa irraa baanuu otomaatoomota rafan yoo taane homaa hin fayyadu.

Jabeenya guddaa tokkoon BEEYILADAA SAMMUU ofuma isaatiif daqiiqaa lamaaf qofa hubannaa sammuu qabaachuu danda’a jechuun dabarsanii dubbachuu miti.

Kun keessatti har’a haala adda ta’een Di’oojiinisii ibsaa isaa irraa kan barbaannu ta’uu isaa ifa dha, haalewwan addaa kunis DHIIRA DHUGAA, BUDDHAA, YESUUS, HERMES, QUETZACOATL, kkf irraa bakka bu’u.

hundeeffattoonni AMANTAA kun HUBANNOO SAMMUU ITTI FUFAA qabu turan, IBSA GURGUDDAA turan.

Yeroo baay’ee namoonni ofii isaanii hubannoo hin qaban. Ofii isaanii hubannaa akka qaban yaadanii dogoggoruun yaadannoo fi adeemsa yaaduu irraa maddu.

Namni jireenya isaa guutuu yaadachuuf shaakala duubatti deebi’ee yaadatu, dhugumaan yaadachuu danda’a, yeroo meeqa akka fuudhe, ilmaan meeqa akka godhate, eenyu akka ta’an, barsiisota isaa, kkf, garuu kun hubannoo sammuu dammaqsuu jechuu miti, kun gochoota hubannoo hin qabnee yaadachuu fi kana qofa.

Boqonnaalee dabran keessatti waan jenne irra deebi’uun barbaachisaa dha. Haalewwan HUBANNOO SAMMUU afur jiraatu. Isaanis: RAFUU, haala Dammaqinaa (VIGILIA), OF-HUBANNOO fi HUBANNOO KAAYYOO (CONCIENCIA OBJETIVA).

BEEYILADAA SAMMUU Hiyyeessi dogoggoraan NAMAA jedhamu, haalewwan sana keessaa lama qofa keessatti jiraata. Kutaan jireenya isaa tokko rafuu keessatti darba, inni kaan immoo haala Dammaqinaa (ESTADO DE VÍGILLA) jedhamee waamamu keessatti darba, inni kunis rafuu dha.

Namni rafaa jiru fi abjootaa jiru, haala dammaqinaatti deebi’uu isaatiin kan ka’e akka dammaqe yaada, garuu dhugumaan haala dammaqinaa kana keessatti abjootuu itti fufa.

Kun barii wajjin wal fakkaata, urjiileen ifa aduutiin dhokatu garuu iji qaamaa isaan hin argin iyyuu jiraachuu itti fufu.

Jireenya idilee keessatti ilmi namaa waa’ee OF-HUBANNOO homaa hin beeku, HUBANNOO KAAYYOO irraa caalaatti.

Haa ta’u malee namoonni boonuu dha, hunduu OF-HUBANNAA qaba jedhee yaada; BEEYILADAA SAMMUU ofii isaa hubannaa akka qabu cimaatti amana, rafaa akka jiru fi ofii isaa hubannaa malee akka hin jiraanne yoo itti himame gonkumaa hin fudhatu.

Yeroowwan addaa kan BEEYILADAA SAMMUU itti dammaqu jiraatu, garuu yeroowwan sun baay’ee muraasa, yeroo balaa guddaa keessatti, yeroo miira cimaa keessatti, haala haaraa keessatti, haala tasaa keessatti, kkf bakka bu’uu danda’u.

BEEYILADAA SAMMUU Hiyyeessi haala hubannaa darbaa sana irratti to’annaa tokko illee qabaachuu dhabuun, isaan waamuu akka hin dandeenye, itti fufiinsa akka hin godheen dhugumaan balaa dha.

Haa ta’u malee BARNOONNI BU’UURAA namni too’annaa HUBANNOO SAMMUU ARGAACHUU fi OF-HUBANNOO argachuu danda’a jedhee mirkaneessa.

PSIICOLOJII RIIVOLUUUSHINARII mala fi adeemsa saayinsawaa HUBANNOO SAMMUU DAMMAQSUU qaba.

HUBANNOO SAMMUU DAMMAQSUU yoo barbaanne, gufuuwwan karaa irratti nu qunnaman qoruun, qorachuun fi ergasii balleessuun jalqabuu qabna, kitaaba kana keessatti HUBANNOO SAMMUU dammaqsuuf karaa manneen barnootaa keessaa jalqabnee barsiisneerra.
