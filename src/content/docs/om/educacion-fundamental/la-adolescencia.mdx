Rakkoo saalaa wajjin walqabatee qaanii sobaa fi yaada dogoggoraa bara baraaf dhiisuun yeroon isaa amma.

Rakkoo saalaa dargaggoota lamaanuu akkamiin akka ta’e karaa ifaa ta’een hubachuun barbaachisaa dha.

Umurii waggaa kudha afuriitti humna saalaa kan qaama dargaggoo keessatti argamu yeroo sana sirna neewroo-simpaatiikii keessatti dhangala’a.

Humna addaa akkanaa qaama namaa jijjiiree, dhiira irratti sagalee jijjiiree, dubartoota irratti immoo hojii hanqaaquu maddisiisa.

Qaamni namaa warshaa dhugaa kan wantoota goggogaa gara wantoota jireenyaaf barbaachisoo ta’anitti jijjiirudha.

Nyaata gara garaachaatti geessinu jijjiiramaa fi qulqullina hedduu keessa darbee dhumarratti gara wanta walakkaa jajjaboo, walakkaa dhangala’aa kan Paracelsus jecha Ens-Seminis (Qaama Sanyii) jedhuun ibsetti ce’a.

Dhangala’aan halluun isaa adii, jijjiiramuu danda’u, laafaa ta’e, kunis sanyiin jireenyaa hunda qaama isaatti qaba.

Gnosticism sanyii keessatti waraansa jireenyaa keessaa akka dhufu beekti.

Alkeemistonni durii akka Paracelsus, Sendivogius, Nicolás Flamel, Raymundo Lulio, kkf ENS-SEMINIS ykn meerkurii ogummaa iccitii gadi fageenyaan ni qoratu turan.

VITRIOLO kun dhugumaan qoricha fayyaa kan uumamaan qaama sanyii keessatti oomishamudha.

Meerkurii ogummaa durii kana keessatti, sanyii kana keessatti dhugumaan wantoonni hunduu ni argamu.

Dargaggoonni baay’een gorsa qalbii dhugaa dhabuun shaakala harkaatiin ykn karaa sirrii hin taaneen saala wal-fakkaataa jaallachuutti kufuun isaanii nama gaddisiisa.

Ijoollee fi dargaggootaaf odeeffannoon sammuu waa’ee mata duree baay’ee kennamaafi karaa ispoortii irratti kan jireenya gabaabsan irratti qajeelfamni kennamaaf, garuu yeroo humna saalaa itti dargaggummaan jalqabu, maatii fi barsiisota mana barumsaa irraa kan ka’e qulqullummaa sobaa fi amala gowwaa irratti hundaa’uun yakkaan callisuu murteessu.

Callisuun yakkaa jira, akkasumas jechoonni salphinaa jiraatu. Rakkoo saalaa irratti callisuun yakka. Rakkoo saalaa irratti karaa sirrii hin taaneen dubbachuunis yakka biraadha.

Warri fi barsiisonni yoo callisan, namoonni saalaan jallatan ni dubbatu, dhiibbaa kan isaanirra ga’us dargaggoota muuxannoo hin qabneedha.

Dargaggeessi maatii ykn barsiisotatti goruu yoo hin dandeenye, hiriyoota isaa mana barumsaa keessatti kanneen karaa dogoggoraa irra jiran gaafata. Bu’aan isaa yeroo dheeraa hin eegu, dargaggeessi haaraan gorsa sobaa hordofuun shaakala harkaatti kennamee ykn karaa saala wal-fakkaataa jaallachuu irraa ka’ee karaa dogoggoraa irra deema.

Shaakala harkaatiin sammuu balleessee dhabamsiisa. Sanyiin fi sammuun walitti dhufeenya cimaa akka qaban beekuun barbaachisaa dha. Sammuu sanyii gochuun barbaachisaa dha. Sanyii sammuu gochuun barbaachisaa dha.

Sammuun humna saalaa jijjiiruun, guddisuun, humna sammuutti jijjiiruun sanyii ta’a.

Haala kanaan sanyiin sammuun ta’ee fi sammuun sanyii ta’ee hafa.

Saayinsiin Gnostiik endokrinoolojii gadi fageenyaan qorattee mala fi sirna humna saalaa jijjiiru barsiisti, garuu kun dhimma kitaaba kana keessatti hin hordofne dha.

Dubbisaan waa’ee Gnosticism odeeffannoo barbaada yoo ta’e kitaabota keenya Gnostiikii baratee barumsatti galuu qaba.

Dargaggoonni humna saalaa bareedina guddifachuun, muuziqaa, bifa, halluu barachuun, gara gaarren ol aanootti imaluun, kkf guddifachuu qabu.

Fuulli meeqa bareedaa ta’uu danda’an ni gogu!

Sammuun meeqa ni bada! Hundi isaa yeroo sirriitti iyyanno dhabuurraa kan ka’eedha.

Shaakala harkaatiin dargaggoota keessattis ta’e shamarran keessatti harka dhiqachuu caalaa babal’achaa dhufeera.

Mana yaalaa warra maraataniin guutameera, dhiirota fi dubartoota sammuu isaanii shaakala harkaatiin balleessaniin. Dhummi warra harka isaaniitiin of gammachiisanii mana yaalaa warra maraatan keessaatti argamu.

Amalli saala wal-fakkaataa jaallachuu hidda dhaloota kanaa kan du’aa fi jal’aa ta’e qacceera.

Biyyoota akka Ingiliz kanneen aadaa fi guddina olaanaa qaban of fakkeessan keessatti sinimaawwan filmii saala wal-fakkaataa jaallachuu agarsiisan bilisaan argamuun nama ajaa’ibsiisa.

Sirna gaa’elaa saala wal-fakkaataa jaallachuu seera qabeessa gochuuf Ingiliz keessatti tattaaffiin gochuun isaan ajaa’ibsiisa.

Magaalaawwan gurguddoo addunyaa keessatti mana ejjitummaa fi kilaboota saala wal-fakkaataa jaallachuu ni argamu.

Gamtaan dukkanaa’aa kan diinota dubartootaa, har’a guyyaa jaarmiyaalee jallataa kan obbolummaa jallataa isaaniitiin nama ajaa’ibsiisu qaba.

Dubbisoota hedduu “obbolummaa jallataa” kana irratti ajaa’ibsiisuu danda’a, garuu yeroo seenaa hundaa keessatti yeroo hundaa obbolummaa yakkaa gara garaatu jiraachaa turuu dagachuu hin qabnu.

Gamtaan dhibee diinota dubartootaa, shakkii tokko malee obbolummaa yakkaati.

Diinonni dubartootaa yeroo hundaa ykn yeroo baay’ee bakka murteessaa keessa seenu.

Yeroo saala wal-fakkaataa jaallatu gara mana hidhaatti deemu, yeroo dhiyootti gargaarsa namoota gamtaa yakkaa keessa jiraniin bilisa ni ta’a.

Yoo dubartoota jibbaa ta’e balaa keessa gale, yeroo dhiyootti gargaarsa dinagdee qaamolee gamtaa yakkaa hundaa irraa argata.

Miseensonni dukkanaa’aa kan saala wal-fakkaataa jaallatan uffata uffatan irraa wal beeku.

Namoonni dubartoota jibban uffata akka uffatan beekuun nama ajaa’ibsiisa, garuu akkasuma. Uffatni warra dubartoota jibban yeroo jalqabaa uffatamu kamuu ni hordofa. Namoonni dubartoota jibban uffata haarawa jalqabu. Uffatni yeroo baay’ee yoo uffatamu, kan biraa ni jalqabu. Haala kanaan uffanni gamtaa yakkaa yeroo hundaa haarawa dha.

Magaalonni gurguddoon addunyaa hundi har’a guyyaa saala wal-fakkaataa jaallatan miliyoonaan lakkaa’aman qabu.

Amalli saala wal-fakkaataa jaallachuu yeroo dargaggummaatti deemu jalqaba.

Mana barumsaa dargaggoota dhiirotaa fi shamarran mana ejjitummaa saala wal-fakkaataa jaallachuuti.

Shamarran dargaggoota miliyoonaan lakkaa’aman karaa dukkanaa’aa warra dhiiraa mormaniin ciminaan deemu.

Dargaggoonni shamarran miliyoonaan lakkaa’aman saala wal-fakkaataa jaallatu. Gamtaan yakkaa saala wal-fakkaataa jaallatu keessatti gamtaan yakkaa dhiirotaa keessa jiru jabaa dha.

Qaanii sobaa karaa murteessaa fi bara baraaf dhiisuun rakkoo saalaa dargaggoota lamaanuuf ifatti himuun barbaachisaa dha.

Dhaloonni haaraan karaa haaromsaa irra deemuu danda’u.
