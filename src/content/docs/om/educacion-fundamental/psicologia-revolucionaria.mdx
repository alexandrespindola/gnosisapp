Barsiisota dhiiraa fi dubartootaa mana barumsaa, kolleejjii fi yuunivarsiitiitti barsiisan, PSIIKOOLOJII RIIVOLUSHINARII kan Dhaabbata Gnoostiikii Idil-addunyaatiin barsiifamu gadi fageenyaan qorachuu qabu.

PSIIKOOLOJIIN RIIVOLUSHINII deemaa jiru hundumaa dura maqaa kanaan beekamurraa adda dha.

Shakkii tokko malee, yeroo "REBELDE SIN CAUSA" fi kabajamtoota ROCK irraa kaasee jaarraa nuti dura dabarsine keessatti, halkanii guddicha umrii hundaa, PSIIKOOLOJII hanga yeroo ammaatti gadi hin buune jechuu dandeenya.

Psiiyikoolojii duubatti harkisaa fi mormituu yeroo ammayyaa kanaa kanaa ol balaa irraan ga'uun hiika isaa dhabsiisee jira, akkasumas kallattiin madda isaa dhugaa wajjin walitti dhufeenya qabu hundumaa dhabsiiseera.

Yeroo hammina saalaa fi guutummaa guutuu miira manca'aa keessatti, jecha PSIIKOOLOJII jedhu qajeelummaan murteessuun dabalataan dhibaa'aa qofa hin taane, dabalataan dhimmoota bu'uuraa Psiiyikoolojii dhugaan hin beekaman.

Namoonni PSIIKOOLOJIIN saayinsii yeroo ammayyaa kan daqiiqaa dhumaa akka ta'etti dogoggoraan yaadan dhugumaan ni burjaajaa'u sababni isaas PSIIKOOLOJIIN saayinsii durii kan madda isaa mana barumsaa durii SIRROOTA ARCAIQUES keessaa qabduu dha.

Akka SNOB gosaatti, Briboon ultra-modern, duubatti harkisaa, PSIIKOOLOJII jedhamu murteessuun hin danda'amu sababni isaas yeroo ammayyaa kana malee, PSIIKOOLOJIIN gonkumaa maqaa mataa isaan hin turre sababni isaas sababa kanaa fi sanaaf yeroo hunda yaada subversives kan amala siyaasaa ykn amantaa qabu shakkiin itti uumamee fi kanaaf uffata hedduutti of dhoksuu barbaade.

Yeroo durii irraa kaasee, qabiyyee adda addaa tiyaatira jireenyaa keessatti, PSIIKOOLOJIIN yeroo hunda gahee isaa taphateera, ogummaadhaan uffata falaasamaatti of dhoksee.

Qarqara Gangaatti, Indiyaa Qulqulluu VEDAS, halkanii sodaachisaa jaarraa irraa kaasee, bifa YOGA kan bu'uuratti PSIIKOOLOJII EKISPERIMENTAALII qulqulluu ta'an jiru, balali'iinsa ol aanaa qabu.

YOGAA torban yeroo hundumaa akka malaatti ibsamaniiru, tarkaanfilee, ykn sirna falaasamaa.

Addunyaa Arabaa keessatti, barsiisaa Sagrada» kan SUFIS, gartokkeen metafizikii, gartokkeen amantaa, dhugumaan tartiiba guutummaatti PSIIKOOLOJIKII dha.

Awurooppaa durii kan lafee keessa lafeetti loluu, loogii sanyii hedduu, Amantaa, siyaasaa fi kkf qabu hanga dhuma jaarraa darbee dabarsineetti, PSIIKOOLOJIIN of dhokse uffata Falaasamaatiin agarsiisee agarsiisa malee dabree.

Falaasamni addaan qoodinsa isaa fi gargar qoodinsa isaa hundaa kan akka Loojikii, tiyoorii beekumsaa Etika, Estetika, fi kkf fakkaatu, shakkii malee mataa isaatiin, AUTO-REFLEXIÓN EVIDENTE, COGNICIÓN MÍSTICA DEL SER, FUNCIONALISMO COGNOSITIVO DE LA CONCIENCIA DESPIERTA.

Dogoggorri MANA BARSUMSAA FILOOSOFII hedduu PSIIKOOLOJII FALAASAMAAN gaditti akka laalaniidha, akka waan haala gad aanaa fi hanga xiqqaa ta'e amala namaatiin wal qabatee qofaatti.

Qorannaan wal bira qabu kan amantaa kan nu dandeessisu goolabuu loojikiitti kan SAAYINSI PSIIKOOLOJII yeroo hunda karaa dhihoo ta'een dhugumaan WALII GALTEE AMANTAA hundaan walqabataa ture. Qorannaan wal bira qabuu amantaa kamiyyuu kan nu agarsiisu LITTERAATURA SAGRADA hundumaa keessatti Ortodoksii kan biyyoota adda addaa fi yeroo adda addaatti, qabeenya dinqisiisaa saayinsii PSIIKOOLOJIKII jiru.

Qorannoon gadi fageenya lafa GNOSTICISMO nu dandeessisa argachuuf walitti qabama dinqisiisaa kan barreessitoota Gnoostiikii adda addaa kan yeroo jalqabaa Kiristaanummaa irraa dhufe kan maqaan isaa PHILOKALIA jedhamu, amma illee manni amantaa Bahaa keessatti kan fayyadamu, keessumattuu leenjii monkeef.

Shakkii malee fi sodaa xiqqoo malee gowwoomsa keessa seenuuf, ciminaan mirkaneessuu dandeenya kan PHILOKALIA bu'uuratti PSIIKOOLOJII EKISPERIMENTAALII QULQULLUU ta'uu isaa.

MANA BARSUMSAA SIRROOTA DURII Giriik, Gibts, Roomaa, Indiyaa, Peersi, Meksikoo, Peruu, Asiriiyaa, Kaldeaa fi kkf fi kkf, PSIIKOOLOJIIN yeroo hundumaa falaasamoota wajjin walqabsiifamaa ture, Artii kaayyoo dhugaa wajjin, saayinsii fi amantaa wajjin.

Yeroo durii PSIIKOOLOJIIN ogummaadhaan bifa bareedaa Shaggaattoota Qulqulluutti haguugamaa ture, ykn qaaccessa hin qabne Jeroglíficos ykn baallee bareedaa, ykn weedduu keessatti, ykn balaa keessatti fi hanga muuziqaa mi'aawaa mana qulqullummaatti.

Saayinsii, Falaasamni, Artii fi Amantaan gargar bahanii gara bilisaatti garagalchuu isaanii dura, PSIIKOOLOJIIN bilisaan MANA BARSUMSAA SIRROOTA hundaa keessatti mo'aa ture.

Yeroo Kolleejjii Eegalummaa cufame sababa KALIYUGA, ykn EDAD NEGRA kan amma illee keessa jirruutiin, PSIIKOOLOJIIN jiraate gidduutti mallattoo MANA BARSUMSAA ESOTÉRICAS adda addaa fi SEUDO-ESOTÉRICAS Addunyaa MODERNO fi keessumattuu gidduu ESOTERISMO GNÓSTICO.

Xiinxalaa gadi fageenya fi qorannoon lafa nu dandeessisa hubachuuf iftoomina guutuun sirna adda addaa fi barnoota Psiiyikoolojikii kan» darbe keessa turanii fi kan amma jiran, bakka lamatti qooduun ni danda'ama.

Tokkoffaa.- Barnoota akka intelaktuwaaloonni hedduun yaadan. Psiiyikoolojii ammayyaa dhugaa keessatti bakka kanaa dha.

Lammaffaa.- Barnoota nama qoratan gama RIIVOLUSHINII CONCIENCIA irraa kaasee.

Isaan kunniin dhugumaan barnoota duraanii dha, kan durii, isaan qofaatu nu dandeessisa hubachuuf madda jireenyaa Psiiyikoolojii fi hiika isaa gadi fageenya.

Yeroo hundi keenya hubannu bifa guutuun fi sadarkaa MIND hundaan, hangam murteessaa ta'uu isaa qorachuu nama gama yaada haaraa RIIVOLUSHINII CONCIENCIA irraa kaasee, ni hubanna yeroo sana Psiiyikoolojii qorannaa WALII GALTEE dha, seeraa fi gochootaa dhihoo walqabatan jijjiirraa RADICAL fi xumuraa INDIVIDUO.

Barattoota Dhiiraa fi Dubartootaa Mana Barumsaa, Kolleejjii fi Yuunivarsiitiitti barachuun dirqama kan hubatan bifa guutuun sa'aa CRITICA kan keessa jiraannuu fi haala BALA'AA burjaaja'uu Psiiyikoolojikii keessa Generación haaraan argamtu.
