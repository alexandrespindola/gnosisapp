Barsiisotaa fi Barsiisotni manneen barnootaa, koolleejjii fi yuunivarsiitii, XIINQALBII INQILAA BARAREE kan Dhaabbata Sochii GNOSTIC Addunyaa barsiisu gadi fageenyaan qorachuu qabu.

XIINQALBIIN INQILAA bara kanaa wantoota dura maqaa kanaan beekaman hunda irraa adda.

Shakkii tokko malee, yeroo dheeraa jaarraa hedduu kan nu duraa darbe keessatti, dukkana gadi fagoo umrii hunda keessaa, XIINQALBIIN yoomiyyuu gadi hin buune yeroo "FINCILA SABABA HIN QABNE" fi dhiirota ROCK kanaa keessatti.

Xiinqalbii duubatti hafaa fi deebii qabu kan yeroo ammayyaa kanaa dabalataan balaa ta’ee hiikkaa isaa dhabuu isaatti gaddisiisaa dha, akkasumas kallattiin madda isaa isa dhugaa wajjin walitti dhufeenya qabu hunda.

Yeroo manca’insa saalaa fi miidhaa guutuu sammuu kana keessatti, hiikaa XIINQALBII jechuu kallatti sirrii ta’een ibsuun hin danda’amu qofa osoo hin taane, dhugumaan qabiyyeewwan bu’uuraa kan Xiinqalbii hin beekaman.

Namoonni dogoggoraan XIINQALBIIN saayinsii ammayyaa yeroo dhihoo kanaa akka ta’etti yaadan dhugumaan ni dogoggoru sababni isaas XIINQALBIIN saayinsii durii hedduu kan madda isaa manneen barnootaa durii kan DHOKSAA DURII irraa qabu dha.

Akka SNOB fakkaatu, akka nama badaa hedduu ammayyaa ta’eetti, akka nama duubatti hafeetti, wanti XIINQALBII jedhamu ibsuun hin danda’amu sababni isaas yeroo ammayyaa kana malee, XIINQALBIIN maqaa isheetiin gonkumaa akka hin turre ifaa dha sababni isaas sababoota kanaaf ykn sanaaf, yeroo hundumaa sochiiwwan siyaasaa ykn amantii kan balleessaniif shakkamaa waan taateef uffata hedduutti haguugamuu qabdi ture.

Bara durii irraa eegalee, mul’ata adda addaa tiyaatira jireenyaa keessatti, XIINQALBIIN yeroo hundumaa gahee ishee taphatti, ogeessaan uffata falaasamaatti haguugamtee.

Qarqara Laga Gangesitti, Hindii Qulqulluu VEDASitti, dukkana sodaachisaa jaarraa keessaa, haalota YOGA kan bu’uuratti XIINQALBII IRRAATTI HUNDAA’E QORNOO isa ol’aanaa ta’an dha.

YOGA torban akka tooftaa, adeemsa, ykn sirna falaasamaatti ibsameera.

Addunyaa Arabaa keessatti, barumsi qulqulluun SUFIS, gartokkeen metaphysical, gartokkeen amantii, dhugumaan sadarkaa XIINQALBII guutuu dha.

Awurooppaa moofa’aa lafee tuullaa keessatti waraana hedduutti, loogii sanyii. Amantii, siyaasaa fi kkf. hanga dhuma jaarraa darbeettis XIINQALBIIN uffata Falaasamaa uffattee ifatti mul’achuu dhiisuu dandeessi turte.

Falaasamni adda baasuunii fi gargar qoqqooduu hunda keessatti loojikii, yaada beekumsaa, naamusa, miidhagina, fi kkf kan akka of keessatti qabu malee, shakkii malee OFII ISAATIIF OF YAADUU IFAA DHA, BEEKUMSA DHOKSAA KAN TA’E, DALAGAA BEKKUMSAA KAN HUBANNOO DAMMAQEEN DALAGAMU.

Dogoggora MANNEEN BARNOOTAA FALAASAMAA hedduu keessaa inni tokko xiinqalbii akka waan FALAASAMAA caalaa gadi aanaa ta’etti, akka waan kallattiin haalota gad aanaa fi hanga xiqqoo ta’an kan uumama namaa wajjin wal qabatu ta’eetti ilaaluu dha.

Qorannoon wal bira qabamee amantii nu dandeessisaa haala loojikiin Saayinsiin XIINQALBII yeroo hundumaa haala dhihoo ta’een akka PRINCIPLES AMANTII hundaan wal qabateera. Qorannoon wal bira qabamee amantii kamiyyuu nu argisiisaa akka barreeffamni qulqulluun Ortodoksii kan biyyoota adda addaa fi yeroo adda addaa keessatti qabeenya ajaa’ibsiisaa kan saayinsii XIINQALBII jira.

Qorannoon gadi fagoo dirree GNOSTICISM keessatti nu dandeessisaa walitti qabama ajaa’ibsiisaa kan barreessitoota Gnnostic adda addaa kan bara jalqabaa Kiristaanummaa keessaa kan PHILOKALIA jedhamu argachuuf, har’as MANNEEN KIRISTAANAA bahaa keessatti fayyadamu, keessattuu leenjii monoksootaatiif.

Shakkii malee fi soda xiqqoo malee gowwoomsaan kufuu, nuti cimnee mirkaneessuu ni dandeenyaa PHILOKALIA keessatti kan bu’uuraa ta’e XIINQALBII IRRATTI HUNDAA’E QORNOO qulqulluu dha.

MANNEEN BARNOOTAA DHOKSAA DURII kan Giriik, Gibxi, Roomaa, Hindii, Persia, Meksikoo, Peruu, Asiriyaa, Chaldee, kkf. fi kkf. keessatti XIINQALBIIN yeroo hundumaa falaasama, Aartii kaayyoo dhugaa, saayinsii fi amantii wajjin wal qabata ture.

Yeroo durii keessatti XIINQALBIIN ogeessaan bifa ajaa’ibsiisaa kan shubbistuu qulqulluu, ykn gidduu iccitii hiroogilaafi addaa ykn siidaa bareedaa keessatti, ykn walaloo keessatti, ykn badii keessatti fi hanga muuziqaa mi’aa’aa mana qulqullummaa keessatti dhokataa ture.

Saayinsiin, Falaasamni, Aartii fi Amantiin gargar bahanii of danda’anii gara qajeeluu isaaniitiin dura, XIINQALBIIN ol aantummaan MANNEEN BARNOOTAA DHOKSAA HUNDAA KEESSATTI NI BULCHAA TURTE.

Yeroo Kolleejjootni Jalqabbii sababa KALIYUGA, ykn bara dukkanaa kan amma iyyuu keessa jirru cufaman, XIINQALBIIN gidduu mallattoowwan MANNEEN BARNOOTAA ESOTERIC kan adda addaa fi SEUDO-ESOTERIC kan Addunyaa AMMAYYAA keessatti fi keessattuu gidduu ESOTERISMO GNÓSTICO keessatti jiraateera.

Qorannoowwan gadi fageenya qabanii fi qorannoowwan gadi fageenya qaban, nu dandeessisuu hubachuuf haala ifa ta’een sirnootaa fi barsiisoowwan Xiinqalbii adda addaa kan bara darbe keessa turanii fi ammas jiraan akka gargar qoodamaniif gara kutaa lamaatti.

Tokkoffaa.- Barsiisota haala hubannoo hubatan. Xiinqalbii ammayyaan dhugumatti kutaa kanaaf kan dha.

Lammaffaa.- Barsiisota kan nama qoratan ilaachaa INQILAA HUBANNOO irraa.

Kunneen dhugumaan Barsiisota jalqabaa dha, isa hundaa duraati, isaan qofatu nu dandeessisuu madda jireenya qabuu kan Xiinqalbii hubachuufi hiikaa isaa isa gadi fagoo.

Yeroo hunduu keenyaa hubanneera haala guutuun fi SADARKAA SAMMUU hunda keessatti, hangam barbaachisaa dha qorannoon namaa ilaachaa haaraa INQILAA HUBANNOO irraa, ni hubannaa yeroos akka Xiinqalbii qorannaa principles, seeraa fi dhugaa dhihoo wal qabatan HAALA GAAREE FI DHAABBIIFFAA TA’EEN GIDDUU GALEE nama dhuunfaa keessaa dha.

Ni ariifachiisaa akka Barsiisotaa fi Barsiisotni manneen barnootaa, Koolleejjii fi Yuunivarsiitii hubatan haala guutuun sa’aatii GADISIISAA kan keessa jiraannuu fi haala balaa kan qajeelcha dhabuu Xiinqalbii keessa kan dhaloota haaraa argamu.

Barbaachisaa dha akka "DALGAA HAARAA" gara karaa INQILAA HUBANNOO keessatti qajeelchuufi kun kan danda’amu XIINQALBII INQILAA BARAREE kan BARNOOTA BU’UURAA keessa qofa dha.
