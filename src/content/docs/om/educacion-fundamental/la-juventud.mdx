Dargaggummaan yeroo lama kan waggaa torbaa torba qoodama. Yeroon jalqabaa umrii waggaa 21tti kan jalqabu yoo ta’u waggaa 28tti xumurama. Yeroon lammaffaa waggaa 28tti kan jalqabu yoo ta’u waggaa 35tti xumurama.

Bu’uurri dargaggummaa manatti, mana barumsaatti fi karaa irratti argama. Dargaggummaan bu’uura BARNOOTA BU’UURESSAA irratti hundaa’e dhuguma ijaaraa fi bu’uuraan ulfina kennuudha.

Dargaggummaan bu’uura sobaa irratti ijaarame bu’aa isaatiin karaa sirrii hin taaneedha.

Dhiironni hedduun jireenya isaanii keessaa kutaa jalqabaa kan hafe gaddisiisuuf oolchu.

Dargaggoonni yaada dogoggoraa kan dhiirumma sobaa irraa kan ka’e, harka sagaagaltootaa keessa galu.

Darbbiin dargaggummaa xalayaa dullooma irratti dabree waggaa soddomaan booda faayidaa guddaadhaan kan kaffalamuudha.

BARNOOTA BU’UURESSAA malee dargaggummaan machii yeroo hundaa ti: ho’aa dogoggoraa, dhugaatii alkoolii fi fedhii bineensotaati.

Wanti nama jireenya isaa keessatti ta’uu qabu hundi yeroo jalqabaa umrii waggaa soddoma keessa jirutti haala dandeesisaa ta’een argama.

Gochaawwan gurguddoo namaa kan beeknu keessaa, bara duriitiis ta’e bara keenyaa, irra caalaan isaanii umrii waggaa soddoma dura jalqabamaniiru.

Namni waggaa soddomatti ga’e yeroo tokko tokko akka waan lola guddaa keessaa ba’ee hiriyoonni isaa hedduun tokkoon tokkoon kufanitti itti dhaga’ama.

Umrii waggaa soddomatti dhiironnii fi dubartoonni jireenya qabaniis ta’e kaka’umsa isaanii guutummaatti dhabu, yoo jalqaba irratti hin milkaa’in abdii kutatanii taphicha gatu.

Abdiin ga’eessumma abdiin dargaggumma bakka bu’a. Barnoota Bu’uuraa malee dhaalli dullumaa abdii kutachuudha.

Dargaggummaan kan darbuudha. Miidhaginni bareedina dargaggummaa ti, garuu kan sobaa dha, hin turu.

Dargaggummaan qaruuraa jireenya qaba, murtiin isaa immoo dadhabaadha. Jireenya keessatti dargaggoota murtiin isaanii cimaa fi qaruuraan isaanii jiru muraasa.

BARNOOTA BU’UURESSAA malee dargaggoonni kan qaanfatan, machooftota, hattu, arrabsitoota, fedhii foonii kan qaban, halaal hin taane, nyaataaf sassatoota, sassatoota, hinaaftota, abbaa humnaa, hattu, of tuultota, dhibaa’oota, fi kkf ta’u.

Dargaggummaan aduu gannaa kan dafee dhokatuudha. Dargaggoonni gatii jireenyaa kan dargaggummaa gubuu jaallatu.

Jaarsoliin dogoggora dargaggoota saamu fi gara waraanaatti isaan geggeessuu raawwatu.

Ummanni dargaggeessi yoo karaa BARNOOTA BU’UURESSAA’N geggeeffaman jijjiiramuu fi Addunyaa jijjiiruu danda’u.

Dargaggummaa keessatti abdiin guutamneerra kan gara abdii kutannootti nu geessu.

EENYUMMAAN ibiddi dargaggumma akka cimee fi humna qabuuf fayyadama.

Eenyummaan fedhii qaba, fedhii foonii gatii kamiinuu yoo dullumni guutummaatti balaa ta’e illee.

Ummata dargaggeessa fedhii foonii, daadhii wayinii fi gammachuu gosa hundaan harka kennu qofaatu barbaachisa.

Dargaggoonni akka hin hubanne fedhii fooniif garboota ta’uun kan sagaagaltoota malee kan dhiira dhugaa miti.

Gammachuun tokkollee gahaa hin turu. Beekumsi gammachuutiin beeyiladoota sammuu qabaniif waan akka malee nama xiqqeessuudha. Walaloo guddaan afaan Ispeeniin Jorge Manrique, akkas jedheera:

"Gammachuun hammam dafee deema,
akkamitti erga yaadatamuu booda,
dhukkubbii kenna,
akkamitti ilaaluu keenyaaf
yeroon darbe kamuu caalaa gaariidha"

Aristooteelis waa’ee gammachuu dubbachuun akkas jedheera: "Waa’ee gammachuu yeroo murteessan dhiironni abbaa murtii sirrii miti".

BINEENSI SAMMUU qabu gammachuu dhugoomsuudhaan ni gammada. Fedriikoo inni Guddaan iftoominaan mirkaneessuuf rakkoo hin qabu ture: "GAMMACHUUN GAAREE JIREENYAA KANA KEESSATTI DHUGAA DHA".

Dhukkubbiin obsa hin qabne kan gammachuu cimaa dheerachuu irraa kan madduudha.

Dargaggoonni mataa ol kaasan akka margaa badaa baay’atu. Eenyummaan mataa ol kaasu yeroo hundaa gammachuu ni dhugoomsa.

Namni mataa ol kaasu KANA HIN TOLCHINE Fuudha hin jaallatu ykn yeroon isaa akka dheeratu filata. Dhimmi cimaan yeroo lafa kana irraa gammachuun hundumaa gammaduuf jecha Fuudha dheeressuudha.

Dogoggora jireenya dargaggummaa dhumsuu fi ergasii fuuchuudha, dhiibbaa kanaa kan dhuma’an ijoolleedha.

Dhiironni hedduun waan dadhabaniif fuudhan, dubartoonni hedduun immoo waan beekuu barbaadaniif fuudhan, bu’aan dogoggora akkasii yeroo hundaa abdii kutachuudha.

Namni ogeessi hundi dhugaan fi onnee guutuudhaan dubartii inni filate jaallata.

Yoo dhuguma dulluma gaddisiisaa qabaachuu hin barbaannu ta’e yeroo hundaa dargaggumma keessatti fuudhuu qabna.

Jireenya keessatti waan hundumaaf yeroon jira. Dargaggeessi fuudhuun waanuma jiruu dha, jaarsi fuudhuun garuu gowwummaadha.

Dargaggoonni fuudhuu fi maatii isaanii ijaaruu beekuu qabu. Bineensi hinaaffaa maatii akka balleessu hin dagannu.

Solomoon akkas jedheera: "Hinaaffaan akka awwaalaatti gara jabeessa; qoraasni isaas qoraasa ibiddaati".

Sanyii BINEENSA SAMMUU qaban akka saree hinaaftuudha. Hinaaffaan guutummaatti BINEENSA dha.

Namni dubartii hinaafu kan isaantuu beeku hin qabu. Dubartiin akkam akka taate beekuuf hinaafuu dhiisuun wayya.

Sagaleen summaa’aa dubartii hinaaftuu ilkaan saree maraattee irraa caalaa nama ajjeessa.

Iddoo hinaaffaan jiru jaalalli jira jechuun sobaadha. Hinaaffaan jaalala irraa hin dhalatu, jaalallii fi hinaaffaan wal hin simanu. Madaan hinaaffaa sodaachuu keessatti argama.

EENYUMMAAN sababa gosa baay’een hinaaffaa ni dhugoomsa. Eenyummaan jaalatamaa dhabuuf sodaata.

Namni dhugaan EENYUMMAA dhabamsiisuu barbaadu yeroo hundaa waan jaalatamaa dhabuuf qophaa’uu qaba.

Hojii irra kan oolchinuuf waggoota baay’ee erga ilaallee booda, qeerransi bilisaa ta’e hundi abbaa warraa hinaafaa ta’a.

Namni hundi akka malee sagaagalaa ta’eera.

Dhiirrii fi dubartiin fedhiidhaan fi jaalalaan walitti hidhamuu qabu, sodaachuu fi hinaaffaan miti.

SEERAA GURGUDDAA duratti dhiirri amala isaatiif, dubartiinis amala isheetiif deebii kennuu qabu. Abbaan warraa amala dubartii isaatiif deebii kennuu hin danda’u ykn dubartiin amala abbaa warraa isheetiif deebii kennuu hin dandeessu. Tokkoon tokkoon isaanii amala isaaniitiif deebii haa kennan hinaaffaan haa dhabamuu.

Rakkoon bu’uuraa dargaggumma Fuudha.

Shamarran hawwatoo kan jaalallan hedduu qaban qeerrattii ta’u "waan hundi isaanirraa abdii kutataniif".

Shamarran yoo dhuguma fuudhuu barbaadan jaalallee isaanii eeggachuu akka danda’an barbaachisaadha.

JAALALA Fedhii FOONII wajjin walitti hin makuun barbaachisaadha. Dargaggoonni jaalatamaa fi shamarran jaalalaa fi fedhii foonii adda baasanii beekuu hin danda’an.

Fedhiin FOONII summii sammuu fi onnee gowwomsu ta’uun isaa hatattamaadha.

Namni fedhii foonii qabu hundi fi dubartiin fedhii foonii qabdu hundi imimmaan dhiigaa dhangalaasuudhaan dhuguma jaallatuu isaanii ni kakatu turan.

Erga fedhiin bineensotaa guutamee booda, gamoon kaartaa lafatti deema.

Kufaatiin fuudhaa baay’ee ta’uu isaa sababni isaa fedhii bineensotaatiin kan fuudhan malee jaalalaan miti.

Tarkaanfiin cimaan dargaggumma keessatti fudhannu Fuudha dha, Mana Barumsaa, Koolleejjii fi Yuunibarsiitii keessatti dargaggoonnii fi shamarran tarkaanfii murteessaa kanaaf qophaa’uu qabu.

Dargaggoonnii fi shamarran hedduun faayidaa dinagdee ykn fedhii hawaasaatiin fuudhuun kan nama gaddisiisuudha.

Fuudhaan fedhii bineensotaatiin ykn faayidaa hawaasaatiin ykn fedhii dinagdeetiin yoo raawwatame, bu’aan isaa kufaatiidha.

Hiriyoonni gaa’ela keessatti kufan baay’een amala wal hin simneef.

Dubartiin dargaggeessa hinaafaa, dheekkamaa fi aarii qabu yoo fuute dhibbaa nama gidirsu taati.

Dargaggeessi dubartii hinaaftuu, aariituu fi dheekkamaa taate yoo fuudhe, jireenya isaa gannata keessa dabarsuu akka qabu ifa dha.

Jaalalli dhugaan namoota lama gidduu akka jiraatuuf fedhiin bineensotaa jiraachuu hin qabu, Eenyummaa hinaaffaa dhabamsiisuun barbaachisaadha, aarii balleessuun dirqama, dantaa hin qabneessa guutuun bu’uura.

EENYUMMAAN maatii miidha, MI WAA’EEN kiyya wal tumsuu balleessa. Yoo dargaggoonnii fi shamarran BARNOOTA BU’UURESSAA keenya baratan Eenyummaa dhabamsiisuuf yoo murteessan, karaa FUDHAA GULAA argachuu akka danda’an ifa dha.

EGO dhabamsiisuudhaan qofa gammachuun dhugaa maatii keessatti argamuu danda’a. Dargaggoonnii fi shamarran fuudhaa keessatti gammachuu argachuu barbaadaniif BARNOOTA BU’UURESSAA keenya gadi fageenyaan akka baratan Eenyummaa dhabamsiisuuf gorsa ni kennina.

Abbootiin manaa hedduun intaloota isaanii akka malee hinaafan jaalallee akka qabaatan hin barbaadan. Gochi akkasii gowwummaa dhibba ta’a waan shamarran jaalallee qabaachuuf fi fuudhuu waan barbaadaniif.

Bu’aan hubannoo dhabuu akkasii jaalallee dhoksaan qabaachuudha, karaa irratti, yeroo hundaa harka galabaa gowwomsaa keessa galuuf balaa qaba.

Shamarran yeroo hundaa jaalallee isaanii qabaachuuf bilisummaa qabaachuu qabu, garuu hanga ammaatti Eenyummaa hin dhabamsiifneef jaalallee isaanii wajjin kophaa dhiisuun hin barbaachisu.

Dargaggoonnii fi shamarran mana keessatti ayyaana isaanii gochuuf bilisummaa qabaachuu qabu. Bashannanni fayya qabeessa nama hin miidhu Dargaggummaan bashannana qabaachuuf barbaada.

Waan dargaggummaa miidhu alkoolii, sigaaraa, fedhii foonii, wal hormuun, bilisummaa, mana bunaa, galma bashannanaa, fi kkf dha.

Ayyaanotii maatii, sirbboonni bareedaa, muuziqaan gaariin, bakkeewwan qonnaa dhaquun, fi kkf nama hin miidhan.

Sammuun jaalala miidha. Dargaggoonni hedduun sodaa dinagdee isaaniitiin, yaaddoo boruutiin, waan guyyaa kaleessaa yaadachuutiin dubartoota ajaa’ibsiisoo wajjin gaa’ela raawwachuuf carraa dhabaniiru.

Sodaan jireenyaa, beela, hiyyummaa fi pirojektii sammuu kan hin barbaachisne sababa bu’uuraa gaa’elli akka dheeratu taasisa.

Dargaggoonni hedduun hanga qarshii murtaa’e qabaatanitti, mana ofii, konkolaataa haaraa fi waan akkamii baay’ee hin fuudhanne yaaduun akka waan hundi isaa gammachuu ta’etti.

Dhiironni gosa sanaa sodaa jireenyaatiin, du’aatiin, waan jedhamuutiin carraa gaarii gaa’ela dhabuun isaanii kan nama gaddisiisuudha.

Dhiironni gosa akkasii qeerransii ta’anii jireenya isaanii hundaaf hafu ykn yeroon dabree erga darbee booda fuudhu, yeroon maatii ijaaruu fi ijoollee isaanii barsiisuu hin jiru.

Dhuguma wanti dhiirri dubartii isaatii fi ijoollee isaa kunuunsuuf barbaachisu hundi ogummaa ykn hojii gad aanaa qabaachuudha, sun qofa.

Shamarran hedduun abbaa warraa filachuuf jecha qeerrattii ta’anii hafu. Dubartoonni herregatoo, dantaa qaban, ofittumma qaban qeerrattii ta’anii hafu ykn fuudhaa keessatti guutummaatti kufuu.

Shamarran dantaa qabdu, herregatoo fi ofittumma qabdu irraa dhiirri hundi akka abdii kutatu hubachuun barbaachisaadha.

Dubartoonni dargaggoonni abbaa warraa qabuuf fedhii qaban fuula isaanii haala malee dibatu, gaddisa ijaa irraa fuudhu, rifeensa isaanii dhabamsiisu, rifeensa sobaa fi gaddisa ijaa kan sobaa godhatu, dubartoonni kun saayikoloojii dhiiraa hin hubatan.

Dhiirri uumama isaatiin waan dibame jibba miidhagina guutummaatti uumamaa ta’ee fi kolfi salphaan ni dinqisiifata.

Dhiirri dubartii keessatti amanummaa, salphummaa, jaalala dhugaa fi dantaa hin qabne, gowwummaa uumamaa arguu barbaada.

Shamarran fuudhuu barbaadan saayikoloojii saalaa dhiiraa gadi fageenyaan hubachuu barbaachisa.

JAALALLI ol aantummaa ogummaati. Jaalalli jaalalaan soorata. Iddoon ibiddi dargaggummaa yeroo hundaa jaalala.
