Yaadachuun waan arginee fi dhageenye, waan dubbisne, waan namoonni biroon nuun jedhan, waan nurra ga’e, fi kkf sammuu keessatti kuusuu yaaluudha.

Barsiisotaan barattoonni jechoota isaanii, himoota isaanii, waan kitaaba barnootaa keessatti barreeffame, boqonnaawwan guutuu, hojiiwwan ulfaatoo, qabxii fi mallattoo addaan kutaa isaanii hunda, kkf yaadachuuf akka barbaadan.

Qormaata darbuun waan nuun jedhan, waan karaa maayikroo dubbisne yaadachuun, yaada afaaniin ibsuun, akka kiyyoo ykn qaaqaa deddeebi’uun, waan sammuu keenya keessatti kuufanne hunda irra deebi’uudha.

Dhaloonni haaraan waan sammuu keessatti galmeeffame hunda akka diraamaa raadiyoo irra deebi’uu jechuun sirriitti hubachuu miti. Yaadachuun hubachuu miti, hubannoo malee yaadachuun faayidaa hin qabu, yaadannoon kan darbee kan du’e, waan jireenya hin qabneedha.

Barattoonni mana barumsaa sadarkaa tokkoffaa, giddugaleessaa fi yuunivarsiitii hunda hiika gadi fagoo hubannoo sirriitti hubachuun barbaachisaa dha, ariifachiisaa dha, yeroo ammaa kana keessatti baay’ee barbaachisaa dha.

HUBACHUUN wanta battalumatti raawwatamu, kallattiidhaan kan raawwatamu, wanta gadi fageenyaan itti fayyadamnu, wanta gadi fageenyaan muuxannu fi wanta hin oollee madda dhugaa tarkaanfii of-beekaa ta’ee dhufuudha.

Yaadachuun, irra deebi’anii yaadachuun waan du’e, kan darbeedha, kan nama gaddisiisu immoo akka dhaadannoo, akka yaadotaatti, akka idealizimii karaa maayikroo hordofuuf yaallu fi of-beekumsa malee itti fufuuf yaallu ta’a.

HUBACHUU DHUGAA keessatti, hubannoo gadi fagoo keessatti, hubannoo keessoo isa gadi fagoo keessatti dhiibbaa keessoo of-beekumsaa qofaatu jira, dhiibbaa dhaabbataa kan bu’uura nuti keessa keenyaatti qabnu irraa dhalate, kunis hunda.

Hubannoon dhugaa akka tarkaanfii of-tumaamee, uumamaa, salphaa, adeemsa gaddisiisaa filannoo irraa bilisa ta’eetti mul’ata; qulqulluu kan murtii adda addaa hin qabne ta’ee. HUBACHUUN madda iccitii tarkaanfii ta’ee jijjiiramuun dinqisiisaa, ajaa’ibsiisaa, ijaaraa fi bu’uuratti ulfina kennuudha.

Tarkaanfiin waan dubbisne, idealii itti hawwinu, seericha, amala nu barsiisan, muuxannoowwan sammuu keessatti kuufanne, kkf irratti hundaa’e, shallagaa dha, filannoo gaddisiisaa irratti kan hundaa’e, lama ta’uu irratti kan hundaa’e, filannoo yaadaa irratti kan hundaa’e yoo ta’u, dogoggoraa fi dhiphinaatti qofa geessa.

Tarkaanfii yaadannootti madaquun, tarkaanfii jijjiiruun yaadannoowwan sammuu keessatti kuufaman wajjin akka wal simu gochuun wanta hin barbaachisne, homaa hin qabne, of-tumaamee kan hin qabne yoo ta’u, kan nu geessu dogoggoraa fi dhiphinaatti qofa.

Qormaata darbuun, waggaa darbuun, nama gowwaa kan malaa fi yaadannoo gaarii qabuun raawwatama.

Barumsa baranne hubachuun fi isa irratti qoramuuf jirru wanta baay’ee adda ta’e, yaadannoo wajjin walitti dhufeenya hin qabu, kan sammuu dhugaa kan ogummaadhaan wal hin wallakkaa’amne dha.

Namoonni gocha jireenya isaanii hunda idealii, yaaxinoota fi yaadannoowwan gosa hundaa kan godoo yaadannoo keessatti kuufaman irratti hundeeffachuuf barbaadan, yeroo hunda wal bira qabuun deemu, iddoo wal bira qabuun jiruutti immoonis ni jiraata. Namoonni kun namoota isaanii, maatii isaanii, ijoollee isaanii ijoollee ollaatiin, namoota ollaa wajjin wal bira qabu. Mana isaanii, meeshaalee manaa isaanii, uffata isaanii, waan isaanii hunda, waan ollaatiin ykn waan ollaatiin ykn waan nama dhihoo wajjin wal bira qabu. Yaada isaanii, ogummaa ijoollee isaanii yaada namoota biroo wajjin, ogummaa namoota biroo wajjin wal bira qabu, immoonis ni dhufa, inni immoo madda iccitii gochaa ta’a.

Addunyaa kanaaf gaddisiisaa, mekaanizimiin hawaasaa hundi immoo immoo fi hafuura argachuu irratti hundaa’a. Addunyaan hundi addunyaa hunda immoo. Yaada, wantoota, namoota immoonna, qarshii fi qarshii dabalataa argachuu, yaaxinoota haaraa, yaada haaraa sammuu keessatti kuufannu, waantoota haaraa kan fakkeenya keenya dinqisiisan argachuu barbaanna, kkf.

HUBACHUU DHUGAA keessatti, kan seera qabeessaa, kan dhugaa keessatti jaalala dhugaa tu jira malee yaada afaaniin ibsuu qofa miti.

Wantoonni yaadatan, waan yaadannoo keessatti kuufaman, dafanii ni irraanfatamu sababni isaas yaadannoon amanamaa miti. Barattoonni godoo yaadannoo keessatti, idealii, yaaxinoota, barreeffamoota guutuu kan jireenya keessatti faayidaa hin qabne, sababni isaas dhuma irratti yaadannoo irraa bakka tokko illee hin hambifne.

Namoonni dubbisuu fi karaa maayikroo dubbisuu qofa jiraatan, namoonni yaaxinoota godoo yaadannoo keessatti kuusuu kan jaallatan sammuu ni balleessu, haala gaddisiisaa ta’een ni miidhu.

Nuti qorannoo dhugaa gadi fagoo fi of-beekaa kan hubannoo gadi fagoo irratti hundaa’e irratti hin dubbanne. Nuti maloota barnootaa bara dabree kan hin barbaachisne qofa balaaleffanna. Sirna barumsaa maayikroo hunda, yaadannoo hunda, kkf balaaleffanna. Iddoo hubannoo dhugaa jiruutti yaadachuun ni dhiifama.

Barachuun barbaachisaa dha, kitaabni faayidaa qabeessa ta’e ni barbaachisa, barsiisota mana barumsaa, koolleejjii, yuunivarsiitii ni barbaachisa. GURUU, qajeelchitoota hafuuraa, mahaatmota, kkf ni barbaachisa, garuu barumsa guutuu ta’een hubachuun barbaachisaa dha malee godoo yaadannoo amanamaa hin taane keessatti kuusuu qofa miti.

Hanga amala wal bira qabu kan yaadannoo keessatti kuufame, idealii, waan ta’uu hawwinu, fi waan hin taane wajjin wal bira qabuu dhiifna, dhugumaan bilisa ta’uu hin dandeenyu.

Yeroo dhugaan barumsa argachuu hubannutti, yaadannoo keessatti yaadachuun, idealii taasisuunis nu hin barbaachisu.

Iddoo wanta asiin fi amma ta’een wajjin waan booda ta’uu barbaannu wal bira qabuun jiruutti, iddoo jireenya keenya hojiin idealii ykn fakkeenya itti madaquuf barbaannu wajjin wal bira qabuun jiruutti jaalalli dhugaa hin jiraatu.

Waliin dorgomuun hundi jibbisiisaa dha, waliin dorgomuun hundi sodaa, immoo, boona, kkf fida. Sodaa waan barbaanne galmaan gahuu dhabuu, immoo guddina nama biraatti, boona waan warra kaan caalaa of ilaalluuf. Wanti jireenya hojii keessatti barbaachisaa ta’e, haala fokkisaa ta’een, immoo qabaannus, ofittummaadhaan, sassattummaadhaan, kkf jiraannus, qulqulluu of goona jechuun, eessaa iyyuu hin jalqabnu, ofuma keenya sirriitti hubachuudha, akka jirru malee akka ta’uu barbaannu ykn akka of goonu osoo hin taane.

OF, OFUMAA dhabamsiisuun hin danda’amu, of ilaaluu, hubachuuf beekuu yoo hin baranne malee wanta dhugaa asiin fi amma ta’een haala bu’aa qabeessa ta’ee fi guutummaatti hojiirra ooluun jiraannu.

Dhuguma hubachuu yoo barbaanne barsiisota keenya, barsiisota, guruta keenya, luboota, gorsitoota, qajeelchitoota hafuuraa, kkf dhaggeeffachuu qabna.

Dargaggoonni, durbartoonnis kabaja, ulfina abbootii keenya, barsiisota keenya, qajeelchitoota hafuuraa, guruta, mahaatmota, kkf dhabaniiru.

Abbootii keenya, barsiisota, gorsitoota ykn qajeelchitoota hafuuraa ulfina kennuu fi kabajuu yoo hin beekne barumsa hubachuun hin danda’amu.

Yaadannoo maayikroo waan baranne hubannoo gadi fagoo malee sammuu fi onnee ni jijjiira, immoo, sodaa, boona, kkf ni dhala.

Yeroo dhugaan of-beekumsa fi gadi fageenyaan dhaggeeffachuu beeknu humna ajaa’ibsiisaa, hubannoo dinqisiisaa, uumamaa, salphaa, adeemsa maayikroo hundumaa irraa bilisa ta’e, cerebration hunda irraa bilisa ta’e, yaadannoo hunda irraa bilisa ta’e nuti keessa keenyatti ni uumama.

Yoo sammuun barataa irra dadhabbi yaadannoo guddaa irraa bilisa ta’e, caasaa bu’uuraa fi gabatee piriyoodikaalii elementootaa barattoota barnootaa lammaffaa barsiisuun guutummaatti danda’amaa dha, walqixxummaa fi Quanta immoo barataa sadarkaa tokkoffaatti hubachiisuun danda’amaa dha.

Akkuma barsiisota mana barumsaa sadarkaa lammaffaa tokko tokko wajjin haasofnee beeknu, barnoota moofaa bara dabree kan hin barbaachisne soda guddaadhaan ni na’u. Barattoonni hundi yaadannoon akka baratan barbaadu malee yoo hin hubanne illee.

Yeroo tokko tokko hubachuun yaadachuu irra akka wayyu ni fudhatu garuu yeroo sana foormulaawwan fiiziksii, keemistirii, herregaa, kkf yaadannoo keessatti galmeeffamuu akka qaban ni jajjabeessu.

Yaadni kun sobaadha, sababni isaas foormulaan fiiziksii, keemistirii, herregaa, kkf yeroo haalaan hubatamutti sadarkaa sammuu qofa irratti osoo hin taane sadarkaa sammuu kan akka sammuu keessaa, sammuu gadii, sammuu gad-aanaa kkf irrattis yaadannoo keessatti galmeessuun hin barbaachisu, psyche keenya keessaa tokko ta’ee ni dhufa, haala beekumsa battalaa of-tumaamee ta’een mul’achuu danda’a yeroo haalli jireenyaa gaafatu.

Beekumsi GUUTUU kun haala OMNISCIENCE, haala ifa mul’isuu of-beekaa nuuf kenna.

Hubannoon gadi fagoo fi sadarkaa sammuu hunda irratti kan danda’amu, xiinxala introspective gadi fageenyaan raawwachuudhaani.
