Dhugaa jiru hundaan hubachuun, akkasumas iddoo yaadni keenya hundinuu argamu keessatti, duuti ofumaan maal akka ta'e sirriitti hubachuun barbaachisaadha. Kana gochuun qofaatu dhugumaan duuti hin jiru jechuun maal jechuu akka ta'e guutummaatti hubachuun danda'ama.

Reeffa nama jaallannuu sanduuqa keessa galchanii arguu jechuun iccitii du'aa hubachuu jechuu miti.

Dhugaan wanta yeroo amma hin beekamnedha. Dhugaan du'aa waa'ee kanaa keessaa hin hafu.

"ANI" yeroo hundumaa, akkuma uumama isaatti, wabii du'aa, mirkaneessa dabalataa, abbaa taayitaa iddoo gaarii nuu mirkaneessuu fi du'a sodaachisaa sana booda duuti hin jiru jechuu kamiyyuu nuu mirkaneessu barbaada.

"OFII" du'uuf fedhii guddaa hin qabu. "ANI" itti fufuu barbaada. "ANI" du'aaf soda guddaa qaba.

Dhugaan dubbii amanuu yookaan shakkii miti. Dhugaan amanamummaa yookaan shakkitummaa wajjin walitti dhufeenya hin qabu. Dhugaan dubbii yaadaa, tiyoorii, ilaalcha, yaada, yaada duraa, tilmaama, loogii, mirkaneessa, marii fi kkf miti. Dhugaan waa'ee iccitii du'aa keessaa hin hafu.

Dhugaan waa'ee iccitii du'aa karaa muuxannoo kallattiin qofa beekamuu danda'a.

Muuxannoo DHUGAA du'aa nama hin beekneef ibsuun hin danda'amu.

Qeeqa bardeesaan kitaaba bareedaa JAALALAA barreessuu danda'a, garuu DHUGAA waa'ee JAALALAA nama isa hin muuxanneef ibsuun hin danda'amu, haaluma walfakkaatuun dhugaa waa'ee du'aa nama isa hin muuxanneef ibsuun hin danda'amu jennu.

Namni dhugaa waa'ee du'aa beekuu barbaadu qorachuu, ofumaan muuxannoo argachuu, sirriitti barbaaduu qaba, karaa kana qofa hiika guddaa du'aa argachuu dandeenya.

Hubannoon fi muuxannoon waggoota hedduu akka hubannu nu taasise namoonni hiika guddaa du'aa hubachuuf fedhii hin qaban; wanti namoota dhugumaan barbaachisu hunda caalaa itti fufuudha, kunis hunda.

Namoonni hedduun qabeenya, kabaja, maatii, amantii, yaada, ijoollee fi kkf fayyadamuun itti fufuu barbaadu, itti fufiinsi xiinqaa gosa kamiyyuu gatii hin qabne, kan darbu, kan hin dhaabbanne, kan hin tasgabbaa'ne ta'uu yommuu hubatan, mirkana malee, nageenya dhabanii, rifatanii, na'anii, soda guddaan guutamu.

Namoonni hiyyeeyyiin hubachuu hin barbaadan, wanti itti fufu hundi yeroo keessatti akka babal'atu hubachuu hin barbaadan.

Namoonni hiyyeeyyiin wanti itti fufu hundi yeroo wajjin akka gadi bu'u hubachuu hin barbaadan.

Namoonni hiyyeeyyiin wanti itti fufu hundi makaanikaawaa, kan barame, kan nama nuffisiisu akka ta'u hubachuu hin barbaadan.

Hiika guddaa du'aa guutummaatti beekuun ariifachiisaadha, barbaachisaadha, dirqama. Karaa kana qofa soda dhabuun ni bada.

Nama hunda sirriitti ilaaluun, yaanni keenya yeroo hunda wanta beeknu keessatti hidhamee akka argamu fi wanti beeknu kun hunda caalaa itti fufuu akka barbaadu mirkaneessuu dandeenya.

Yaanni wanta beeknu keessatti hidhame, wanta hin beekamne, kan dhugaa, kan dhugaa gonkumaa muuxannoo hin argatu.

Madaallii sirrii fayyadamuun qaruuraa yeroo cabsuun qofa, Wanta BARABARAAMAA, kan YEROO HIN QABNE, kan DHUGAA muuxannoo argachuu dandeenya.

Warri itti fufuu barbaadan du'a sodaatu, amantaanii fi tiyooriiwwan isaanii qoricha dhiphinaa isaanii qofaaf gargaaru.

Duuti ofumaan homaa sodaachisaa hin qabu, waan baay'ee bareedaa, ol aanaa, kan hin ibsamnedha, garuu yaanni wanta beeknu keessatti hidhame: amanamummaarraa hanga shakkitummaatti marsaa hamaa keessa qofa socho'a.

Yommuu hiika guddaa fi gadi fagoo du'aa dhugumaan guutummaatti beeknu, yeroo sanatti ofuma keenyaa karaa muuxannoo kallattiin, Jireenyi fi Duuti guutummaa guutuu, walitti makamuu tokko ta'uu arganna.

Duuti bakka Jireenyi kuufamudha. Daandiin Jireenyaa faana harma du'aan uumame.

Jireenyi humna murtaa'aa fi murteessaadha. Dhalootarraa hanga du'aatti gosa humna adda addaa qaama namaa keessa yaa'a.

Gosa humna qaamni namaa dandamachuu hin dandeenye gosa tokkicha, Ibsaa DU'AATI. Ibsaan kun voltii elektirikii baay'ee ol'aanaa qaba. Qaamni namaa voltii akkasii dandamachuu hin danda'u.

Akkuma ibsaan muka tokko caccabsuu danda'u, akkasumas ibsaan du'aa qaama namaa keessa yaa'uun, ni balleessa.

Ibsaan du'aa wanta du'aa ta'e, wanta dhaloota ta'e wajjin walitti qaba.

Ibsaan du'aa dhiibbaa elektirikii baay'ee gadi fagoo fi sagalee furtuu murteessa ta'e kan dandeettii gene keessa buubsan qaba.

Ibsaan du'aa qaama namaa elementoota bu'uuraatti gadi qaba.

"ANUMMAAN", "ANUMMAAN" humna qabu, dhaloota keenya keessa itti fufa, kun immoo kan nama gaddisiisudha.

Dhugaan waa'ee du'aa maal akka ta'e, wanti du'aa fi ulfaa'uu gidduu jiru yeroo kan hin qabne fi karaa saayinsii madaallii qofaan muuxannoo argachuu dandeenyudha.

Barsiisota fi Barsiisotni Mana Barumsaa, Kolleejjii fi Yunivarsiitii daandii muuxannoo DHUGAA, DHUGAA geessu barattoota isaanii barsiisuu qabu.
