Dhugaa kana sirriitti fi iddoo yaadaa hunda keessatti hubachuun barbaachisaa dha, kunis DUUTI maal akka ta’e ofuma isaatiin, akkasumas dhugumaan hiika guutuu du’aa booda jiraachuu hubachuun kan danda’amu yoo ta’e qofa dha.

Qaama nama jaallatamaa sanduuqa keessa galfame arguu jechuun iccitii du’aa hubatte jechuu miti.

Dhugaan wanta yeroo amma hin beekamneedha. Dhugaan du’aa irraa kan hafe ta’uu hin danda’u.

"ANI" yeroo hunda, akkuma uumamaan ta'ee, wabii du'aa, mirkana dabalataa, abbaa aangoo iddoo gaarii nu mirkaneessuu fi du'aa booda immoo du'a sodaachisaa irraa bilisa nu taasisu barbaada.

"ANUMTI" du'uuf fedhii hin qabu. "ANI" itti fufuu barbaada. "ANI" du'aaf soda guddaa qaba.

DHUGAA jechuun waan itti amanuu ykn shakkii qabu miti. Dhugaan waan amanuu fi shakkuu wajjin walitti dhufeenya hin qabu. Dhugaan yaada, tiyoorii, yaada, hubannoo, dursaa, tilmaama, jibbaa, ibsa, marii, fi kkf miti. Dhugaan iccitii du'aa irraa kan hafe miti.

Dhugaan iccitii du'aa keessatti argamu, muuxannoo kallattiin argamuun qofa beekamuu danda’a.

Muuxannoo DHUGAA du'aa nama hin beekneef ibsuun hin danda'amu.

Walaloo barreessaan kamiyyuu kitaaba jaalalaa bareedaa barreessuu danda'a, garuu DHUGAA jaalalaa namoota matuma isaaniitiin hin muuxanneef ibsuun hin danda'amu, haaluma walfakkaatuun dhugaa du'aa namoota hin muuxanneef ibsuun hin danda'amu jenna.

Namni dhugaa du'aa beekuu barbaadu ofuma isaatiin qorachuu, muuxannoo qabaachuu, sirriitti barbaaduu qaba, karaa kanaan qofa hiika guddina du'aa argachuu dandeenya.

Hubannoo fi muuxannoo waggoota dheeraa keessatti argameen, namoonni hiika guddina du'aa dhugaan hubachuuf fedhii akka hin qabne hubanneerra; namoota dantaa isaanitti kennu keessaa isa guddaan gara fuulduraatti itti fufuu qofaadha, kunis hunda caalaa.

Namoonni hedduun qabeenya, kabaja, maatii, amantaa, yaada, ijoollee, fi kkf’n itti fufuu barbaadu, yeroo itti fufiinsi xiinsammuu kamiyyuu gatii hin qabne, kan darbu, kan hin dhaabbanne ta'uu hubatanitti, wabii tokko malee, kan hin mirkanoofne ta’anii sodaatanii, gubatu, soda guddaan guutamu.

Namoonni hiyyeeyyiin hubachuu hin barbaadan, wanti itti fufu hundi yeroo keessatti akka babal'atu hubachuu hin barbaadan.

Namoonni hiyyeeyyiin wanti itti fufu hundi yeroo wajjin akka badaa deemu hubachuu hin barbaadan.

Namoonni hiyyeeyyiin wanti itti fufu hundi makaanikaawaa, kan barame, kan nama nuffisiisu ta'uu hubachuu hin barbaadan.

Hiika guddina du'aa guutummaatti beekuun barbaachisaa, dirqamaa, fi murteessaa dha, karaa kanaan qofa sodaan dhabamuu ni bada.

Ilmaan namootaa sirriitti ilaaluun, yaanni yeroo hunda wanta beekamu keessatti akka kufee fi wanti beekamu sun du'aa booda itti fufuu akka barbaadu mirkaneessuu dandeenya.

Yaanni wanta beekamu keessatti kufe, gonkumaa wanta hin beekamne, dhugaa, kan dhugaa hin muuxannu.

Qaruuraa yeroo xiinxala sirrii ta'een cabsuun qofa EETERNAAL, ATEMPORAL, DHAABATAA ta'e muuxannoo qabaachuu dandeenya.

Warri itti fufuu barbaadan du'aaf sodaatu, amantaan isaanii fi tiyooriin isaanii qoricha hirribaa isaaniif qofa ta'a.

Duuti ofuma isaatiin waan sodaachisaa tokkollee hin qabu, waan baay'ee bareedaa, ulfina qabuu fi kan ibsuun hin danda'amnedha, garuu yaanni: wanta beekamu keessatti kufee, amanuu irraa gara shakkuutti deemuun daandii dogoggoraa keessa qofa socho'a.

Yeroo dhugaan hiika guddaa fi gadi fagoo du'aa guutummaatti hubannutti, ofuma keenyaan muuxannoo kallattiin, Jireenyaa fi Duuti guutuu, tokkummaa-gutuun ta'uu arganna.

Duuti bakka Jireenyi itti kuufamuudha. Daandiin Jireenyaa faana lukkuu duutiin uumameera.

Jireenyi humna murtaa'aa fi murteessaa dha. Dhalootaa hanga duuti qaama namaa keessatti gosa humna adda addaa keessa yaa'a.

Gosti humna qaama namaa danda'uu hin dandeenye isa qofa RAAYYOO DU'AA dha. Raayyo kun voltajjii elektirikii baay'ee ol'aanaa qaba. Qaamni namaa voltajjii kana danda'uu hin danda'u.

Akkuma raayyo muka caccabsuu danda'u, raayyon du'aas yeroo qaama namaa keessa yaa'utti, dhumaatii hin oolle dha.

Raayyon du'aa mul'ata du'aa wajjin, mul'ata dhalootaa walitti qabsiisa.

Raayyon du'aa dhiibbaa elektirikii baay'ee gadi fagoo fi nota murteessaa tokko kan humna qabu, jeenota bu'uuraa keessaatti walitti qabuuf sababa ta'a.

Raayyon du'aa qaama namaa gara wantoota bu'uuraatti hir'isa.

EGO, ANUMNI humna qabu, dhaloota keenya keessatti itti fufa, kan nama gaddisiisu.

Dhugaan du'aa maal akka ta'e, gidduu du'aa fi ulfaa'uu gidduu jiru kan yeroo keessaa hin taanee fi saayinsii xiinxalaan qofa muuxannoo qabaachuu dandeenya.

Barsiisotaa fi barsiistonni Mana Barumsaa, Kolleejjii fi Yuunivarsiitii, daandii gara muuxannoo DHAABATAA, DHUGAA geessu barattoota isaanii barsiisuu qabu.
