## DUBBISA
### BARAACHA BARSIIFATA GARGHA KUICHINES IRRAA

Astirolojiin Saayinsaawaa ykn Lakkoofsaan kan barbaachisu barumsa bal'aa barachuuf, isaan Astrarolojistoonni bara hundaa keessaatti taateewwan barbaachisoo ta'an ilaalchisee raaggaa ni dubbatu. Akkasumas beekumsa kana irraa faayidaa kan argatan namoota guguddoo akka kan Hitlera Jarman keessatti Astrarolojistoota addaa ta'an waraana isaanii keessatti akka isaan geggeessan godhaniiru.

Nutii Gnostics barnoota kana irraa adda baanaan, sababiin isaas namichi akka taphaatti kan ilaalamu raaggaa irratti hundaa'uun, mallattoolee fi bakka bu'oota addeessota adda addaa keessa darbuu irraa ofirraa ittisuu hin danda'u, nutii Astirolojiin nu barsiisuu addeessotaan haala itti fayyadamnu nu barsiisa, akkasumas taateewwan Astirolojii Lakkoofsotaatiin addaa ta'an raaggaa irraa fagaachuu dandeenya. Kanaafis qaamota ji'aa nuti ittiin dhalannee, qaamota biiftuu ykn ifaa gara guddina keenyatti jijjiiruun barbaachisaadha.

Diinni beekumsa Waaqayyoo guddina nurraa dhorku, ani seexanaa ykn hoogganaa raayyaa qaama keenya kan bulchu dha. Karaan gaariin tiksee balbala irraa dhabamsiisuuf kan akka nama harka bitaa qabuuf jalqabsiisa, innis nuti Gnostics galma ykn Luma keessa seenuun jalqaba, barumsi inni geggeessuun fi fuulduratti olqabuun Avatar Aquarius "Samael Aun Weor".

Yeroo jalqabaaf Yesus barumsa isaa gara jaalala karaa irratti kennutti, "Ani karaadha, ani dhugaadha, ani jireenyadha" jedhe, yeroo sana osoo hin darbin falfala guddaan kan Simoon Magician jedhamu, humna guddaa fi badhaadhina qabu, bartoota isaa akkana jedhaan ture: "Yoo Yesus abbaa isaa karaa isaa dhuunfaa irratti gahe, ani Simoon, akkasumas abbaa kootti karaa koo dhuunfaa irratti nan gaha, inni immoo karaa bitaa irratti hordofuun, jaalala isaa irraa fagaachuun kan hojjatu, balaan kun yeroo jaalalaa karaa irratti hordofan isaan dhuunfaadhaan karaa kana ce'uu danda'u jedhanii amanan yeroo hundaa ni dhufa.

Warreen yeroo jalqabaaf ogummaa kana harka isaaniitti galchan, ji'oota ja'aa barumsa jalqabaa, hojii Samael Aun Weor barachuun fi beekumsa isaanii hojiirra oolchuun isin barbaachisa. Sana booda yoo isin dhugumaan galmaan gahuu barbaaddan, fi jireenya ol'aanaa yoo barbaaddan, luma keessa seensa isiniif kennama adabbii addaatiin.

Yeroo nama karaa irratti hordofuuf qophaa'eef, qormaanni jalqabaa inni darbuu qabu tiksee balbalaa ykn seexana isaa ofii wajjin walitti bu'iinsa, kan nu geggeesse fi barsiisaa keenya jaarraa hedduudhaaf ture. Bara 1949 yaadadha, barataa Hierofant Misterios Minore Aun Weor, fi ji'oota afur caalaa eega eeggate booda guyyaa 27ffaa ji'a July qormaata Tiksee irratti dhihaate. Namichi jalqabaa iddoo tokko kophaa isaa argama ture, keessa isaatti argama ture, garuu hin beekne, yeroo Tiksee balbalaa waammatutti, inni hin turre, namichi jalqabaa yeroo jalqabaaf qabbana du'aa akka itti dhagahame dubbata, yeroo ifaa ture gara dukkanatti deema, qabbana kana guddisuun, urgaa badaa soda guddaadhaan akka gurgura'u taasisa, bakka sanaa baqachuuf fedhii qaba ture, garuu humna Kiristoos kan qaama isaa keessatti kuufate jijjiirama alkimiyaatiin, Arkan A.Z.F., bakka hin barbaachifne sana keessatti akka argamu jajjabeessa. Battalumatti bineensa fakkeenya qamaleen qabu gara isaatti dhufaa akka jiru arge, guutummaan guutuutti rifeensa qabu, gaanfa fuula isaa irraa ifa kennu kan yeroo socho'u sagalee kennu, funyaan fi afaan akka gaangaan qaba, innis akkana jedheen: Akka anaan dhiistee deemtu barbaaddaa? Akkas na galateeffattaa? Nama hin beekneef na jijjiirtaa? Innis soda guddaadhaan akka isa dhiisu deebiseef, bineensichi gara isaatti fiigaa kan weeraru fakkaata ture, namichi jalqabaa isa hammeesse, garuu hammeessuun dadhabaa dhiira dadhabaa fi soda qabuuf hin fayyadamne, garuu Chela Kiristoos akka ta'e yaadate, maqaa Kiristoosiin isa hammeesse, innis xiqqoo gara duubaatti deebi'e, yeroo hundaa gara isaatti fiigutti, eeggumsa Kiristoos fi barsiisota isaa jaallataman irraa barbaada ture, innis akkana jedheen: amma na irratti hin danda'u, si nan mo'adha bineensichis soda guddaadhaan deebi'e. Raafama barataa sanaa guddaa ture, sababiin isaas Hierofant gara magaalaa biraatti deeme ture, kunis guyyoota sadii ykn isaa ol isa barbaachisa, garuu yeroo inni deebi'utti isa gaafate innis akkana jedheen: Sin galateeffadha, qormaata jalqabaa tiksee balbalaa waliin sirriitti darbite, uumamni kun seexana keessan, isaaf tajaajilteetta fi isa qubsiisteetta jaarraa hedduudhaaf, akkamittan isa qubsiisaa? Barataan soda qabu gaafate, Barsiisaan akkana jedheen, inni fedhii keessan gadhee irraa quubsa, hawwii keenya gadhee, fedhii hamaa, halalummaa, gaalawwanii fi jireenya xuraa'aa irraa quubsa, kun hundi daldaltoota mana qulqullummaa Kiristoos nuuf dubbate dha, mana qulqullummaa keenya waliin daldalu, amma sibixila fedhii gara keessaa keessaa daldaltoota si gabroomsan keessaa baasuuf ati si barbaachisa, amma isaan keessaa tokko tokko mo'uu si barbaachisa yoo dhugumaan hammeenya irraa bilisa bahuu fi karaa adii irratti deemuu barbaaddan.

Hojii kanaan nutii ilmaan addeessaa daaw'achuu ni dandeenya, gara mana qulqullummaa addeessotaatti argamuun, kadhannaa fi ergamtoota addeessaa waliin hojjechuun, akka nuyi tapha haalota hin taane, garuu jalqaba daldaltoota mana qulqullummaa keenya keessaa baasuu qabna, alaqaa keenya keessatti haala itti fayyadamnu barachuu qabna. Kanaafis jaalalli gara aadaa Lumaatti dhaqa, achitti Waaqa caalaa waan hundumaa jaallachuu fi tajaajiluu fi ollaa isaa, aadaa isaa ni barata, yeroo booda beeksisa aadaa guutuun alaqaa jireenya waliin walitti dhufeenya cimaa qaba, dinqii kan hin dubbatamne argata. JACHIN fi BOAZ alaqaa isaanii fayyadamuuf barbaachisaa, hanga yeroo tarkaanfii torba alaqaa Waaqa jireenya keessatti hubannoodhaan hojjechuuf barbaachisutti argamuutti, fuuldura waaqayyoo eebbifamaatti.

Barumsa du'aa ka'uutiin hoolota keessa keenyaatti ajjeesuu ni baranna, yeroo booda bushaayyee Hoolaa Irra-Butuu ni uumna. Akkasumas Gooftaa Yeroo irraa bilisa baanee jireenya dhuma hin qabne gammachuutiin guutame jiraanna.
