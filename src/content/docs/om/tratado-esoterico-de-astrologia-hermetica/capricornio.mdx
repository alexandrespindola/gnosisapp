Killeen SAAYINSII JINAS keessatti hedduu fayyadama kanaaf TRATADO ESOTÉRICO kan TEÚRGIA, kan LAMAFFAA EDICIÓN keessatti dubbanneerra.

HOJII KAN CAPRICORNIO. Yeroo mallattoo CAPRICORNIO keessatti sanduuqa ykn saanduqa du’aa lafa irraa yaadaa. Sanduuqa yaadaa kana irra deema, garuu gidduu miillaa akka yaadamuu taasisi; yeroo deemtutti jilbi ni dabala, akka waan gufuu sana dandamattuutti, akka miilla sanduuqa sana irra darbituutti, garuu jilba mirgaa gara bitaatti sochoosuun, yaada keessaatti jilbatti xiyyeeffachuun, kaayyoo jabaa kan PLOMO kan SATURNOtiin akka guuttaman qabduu.

MAASTAR MASONES hojii SATURNO kana sirriitti hubachuu danda’u, SABABIIN ISAA tarkaanfiin MAASTAR MASÓN gara LOGIA seenuu kanuma.

Namoonni CAPRICORNIO keessatti dhalatan qophaa’ina barnootaaf qabu, hedduu ni dhiphatu, HUBANNOO GUDDAA KAN DIRQAMAA qabu, uumamaan hojiitti ni seenu jireenya isaanii keessatti dhiphina guddaa keessa darbu, namni tokko isaan ganu.

Dubartoonni CAPRICORNIO haadha manaa ajaa’ibsiisaa, hanga du’aatti amanamtoota, hojii jaalatoota, hojii hojjatoota, hamma hin ibsamnetti dhiphattoota, garuu dandeettiwwan kana hundaaf malee, abbaan manaa isaan ni gana, ni dhiisa, yeroo baay’ee jaalala isaaniitiinis yoo ta’e, KARMA isaanii ti kan nama gaddisiisu.

Dubartoonni tokko tokko CAPRICORNIO dhiirota biraa wajjin walitti seenu, kunis erga abbaan manaa isaan dhiisee fi erga sodaachisaatti dhiphatanii booda qofa.

Dhiirootaafi dubartoonni CAPRICORNIO ofittummaadha, kan hundi yoo hin ta’inillee; gara gosa isa gadi aanaa kan CAPRICORNIO refer ni goona. Kanaaf, ofittummaaf, waadaa hedduu galu akkasumas diina hedduutiin ni guutamu.

Namoonni CAPRICORNIO waanitti, qarshiitti hedduu maxxanu tokko tokko garmalee itti ta’u.

CAPRICORNIO mallattoo lafaa ti, dhaabbataa, kan hin jijjiiramne. Haa ta’u malee, namoonni CAPRICORNIO keessatti dhalatan imala hedduu ni godhu yoo gaggabaaboo ta’anillee.

Dhukkubbiin amalaa kan CAPRICORNIANOS sodaachisaa dha, garmalee ni dhiphatu, carraan isaanii hubannoo jireenya isaanii isaan baraara yeroo gabaabaatti gara hadhaa jireenya guddichaatti ni deebi’u.
