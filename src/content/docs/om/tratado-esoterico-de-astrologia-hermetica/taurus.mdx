## Ebla 21 Hanga Caamsaa 20

Mallattoo horoskooppii TAURUS UUMAA qoqaa (LARINGE) yoo ta'u, kunis gadameessa ajaa'ibsiisaa kan jecha, dubbiin keessatti uumamu waan ta'eef, gabaabumatti yeroo barnoota kana keessatti jechoota YOHAANNIS yeroo akkana jedhu hubachuun ni barbaachisa: "Jalqabatti dubbiin ture, dubbiinis Waaqayyoo wajjin ture, dubbiinis Waaqayyo ture, isaaniin waanti hundinuu ni ta'e, isa malee waanti ta'e tokko illee hin taane."

Sadarkaa addunyaa torbatu jira, cosmos torbaan humna dubbiitiin, muuziqaan, sagaleedhaan uumaman.

COSMOS inni jalqabaa ifaa hin uumamne ABSOLUTE keessa liqimfamee argama.

Sadarkaan addunyaa inni lammaffaan addunyaa bal'ina daangaa hin qabne keessa jiran hundaan ijaarameera.

Sadarkaan addunyaa inni sadaffaan ida'ama guutuu aduu bal'ina urjiilee qabdu hundaati.

Sadarkaan addunyaa inni afraffaan aduudha, seera ishee hundumaa fi DIMENSIONS ishee hundumaatiin nu ibsitu.

Sadarkaan addunyaa inni shanaffaan pilaanetoota sirna aduu hundaatiin kan ijaarame.

Sadarkaan addunyaa inni ja'affaan lafa mataa isheetii, dimiyeenshii torbaa fi naannoowwan jiraatoota daangaa hin qabneen jiraataman qabdi.

Sadarkaan addunyaa inni torbaffaan geengoo torba walgidduutti argaman yookaan ADDUNYAA JAHANNAM kan MOTUMMAA MINEERAA kan gogaa lafaa jala liqimfamaniin kan ijaarame.

Muuziqaan, DUBBIIN, kan LOGOS tiin oktobaawwan muuziqaa torba keessa kaa'ame, addunyaa kana ejjennoo isheetiin jabeessee qaba.

Sadarkaa jalqabaa addunyaa, mallattoo DO. Sadarkaa lammaffaa addunyaa, mallattoo SI. Sadarkaa sadaffaa addunyaa, mallattoo LA. Sadarkaa afraffaa addunyaa, mallattoo SOL. Sadarkaa shanaffaa addunyaa, mallattoo FA. Sadarkaa ja'affaa addunyaa, mallattoo MI. Sadarkaa torbaffaa addunyaa, mallattoo RE. Ergasii hundinuu mallattoo DO tiin ABSOLUTE tti deebi'a.

Muuziqaa malee, DUBBII malee, JECHA GUDDAA malee, jiraachuun ajaa'ibsiisaa COSMOS TORBAAN hin danda'amu ture.

DO-RE-MI-FA-SOL-LA-SI. SI-LA-SOL-FA-MI-RE-DO. Mallattoolee torban ISKEELII DUBBII UUMAA GUDDAA, waanta uumame hundumaa keessatti ni dhageettii kennu, sababiin isaas jalqabatti dubbiin ture.

Sadarkaan jalqabaa addunyaa seera tokkicha, SEERA GUDDAA tiin ogeessaan bulfama. Sadarkaan lammaffaa addunyaa SEEROTA SADAN bulfama. Sadarkaan sadaffaa addunyaa seerota ja'aan bulfama. Sadarkaan afraffaa addunyaa seerota kudha lamaan bulfama. Sadarkaan shanaffaa addunyaa seerota digdamii afuriin bulfama. Sadarkaan ja'affaa addunyaa seerota afurtamii saddeetiin bulfama. Sadarkaan torbaffaa addunyaa seerota sagaltamii ja'aan bulfama.

Yeroo DUBBIIN haasa'amu, sagalee muuziqaa, RITMOOTA, IBIDDA kompaasii SADIITIIN kan MAHAVAN fi CHOTAVAN tiin addunyaa kana ejjennoo isheetiin jabeessee qabuunis ni haasa'ama.

SOBA-DHOKSAATOONNI fi SOBA-SAAKRATTOONNI, MICROCOSMOS qofa, MACROCOSMOS qofa ni dubbatu, sadarkaa addunyaa lama qofa ni caqasu, yeroo dhugaan COSMOS TORBA, sadarkaa addunyaa torba DUBBIIN, muuziqaan, FIAT IBSAAN fi spermata jalqabaatiin kan jabeessee qabu.

COSMOS TORBAAN hundi isaanii lubbuu qabeessa hafuura baafatu, miira qabu fi jiraatu ta'uun isaanii shakkii hin qabu.

Ilaalcha DHOKSAATIIN, guddina olitti argamu hundinuu guddina gadiitiin argamuu mirkaneessuu dandeenya. Gadi hin bu'iin ol ba'uun hin danda'amu. Jalqaba gadi bu'uun booda ol ba'uutu barbaachisa.

COSMOS beekuu yoo barbaanne, dura lamaan isa waliin jiran, kan olitti fi kan gaditti jiru beekuu qabna, sababiin isaas isaan lachuu haalota fi raawwii lubbuun qaban COSMOS barachuu, beekuu barbaannu murteessu.

Fakkeenyaaf: Yeroo saayintistoonni injifannoo bal'inaaf qabsaa'an kana keessa, guddina cimaa, garuu xiqqaa, addunyaa atomikii keessatti ni argama.
