毫无疑问，每个人都有其独特的心理，这是无可辩驳的，无可争辩的，无可反驳的。

不幸的是，人们从未思考过这一点，许多人甚至不接受它，因为他们被困在感官的头脑中。

任何人都承认肉体的存在，因为他们可以看见和触摸它，然而心理学是不同的，它对五种感官来说是不可感知的，因此普遍的趋势是拒绝它，或者只是低估和轻视它，认为它无关紧要。

毫无疑问，当一个人开始自我观察时，这明确表明他已经接受了自己心理的巨大现实。

很明显，如果一个人没有找到一个根本的原因，他就不会试图自我观察。

显然，开始自我观察的人会变得与其他人非常不同，事实上，这表明了改变的可能性。

不幸的是，人们不想改变，他们对自己的生活状态感到满意。

看到人们像野兽一样出生、成长、繁殖，遭受难以言喻的痛苦却不知道为什么而死，这令人痛苦。

改变是根本的，但如果不开始心理上的自我观察，这是不可能的。

有必要开始审视自己，以便自我了解，因为事实上，理性的人形生物并不了解自己。

当一个人发现一个心理缺陷时，实际上已经迈出了一大步，因为这会让他能够研究它，甚至彻底消除它。

事实上，我们的心理缺陷是无数的，即使我们有千言万语和钢铁般的味觉，也无法完全列举它们。

这一切的严重之处在于，我们不知道如何衡量任何缺陷的可怕现实；我们总是以虚荣的方式看待它，而没有给予应有的关注；我们认为它无关紧要。

当我们接受“众多”的教义，并理解耶稣基督从抹大拉的玛丽亚的身体里驱逐的七个恶魔的残酷现实时，我们对心理缺陷的思考方式显然会发生根本性的变化。

以强调的方式肯定“众多”的教义百分之百起源于西藏和诺斯替教义，这一点并不过分。

事实上，知道在我们的内心深处生活着成百上千个心理人物，这并不是一件令人愉快的事情。

每一个心理缺陷都是一个不同的人，此时此刻存在于我们自身之中。

伟大的导师耶稣基督从抹大拉的玛丽亚的身体里驱逐的七个恶魔是七宗罪：愤怒、贪婪、淫欲、嫉妒、傲慢、懒惰、暴食。

当然，这些恶魔中的每一个本身都是军团的首领。

在古埃及法老时代，如果想实现意识的觉醒，发起者必须从他的内在本质中消除赛特的红色恶魔。

鉴于心理缺陷的现实，有抱负的人渴望改变，不想继续生活在如此多的人潜伏在他的精神之中的状态，于是开始了自我观察。

随着我们在内在工作中的进步，我们可以亲自验证消除系统中的一个非常有趣的秩序。

当一个人发现与消除代表我们错误的众多心理聚集体相关的工作中的秩序时，他会感到惊讶。

这一切有趣之处在于，缺陷的消除中的这种秩序是以渐进的方式进行的，并且是根据意识的辩证法进行的。

理性辩证法永远无法超越意识辩证法的强大工作。

事实向我们证明，缺陷消除工作中心理的秩序是由我们自己内在深处的存在建立的。

我们必须澄清的是，自我和存在之间存在着根本的区别。自我永远无法在心理问题上建立秩序，因为它本身就是混乱的结果。

只有存在才有能力在我们的精神中建立秩序。存在即存在。存在的存在理由就是存在本身。

通过心理自我观察的明智意义，可以证明自我观察、评判和消除我们的心理聚集体的工作中的秩序。

在所有人类中，都存在着处于潜伏状态的心理自我观察意识，但随着我们使用它，它会逐渐发展。

这种意识使我们能够直接地感知，而不是通过简单的智力联想来感知生活在我们精神中的各种“我”。

这种超感官知觉的问题开始在超心理学领域进行研究，事实上，已经在随着时间推移进行的多次实验中得到了证明，并且存在大量相关文档。

那些否认超感官知觉现实的人是百分之百的无知者，是被困在感官头脑中的智力恶棍。

然而，心理自我观察意识是一种更深层次的东西，它远远超出了简单的超心理学陈述，它允许我们进行亲密的自我观察，并充分验证我们各种聚集体的巨大主观现实。

与消除心理聚集体这个非常严重的主题相关的各项工作的连续顺序，使我们能够推断出一个非常有趣的“工作记忆”，甚至在内在发展问题上非常有用。

虽然这种工作记忆确实可以给我们提供过去生活中各个阶段的不同心理照片，但将它们全部组合在一起会给我们的想象力带来一幅生动甚至令人厌恶的画面，描绘出我们在开始激进的心理转变工作之前的样子。

毫无疑问，我们永远不会想回到那个可怕的形象，那是我们过去样子的生动写照。

从这个角度来看，这张心理照片将作为在转变的现在和倒退的、陈腐的、笨拙的和不幸的过去之间进行对比的一种有用手段。

工作记忆总是基于心理自我观察中心记录的连续心理事件来书写的。

在我们的精神中存在着我们甚至无法想象的有害元素。

一个诚实的人，永远不会拿走任何不属于自己的东西，值得尊重和荣誉，以一种不寻常的方式发现一系列小偷“我”居住在他自己精神的最深处，这是一件可怕的事情，但并非不可能。

一个充满伟大美德的伟大的妻子，或一个具有卓越灵性和出色教育的年轻女子，通过心理自我观察意识以一种不寻常的方式发现，在她亲密的精神中居住着一群妓女“我”，这对于任何明智的公民的智力中心或道德意识来说都是令人作呕甚至无法接受的，但所有这些在心理自我观察的精确领域内都是可能的。
