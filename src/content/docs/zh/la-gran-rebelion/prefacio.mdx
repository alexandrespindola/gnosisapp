序言

作者：V.M. GARGHA KUICHINES

尊敬的萨麦尔·奥恩·维奥尔大师的“大叛乱”切实地向我们展示了我们在生活中的位置。

我们必须打破所有束缚我们于这个生命中虚幻事物的东西。

这里我们汇集了每一章的教义，以指导勇敢者投入到与自己的战斗中。

这部作品的所有关键都指向我们小我的毁灭，以释放本质，这才是我们身上有价值的东西。

小我不想死亡，而主人觉得自己不如缺点。

世界上充斥着无能的人，恐惧随处肆虐。

“没有不可能的事，只有无能的人。”

# 第1章

人类失去了内在的美丽；肤浅抹杀了一切。虔诚不为人知。残酷有追随者。平静不存在，因为人们生活在担忧和绝望中。

受苦受难者的命运掌握在各色罪人的手中。

# 第2章

饥饿和绝望每时每刻都在增加，化学产品正在破坏地球大气层，但存在一种对抗我们周围邪恶的解药：“科学的贞洁”或利用人类的种子，在我们的实验室中将其转化为能量，然后当我们学会掌握意识觉醒的3个因素时，将其转化为光和火焰：1. 死亡我们的缺点。 2. 在我们体内形成太阳体。 3. 服务于可怜的孤儿（人类）。

土地、水和空气，都被当前的文明污染了；世界的黄金不足以弥补邪恶；只有我们每个人产生的液体黄金，我们自己的种子，才能明智地、有意识地使用，这样我们才能有能力改善世界并以清醒的意识服务。

我们正在组建世界救世军，所有勇敢的人都通过基督化的教义与水瓶座的阿凡达并肩作战，这将使我们摆脱一切邪恶。

如果你变得更好，世界就会变得更好。

# 第3章

对于许多人来说，幸福并不存在，他们不知道这是我们自己的工作，我们是它的创造者，是建设者；我们用我们的液体黄金，我们的种子来建造它。

当我们快乐时，我们会感到幸福，但这些时刻是短暂的；如果你无法掌控你的尘世的心智，你就会成为它的奴隶，因为它永远不会满足。我们必须生活在世界上，而不是成为它的奴隶。

# 第4章 关于自由的讨论

自由让我们着迷，我们渴望自由，但当我们听到别人说某人的坏话时，我们会被迷住，从而变成放荡者，并变成恶人。

重复恶意谣言的人，比发明谣言的人更邪恶，因为后者可能是出于嫉妒、羡慕或真诚的错误；而重复者则是作为邪恶的忠实门徒这样做，他是一个潜在的恶人。“寻找真理，真理必叫你们得以自由。”但是，说谎者如何才能找到真理？在这种情况下，他每时每刻都在远离对立的一极，即真理。

真理是慈爱天父的属性，信仰也是如此。说谎者如何才能拥有信仰，如果信仰是天父的恩赐？充满缺点、恶习、权力欲望和专横跋扈的人无法获得天父的恩赐。我们是我们自己信念的奴隶；逃离那些谈论他们内心所见之事的先知；这样的人出卖天堂，一切都将被剥夺。

“谁是自由的？谁获得了著名的自由？有多少人获得了解放？唉！唉！唉！”（萨麦尔）。说谎的人永远不会自由，因为他反对纯真真理的慈爱者。

# 第5章 关于钟摆定律的讨论

万物流动和回流，上升和下降，来来回回；但是人们更关心邻居的波动，而不是他们自己的波动，因此他们在他们存在的狂风暴雨中行走，利用他们有缺陷的感官来评价邻居的波动；而他自己呢？当人杀死他的小我或缺点时，他就解放了，摆脱了许多机械定律的束缚，打破了我们形成的众多外壳之一，并感到渴望自由。

极端总是会有害的，我们必须寻求公正的中间，天平的平衡点。

理性在既定事实面前恭敬地倾斜，概念在晶莹剔透的真理面前消失。“只有消除错误，真理才会出现”（萨麦尔）。

# 第6章 概念与现实

读者最好仔细研究本章，以避免被错误的评价所引导；只要我们有心理缺陷、恶习、怪癖，我们的概念也会是错误的；“事情就是这样，因为我已经证实了”，这是愚蠢的，一切都有方面、棱角、起伏、高低、距离、时间，在那里，片面的傻瓜以自己的方式看待事物，以暴力强加于人，吓唬他的听众。

# 第7章 意识的辩证法

我们知道，并且这教导我们，我们只能通过有意识的工作和自愿的痛苦来唤醒意识。

当虔诚的追随者认同他存在的事件时，他会浪费一小部分意识的能量。

一位有能力的大师，参与生活剧，不会认同该剧，他感觉自己像是生活马戏团里的观众；在那里，就像在电影院里一样，观众偏袒冒犯者或受害者。生活的大师是那些教导追随者善良和有用的事物的人，使他们比他们原本更好，大自然母亲服从他，人们带着爱追随他。

“意识是无意识无法感知的光”（萨麦尔·奥恩·维奥尔），睡眠者对意识之光的感觉，就像盲人对太阳之光的感觉一样。

当我们意识的半径增加时，一个人会在内心体验到真实，什么是真实。

# 第8章 科学术语

人们面对自然现象感到害怕，并希望它们过去；科学对它们进行标记并赋予它们难懂的名字，以使无知者不再打扰他们。

有数百万人知道他们疾病的名字，但不知道如何摧毁它们。

人熟练地操纵他所创造的复杂车辆，但他不知道如何操纵自己的车辆：他每时每刻都在其中移动的身体；人为了了解它，会发生这样的情况，就像实验室里有污垢或杂质一样；但是人们被告知要通过杀死他们的缺点、习惯、恶习等来清洁它，但他们做不到，他们认为每天洗澡就足够了。

# 第9章 反基督

我们把它带在里面。它不允许我们到达慈爱的天父。但是，当我们完全控制它时，它在表达上是多重的。

反基督仇恨基督教的美德，如信仰、耐心、谦卑等。“人”崇拜他的科学并服从它。

# 第10章 心理小我

我们必须每时每刻观察我们的行动，了解我们所做的事情是否让我们变得更好，因为破坏他人对我们没有任何用处。这只会让我们相信我们是好的破坏者，但当我们摧毁我们内在的邪恶，从而使我们按照我们潜在的活着的基督来改善自己时，这是好的，目的是为了照亮和改善人类。

教导仇恨，这是每个人都知道的，但教导爱，这才是困难的。

亲爱的读者，如果你想彻底摧毁你自己的邪恶，请仔细阅读本章。

# 第11至20章

人们喜欢发表意见，将他人呈现为他们所看到的样子，但没有人想了解自己，这才是基督化之路上的重点。

说谎最多的人是最流行的；光明是意识，当它在我们身上显现时，是为了执行更高的工作。“凭着他们的果子，你们就可以认出他们来”，耶稣基督说。

他没有说凭着他们所做的攻击来认出他们。诺斯替教徒...醒来！！！

有智力或情感的人根据他们的智力或情感行事。这些人作为法官是可怕的，他们只听他们想听的，并把一个比他们更大的说谎者所断言的东西当作上帝的真理来评判或给出。

哪里有光，哪里就有意识。诽谤是黑暗的工作，它不是来自光明。

在第12章中，讨论了我们拥有的3种心智：感官心智或感官心智，中间心智；这种心智相信它所听到的一切，并根据冒犯者或辩护者进行判断；当它受到意识的指导时，它是一个强大的调解者，它变成了一个行动的工具；储存在中间心智中的事物形成了我们的信念。

拥有真正信仰的人不需要相信；说谎者将无法拥有信仰，这是上帝的属性和直接的经验，也无法拥有内在的心智，当我们杀死我们精神中携带的不良事物时，我们会发现它。

了解我们的缺点，然后分析它们，然后借助我们的母亲 RAM-IO 摧毁它们的这种美德，使我们能够改变，而不是成为在所有信念中出现的暴君的奴隶。

小我，自负，是我们内在的混乱；只有存在有能力在我们内在，在我们的精神中建立秩序。

通过对第13章的仔细研究，我们意识到了当有缺陷的先知遇到道路上任何兄弟的不良小我时会发生什么。当我们自我观察时，我们就不再说别人的坏话。

存在和知识，必须相互平衡；这样理解就诞生了。没有对存在的认识的知识，会带来各种各样的智力混乱；骗子诞生了。

如果存在大于知识，那么愚蠢的圣徒就会诞生。第14章为我们提供了了解自己的重要线索；我们是神圣的上帝，周围有一个不属于他的随从；放弃这一切就是解放，让他们说...

“罪行披着法官的托加，大师的长袍，乞丐的衣衫，老爷的西装，甚至基督的长袍”（萨麦尔）。

我们的神圣母亲马拉赫、玛丽亚或 RAM-IO，就像我们诺斯替教徒所称呼的那样，是慈爱天父和我们之间的调解人，是大自然基本神灵和魔法师之间的调解人；通过她和通过她，大自然的元素服从我们。她是我们的神圣德瓦，是世界神圣母亲和我们的物理载体之间的调解人，以实现惊人的奇迹并为我们的同伴服务。

通过与女祭司妻子的性结合，男人变得女性化，妻子变得男性化；我们的母亲 RAM-IO 是唯一可以让我们的小我和他们的军团变成宇宙尘埃的人。有了敏感的规范，我们就无法了解存在的事物，因为感官是密集的工具，充满了缺陷，就像它们的主人一样；有必要通过消除我们的缺点、恶习、怪癖、依恋、欲望以及所有令尘世心智愉悦的事物来疏通它们，这会给我们带来如此多的疑问。

在第18章中，我们根据二元性法则看到，正如我们生活在地球上的一个国家或地方一样，在我们的内心深处也存在着我们所处的位置的心理场所。亲爱的读者，阅读这有趣的章节，以便你知道你在内在的哪个社区、殖民地或地方。

当我们使用我们神圣的母亲 RAM-IO 时，我们就会摧毁我们的撒旦小我，并在意识的96条法则中从如此多的腐烂中解放自己。仇恨阻止我们在内心取得进步。

说谎者违背了自己的父亲，而淫乱者违背了圣灵；他们在思想、言语和行为上犯淫乱。

存在着一些暴君，他们赞美自己，引诱了许多无知者，但是，如果分析他们的工作，我们会发现破坏和无政府状态；生命本身会负责隔离和遗忘他们。

在第19章中，它给了我们光明，使我们不会陷入觉得自己优越的错觉。我们都是为阿凡达服务的学生；专制者会对伤害他感到痛苦，而愚蠢的人则会对不赞扬他感到痛苦。当我们理解我们必须摧毁人格时，如果有人在这项艰巨的任务中帮助我们，那就值得感谢。

信仰是纯粹的知识，存在的直接实验智慧，“自我意识的幻觉等同于毒品引起的幻觉”（萨麦尔）。

在第20章中，它给了我们线索，以消除我们身处其中的月球寒冷，并发展自己。

# 第21至29章

在第21章中，它告诉我们，教导我们冥想和反思，知道如何改变。不会冥想的人将永远无法消除自我。

在第22章中，它谈到了“回归与重现”。它以简单的方式告诉我们回归；如果我们不想重复痛苦的场景，我们必须瓦解那些向我们呈现这些场景的小我；它教导我们如何提高我们孩子的质量。重现对应于我们存在中的事件，当我们拥有肉体时。

内在的基督是火中之火；我们所见所感的是基督之火的物理部分。基督之火的降临是我们生活中最重要的事件，这种火负责我们所有气缸或大脑的运作过程，我们首先必须借助我们仁慈的母亲 RAMIO 的服务，使用大自然的5种元素来清洁这些气缸或大脑。

“启蒙者必须学会危险地生活；就这样写着。”

在第25章中，大师告诉我们关于我们自己不为人知的一面，我们将其投射出来，就像我们是一台电影放映机一样，然后，我们在别人的屏幕上看到我们的缺点。

所有这些都向我们展示了真诚的错误；正如我们的感官欺骗我们一样，我们也是说谎者；当我们不杀死我们的缺点就唤醒隐藏的感官时，隐藏的感官会导致灾难。

在第26章中，它告诉我们关于三个叛徒，希拉姆·阿比夫的敌人，内在的基督，恶魔：1.- 思想 2.- 恶意 3.- 欲望

我们每个人都在我们的精神中携带三个叛徒。

它教导我们，内在的基督是纯洁和完美的，它帮助我们根除我们内在携带的成千上万的不良事物。在该章中，它教导我们，秘密的基督是伟大的叛乱之主，被祭司、长老和圣殿的文士拒绝。

在第28章中，它告诉我们关于超人以及大众对他的一无所知。

类人为了变成超人所做的努力是与自己、与世界以及与一切将这个世界视为苦难的东西作斗争和斗争。

在第29章，最后一章中，它告诉我们关于圣杯，赫尔墨斯的容器，所罗门的杯子；圣杯以独特的方式寓意着女性的尤尼，性，神秘主义者的 soma，圣神在其中饮用。

这种喜乐之杯在任何神秘圣殿中以及在诺斯替教祭司的生命中都不可或缺。

当诺斯替教徒理解了这个奥秘时，他们的婚姻生活将发生改变，活生生的祭坛将为他们服务，像祭司一样在神圣的爱之圣殿中举行仪式。

愿最深的平安在你的心中统治。

GARGHA KUICHINES
