བོད་ཡིག དཔེ་དེབ་ཀྱི་མདུན་ཤོག

རྩོམ་པ་པོ། གདོང་དྲུག་ཀུའི་ཆིས་ནས།

བླ་མ་སམ་ཨེལ་ཨོན་ཝོར་མཆོག་གི་"གྱེན་ལོག་ཆེན་པོ་"ཞེས་པའི་བརྩམས་ཆོས་འདིས་ང་ཚོའི་འཚོ་བའི་གནས་བབ་ཇི་འདྲ་ཡིན་པ་མངོན་སུམ་མཚོན་ཡོད།

འཇིག་རྟེན་འདིའི་སྣང་བ་འཁྲུལ་སྣང་གི་ཆོས་སུ་འཆིང་བའི་དངོས་པོ་ཐམས་ཅད་བཅོམ་དགོས།

འདིར་ལེའུ་རེ་རེའི་ནང་གི་བསླབ་བྱ་བསྡུས་ཏེ་རང་ཉིད་ལ་གཡུལ་སྤྲོད་པའི་སྙིང་སྟོབས་ཅན་དག་ལ་ལམ་སྟོན་བྱེད།

བརྩམས་ཆོས་འདིའི་གནད་འགག་ཐམས་ཅད་ང་ཚོའི་ངོ་བོ་འཇོམས་པར་འཁྲིད་དེ་སྙིང་བཅུད་བཅིངས་གྲོལ་ཐོབ་པ་ནི་ང་ཚོའི་ནང་གི་རིན་ཐང་ཅན་དེ་ཡིན།

ངོ་བོས་འཆི་འདོད་མེད་ལ་བདག་པོ་ནི་སྐྱོན་ལས་དམན་པར་སྣང་།

འཇིག་རྟེན་ན་ནུས་མེད་མང་ཞིང་འཇིགས་སྐྲག་གིས་གང་སར་གནོད་འཚེ་བཟོ།

"མི་སྲིད་པ་མེད་ལ་མི་ནུས་པ་ཅན་ཡོད།"

# ལེའུ་དང་པོ།

མིའི་རིགས་ལ་ནང་སེམས་ཀྱི་མཛེས་སྡུག་མེད་ཅིང་། སྟེང་ཤུན་གྱིས་ཐམས་ཅད་ཟིལ་གྱིས་གནོན་པས། སྙིང་རྗེ་མི་ཤེས། གདུག་རྩུབ་ལ་རྗེས་འཇུག་པ་ཡོད། ཞི་བདེ་མེད་དེ་མི་རྣམས་སེམས་ཁྲལ་དང་ཡིད་ཆད་ཀྱིས་འཚོ་བའི་ཕྱིར།

སྡུག་བསྔལ་ཅན་གྱི་བསོད་ནམས་ནི་ངན་སེམས་ཅན་གྱི་ལག་ཏུ་ཡོད།

# ལེའུ་གཉིས་པ།

བཀྲེས་སྐོམ་དང་སྡུག་བསྔལ་སྐད་ཅིག་སྐད་ཅིག་ལ་འཕེལ་ཞིང་རྫས་འགྱུར་གྱིས་སའི་གོ་ལ་བརླག་ཀྱང་ང་ཚོར་གནོད་པའི་གདུག་པ་ལ་སྨན་ཡོད་དེ། "ཚན་རིག་གི་བཙུན་སྤྱོད་" ཅེས་པའམ་མིའི་ས་བོན་ནུས་པར་བསྒྱུར་ནས་མིའི་བཟོ་གྲྭའི་ནང་དུ་དང་དེ་ནས་འོད་དང་མེར་བསྒྱུར་དུས་ང་ཚོས་རིག་པ་སད་པའི་རྒྱུ་རྐྱེན་གསུམ་ལ་དབང་འབྱོར་དགོས། ༡ ང་ཚོའི་སྐྱོན་ཆ་འཆི་བ། ༢ ང་ཚོའི་ནང་དུ་ཉི་མའི་ལུས་གྲུབ་པ། ༣ དབུལ་ཕོངས་སྡུག་པ་ཅན་ལ་ཞབས་ཞུ་སྒྲུབ་པ།

ས་དང་ཆུ་དང་རླུང་བཅས་དེང་སང་གི་ཤེས་རིག་གིས་བསླད་པས་འཇིག་རྟེན་གྱི་གསེར་གྱིས་གནོད་པ་བཅོས་མི་ཐུབ་པས་ང་ཚོའི་གསེར་གཤེར་དེ་ང་ཚོ་རང་ངོས་ནས་ཐོན་སྐྱེད་བྱེད་ཅིང་རང་ཉིད་ཀྱི་ས་བོན་བཀོལ་ནས་རྒྱུ་མཚན་ཤེས་ཏེ་བེད་སྤྱོད་གཏོང་ན་ང་ཚོས་འཇིག་རྟེན་ཇེ་ལེགས་སུ་གཏོང་བ་དང་རིག་པ་སད་པའི་སྒོ་ནས་ཞབས་ཞུ་སྒྲུབ་པར་ནུས་པ་ཐོན་ཡོད།

ང་ཚོས་འཛམ་གླིང་སྐྱོབ་པའི་དམག་དཔུང་འཛུགས་བཞིན་ཡོད་པ་དེ་དག་ནི་སྙིང་སྟོབས་ཅན་གྱིས་ཁྱིམ་མཚེས་བླ་མ་ཨེ་ཀུ་ཨེ་རི་ཡུ་སི་དང་མཉམ་རུ་སྒྲིག་སྟེ་ང་ཚོ་གདུག་པ་ཐམས་ཅད་ལས་སྐྱོབ་པའི་ཁེ་རི་སི་ཏི་ཕི་ཀེ་ཤོན་གྱི་ཆོས་ལུགས་བརྒྱུད་ནས་གྲུབ་པ་ཡིན།

ཁྱེད་རང་ཇེ་ལེགས་སུ་སོང་ན་འཇིག་རྟེན་ཡང་ཇེ་ལེགས་སུ་འགྲོ།

# ལེའུ་གསུམ་པ།

མང་པོ་ཞིག་ལ་མཚོན་ན་སྐྱིད་པོ་མེད་དེ་ཁོ་ཚོས་ང་ཚོའི་ལས་ཡིན་པ་མ་ཤེས་ལ་ང་ཚོས་བཟོ་སྐྲུན་བྱེད་པ་ཡིན་པས་ང་ཚོས་གསེར་གཤེར་དང་ས་བོན་བཀོལ་ནས་བསྐྲུན་དགོས།

ང་ཚོ་སྤྲོ་སྣང་སྐྱེས་དུས་སྐྱིད་པོར་ཚོར་ཡང་སྐད་ཅིག་དེ་དག་མྱུར་དུ་ཡལ་འགྲོ་སྟེ་ཁྱེད་ཀྱི་སེམས་ལ་དབང་བསྒྱུར་མ་ཐུབ་ན་ཁྱེད་དེའི་བྲན་དུ་འགྱུར་ཏེ་དེས་ཅི་ཡང་ཚིམ་པ་མི་སྲིད། འཇིག་རྟེན་ན་བྲན་མེད་པར་འཚོ་དགོས།

# ལེའུ་བཞི་པ། རང་དབང་སྐོར་བཤད་པ།

རང་དབང་ལ་ང་ཚོ་ཧ་ཅང་དགའ་ཞིང་རང་དབང་ཐོབ་འདོད་ཡོད་ཀྱང་སུས་ཀྱང་ང་ཚོར་སྐྱོན་བརྗོད་ན་ང་ཚོ་བཟི་འགྲོ་སྟེ་རང་དབང་མེད་པར་གྱུར་ནས་ངན་པར་འགྱུར།

གཞན་གྱི་སྐྱོན་བརྗོད་པ་བསྐྱར་ཟློས་བྱེད་མཁན་དེ་སྐྱོན་བརྗོད་པ་གསར་རྩོམ་བྱེད་མཁན་ལས་ཀྱང་སྡུག་སྟེ་གསར་རྩོམ་བྱེད་མཁན་དེས་ཕྲག་དོག་དང་ཞེ་སྡང་ལ་བརྟེན་ནས་བསམ་བློ་ནོར་བ་གཞན་ལ་འཁྲི་བའི་གནས་སྟངས་འོག་ཏུ་བརྗོད་སྲིད་ཀྱང་སྐྱོན་བརྗོད་པ་བསྐྱར་ཟློས་བྱེད་མཁན་དེས་གདུག་རྩུབ་ཀྱི་སློབ་མ་དམ་པ་ཞིག་ཡིན་པས་གདུག་རྩུབ་ཅན་ཞིག་ཏུ་གྱུར་སྲིད། "བདེན་པ་འཚོལ་དང་དེས་ཁྱེད་རང་རང་དབང་ཅན་དུ་བསྒྱུར།" འོན་ཀྱང་རྫུན་བཤད་མཁན་དེ་བདེན་པའི་གནས་སྟངས་སུ་ཇི་ལྟར་སླེབས་སྲིད། གནས་སྟངས་དེའི་འོག་ཏུ་ཁོ་རང་སྐད་ཅིག་སྐད་ཅིག་ལ་ལྡོག་ཕྱོགས་དང་བྲལ་བར་འགྱུར། བདེན་པ་ནི་བདག་པོ་བཟང་པོའི་ཁྱད་ཆོས་ཡིན་ལ་དད་པ་ཡང་དེ་བཞིན་ནོ། རྫུན་བཤད་མཁན་དེས་དད་པ་ཇི་ལྟར་བརྙེས། དེ་ནི་བདག་པོས་སྦྱིན་པ་ཡིན་ནོ། སྐྱོན་དང་དུག་དང་དབང་ཆ་སྒེར་སྤྱོད་དང་གཞན་འོག་ཏུ་མི་འདུག་པའི་ང་རྒྱལ་གྱིས་ཁེངས་པ་དེས་བདག་པོའི་སྦྱིན་པ་བླངས་མི་ཐུབ། ང་ཚོ་རང་ཉིད་ཀྱི་ཡིད་ཆེས་ཀྱི་བྲན་དུ་གྱུར་ཏེ་ནང་སེམས་ཀྱིས་མཐོང་བ་བརྗོད་པའི་ལུང་སྟོན་པ་ལས་ཐར་དགོས་ཏེ་དེས་ཞིང་ཁམས་བཙོང་བས་ཁོ་ལ་ཡོད་ཚད་འཕྲོག་ངེས།

"སུ་ཞིག་རང་དབང་ཅན་ཡིན། སུ་ཞིག་གིས་རང་དབང་ཐོབ་ཡོད། སུ་དག་བཅིངས་གྲོལ་ཐོབ་ཡོད། ཨ་ཙི་ཙི། ཨ་ཙི་ཙི། ཨ་ཙི་ཙི།" (སམ་ཨེལ།) རྫུན་བཤད་མཁན་ནམ་ཡང་རང་དབང་ཅན་དུ་མི་འགྱུར་ཏེ་ཁོ་རང་བདག་པོ་བཟང་པོ་དང་འགལ་ཟླར་གྱུར་པས་བདེན་པ་རྣམ་པར་དག་པ་ཞིག་མེད།

# ལེའུ་ཉེར་གཅིག་ནས་ཉེར་དགུའི་བར།

ཉེར་གཉིས་ནང་དུ་ང་ཚོར་"ཕྱིར་ལོག་དང་ཡང་བསྐྱར་འབྱུང་བའི་"སྐོར་བཤད་དེ། ཕྱིར་ལོག་གི་སྐོར་གླེང་བའི་ཐབས་ནི་སླ་མོ་ཞིག་ཡིན་ཏེ་ང་ཚོས་སྡུག་བསྔལ་གྱི་ཚོར་བ་བསྐྱར་ཟློས་བྱེད་འདོད་མེད་ན་ང་ཚོས་ངོ་བོ་སྟེ་ཚོར་བ་དེ་སྟེར་མཁན་དེ་དག་བཤིག་དགོས་ཏེ་ང་ཚོའི་བུ་ཕྲུག་གི་སྤུས་ཚད་ཇེ་ལེགས་སུ་གཏོང་བར་བསླབ
