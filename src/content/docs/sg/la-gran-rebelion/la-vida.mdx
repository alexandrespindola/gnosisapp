Dunia aga ngangu mingi, alege ti lakue lakue so nga na ada ti ngangu so ayeke na ndo kue.

A yeke même bongo ti lakue lakue: sioni, so asara zo ti bê, so ayeke na ngangu pëpe. “Alege ti fadeso!” azo mingi atene.

E yeke tongana a-paon so ayeke pika kate na bongo so e yü nga na poro so e zia na gere ti e, atâa so azo so nzara agi ala, so kobe ayeke nzoni pëpe, so ayeke na mawa mingi ayeke tambela na ndo so, na ndo so nga na ndo so.
