ຄົນ​ຜູ້​ໜຶ່ງ​ເກີດ​ມາ, ມີ​ຊີວິດ​ຢູ່​ໄດ້​ຫົກ​ສິບ​ຫ້າ​ປີ​ແລ້ວ​ກໍ​ຕາຍ​ໄປ. ແຕ່​ວ່າ​ລາວ​ຢູ່​ໃສ​ກ່ອນ​ປີ 1900 ແລະ​ລາວ​ຈະ​ຢູ່​ໃສ​ຫຼັງ​ປີ 1965? ວິທະຍາສາດ​ທາງ​ການ​ບໍ່​ຮູ້​ຫຍັງ​ກ່ຽວ​ກັບ​ສິ່ງ​ເຫຼົ່າ​ນີ້. ນີ້​ແມ່ນ​ການ​ສ້າງ​ເປັນ​ທົ່ວ​ໄປ​ຂອງ​ບັນ​ຫາ​ທັງ​ໝົດ​ກ່ຽວ​ກັບ​ຊີ​ວິດ​ແລະ​ຄວາມ​ຕາຍ.

ໂດຍ axiomatically ພວກເຮົາສາມາດຢືນຢັນວ່າ: "ມະນຸດຕາຍເພາະວ່າເວລາຂອງລາວສິ້ນສຸດລົງ, ບໍ່ມີມື້ອື່ນສໍາລັບບຸກຄະລິກກະພາບຂອງຄົນຕາຍ".

ທຸກໆມື້ແມ່ນຄື້ນເວລາ, ທຸກໆເດືອນແມ່ນຄື້ນເວລາອີກຄື້ນຫນຶ່ງ, ທຸກໆປີກໍ່ແມ່ນຄື້ນເວລາອີກຄື້ນຫນຶ່ງ ແລະຄື້ນຟອງທັງໝົດເຫຼົ່ານີ້ຖືກຕ່ອງໂສ້ເຂົ້າກັນກໍ່ປະກອບເປັນຄື້ນຟອງອັນຍິ່ງໃຫຍ່ແຫ່ງຊີວິດ.

ເວລາເປັນຮູບກົມ ແລະຊີວິດຂອງບຸກຄະລິກກະພາບຂອງມະນຸດເປັນເສັ້ນໂຄ້ງປິດ.

ຊີວິດຂອງບຸກຄະລິກກະພາບຂອງມະນຸດພັດທະນາໄປຕາມເວລາຂອງລາວ, ເກີດມາໃນເວລາຂອງລາວ ແລະຕາຍໃນເວລາຂອງລາວ, ລາວບໍ່ສາມາດມີຢູ່ເກີນເວລາຂອງລາວໄດ້.

ເລື່ອງເວລານີ້ເປັນບັນຫາທີ່ນັກປັນຍາຊົນຫຼາຍຄົນໄດ້ສຶກສາ. ໂດຍບໍ່ຕ້ອງສົງໃສເວລາແມ່ນມິຕິທີ່ສີ່.

ເລຂາຄະນິດຂອງ EUCLIDES ພຽງແຕ່ສາມາດນໍາໃຊ້ກັບໂລກສາມມິຕິ, ແຕ່ໂລກມີເຈັດມິຕິ ແລະອັນທີສີ່ແມ່ນເວລາ.

ຈິດໃຈຂອງມະນຸດຮັບຮູ້ຄວາມເປັນນິດນິລັນດອນເປັນການຕໍ່ເວລາໃນເສັ້ນຊື່, ບໍ່ມີຫຍັງຜິດພາດຫຼາຍກ່ວາແນວຄວາມຄິດນີ້ເພາະວ່າຄວາມເປັນນິດນິລັນດອນແມ່ນມິຕິທີຫ້າ.

ທຸກໆຊ່ວງເວລາຂອງການມີຢູ່ເກີດຂຶ້ນໃນເວລາ ແລະເກີດຂຶ້ນຊໍ້າຕະຫຼອດໄປ.

ຄວາມຕາຍ ແລະຊີວິດແມ່ນສອງຂົ້ວທີ່ສຳຜັດກັນ. ຊີວິດສິ້ນສຸດລົງສໍາລັບຜູ້ຊາຍທີ່ຕາຍແຕ່ອີກຊີວິດຫນຶ່ງເລີ່ມຕົ້ນ. ເວລາຫນຶ່ງສິ້ນສຸດລົງ ແລະອີກອັນຫນຶ່ງເລີ່ມຕົ້ນ, ຄວາມຕາຍແມ່ນກ່ຽວຂ້ອງຢ່າງໃກ້ຊິດກັບການກັບຄືນນິລັນດອນ.

ນີ້ຫມາຍຄວາມວ່າພວກເຮົາຕ້ອງກັບຄືນ, ກັບຄືນສູ່ໂລກນີ້ຫຼັງຈາກທີ່ຕາຍແລ້ວເພື່ອເຮັດຊ້ໍາການສະແດງອອກດຽວກັນຂອງການມີຢູ່, ຫຼາຍຖ້າ, ບຸກຄະລິກກະພາບຂອງມະນຸດຕາຍໄປກັບຄວາມຕາຍ, ຜູ້ໃດຫຼືສິ່ງທີ່ກັບຄືນມາ?

ມັນເປັນສິ່ງຈໍາເປັນທີ່ຈະຊີ້ແຈງເທື່ອຫນຶ່ງແລະສໍາລັບທຸກຄົນວ່າ SELF ແມ່ນຜູ້ທີ່ສືບຕໍ່ຫຼັງຈາກຄວາມຕາຍ, ວ່າ SELF ແມ່ນຜູ້ທີ່ກັບຄືນ, ວ່າ SELF ແມ່ນຜູ້ທີ່ກັບຄືນສູ່ຮ່ອມພູແຫ່ງນໍ້າຕານີ້.

ມັນເປັນສິ່ງຈໍາເປັນສໍາລັບຜູ້ອ່ານຂອງພວກເຮົາບໍ່ໃຫ້ confuse ກົດຫມາຍຂອງ RETURN ກັບທິດສະດີຂອງ REINCARNATION ສອນໂດຍ THEOSOPHY ທີ່ທັນສະໄຫມ.

ທິດສະດີທີ່ກ່າວມານັ້ນຂອງ REINCARNATION ມີຕົ້ນກໍາເນີດມາຈາກການນະມັດສະການຂອງ KRISHNA ເຊິ່ງເປັນ RELIGION HINDUSTAN ຂອງປະເພດ Vedic, ແຕ່ຫນ້າເສຍດາຍທີ່ຖືກແກ້ໄຂແລະປອມແປງໂດຍຜູ້ປະຕິຮູບ.

ໃນການນະມັດສະການທີ່ແທ້ຈິງຕົ້ນສະບັບຂອງ Krishna, ພຽງແຕ່ Heroes, Guides, ຜູ້ທີ່ມີ INDIVIDUALITY SACRED, ແມ່ນຜູ້ດຽວທີ່ເກີດໃຫມ່.

YO PLURALIZED RETURNS, ກັບຄືນມາແຕ່ບໍ່ແມ່ນ REINCARNATION. ມະຫາຊົນ, ຝູງຊົນກັບຄືນມາ, ແຕ່ນັ້ນບໍ່ແມ່ນ REINCARNATION.

ແນວຄວາມຄິດຂອງ RETURN ຂອງສິ່ງຕ່າງໆແລະປະກົດການ, ແນວຄວາມຄິດຂອງການເຮັດຊ້ໍານິລັນດອນແມ່ນບໍ່ເກົ່າແກ່ຫຼາຍແລະພວກເຮົາສາມາດຊອກຫາມັນຢູ່ໃນ WISDOM PYTHAGORIC ແລະໃນ cosmogony ວັດຖຸບູຮານຂອງ HINDUSTAN.

ການກັບຄືນນິລັນດອນຂອງມື້ແລະຄືນຂອງ BRAHAMA, ການເຮັດຊ້ໍາຢ່າງບໍ່ຢຸດຢັ້ງຂອງ KALPAS, ແລະອື່ນໆ, ແມ່ນກ່ຽວຂ້ອງກັນຢ່າງບໍ່ປ່ຽນແປງໃນຮູບແບບທີ່ໃກ້ຊິດກັບ Wisdom Pythagorean ແລະກົດຫມາຍວ່າດ້ວຍ RECURRENCE ນິລັນດອນຫຼື RETURN ນິລັນດອນ.

Gautama the BUDHA ໄດ້ສອນຢ່າງສະຫລາດກ່ຽວກັບ DOCTRINE ຂອງ ETERNAL RETURN ແລະວົງຈອນຂອງຊີວິດຕິດຕໍ່ກັນ, ແຕ່ DOCTRINE ຂອງລາວໄດ້ຖືກດັດແກ້ຢ່າງຫຼວງຫຼາຍໂດຍຜູ້ຕິດຕາມຂອງລາວ.

ທຸກໆ RETURN ຫມາຍເຖິງແນ່ນອນການຜະລິດບຸກຄະລິກກະພາບຂອງມະນຸດໃຫມ່, ນີ້ແມ່ນຮູບແບບໃນລະຫວ່າງເຈັດປີທໍາອິດຂອງໄວເດັກ.

ສະພາບແວດລ້ອມຂອງຄອບຄົວ, ຊີວິດຕາມຖະຫນົນແລະໂຮງຮຽນ, ເຮັດໃຫ້ບຸກຄະລິກກະພາບຂອງມະນຸດ, ສີລັກສະນະຕົ້ນສະບັບຂອງມັນ.

ຕົວຢ່າງຂອງຜູ້ເຖົ້າແມ່ນແນ່ນອນສໍາລັບບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍ.

ເດັກນ້ອຍຮຽນຮູ້ເພີ່ມເຕີມດ້ວຍຕົວຢ່າງກ່ວາດ້ວຍສິນລະລຶກ. ວິທີການດໍາລົງຊີວິດທີ່ຜິດພາດ, ຕົວຢ່າງທີ່ບໍ່ມີເຫດຜົນ, ຮີດຄອງປະເພນີທີ່ເສື່ອມໂຊມຂອງຜູ້ໃຫຍ່, ໃຫ້ບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍທີ່ມີສີລັກສະນະທີ່ແປກປະຫຼາດ, ບໍ່ເຊື່ອຟັງແລະຂີ້ຕົວະໃນຍຸກທີ່ພວກເຮົາອາໄສຢູ່.

ໃນຍຸກສະ ໄໝ ໃໝ່ ນີ້ການຫລິ້ນຊູ້ໄດ້ກາຍເປັນເລື່ອງ ທຳ ມະດາຫຼາຍກ່ວາມັນຕົ້ນແລະຜັກບົ່ວແລະດັ່ງທີ່ມັນເປັນພຽງແຕ່ເຫດຜົນນີ້ກໍ່ໃຫ້ເກີດສະຖານທີ່ Danteesque ຢູ່ໃນເຮືອນ.

ເດັກນ້ອຍຈໍານວນຫຼາຍໃນເວລານີ້ຕ້ອງທົນທຸກທໍລະມານຢ່າງເຈັບປວດແລະຄວາມຄຽດແຄ້ນ, ແສ້ແລະໄມ້ຕີຂອງພໍ່ລ້ຽງຫຼືແມ່ລ້ຽງ. ມັນເປັນທີ່ຈະແຈ້ງວ່າໃນທາງນັ້ນບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍພັດທະນາຢູ່ໃນຂອບເຂດຂອງຄວາມເຈັບປວດ, ຄວາມຄຽດແຄ້ນແລະຄວາມກຽດຊັງ.

ມີຄໍາເວົ້າ vulgar ທີ່ເວົ້າວ່າ: "ລູກຊາຍຂອງຄົນອື່ນມີກິ່ນຫອມໃນທຸກໆບ່ອນ". ໂດຍທໍາມະຊາດໃນນີ້ຍັງມີຂໍ້ຍົກເວັ້ນແຕ່ສິ່ງເຫຼົ່ານີ້ສາມາດນັບໄດ້ດ້ວຍນິ້ວມືຂອງມືແລະມີນິ້ວມືຫຼາຍເກີນໄປ.

ການໂຕ້ຖຽງກັນລະຫວ່າງພໍ່ແລະແມ່ຍ້ອນຄວາມອິດສາ, ການຮ້ອງໄຫ້ແລະການຮ້ອງໄຫ້ຂອງແມ່ທີ່ໂສກເສົ້າຫຼືຜົວທີ່ຖືກກົດຂີ່, ຖືກທໍາລາຍແລະສິ້ນຫວັງ, ອອກຈາກເຄື່ອງຫມາຍທີ່ບໍ່ສາມາດລົບລ້າງໄດ້ຂອງຄວາມເຈັບປວດແລະຄວາມໂສກເສົ້າຢ່າງເລິກເຊິ່ງຢູ່ໃນບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍທີ່ບໍ່ເຄີຍລືມຕະຫຼອດຊີວິດ.

ໃນເຮືອນທີ່ສະຫງ່າງາມ, ແມ່ຍິງທີ່ພູມໃຈທາລຸນຜູ້ຮັບໃຊ້ຂອງພວກເຂົາເມື່ອພວກເຂົາໄປຮ້ານເສີມສວຍຫຼືແຕ້ມຫນ້າຂອງພວກເຂົາ. ຄວາມພາກພູມໃຈຂອງແມ່ຍິງແມ່ນມີຄວາມຮູ້ສຶກເຈັບປວດ.

ເດັກນ້ອຍທີ່ເຫັນສະຖານະການທີ່ໂຫດຮ້າຍທັງຫມົດເຫຼົ່ານີ້ຮູ້ສຶກເຈັບປວດໃນສ່ວນເລິກຂອງຫົວໃຈຂອງລາວບໍ່ວ່າລາວຈະເຂົ້າຂ້າງແມ່ທີ່ຍິ່ງໃຫຍ່ແລະມີຄວາມພູມໃຈຂອງລາວ, ຫຼືເຂົ້າຂ້າງຜູ້ຮັບໃຊ້ທີ່ບໍ່ມີຄວາມສຸກແລະອັບອາຍແລະຜົນໄດ້ຮັບມັກຈະເປັນໄພພິບັດຕໍ່ບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍ.

ນັບຕັ້ງແຕ່ໂທລະພາບໄດ້ຖືກປະດິດຂຶ້ນ, ຄວາມສາມັກຄີຂອງຄອບຄົວໄດ້ສູນເສຍໄປ. ໃນອະດີດ, ຜູ້ຊາຍໄດ້ມາຮອດຈາກຖະຫນົນແລະໄດ້ຮັບການຕ້ອນຮັບຈາກພັນລະຍາຂອງລາວດ້ວຍຄວາມສຸກຫຼາຍ. ມື້ນີ້, ຜູ້ຍິງບໍ່ໄດ້ອອກໄປຕ້ອນຮັບຜົວຂອງລາວຢູ່ທີ່ປະຕູເພາະວ່ານາງຫຍຸ້ງຢູ່ກັບການເບິ່ງໂທລະພາບ.

ພາຍໃນເຮືອນທີ່ທັນສະໄຫມ, ພໍ່, ແມ່, ລູກຊາຍ, ລູກສາວ, ປະກົດວ່າເປັນເຄື່ອງຈັກທີ່ບໍ່ມີສະຕິຢູ່ຕໍ່ຫນ້າຫນ້າຈໍໂທລະພາບ.

ດຽວນີ້ຜົວບໍ່ສາມາດສະແດງຄວາມຄິດເຫັນກັບຜູ້ຍິງກ່ຽວກັບບັນຫາຂອງມື້, ວຽກງານ, ແລະອື່ນໆ, ເພາະວ່ານາງເບິ່ງຄືວ່ານອນຫລັບເບິ່ງຮູບເງົາຂອງມື້ວານນີ້, ສະຖານະການ Danteesque ຂອງ Al Capone, ການເຕັ້ນລໍາຄັ້ງສຸດທ້າຍຂອງຄື້ນໃຫມ່, ແລະອື່ນໆ.

ເດັກນ້ອຍທີ່ລ້ຽງດູໃນເຮືອນປະເພດໃຫມ່ທີ່ທັນສະໄຫມພຽງແຕ່ຄິດເຖິງປືນໃຫຍ່, ປືນ, ປືນກົນຂອງຫຼິ້ນເພື່ອຮຽນແບບແລະດໍາລົງຊີວິດໃນແບບຂອງຕົນເອງສະຖານະການ Danteesque ທັງຫມົດຂອງອາຊະຍາກໍາຕາມທີ່ເຂົາເຈົ້າໄດ້ເຫັນໃນຫນ້າຈໍໂທລະພາບ.

ມັນເປັນຄວາມສົງສານທີ່ການປະດິດສ້າງທີ່ປະເສີດນີ້ຂອງໂທລະພາບຖືກນໍາໃຊ້ເພື່ອຈຸດປະສົງທີ່ທໍາລາຍ. ຖ້າຄົນເຮົາໃຊ້ສິ່ງປະດິດສ້າງນີ້ໃນວິທີການທີ່ດີ, ບໍ່ວ່າຈະສຶກສາວິທະຍາສາດທໍາມະຊາດ, ບໍ່ວ່າຈະເປັນການສອນສິລະປະທີ່ແທ້ຈິງຂອງ MOTHER NATURA, ບໍ່ວ່າຈະເປັນການໃຫ້ຄໍາສອນອັນສູງສົ່ງແກ່ປະຊາຊົນ, ຫຼັງຈາກນັ້ນການປະດິດສ້າງນີ້ຈະເປັນພອນແກ່ມະນຸດ, ມັນສາມາດນໍາໃຊ້ຢ່າງສະຫຼາດເພື່ອປູກຝັງບຸກຄະລິກກະພາບຂອງມະນຸດ.

ມັນເປັນເລື່ອງໄຮ້ສາລະຢ່າງແທ້ຈິງທີ່ຈະບໍາລຸງລ້ຽງບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍດ້ວຍດົນຕີທີ່ບໍ່ມີສຽງ, ບໍ່ມີສຽງດັງ. ມັນເປັນສິ່ງໂງ່ທີ່ຈະບໍາລຸງລ້ຽງບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍດ້ວຍເລື່ອງຂອງໂຈນແລະຕໍາຫຼວດ, ສະຖານທີ່ຂອງຄວາມຊົ່ວຮ້າຍແລະການຄ້າປະເວນີ, ລະຄອນການຫລິ້ນຊູ້, ຮູບພາບລາມົກ, ແລະອື່ນໆ.

ຜົນໄດ້ຮັບຂອງຂັ້ນຕອນດັ່ງກ່າວທີ່ພວກເຮົາສາມາດເຫັນໄດ້ໃນ Rebel ໂດຍບໍ່ມີສາເຫດ, ຄາດຕະກອນກ່ອນໄວອັນຄວນ, ແລະອື່ນໆ.

ມັນເປັນເລື່ອງທີ່ຫນ້າເສຍໃຈທີ່ແມ່ຕີລູກຂອງພວກເຂົາ, ໃຫ້ໄມ້ຕີ, ດູຖູກພວກເຂົາດ້ວຍຄໍາສັບທີ່ແຕກຫັກແລະໂຫດຮ້າຍ. ຜົນໄດ້ຮັບຂອງການປະພຶດດັ່ງກ່າວແມ່ນຄວາມຄຽດແຄ້ນ, ຄວາມກຽດຊັງ, ການສູນເສຍຄວາມຮັກ, ແລະອື່ນໆ.

ໃນການປະຕິບັດ, ພວກເຮົາສາມາດເຫັນໄດ້ວ່າເດັກນ້ອຍທີ່ລ້ຽງດູລະຫວ່າງໄມ້ຕີ, ແສ້ແລະສຽງຮ້ອງ, ກາຍເປັນຄົນທໍາມະດາທີ່ເຕັມໄປດ້ວຍຄວາມໂງ່ຈ້າແລະຂາດຄວາມເຄົາລົບແລະການນະມັດສະການ.

ມັນເປັນສິ່ງຈໍາເປັນຢ່າງຮີບດ່ວນທີ່ຈະເຂົ້າໃຈຄວາມຕ້ອງການທີ່ຈະສ້າງຄວາມສົມດູນທີ່ແທ້ຈິງພາຍໃນເຮືອນ.

ມັນເປັນສິ່ງຈໍາເປັນທີ່ຈະຮູ້ວ່າຄວາມຫວານແລະຄວາມຮຸນແຮງຕ້ອງມີຄວາມສົມດູນກັນໃນສອງດ້ານຂອງຂະຫນາດຂອງຄວາມຍຸຕິທໍາ.

ພໍ່ເປັນຕົວແທນຂອງ SEVERITY, ແມ່ເປັນຕົວແທນຂອງ DULZURA. ພໍ່ເປັນຕົວແທນຂອງ WISDOM. ແມ່ເປັນສັນຍາລັກຂອງ LOVE.

WISDOM ແລະ LOVE, SEVERITY ແລະ SWEETNESS ມີຄວາມສົມດູນກັນຢູ່ໃນສອງດ້ານຂອງຂະຫນາດ cosmic.

ພໍ່ແລະແມ່ຄວນມີຄວາມສົມດູນກັນເພື່ອຜົນປະໂຫຍດຂອງເຮືອນ.

ມັນເປັນເລື່ອງຮີບດ່ວນ, ມັນເປັນສິ່ງຈໍາເປັນ, ສໍາລັບພໍ່ແມ່ທຸກຄົນທີ່ຈະເຂົ້າໃຈຄວາມຕ້ອງການທີ່ຈະປູກໃນຈິດໃຈຂອງເດັກນ້ອຍຄຸນຄ່າ ETERNAL ຂອງ SPIRIT.

ມັນເປັນເລື່ອງທີ່ຫນ້າເສຍໃຈທີ່ເດັກນ້ອຍທີ່ທັນສະໄຫມບໍ່ມີຄວາມຮູ້ສຶກ VENERATION, ນີ້ແມ່ນຍ້ອນວ່າເລື່ອງເລົ່າຂອງ cowboys ໂຈນແລະຕໍາຫຼວດ, ໂທລະພາບ, ຮູບເງົາ, ແລະອື່ນໆໄດ້ເຮັດໃຫ້ຈິດໃຈຂອງເດັກນ້ອຍເສື່ອມໂຊມ.

ຈິດຕະວິທະຍາ REVOLUTIONARY ຂອງ MOVEMENT GNOSTIC, ໃນຮູບແບບທີ່ຊັດເຈນແລະຊັດເຈນເຮັດໃຫ້ຄວາມແຕກຕ່າງລະຫວ່າງ EGO ແລະ ESSENCE ຢ່າງເລິກເຊິ່ງ.

ໃນໄລຍະສາມຫຼືສີ່ປີທໍາອິດຂອງຊີວິດ, ພຽງແຕ່ຄວາມງາມຂອງ ESSENCE ສະແດງອອກໃນເດັກ, ຫຼັງຈາກນັ້ນເດັກນ້ອຍອ່ອນໂຍນ, ຫວານ, ງາມໃນທຸກໆດ້ານທາງຈິດໃຈຂອງລາວ.

ເມື່ອ EGO ເລີ່ມຄວບຄຸມບຸກຄະລິກກະພາບຂອງເດັກນ້ອຍທີ່ອ່ອນໂຍນ, ຄວາມງາມທັງຫມົດຂອງ ESSENCE ເລີ່ມຫາຍໄປແລະແທນທີ່ຂໍ້ບົກພ່ອງທາງຈິດໃຈຂອງມະນຸດທຸກຄົນເລີ່ມຕົ້ນ.
