ສັດລ້ຽງລູກດ້ວຍນົມທີ່ສົມເຫດສົມຜົນທີ່ເອີ້ນວ່າຄົນ, ແທ້ຈິງແລ້ວບໍ່ມີເອກະລັກທີ່ແນ່ນອນ. ໂດຍບໍ່ມີຂໍ້ສົງໃສ, ການຂາດຄວາມສາມັກຄີທາງຈິດໃຈໃນມະນຸດແມ່ນສາເຫດຂອງຄວາມຫຍຸ້ງຍາກແລະຄວາມຂົມຂື່ນຫຼາຍຢ່າງ.

ຮ່າງກາຍທາງກາຍະພາບເປັນຫົວໜ່ວຍທີ່ສົມບູນ ແລະເຮັດວຽກເປັນທັງໝົດທາງອິນຊີ, ເວັ້ນເສຍແຕ່ວ່າເຈັບປ່ວຍ. ແຕ່ຊີວິດພາຍໃນຂອງມະນຸດບໍ່ແມ່ນຄວາມສາມັກຄີທາງຈິດໃຈ. ສິ່ງທີ່ຮ້າຍແຮງທີ່ສຸດຂອງທັງໝົດນີ້, ເຖິງວ່າຈະມີສິ່ງທີ່ໂຮງຮຽນປະເພດ Seudo-Esoteric ແລະ Seudo-Ocultist ເວົ້າວ່າ, ແມ່ນການຂາດອົງການຈັດຕັ້ງທາງຈິດໃຈໃນຄວາມເລິກລັບຂອງແຕ່ລະຄົນ.

ແນ່ນອນ, ໃນສະພາບການດັ່ງກ່າວ, ບໍ່ມີການເຮັດວຽກທີ່ກົມກຽວກັນເປັນທັງໝົດໃນຊີວິດພາຍໃນຂອງຄົນ. ມະນຸດ, ກ່ຽວກັບສະຖານະພາຍໃນຂອງມັນ, ແມ່ນຄວາມຫຼາກຫຼາຍທາງຈິດໃຈ, ລວມຍອດຂອງ "ຂ້ອຍ".

ຄົນໂງ່ທີ່ສະຫວ່າງຂອງຍຸກທີ່ມືດມົວນີ້ເຄົາລົບ "ຂ້ອຍ", ພວກເຂົາເຮັດໃຫ້ມັນເປັນພະເຈົ້າ, ພວກເຂົາເອົາມັນໄວ້ເທິງແທ່ນບູຊາ, ພວກເຂົາເອີ້ນມັນວ່າ "ALTER EGO", "ຂ້ອຍທີ່ສູງກວ່າ", "ຂ້ອຍທີ່ສູງສົ່ງ", ແລະອື່ນໆ. "Sagebrush" ຂອງອາຍຸສີດໍາທີ່ພວກເຮົາອາໄສຢູ່ບໍ່ຕ້ອງການທີ່ຈະຮັບຮູ້ວ່າ "ຂ້ອຍທີ່ສູງກວ່າ" ຫຼື "ຂ້ອຍຕ່ໍາກວ່າ" ແມ່ນສອງພາກສ່ວນຂອງ Ego ດຽວກັນ pluralized ...

ມະນຸດແນ່ນອນບໍ່ມີ "ຂ້ອຍຖາວອນ" ແຕ່ເປັນຝູງຊົນຂອງ "ຂ້ອຍ" ທີ່ບໍ່ເປັນມະນຸດແລະບໍ່ມີເຫດຜົນທີ່ແຕກຕ່າງກັນ. ສັດທາງປັນຍາທີ່ທຸກຍາກທີ່ເອີ້ນວ່າຄົນຜິດພາດແມ່ນຄ້າຍຄືກັບເຮືອນທີ່ບໍ່ເປັນລະບຽບບ່ອນທີ່ແທນທີ່ຈະເປັນນາຍຈ້າງ, ມີຄົນຮັບໃຊ້ຈໍານວນຫຼາຍທີ່ຕ້ອງການທີ່ຈະສັ່ງແລະເຮັດສິ່ງທີ່ເຂົາເຈົ້າຕ້ອງການ ...

ຄວາມຜິດພາດທີ່ໃຫຍ່ທີ່ສຸດຂອງ Seudo-Esotericism ແລະ Seudo-Occultism ລາຄາຖືກແມ່ນການສົມມຸດວ່າຄົນອື່ນມີຫຼືມີ "ຂ້ອຍຖາວອນແລະບໍ່ປ່ຽນແປງ" ໂດຍບໍ່ມີການເລີ່ມຕົ້ນແລະບໍ່ມີທີ່ສິ້ນສຸດ ... ຖ້າຜູ້ທີ່ຄິດເຊັ່ນນັ້ນຕື່ນຂຶ້ນສະຕິເຖິງແມ່ນວ່າສໍາລັບຊົ່ວຄາວ, ພວກເຂົາສາມາດສະແດງໃຫ້ເຫັນຢ່າງຊັດເຈນດ້ວຍຕົວເອງວ່າມະນຸດທີ່ມີເຫດຜົນບໍ່ເຄີຍຄືກັນເປັນເວລາດົນນານ ...

ສັດລ້ຽງລູກດ້ວຍນົມທາງປັນຍາ, ຈາກທັດສະນະທາງຈິດໃຈ, ມີການປ່ຽນແປງຢ່າງຕໍ່ເນື່ອງ ... ການຄິດວ່າຖ້າຄົນຖືກເອີ້ນວ່າ Luis ລາວແມ່ນ Luis ສະເຫມີ, ແມ່ນບາງສິ່ງບາງຢ່າງເຊັ່ນເລື່ອງຕະຫລົກທີ່ບໍ່ດີ ... ຫົວຂໍ້ນັ້ນທີ່ເອີ້ນວ່າ Luis ມີ "ຂ້ອຍ" ອື່ນໆໃນຕົວມັນເອງ, egos ອື່ນໆ, ທີ່ສະແດງອອກໂດຍຜ່ານບຸກຄະລິກຂອງລາວໃນເວລາທີ່ແຕກຕ່າງກັນແລະເຖິງແມ່ນວ່າ Luis ບໍ່ມັກຄວາມໂລບ, "ຂ້ອຍ" ອື່ນໃນລາວ - ໃຫ້ເອີ້ນວ່າ Pepe - ມັກຄວາມໂລບແລະອື່ນໆ ...

ບໍ່ມີໃຜຄືກັນໃນແບບຕໍ່ເນື່ອງ; ແທ້ຈິງແລ້ວມັນບໍ່ຈໍາເປັນຕ້ອງມີສະຕິປັນຍາຫຼາຍເພື່ອຮັບຮູ້ການປ່ຽນແປງແລະຄວາມຂັດແຍ້ງທີ່ນັບບໍ່ຖ້ວນຂອງແຕ່ລະຄົນ ... ສົມມຸດວ່າບາງຄົນມີ "ຂ້ອຍຖາວອນແລະບໍ່ປ່ຽນແປງ" ເທົ່າກັບການລ່ວງລະເມີດເພື່ອນບ້ານແລະຕົນເອງຢ່າງແນ່ນອນ ...

ມີຄົນຈໍານວນຫຼາຍອາໄສຢູ່ພາຍໃນແຕ່ລະຄົນ, "ຂ້ອຍ" ຈໍານວນຫຼາຍ, ສິ່ງນີ້ສາມາດຖືກກວດສອບໂດຍບຸກຄົນທີ່ຕື່ນນອນ, ສະຕິ, ໂດຍກົງ.
