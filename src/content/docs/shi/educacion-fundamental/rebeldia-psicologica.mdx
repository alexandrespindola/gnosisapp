Winna iran ifkan ad dduṛun akk tamurt n ddunit bac ad qqnen ɣef akk tewsatin n ufgan, ufan s yiman nsen belli tbiɛt n aɣersiw amsikel azekkʷaɣ sɣalen as s yisem n ufgan, tga kullu-tt samaḥ, ad tili deg Uruppa taqdimt neɣ deg Afryqa yuḍnen s wawal n tkla, deg akal idusan n Vedas neɣ deg Indyas Tigut, deg Ustriyya neɣ deg Ccinwa.

Asiggl agi yuggit, tidett tameqqrant agi yuḍnen aṣmmuḍu i kullu argaz ameqqran, izmer ad ittwafham s wudem igadda ma iran umessfr ad iẓer Timlmadin, Timelsin d Tasdawiyin.

Nddaɣ s tallit n uzwiru deg tayyuga. Ḍar kullu iṭṭas ittwazwer deg cinta taysayt d tameqqrant. Taysatin n Iferḍisen, Tikarrusin, Tiggaṭ n lux, atg., atg., atg.

Ḫas ma yebdda, kra n uzurɣi, iga tidett mqqurten belli Timlmadin Tinesliyin, Tasdawiyin, atg. uɣalent daɣen d tiẓẓuẓin imsiklin deg uzwiru deg tayyuga.

G tiramawin agi n uzwiru deg tayyuga iɣtas ukan amzwaru deg tudert iga ttafent tazwert tanemɣurt. Imdanen ggʷadan kullu aylliɣ ddrun tazwert.

Afker amsuṭa deg tallit agi n uzwiru deg tayyuga, ittyebɣa ar yefsus acku anaw amaynu n Asgmi ittgadda ɣef tiɣtasin irɣan.

"Tafuyt Tamaynut" teddar bahra tettafag d tazzwart tamsikelt agi. Ma iran kra ad yili meẓẓi ɣef wid akk yaḍnin, kullu argaz ttaḥecciḍit, kullu argaz iḍerriqit, ittwafriq, ittwannɣaɣ uɣrum, atg.

Aseqsi ad ɣlin idrimen bac ad idder umdan d ad ismirir, asekwti ad issaɣ amessɣar deg tudert, asgiddi n tazwert, ad isɣuḍa kra mqqar ad ismirir zdat n wiyaḍ, atg., ad isserwes afker iffɣn, agennu d aɣtusu.

Ittwafahem s wudem kamel belli agwdi iẓuḍ afker d iqesseḥ ul.

G tiramawin agi n agwdi d asgiddi n tazwert, imdanen ttḍarren deg tiɣarḍayin nsen, deg tizwar nsen, deg takʷnnart nsen, deg adeg anda sɣalen zemren ad ilin bahra ddren, meẓẓi n uguren ur iran ad ffuɣn ɣinna, ggʷadan tudert, agwdi n uɣarasen imaynuten, iɣarasen imaynutn, atg., atg., atg.

Kullu asgmi agi ittyaɣran iṭṭas ittgadda ɣef agwdi d asgiddi n tazwert, imdanen ttuggʷadn, ggʷadan ar tɣḍa nsen yiman nsen.

Imdanen, ggʷadan kullu, tekksen ad ffɣen seg idɣaren ittwabadln, ad ilin meẓẓi ɣef imdanen yaḍnin, ad fikiren s wudem azemni, ad qezḍen d uggugen n Tgemmi yuḍnen, atg.

Ṣah tddar deg ddunit kra n iḍuran d ifehmnen, aylliɣ iran ad ssaɣen bahra uguren n afker, maca deg ungalu ameqqran nnekni ur illa ula d ul n usmuṭṭeg d uzenni.

Illa snat tewsatin n UZENNI iddaw ttwabdlen. Amzwaru: Azenni Amsikel amesɣuḍi. Anmuggar: Azenni Amsikel ɣaran n TMAZGIRT.

Anaw amzwaru n Uzenni iga Amqqeddem amɣar d amserrɣi. Anaw wis sin n Uzenni iga AZEMNI.

Deg unaw amzwaru n Uzenni Amsikel nttafa Amɣarras iṛemḍen ticeṭṭaḍ tiqdimin d iɛemmen aɣrub n igudar tiqdimin bac ur ttgʷrrint, anaw aṣṣaq, Azemni n idammen d tizwit, anɣlaf n trummiwin d tignatin n Tamurt, argaz n ufḍal deg tɣmrt, Amɣar iɣtan isserwes ad yawi ɣef uɣrub wid akk ur iqbeln tiramawin nsen, tiyurin nsen.

Deg unaw wis sin n Uzenni Amsikel nttafa BUDDHA, a JESUS, a HERMES, amqqeddem, AMAZNI AMSIKEL, AMASSAY, iɣarn mqqurnin n TRUMMIT N TADWI, atg., atg., atg.

Wid ukkan tssɣalin ukan s tɣtasit taɣewwaɣt ad nɣlin idɣaren imeqqranin deg tazzit tazurɣit, ad alin, ad rkkn ɣef uɣrub n tarrayt, ad issinɣen yiman nsen, atg., teksen degsen tidett ɣarant, ugin Iɛemmagen s tbiɛt, ẓaydin, uɣriɣn, miyya s miyya iɛerriḍn.

Ittrar bahra ar azduɣu belli mara deg ufgan ur illa izdi idusan n afker d ul, xas nerḍa a nerḍa Asgmi ameqqran, tudert terẓa teẓli, tettuweḍḍa, tettuḥerrq s ugwda ur ittufsisen n kullu anaw.

Berra n kullu uggʷad d war uggʷad, nezmer ad nɣer s tazzwart belli war Asgmi IZDAN, tudert terẓa tḍur, ur terɣi d tettaḍar.

AƔERSIW AMSIKEL illa yas EGO IDUSAN ittuybedd s tidett ɣef TIZDAɣIN timqqurrin ittwazḍaṛen d ASGMI AZERRɣ.

YO PLURALIZADO i kullu nnekki nttawi deg, iga ssebba tamzwarut n kullu uggugen nneɣ d tzegnanin.

ASGMI AMIKSAN ilaq ad iselmed i tuzzulin timaynutin DIDÁCTICA nneɣ Amsikelt bac ad NAWWI ɣef YO.

S ukkan ad ttwassulin tiẓḍrin i deg ukan tkʷnnan Ego (YO) nezmer ad nebdar degnneɣ ammas anessaw n tadwit yiman, iseqsa ad nili NEZDANEN.

Ma illa deg kullu nneneɣ EL YO PLURALIZADO, ur ukan neɣɣir tudert i yiman nneɣ amma neɣɣirtt i wiyaḍ.

Ma yella tssɣurin azref d nttili imqqedmn, ma nesmers uggugen? Ma yella nggem deg afker nneɣ kra n iswilen, ma nssenkert? Ma yella izmar ibaddln d inesliyen ma nnessmersitent bac ad nɣen imettiwen nneɣ?

Ur terɣi ad nsselmed, ad nɣer deg tmlmadin, ad nɣer, ma deg usmeres n uzal nesserni ad nenɣ yiman nneɣ.

Iɣtas n usgmi ur ilaq ad yili ukan ittwazwer kullu aseggas imaɣranen imaynuten n umursi, anaw amaynu n iɛerriḍn, iqerqacen imaynuten ur issinen ula ad rezqen Din n umetti, atg.

Tidett n iɣtas n ASGMI AMIKSAN ilaq ad yili usbedd n irgazen d tmɣarin IZDANEN d iseqsa d imsiklen.

S tidett Imɣaren d Tmeɣarin n Tmlmadin, Timelsin d Tasdawiyin, kullu fikiren, meẓẓi deg unzar n TMAZGIRT IZDAN n imelmaden.

Kra n ufgan izmer ad iḥassd d ad iɣtas izwilen, tiɣerniwin, iḍibelmn d ar ad yili s wudem ifulkin deg adeg amkini n tudert, maca agi ur tssengad iga Amsikel.

TMAZGIRT ur tezmer ad tili ukan isserwes tazmert tamsilɣt, TMAZGIRT ur tezmer ad tili targga n wawal aḍif n iktabn, TMAZGIRT ur iga ɣur ad tesserwes s wawal deg kra n tɛessast. TMAZGIRT ur iga wawal n uskan. TMAZGIRT iga tazmert bac ad taqqen s wudem igadda ESENCIA, TIDETT, aylliɣ tidett IGA.

ASGMI AMIKSAN iga tussna s aɣ d ijerreḥn tazmert agi deg yiman nneɣ d deg wiyaḍ.

ASGMI AMIKSAN ittawiɣ i kullu UMDAR ad iggdem Izweran idusan aylliɣ ddin s tagga n umessi ɣaran d UTTAFI IZDAN n YIMAN.

Mara degnneɣ ur illa AUTO-CONOCIMIENTO, iseqsa AUTO-EXPRESION tettuli d AUTO-AFIRMACION iɣes d amesserɣ.

ASGMI AMIKSAN ittgadda ukan ɣef unzar deg kullu umdar tazmert bac ad ifhem yiman nes deg kullu igenwan n afker d ur ukan ad iggdem i asmirir n AUTO-EXPRESION azerrɣ n YO PLURALIZADO.
