G tmguri, nssnufa ad nẓṛ is akkʷ id mddraben imatarialistn d wid iruḥanin gantn akkʷ ṛẓmnin ɣ ddu udɣar n ddugma n ufraḍ.

Arra n zriɣ ɣ tizi ad f tɣssa n ufgan d ufraḍ nns iɛdan, ɣ wul nns gantn tignit n usuggl ibzan, ur tzdɣrn i ugḍuf iɣusan.

Bla ma nẓẓuẓun i akkʷ imzaray n Darwin ittwafɣan zun d isli n iman amɣẓan sɣ ɣur Karl Marx d umatarializm nns ittwabdrn, wla kra ẓran imusnawen n tizi ad f tɣssa n ufgan, wla kra isstn, wla kra ujrn s wudem ibdda d urta gn kra n tsddastin tusdidin, tid usdidin, f ufraḍ n ufgan.

Macc, ma nɛmmd iman n ufgan amaziɣ, aylliɣ d ɣilad n ṛbiɛin id mlyun n usggʷas dat Ɛisa, nẓṛ tmassanin tusdidin, isklan ur ittnɣlin n yan wanaw f ufgan iggutn, ur ittaqbal ufgan n tizi ad, d mani nnaɣ nkka nẓẓul s mddn imqqurn, igḍaḍ imzwura, id buṛyamatn ittawssan, iɣẓwan icmmnan, igḍaḍ igḍfn d imuqriyn imzwura.

Ɣef ufgan udat amaziɣ, i ikritusn iɛjbn d irɣayn ɣ wudem ittuqqsn i unɣmisu amsuggl mac ar tn mgdaln, ar tn mgdaln, ar tn mgdaln imuqriyn d ibddar nnɣan s ddaw n wul nns ɣ yigran innaɣ n tmizar n uɣris n tafukt, wla kra ẓran imusnawen n tizi ad s wudem ibdda d ugmr-inkr.

Tmassant Agnustica tsslad is unɣmis amsuggl zun kullu ma nẓṛn, ur igi yan ɣ uɣaras, ur illi ufgan ɣ tɣssa tumlilt n awal; tayri ttmjjaṛt ar tɣssa, dffir ɣ ttnṭṭa ar ttgʷt ar ttmjjaṛt nns neɣ ad tɣrs akkʷ igiten nns d ad trẓ.

Tiggura n ufraḍ d unɣal gan taɣẓunt tamikant n tayri, ur gan kra n ufraḍ n wul nns.

G ddu unɣmis amsuggl gantt igiten imqqurn nnaɣ ittmjjaṛn neɣ ittrẓun, ur illi uɣaras nnaɣ ittmjjaṛn. Amik n ufraḍ ur ittmjjaṛn.

Amjjaṛ n wannaw n igiten imẓẓiyn, ar nns nkkatin ɣ tɣssa yusinɣan d aya iga timhalin tigʷḍaḍanin sɣ ɣur mddn d tmusa tuzzumt sɣ ɣur imɣarn nnaɣ ujrn tawwuri-nnaɣ ɣ tizi iɛdan.

Win irẓm ad imjjaṛ akkʷ igiten nns imẓẓiyn ad tɣwrri ɣ ufgan, urta ittaɣ ɣ ubrid n tṛwla n tɣssa.

Unɣmis amsuggl iga ajgu, tamagit; sɣ tamagit-nnaɣ nkkatin ad ilul umɣar n tudrt, ufgan amqqran, ufgan-nnaɣ illa ittnṭṭaf Diogenes s tagḍiḍt tzddɣa s imruray n Atenas d ɣ yat n wass d ur tn igi.

Ur illi uɣaras ɣ wanazuṛ aɣssa, tamagit i yusinɣan ad ttmjjaṛt, aylli ittawssan, aylli iga iga ad trẓ.

Ufgan amqqran iga ɣ ufgan amsuggl, zun d usigna iga s tagut.

Ma ur imut ajgu, tamagit ur ittmjjaṛ, illa urta trẓt tanaɣ, iwiw, iman nnu, fkka ad ilul ufgan.

Imɣarn d imɣartn n id mddraben, id kulijin d id unibersitatn, urta tslmdn i inlmudn nsn abrid n tannirt tarurayt, anɣa kkan nkkatin ad nɣn tanaɣ.

Nɛmmd fkka tamawt nɣ is tṛwla n tɣssa ur igi anɣa ɣ ddunit-aɣ, maca kullu ma ad tga anɣa d anɣa.

Tṛwla n tɣssa tɣlla tɣawsiwin iɣusan: Amzwaru, ad nmt; Ayyur, ad nlul; Ayyad, ad nttaɣar i mddn. Tagglayt n tɣawsiwin ur taɣal taɣssa.

Ad nmt illa tmhḍart n tannirt tarurayt d uddis n tanaɣ tanfsant.

Ad nlul iga tmhḍart n ussudds n tzdayt, aya iga taɣssa i taɣssa tanamurt, win irẓmn ad ittnɣam aya, urta inɣam d ad nssn id ktbn nɣ tid Agnostic.

Ad nttaɣar i mddn iga tannirt tamagdayt tuzdidt.

Ma ur nra tṛwla n tɣssa, ma ur nga timhalin tigʷḍaḍanin ad nmjjaṛ igiten imẓẓiyn nnaɣ ad ntdda ar ad nẓṛ iman nnu, illa is igiten-inna ur taɣurn ad ttumjjaṛn.

Gan anɣa wid ittagin iman nsn, wid ittuslkn d g aya ur illa urtaɣi, ɣassa ad yili unɣmis amsuggl iggutn aylli ur irẓm?

Nttaɣ azɣar iggutn akkʷ d ubdda mac ur illi mddn akkʷ iran azɣar-nnaɣ, ur tn irẓm, ur tn issn d nnɣ is d ur ttqbaln, ur tn ittaɣ aray. Ɣassa ad yili kkn s tɣssa ma ur iran?

Tɣssa illa dat ad nɣrs yan ammas n uzddig neɣ ammas n uzddar, ma ur issin ula nḍɣar d ma ur nn illan, urta nɣrs ammas d uzddar nnaɣ ittwahaln s tɣssa mac ur nn illan.
