G ufella n tala tuggugt n Delfus, illa yat tẓerbit tazaykut ittwaksin ɣef uzru araqqaḍ, tettini: "UR teẓẓuẓḍ iman-k". Ẓẓuẓḍ iman-k ad taẓẓuẓḍ ula asmam d izuzen.

Tussna tazzaykut n usarag tesɛa zdat-s awal ameqqran n imqranen igrikiyn iqburen.

Ma nerda s tideț ad nesbedd tiggejda i usarag ilan, ilaq ad nɛeqqel iman-nneɣ deg akk iswiyyat n ummaqqel.

Asbedd n tiggejda ilan i usarag iga d tidet ad teffɣeḍ seg uzzaɛ, ufgan, uggad, uray, ṭṭmeɛ n tezmertin tisiḥriyyin, arz n igmamn, atg., atg., atg.

Iban i medden d ur illa kra n cck, belli mrawad asbedd n TUGGUGT TAMEQQRANT n usarag, ummaqqel ittuɣal ilelli d amseɛdel deg iɣummaren iqqbuṛen.

Seg tama n umẓẓan iṣḥan, ur yusi kra ad nɛic AYEN NEȚȚA N TIDET bla ma nɛeqqel iman-nneɣ.

Yeẓḍar ad nɛeqqel deg umata d deg akk iberdan n ummaqqel, yal iɣtan s yat-nnɣ, yal aray, yal asmekti, yal tɣawsa iṭṭefn tudert-nneɣ, atg.

Iban i medden belli g tɣawsiwin n usarag, ittɛedda ɣef uɣaras n ummaqqel deg tɛeccamt taberkant, akk tɣawsiwin n tudert-nneɣ iṭṭefn tudert-nneɣ, akk ifrah d tɣawsiwin icemmen, ismektayen ur ttamun, igiten iḍran seg agʷns neɣ seg barra, arayen n yal anaw, izmzalen n yal tamdda, irɣuten iqburen, irayen, atg.

Win yuran ad isbedd deg ummaqqel-nnes tiggejda tameqqrant n usarag, isenta ad issentaɣ i warraten ila d imarayen n tɣuri-nneɣ d ad ten-yeɛqel deg umata, ur tḥsabnt ila deg iswi n ummaqqel, maca ula deg akk iberdan imsedfen, imẓen d ur ilin n ummaqqel. Ur nttafa ad nttu belli ummaqqel ila iswiyyat uggar.

Aɣuru n uẓuran n akk warraten-a iga d tidet taẓẓurt n iman.

Akk tfilmt ɣef uɣaras n ummaqqel ila tazwara d taggara. Mrawad ad tefk tfulki n iɣriḍen, arayen, izmzalen, uẓẓaɛen, ismektayen, atg., ummaqqel ittuɣal ilelli d amseɛdel deg iɣummaren imqqranen ILELLI seg akk anawn n immaqqalen.

Inelmaden imaynutn n tussna n ifganen ḥwaǧn ad ɛicin ILELLI YEZZEN. Assaɣ n ILELLI deg ummaqqel-nneɣ yettarra ad nɛic, ad nɛiqqel, ad ndarɣal yat tɣawsa itbeddiln, TɣAWSA-NNEƔ d AYEN NEȚȚA N TIDET.

Mmeyyez gar ummaqqel illan ilelli d ummaqqel iṭṭafen ḍerṣ.

Mmeyyez gar ummaqqel illan deg iɣummaren d ummaqqel yeggugmen s ḍerṣ.

Ɣef uɣaras n kra n uɣtas ilan ilaq ad nɛeqqel belli mi illa ummaqqel ilelli s ḍerṣ, deg iɣummaren n wiyyaḍ ur illa ilelli yettɛaya ad yeffeɣ.

Seg tama n uẓẓan ilan ilaq ad nɛeqqel belli mi illa ummaqqel yeggugmen s ḍerṣ, deg iɣummaren ur illa deg iɣummaren, yettɛayya d ittaɣal mqqar.

Taɣyult d iɣummaren iṣḥan n ummaqqel, tussaɣ-d ɣurneɣ am tɣawsa ihwan, am tazdayt, mi ttfakka tfilmt ittaḍun tudert-nneɣ deg uɣaras izdaɣen n ummaqqel.

Ila mrawad illa ummaqqel ilelli s tɣarast-nnes, ila mrawad illa ummaqqel deg iɣummaren ilan, tussaɣ-d tirrugza n ILELLI YEZZEN.

ILELLI ur yusan ad ittyara. Ur yusan ad ittyara neɣ ad ittyawi, kra n aray s نححح yusan ɣurneɣ yusa ad igʷddi deg udɣar ameqqran.

ILELLI ur yusan ad ittyara neɣ ad ittyara deg awal. Agʷdira n uɣbalu illa ittwafḍal i sinen tɣawsiwin, arayen d imrayen illan; ur ilaɣ ad yawi s tɣarast ilan d tazzaykut, tɣawsiwin, tɣawsiwin d imrayen UR LLININ.

Ad nzuzzer ɣef ILELLI deg imzirugen n uɣbalu يززن s tɣawsiwin illanin, illa s tideț, iga d tazzaykut d ur ilan.

"ILELLI iga d UR ILLI, D ILLI UR IGI ILELLI".

"TɣARAST UR TGLA ILELLI, D ILELLI UR IGI TɣARAST".

"TɣARAST IGI ILELLI D ILELLI IGI TɣARAST, IGI SABAB ILELLI TɣAWSIWIN TILLANIN".

"ILELLI D ILLI TTMEZWARIN GARA-SEN D UR TTSMMAGARN". ILELLI D ILLI ḤWAḌN D TTZDAƔN.

"MI ẒRAN IḌṛANEN AGLAYN YAT TɣAWSA, ẒRAN ALA TɣARAST-NSN ILLANIN, UR ẒRAN TɣARAST-NSN ILELLIN".

"Akk IḌṛANEN YEZZN yusan ad ẓran s yat tikklt tɣarast illanin d ILELLIN n kra n tɣawsa.

"ILELLI iga d awal yuran i ugʷdira UR ILIN d ur ufganen n iḍranen, d yat tɣawsa yuran n uɣaras n uzdar d tilelli".

Imɣaren d imɣaren n ibrdan, tismɣurin d tɣiwaniwin ḥwaǧn ad ɣren ɣef tussna-nneɣ tettbeddiln d sinen ad sɣren inelmaden-nnesn uɣaras yettwiyen i tɛic n AYEN NEȚȚA N TIDET.

Ilaɣ ad nggaɣ i TƐIƐT N AYEN NEȚȚA N TIDET mi ttfakka tira.

Tirrugza n ILELLI tettarra ad nɛic TARAGT TAZAYKUT n TIDET TULLIST.

Taẓẓurt tllant deg tidet ILELLI, bla tɣawsa d bla unɣi, ILELLI SEG AGʷDIRA, iga d TIDET IṢḤAN, TAɣYULT TANSMANT.

TARRA TɣURI-K, i ugʷdira-k illa ILELLI ur yusan ad ittyara am ILELLI n UR ILLI maca am TARRA ɣURI YATNIN bla tɣawsiwin, teggen, tanasmant d tazdaykut iga d TƐIC, BUDDHA UMATAN YEZZN.

TƐIC-K ILLI d TARRA YEZZN d tazdaykut ur ttafɣen. TAZDAYT-NSN n DHARMA-KAYA; UɣARAS N TƐIC TULLIST.

TƐIC-K TEGGEN, ILLI d ur ttafɣen n tfekka tameqqrant n tfulki, ur tesɛi tarwa neɣ tmut d taragt tamaynut AMITARA BUDDHA.

Taẓẓurt-a terwḥ. Ad tɛeqqleḍ ILELLI n TARRA TɣURI-K am UɣARAS n BUDDHA d ad tḥsbeḍ am TƐIC-K, iga ad tkemmel deg ṚṚUḤ AMADAN n BUDDHA.

Sɣur-d tarra taɣuri-k bla tɣawsiwin mi telliḍ deg UZARAG, tttu belli telliḍ deg UZARAG, ur tḥsabt belli telliḍ tettarɣagt acku mi tḥsabt belli telliḍ tettarɣagt, tazzurt-a terwḥ i tugʷddi usarag. Ummaqqel-nnek ila ad yili ILELLI ad tɛic AYEN NEȚȚA N TIDET.
