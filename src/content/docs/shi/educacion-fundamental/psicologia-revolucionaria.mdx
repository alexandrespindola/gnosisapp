Ila ssutad aselmaden d selmadat n iɣerbazen, tisnawiyin d tasdawiyin ad gren deg ussakun n tsikoloji tarewrawt i yesseɣran umussu agnusan agraɣlan.

Tasikoloji n rewruj deg tilawt temgarad mliḥ ɣef wayen akk i yellan zik s isem-a.

Ur yelli kra n ccek, nzemmer ad ini s wul yeṣfan belli deg lqernun i ɛeddan zdat-nneɣ, seg yiḍ ameqqran n yal leɛṣur, ur teɣli ara tsikoloji ɣer tama n wadda am akken tura deg tallit-a n "Amrebbuz war ssebba" d imsebnayen n ROCK.

Tasikoloji yeǧǧan deffir d tamatart n leɛṣer-a attan tderɣel-itt tbaḍit, tzegzel-itt lmeɛna-s, d yal tamunt tusridt ɣer aẓar-is n tidet.

Deg tallit-a n texnuft tazduɣrant d ubuhri amcum n wallaɣ, ur yettili ara kan wawal i yessemmaɣen s tikelt i tira n tsikoloji, maca daɣen ur ssinen ara tidet n tɣawsiwin tigjdanin n tsikoloji.

Wid yeqqaren s uɣalat belli tsikoloji d tussna tatrart n tallit-a, tɛuccent fell-asen, acku tsikoloji d tussna taqburt i d-yekkan seg iɣerbazen iqburen n tirrugza tinɛurfanin.

Amek i yeẓra usnob, amejḍiḍ amaynu, ameɣlal, ur yezmir ad yessifel ayen yettwassnen s tsikoloji acku ɣir tallit-a tatrart, iban belli tsikoloji ur telli ara s yisem-is, acku ɣer kra n ssebbat, tettwaḥseb i tɣellabin n tɣellabt tazrimant neɣ tadinit, ɣef wayi ad tenjer ad teslewyeɣ iman-is s iberdan.

Seg tallit iɛeddan, deg imesriyen yemgaraden n umezgun n tudert, tsikoloji tetturar yal ass, tettseggi iman-is s tmusni deg lbise-nnes n telfalsaft.

Deg rrif n Ganges, deg Lhend Iqedsen n VEDAS, seg yiḍ i ttagwaden n lqernun, llan ṣṣurat n YOGA i yellan deg uẓar, d tsikoloji tajjerrabt i ɛlayan.

Sebɛa yogas tettwassen d tikkwatin, ibeddilen, neɣ imagrawen ifelsafiyen.

Deg umaḍal aɛrab, ussɣan iqedsen n SUFIS, deg tama tafizika, deg tama tadinit, d imezwura n tɣawsa tsikulujit.

Deg Turuft taqburt yesseɣwan ɣer ɛḍel n iɣsan s umenɣi, leḥqiqa-s, tadinit-is, tazrimt-is, siyasit-is atg. Ar taggara n lqern iɛeddan, tsikoloji tesleweɣ iman-is s tlebset n telfalsaft i wakken ur tettwabdar.

Telfalsaft ɣas tella tebḍa-tebḍi am Lógica, asnerni n tmusni, Ética, Estética, atg., Tella berra n yal ccek iman-is, d tukksa n tira, tikkit n ummystic n ser, taɣiwant tusnakt n teɛɛurt teɣra.

Aɣalat n waṭas n iɣerbazen ifelsafiyen d asnerni n tsikoloji am ayen yeẓẓuṛen ɣef telfalsaft, am wayen yeqqnen ɣer tɣawsiwin n wadda am tadusi tanemgalt n umdan.

Usnerni amesgadi n ddiyen yettarra-aɣ ɣer tagrayt tanemgalt belli tussna n tsikoloji tella yal ass tettwassen s tarrayt tagmant mliḥ ɣer yal aẓaṛ adinan. Yal usnerni amesgadi n ddiyen yettarra-aɣ ad neẓer deg tira iqedsen kter n ddiyen yettwassenen seg tmura yemgaraden d tallitin yemgaraden, llan ayen yifen tussna tsikulujit.

Amestasriqen n uẓar deg tlemmast n GNOSTICISMO yettarra-aɣ ad naf asemlili yesɛan azal seg yimezwura n Gnosticos i d-yekkan zdat n tmasiḥt d ayen yettwassnen s isem n PHILOKALIA, tettusemres tura deg iglesias n usserɣin, kter i useɣri n imunak.

Berra n yal ccek d war ccek aɣef ad neɣlu deg uɣuṛru, nzemmer ad ini belli PHILOKALIA d tsikoloji tajribit iṣfan.

Deg iɣerbazen iqburen n tirrugza n Grecia, Egipto, Roma, India, Persia, México, Perú, Asiría, Caldea, atg. Atg. atg., Tsikoloji tella tettqeṛreb ɣer telfalsaft, ɣer tẓuri tagejdant n tidet, ɣer tussna d ddin.

Deg tallit iɛeddan tsikoloji tettṣegger iman-is s tmusni gar ṣṣurat ḥlunt n tmeɣnasin tiqedsen, neɣ gar uɣebbar n tira yeɣḍen, neɣ tẓuriwin yelhan, neɣ deg tmedyazt, neɣ deg tagedyazt d yal musiqi yelhan n iẓuran.

Zdat ad tefrez tussna, telfalsaft, tẓuri d ddin i wakken ad tbeddel iman-nsen, tsikoloji teɣleb deg yal iɣerbazen n TIRRUGZA TIQBURIN.

Mi ttmudden iɣerbazen imseɣriyen ɣef ssebba n KALIYUGA, neɣ EDAD NEGRA deg nettat tura, tsikoloji tɛuccent gar tmettelt n yal iɣerbazen ESOTÉRICOS d SEUDO-ESOTÉRICOS n UMAḌAL AṬRAR d kter gar ESOTERISMO GNÓSTICO.

Aḥlil ameqqran d amastasri ameqqran, yettarra-aɣ ad neḥsu s timmert tajawant belli imezwura n imagrawen d ttawil tsikulujin i yellan zik, i yellan tura, nzemmer ad ten-nebḍu ɣer sin imezwura.

Amzwari.- Ttawil am akken waṭas seg iɣerbazen ssnent-ten. Tsikoloji tatrart teqnen ɣer umzwari-ya.

A sin.- Ttawil i yesɣaren amdan seg tama n REWRUJ N TEƐƐURT.

Imezwura-ya deg tidet d ttawil azwaran, kter n yimqedmin, nettat kan tettarra-aɣ ad neḥsu ɣer iẓuran iddurren n tsikoloji d lmeɛna-s teɣran.

Mi nettmuddu ad neḥsu deg wudem iduran deg yal TIWEN N WALLAƔ, ayen yella yesɛan azal usnerni n umdan seg tama tamaynut n REWRUJ N TEƐƐURT, Ad neɛqel belli tsikoloji d asnerni n lqaydat, lɛeqqan d tɣawsiwin iqerben ɣer tbeddilt tameqqrant d taggara n INDIVIDUO.

Ur yella kra n ccek belli selmaden d selmadat n iɣerbazen, tisnawiyin d tasdawiyin, ad fehmen deg wudem iduran tallit tajaddit deg nettat nedder d lḥala iqquren n tɣellabt tsikulujit deg nettat tella tgelda tamaynut.

Yessefk ad nessiɣer "NUEVA OLA" s ubrid n REWRUJ N TEƐƐURT, wagi yezmer kan s TSÍKOLOGÍA REVOLUCIONARIA n USEƔRI AWEZWRI.
