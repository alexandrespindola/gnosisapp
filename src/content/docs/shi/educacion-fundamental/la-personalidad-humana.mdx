Yiwen ur d ilul, idder setta u semmus n iseggʷasen yemmut. Maca mani illa uqbel useggʷas n 1900, mani rad yili deg useggʷas n 1965? tussna taɣerfant ur tssin ula kra deg waya. Atiggi ad tga tferkit tumlilt n akk isteqsitn ɣef tudert d lmut.

S useɣẓan nɣir ad nini: "Yiwen immut acku lweqt nnes ikfa, ur illi kra n azekka i tɣeltma n win yemmuten".

Kull ass iga-t afalkay n lweqt, kull ayur iga-t afalkay nniḍen n lweqt, kull aseggʷas iga-t daɣ afalkay nniḍen n lweqt d akk ifalkayen-aggi s umata ad skaren afalkay ameqran n tudert.

Lweqt d awraɣ d tudert n tɣeltma n yiwen d tawlaɣt yeqqnen.

Tudert n tɣeltma n yiwen tettnerni deg lweqt nnes, tettlal deg lweqt nnes, tettemettat deg lweqt nnes, ur tezmir ad tili berra n lweqt nnes.

Mayad n lweqt d uguren ittewsensen s waṭas n imusnawen. Ur telli uɣil ɣef belli lweqt iga-t tiddi tis kkuẓ.

Tasengama n EUCLIDES tettusemres anagar ɣer umaḍal ittusnidren maca umaḍal illa gis sebɛa tdidin tiddi tis kkuẓ d lweqt.

Tixmemt n umdan tettḥssu i d timittarut am asarag n lweqt deg uḥeddad usrid, ur yelli kra izedɣen ɣef tagi acku timittarut d tiddi tis semmus.

Kull tazlagt n tella tettḍer deg lweqt tettɛawad i timittaruten.

Lmut d tudert d sin igguggen ittemsasanen. Tudert tekfa i yiwen ittmettaten maca tettbedaɛ tayeḍ. Lweqt ikfa, itbeddaɛ wa nniḍen, lmut tella deg tɣalt yeqqnen ɣer tuɣalin timittaruten.

Mayad isseqsi belli ilaq a nuɣal, a neddu ɣer umaḍal-aggi deffir n lmut i nɛawed amanar n tella, mqqar, tɣeltma n umdan tettemmettat s lmut, anwa, ma d acu ad yuɣalen?

Ilaq a nḍeffer deg yiwet n tikkelt abedda belli Nekk ad ikemmlen deffir n lmut, belli Nekk ad yuɣalen, belli Nekk ad yeddun ɣer wagu n imettawen.

Ilaq ad ur qqaren wid ittɣaranen ɣer tmurt n Taggayt n tuɣalin d tanqist n Tɛawada ittusslmaden s Teosófica Moderna.

Tanqist ttussawalen n Tɛawada tbeddaɛ deg udwas n KRISHNA iga-t din Védico n INDOSTAN, lḥal ur yessiweḍ ad ittubeddel d ittuɣar deg imbeddalen.

Deg udwas awraɣ aẓaṛan n Krishna, anagar iḍerman, imallasen, wid ɣuṛ-s illa yagi tɣeltma tusdidt, ad tɛawaden ad lalun.

NEKK Amegraw ad yuɣal, ad yeddu maca agi ur d tɛawada n tlallit. Igelliden, imɣuriyen ad yuɣalen maca agi ur d tɛawada n tlallit.

Tameɣra n tuɣalin n tɣawsiwin d tɣawsa, tameɣra n tɛawada timittarut ur telli taqdimt waṭas d neqqar ad nttaf deg tmusni n Pitágoras d tasengama taqdimt n INDOSTAN.

Tuɣalin timittarut n assen d iḍan n BRAHAMA, tɛawada ur tettaḍaɛ n KALPAS, tella zɛement deg talɣa yeqqnen ɣer tmusni n Pitágoras d tgemmi n tɛawada timittarut nɣ taggayt n tuɣalin timittarut.

Gautama el BUDHA isselmed s tmusni tameqqrant taneqist n tuɣalin timittarut d tazlawt n tudert ittemsalanen maca taneqist nnes tettuɣar waṭas s wid ittettbinen.

Kull tuɣalin terna asɣur afares n tɣeltma tumlilt, tggi tettaɣ deg useggʷas amezwaru n tmeẓi.

Tamezruyt n twacult, tudert n ubrid d tinmal, tteggen i tɣeltma tumlilt aɣawas aẓaṛan izdin.

Amedya n imɣaren iga-t usdid i tɣeltma tameẓyant.

Umeẓyan ittḥerrez s umedya wala s tɣawsa. Talɣa ur yusin amedya usnɛlil tennarugatin n imɣaren, tteggen i tɣeltma n umeẓyan aɣawas yusin ameskir d win ittleɛben n wakud ittuddren.

Deg wussan-aggi imaynuten zzna yuɣal isehlen wala batata d buṣil d lḥal ur yessiweḍ agi itteggen imanaren deg iduwar.

Ganin umeẓyanen ɣef wussan-aggi illa ɣuṛ-sen ad sedren yettwaččaren s wugur d wurar, igedren d tiɣanimin n ubaṭran nɣ tinmetran. Yussen belli deg talɣa-nni tɣeltma n umeẓyan tettnerni deg izmez n wugur, wurar d tkerḍa.

Illa usawal ur yusin d yeqqar: "Umeẓyan n wa nniḍen ittettɛara deg akk imukan". Deg tagi daɣ illa imezwar, maca agi neqqar ad nesɣer s uḍad n ufus isehlen.

Imnattčen gar baba d yemma ɣef talɣa n tayri, imeṭṭawen d tirɣas n yemma ittwaguren nɣ urgaz ittugaṛen, ittujerden d win ittumaguren, tteggen deg tɣeltma n umeẓyan aɣawas ur ittaḍaɛn n uɣwas adran d tɣelbi ur yettettu deg tudert.

Deg tiɣwanin yusin tigellidin ttuguren ɣer tmɣarin mara ddan ɣer uguns n tukkin nɣ sedun udem. Aɣawas n tigellidin ittḥssu is ittwajreh i timetta.

Umeẓyan ittṭṭafar akk amanaren-aggi n tennarugatin ittḥssu s tukkin deg tɣalt nɣ iga iman n yemma imexḍaɛen d tusdin nɣ iman n texdamt tarusit ttuɣuren d tgi tamat n tukkin ittusḍerrɛen d tɛuqqast ittetten deg tɣeltma n umeẓyan.

Seg mi ittwafares tilivizyun ittufg lqema n twacult. Deg wussan nniḍen argaz iḍfuḍ s ubrid ittemuqal s tameṭṭut nnes s sers n wul. Ass-aggi tameṭṭut ur tettiffiɣ ad temuqal argaz nnes ɣer tabburt acku tettaɣ deg tilivizyun.

Deg iduwar imaynuten baba, yemma, warraw, tibniqin zund ibriden ittuffren deg udem n tazdayt n tilivizyun.

As-aggi argaz ur yezmir ad yennɣ s tameṭṭut kra n uguren n wass, axeddim zund tagi acku ttemɛassa ɣer tilivizyun.

Warraw ittusɣanen deg talɣa-aggi n uduwar ultrama yusin teɣɣaren ɣef yettwassen-issen, ittwatfen n uḍar, ifullusen n tayyugi ad tɛawaden ad uddren akk amanaren ittaḍaɛen n tɛuqqast zund ma ttwattin deg tazdayt n tilivizyun.

Teɣleb acku afares-aggi ushel n tilivizyun ittuššermes s tuɣmiwin tinerratin. Ma d imdanen ittuššermesen deg afares-aggi deg talɣa tusdin nɣ ad isslmaden tussniwin tumlilin nɣ ad isslmaden tẓuri yusin n tewmat n NATURA, nɣ ad tteggen aslmaden yusin i imdanen, d afares-aggi rad iga timɣurin i imdanen, neqqar ad nessemres s tmusni ad neggezn tɣeltma tumlilt.

Yussen belli ur yessusum ad iḍfuḍ tɣeltma tameẓyant s uẓawan ur yusin, ur ittwajadden. Usgil ad iḍfuḍ tɣeltma n warraw, s timɛeqlin n imukrad d imrigen, imanaren n ddwal d tɛerqub, amanaren n uzznu, tullsa, zund tagi.

Taggayt n wudem isehlen neqqar ad nesɣer deg imukrad bla ssebba, imettaten ifeggagen.

Teɣleb acku tmedduren ttuquren ɣer warraw nsen, tteggen isegren, tteggen asawalen ur yusin d icerquben. Taggayt n tudda d ugur, tisselt n tayri, zund tagi.

Deg tɛemmart nettaf belli warraw ittusɣanen gar isegren, tigedren d temyurin ad yuɣalen ɣer imdanen usgil ittwaččaren s tukkin d tuḍfa n kull tarusi n tisselt d tušḍaɛ.

Yessenqes ad nessuḍar tisselt n ussuɣer azzayan deg iduwar.

Yessenqes ad neẓer belli tɛewwant d tkerḍa ilaq ad ittuzemzen deg iseggilen n tsedda n tmazɣa.

BABA ittḥekkem tkerḍa, YEMMA tettḥekkem tɛewwant. Baba ittḥekkem tmusni. YEMMA tetteddukan tayri.

Tmusni d tayri, tkerḍa d tɛewwant ttuzezment deg iseggilen n tsedda tazdayant.

Baba d yemma n twacult ila qad zemzemen i twacult.

Yessenqes, yussen ad kull baba d yemma n twacult ad suḍren tisselt ad neggzen deg tixmemt n umeẓyan iğuren timittaruten n yiman.

Teɣleb acku warraw imaynuten ur ɣuṛ-sen illa uɣawas n tušḍaɛ, agi d talɣat n imukrad n imrigen, tazdayt n tilivizyun, asinar ttuɣren tixmemt n warraw.

Tafaggurt tusdidt n UZZI usdid, deg talɣa tusdidt tettggez tifawt gar tgemmi d tella.

Deg krad n iseggʷasen imezwura n tudert, iḥmel deg umeẓyan taẓult n tella, umeẓyan iga-t waɛun, usgil, izdi deg akk igwran n tesnaggurt.

Mara iddukan tella ad iḥekkem deg tɣeltma taɛunt n umeẓyan akk taẓult-nni n tella tettemettefat d ad tteggen imezwar uṣḍaɛen n tesnaggurt ad yeddren deg kul yiwen.

Zund iqqa a neggez gar tella d tgemmi illa ɣuṛ-nɣ a neggez gar tɣeltma d tella.

Yiwen ilul s tella maca ur ilul s tɣeltma, taneggura tessenqes a ttunefres.

Tɣeltma d tella ilaq ad ittnernin deg talɣa ittwajadden d yusin.

Deg tɛemmart nettaf belli mara tnerni tɣeltma s tella, taggayt d Amgerḍ.

Taẓrut d tasengama n waṭas n iseggʷasen ttaɣen ad nesseḍar mara tnerni tella ur ttemmuqulen kra n tnermit n ussuɣer usdid n tɣeltma taggayt d Ameskin bla tafeggurt, bla tɣeltma d usgil n wul maca ur yettwassen, ur yezmir.

Asnerni usdid n tɣeltma d tella itteggen argazen d tmɣarin yusin.

Deg tella illa ɣuṛ-nɣ akk mayan illan ɣuṛ-nɣ, deg tɣeltma akk mayan ittwawer.

Deg tella illa ɣuṛ-nɣ igwran ittulalen, deg tɣeltma illa ɣuṛ-nɣ amedya n imɣaren nɣ, mayan nɣra deg uduwar, deg tinmal, deg ubrid.

Yessenqes ad iḍfuḍ warraw ikessen i tella d ikessen i tɣeltma.

Tella tettkessen s taɛewwant, tayri bla iğur tayri, uẓawan, iǧeǧǧigen, taẓult, tzemzmet zund tagi.

Tɣeltma ilaq ad tettkessen s umedya usgil n imɣaren nɣ, s aslmad usgil n tinmal, zund tagi.

Yussen belli warraw iḍfun ɣer temzwarutin deg lɛmer n sebɛa n iseggʷasen uqbel iɛedda s kinder.

Warraw ad isslmaden tigemmin imezwura ttleɛben zund talɣa nneɣra tettgi i ɣuṛ-sen, tettaɣ, taɛeḍḍawt d yusin.

Aslmad aẓaṛan isslmad belli seg kinder illa ilaq ad ttemmuqal deg talɣa tusdidt kull yiwen deg krad n tgurawin n tɣeltma tumlilt, ittussanen am tixmemt, timetta d tɛemmart, zund talɣa-nni tɣeltma n umeẓyan tettnerni deg talɣa ittwajadden d yusin.

Tasteqsit n ufrares n tɣeltma n umeẓyan d usnerni nnes, d tagrayt yusin i BABA n TWACULT d IMASTREWIN n TINMAL.

Talla n tɣeltma tumlilt tetteddukan anagar ɣef talɣa n uggar n tesnaggurt ittwanefren d tettwakessen.

Izzuɣer n tɣeltma, tella, tgemmi illa gar imɛerḍen n tasengama waṭas n teqquma.

Kra qqaren belli tɣeltma tettɛeddaɛ s tella d kra qqaren belli tgemmi tettɛeddaɛ s tella.

Ganin tinmal ur yusin tettḍfirin, tella ɣuṛ-sent amedya n asnerni n tudert ur ittiṭṭafen.

Ilaq a nessuḍar ur d tɣeltma ad nessefḍer.

Yussen ad neẓer illa ɣuṛ-nɣ ad nemsegzer tgemmi, Nekk Nekk AM I MAN, a nzerz iḍes deg uɣbar tazdayan.

Tɣeltma d imarayn umezdaɣ n tɛemmart d umezdaɣ illa ɣuṛ-nɣ a nfarssen.

Deg umaḍal illa CALIGULAS, ATILAS, HITLERES, zund akk talɣa n tɣeltma mara telli taɛuqast, neqqar a ttubeddel mara tgemmi ad tessegzer deg tɛemmart.

Agii n UMSUGZEZ n TGEMMI Iɛesbeḍ waṭas deg udwas d isekren waṭas n imɛerḍen n tesnaggurt. Agi ittwaqnen s belli tgemmi d DIVINA, ḥessben belli tgemmi d SER, MONADA DIVINA zund tagi.

Yussen, yussen, yussenqes ad nesseḍar belli tgemmi kra tella deg DIVINA.

TGEMMI d SATÁN n BIBLIA, amesrar n ttiɣiten, imuras, tuɣmiwin, ikeraḍen, urar, izdanen, uzznu, tɣawsa n twacult, tmura, agdud, zund tagi.

Waṭas ttinin deg talɣa usgil belli illa ɣuṛ-nɣ NEKK yusin d NEKK adran d UN NEKK izdaɣ.

YUSIN d IZDAɣ d sin igzumen n yiwet n tɣawsa. NEKK YUSIN, NEKK IZDAɣ d sin igzumen n yiwet n TGEMMI.

SER DIVINAL, MONADA, INTIMO, kra illa ɣer talɣa n TGEMMI. SER iga-t SER d mayad ad igga.

Tɣeltma deg iman-nnes d imarayn d mayad. S tɣeltma ittezmir ad innerni tgemmi nɣ SER kul yiwen degɣ.

YUSSTEN ad nessefḍer NEKK, TGEMMI d ad tetterzi anagar deg Tɣeltma nɣ TAGGAYT TSINAGURT n SER nɣ AZDI.

Yussen belli IMASTREWIN ad suḍren tisselt n tnermit ad ssaɣn tgurawin n tɣeltma tumlilt.

Azemzems usgil gar tɣeltma d tella, asnerni usdid n TIXMEMT, TIMETTA d Timetta, ETHIQUE TUSDIDT tbeddaɛ asɣar n aslmad aẓaṛan.
