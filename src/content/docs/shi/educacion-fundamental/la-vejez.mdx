Arbaɛin isggʷasn izwaran n tudrt aɣ d-ittddn adlis, kraḍin d id sḍis lli d-ittddn aɣ t-ittfssirn.

Ɣ isggʷasn n ɛecrin, argaz iga d uṭṭaws; ɣ tlataḍ, iga d izm; ɣ arbaɛaḍ, iga d alɣem; ɣ smmusḍ, iga d ifis; ɣ sḍisḍ, iga d aydi; ɣ saɛḍ, iga d abuki, u ɣ tam aṭṭam, anɣ tmazɣat d tɣrat kan d tẓallit.

Tizi ittẓahir kullu mayllan: iga d ajṭṭaf amaynu ixccn ittḥḍaṛ s yiman-ns ula ma ur t-stiniḍ hta kra.

Ur illi kra ikkan s ufus n AYWAN AQQRAY, sɣalḍn s yism n urgaz, ar tizi rad tizi ur t-isɣiwd.

"FUGIT IRRÉPARABILE TEMPUS", tizi lli izzlɣn ur ittmḍfṛ.

Tizi ittffiɣ kullu mad illa ɣ assa yuffr ittɣummuy, ittɣummuy d ittɣummuy kullu mad ɣ assa ikccum s uẓru.

Tiɣzi iga zund tayri, ur ittaẓẓu ma nn t-isburɣs s iḍrisn n tarwa.

Tiɣzi tskat tɣrdit n imɣarn tsbddl-tn, macc kra iga tt ibddl kra iga tt tḍr.

Mqqar immut, imɣarn lli ɣ tntfal tudrt nttanf tiɣzi ur tentzdiɣ.

Imdan kullu gan tamatart ad ddrn tudrt iɣzzifn ad kkin d imɣarn, dɣya tiɣzi tentzdiɣ.

Tiɣzi tbddu ɣ smmus d id sḍis isggʷasn d ttfssir ɣ tizi n sbaɛ aɣ d-ittddn ar tɣzzift d tmut.

Tawda akkʷ imqqurn n imɣarn tla, ur ɣ manik nn gan imɣarn, wanti ɣ tntfalt ur bɣin ad asn-sllḥn d tmahbalt ad ttamnn tarwa zund amgdl, tiɣzi tra ad ttffɣn.

Aɣaras akkʷ izdin ɣ tiɣzi, nttat aɣ tẓraḍ ur tɛḍal.

ANAZUR APSIKULUJIN, MI YIMAN, IGU, ur ittẓil s isggʷasn d tawalt; ittɛqql, ittɛqql ktr, ittaḥya, dɣya ittini mddn: "GENIO Y FIGURA HASTA LA SEPULTURA".

ANAZUR APSIKULUJIN n imɣarn lli yḍfṛn zɛama ad sulsn izwaran tṭṭfn tmɣarin ɣ umal, ssbḥan izwaran zɣ imal ɣ mani ɣ ur tḥsin ad tṭṭfn timɣarin ɣ tazwart.

Imɣarn ḥsan yaggi tiɣzi iga d agʷllid ɣas ixccn lli yttakʷfaln ɣ id tɣzi n tiram n tazɣart tazzulant dɣya tanf ad tṭṭfn tamɣart n yiman-ns ssbḥan izwaran.

ANAZUR yuffr ANZYAR, ANIZUR issufu kra n ANZYAR, d kullu ttruṭa s tgʷmmar iẓiln d tmazɣin iwulmn.

KRA n MI YIMAN issufu kra n MI YIMAN. ANAZUR issufu mad ur t-iɛjjbn.

Ittban s tmẓra d tawalt mqqar dɣya tiɣzi tnɣlb, iɛjjbn aɣ ntfrẓ aɣ nɣlbt.

Ull n AYWAN AQQRAY ur ittẓil s isggʷasn, anɣ ixccn, ɣ tazɣart nɣlla imḥsasn, imzran, iɣwwaln, ɣ tiɣzi nɣlli ad nn ẓẓarn.

Imɣarn ddrn ɣ tizi izrin, imɣarn gan d tazzat n yirmdan, imɣarn ur ḥsin yat mnzɛt nɣlla ddrn, imɣarn gan d tummt n tmɣarin.

Aɣaras wahdu ad taweḍ ɣ tummt ikmaln nttat aɣ tssḍfuḍ ANAZUR APSIKULUJIN. Mqqar ntḍfṛ ad nmmat zɣ mnzɛt ɣ mnzɛt, ntkka ɣ tummt taẓilit.

Tiɣzi tla anamk amazlan, n uḥbu d uzrf n wid ann ssḍfn ANAZUR.

Mqqar mmtnt tazzulin s trmat, tasusit, igmmaḍn, ittɣamaḍ ur zɣ mas, anɣ zɣ mraw mas.

Iɣʷi ad taffeḍ ɣ tudrt imɣarn imskrin lli ur ttḥmln ur ttḥmln kra n iḍrisn n ANAZUR, tawwjaḍ n imɣarn gan d imqqurn ittmḥallan d ddrn zɣ mnzɛt ɣ mnzɛt.

Argaz igan ɣ tsnsit. Amɣar ɣ tɣri, agʷllid n tayri, ibddl ɣ yat ayyur n tafawt aɣ ittnfarn s tɣri tazzarant n tmazɣatin d umḍan n isggʷasn.

Ɣ umaḍal llan d llan ɣ assa kra n IMAGDRAN IMARIRN lli ur ttḥmln hta kra n imɣarn n ANAZUR. IMAHART GNUSTIKI gan d irjlan d iggʷrn zund izɣar n luttus.

AMZWAR IRMMI AMARIR lli isḍufn ANAZUR S UẒMAY s trmat d tasusit nttat attfal ikmaln n TSNSIT TAKMALNT, N TAYRI TASSILANT D UẒMAY IẒILN.

AMZWAR AMARIR lli ur illi ANAZUR, iga zɣ yat tmazɣat tazzarant n ASN DIVINAL.

IMɣARN IẒILN, IMAHART GNUSTIKI uɣddmn umaḍal zɣ zikk, nttfrẓ ALBUDHA, MUISIS, IRMIS, RAMAKRISHNA, DANYAL, ASANTU LAMA, ATTAW, ATTAW, ATTAW.

Imɣarn n iɣrfan, tmagdal d tsddawin, timɣarin, ɣilli d baba-tmɣarin, ttnfln ɣ tumast ad ttẓallan d ad ttmḥallan ɣ izwaran.

Awal lli ur illi yism-ns, Awal d DIVINAL, Awal d AD REAL, ila kraḍ yannar: TASNSIT, TAYRI, AVERVU.

AWAL AD DIVINAL zund BABA nttat TASNSIT TAKUSMIKT, zund YUMMA nttat TAYRI TAZGURT, zund ɣuri nttat AVERVU.

Ɣ BABA n tgmmi illa udɣar n tsnsit. Ɣ YUMMA n tgmmi illa TAYRI, ɣurri ttnfḍn awal.

Baba amɣar ira ad yaẓ n ɣurri. Baba yaggur ur itttaḥya ad igʷmm, d yat aɣ ira nɣ tɣri d ad as-ttṛẓmn.

Yumma tayyurt taymmurt ur ittaḥya ad igʷmm dɣya ira aɣ iẓraɣ ɣurri n wazn d tayri nɣ tg tayri d ddin.

Win ur issinn ad iɛzza ɣ BABA-s, win ur issinn ad iɛzza ɣ YUMMA-s, ittgur ɣ aɣaras n ufus uẓẓlmḍ, ɣ aɣaras n uɣlḍ.

Ɣurri ur ttḥqqan ad tɣrrẓn ɣ IMMAWN-nsn, ulac win ikmaln ɣ umaḍal-a d win ur illi uɣṭas iṣuln ɣ yat tirikt, llan ɣ gʷmaḍ, kullu nɣlla ɣ taḍfiḍt yumaḍn.

Kra ttṛẓmn tayri n YAMA, kra ttsqqsn tayri n YAMA. Wid ann ɛmln zɣ tudrt ur dɣya kkin s aɣaras lli ittdduɣ ar Awal ur illa yism-ns.

ɣuri ur iccln lli ykrbn ɣ BABA-s d ittu ɣ YUMMA-s iɛzzan iga d aywan lli ikkrbn kullu mad illa DIVINAL.

TRWMIT N TUKRA ur tra INGRATITUD, aɣ ttfu baba, ad ttṛẓm YUMMA taɛzzant. TRWMIT N TUKRA nttat TASNSIT TAYRI d UZM IKMALN.

Ɣ BABA illa udɣar n tsnsit ɣ YUMMA taffeḍ aɣbalu ddrn n TAYRI win ur illa tazzart-ns taẓṛant ira ad t-taffḍ TIRMATIN IẒILN.
