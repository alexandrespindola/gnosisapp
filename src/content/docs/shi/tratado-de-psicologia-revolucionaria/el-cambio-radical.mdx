Ma ad idder urgaz ittwala iman-ns yiwen, waḥdu, iga yan, yubayn belli badal itran ur igi mayt yattu.

Akkʷ id tiddi nɣ awal ɣef asggʷs n tɣawsiwin ɣer wman ar ittmggar d usyafa n iman nns, ar aɣ inn ikkan asyafa n tugtt n iwtta n uɣaras, n imẓẓiyn nɣd imugriyn ur bahra nḍar ar asn-nssḥbu, nssḥdd uɣaras nnsn seg ɣur-nɣ.

Ur yugir ḥtta mayt yattu ad nḥḍu iɣlatn ur nssin, ilza ad nẓr zikkʷ ɣlli nra ad nḥḍu seg tmẓẓa nɣ.

Anaw ad ur igi agʷnsan nɣd anafaw, kra n wi d-itmnad ar maniwal n tdrfi nɣd agʷns anafaw d ugʷnsan ad asn-yakʷi s yat, illa wan ittyaɣʷzn.

Akkʷ id tiddi d tawssna n usyafa n wman ar ittmggar d tmuya ɣef usyafa f imal n iman, iggʷad iqqan ad nsskd belli iga usyafa n kraygat yan nɣ.

S usɣʷrri d war aɣʷri, ar nttini d war ttruḥu: ur t-ikki yan urgaz ad yaggʷ yan usyafa ɣef uɣar-nɣ.

Ur igi mayt yattu badal ɣer tmẓẓa nɣ, war usyafa i tugtt n iwtta igʷnsan ɣlli nttddu.

Ad nqqbal tugtt n iɣlatn, ad nḥḍu azwl i usyafa d usntl nsn, yagʷi ad nssḍf nɣd ad nttḍfr seg iman, agʷns n tfawt.

Kraygat usyafa f usyafa n iman, war asyafa d war ttruḥu ad nsskd s trggi belli ur ngi "Yan" macc "Tugtt".

Ad nqqbal tugtt n YO d usskd nns s usyafa n trggi, iga snat tɣawsiwin yaḍni.

Imkin ad yaqqbal usntl n tugtt n iYO war ittsskd, wawal ar igi mayt yattu s usyafa n iman-ns.

Ad nttḍfr usyafa igʷnsan, ad nẓẓr ssnfln, iga uskn n walliɣ.

Ma idder urgaz s tiddas d ar ttgi akkʷ yan, ur itikki ad ibddl, d yubayn belli usyafa n usyafa iga usyafa n usyafa ɣer tiddi nɣ tagʷnsant.

Asnfl itran iga tiṭṭawin, tɣʷzan, tḍar s tayda ɣlli ur ittiwsɣʷr ɣef iman-ns.

Awttas amzwaru n usyafa itran yugga ɣlli urgaz ittwala iman-ns iga Yan.

Wlli yagʷan usntl n tugtt n iYO, ar asn-skidn belli ur skrn akkʷ usyafa ɣef iman-nsn.

Usyafa iẓẓn n iman war asyafa n tɣawsiwin, ar aɣ yakʷi ad nẓr s iman-nɣ asyafa n trggi ɣlli ur ngi "Yan" macc "Tugtt".

Deg unɣal n iwtta, tigmmiwin tiɣʷẓanin n usyafa, ar ttkkan s umyur n ad nɣʷl seg iman...

Ur igi mayt yattu tiddas ɣlli ttgi akkʷ yan urgaz, iga agʷns i usyafa n iman...

Ad yini yan: "Ssnsɣ ur igi Yan macc Tugtt, Gnosis aɣ-t-inn ikkan". Awal ad war yaggʷa akkʷ tamattay ɣef usntl, ad iga awal anafaw.

Asskd, Asyafa, Asyafa iga ɣlli ittrran, mayt yattu ad nɣʷr s tawssna akkw n badal itran.

Ad tini agat d usyafa agat. Ma yini yan: "Asyagh ur igi Yan macc Tugtt", ma iga usyafa nns d trggi, ar inn inn ikkan, ar inn inn ittn, ar inn inn ikkan ad isskd usntl n Tugtt n iYO.

Tawssna d Asyafa gan snat. Tazwarat seg tmẓẓa, tanɣurt seg ul.

Maniwal n tawssna n Usntl n Tugtt n iYO ur tettaɣʷzn; s tayda ɣlli nttddu, tawssna tddaɣ s usyafa, ackʷu aɣyul iflfal yaggʷa akkʷ agʷns n tawssna, ur ittmnad agʷns n Ser.

Ad tssnt Usntl n Tugtt n iYO d usyaf nns, iga ɣlli ilzan s badal itran d trggi.

Ma yaggʷa urgaz ad isafr ɣef iman-ns, seg unɣal n ur igi Yan macc Tugtt, yubayn illa yaɣ ad ikcm usyafa ɣef tagʷnsant nns.
