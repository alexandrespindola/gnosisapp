Tawiḍ "Amammu Aɣsan", yan ufgan zɣ wid ar ttinin ɣ tudrt ɣan akkʷ idusn, ddut d nttat s ammas n tnzruft n SAHARA, ḥettut t din iddr zɣ kra n tazuri, tẓrat zɣ tnafit aylli iga...

Tiɣawsiwin ad suln afssarnt; "Afgan Aɣsan" ḥtta iggura idus izddɣ d ufgan bahra, ɣ usntm nns ar ittafa dḍɛif bahra...

"Aɣersiw Aɣsan" iga aqjjiḍ ɣ yat tmrawt n tmrawin; ar ittafa ɣ iman nns ayen akkʷ yaggugn; ittnini is dars kra n ufɣu zɣ KINDERGARTEN, iḍrisn n tɣawsiwin timdinin, tiɣriwin timzwura, timgrayin, baccaluriyat, tasdawit, tiggʷt n baba nns, atg., atg., atg.

Aɣaras n usddɛ, ɣar dffir n kra n tira d timlsit yaɣutn, izwal d idrimn, nssn akkʷ belli kra n tɣrsut n uɛḍis ar aɣ ttɣʷʷḍn ul, d ɣ usntm nss ar nttini nɣiy d imikiyen...

Aqqra n Uḍris Agraɣlan aẓdar nnɣ ad nssn belli nkkni gan kuyan wid akkʷ ifuḍn n zikk d belli s usnflul nɣ nggʷra yugr...

Asggʷas ad n 20 s kullu mayllan, imɣurn, imnɣan, iznakan, tẓnṭaḍt n umaḍal, asddr n iɣsan, dduxan, tmunin, lqṣawa timɣurnin, tɛkkast tafarut, tɣawsiwin tulɣamin, atg., atg., atg., iga azru n ttmuɣlyat i wannaɣ nmuɣl; ur illi maniɣ nnnɣ aqqayu i wannaɣ nninin nnnɣ nssaɣ s yan uɣaras yugrn ɣ usnflul...

Aknay belli tallit tggut usnflul iga akṛḍḍ, aɣaras n usddɛ, "imjhal imjjur", ad suln ggutn ɣ "Dogma n Usnflul"...

ɣ kullu isffuḍn iẓḍmnin n "Tarix Taẓḍmant" ad nttafa kuyan lqṣawa iɣzzifn, tɣawsiwin tɣarasin, imnɣan, atg.

Maca imalassn nnɣ "Súper-civilizados" ad suln ggtun belli tɣawsa n Imnɣi ar tga tɣawsa taɣbalut, yan uḍriṣ ɣ tiggʷt iɣḍn aylli illan ggutn d "Tamagit Tamzwarut"

Zayd aylli illa ar adduɣn aylli iga amud n umudduɣn; kra n imudduɣn ad gan iznayen, kra ur gan iznayen, winna d iɛdiln d wiyaḍ ur d iɛdiln; kullu illan ɣ tudrt...

Ggan dɣya d illan uggʷar n Turarin n Yan. Maylli nkkni nnɣ nlla ɣ usntm, izdirn neɣ imikiyen, imjjur neɣ iḥassasn, igẓẓaln neɣ imḍaln, ad ittɛawan tiɣawsiwin yaḍnin ɣ tudrt...

Amḍal ad ittɛawan d kuyan iɣarasn, iɛḍiln d ḥtta tiɣawsiwin iɣzzifn n asddr ɣ illan...

Aznay ad ittɛawan s iznayen d ad illa ɣ iznayen d imunan, illa izri...

Maca ufgan iḥuqn, tɣtan d ugraw, dars ikufi d ad issnflul, ad issḍfur asffuḍ n uḍris nns...

Man illi amud nɣ izdiɣ? Neɣ yaggurn nnnɣ: Man illi Turart nnɣ n Yan?

Wannar Turart n Yan ur issnflul s usddɛ, ad issul asḍllil n kullu tigwnditn, iɣarasn, tiɣawsiwin iɣzzifn d tɣawsiwin n tɣṛḍawt...

Kullu tiɣawsiwin, kullu tiɣawsiwin, iɣʷssan ɣ azɣʷr nnɣ, ɣ ammas n umaḍal, gan wanakkku aylli nkkni nkkaz ɣ usntm.

S tɛdil ad nnnɣ belli "aylli illan ɣ uzɣʷr iga wanakkku n aylli illan ɣ usntm".

Wannar yan issnflul ɣ usntm d tannfult ad tga tɣawsa tafarut, azɣʷr, tiɣawsiwin, tudrt, snflul kuyan.

Lliɣ sulɣ tẓra ɣ tallit ad, (Asggʷas 1974), yan ugraw n ufgan iṛzzm aɣaras yaḍnin. ɣ maniɣ ɣ Miskik ufgan ad ittɣʷssa s tɣawsa taɣribt n "PARACAIDISTAS".

Aggura tɣiyad ur aqqaya atgga tɣawsa taɣbalut, maca aylli iɣzzifn ur illi din, maca ɣ Turart n Yan...

Ad igan ammas wannaɣ ad issɣssi ufgan, aylli yaɣutn "cochera", "cloaca" imnɣi...
