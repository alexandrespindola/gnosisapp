Blaḥeqq, akken yella tmurt n berra anda nella nettwili, akken daɣen yella deg ulawen-nneɣ tmurt tasemrant.

Medden ur țțuɣalen ara ɣef temdint neɣ taddart anda zedɣen, mačči d lɛib ma ur ssinen ara amkan asemran anda țțilin.

Di kra n lweqt, yal yiwen yessen anwa tzeqqa neɣ lǧiran anda yella, maca deg umkan asemran ur yettili ara akka, aṭas n medden ur ẓran ara ɣef wayen yellan deg tmurt-nsen tasemrant anda kecmen.

Akken yella deg ddunit tilisa anda llan medden yelhan d wid yeqqaren, akken daɣen yella deg tmezdaɣt tasemrant n yal yiwen seg-neɣ; ulac ccek belli llan tilisa yecban ddunit.

Akken yella deg ddunit tilisa neɣ tzeqqa anda llan iberdan yesxetren, yeččuṛen d imakaten, akken daɣen yella deg tmezdaɣt tasemrant n ulawen-nneɣ.

Kullu yettɛeggiḍ ɣef ṛṛezq n medden anda nettwali; ma llan ɣur-nneɣ imdukkal imsekranen ad nruḥ ɣer lmḥerra, ma llan d imdellaɛen, bla ccek ad nruḥ ɣer yiɣerbazen n tlawin.

Deg tmurt-nneɣ tasemrant yal yiwen yesɛa imdukkal-is, "Nekkiniyen-is", ad t-awin anda ilaq ad t-awin ɣef leḥsab n leṣnaf-is isemranen.

Yiwet n teqcict yelhan, tameṭṭut yelhan, tetteṭṭef leḥya, tzedɣ deg uxxam yelhan deg ddunit, ɣef leḥsab n "Nekkiniyen-is" yeččuṛen d ṭṭmeɛ, tzemmar ad tili deg yiɣerbazen n tlawin deg tmurt-is tasemrant.

Argaz yeṣfan, yesɛan leḥya, d bab n tmurt yelhan, yezmer ad yili deg tmezdaɣt-is tasemrant deg yiɣzer n imakaten, ɣef leḥsab n imdukkal-is n diri, "Nekkiniyen" n tukerḍa, yettwaznen deg ul.

Yiwet n terwiḥt yexedmen lḥers d lɣalat, ahat d arebban akka yettɛic deg uxxam-is, deg kra n lemṛabt, yezmer ad yili deg tmezdaɣt tasemrant n yemseɣḍen, imseḍwalen, iɛettaren, imsekranen, ɣef leḥsab n "Nekkiniyen" n daxel neɣ n ul, yettwaznen deg wanda yesxetren deg ul.

Iruḥ wawal-nneɣ belli aṭas n lxir yellan deg yemcumen, aṭas n diri yellan deg wid yelhan.

Aṭas n wid yettwasmen yella mazal-iten deg yiɣerbazen isemranen n tukerḍa neɣ deg ixamen n tlawin.

Ayen i d-neqqar s ṣṣut ɛlayen yezmer ad yexleq ugur i wid yettqadaɛen, wid yesɛan laman, wid ur nessin, wid yesɛan lefhama, maca ulac leɛyun i wid yesnen taɣuri.

Ɣas ma yebɛed, gar ttejmaɛ n tẓallit yeffeɣ daɣen ddin, gar umqelɛa n tmedyazt yeffeɣ daɣen ddin, ddaw tzeqqit n leknes n yettwagazen s tidet ddin yebḍa d lebsa n tẓallit d wawal ɛlayen.

Gar ulawen-nsen wid yettwagazen, ɣellin "Nekkiniyen" n tlawin, n tukerḍa, n tmettant, atg.

Imdukkal imcumen yettwafen gar ulawen-nsen.

Aṭas i sen-yuɣalen ɣef waya wid yettwasmen n umezruy; ad d-nekteɛ tidet n St. Anthony, ayen yesbubbuden ɣef wayen iɣef ilaq ad yennaɣ gma-tneɣ Francisco de Asís.

Maca, ulac ayen i d-nnan wid yettwasmen, aṭas n wid yettɛicin deg iɣzer ur d-nnin ara.

Yiwen yettɛeǧeb mi yeẓra kra n wid yettɛicin deg iɣzer n tlawin d tukerḍa.

Llant d wid yettwasmen, ma mazal ur d-ufin ara ayen yebɛaden deg ulawen-nsen, mi ara t-id-afen ad ssekren azekka ɣef tfekka-nsen, ad uẓumen, ahat ad t-id-ssekren, ad ẓallen ɣer yemma-tsen KUNDALINI ad tekkes seg ulawen-nsen imdukkal-nni n diri i ten-iɣwlen deg yiɣzer-nni.

Aṭas i d-nnant ddinin ɣef tudert d tameṭṭant d ayen yellan dinna.

Ad ur tezzin ara i medden ɣef ayen yellan dinna, ɣer tama n uẓekka.

Blaḥeqq, deffir tameṭṭant yal yiwen mazal yettɛic deg tmezdaɣt tasemrant n zik.

Amakaten ad yezdeɣ deg yiɣzer n imakaten; ameɣbun ad yezdeɣ deg ixamen-nni akka am uẓekka; amqerraɛ, aɛerray ad yezdeɣ deg iberdan yesxetren n tɣessa d uray, dinna daɣen anda yettfeǧiǧ ucḍad.

Tiṭṭ-is tecbeḥ aṭas, tusa-d seg yitriwen, teffeɣ deg "Nekkiniyen-nneɣ" n daxel.

S tugga tiṭṭ tzemmer ad terreɛ iberdan, ad tuɣal ɣer amkan n zik, ad tuɣal ɣer yitriwen, maca ilaq ad teffeɣ seg imdukkal-is n diri.

Mi ufan Francisco de Asís d Antonio de Padua, wid yesnen iselmaden yellan deg المسيح, "Nekkiniyen" n daxel, uɣalen-d ɣer amkan n zik.

Uqbel kullu ilaq, d lemɛunya, d lemɛajla, ad yerru tmurt n tazmert i yellan deg temsal n tmettant.

Ma mazal tmurt n tazmert deg temsal-nneɣ ad nɛic deg yiɣzer isemranen yebɛaden, ɣas ma nella d bab n tmurt yelhan.

Yal yiwen yesɛa tmurt n tazmert i d-yettbegginen; ameɣbun yesɛa tmurt n tazmert n tɣessa dɣa yettɛic deg suq dɣa yettɛiḍ ayen yellan yid-s, yettessuɣu d yetteddmuɣ.

Argaz n tussna deg temsal-is yesɛa tmurt n tazmert n tussna dɣa yettɛiḍ sɣur-s ayen yellan n tussna, idlisen, iɣerbazen, atg.

Ameɣbun yesɛa deg-s tmurt n tazmert n taɣuri, dɣa yettbeddil ayen yellan n temsal.

Mi ara tɛeddi tmurt n tazmert ɣer ul, dɣa ad teffeɣ seg wanda yella argaz.
