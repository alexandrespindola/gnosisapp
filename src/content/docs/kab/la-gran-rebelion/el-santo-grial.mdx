Akal-n-ddheb iffeɣ-d deg yiḍ ameqqran n yal lqern. Ixeffafen n lqern alemmas deg lawan n Timenɣiyin n Yiri ttnadin amecṭuḥ Akal-n-ddheb deg wakal Imqeddes maca ur t-afen ara.

Mi yuɣal nnbi Brahim si ṭṭrad mgal igelliden n Suddum d Ɛamurra, qqaren yufa-d Malkizedek lǧenni n wakal. Ṣeḥḥ, win yellan d ameɣɣar yeɛic deg yiwen n leɛqel yellan deg umkan anda i țwabnan mbeɛd Yerusalem, tamdint eɛzizen n llanbiya.

Teqqar-d talɣiwt n lqernat u ayagi ssinen-t imeqqranen d medden, belli Brahim yeḥseb taɛaweddit Ɛinussit s tḥeqqit n uɣrum d ccṛab gar-asen deg tilisa n Malkizedek.

Ur yettḥebbib ad yini belli deg wakud-nni Brahim yefka i Malkizedek leɛecṛ d lewḥi, am akken i yura deg Kitab n lqanun.

Brahim yeṭṭef si ifassen n Malkizedek Akal-n-ddheb; aṭas n wakud mbeɛd deg wakud-a taɛeqqa tṛuḥ ɣer lǧameɛ n Yerusalem.

Ulac ccekk belli tagellidt n Saba tsexdem am umeẓẓi-gareɣ i lḥeq-a. Tban-d ɣer ugellid Sliman s Akal-n-ddheb u mbeɛd ma tesseɣwet-it i tjeṛṛubin igdudran tessekker-it i tejyaɛ.

Kabir ameɣɣar Ɛisa yesswa deg taɛeqqa-nni deg tmeɣra imqeddsen n umensi aneggaru am akken i yura deg Rebɛa n Linǧil.

Yusef n Ɛarimatea yečča tasekkurt s idammen i d-ffeɣen si leǧruḥ n win yetɛezzen deg Uzrar n Tuɣmas.

Mi d-kecmen iberruyen irumaniyen adeg n usinat-nni, ur ufin ara tejyaɛ-a ɛzizen.

Asinat arumani ur yesder-ara kan tejyaɛ-nni ɛzizen maca daɣen, yessenqed gar-as s wadda n wakal axenzir n Lungiẓu s wayes ijerḥ Centurion arumani lǧenb n Mass.

Yusef n Ɛarimatea yeḥbes deg lḥebs iderɣalen imi ur yebɣi ad yefk Akal-n-ddheb.

Mi yeffeɣ usinat-nni si lḥebs yella yettuɣal ɣer Ṛuma s Akal-n-ddheb.

Mi yewweḍ ɣer Ṛuma Yusef n Ɛarimatea yufa-d taɛeqqibt n Nirun mgal ime المسيحيين u yeṛwu ɣer tama n yiri n Agrakal.

Yiwen n yiḍ deg yirgi yettban-d iḥeffaf i yinna-as: "Taɛeqqa-ya tesɛa tazmert tameqqrant imi deg-s llan idammen n Umsefsi n Amaḍal." Yusef n Ɛarimatea yerra-d i lumuṛ n lḥeffaf yessenqed taɛeqqa-nni deg yiwen n lǧameɛ yellan deg Muntserat, Katalunya, Espania.

S wakud taɛeqqa-nni teɣbeṛ gar-as akk d lǧameɛ d tama n idurar.

Akal-n-ddheb d taɛeqqa n Ḥirmis, tasekkurt n Sliman, taɛeqqa ɛzizen n yal lǧameɛ n tmenna.

Deg Tsanduqt n Umnay ur yella ara Akal-n-ddheb deg ṣṣifa n tasekkurt neɣ ɣumur, deg-s illa ddehn n usafag.

Akal-n-ddheb yettwaḥsab deg ṣṣifa yersen yuni taḥḥeccact, deg taɛeqqa taqeddsent-a illa nnitra n tmettant-nni, Suma n lmuṭṭaɛa, tasekkurt tabarurant n Yillu imqeddsen.

Lmasiḥ azeggaɣ yesswa si Akal-n-ddheb deg tsawa tabarurant n takristuft, akka yura deg Linǧil n Mass.

Ur yella ara Akal-n-ddheb deg usɣar n lǧameɛ. Ɣer-s akken yettwaḥseb lmuseɛ ad yessiw ccṛab n tafat deg Tasekkurt Taqeddsent.

Ad yili d imerɣi ad yettwaḥseb lǧameɛ n tmenna gar-as ur illa ara taɛeqqa yettbarkanen n yal lqern.

Ayagi yettwassel-aɣ-d i Ginebra Tagellidt n Lǧina win yessiw ccṛab i Lanẓaruti deg tsekkuriyin n Sufra d Manti.

Yillu ur țmettaten ur ttɣaran ara s tasekkurt yellan deg Taɛeqqa Taqeddsent; win akken ikṛhen Taɛeqqa Yettbarkanen, ttḍemɛen mgal Uruḥ Imqeddes.

Ameddakel-Ugbar yettḥebbib ad yeɣḍ s nnitra n tmettant-nni yellan deg taɛeqqa illuhi n lǧameɛ.

Abeddel n tazmert texleq d azur mi nebɣa ad nessiw deg Tasekkurt Taqeddsent.

Lmasiḥ azeggaɣ yettbeddilen, yettqabalen, yettḥarabiyen, yettaɣen daɣen, yettṣebbiber i Yillu yettessawa deg tasekkurt n wurɣu.

Ɛellut-d akken iwata tasekkurt nwen u ḥadret ad terzuḍ aṭṭas n ẓẓerbet n ccṛab eɛzizen.

Ɛqalet belli awal nneɣ-a yellan d tazwert d ṬILIMA (lebɣi).

Si tɣerɣert n tasekkurt - ṣṣifa tameɣrut n umeɣzu n tmeṭṭut - ttinɣerɣen imesdaɣ i d-ttfeɣen deg udem yecceḍlen n Umeddakel-Ugbar.

Yillu ur yettwaḥsuben n yal galaksiya ttessuɣen daɣen si tasekkurt n tmettant-nni deg taɛeqqa n lebda.

Aseɣser n waggur yettkemmil leḥwayeǧ deg wakud; ilaq ad nessiw si ccṛab imqeddes n tafat deg tasekkurt taqeddsent n Lakimiya.

Tizrig n igelliden imqeddsen, taɛeqqa tageldit d wurɣu yecceḍlen anagar i Lmasiḥ Azeggaɣ.

Mass n Uẓenzar d Rɛud yettṭef deg ufus-is ayeffus Akal-n-ddheb u yettessu ccṛab n wurɣu ad yeɣḍ.

Win akken yerzun Taɛeqqa n Ḥirmis deg tmellit takimiyant, s tidet ttbeddilen ɣer iɣersiwen idrawawiyen n wadda-amaḍal.

Akk ayen daɣen nura-d yufa-d isekkiren yessawden deg Kitab-iw yettusemman "Maɛawedda Tagdudrant".
