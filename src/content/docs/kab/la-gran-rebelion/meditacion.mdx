Deg yellan d ameqqran deg tudert d beddel ameqqran, amcum akk-d umeggur; ayen yellan iden ur yesɛi ara azal.

Tameɣruyt (méditation) teqqen mliḥ ma yella nebɣa s tideţ beddel ameqqran-a.

Ur nebɣi ara tameɣruyt ur yesɛin ara lmeɛna, tawsit akk-d tilfi.

Ilaq ad nili d imeqqranen, ad neǧǧ ayen akk yellan d uffɣen i ttafen deg win yellan d aseɛdi akk-d win yeffren n biɛa.

Ilaq ad nili d imeqqranen, ilaq ad nebdel ma yella s tideţ ur nebɣi ara ad nɣleb deg lxedma n tseɛdit.

Win ur nessin ara ad yettmeyyiz, win yellan d aqcic, win ur nessin ara, ur yezmir ara ad yefsi Ego; ad yeqqim d isɣi d ilemẓi ger lebḥer n tudert.

Aɣalat yettwafgen deg wakal n tudert, ilaq ad yettwafehm mliḥ s tiknik n tameɣruyt.

Aɣbalu n uselmed i tameɣruyt yella deg lumur akk-d tewtilin n yal ass n tudert, ayagi d ayen ur yettnkeṛ.

Medden yal ass ttenqalaben ɣef lumur ur yeɛǧiben ara, ur ssinen ara azal n lumur-a.

Nekkni deg umkan ad ntenqaleb ɣef tewtilin ur yeɛǧiben ara, ilaq ad d-negmer seg-sent, s tameɣruyt, ayen yelhan i wemɣi n teṛwiḥt-nneɣ.

Tameɣruyt tameqrant ɣef tewtilin-a ur yeɛǧiben ara, teqqen-aɣ ad naru deg iman-nneɣ taɛqubt (le goût), agmuḍ.

Ilaq ad nexleq aferriq gar wayen yellan d taɛqubt n lxedma akk-d wayen yellan d taɛqubt n tudert.

Deg yal amur, akken ad naru deg iman-nneɣ taɛqubt n lxedma, teqqen-aɣ ad nbeddel akken ma llan lumur n ddunit-a.

Ulac win yezemren ad yaɣ aɣilif n taɛqubt n lxedma ma yella yesɛa aɣalat n tmuɣli akk-d wayen yeddren.

Tideţ, tmuɣli teḥbes taɣdemt n wayen yeddren.

Ma yella yiwen yeqqarru akk-d wayen yeddren, ur yezmir ara ad d-yegmer seg-s ayen yelhan i tɣerda akk-d wemɣi n teṛwiḥt n tmuɣli.

Axxam n tseɛdit yettuɣalen ar tmuɣli seld mi yeffeɣ amrar, yettwali taɛqubt n tudert deg umkan n taɛqubt n lxedma.

Ayagi yesken belli tmuɣli yettubeddelen zik, tuɣal ar tmuɣli-ines.

Yal tewtilin ur yeɛǧiben ara ilaq ad yettwabna s usemres n lxyal s tiknik n tameɣruyt.

Tebni n yal tzeɣwa teqqen-aɣ ad nefhem s iman-nneɣ akk-d ṣṣifa timsegreft, tgellet n kra n yuwan yeqqenen deg-s.

Amel: Tzeɣwa n tismin n leḥbuk; deg-s ttafen yuwan n ɣas, tismin akk-d tkerḥit.

Ad nefhem yal yiwen seg yuwan-a, yal yiwen seg lumur-a, teqqen-aɣ aseqsi, tigawt, tameɣruyt.

Aɣilif ad nɣalat ɣef wiyaḍ d aɣbel i telfat n tuccḍiwin nneɣ.

Ma ɛesfa, tella d tameɣrut ayen ur yelḥin ara ad yettwaɣal deg-neɣ aɣilif ad nɣalat ɣef wiyaḍ.

S yisem n tideţ ilaq ad nini belli d nekkni kan i d imḍerrben ɣef lumur ur yeɛǧiben ara n tudert.

Lumur-a yeɛǧiben neɣ ur yeɛǧiben ara ttafen yid-neɣ neɣ war-aɣ, yettales s lmakina yal ass.

Ma nbedda ɣef llsas-a, ulac aɣbel yezemren ad yaf ḥellu umeggur.

Aɣebbel d aɣbel n tudert, ma yella yella ḥellu umeggur, tudert ur tella ara d tudert maca d lmut.

Ihi yezmer ad yili beddel n tewtilin akk-d aɣebbel, maca ur ttkelsen ara ad ttalesen, ur ttfaqen ara ḥellu umeggur.

Tudert d taɛeqqamt tettweḍḍuf s lmakina akk-d tewtilin akk yeɛǧiben neɣ ur yeɛǧiben ara, yal ass tettales.

Ur nezmir ara ad nḥebbes taɛeqqamt, tewtilin yelhan neɣ ur neḥlin ara tettwaɛmeṛ s lmakina, nezmer ad nebeddel kan tmuɣli nneɣ ɣef lumur n tudert.

Akk nekkat ad negmer ayen yelhan i tameɣruyt gar lumur n ddunit, ad nɣerḍeb iman-nneɣ.

Deg yal tewtilin yeɛǧiben neɣ ur yeɛǧiben ara, ttafen yuwan yemgaraden ilaq ad ttwafehmen akken ma llan s tiknik n tameɣruyt.

Ayagi yesken belli yal agraw n yuwan yeqqenen deg kra n umezgun, neɣ tmuɣli n tudert, seld ma ttwafehmen akken ma llan ilaq ad ttwaḥeṛsen s tezmert n Tmeṭṭut Ulahiyya Kundalini.

Akk nexdem s teẓri n tmuɣli, ayagi ad yettnerni. Ihi nezmer ad neẓer deg lɣaci kan yuwan uqbel ad ttwaqeddmen, neɣ deg wakud n lxedma.

Mi ttwatteqqsen yuwan-a yerna ttwattwexṛen, naru d asuref ameqqran, d ussegdef ameqqran.
