Lmasiḥ d times n times, d uḥeffaf n uḥeffaf, d tsignatur astral n times.

Fell n tmeskert n umeɣras n Qalvariyu tettwabḍaḥ tmeɣrut n Lmasiḥ s yiwen wawal kan yesɛan ukkuẓ isekkilen: INRI. Ignis Natura Renovatur Integram - Times tessuddes tettɛeddi ddunit.

Asmi i d-yeɣli Lmasiḥ deg wul n umdan, yettbeddil-aɣ s yiwen n uswir.

Lmasiḥ d LOGOS SOLAR, yiwen d waṭas yeṣfan. Lmasiḥ d tudert yettwerɛac deg yal taɣalt n lebda, d ayen yellan, d ayen yellan zik, d ayen ara yettili daymen.

Aṭas i d-nnan ɣef Drama Takuzmikt; ulac ccek belli Drama-agi yettwasegrew s ukkuẓ injilen.

Nnan-aɣ-d belli Drama Takuzmikt tewwi-tt-id ɣer tmurt n Elohim; Agellid Ameqqran n Atlantida yettmetti drama-agi s Aqcic d Iɣsan.

Lkabir ameqqran Ɛisa uqbel ad yettmetti Drama yellan deg Uǧennet ɣer tmurt iqeddsen.

Ɣas ma yettlal Lmasiḥ alef iberdan deg Bitḥlem, ur yesɛi ara azal ma ur yettlal ara daɣen deg wul-nneɣ.

Ɣas ma yemmut yeḥya ass wis tlata seg lmegtin, ur yesɛi ara azal ma ur yemmut ur yeḥyi deg-nneɣ daɣen.

Ad nɛreḍ ad naf ttabɛa d tɣawsa n times d acu i d Ṛebbi, anda tilisa-ines ɣer tilawt tban-d daymen s ṣṣifa n times.

Tẓeggwa ttefsex (Ɛarfa, III, 2) d uɣerday n Sinai asmi i d-teffeɣ lɛahd (Ɛarfa, XIX, 18): d snat n tbanit anda i d-ibeggen iman-is Ṛebbi i Musa.

S ṣṣifa n yiwen n ugama n Yaspe d Sardunik n ini n uḥeffaf, yeqqim ɣef Ukersi ucqiq d umawaw, Yuxenna i d-yesselḥeq s ṣṣifa n bab n Lebda. (Awal n Ṛebbi, IV, 3,5). "Ṛebbi-nneɣ d Times Iseṭṭren", yura San Paul deg Tabrat-ines i Ubrukiyen.

Lmasiḥ i yellan deg wul, Times yellan deg Uǧennet, ilaq ad yettlal deg-neɣ yettlal s tidet mi nezdi mliḥ deg lxedma n ssḥa n nnfs.

Lmasiḥ yellan deg wul, ilaq ad yekkes seg Ugama n Ssḥa n nnfs-nneɣ, d snat seg usufɣuɣ ara d-yettaggen tɣaltin; TIYUCA CAUSAS.

Ur yezmir ara ad yesseɣṛeb aseɣẓeb n tɣaltin n EGO asmi Lmasiḥ Intimu ur yettlal deg-neɣ.

Times tudirt d Tafelsufit, Lmasiḥ deg wul, d Times n Times, d ayen yeṣfan deg ayen yeṣfan.

Times tezzzi-yaɣ-d tessal-aɣ-d seg yal tama, tettawi-yaɣ-d s usigna, s waman d tmurt s timmad-is yellan d tigga d tikkal-ines yemgaraden.

Times n Uǧennet ilaq ad tteḍfer deg-neɣ, d Lmasiḥ Intimu, amsellek-nneɣ ameɣlali.

Agellid Intimu ilaq ad yeddem tamurt-ines deg Psiquis-nneɣ n Xemsa Cilindros n Maquina Urganica; deg yal tikli-nneɣ n Lɛeqel, Emucyunel, Mutur, Instintivu Sikswal.
