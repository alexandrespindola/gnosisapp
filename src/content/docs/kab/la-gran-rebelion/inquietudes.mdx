Ulac ccek dakken ger tmuɣli d ussermed yella umgarad ameqqran, ayagi ur yettargu ccuba.

Yella usemmiḍ ameqqran ger medden, d asemmiḍ n wayen ur yesɛi ara azal, n wayen yesdersen kan.

Tettamen tdukkla tamaneɣt dakken ayen yesɛan azal d ayen ur nesɛi ara azal, tɣilen dakken aḥric aneggaru n lmoda, neɣ takerrust n umeddel aneggaru, neɣ ameslay-agi ɣef ujur ameqqran d ayen yellan d ṣṣeḥ kan.

Qqaren ṣṣeḥ ɣef tira n wass, talalit n tayri, tudert tasettart, tacekkunt n ccṛab, tazzla n imɣan, tazzla n tkerṛusin, tazzla n ibulisen, leɣder, acewwel, atg.

S tidet, m'ara yeslen argaz n wass neɣ tameṭṭut n tɣuriwmin ɣef useɣwen, imi ayagi ulac deg uɣerbaz-nsen, neɣ deg timlilitin-nsen, neɣ deg tulawin-nsen n seks, ttarun-d s ulac dɣa asemmiḍ yessexlaɛen, neɣ kan xarxen-d imi, refɛen-d iɣsan, ṭṭsen s ulac n lhem.

Aɣḍas-agi afsigan, asemmiḍ-agi yessaxlaɛen, yesɛa snat n tsennayin; amezwaru d tuggdi tameqqrant, wis sin d ulac ameqqran n tuɣyin tiṛuḥaniyin.

Ixerṣ uṭṭun, aseɣẓan aɛeẓẓan, ulac win i d-yefkan deg ddkan, ulac daɣen ger wayen iɣilen ṣṣeḥ, neɣ kter seg wayen yellan deg tulawin n weṭṭef.

Lemmer yella win i d-yefkan i bu-usemmiḍ adabuḍ neɣ i tcacit tasdersant taṭṭeft taɛeẓẓant n lawan, aburɣes n wul, kra n tnekra n berra, ulac dɣa yettɛebbaɛen, imiren ad yella umgarad.

Lakin kra yettḥariren i tɣeɣṭeɣt tufessast, i wul amezwaru, i tṛwiḥt taɣezzayant; yezmer ad yili d abelɛaq, taɛeqqact taḥlawt n kra n tɣuriwmin neɣ aseɣti n umeddakel ur nesɛi ara ɣur-s azal, atg.

Abalɛaq, tiɣeɣɣḍiwin ur nesɛi ara afus, lakin yesɛa tezmert deg tikkelt i d-yettunefken i tmeɣṛa tamezwarut taṛuḥanit, tṛwiḥt taɣezzayant, tṣuffiwt n tafat, wul i ɣ-yeḍleqen deg kra n lawan.

Lemmer ayen i yellan ass-a d agḍiḍ i yettidiren, isemmiḍen ineqqalen n wegrar neɣ kan imzenzaɛ n tɣellatin deg ddkan n webrid ameqqran, ur ten-yeɣḍesen ara tmeɣṛa tamezwarut taɣezzayant, ad ilin deg lawan-agi tafat n ṛṛuḥ, imdanen n tafat, irgazen ṣṣeḥ deg tmuɣli tameqqrant n awal.

Aburɣes, wul, amenɣiḍ ucbiḥ, ulac dɣa yettwaḥses-as deg kra n lawan sɣur uqessaḍ n tɛeqqabt, sɣur umezzenzaɛ n iḍarren neɣ sɣur uḥkim ameqqran, lakin meṛṛa yella d abrid, tiɣeɣɣḍiwin n tilawt yettɣedsen i burɣes amezwaru n tafat; deffir-s yettaɣ asemmiḍ n ulac n lhem ameqqran.

Ur yettargu ccuba dakken medden yettɣummu-ten ayyur deg kra n lawan; tideț-agi ur tɛeddi ara ccuba.

Ulac win deg tudert ur yessermed ara kra n lawan wul, aseɣẓan ucbiḥ, meṛṛa kra n tilawt, ɣas ma d agrud, yettwassefḥan akken ad yeṛṛu yettwassefḥan akken ad yeṛṛu tajiṭ tajiṭ n tlemmast ayen i ɣ-yeḍleqen deg tsedmi n yiḍ deg kra n lawan.

Ayyur yettɛegg i tneglabin-agi, yettɣerri, yesnerni ɣef tidett s tɣawsiwin-nneɣ iduɛfanen.

Ayyur d amekkan ameqqran; amdan n ayyur, ur yettwasufɣ ara s lekmal seg tuɣyin n tafat, ur yettwafham ara yettḥerril deg ddunit n iṭerriwen-is.

Lemmer yella win i d-yettunefken i tmeɣṛa taɣezzayant i d-yettunefken deg uɣur n kra n yiḍ, ulac ccek dakken deg lawan ad yettḍafar tiɣri n tafat d ad yettwaru deg tmuɣli-agi s argaz n tafat.

Ayen ayagi i yebɣa Tafat, lakin i teɣliwin-agi n ayyur isemmiḍen, i yettɣiman d ulac n lhem, ayyur yettɣummu-ten; deffir-s yettɣimi aseɣti n lmut.

Lmut tetteddukul meṛṛa. Agḍiḍ yettidiren yettwasufɣ seg tuɣyin n tafat, yettɣummu s lekmal deg tmuɣli tagriglit alamma ayyur i t-yeggar.

Tafat tebɣa ad tessen irgazen, tettarra tamtil deg ulaboratur n taggara; s tɣerma, timtil-agi ur tt-tefki ara tigratin, ayyur yettɣummu lɣaci.

Lakin ayen i d-neqqar ulac win i t-yeḥwaǧen, kter seg imeddalen i wumi d-yekkes teẓri; ḥussen-d iman-nsen d yemma n yefṛuxen neɣ d baba n Tarzan.

Tafat tessers deg yexxamen n sex n umɣar aɛeqli yeɣleten s yisem n urgaz, kra n tajeṛṛumt n tafat i yettwaḥḥeqqen s lekmal yezmer ad aɣ-ibeddel deg irgazen ṣṣeḥ.

Lakin timtil n tafat tezmer ad tili d tiɣẓiwt imi ayyur isemmiḍ.

Lɣaci ur bɣin ara ad ɛiwnen Tafat ɣef tudert, taɣawsa-agi tajeṛṛumt n tafat tettɣummu, tettɣeddir d tettenzu deg lɣella.

Taqeffalt tameqqrant n lxedma n Tafat tella deg ulac n tɣawsiwin iḍuɛfanen i d-nessuɣu gar-aneɣ.

M'ara tenger tazzla n wawal ɣef ixeyyal n tafat, Tafat tessefsay-itt imi ur t-teɛɛin ara i timtil-is.

Imi tasekkurt-agi n tura teggummu slekmal, tettwaxxer s lekmal d amekkan, ur tɛɛin ara i timtil n tafat, talɣa kter s lekmal iɣ ad tessefsay.

Akken ad yella aseɣwen aṛuḥani ameqqran iḍlem ammas n tɣerma n tezmert i lḥella, i tɛeqlit.

S tɣerma, lɣaci sɛan ammas n tɣerma n tezmert deg tilawt, deg lqahwa, deg lkan, deg leḥwayeǧ n lbunuk, deg taddart n tirmitin neɣ deg ugar n ddkan, atg.

S tidet, meṛṛa ayagi d tɣawsiwin n tilawt d ammas n tɣerma n tezmert yettɛagg i meṛṛa tɣawsiwin-agi; ayagi ur yettargu ccuba d kra n umdan i yesɛan ṣṣeḥ yezmer ad t-yessen s yiman-is deg talɣa tadayt.

S tɣerma, m'ara yeɣṛa meṛṛa ayagi, imcawaḍen n uɣerbaz, yettwaɛawden ad msawalen kter neɣ ad ṭṭsen s zzux ur yettusuffuɣ ara, bɣan ad ḍeggṛen adlis s ulac n lhem d ad qren aɣmis.

Kra n iqerraɛ n lqahwa u ad tettɣimi deg wass d aɣrum ameqqran i yettidiren.

Lakin, ḥussen-d iman-nsen ṣṣeḥ; s tidet tmusni-nsen tetteẓẓuẓu-ten, d tɣawsiwin-agi n tafat yettwarun deg udlis-agi uselḥeqqen tetteḍleqen s lekmal. Ulac ccek dakken allen timeqqranin n uḥekkam n teẓri ur zmir ara ad yettkemmilin tɣuri n lxedma-agi.
