Aẓayer agejdan, amedya ibanen n tuffɣa n teɣzi n wul, s tideț d AMASIḤ D AJENNAWI.

Wid yețɣilen d AMASIḤ D AJENNAWI d yiwen umdan yellan d abeṛṛani, i d-ilulen deg yiwen n wemkan n tmurt neɣ i d-yusan seg tmurt-a neɣ seg tmurt-nni, tella fell-asen lɣelta tameqqrant.

Nenna-d s tuffɣa tameqqrant d AMASIḤ D AJENNAWI ur yelli ara d asentel i d-yettwakksen, maca d asentel yal wa.

D tidet d AMASIḤ D AJENNAWI yella deg uzɣar n yal yiwen, yettban-d deg waṭas n wudem.

Teffɣa n teɣzi n wul i yellan deg umahil n tmetti d ayen ifettlen; teffɣa n teɣzi n wul i yeɣlen seg tmetti teqqar-d d ayen ur ifettlen ara.

Seg tuffɣa n teɣzi n wul war tmetti, teffɣen-d wid yettɣuṛṛun, amedya ibanen n AMASIḤ D AJENNAWI.

D tidet win yettɣuṛṛun s timmad-is d AMASIḤ D AJENNAWI. Aɣbel, ddunit i deg nella tura, s lumuṛ-is akk d tiɛeggalin-is, tettwaḥkem sɣur AMASIḤ D AJENNAWI.

Lḥala n ur tukki deg i yella lɣaci tura, s tideț teḍra-d ɣef sebba n AMASIḤ D AJENNAWI.

Win i d-yenna Pawlus n Ṭarsus deg tebṛatin-is, d tideț d ayen yettwaḥeqqeqen deg tallit-a.

Win ur neṣliḥ ara yusa-d yettban-d deg yal amkan, s tideț yesɛa adrim n umkan akk.

Yettmeslay deg yixxamen n qahwa, yettaṭṭaf deg UN, yeqqim deg Ženeva, yettawi tajṛiba deg usinag, yettales deg yibumbat atomikan, iɛeqqayen i yettubeddlen, iɣezzan i yettseqsiḍen, ibumbat biulujikan, atg., atg., atg.

Yetḥeyyeṛ AMASIḤ D AJENNAWI s tuffɣa-ines n teɣzi n wul, axsiṛ n wid yettmeslayen, yetɣil yessen lumuṛ n ddunit.

AMASIḤ D AJENNAWI yetɣil yessen kullec, yettenɛeq deg akk ayen yellan d leɣḍer, yeṭṭeggiṛ kullec yellan d Rebbi neɣ kullec yettwaɛbaden.

Axxam n AMASIḤ D AJENNAWI, zzux d tmeɣṛa yesɛa, d ayen ur yettwaberrḥen ara.

AMASIḤ D AJENNAWI yeɛreq s lmut tidiwin n Imsiḥiyen n liman, ṣṣber d tameɣrant.

Yal tazegzewt ttekken zdat AMASIḤ D AJENNAWI. D tidet, yettales deg yefḍagen ultrasunikan, lbabur i d-yettḥeyyiṛen, tikeryas i d-yettbegginen, ddwa i d-yettḥeyyiṛen, atg.

Deg lḥala-yagi, anwa ara yettḥeyyiṛen deg AMASIḤ D AJENNAWI? Win yettarun deg tallit-a mgal lumuṛ-agi akk i d-yettbanen n mmi-s n tmettant, yettwaɛreq deg aɛeggeɛ n watmaten-is, adlis, taluft, d win yellan d aqeffaf d ameggagal.

Yeɛteb ad yettwafhem aya i lɣaci iqeddcen d wid yeqqaren, wid-agi i yettwaɣen, nessenɣalen tuggta.

Yettban-d d aɣersiw aɣezzan sɣuṛ lɣelta yettusemman d amdan, d arubut i yettwaḥeṭṭen s kinder, tazwara, taleɣwest, taddart, tasdawit, atg.

Ulac win ara yeṭṭeggiṛen d arubut i yettwaḥeṭṭen yettwaḥkem sɣur aḥeṭṭu, ulac amek ara yettwaḥkem ma yettwakkes seg aḥeṭṭu.

AMASIḤ D AJENNAWI yeɛreq aḥeṭṭu s win yettwaḥeṭṭen arubuten amdan deg tallit-a i yeɣlen.

Ad nerr tajṛiba-yagi, ad nerr adrim deg ayen d-ttiniɣ, yeɛteb aṭas ɣef tella berra n uḥeṭṭu, ulac arubut amdan ara yeqbel lumuṛ yellan berra n uḥeṭṭu.

Tameɣrant-a teqqar-d d umezgun i d-yettwaqqnen deg ul, ulac amek, arubut amdan yellan d wa yeqqar-d d aḥeṭṭu ur ifettlen ara, imi netta yettwaqqnen ɣef uḥeṭṭu, ad yetḥeyyiṛ fell-as d ayen ur neṣliḥ ara, d ayen ur yefhim ara d ayen yesseḥsayen.

Arubut yetḥeyyiṛ deg uḥeṭṭu-ines d ayen ur d-iban ara, d ayen ur yezmir ara imi tella-ines teqqar-d ɣef uḥeṭṭu.

Aɣbel, lumuṛ ur d-ibanent ara amek yetɣil arubut amdan; tella-d tussna-nniḍen, teɣzi-nniḍen n wul, ur yeqqim ara i arubut amdan.

Arubut amdan yettwaɣ, yesɛa lḥeqq ad yettwaɣ imi ur yettwaḥeṭṭ ara i tussna-nniḍen neɣ i tmetti-nniḍen, neɣ i kra yellan d umezgun i uḥeṭṭu-ines.

AMASIḤ D AJENNAWI yeɛreq iḥeṭṭunen n arubut amdan, arubut yettwaxxer zdat umaɣlaf-ines. Amek ara yettḥeyyiṛ arubut deg teɣzi n umaɣlaf-ines?

Yeẓẓu umecṭuḥ iṣfan d iṣfan; tidet tban-d deg yal lexluq teqqar-d d ayen yesɛan azal.

D tidet, ddunit tesseṛɣay deg yiɣef n win i d-yezlulen ayen yellan d ayen iberrani, aẓar, agellil, n ddunit, asirem, amaynuten, i yesseḥqen ad yaɣ neɣ ad yerr tideț yellan deg yiwet n lḥala i tɣara ddunit.

Aya yeɛni umecṭuḥ i d-yezlulen yezmer s timmad-is ad yebder tidet n yal lḥala n ddunit, aɣbel, yettwaqqen aḥeṭṭu n AMASIḤ D AJENNAWI, d tissas ibanen i d-yerra ddunit deg yiɣef n win i d-yezlulen qrib yettwakkes.

AMASIḤ D AJENNAWI yeṭṭeggiṛ ad ttmeyyizen deg wudem-nniḍen; yal lexluq i d-yezlulen, ɣef lumuṛ n AMASIḤ D AJENNAWI ad yettwaḥeṭṭu.

Ulac ccekk d AMASIḤ D AJENNAWI yeɛreq s lmut tussda-nni n Yilem, i yettwassnen am "tazmert n ad tenzeɣ seg iɣes n tidiwin n ddunit".

Tussna yeṣfan, teggul seg akk lumuṛ i yettwaɛeqlen deg tesdawit yellan da, dinna d dinna, d ayen ur neqbal ara i arubuten n AMASIḤ D AJENNAWI.

Aṭas n ṭrad, lluz, d iḍan i yessawaḍ AMASIḤ D AJENNAWI deg akk iɣil n ddunit, ulac ccekk ad yessawaḍ iɛeqqayen-nni qbel ad tewweḍ tmettant.

Aɣbel, tewweḍ-d tezmart n tuṭṭfa tameqqrant i d-yenna yal anabi ulac amdan ara yettaru mgal AMASIḤ D AJENNAWI.
