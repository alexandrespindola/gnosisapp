Asegnew n yiman n wemdan yettaǧǧa-aɣ ad nessen tidet n yiwen n uswir yugaren deg yal yiwen deg-neɣ.

Mi ara yizmer yiwen ad iẓer s yiman-is, s tarrayt tusdidt, tilawt n sin n yergazen deg yiwen iman-is, win yellan d asadda deg uswir aɣarim amagnu, win yellan d asawen deg uktaw yugaren, imiren kullec yettbeddil yerna nettḥawal deg wanect-a ad neddu deg tudert ɣef leqwanen tilellasin i yellan deg tlemmast n TEBDANT-is.

Akked akken tella tudert tazdayant, akked akken daɣen tella tudert tadigant.

Argaz azdayan ur yelli d kullec, asegnew n yiman yesselmad-aɣ-d tidet n urgaz adigan.

Argaz azdayan yesɛa tarrayt-is n uɣerbaz, d ayen yesɛan aṭas n tikta d tugniwin tidiganin deg tudert, d taceṭṭiḥt tetteddu s isufa ur nettmuqul.

Argaz adigan d TEBDANT tunṣibt, tettwaṣfa deg leqwanen nniḍen meṛṛa gar-asen, ur yezmir ara ad yuɣal d aṛubuti.

Argaz azdayan ur yettaf ara tawla bla tuzzya, yetḥuss d akken xelṣen-t d amcum, yettḥunn fell-asen, yettqadar iman-is s zzyada, ma yella d aserdas yettmenni ad yuɣal d ajeneral, ma yella d axeddam n uskar yettɛeggiḍ mi ur t-yessulin ara, yebɣa ad ttwassen lemaɛna-s am lazem, atg.

Ulac win yezmer ad yaweḍ ɣer tlalit TISSIN, ad yelḥu akken yeqqaṛ Injil n Mass, ɣer tama n yettidir s ussegnew n urgaz asadda aɣarim amagnu.

Mi ara yissin yiwen tagdi-s d leḥqara tadigant, mi ara yesɛu tigullelt ad iɛawed tudert-is, war ccek ad yessen s yiman-is belli ur yesɛi ara lemaɛna n kra n telmest.

"Tawenza i igellilen n uṛuḥ, acku nutni ara yewṛten tagelda n igenwan".

Igellilen n uṛuḥ neɣ imeḥtajen n uṛuḥ, d wid yessnen tidet tagdi-nsen, tkerḍi d leḥqara tadigant. Amezruy-a n wid ur yesɛin ccek yettwasnerni.

"Yesshel i ulɣem ad yeɛber ɣef teɣmert n uẓeffun, wala i umeṛkanti ad yekcem ɣer tagelda n igenwan".

D ayen yettwabeyyn belli aqelmun yettwaṣebḥen s aṭas n lemaɛna, tmedalin d imedylen, tikliwin timettiyin ttwamgaraden d tmuɣliwin tusnakiwin ttwagalden, ur yelli d igellil n uṛuḥ yerna ɣef waya ur yezmir ara ad yekcem ɣer tagelda n igenwan.

Akken ad yekcem ɣer Tagelda yettwagellel uɣerbaz n liman. ɣer tama ur yettwaṣfa ara deg yal yiwen deg-neɣ asegnew n yiman, LIMAN yettban-d d ayen yugaren ayen ur yezmir.

LIMAN d tussna idsan, tussna tajṛibt tusdidt.

LIMAN yettwamegrez dɣa d amennuɣen ilmawen, Ijnusiyen ur ilaq ara ad ɣlin dɣa deg tuccḍa-agi tameqqrant.

LIMAN d tajṛiba tusdidt n wayen yellan d tidet; tudert tamesɣant n urgaz adigan; tameslayt tarufant tunṣibt.

Argaz adigan, mi ara yessen s tajṛiba tamessugurt tusdidt imalen-is adiganen yiman-is, yettwabeyyn belli yessen daɣen imalen adiganen n medden meṛṛa yellan deg wudem n tmurt.

Ulac win yezmer ad yessen imalen adiganen n umtiweg Tmurt, n usud anafuk d tgelda anda nettidir, ma ur yessen ara uqbel imalen-is adiganen. Ayagi icuba i win yemmuten i yerwel ɣef tudert seg tewwurt tameẓẓugant.

Tilufa tewwḍen n uɣelmed yettakken azuɣar-is udmawan deg uglim amcum KUNDARTIGUADOR (azrem yettɣuṛṛun n Eden).

Tizzla yettwaṭṭfen gar imuqranen i yesbeddayen Iɣil yettwaṣfa ɣef teɣzi n uṭṭfa-ines iman-is.

Tizzla Iɣilan tuɣal-d dɣa, deg tegda takumatusant, s ussugen amehni ɣas akken icuba i yal amdud yettwaṭṭfen seddaw taɣellist n tal neɣ tal dawa.

Nezmer ad nesserɣed taluft-agi deg tarrayt-agi: ussugen n tizzla Iɣilan yessawaṭ ɣer ussugen yettwakemmel s ddwa.

S tideț snat telmas-agi n ussugen yesɛant azuɣar-insent amezwaru deg uglim amcum KUNDARTIGUADOR. (Ẓer taɣellist XVI n udlis-agi).

War ccek ddwa ttneɣnayen-d usɣar alfa, imiren war ccek yettruḥ i wayen yellan d aɣarim gar aqelmun d uɣaras; ayagi s tideț yettban-d d azɣar ummid.

Aɣelmed yessenkul tiɣri s asɣar d win yettwazedɣen yettxemmim ad yessejṛeb ayen yellan d tidet seddaw taɣellist n ddwa, ur yettssen ara belli tilufa tewwḍen yettwakemmel s lḥeccic, L.S.D., lmurfin, iɣersiwen i yettɣuṛṛun, tukayin, lhiruwan, lḥaci, tiɣula timsuqtiyin s tuget, anfitamin, barbituriques, atg., atg., atg., d ussugen kan i yettwaṣebḥen s uglim amcum KUNDARTIGUADOR.

Iɣelmaden ttneɣnayen, ttruḥun deg lawan, ttwassermen deg taggara s tarrayt tusdidt deg imalen iṛuḥen.
