Tettunatin-nniḍen tiseddawiyin i d-yesdukkel yiḥsas-nneɣ (l’ego) sɛant izuran i d-yekkan seg tsebbiwin.

Iḥsas-nni n tsebbiwin yeqqnen ɣer yisalen n Cause et effet. Bla ccekk ulac tsebbi war agris, ulac agris war tsebbi; ayagi ur yettunqal, ur yettettuḥellil.

Ad yili d ayen ur nessawal ara ma nessuffeɣ akk iɣalen-nni yellan deg-nneɣ deg wulawen-nneɣ ma ulac ma nefsek akk izuran n leɛyubat-nneɣ tisikolojiyin.

Bla ccekk iḥsas-nni n tsebbiwin ttwaɛeqlen ɣer leɣṛamat-nni n Karma.

Ala ṣṣura-nni teqquren n tuɣẓa d ddwa n yimerayen n yisalen, ara aɣ-yefken ssɛada iwakken ad nessenger akk iɣalen-nni n tsebbiwin, s yiwen n uɣal yettqeddimen aɣ-neṣṣaweḍ ɣer usefsi n iɣalen-nni ur neḥwaǧ ara.

Izuran-nni n ṣṣura n tuccḍa-nneɣ, yettwazmir ad ttwasuffɣen seg yiman-nsen s ixeddimen i d-yesdukkel Ɛisa deg ul.

Bla ccekk iḥsas-nni n tsebbiwin ad yilin yewɛer nezzeh.

Amedya: Anelmad iḥeqqeqen yezmer ad yettwaɣḍer seg uselmad-is dɣa anelmad-nni ad yuɣal d ameɣlal. Deg umedya-agi, aḥsas-nni n tsebbi i d-yeǧǧan tuccḍa-nni, yezmer ad yettwasenger s ṣṣura n tuɣẓa deg ul d ddwa meqqren.

Ɛisa deg ul deg yiman-nneɣ yexdem iwakken ad yesfeḍ s ixeddimen ibanen d leɛtab-nni tella i lebɣi akk tsebbiwin-nni n tuccḍa-nneɣ.

Mass n lkemal ilaq ad yidir deg tiqqa-nneɣ taneggarut akk drama n lɣerḍ-agi.

Yettɛejjab win yettmuqulen deg umaḍal-agi yal leɛtab i d-yettɛeddin fell-as Mass n lkemal.

Deg umaḍal-nni n tsebbi Ɛisa yettɛeddi ɣef akk leɛtab-nni ur nezmir ad neḥku deg ubrid-is n leɛtab.

Bla ccekk Bilatus yessared ifassen-is dɣa yesseḥbibir fell-as maca yesseḥkem-it ɣer lmut deg umidag.

Tettban d tameqqrant ɣer umezday aɣerfan ulaɛɣal ɣer Calvaire.

Bla ccekk ṭṭbiɛa tanefrawt ddukkel ɣer Ɛisa deg ul, yettwaẓerbent deg umidag n Calvaire, yessawal awal yewɛren i wemdan ur yezmir ad yefhem.

Awal aneggaru (Baba deg ifassen-ik rxeɣ ṛṛuḥ-iw), yettḍefar-it ureq deg usigna d lɣerbaj d tissas meqqren.

Deg taggara Ɛisa deg ul, mbeɛd ma yenṭel deg uẓekka-s.

S lmut Ɛisa deg ul yeneɣ lmut. Akken ad ɣlin wussan, Ɛisa deg ul ilaq ad d-yekker deg-nneɣ.

Bla ccekk aɣelluy n Ɛisa yetbeddil-aɣ s ṣṣura teqquren.

Ssid i d-yeɣlin yella yella yesɛa tazmert meqqren ɣef tmes, ṭṭir, aman d akal.

Bla ccekk ssdaten-nni yettwaɣellin, ttɣellen tameɣrant, mačči ala deg iman maca ula deg tfekka.

Ɛisa lkevir mazal yidir s tfekka nni yesɛa deg tmurt n leɛqed; Ulaɣez San Germán i yettbeddilen amellal ɣer wurɣu, yettnufaḍ lǧuherat n ṣṣura meqqren deg leqrun wis XV, XVI, XVII, XVIII, ula tura yella yidir.

Ulaɣez Cagliostro i d-yeǧǧan tira deg umaḍal s tazmert-is deg leqrun wis XVI, XVII d XVIII yella d Ssid yettwaɣellin mazal yeṭṭef tfekka-s.
