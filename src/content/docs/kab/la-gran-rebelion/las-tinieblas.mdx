Yal waṭas n uruyen i d-yeṭṭeffren deg yiwet n tallit-nneɣ, d amgarad n tmuɣliwin.

Bla ccek, deg zzman-agi, rnan-d aṭas n lemdaris tilmas-nniḍen n tẓuri d tẓuri taḥeqqanit.

Tiddukla n leṛwaḥ, idlisen, d tmuɣliwin t-tṛwiḥ; dɣa d ayen iɣef d aɣilif ad taf amɣar yellan deg iẓẓi tiggara n tmuɣliwin-a ad yaf abrid n tuffɣa.

Ayen i ten-yugaren akk d leɛǧeb n lefhama; yella lɛada n usenɛeddi iman deg wayen yellan d lefhama kan.

Imessufen n lefhama ur qnɛen ara s idlisen-nniḍen n tmuɣliwin, meɛna imeslayen n tẓuri d tẓuri taḥeqqanit-nniḍen daɣen i ten-ttawin deg ayen yellan d ayen.

Aḍris n yimedyazen-a d ayen iḍefren d umesruf n lefhama.

Seltent-iyi-d tira d idlisen n yal aɣrud; imesfayen-a ttessennayen-iyi ɣef lemderza-nni neɣ adlis-a, nekk ad ten-rreɣ i ayen i d-yesnelfen: ǧǧ-et laɛyuḍ n uqelleb, t-tuɣalt i tudert n wiyaḍ, esserreɣ-d iman-ik, ur k-tentɛan ara lemdaris n wiyaḍ, ttuɣal d win yesɛan lḥisab, ḥess-d iman-ik, ɣur-k iman-ik, tmuqleḍ iman-ik, atg., atg., atg.

Win i ten-yugaren akk d asenɛeddi iman s teɣzi deg yal tama n uqelleb.

Tiddukli d tiddukla; tafat d asenɛeddi; yessefk ad nserreḥ i tafat ad tekcem deg tiddukli-nneɣ; tanetla tafat tesɛa tazmert ad tɣelb tiddukli.

Lɣaci ttatfen iman-nsen deg tama yeččuren d iɣessalen n uqelleb-nsen, ttafen iman-nsen deg iman-nsen.

Ur bɣin ara lɣaci ad ɛeqlen ur d-nsen ara deg tudert-nsen, yal wa yettwaḥkem seg ičči-s seg lɣaci-nniḍen, nekk bɣiɣ ad d-ɛerḍeɣ ɣef ttawilat-a n uɛeddad n iman-nneɣ deg ičči.

Yal wa deg yimanen-nni ad yessekcem deg uqelleb-nneɣ ayen yessefk ad nemmeslay, deg imi-nneɣ ayen yessefk ad naru, deg wul ayen yessefk ad nḥess, atg.

Deg tallit-a, timettit n umdan ur tesɛi ara s wayes ara teḥkem iman-is.

S yisem n tidet, yessefk ad naru s tserreḥt d amɣar aɣrib i yesɛan asenɛeddi deg iman-is yettidir deg tlalit n tussna.

Aɣrib ur yesɛi ara aɣrud ɣer tama, ma yella yella yella yella yeqqen.

Aɣrib yella-d ɣer tama-nniḍen d ayen yettwawḍen.

Amek ara yeqqen umdan-agi? I wakken ad yili usenɛeddi yella usenɛeddi.

Anagar tafat n tussna d imeslayen ur d-yerriren ara meɛna d tadukli ɣef iman-nneɣ, dɣa ad tessexsi aɣbalu n usenɛeddi.

Ma nesseɣṛeb ayen yellan deg ičči-nneɣ, ad yili usenɛeddi n tussna akked yiwet n tussna n tissas.

Mačči d ayen iɣef bɣin lɣaci ad ɛeqlen tiddukli-nni i deg idiren; ttnuɣun.

Ma lɣaci uɛen-d, yal wa ad yessen ɣer tɣellist-is deg iman-is.

Ma lɣaci uɛen-d, tɣellist-nneɣ ad aɣ-ssen deg ičči-nsen.

Dɣa ur teḍru ara ttaɣust, ddunit meṛṛa ad teqqen.

Tafat n tussna, dɣa ad aɣ-tesserɣeḍ asenɛeddi n tussna, ad tbeddel yal taɣawsa deg wemkan-is, ayen yettili-d deg ičči-nneɣ, ad yili deg wemkan-is.

Dɣa d ayen ur yessen lɣaci deg tussna yellan gar tafat d tussna.

Bla ccek, tafat d tussna d ayen yellan gar tafat d tussna.

Tiddukli d tiddukli-nni deg ičči-nneɣ.

Anagar s usenɛeddi n tussna ad nserreḥ i tafat ad tekcem deg tiddukli-nneɣ.

"Tafat t-tḍefren i tiddukli meɛna tiddukli ur tt-tefhimen ara."
