Deg yettwaxemmem deg iman-nneɣ, deg tilisa n ususru n iman-nneɣ s tmuɣliwin tisikolojiyin, ilaq ad nɛic, s tikta ur neɣwuṛṛu, akk taseɣẓant talsimanit.

Lmasiḥ Iqerben ad yekkes akk iberdan iderɣalen i d-neṭṭef deg ulawen-nneɣ.

Aṭas n yernuẓẓaẓen isikolojiyen deg tizzegnit tisikolojit-nneɣ ttɛeggiḍen, ssuturen ad ttusmirṛen deg umɣar-nneɣ.

Bla ccekk, yal yiwen seg-neɣ yeṭṭef deg iman-is agrawliw iḍehheṛen ṛebɛa.

Yehuda, amcumen n tayri; Pilatus, amcumen n wallɣu; Qayafa, amcumen n nniya-yir.

Agrawliw-agi smirṛen umɣar n telwit deg taqerrast n iman-nneɣ.

D tlata n leṣnaf id-ireɣlen n ibardan ur nsanɣararen deg taseɣẓant talsimanit.

Ur yelli uccekku, taseɣẓant i d-yettwabedlen deg wul d tilisa n tussna tagennawt n Umazzag.

Ur d taseɣẓant talsimanit n Umeɣlal Ameqqran Ɛisa, am akken ttamnen medden ur nessin.

Imuqren n akk lawan, Imagrawen n akk leqrun, ilaq ad ɛicen taseɣẓant talsimanit deg iman-nsen, da, tura.

Acu kan, Ɛisa, Amezwaru Ameqqran, yesɛa lǧehd ad yessegzi taseɣẓant i d-iqerben deg ubrid d leɛniq n wass, iwakken ad yeldi lmeɛna n ukemmel i medden, ur yelli lxilaf n tdersi, n ǧins, n ddab ula d ini.

D leɛǧeb i yellan win id-yessenɣaden taseɣẓant i d-iqerben i akk agdud n ddunit.

Lmasiḥ Iqerben, ur d amcumen n tukerḍa, ilaq ad yekkes seg iman-is iberdan isikolojiyen n tukerḍa.

Lmasiḥ Iqerben, deg iman-is d lehna d leḥmala, yessefk ad yekkes seg iman-is iberdan iderɣalen n ɣuṛ-es.

Lmasiḥ Iqerben, ur d amcumen n tamaɛ, ilaq ad yekkes seg iman-is iberdan iderɣalen n tamaɛ.

Lmasiḥ Iqerben, ur d amcumen n usḥissi, yessefk ad yekkes seg iman-is agrawliw isikolojiyen n usḥissi.

Lmasiḥ Iqerben, d tissas tagellilt, taɛeffant tugtama, d teɣẓant tazeqqant, yessefk ad yekkes seg iman-is iberdan iṭeqqiren n zzux, n yisem, n uɛeqqer.

Lmasiḥ Iqerben, awal, Awal-Akelkaw aɣaras deg tudert yellan tettqeddimen, yessefk ad yekkes deg ulawen-nneɣ, deg iman-is, s iman-is, iberdan iderɣalen n tuggdi, n tiɛeffanin, n uqeddem.

Umɣar n Telwit yebda akk iẓumman, yessers-iten, ulac imeɣri yeqɛaden n tɛeccant d wučči. I ilaq ad yekkes seg iman-is iberdan iɛeṭlen n tayri n wučči.

Tiggugga taɛjibant n Lmasiḥ-Ɛisa; Lmasiḥ-Argaz; axelluḍ ur nessin n wayen akelman d wayen ajjedan, n telwit d wayen ur teggren ara; assaɣ yellan yeqqaren i Uwal.

Akk ayen iɛejben deg wayagi, Lmasiḥ iɣelben deg teɣzi; yiwen yettaɣen tixuḍi; yiwen yekkessen tixuḍi deg iman-is, da, tura.

Lmasiḥ Yuɣelen d Umɣar n Taggugga Tameqqrant, yeṭṭuɛṛeq seg iqecɛacen, seg imɣaren d imsebriden n lǧameɛ.

Iqecɛacen ttenfufen-t; anamek-is, ur t-fhimen ara, bɣan Umɣar n Telwit ad yidir kan deg lawan nniḍen, s tmanit-nsen ur yesɛin ara lxilaf.

Imɣaren, anamek-is, imezdaɣ n ddunit, at tizegginin, medden ileqqmen, medden n tmusni ttagwaden awal, Lmasiḥ Azeggaɣ, Lmasiḥ n Taggugga Tameqqrant, imi iteffeɣ seg ddunit n tɛadda-nsen, d leɛwayed-nsen iɛeddan, id-yessuffuɣen, ur tettbeddel, deg waṭas n yizri.

Imsebriden n lǧameɛ, iberdan n leɛqel, ttagwaden Lmasiḥ Iqerben, imi yella d amjir n Umasiḥ, aɛdaw ad-yeqqnen seg wacu yeffeɣ deg inemgalen timsiraniyin, i yettazalan deg igelda n lǧasad d timawin.

Agrawliw-agi ttagwaden s lmut Lmasiḥ Yuɣalen, d wi id-yessekrewen seg lmut deg ulawen-nneɣ, d tallit tisikolojit-nneɣ.

Yehuda, amcumen n tayri, yebeddel Umɣar s teltin ijdiyen n sserfen, anamek-is, s ccṛab, s yedrimen, s yisem, s tayri, s tukerḍa, d zzna, atg.

Pilatus, amcumen n wallɣu, yessufuɣ-d akk afus-is, yettɣaḍar iman-is d amseffed, ur yelli d ameslub, yeqqar iman-is zdat iman-is d zdat wiyad, yettnadi ad yeffer, ad yeffer iwakken ad yeqbel taɛkkumt-is, atg.

Qayafa, amcumen n nniya-yir, yessers-d Umɣar deg ulawen-nneɣ; Uɣerfan Iqerben yesɛa tεekkwamt ad yesɛu tiɣeṭṭatin-is, acu kan, agrawliw ad-yeqḍen yessuffuɣ-d lǧameɛ i tɛebbuḍ n tayri, yettaɣ yezni, yessufeɣ-d lesrar, atg.

Agrawliw-agi ssarɣanen fell-as Umɣar Uɣerfan Iqerben bla tarwa.

Pilatus yettger tamɣart n tizegzewt deg teɣmas-is, medden iɛeglun tettneḍḍen-t, yettεeyyṛen-t, tettelɛanen-t deg tallit tisikolojit i d-iqerben bla tarwa n kra yellan.
