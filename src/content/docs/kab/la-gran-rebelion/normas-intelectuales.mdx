Deg umuɣ n tudert, yal yiwen yesɛa ccɣel-is, ttawil-is yellan d aqdim neɣ d amaynut, ur yettaqabal ara ayen ijdiden; aya d ṣṣeḥ, ur yettḍeggiṛ, ur yettbeddil.

Uɣal n umdan aqlaɛ aqlaɛ yexser, yeqqur, yella deg ussan n uɣuru.

Tideț, ifehhem n ddunit tura yettemcabi i lebni taqdimt teɣziwt ur teḥric, ur tezmir s yiman-is i kra n tɣawsa n tlelli n tidett.

Xusṣen imeslayen deg uɣal, yella deg waṭas n lɛwayed iquranin d leqdicin ur nesɛi azal.

Yal yiwen yesɛa ccɣel-is d lɛwayed iquranin deg-sent i yettaɣ ara yettuɣal.

Ayen yugaren akk deg waya d ugar n waṭas n lɛwayed yettaɣen azal n waṭas n lɛwayed yellan d irqiqen d imaruden.

Deg yal lḥal, lɣaci ur ḥulfun ara d akken tenɣaben, yal aqerru yettemcabi i ddunit, ulac ccekk belli gar waṭas n iɣallen n uɣal llan waṭas n yimeslayen n uɣuru d tbuqyaɛt ur ttettḍellɛen.

Lameɛna ccɣel aḍeqqiq n waṭas n lɣaci ur ttaḍeqqiq ara uɣuru n uɣal deg-s llan.

Lɣaci-agi imura s uɣal am tbuqyaɛt ḥesben iman-nsen d rnayen, țțaɣen iman-nsen d ilelliyen, d imeqranen, țɣilen ḥesben waṭas.

Iḍelma aqlaɛ aqlaɛ yugaren akk aṭas, acku s tidett, ma nmeslay tikkelt-a deg ussan n Sokrat, ad d-nini: "Ur ḥsin ara kan, meɛna daɣen ur ḥsin ara d akken ur ḥsin ara".

Ibezganen n uɣal yettṭefaen deg lɛwayed yellan d tiqdimin n zik, yettaɣen deg iman-nsen s ṣṣeḥha akken i ttaɣen deg-sen uɣuru-nsen, țeddun ḥettman ad ttaqbalen ayen ur yezmir ara ad yekcem deg lɛwayed-nsen n wuzzal.

Iɣilen lɣaci yessnen yessnen d akken yal ayen yeffɣen seg ubrid aquran n leqdicat-nsen yellan d irqiqen d amaru s wugar n meyya n tmerṭanin. Akka am aya, lɣaci-agi n ccɣel-nsen aḍeqqiq ɣur-sen zayren iman-nsen s lɛib.

Țțaɣen iman-nsen d iɛeqliyen ɣur-sen, țessenɣaḍen wid yesɛan azal ad ḥessnen seg lɛwayed-nsen iquranin s lawan, ayen yugaren akk d akken ur ḥsin ara s tidett taqdimt n tbuqyaɛt-nsen.

Tettmenniḍt n uɣal n uɣal aqdim yettemcabi i wayen yessnen d akken daɣen tettunefk i yiman-is ad tessutur ad ttbanay ayen yellan d ṣṣeḥ, ɣef wayen ur yellan seg uɣal.

Ur ttmenniḍen ara ad fehmen lɣaci n ifehhem iḍeqqiqen d wid ur nzemren ad țțaɣen sḥissi deg wayen yellan d ṣṣeḥ kan deg umkan n lḥesda.

Ur yezmir ara d ṣṣeḥ ad nɛeqqel srid leɣyub n tudert d lmut ɣas ma ur neḍri ara deg-neɣ deg iman-neɣ uɣal amezwaru.

Ur yelhi ara ad nɛawed deg yisefka-agi belli kan tamusni meqqren n wudem tezmer ad tessin tideț.

Uɣal amezwaru kan yezmer ad yexdem s yisallen i d-tekka tamusni n ddunit n WUDEM.

Uɣal yellan d ajerḍi, s yimeslayen-is, ur yezmir ad yessen ɣef wayen yeffɣen seg udmawen-is.

Nessen belli yimeslayen n yisallen n yimeslayen yeɛqal ttekkan seg yisallen i d-tekkant tissirtin n tudert n berra.

Wid yettḥettmen deg leqdicat-nsen n uɣal d lɛwayed iquranin, ttaḍeqqiqen kan deg yisallen-agi yuɣalen.

Kan s uɣuru n TEGO deg umkan n uɣal d ṣṣeḥ i nezmer ad neɣtes tamusni ad neḍri tidett uɣal amezwaru.

Lameɛna, akken tella, yimeslayen-agi yuɣalen ur ttaqbal ara deg ubrid alogan neɣ deg ubrid ameslay, ttuɣalin n yimeslayen n uɣal yettɣurru ttaḍeqqiq ṣṣeḥha.

Țmenniḍen aɣal-nsen iḍeqqiqen ad ddmen abḥar deg uglass, ḥesben belli tasdawit tezmer ad tḥazem tamusni n ddunit, belli akk lɛwayed n ddunit llan ḥettma ad ttaqbalen deg lɛwayed-nsen tiqdimin n tasdawit.

Ur ttaḍeqqiqen ara d ṣṣeḥ ɣef wid ur nessin, ad ttnadin tamusni, deg ussan iɣurran deg-sen llan.

Iɣallen ttettḍeqqiqen kan ɣur-sen mi ttnadin aɛlam n Ddunit, meɛna s zik ad ɣurun am times iɣurran, ḥessen seg ubrid n tirawin yellan d iḥbiben, țetten deg uɣal ḥessen seg wudem i lebda.

Tettmenniḍt n uɣal ur tezmir ara ad tekcem deg uɣuru iɣuran n WUDEM, meɛna leqdicat yellan d ajerḍi n wawal yezmer ad ddmen lɣaci ur nessin ɣer kra n tɣawsiwin tiɛeqliyin meɛna d tid ur nesɛi azal.

Tazmert n ugazu n yimeslayen alogan ur tezmmer ara ad d-tban tidett.

Uraren-agi ẓidanen n uɣal d awal, ttaɣen wawal s yiman-nsen yessawaḍen ad ḥessen deg umeslay d akken i yettaɣ arẓaz s tɛeqliɛt.

Tettḥussuɣ tazzla n tɣawsiwin deg uɣal i yettaɣen uɣal, ttefkan i iman-is tazmert yellan d amaru ad yeggul ayen ur yettuɣun ara araẓ dɣa iɛeqliyen n tesdawit.

"Aḍeggiṛ d aḍeggiṛ" n lɣaci iǧǧuǧǧigen yesɛan yimeslayen ur yettbeddilen, lameɛna wid yeɣɣazen s yimeslayen yettaɣen d akken ur nessin d akken teffeɣ d tamusni.

Mi newweḍ ɣer tama-agi n yisefka-nneɣ, ad d-nini belli d ṣṣeḥ, yezɛaf aṭas ad nessin anda i yettaɣ uɣal n lɣaci d anda i yebda aḍeggiṛ.

Ma nkemmel deg lɛwayed yellan d irqiqen d tiqdimin n uɣal, ad yugar ayen ur nezmir ara ad nessen ɣef wayen ur yellan seg uɣal, ɣef wayen ur yellan seg lawan, ɣef wayen yellan d ṣṣeḥ.
