Ma yettwamnen, d tideț tameqqrant, taddart-agi tatrart yettmeslayen fell-as mazal tecbeḥ mliḥ, ur tesɛi ara lmeɛnat tameqqrant n lemɛani n tesnarrayt, tella d tarat deg tezmert n tizeɣt taḥeqqanit.

Aṭas i neẓẓer ɣef lebni n tewwura-agi n zzman, yellan am tewwura n yiɣerda.

Dunnit tuɣal tugar-d mliḥ, iberdan yellan d imezwura, iɣerban-nni iweḥḥcen ɣer yal tama.

Ayagi akk yuɣal yuɛer, deg ugafa d unẓul, deg umalu d usammar n ddunit.

D yiwen uqeddic yellan d imezwura: iweḥḥec, iɣelben, d aɛqim. « Ttrarit! », ɣɣaren aṭas n medden.

Nettban am ifran yefɣen s ucebbub-nneɣ d yerkasen yellan d imeqqranen, ɣas akken da, da, da ttɛeddin imelyunen n medden yellan d ilemẓiyen, ur ttɛetten, d imeɣban.

Tseḥḥer d ṣṣifa taɣellawit, tatrawit, taḥeqqanit, ur tesɛi ara lemɣawla d usenqef deg tmeṭṭut. Tura nella d itrariten, akka i tedder tudert.

Medden uɣalen d imeɣben: ɣer tjeḥḥa, ur yelli win ara iɣaḍen.

Lemɛun n tewwura n tnezduɣin n uɣerɣur tesseɣzayen s leɛtab, fell-asent targa n lɛali, ur ttaweḍ ara ɣer ilemẓiyen.

Akayen ara xedmen Iberkanen n tudert d yal ufiɣ ur ttɛetten, akken ad ɛetten d teɛeccawin, iɣezdisen n iɣeblan n lɛali d tɛeccawin n uɣelli; ad ɛetten mbla ad asen-ten yawi, aɛeqqab yellan am Tantalus.

Medden n zzman-agi itraranen uɣalen d imeɣben: taɛeccawit n tayri d tɣellawt n lḥeqq ur tbanent ara.

Ṭṭsen tejmuɛan n medden i d-yefɣen ɣef leɣramat; dunnit akk deg uɣilif, ɣur-nneɣ i d-ḥqaqen d nekkni i d-ḥqaqen; ḥeqqen fell-aɣ d nekkni ur nesɛi ara ayen ara nxelṣen, lxetyar iqqesen aqerru, ulac win iteddren deg talwit.

Iberqan n uqeddic s telɣa n taɛeccawt deg uɛebbuḍ-nsen d tmeẓẓit tameqqrant deg imi-nsen, i ten-yettɛawanen deg wussan-nsen, ttxellisen aɛqab s yir tɣerɣar d aqerru, ur ten-iḍerr ara deg wussan-nsen deg leɣben n medden.

Ulac win yellan d ameɣbun deg wussan-agi d waɛrar n tgejdit, agi yella ger lmus d tɣawsa.

Iɣellayen d imeɣban, wid yettaɣen deg uɣerɣur d wid yettḍuruɛen, wid yettaɣen yerkasen d wid yettaɣen tɣawsiwin n tewwura, ttidiren axater ilaq ad ttidiren, ttexneɛɛun deg ccṛab ttaɣen-d lxetyar d tɛeccawin n yiman-nsen ad feɣen.

Medden uɣalen d imeɣben, d imeɣɣben, d imḥezben, d imeɣzayen, d imeɣben; ulac win itettmunen deg ulac; ttwexlaḍen yal ass tiwtitin timejrudin, iḍrisen, tɛuccurin n yal tawsit, iqecɛuren, tɛuccurin, atg., d akk ayagi ur yettxeddem ara, imeɣzayen ttṣeddiqen ɣef kullec agi: ur ttxellisen ara, tteffɣen ɣef leɣrur ɣas akken ad asen-ttawi s yessumaten-nsen ɣer lḥebs.

Ulac uqeddic ara yeɣben; lmeɛna n tayri taḥeqqanit teḍḍeɛ d medden ttzewǧen ass-a d yefḍen azekka.

Tajmaɛt n iɣerban teḍḍeɛ, lehwuḍ ur yelli ara, taɛeccawit d tmectut uɣalent d imeɣban mliḥ ɣef ussuref n ifassen.

Akk agi ad tessnen fell-as, ad ttɛetten ad tessnen tawsit n kullec deg teɛeccawit, ad stenqsen, ad qellben, d tidet d ayen ara nesɛi deg udlis-agi.

Tmeslayeɣ deg tutlayt n tudert taɣellawit, teɛɛawaneɣ ad ssenɣ ayen itettwaɣben deg udem-agi iweḥḥcen n tmettant.

Ttxemmimeɣ s ṣṣut ɛali d ad yinin iddan n leɛqel ayen i d-yettxemmimeɣ.

Tɛuccurin uɣalent yugar d armi zzenzent d yettɣazen deg ssuk. Ihi acu?

Tɛuccurin xedmen akken ad ttɣawalen lxetyar ad ɣelben tudert.

S lḥeqq yenna Goethe: « Akkul tɛuccurt tella d tameqqrant d tayri tameqqrant deg uɣerbaz, akken ad tɛetten tikelt-nniḍen... »

Medden ilemẓiyen ɣelben s aṭas n tɛuccurin, tura tmeslayen ɣef tɣawsiwin taɣellawin, ilaq ad nili d imeɣban d ad nessnen tidet tawsit n leɣben-nneɣ.
