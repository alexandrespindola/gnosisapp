War ur d-nettwali neɣ ur n-neqbil ara, yella degneɣ yiwen uɣilas ilemẓi iɣef ur nessin. Ilaq ad nessali tafat n tmusni ɣer uɣilas-nni.

Asegget n unadi-nneɣ n Usnussi (Gnostique) d amek ara nesserkem tamusni-nneɣ s timmad-nneɣ.

Mi ara yili degneɣ aṭas n lecɣal iɣef ur nessin neɣ ur n-neqbil ara, ad aɣ-yawi lecɣal-nni ɣer usnerni n tudert-nneɣ s tuɣmas, ad aɣ-yeɣɣiɣen deg yal aḥala ara yettwaggazen s tmusni-nneɣ.

Amedya, ad n-tenwali am akken d attekksen, d imselken, d imeqranen, deg wayen yellan deg ul-nneɣ.

Isnussi (Gnosis) yenna ɣef wayagi, nettaɛic deg yiwen umur meẓẓi seg timmad-nneɣ.

Asefti-nnes d amek ara nesserkem tamusni-nneɣ ɣer umur meẓẓi kan seg timmad-nneɣ.

Amezruy n lxedma n Usnussi (Gnostique) d amek ara nessali tmusni-nneɣ.

Bla ccekk, ma yella ur nxeddem ara d timmad-nneɣ, ur nxeddem ara d wiyaḍ, ad aɣ-yawi ɣer umennuɣ n yal ṣenf.

Ilaq ad yuɣal lxedma-nneɣ s timmad-nneɣ akken ad yettwaḥrez.

Aɣerbal n Usnussi (Gnostique) deg lxedma n Usnussi (Gnostique), mi ara ur nettafeq ara d yiwen n umdan, ad yili ccekk belli d netta ay d lexṣas ayɣef ilaq ad nexdem fell-as.

Aswexxer ayɣef nessexṣer deg wiyaḍ, d ayen yellan deg uɣilas ilemẓi-nneɣ, ayɣef ur nessin, neɣ ur nebɣi ara ad n-neḥrez.

Mi ara nili deg wayagi, uɣilas ilemẓi-nneɣ yeqqen, maca mi ara tafat n uḥraz ɣer uɣilas-nni, tamusni-nneɣ ad tnerni s tmusni-nneɣ.

D amedya d amellal n uɣersiw, yugar tikkest n temlilt, aṭas ay d-yeffɣen, maca drus ay d-yeɣran.

Akkenni d tallut (lqamar) tesɛa yiwen uɣilas uɣur ur tettwattaf ara, uɣilas ayɣef ur nessin, akken daɣen ay d-yeddukel d tallut (lqamar) n timetti ay nelqeḍ deg ul-nneɣ.

S tideț, tallut-nni (lqamar) n timetti teggugem seg Ugu, I, Mma, Timmad-nni.

Deg tallut-a (lqamar) n timetti, nelqeḍ lecɣal iberraniyen ay ssexṛaɛen, ay ssaxḍaɛen, ayɣef ur neqbil ara ad ilin degneɣ.

Abrid iweɛren d win n LḤEQIQAT UḌṚAN N TɣESSA, Acḥal n tmesḍaɣ! Acḥal n leḥyuḍ iweɛṛen! Acḥal n tmezɣunt tewḥacen!

Tikkal, abrid ilemẓi, seld mi ara yili aṭas n tseddiwin d tlawin, tɛellawin tewḥacen d tnaẓẓalin tigejdanin, ad yettɣelli deg uneẓruf n ṛṛmel, ur nettafeq ara anda yella, ur d-yetteddu ara fell-aɣ yiwen uccen.

Abrid yeččuṛ d lxetyar ɣer daxel d berra, abrid n tɣawsiwin uɣur ur nezmir ara ad n-ininti, anda yettaweḍ yiwen uḍris n lmut.

Deg abrid-a ilemẓi, mi ara nɣil belli nexdem akken iwulmen, nexdem akken ur iwulmen ara.

Deg abrid-a ilemẓi, mi ara nɣil belli nexdem akken ur iwulmen ara, aql-aɣ nexdem akken iwulmen.

Deg abrid-a n tuffra, llan tikkal anda ur nessin anda yella wayen yelhan neɣ wayen diri.

Ayen yettwaggaden, tikkal yezmer ad yili d ṣṣwab; akken ay yella abrid ilemẓi.

Yal adɣaɣen n tewtilin deg abrid ilemẓi, ttwaxṛen; yiwen n awal neɣ yiwen n umuɣ n tewtilin, deg tikkal yezmer ad yuɣal d lɛeqba meqqren i Lḥeqiqat uḍṛan n Tɣessa.

S ṣṣeɛd, Umasiḥ ilemẓi seg ul n tɣessa-nneɣ yexdem s tezmert, yetterzu, yettru, yettɣimi deg lecɣal iberraniyen ay nelqeḍ deg ul-nneɣ.

Umasiḥ yettlal am umeẓyan deg ul n umdan, maca mi ara yettɣimi deg lecɣal ur nebɣi ara ad ilin degneɣ, yettnerni s tuffra ar ad yuɣal d argaz akk.
