Kraḍen kra n yimusnawen n tmusni, tmetti tmettelt am ucḍiḍ yesɛan tazmert tameqrant iwakken ad iɛzel gar-aneɣ d wayen yeddukul ɣef-neɣ, ad iɣtes lquwwa-nneɣ.

Umen yimusnawen-agi belli anagar yiwen n ubrid iwakken ad ttmessexsiḍ si tezmert n "nekk"-agi neɣ "nekk"-agi, d ussidmer-it yal tikkelt s tuẓẓuft yugar akken ad t-nefhem, ad nuɣal d imusnawen fell-as.

Ttxemmimen medden-agi belli akka ara nɛizel iman-nneɣ ɣef "nekk"-agi neɣ "nekk"-agi, ɣas ma d tazmert n ucḍiḍ.

Akka, qqaren, "nekk" iɛzlen s tmetti, iban-d am yimɣi yețwazmer.

Ad tili-d tmusni ɣef yal "nekk", ɣur-sen, tɛni ad t-neɛzel ɣef tnefsit-nneɣ, ad t-neḥkem s lmut.

Bla ccekk, amsefhem-agi, i d-ibanen yesɛa tazmert tameqrant, yeɣleḍ deg tɣuri.

"Nekk" iɛzlen s ucḍiḍ n tmetti ɣef tɣuri-nneɣ, yețwaḍeggṛen si taddart am tixsi taberkant, yettkemmil deg tallunt tusnakin, yettili d ajennun yeɛǧiben, yettḥettim ad yuɣal ɣer taddart, ur yettqabal ara s lhes, ulac amek i yebɣa ad yečč aɣrum aderɣal n usuffeɣ, yeqqelleb ɣef tagnit, deg yiwet n tuccḍa n lḥader yeqqim tikkelt nniḍen deg tnefsit-nneɣ.

Ayen i gerrzen yugar, d akken deg "nekk" yețwaḍeggṛen yella yal waṭas tebḍit n lǧawher, n tmetti.

Yimusnawen-agi akk i yexemmimen akka, ur ɛurḍen ad sserɣayen ula d yiwen n "nekk" seg "nekni"-nsen, deg tidet nɣelben.

Ɣas ma nɛerḍ ad nenjerr si temsalt-agi n "KUNDALINI", aɣbel d ameqran.

Deg tidet "Mmi-s Ur Nkiṛ Ara" ur yettraqqi ara deg lxedma taɛeqlit ɣef iman-is.

Yettban-d "Mmi-s Ur Nkiṛ Ara" d win yețḥeqqren "ISIS", yemma-tneɣ Taruganit Tameɣlalt, tamasturit, tagensant.

ISIS d yiwet seg tibḍatin timahrayin n iman-nneɣ, maca yeḍfṛen, taɣẓunt tanariwt n tezmert-nneɣ tasɣarat, "KUNDALINI".

Yettban-d anagar "ISIS" yesɛa tazmert tamatant iwakken ad yesserɣay yal "nekk"; ayagi d ṣṣeḥ, ur yettwankeṛ, ur yettwettwed.

KUNDALINI d awal yedduklen: "KUNDA yettid-aɣ-d ɣer wul ɣef Uẓerẓur Aɛefyan KUNDARTIGUADOR", "LINI d awal Atlanti yettɛni tameggarut".

"KUNDALINI" yettɛni: "Tameggarut n Uẓerẓur Aɛefyan KUNDARTIGUADOR". Ilaq ur nexleḍ gar "KUNDALINI" d "KUNDARTIGUADOR".

Nenna-d yakan deg yiwet n tegzirt iɛeddan belli Taɣẓunt Tanariwt n tezmert-nneɣ tasɣarat tettweṭṭef tlata tikkal d wezgen deg tlemmast tasɣarawt tezga deg yiɣes Coxígeo, lsas n usadar ameqqran.

Mi ara tɛali taɣẓunt, d KUNDALINI, mi ara terṣi, d Uẓerẓur Aɛefyan KUNDARTIGUADOR.

S "TANTRISMO AMLAL" taɣẓunt teɛla s tnesbaɣurt ɣef ubrid ameqqran asadar, tettseḥyu tezmert i yesdukulun.

S "TANTRISMO ABERKAN" taɣẓunt teɣli si coxix ɣer iǧehhiman itamiyen n umdan. Akka i uɣalen waṭas d iǧennunen iweḥcicen mliḥ.

Win yeḍfen tuccḍa ad yefk i taɣẓunt taɛlalt akk lamanat izeɣwan d idubaɣen n taɣẓunt tarrast, tɣelli deg lxedma ɣef iman-is.

Aɣerruj-nniɛma n "Uẓerẓur Aɛefyan KUNDARTIGUADOR", anagar d "KUNDALINI" i yezmer ad ten-yesseḥyu.

Ur yeqqim deg-neɣ ad nessifsay belli aɣerruj-nniɛma-agi ttwaseɣren deg "NEKK" yettwajerren n Tmusni taruglant.

Tazmert tamseflalit n Taɣẓunt tarrast tesɛa tamdanit tettemɣaq deg tlemmast n tukrist.

Anagar Taɣẓunt taɛlalt, s tektiḍt, tezmer ad aɣ-tessekker; tideț-agi d aglaw ameqran n Tussna Tahirmant. Tura ad nefhem tuɣast tusridt n awal uḥric "KUNDALINI".

Lebɣi amussan yettban-d yal tikkelt s tameṭṭut taḥrict, María, ISIS, yettɣaḍen aqerruy n Taɣẓunt tarrast.

Tbeddaɣ daɣen s ṣṣeḥ, bla ccekk belli ddu tamɣalt n tafat, times tudirt d tsegganet n tmurt, tettwabeddleɣ s taɣẓunt s uqerruy n uɣersiw, n uɛekṛuc neɣ n ukeṛṛa deg Tmeɣriwin Tiqdimin.

D tɣeẓẓunt timeddukklin n Caduceo de Mercurio; d Taɣẓunt yeɛǧiben n Edén; maca daɣen bla ccekk, d Taɣẓunt n Nuḥas n Musa yeṭṭfen deg "TAU", tɛni, deg "LINGAM Aserɣan".

D "Aɛekṛuc" n Sabbat d Baphometo n Templarios Gnósticos; d HYLE n Gnosticismo Amadlan; d aɣanim ameddukal n taɣẓunt i yessegren iḍaren n Weẓru Aɣafaw n ABRAXAS.

Deg "LINGAM ABERKAN" yeṭṭfen deg "YONI" amegdi, imuqran n Rebbi SHIVA, Taruganit Tahendit, tezga-d tazrawt taguffert iwakken ad tesseḥyu, ad tessemlili Taɣẓunt taɛlalt neɣ KUNDALINI, ɣas ma ur yezmer ara ad yessexser deg tudert "Aɣerrabu n Hermes Trimegisto", Rebbi Aqdim tlata tikwal "IBIS DE THOTH".

Nmeslay-d gar izenzanen i wid i izemren ad fehmen. Win yesɛan afehem, ad yefhem, acku daɣen tella tussna.

Tánctrios Iberkanen d ugaren, nutni tesseḥyu, tessemlili Uẓerẓur Aɛefyan KUNDARTIGUADOR, Taɣẓunt teɛǧiben n Edén, mi ara xedmen deg yidles-nsen tajeṛmumt ur yettwasemmaḥen n ussexser n "Ccṛab Uḥric".
