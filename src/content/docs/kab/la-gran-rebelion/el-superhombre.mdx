Yenna-d yiwen uqanun n Anawak: "Ixellafen xelqen irgazen seg wesɣar, mi ten-xelqen, rsen-ten d teẓri"; syin yerna-d: "Mačči d irgazen akk ara yaweḍ ad yersen d teẓri".

War ccekk, ayen yellan d amezwaru i ɣ-ilaq d axleq n urgaz uqbel ad nessiweḍ ad n-ners d tideț.

Aɣersiw amsekkel yettusemman d argaz sɣelt, mačči d argaz.

Ma nmuqel urgaz s uɣersiw amsekkel, ad nezmer ad nessenṣeb ayen yellan d tideț, ɣas akken aɣersiw amsekkel icuba urgaz s tɣara, s taɣessa, d amgerrad aṭas s tikli.

Aɣbel, akk ssnen dɣel, țḥesiben iman-nsen d irgazen, țsemmin iman-nsen akka.

Yal ass numen belli argaz d agellid n lxelq; aɣersiw amsekkel ar ass-a ur d-iban ara ula d agellid ɣef yiman-is; ma yella ur yelli d agellid ɣef tikli-ines, s taɣessa, ma ur yezmir ara ad ț-yawi ɣef lebɣi, amek ara yaweḍ ad yḥekkem ɣef ugama?

Ur nezmir ara ad neqbel argaz yuɣalen d akli, ur yezmir ara ad yeḥkem ɣef yiman-is yuɣalen d amurar n tezmert timedmamin n ugama.

Neɣ ad yili urgaz d agellid n tegnit neɣ ur yelli; deg umeggaru n wanwa-agi, iban-d ayen yellan d tideț ur newwiḍ ara ɣer tɣara n urgaz.

Deg tɣemmaṛ tisselmunin n uɣersiw amsekkel, iṭij yesserwes iẓuran i urgaz.

Ayen ibanen, iẓuran-agi zemren ad ten-yesnerni neɣ ad ten-negger maḍalen.

Ma nebɣa iẓuran-agi ad snernin, ilaq ad nɛiwen s جهود i yexdem iṭij akken ad yexleq irgazen.

Argaz yellan d aḥeqqani ilaq ad ixdem s tezmert s lebɣi n usuffeɣ seg yiman-is iɣewwan ur nebɣi deg umeẓẓuɣ-nneɣ.

Ma yella argaz yellan d aḥeqqani ur yessuffeɣ ara seg yiman-is iɣewwan-agi, ad iɣelleḍ, ad yuɣal d tiṭṭ n Tmeṭṭut Tigrunit, d aɣlati.

Argaz ara ixedmen tideț ɣef yiman-is s lebɣi n usekker n tmuɣli, ad yezmer ad yers d teẓri.

Ayen yettbanen, argaz aṭijan yersen d teẓri, yuɣal s tidet d azref-is d ARGAS UWESSAR.

Ur d-yeshel ara ad taweḍ ɣer ARGAS UWESSAR. War ccekk, abrid ara yawin ɣer ARGAS UWESSAR yugar lxir d ccer.

Ayen yelhan d ayen i aɣ-yufan, ayen diri d ayen ur aɣ-yufan ara. Gar tẓuriwin n usefru, yeffer-d daɣen ddin. Llan aṭas n lɛali deg umeɣlal, llan aṭas n diri deg uɛeqli.

Abrid ara yawin ɣer ARGAS UWESSAR d abrid n wudem n tmesṭart; abrid-agi yeččur d aɣbel deg daxel d berra.

Ccer d aɣbel, lxir daɣen d aɣbel; abrid ara icemmuɛen yugar lxir d ccer, d axatar aṭas.

Yal aɣanun n tikli yezmer ad aɣ-iḥbes deg umseɛy ɣer ARGAS UWESSAR. Taɣḍect ɣer ass-nni neɣ ass-nni, ɣer tmuɣliwin-nni neɣ tin, yezmer ad aɣ-iḥbes deg ubrid ara yawin ɣer ARGAS UWESSAR.

Iẓuran, tikliwin, ɣas akken yellan d ukyisen, ma yella ṭṭfen deg usɛeddi, deg uḥellil, deg tmuɣli-nni neɣ tin, yezmer ad aɣ-ɛeweq deg uzzal ɣer ARGAS UWESSAR.

ARGAS UWESSAR yessen ayen yelhan deg wayen diri, d ayen diri deg wayen yelhan; yeṭṭef asif n telwit tagrunit, yugar lxir d ccer.

ARGAS UWESSAR, mi yeṣfa deg yiman-is akk azal yellan d ilhan d diri, yuɣal d ayen ur yefhim yiwen, d uzzal, d tmes n teẓri tagrunit n tudert tețfeǧğiǧ deg udem n Musa.

Deg yal tzeḍɣa n ubrid, kra n umexluq yețfekkir yefka tiyita-s i ARGAS UWESSAR, maca netta yettkemmil abrid-is yugar lebɣi yelhan n imexlaqen.

Ayen i d-nnan medden deg tẓegwa iqeddsen n lemɛabed, yesɛa aṭas n ccbaḥa, maca ARGAS UWESSAR yugar ayen i d-nnan medden n usɛeddi.

ARGAS UWESSAR d uzzal, awal-is d rrɛud ara yettuxriɛen tiẓẓawin n lxir d ccer.

ARGAS UWESSAR yețfeǧğiǧ deg ṭṭlam, maca ṭṭlam yuker ARGAS UWESSAR.

Aṭas n medden țsemmin ARGAS UWESSAR d awɛir ɣef wayen ur yettmuqul deg umahil n laman, deg tmeslayin n usɛeddi, deg tikli yelhan n irgazen yellan d imuqranen.

Medden kukren ARGAS UWESSAR, țșemmiren-t gar imexmaren ɣef ayen ur t-fhimen, ɣef ayen țḥesiben-t, țmuqulen-t s allen-nsen s wayen i țḥesiben d ukyisen, ɣas akken yella d aɣlal.

ARGAS UWESSAR d lehma ara yeɣlin ɣef iweɛraten neɣ d țfeǧğiǧ n ayen ur nettfhem, ara yettḍalen syin ad iruḥ deg uɣrib.

ARGAS UWESSAR mačči d ukyisen neɣ d awɛir, yugar tkyessi d tɛewwiṛt; maca medden țsemmin-t d ukyisen neɣ d awɛir.

ARGAS UWESSAR yețfeǧğiǧ kra n tukrist gar ṭṭlam n ddunit-a, syin ad iruḥ maḍalen.

Deg daxel n ARGAS UWESSAR, yețfeǧğiǧ s lḥamu Lmasiḥ Azeggaɣ. Lmasiḥ amrewli, Agellid n Rebba Tamqqrant.
