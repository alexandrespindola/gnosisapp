Argaz d ayen yettidir: ma yella ur yexdim ara argaz tudert-is, ittezni lweqt-is deg lbaṭel.

Anagar ma nesseḥḥu tirebgaḍ timecṭuḥin i nella nerfed deg ulawen-nneɣ, ara nesken tudert-nneɣ d tifeɣɣelt.

Lmut d tuɣalt ɣer tazwara n tudert, s wazal n tuɣalt ɣur-s deg tmezgunt n tilawt tajdiṭ.

Tiɣriwin yemxalafen n usɣan-agrud d usɣan-ufellay ɛawjen ttḍerrent tasertit n tudert yeṭṭuqten, amur-a ulac fell-as ṣṣeḥ.

Tudert d ifilm; mi tefra tazzla-ines, neṣṣu ifilm-a deg teflukt-ines, nettawi-t yid-nneɣ ɣer dayem.

Tuɣalt tella, uṭṭan yella; mi nuɣal ɣer ddunit-a, nessers fell-as iɣef n tilawt, ifilm-a yellan, tudert-a yellan.

Nezmer ad nerr tasertit n tilawin yeṭṭuqten; maca ulac tudert yeṭṭuqten acku ifilm yella.

Amdan yesɛa tlata n tmerna deg tiddi tilellit d tsɛa u sebɛin deg tmerna tiddi yeṭṭuqten gar yiwaṭen.

Mi tuɣal tlata n tmerna deg tiddi tilellit tesseɣleb akk tama n tuccda yebɣan ad tegrer; bla ccekk nenkemmel deg tẓur-nneɣ n yeglen.

Tɣimit temgarrad; ulac ass-a n uzekka ɣer tɣimit n win yemmuten; taneggarut-a tettḍefṛu ɣer tama s wudem yessersen deg yeqbur neɣ deg temqebbert.

Deg umedluk-a kan yella yessers kan amur amecṭuh deg tiddi tilellit; aya yessegza i yezmer ad yemsewwiɛef d tezdeg deg ul.

Iwaṭen yemxalafen yettraḥen ttḍerren yezzi ɣer umedluk-a, tettraḥen yettasen s tilelli deg yal tama, bɣan ad ksen deg uɣbalu n tɣimit maca aya ur yezmir ara ad yili ɣer yiman-nsen acku ur sen-yexdem ara tɣimit tajdiṭ.

Ilaq ad tḥeqqeḍ belli tɣimit d tazmart d wugur deg tmazɣa d lweqt.

Yettwaru belli tɣimit ad tesken deg uqbel sebɛa iseggasen deg umecṭuh, ma d wamek tessaweḍ tssebded tesseǧhed s tɣuri.

Iwaṭen bdan ad ssexdaɛen deg uɣbalu n tɣimit ɣer tama ɣer tama s wudem tɣimit tajdiṭ tessewjed.

Lmut d asseḥḥu n wuffar, mi tefra temlilit n tusnakt, ayen kan yettkemmilen d wazalen (aya d iwaṭen iǧehden d imcḥal, yesɛan lfayda d ulac lfayda, yesɛan ṣṣeḥ d yesɛan leɣlaḍ).

Wazalen deg tafat tusdadt ttɛeggilin d ttḍeggiren gar-asen s ccariɛa n tɣimit yessawaḍen akk.

Nekkni d tiqqa tusnakin deg tallunt yellan nesseqeddem ɣer idles yessawaḍen i tdukli n wazalen.

Deg tɣimit n amdan deg yal yiwen seg-neɣ yella dayem wazalen-a yeqqimen d lsas i usun n tɣimiwin yettɛawaden.

Akk tuɣal ad tddu am akken i tḍerru maca aẓru neɣ tanzagt n wemdan nneɣ yezwaren.

Am akken deg yal yiwen seg-neɣ yella aṭas n yiwaṭen n tudert yezwaren, nezmer ad nini s wudem yessenṭaq belli yal yiwen deg wid-akken d amdan yemgarraden.

Aya yesɛedday-aɣ ad nefhem belli deg yal yiwen seg-neɣ yettidir aṭas n imdanen yellan yesɛan tasekkesin yemxalafen.

Deg tɣimit n umakar yella tighri tella n tidet n yimakaren; deg tɣimit n umseqqiɛ yella akk tirɣi n yiqettalen; deg tɣimit n umxelluɛ yella deg tama n tmezla; deg tɣimit n yal tawtemt yella taɣiwant meqqren.

Yal yiwen deg wigi n imdanen yettidir deg tɣimit-nneɣ nerfed, yesɛa lemɣawen-is d tasekkesin-is.

Aɣref yettidir deg wemdan, imdanen ttidirin deg wemdan; aya d win yerzan, ur nettwaraɣ.

Ayen yexṣen akk deg aya d belli yal yiwen seg wigi n imdanen neɣ iwaṭen deg-neɣ yettidir, yellan yettwaǧǧed seg tilawin zik d yesɛa tasekkesin yettwaǧǧden.

Iwaṭ-a yellan deg tmettiyezda iɛeddan yesɛa aɛyyez deg wessar deg leɛmer tlata iseggasen, deg tmettiyezda tajdiṭ ad yettraǧu leɛmer-a akken ad yebyan, mi tewweḍ ad iqelleb aɛyyez-ines, ad iṭṭef tameɣra telebɛa yellan yid-es, ad yesseqreb s telefon yellan yid-es ad iǧǧal ad d-tuɣal tirɣi n temlilit.

Iwaṭ-a deg leɛmer rebɛa iseggasen yesɛa tumert ɣef tɣawsiwin, deg tilawt tajdiṭ ad yettraǧu leɛmer-a akken ad yerreɣ ad yefraḥ.

Iwaṭ-a deg leɛmer n ɣesrem d semmus iseggasen yenneɛtab yid-en deg uɣerbaz neɣ deg ubaru, ad yettraǧu deg tilawt tajdiṭ leɛmer tajdiṭ deg ɣesrem d semmus iseggasen akken ad iqelleb iɛdawen-is ad yerreɣ tirɣi.

Ttraɣen gar-asen iwaṭen n yiwen yeqqen s telefon, mi tewweḍ ad msemlilen akken ad rren amekken ameskan yellan.

Aya d tidet usun n ccariɛa n Tuɣalt, aya d lmeḥna n tudert.

S idis n azzaren n yiseggasen imdanen yemxalafen ttɣalen akken ad ṣṣen tirɣi n lmeḥnat, d tirwi d lmeḥnat.

Tɣimit n amdan ulac ayen yexṣen am uɣbalu ɣer tqeddicin-a iwaṭen yesɛan tasekkesin meqqren.

Ayen yexṣen akk deg tirwi-a d belli akk tasekkesin-a n uɣref yellan deg ulawen-nneɣ ttwafren akka win yettfhamen yella kra idles.

Tɣimit-nneɣ n amdan deg usun-a tebban ɣer uɣrub yettwaɣɣaz s aṭas n yessan.

Tudert meqqren n tuɣalt teṭṭuqten tilawt ulac amek ttwbeddelen.

Ur nezmir ara ad nerr tizerwi d tirwi d lmeḥnat n tudert ɣef yefṛun n tilawt, ma ulac usḥissi.

Imesḥussi n akk usunen-a d iwaṭen yellan deg ulawen-nneɣ nerfed d yettwaǧǧden seg tilawin zik.

Ma nessaḥḥu ɣer iwaṭen n ẓẓɣeb, usunen n terwi n lbaṭel ad fukkun s tɣawsa ttwazwazen.

Ma nerẓeq ɣer igwmaten n uɣerɣur, lemɣawen-a ad fukkun akk.

Ma neɣrez ɣer iwaṭen n tneffust, usunen n tmezla d uɛewwuz ad fukkun.

Ma nerẓeq ɣer iɣsan igwmaten n tneqqa, temliliyin-nneɣ ad fukkun.

Ma nenɣa ɣer iwaṭen n zzux, n temɣer, n temɣer, n tewwit n yiman, usunen d ixettisen-a ad fukkun.

Ma nesseḥḥu seg ulawen-nneɣ ayen yexṣen deg tneẓẓi, n ur ttṣebbiɛ d teɣzi, usunen ttwagernen n tlata-a ulac amek ad ttwarrin acku ulac imesḥussi.

Ma nesrefref iwaṭen yetthuyyujen n tɣerɣi, n tezzyat, ad fukkun ifaten, timensiwin, atg...acku ulac imesḥussi.

Am akken iwaṭen-a ttɛeddin deg yeqqnen yemxalafen deg tidi n win yellan, iɛeggel ad nettwali llsas-nsen, tamhelt-nsen d tamhelt n yesiḥen ad nesɛeddi akk ar lmut n yiman-nneɣ d tilelli taneggarut.

Ad nɣer ɣer Lmasiḥ yellan, ad nɣer ɣer Lmasiḥ yesɛan llsas mi yettwaǧǧed ad nekkil deg ulawen-nneɣ beddel meqqren d taggara; aya ayen ara nɣer deg ixfawen-a yettwasan.
