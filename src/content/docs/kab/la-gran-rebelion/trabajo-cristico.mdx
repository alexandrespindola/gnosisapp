Ɛisa aɣarim yeffeɣ-d seg daxel deg umahil i d-yeqqnen ɣer usenneɛ n Nnefsi Apsikulujikan.

Awal-agi yessefk ad yettwabder, Ɛisa aɣarim iteffeɣ-d anagar deg tallit n ucengu n tɣermiwin-nneɣ i d-yeṭṭfen ɣer tmerẓiwt-nneɣ d tlufa-nneɣ i d-nebɣan.

Tayri n tmes Ɛisawi d tukkist i d-yeqqnen ɣer tudert-nneɣ.

Ɛisa aɣarim yettḍafar usligen taswiɛt n wallalen-nneɣ n lɛeqliya, tamusni, timhezzt, tagnawt, timhezzt-nneɣ n sex.

Ulac ccek fell-as, Ɛisa aɣarim d amsellek-nneɣ aɣarim usligen.

Netta d imseddaren ɣer daxel-nneɣ ad yemmeslay fell-as am umisedder ara yilin, am d asafi ad yemmeslay fell-as am umisafi ara yilin, am d aɛeddel ad yemmeslay fell-as am umaɛeddel ara yilin.

Ayagi icuba ɣer teɣdemt n tafat. Ma nesseqdec tilist n luṛ akken ma yella ad yeḍher d luṛ ma nesseqdec tilist n uzeggaɣ ad nẓer ayen ma yella d azeggaɣ.

Netta ɣas akken d amellal, ɣer berra yal yiwen ad t-iẓer seg tilist apsikulujik i s-iẓer, ɣef ayagi i d-yenna lɣaci m'ara t-ẓren ur t-ttẓernen.

Akken ad yeddukel ɣer wallalen-nneɣ apsikulujikan, Mass n tkemlel yettɛeqqab acḥal.

Yuɣal d argaz ger yirgazen, ad yeggri ɣer cɣal d aṭas d ad iṭṭef deg tukkitin-nni ur n-ttusemman.

Tukkit d tmes, azref ɣef tukkit d tafat.

Amezwaru ad yelmed ad yidir deg tagnit n lɣerḍ, akka tettwaru, ayagi ssnen-t lkimiyen.

Amezwaru ad yelḥu s lḥerz deg ubrid n useɣẓen n tgerrest, ɣer tama-s yal yiwen seg ubrid-nni aɛeẓẓan illa ubuɛeccac yewḥecen.

Deg ubrid-nni aɛeẓẓan n usenneɛ n Nnefsi illa ubrid n usenneɛ i d-yekkan seg ubrid anemlay.

Awal-agi yessefk ad yettwabder seg ubrid-nni n useɣẓen n tgerrest d-ffɣen waṭas n yiberdan ur nessawan ɣer wara; kra seg-sen ttawin-aɣ ɣer ubuɛeccac d tlemmast.

Illa ubrid i zemren ad yuɣal d agellid n umalu neɣ n ugama-nni, maca ur n-ttawin ara ɣer taɣdemt n Baba Amenzawi Amegdul.

Illa ubrid yettaṭṭfen ulawen, yellan d usafi usligen, ur n-ttusemman, maca zemren ad awin ɣer tukka i d-yekkan seg umaḍal.

Deg umahil n usenneɛ n Nnefsi neḥwaj ad nezdem ɣer Ɛisa Aɣarim.

Kra n temsal i d-yeɣellten ad n-fsen; akka kan; ubrid yettruḥu deg uḥeffaf ur n-ttusemman ur nessen anida id iwakken ad yemmeɣ; anagar tṭeɛa ɣer Ɛisa Aɣarim d Baba yellan deg uffer, ttawin-aɣ ɣer uɛqel deg leḥwayeǧ-nni.

Ubrid n useɣẓen n tgerrest yettwaccen s lɣerḍ deg daxel d berra.

Tizdeggeɣt-nni yemmalen ulac fayda deg-s; tizdeggeɣt d aqeddac n leɛwayed; n tallit; n umkan.

Ayen yellan d tizdeggeɣt deg talliyin iɛeddan tura yuɣal d leɛyub; ayen yellan d tizdeggeɣt deg tallit n wass-a tura yezmer ad yuɣal d leɛyub. Ayen yellan d tizdeggeɣt deg tmurt-a deg tmurt-nni yuɣal d leɛyub, atg.

Deg umahil n usenneɛ n Nnefsi iɛedda-d m'ara nessin diri-yaɣ ad nruḥ akken ilaq, maca ad naruḥ s diri.

Beddel d illa-t ger tefriqt-nneɣ deg teɣermiwin-nneɣ, maca lɣaci-nni iɛerqen deg tezmert qqimen deg yizri; qqimen deg wakud d yexlaɛ-d ɣer yiffus-nneɣ akken ad nexdem tasertit apsikulujikat d tbeddilt.

Lɣaci ur ttaṭṭafen ara tbeddilt n umezwaru; bɣan ad qqimen meḍlen deg yiḍelli.

Abeddel-nni ara yexdem umezwaru ad t-ssersen s leɛyub.

Ma nẓer leḥwayeǧ seg taɣmert-a ɣer tafat n umahil Ɛisawi, nezmer ad nẓer s tuẓẓma lɣella-nni n waṭas n yiḍrisen n tizdeggeɣ i yettwarun deg umaḍal.

Ulac ccek fell-as, Ɛisa yemmalen, maca yeffer deg ul n wargaz anemlay; akken ad yeddukel ɣer wallalen-nneɣ apsikulujikan, maca yettwassen kan i lɣaci, s tideț yettwassemma d amcum, d aleɛyub d amcum.

Yeḍher-d akken d lɣaci ḥemmlen Ɛisa maca yettawin fell-as izenqen.

Awal-agi yessefk ad yettwabder, lɣaci-nni iɣeflen d wid yettgen ṭṭes bɣan kan Ɛisa n usḍar, antrupumorfikan, n tselyatin d ufrag ur yemmuten, iwumi zemren ad sersen tizdeggeɣ-nsen i yettɛeqqaben d leɛwayed-nsen d tuggdiwin-nsen.

Lɣaci ur zemren ad ttalin Ɛisa Aɣarim deg ul n wargaz; lɣaci ḥemmlen kan tselɣa n Ɛisa kan.

M'ara yemmeslay yiwen ɣer lɣaci, m'ara yemmeslay yiwen ɣer usideg amezzayan n Ɛisa aqlalay; n Ɛisa azeggaɣ, n Ɛisa yeqqimen, s tikkelt ad yefren izenqen am agi: umceɛɛeɛ, abidl, amcum, usermem, asakrilegu, atg.

Akkagi lɣaci, daymen d iɣeflen; daymen ttgen ṭṭes. Tura ad nefhem ɣef acu i d-yenna Ɛisa deg yiffus n Galguta: Baba ɛfu-yasen acku ur ssinen ara ayen xedmen!

Ɛisa deg iman-is ɣas akken d yiwen, yeḍher am waṭas; ɣef ayagi i d-yenna akken d taḥidant tuslift tkemlel. I win yesnen, awal yefka-d taɣmert; ulac win t-yennan, ulac win ara t-yinin, anagar win yilin yettwassenfud.

Akken ad t-id-nessenfed d ayen i d-ilaqen deg umahil n ussegzel n Nnefsi.

Mass n tkemlel ixeddem deg-neɣ akken ad neɛreḍ ɣef leɛqel deg umahil ɣef iman-nneɣ.

Yeɣzif akk waɛeẓẓan umahil ara yexdem Ɛisa Aɣarim deg tagnawt-nneɣ.

S tideț Mass-nneɣ aɣarim ad yidir akk tawuri-ines deg tama-nni i deg illa ul-nneɣ.

Yettwaru: "S utaf yettḥellilen ɣer Ṛebbi d uɣeddu yettunefken". Yettwaru daɣen: "ɛiwen iman-ik ad k-ɛiwneɣ".

Tḥellilt ɣer Yemma Taɣurart Kundalini d ayen ila-t m'ara d-ffeɣ aggrege apsikulujikan ur n-ttwabɣan, maca Ɛisa Aɣarim deg tama taqdimt akk n iman-iw, ixeddem s uɛqel akken illa urmud-nni i yeṭṭef ɣef tuyat-is.
