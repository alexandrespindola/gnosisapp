ZI 20 FUṚAṚ AR 21 MEƔRES

Newweḍ ɣer TGIḌ-YUMMAT n Tusnakt-Tagnayt ta Maṣrit, ilel aɣelben n ISELMAN, tillas tigellilin ur yesɛin tilisa n tallunt TAGENSUZT TAYEẒLANT; d ameggi amezwaru n tlemmast anda tidyanin tɣellayent ɣef UREƔ n RHIN neɣ TIMES n UMEYYAZ UWSIAN UDLALAN.

ISELMAN yettumeggez s ṣṣwab s sin ISELMAN; aslem, Aslem, d SOMA n TUDERTIWIN N ISIS. ASLEM D AMATAR YEDDREN N UMASIḤIYEN AGLUSIYANEN IZEWWARANEN.

Sin ISELMAN n ISELMAN yettwabḍanen s asekkil d ameqqran maḍi deg unamek AGLUSIYAN, ttilin-d i sin YELMAWEN n ILYASEN IZEWWARAN yettwakksen gar waman ineqqren n TGIḌ-YUMMAT.

Nesruḥ-d yakan, deg yixefawen yezrin belli IMI, AZAL, ATMAN, yesɛa sin YELMAWEN: yiwen D AMEṬṬUṬ, wayeḍ D AMESSAS.

Nesruḥ-d yakan belli ALMA TANEFSAN, BUDDHI, d ametṭuṭ. Nenna-d yakan nettuɣal a d-nini, belli ALMA URGHAZ, MANAS ayfella, d amessas.

Tameṭṭut iqeddsen, TAZEWWAJ UWSIAN D IMAWER, yettumeggez daymen s sin iselman yettwabḍanen s asekkil; wagi aneggaru d AZAL ATMAN.

Tameṭṭut iqeddsen, sin ISELMAN IMAWER, xeddmen gar waman n tlemmast mi d-tecceɛ talla n MAHANVANTARA.

Sin ISELMAN ur nelli ara ad ngen fell-asen xeddmen seddaw ufus n ATMAN, mi d-teḍher tsafa n tallit n tɣuri.

Acu kan, yelha ad nmekti belli ISIS yakked OSIRIS ur yezmir ara ad xedmen daymen deg UẒU UWSIAN, bla MERCURIO ifazen n TFLIST TAFFESSAST. Deg MERCURIO ADIMAYAN-A tella tigawt n yal tazmert.

Taggugt yakked asekkil yemḥeqen d aqezzaz amatar aneggaru, d tdukli iqeddsen maḍi n YETTUT AMEṬṬUṬ yakked UMESSAS IMAWER; TAMEKLA n imugar deg UMELMAD ABṚIDAN, ur yettwaggaden ara akked UWSIAN.

Seg TMEṬṬUT-TAMEQQRANT-TALLUNT d-tettili TAMUNA, AZAL. Seg ULEL AMEQQRAN d-kkren ILYASEN akken ad xedmen deg tsafa n MAHANVANTARA.

AMAN d ameggi ametṭuṭ n yal ayen i d-yettwaxelqen, anda d-tekka MATER LATINA, akked usekkil M, d ameqqran maḍi UWSIAN.

Deg UMASIḤIYEN AGLUSIYANEN d MARÍA tagi d ISIS s timmad-is, YUMMAT N TESNANT, YUMMAT-TALLUNT IMAWER, aman ineqqren n tlemmast.

Awal MARÍA yettwabḍa ɣef sin yigiten; amezwaru d MAR, ara aɣ-d-imektin ilel aɣelben n ISELMAN. Wis sin d ÍA, d amsefrak n IO (iiioooo), isem ucbiḥ n YUMMAT-TALLUNT, taggugt n WALA, anda yal ayen d-yettili, ɣur-s yal ayen yettɣimi; Yiwen, YIWWEN YEKKA deg umenɣi n tallunt, deffir yiḍ n Pralaya tameqqrant neɣ asenger.

Ssedfant-d Aman ayfella seg wid idellan, yella-d tafat, anamek-ines, d-yekka deg tudert UẒU d-yessalayen TESNANT, MMI, akked tudert-a tekkes ameggi amessawal ɣer ṬEJJA, ayiɣef yella deg tlemmast n udur-nneɣ aneẓɣar, am ul deg teɣdemt-nneɣ.

Tidyanin tijerdiwin n ṬEJJA d TIMES TIJELMIST YEDDREN ayyettwaḍfen deg tlemmast n yal akal, yettwabna deg ul n yal yiwen seg-sen.

Tafat-nni akk, tudert-nni akk, ttilint-d s sebɛa yelman ɣer zdat n uqerru, deg UDEBBAR-UL n yal yiwen seg sebɛa yikawken n UDUR ANEẒƔAR.

Axeddim n usefsex n waman n waman, yettunefk i tmeṭṭut iqeddsen. Yal yiwen seg SEBƐA YELMAN ɣer zdat n UQERRU, d-yekka seg WALI-IS, ɣer tmeṭṭut iqeddsen n ISELMAN akken ad xedmen deg TSAFA n tɣuri yakked tazmert n KRIYASAKTY, tazmert n wawal yeḍṛen, tazmert n URIR akked YUGA.

TIRIT n TIRAT, ASSAɣ ASEKLAN n tmes tagergart gar URGAZ IMAWER yakked TEKKALT UWSIAN, d ilɣanen akken ad ssefsixen AMAN AYFELLA seg AMAN IDELLAN.

Deg UXEDDIM-A tella MAITHUNA TASEMMESLANT; KRIYASAKTY, AWAL D-IYETTGERRAEN.

Yettarra-d TIMES yakked tbeddel aman tessefsix ayfella seg wid idellan.

Sin Iselman sseqdɛen imir-nni times-nni yakked aman-nni ayfella yettwabeddlen ɣef waman n CAOS ɣef tewmat takusmayt neɣ tamatrayt i yimagaz, ɣef yiɣsan yettgennen n tella akked telfut tudert.

Axeddim akk yettwaxeddem s tallilt n WAWAL akked URIR akked YUGA.

Deg tazwara, TESNANT d tafessast, deffir-s tettwaḍfar s tewmat, tettedda ɣef tallit uffiren n TSERSIWET taɣelnawt.

Llan imelyunen n TESNANIWIN deg tallunt tigellilit, gar uqerru n YUMMAT-TALLUNT.

Kra TESNANIWIN tteddunt d-kkrent seg PRALAYA, d-teffɣent seg gar waman ineqqren n ISELMAN, wiyaḍ llan deg umenɣi, wiyaḍ llan ttwattun gar waman yettumiren.

Ur yezmir ara ISIS yakked OSIRIS ad xedmen bla MERCURIO ADIMAYAN, sin iselman IMAWER, hemmlen, ẓɣaren, ddren daymen d-ttgerren ttɣalent ad gerren.

Aslem d amatar yeqqden maḍi n UGASTIKIZM UMASIḤI AYZEWWARU. D leɛyub imelyanen n inelmaden n tuffra ad ttun AGLUSIS n uyslem.

Deg ukal-nneɣ ggen sebɛa YEMDANEN yakked tɣessiwin tigensiyin akked sebɛa akk, aneggaru d nukkni, yiwen ayefraren ɣef tunɣilt AGLUSIS.

Sḍis YEMDANEN wiyaḍ dddren deg ussan n JINAS, deg TELLINT TISSAƐT, yakan deg tlemmast n UKAL, neɣ deg tamaṭ n tiɣaltin akked temnaṭin n JINAS.

LAƐMER n ISELMAN ur yella ara ad yili d aferru am akken yella. Taggalt taɣelnawt n uferru n PISCIANO teḍra ɣef kra n yimeggiyen imellalen ayiɛerḍen AGLUSIS ttininn-d kra TISENGAWIN AGNÓSTICAS neɣ ANTI-GNÓSTICAS, ttettxṣaren ASLEM, ttserrḥen TIRGA UZMAM, ttekksen taɣemsa i yemdakel deg tewmat.

Ad nemmekti LUCIU newweḍ ɣer tamdint n HYPATIA, yezdeɣ deffir-s deg uxxam n MILON, anda tmeṭṭut-is PÁNFILA d aseḥḥar yeqqimen. Yeffeɣ uḍeyyer LUCIU ad yaɣ aslem (EL ICTUS, AMATAR N UMASIḤIYEN AGLUSIYANEN D-IYETTELFUTEN, ASLEM, ASLEM, SOMA, n TUDERTIWIN N ISIS).

Iɣewwaten zzenzen-as ɣef tlata d wessin dinar s uḥessir yezgaren, ayen yezzaren zzenzen ɣef miya n uskad, tigejdit yeqqimen anda yettwatteɣ imeqqren i UMASIḤIYEN AGLUSIYANEN D-IYETTELFUTEN.

Aḍris n UMASIḤIYEN AGNÓSTICO neɣ ANTI-GNÓSTICO, d DIALÉCTICA MATERIALISTA MARXISTA.

TUGGNA ɣer UGLUSIS d TMATERIA YEQQIMEN bla YIRBAḌ akked bla uselɣu.

Yezmer ad yettuseɣtel belli laɛmer n ISELMAN yeffraren ɣef UAGNÓSTICO. Aɛered ɣer AGLUSIS d lɣella yeqqsen maḍi n laɛmer n ISELMAN.

JESÚS UMSIḤ yakked tneẓẓi-ines n iɣewwaten, IBEDD laɛmer ara yilin n teẓɣert tameqqrant.

JESÚS yakked TNEẒẒI-INES N URBEƐTA YELMAWEN AGLUSIYANEN ussent-d abrid yesḥiḥen i laɛmer n ISELMAN, AGLUSIS, tmusni n UYSLEM.

D lehḍa ad ttwarun akk idlisen yeqqden n SANTA AGLUSIS ɣlin yettwaḥerqen akked ttserrḥen amatar yeqqden n uyslem.

ASEGDAW. Deg usekkil n ISELMAN ilaq ad tesselɣeḍ saɛa yal ass. Ad nemmekti deg tazwara yella UẒU yakked UẒU yella yakked YIRBAḌ yakked UẒU yella YIRBAḌ.

Deg ussan izewwran sebɛa timsal n tẓuri ttseflalayen deg tɣessiwt taɣelnawt seg uqerru ar yidaḍen, akked tura ilaq ad nerr sebɛa tifina deg terba taɣelnawt n tɣessiwt-nneɣ, akken ad nerr tazmert yeḍṛen.

Taslent "I" tessenṭiq tɣerɣart PINEAL akked PITUITARIA; snat tɣerɣarin-a n uqerru yettwabḍanen s uɛaqeb neɣ aqebdan d ifessasen maḍi, yakan tḥeqlen deg yemettanen.

PINEAL tella deg amur ayfella n umeyyeẓ akked pituitaria deg uɣbalu ameqran gar snat tiẓra.

Yal yiwen seg snat tɣerɣarin-a yesɛa AURA-INES UWSIAN akked mi ttwaḍfen snat AURAS, yettwelfu USENSIW AMEKAN nwala ULTRA n yal ayen.

Taslent "E" tessenṭiq tɣerɣart tiroides ayiɣef yettarra YODO ABIOLÓGICO. Tɣerɣart-a tella deg tmara anda tettidir takrura n umeẓẓuɣ aseklan.

Taslent "O" tessenṭiq takrura n ul, tlemmast n UẒAYYAS, akked yal ajen n tizmert akken ad teffɣeḍ deg ASTRAL, ussan JINAS, atg.

Taslent "U" tessenṭiq UɣBALU ANEẒƔAR, yellan deg temnaṭ n tallunt. UɣBALU ANEẒƔAR-A d TLEMMAST TELEPÁTICA akked UMEYYEẒ UMEẒLIW.

Taslent "A" tessenṭiq takrura tasedrant ayiɣef d-nemmekti tudert-nneɣ yezrin.

Taslent "M", yettwakkes seg taslest s tikkelt, ttasselɣen yakked yemḍaren yeqqnen, bla teldiḍ imi, taɣect d-tettuffɣen imir-nni s nger d "M".

Taslent "M", tessenṭiq ENS SEMINIS, aman n tudert, MERCURIO n tflist taffessast.

Taslent "S" d aseɣbel amezdi akked isḥanen ayiɣef tettessenṭiq times deg nukkni.

Teqqimen deg tkerzi teɣdemt ilaq ad tesselɣeḍ I. E. 0. U. A. M. S. ttedduḍ taɣect n yal yiwen seg sebɛa taslatin-a seg uqerru ar yidaḍen.

Yelha ad ttenaffaseḍ deffir tesseffeɣḍ-d yiḍes yakked taɣect yeqqnen tessafses-itt, ḥettad tekkesḍ taneffast.

Asegdaw-a ilaq ad texeḍmeḍ yal ass akken ad telfuḍ tazmert tashilwit IMAWER.

ISELMAN yettweḥkem seg NEPTUNO, akawken n tuffra yettwaxedmen akked JÚPITER TONANTE, Baba n ILYASEN.

Uɣesru n Iselman d ESTAÑO n Júpiter; iẓra, AMATISTA, CORALES. Iselman ttweḥkmen iḍaren.

Inexdamen n Iselman s timmad d sin inegḍaren, kra warrac. Llan deg tẓuri tabiddit akked ssɣirent i sin tiẓẓuna neɣ ixeddimen. Inexdamen n Iselman yettunfaqen maḍi ad mmers, ddren am uyslem, deg yal ayen, acu kan ttwafaqen yal ayen s yessili azuran. Ttwattfen yal ayen, acu kan deg uɣerru tteɣẓayen akk ayen deg dunnit. Llan yeqqnen maḍi tijerdiwin, inzayen, ineqqren akked medden ur yezmir ad mmers.

Inexdamen n Iselman ssɣirent tameqqrant i tuffra, ɣef Iselman yettweḥkem seg NEPTUNO, akawken n ESOTERISMO.

Tilawin n ISELMAN llan d irqiqen maḍi, qqnent am ujerrid ifessen maḍi; inzayen, yettwaḍfen.

Ipiscianos ssɣiren ucziren imdanen, d ifraḥen, yettḥessisen, yettseḥbibiren s tẓuri.

Ameskal n Yiselman d ad yettɣem i tɣara, d tuggma, taneflit akked tamesɣelt i tudert.

Iselman yezmer ad newweḍ ḥettad teqqel tɣelt uḥaramen. Umeyyez n Yiselman yettwexxer gar tallunt taḍferrant neɣ tamerɣant, tɣara akked teɣẓiṭ i ayen d ilɣanen maḍi i tudert. Llan sin tilisa yakked tikkelt ddan seg tiliɣt am akken deg tiliɣt-nni. Urir n Yiselman kra n tikkal d yeqqsen, acu kan ttbeddilen kra n tikkal.

Mi tɣlin Yiselman deg tamesɣelt akked taneflit taɣelnawt, ttxelliqen ad d-wten seg tazzla n wasif n tudert, acu kan mi walaen timaḍit n tɣellist-nsen, sseqdɛen deg uɣerru urir-nsen n uzɣal imir-nni beddlen tallit akk n tudert-nsen.

Iselman n telɣa ayfella d AGLUSIYANEN deg miya n imiya, ssɣiren urir n uzɣal ur nzegdig ara akked tagergart tamellalit n uḥaramen yettuneḥkmen.

Telɣa ayfella n ISELMAN tteddu ILUMINADOS imeqqranen, MAESTROS, AVATARAS, RREBƐAT, INICIADOS, atg.

Telɣa idellan n ISELMAN yesɛa tayessert tqeddsen ɣer LUJURIA, tamsit, tɣara tameqqrant, tɣara.

Iselman ssɣiren i tiɣriwin, acu kan ur yal yiwen yezmir ad yeffeɣ. Iselman yesɛan imeyyaz ameqqran akked talɣa tigejdant.

Yettunfaq maḍi ad mmers i ISELMAN, anagar ISELMAN yezmer ad mmers i Iselman.

Yal ayen i medden imdanen yesɛa tigejda tameqqrant, i Yiselman ur yesɛa ara azal, acu kan d aserdawi, yettwattef ɣer wid.

Ayen yeqqsen maḍi i Inexdamen n Iselman, d yesla ad yettuseɣtel deg tayessa n terba, ɣef qrib daymen snat turitin tigejdiyin imeqqranin, ttrarant-iten deg abrid bla tuffɣa.

Telɣa ayFELLA n ISELMAN yakan ad yettuseɣfel ɣer timenzuyin-a yakked CASTO deg talɣa ABSOLUTA.

S timmad ttɛezzaben aṭas Yiselman yakked terba deg iseggassen-nsen izewwran.

Yettunfaq ad telfeḍ Iselman d-yesɛan afrah yakked terba deg iseggassen-nsen izewwran.

Telɣa idellan n tilawin n ISELMAN, tteɣlint deg taznatt akked tamsit.

Telɣa ayfella n tilawin n ISELMAN ur teɣli ara akka, am ujerrid ifessen maḍi, am ujerrid yesɛan taɣara n LOTO.
