## 22 YULYU ɣER 23 ƔUŠT

ANNIE BESANT teḥkay-d yiwet n temsalt ɣef USTAḌ NANAK i yuklal ad t-id-nekteb.

"D lǧemɛa ass-nni, mi d-yewweḍ lawan n ṣṣla, ruḥen bab-is d aqeddac-is ɣer lǧameɛ. Mi yebda lqari (AMUSLEM AƐEQQA), lmeṛẓuq d lɣaci-s ɣlin ɣef wudem, akken i d-yettarra UṚṚUṚ AMAHUMETANI, NANAK yeqqim yeddaɣ, ur yeqqim d akken, d asus. Mi tefra tẓallit, lmeṛẓuq yezmeɣ ɣef ulemẓi-nni yesteqsa-t s lɣeḍab: Acuɣer ur teḍfiṛeḍ ara igerdan n Uɣanim? D akeddab d ukennac i telliḍ. Ur ilaq ara ad d-tasedd ɣer da akken ad teqqimed am agerdas".

NANAK YERR-AS:

"Tɣellim ɣef wudem mi tella tmuɣli-nwen tetberriɛ di lecṛuf, acku tettxemmimeḍ ɣef ujemmeɛ n yisaffen seg CANDAR ur tettxemmimeḍ ara ad tɛaweddd tẓallit. Ɣef wayen yeɛnan AƐEQQA, yettɛawad igerdan n uɣelluy s usukkud, mi yettarra tixmimin-is ɣef uɣersiw i d-ilulen kra n wussan yezrin. Amek ara zalleɣ akk d lɣaci i yettɣellin s usukkud tɛawaden awal am tbeṭṭuṭ?"

"Iɛteṛ LMEṚZUQ belli s tidet yella yettxemmim deg ujemmeɛ yisaffen deg uɣerɣer n tẓallit. Ɣef wayen yeɛnan Lqari, yessken-d nnig deg ufus yesteqsa ulemẓi-nni s waṭas n yesteqsiyen".

S tidet ilaq ad neqqen ad NẒALLEɣ s tussna; win ara yaɣen ad yesdukel s tussna TaẒALLIT akk d UMERREK, ad yaweḍ ɣer leɛnaya YESƐAN ƐINI.

Lameɛna yella uswiren ad neqder belli llan TaẒALLIWIN yemgaraden dɣa tigemmi-nsent tga d yemgaraden.

Llan TaẒALLIWIN yettḍefren deg-sent d ssuter, maca mačči akk TaẒALLIWIN iteddun akk d ssuter.

Llan TaẒALLIWIN teqdamin aṭas i d-tegzem timessent ɣef wayen yedran CUKTI, yezmer ad nɛic kullec deg-sent ma nemmerked ɣef yal awal, ɣef yal tanɣa, s lebɣi n tidet s lefhama.

Baba-tneɣ Yella di igni d tarrayt TAƐEZGIT n tezmert TASEɣWIT ur nettɛeddi, maca yella uswiren ad neqder ɣer lqem dɣa s wayes kan lmeɛna iduren n yal awal, n yal tanɣa, n yal taɣawsa.

Baba-tneɣ yella di igni d taẓallit n ssuter, taẓallit akken ad nessawel akk d BABA yellan deg uffir. Baba-tneɣ Yella di igni yeddukel akk d UMERREK n lqem, yettarra tigemmi YESƐAN ƐINI ƔUR-S ƔER ƔUR.

IṚṚUṚAN IGUNASTIYEN, ISELƐADIWEN IMAẒẒALEN, d iberdan n tidet n TUSNA UFFIR, i win yessnen ad yemmerked, i wid i ten-yefhumen s wul.

Win yebɣan ad yeqdec iberdan n WUL ISEMMḌEN, ilaq ad yeẓẓel PRANA, TUDERT, TEZMER N TMEṬṬUT deg uɣaras akk d tmuɣli deg WUL.

Yella USWIREN ad neqqen ad nexdem s wul, ad nesers tmuɣli deg WUL AMANZU. Ugerruz n TILALIT yettɛeddi yal ass deg WUL AMAZWAR.

NANAK, USTAḌ i yesbedden DI LƐIQIDA SIKH deg tmurt yeẓẓilen n YIVIDAS, yesselmed iberdan n WUL.

NANAK yesselmed tegmat gar akk DI LƐIQIDA, IƔEṚBAN, iweẓlaɛen, iḍrisen, atg.

Mi nettaɛṛad akk DI LƐIQIDA neɣ s timmadit i kra n DI LƐIQIDA, neɣdeṛ ayen i d-tewwet TAZRIT n WUL.

Deg WUL AMANZU yella iṭṭef i akk DI LƐIQIDA, IWEẒLAƐEN, IḌRISEN, atg., atg., atg.

Akk DI LƐIQIDA d tiẓunin deg ufus ttwaxedmen deg ugerdas n wurɣu n TEẒṚI.

Iɣil-tneɣ IGUNASTIYEN ttexdimen-t lɣaci n akk DI LƐIQIDA, IƔEṚBAN, IWEẒLAƐEN, TIṚMIWIN TIRUḤANIYIN, atg., atg., atg.

Deg WUL AMANZU yella iṭṭef i akk DI LƐIQIDA, i akk leɛwayed. Ɛisa yeqqaṛ: "Deg wayen tḥemmlem yiwen waḍen, ad tesbuɣrem belli d Inelmaden-iw".

IḌRISEN SIKIYEN, am tid n akk DI LƐIQIDA, s tidet ur zemmer ad sen-fkeɣ azal.

Gar usikiyen yella OMKARA d ASELWAY AMAẒẒAL IṬFAD IḌERRU i d-yexdem igenni, akal, aman, kullec ayen yellan.

OMKARA d IṚUḤ AMAẒẒAL, UR YETTBAN ARA, UR NETTMAṬṬAF, ur yelli lebda n wussan, ur yelli taggara n wussan, tayri-ines teffeq CƐAṚṬA MṚAW-D-ƐEṚḌA TUDRIN, yessnen s telqey; yesselhu taɣawsa n wul.

"Tallast d taneflit-ik. IṬIJ d WAYYUR d tiftilin-ik. Iɣil n yetran d tiẓunin-ik. A Baba!. Tajeǧǧaɛt tayyurt n yimalayas d tibeɣra-k. Aḍu yeɣṛeb. Aɣersiw yemɣin yettarra-k iden, a taɣawsa!. I kečč, tidetti n umeɣnas, a win yegren deg tugdi!. ANATAL SHABDHA (TIƔRI TAKESSUMT) tessexlaq am tɣembatin-ik. Ur tesɛiḍ allen akken ma llan. Ur tesɛiḍ idaṛen akken ma llan. Ur tesɛiḍ tinzert akken ma llan. Aqeddic-ik amazwar yettwassen-aɣ. Tayri-ik, a lqeḍra! teffeq deg kullec. Seg akk yemdanen teffeq Tayri n Tayri-k. Seg asselmed n USTAḌ teffeq tayri-agi. D ARATI".

USTAḌ AMAZWAR NANAK, s lemɛawna akk d YUPANISHADAS, yefhem belli BRAHAMA (BABA), d YIWEN dɣa IṚEBBI ILAƐLIYEN gan kan d tiɣawsiwin-ines yesɛan azal meẓẓiyen, ademmiyen n TAẒṚIT TIBERKANIT.

GURÚ-DEVA d win yettwaḥḥden akk d BABA (BRAHAMA). D aswaḥ win yesɛan GURÚ-DEVA d imselmed d win yeselhu. Yettwaḥmed win i yufan USTAḌ n TNEFLIT.

Iberdan d iɣzizen, d iduqqsen d iwehḥalen. Tḥeqqeqet deg GURÚ-DEVA, imselmed, win yeselhu.

Deg WUL AMANZU ad naf HARI SER. Deg WUL AMANZU ad naf GURÚ-DEVA.

Tura ad nerr kra n temsal CIKIYEN ɣef Lebɣi ɣer GURÚ-DEVA.

"A NANAK! Qeder-it i GURÚ n tidet, win i tḥemmeḍ yettwaḥḥeden kečč akk kullec..."

"Mṛa-nneɣ yal ass ad rreɣ iman-iw ɣef GURÚ-iw i yi-yerran IṚEBBI deg wakud azelmaṭ".

"Ɣas akken ad yeǧǧeǧ CƐAṚṬA WAṬṬAS d WALF TIƔAWSIWIN, ad tɛebber nnig deg GURÚ tillas tberkanin".

"Ad yettwaḥmed GURÚ-iw YESSERƐAN i yessnen HARI (SER) dɣa yesselmed-aɣ ad nemɛamel am yimdukkal akk d iɛdawen".

"!A Sidi!. Sewjed-aɣ akk d tdukla n GURÚ-DEVA, akken ad nɛiwen akk yid-s, nukkni i yelqen, amek nessen ad nɛeddi armi nesselkeɣ".

"GURÚ-DEVA, GURÚ n tidet, d PARABRAHMAN Sidi Amellay. NANAK yella yettɣelliyen zdat GURÚ DEVA HARI".

Deg INDOSTÁN SAMYASIN n tixmimin yella yettqeddec GURÚ-DEVA n tidet, win yufan deg wul, win yqeddcen deg TLELLI N YIGU AYURAN.

Win yebɣan ad yessufeɣ AGU, akk d AYI, ilaq ad yesegder LƔEḌAB, ṬMEƐ, TMEṬṬUT, ISENƐANI, UƐEQQEL, AƐEQQAL, TIɣRI. Kan s wesegder akk tiɣewɣat-agi deg yal ASWIRE n Tɣellist, ad yettmettat AYI s tarrayt TERKAZT, telkemt dɣa taneggarut.

UMERREK deg isem n HARI (SER), yettarra-aɣ ad nɛic ayen yellan, ayen n tidet.

Yella uswiren ad neqqen ad NẒALLEɣ baba-tneɣ Yella di igni, ad neqqen ad nessawel akk d BRAHAMA (BABA) yellan deg uffir.

Kan yiwet baba-tneɣ Yella di igni yettwaẓallaen dɣa yettwaqedder s tussna yeddukel akk d UMERREK, d aqeddic AMAZWAR n tezmert taneggarut.

Kan yiwet baba-tneɣ Yella di igni yettwaẓallaen s useqsi ad iḍfer deg ssaɛa neɣ deg kra n ssaɛa.

Sakkin taẓallit ilaq ad neqqen ad nergagu deg tγersa n BABA dɣa tagi tga ad neqqen ad nemmerked, ad nesɛu tmuɣli nɣelli dɣa d asus, teɣreq deg kullec ayen yettxemmim, tergagu deg tγersa n BABA.

Mi TEƔLLIST tella tɣelli deg iɣil d teɣra, mi TEƔLLIST tella deg UɣUBU deg iɣil d teɣra, mi tmuɣli tella tyeɣreq seg UBƐADISMA, imiren teqqed ɣur-neɣ ayen IDHEṚ.

Yella uswiren ad NESREƔ tmuɣli n yal tikkelt ayen yettxemmim, ssiḥ, ssawalen, lemri, tugdi, atg., akken teqqed ɣur-neɣ tamussent n ayen YELLAN.

Tafɣert n UɣUBU, TAMUSSENT deg UɣUBU ITTEFRUSEN, gan kan imseɣlen mi TAMA, ALMA, BUDHATA, tettwareqqeq seg tbuṭult n uɣaras.

TAMA tella tetwatteṛ deg gar uḍuffan aqurar d ɣili, ttuga dɣa ur ttuga, ih dɣa la, ɣer lxiṛ dɣa ɣer lɣaci, yesɛan iɣil dɣa ur yesɛi iɣil.

Mi TEƔLLIST tella teɣli, mi TEƔLLIST tella deg UɣUBU, imiren TAMA tetteɣreq dɣa teqqed TAMUSSENT n ayen YELLAN deg UɣUBU ITTEFRUSEN.

NẒALLEƔ, i nelmad amellay akkin akk d tmuɣli tɣellit dɣa deg uɣubu, yeɣreq seg yal tikkelt ayen yettxemmim, tergagu deg tγersa n BABA: "Essuter dɣa ad tettunefk kečč, ssaɛ dɣa ad tettwaɣreq".

NẒALLEƔ tga ad tessawaled akk d IṚEBBI dɣa s tidet ilaq ad neqqen ad nessawel akk d BABA, akk d BRAHAMA.

WUL AMANZU d axxam n TAẒALLIT. Deg ugerruz yella ugerdas n SELUMUN.

Yella uswiren ad nẓalleɣ dɣa ad nemmerked lqem. Yella uswiren ad neqqen ad nseddel aɣersiw akken UMERREK ad yegemmed.

Uqbel ad nebdou Leɛwayed n TAẒALLIT dɣa UMERREK zdukklen, seddel aɣersiw.

Ad yekker UMNADI IGUNASTIYEN deg taɣiwant YESKUNEN IDURAN, yaɛni, yeǧǧed seg wul deg uqerreɛ neɣ deg tɣalt, idaṛen dɣa iɣalen d ireqqiɣen ɣer yeffus dɣa zelmeṭ, deg tagga n YETRI deg xemsa n yiserɣasen.

Tagga-agi n YETRI PENTAGONAL tga tameqqrant i taɣiwant-ines n lqem, maca yemdanen i ur zemren ad merkeden deg tagga-agi, imiren ad merkeden asersan aɣersiw deg TAɣIWANT N URGAN: taɣessart zdukklen, tisekkurt n idaṛen ttureqqeqen deg tagga n tayda, iɣalen tettuɣalen ɣer rrif ur tettwaẓẓilen, ttuɣɣalen ɣer lqem n tzunzaɣ.

Allen ilaq ad tettmedlent akken tiɣawsiwin n ddunit ur kem-teqqeden. Iḍes yeddukel akk d UMERREK ad yegemmed deg leɛnaya n UMERREK.

Yella uswiren ad nesɛu urǧan ad nseddel akk idmus deg uɣersiw dɣa sakkin ad nemsiddeg LƐINI deg tisekkurt n tinzert armi ad nḥess s telqey deg tmerriwt n wul deg uɣersiw n tilif, sakkin ad neddu akk d umeẓẓuɣ yesɛan azref armi ad nḥess tmerriwt n wul deg-s, sakkin ad nkemmel akk d ufus yesɛan azref, aḍaṛ yesɛan azref, aḍaṛ yesɛan zelmeṭ, ufus yesɛan zelmeṭ, ameẓẓuɣ yesɛan zelmeṭ dɣa nneɣ, aseqsi ad nḥess s telqey deg tmerriwt n wul deg yiwet yal uɣersiw ɣer win nemsiddegen deg-s LƐINI.

Aɣaras ɣef uɣersiw yebda akk d uɣaras ɣef tmerriwt. Tmerriwt n wul teɣli ad teqqed deg yiwet yal deg-s deg tmellt s lekmal deg tɣellist, maca IGUNASTIYEN yezmer ad tt-ḥessen s lebɣi deg yal amkan n uɣersiw, yaɛni tisekkurt n tinzert, yiwet tmeẓẓuɣt, yiwen uɣal, yiwen uḍaṛ, atg.

Tettwabayan s leɛwayed s tarrayt n temlilt ad teqqen ad testeqsi neɣ ad tleqqem tmerriwt, zemren ad testeqsin neɣ ad lhewwlen tulɣaɣ n wul.

Aɣaras ɣef tulɣaɣ n wul ur yezmer ad yeqqed seg tidmus n wul, maca yettɛelleq deg uɣaras ɣef tmerriwt. Tagi tga ɣer tallast, LATAY D SEḌIS neɣ AMAZWAR N WUL.

Aɣaras ɣef tmerriwt neɣ aɣaras ɣef wul d useɣṛef n tidet, yettwaweḍ s LEDDEL TLEMMT n akk tidmus.

S LƐINI yezmer ad nsewwel neɣ ad nessiddeg TUḌAƐ n WUL D SEḌIS dɣa tulɣaɣ n wul AMAZWAR.

SHAMADHÍ, LEXȚISI, SATORI, ad ilḥeqen yal ass akk d tuḍɛatin tɣellanin aṭas, dɣa deg MAHA-SHAMADHÍ tuḍɛatatin ttferraqent.

Deg tallit n SHAMADHÍ TAMA, BUDHATA, tteɣreqent seg TMESKINT, imiren tettwareqqeq akk d SER dɣa teqqed TAMUSSENT n ayen YELLAN deg UɣUBU ITTEFRUSEN.

Kan deg uḥerrew n AYI yezmer ad nḥekku akk d BABA, akk d BRAHAMA.

NẒALLEƔ dɣa NEMMERKED, akken kemm ad teslem TAMESRI ɣef UɣUBU.

Lيو d ugerdas n TIƔRI, wul n ZUDIAKU. Lيو yesselhu wul IMAZAN.

TIƔRI n teɣri tga d WUL. Deg wul tizemmar n uεellay tgezment akk d tizemmar n tlata, akken win n tlata ad tteɣreqen.

Umeɣres n Lيو yella d AURƔU azedgan. Azru n Lيو tga DIAMANTE; uɣersiw n Lيو yella d AURƔU.

Deg leɛwayed nekkni nefhem belli iɣelliyen n Lيو gan am ɣersiw, d imellalen, d iɣellanen, d ineɛṛaben, yessnen lqeḍra, yettṣebbiten.

Eɛni lɣaci yella waṭas dɣa yebayan belli gar iɣelliyen n Lيو kan nennaf deg uɣerda ɣer targa, uɣersen, iɛewwen, iḍefɛen, atg.

Iɣelliyen n Lيو sɛan lemri n wexdam, ɣur-sen yettwesseqdiḍ ssiḥ dɣa taɣiwant n ɣersiw. Iɣelliyen yessulin di tɛebbuḍt, gan d IMSEBDEN IMIQQAN.

Asnak meddeγ n Lيو yella ssiḥ yettwesseqdiḍ dɣa yella iɣellalen. Asnak meddeγ n Lيو ɣur-sen azal meqqren ɣef tizmerin-ines.

Deg yal iɣelli n Lيو yella yal ass MÍSTICA yettwessuliḍ di tagga tameẓẓyant; kullec yettɛelleq deg tɛebbuḍt n umdan.

Iɣelliyen n Lيو llant yal ass ttezzen iɣella deg iɣalen dɣa seg iman.
