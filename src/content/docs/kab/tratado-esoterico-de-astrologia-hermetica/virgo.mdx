PRAKRITI d YEMMAT TAƔERFANT, AGAZDI AMENZAW n ugama.

Deg tlemmast, llan kra n ugazdi, kra n yifadden d udfadden, maca ayagi akk d amgarrad n kra n UGAZDI YIWET.

TADDA AMENZAW d AKASA IZEDGAN yellan deg yal amkan, YEMMAT TAMEQQRANT, PRAKRITI.

MAHANVANTARA d PRALAYA d snat n wawalen SANSKRIT yugaren azal i wid yettɣaren GNÓSTICOS ilaq ad ilmed tamusni-nsen.

MAHANVANTARA d ASS AMEQQRANT AKUSMİK. PRALAYA d IḌ AMEQQRANT AKUSMİK. Deg WASS AMEQQRANT yella umaḍal. Mi d-tewweḍ IḌ AMEQQRANT, amaḍal yettḥebbes ad yili, yettḥellil deg uqerdus n PRAKRITI.

Amkan ur yeṭṭafar ara yella d ayla n YISEGMAMEN ITRIWIYEN yesɛan MAHANVANTARAS d PRALAYAS-nsen.

Mi kra deg-sen llan deg MAHANVANTARA, wiyaḍ llan deg PRALAYA.

Igemmaḍ d igelman n YUMADALEN d-ttlalent d-tteggwarent deg uqerdus n PRAKRITI.

Yal COSMOS yettlal seg PRAKRITI d yettḥellil deg PRAKRITI. Yal amaḍal d takurt n tmes yettceɛɛun d tteɣḍef deg uqerdus n PRAKRITI.

Yal ayla yettlal seg PRAKRITI, yal ayla yettqelleb ar PRAKRITI. Nettat d YEMMAT TAMEQQRANT.

BHAGAVAD GITA yeqqar: "PRAKRITI TAMEQQRANT d tɣremt-iw, dinna ad rreɣ asekkil, yerna seg-s, a Bharata!, ad tlalent akk temsefkiwin".

"A Kountreya!, PRAKRITI d tɣremt taḥeqqanit n kra n wayen yettlal seg tɣremt yemgarraden, yerna nek d asenttaq PATERNO".

"SATTVA, RAJO d TAMO, agi tlata GUNAS (aɣbalu neɣ tilmas), d-ttlal seg PRAKRITI, a win yesɛan iɣil ussid!, ttarun aɛebbuḍ ar win yellan d amdan".

"Seg-sen, SATTVA yellan d azedgan, d ameqqran yerna d aḥeqqani, ttaru amdan!, A win ur yeṭṭaṭṭafen ara, s ttegrew ar lferḥ d tmusni".

"A KOUNTREYA!, ssen belli RAYAS d ayla n tmeɣra yerna d aɣbalu n TRWIḤ d ttegrew; GUNA agi ttaru amdan ar tigawt".

A Bharata!, ssen belli TAMO yettlal seg teẓɣelt yerna yessaɣd awal akk iɣersiwen; nettaru amdan s uɣilif, tiɛessast d tnaḍ". (TASEMDA N TASEMDEST, TALMEST N TASEMDEST.)

Deg GRAN PRALAYA agi tlata GUNAS llan deg taɣiwant deg TILAS TAMEQQRANT n tḥeqqanit; mi yeffeɣ uɣiwet n tlata GUNAS, yettwali tuffɣa n MAHANVANTARA yerna yettlal UMAḌAL seg uqerdus n PRAKRITI.

Deg GRAN PRALAYA, PRAKRITI d UNITOTAL, INTEGRA. Deg MANIFESTACIÓN, deg MAHANVANTARA, PRAKRITI tettɣareḍ ar tlata n YIFADDEN IKUSMİK.

Tlata n YIFADDEN n PRAKRITI deg MANIFESTACIÓN, gan: Amenzu, win n UMAKAN UR NETTFEKKIR ARA; Wis sin, win n UGAMA; Wis tlata, win n WEMDAN.

YEMMAT TAƔERFANT, deg umkan ur nettfeqqir ara; YEMMAT TAƔERFANT deg UGAMA; YEMMAT TAƔERFANT deg wemdan. Agi gan Tlata n YEMMATEN; Tlata n MARYAM n UKRISTIANISM.

Wid yettɣaren GNÓSTICOS ilaq ad fehmen aṭas agi tlata n YIFADDEN n PRAKRITI, acku ayagi yugaren azal deg lxedma ESOTÉRICO. Ama d ayagi, yella d AƔBALU ad nessen belli PRAKRITI tesɛa taɣiwant-ines deg yal argaz.

Wid yettɣaren GNÓSTICOS ur ilaq ad tḍewɛen ma nerra-asen belli PRAKRITI n yal argaz tesɛa isem-is. Ayagi yeɛni belli nesɛa yal yiwen seg-neɣ YEMMAT TAƔERFANT. Ad tfahmed ayagi, d AƔBALU i lxedma ESOTÉRICO.

TLAL TISEMT d taɣawsa nniḍen. TALES TAMEZWARUT, TAMES TAQDUSET, ilaq ad iḥezzib tɣremt taqduse n YEMMAT TAƔERFANT, syin ad d-tas TLAL TISEMT.

Nettat, PRAKRITI, tella daymen d TIZDEGT, uqbel lweḥm, deg lweḥm d deffir lweḥm.

Deg udis n uḍris n tektabt-agi ad nesseɛti lxedma yellan d praktika i tlal tisme. Tura ad nerr kra n tɣermiwin ad nessen ɣer anda nteddu.

Yal AΣERBI n LOGIA TAWEΣLANT yesɛa yemma-s taɣerfant, taɣiwant-is, PRAKRITI-s.

Yal AΣERBI d mmi-s n tzeɣt ur yettwaxḍaɛen ara. Ma nɣer Religions comparadas, ad naf dima tameɣra ur yettwaxḍaɛen ara; JESUS yettwacerreg s tigawt d leɣḍa n TNEFREST TAQDUSET, YEMMAT n JESUS tella d TZEƔT UR YETTWXḌAƐEN ARA.

Tiktabin tiɣerfanin qqarren belli BUDHA, JÚPITER, ZEUS, APOLO, QUETZALCOATL, FUJI, LAOTSE, atg., atg., llan d arrac n TIZDEĞT UR YETTWXḌAƐEN ARA, tizeɣtin uqbel lweḥm, deg lweḥm d deffir lweḥm.

Deg wakal aqdus n VEDAS, DEVAKI, TZEƔT INDOSTÁN tewhem KRISHNA d deg BELEM TZEƔT MARYAM tewhem JESUS.

Deg CHINA AMARILLA, ɣef yiri n wasif FUJI, TZEƔT HO-AE, tesseffad aseklu n UR AMEQQRANT, tettɣummu-tt d taẓẓalin, yerna tɣremt-is tewhem s tigawt d leɣḍa n TNEFREST TAQDUSET ar CRISTO CHINO FUJI.

D aḥric aḥeqqani i TLAL TISEMT belli lwel ilaq ad yettɣareḍ TALES TAMEZWARUT, TNEFREST TAQDUSET, tḥezzeb tɣremt tazdegt n YEMMAT TAƔERFANT.

TAMES TAJNESLANT n TALES TAMEZWARUT deg INDOSTÁN tettwassen s isem n KUNDALINI yerna yettwamel s yiles n tmes d-yuɣalen.

YEMMAT TAƔERFANT d ISIS, TONANTZÍN, KALI neɣ PARVATI, tameṭṭut n SHIVA, TALES TAMEZWARUT d lmel-is yugaren tazmert d taɣaṭ taqduse.

Tiles ilaq ad yerkeb deg ugaraw anammas n taɣaṭ taqduse, tiles ilaq ad iḥezzeb tɣremt n YEMMAT TAƔERFANT, d aya kan ad d-yas tameɣra ur yettwaxḍaɛen ara d TLAL TISEMT.

KUNDALINI, yiman-is, d tmes UTTUSAN yellan yettwaɣleq deg tlemmast MAGNETIK yellan deg usɣar n uɣesdis, aɣbalu n tseddart taɣerfant.

Mi d-tekker tmes taqduse, tettali deg ugaraw anammas deg tseddart, teccar ṣa n tlemmast n tseddart yerna tettḥezzeb PRAKRITI.

TAMES n KUNDALINI tesɛa ṣa n tlemmast n tazmert yerna yella d aḥeqqani ad nerkeb taɣuri n ṣa n tmes ad nger TLAL TISEMT.

Mi PRAKRITI tettwaberreḥ s tmes taceɛɛut tesɛa tazmert meqqren ad aɣ-tɛiwen.

Ad TLAL tikkelt nniḍen yeqqen ar ad NGER ar TLAM. D waɛra ad naf win d-ilalen snat n tikwal. Waɛra d win d-illulen TIKKELT TISEMT.

Win yebɣan ad TLAL tikkelt nniḍen, win yebɣan ad iɣers LƔACI AMEGGU, ilaq ad yessufeɣ seg ugama-s Tlata n GUNAS n PRAKRITI.

Win ur yessufɣen ara GUNA SATTVA, yettɣimi deg uɣelben n TEORÍAS yerna yeṭṭerqiq lxedma ESOTÉRICO.

Win ur yessufɣen ara RAYAS, yesnerni EGO LUNAR s IRA, TMUZEƔT, TƐAFFANT.

Ur ilaq ad nettu belli RAYAS d aɣbalu-yiman n trwiḥ amɣar d tmeɣratin yugaren.

RAYAS d aɣbalu n yal taɛessast. Agi yella d aɣbalu n yal trwiḥ.

Win yebɣan ad yessufeɣ TRWIḤ, ilaq ad yessufeɣ GUNA RAYAS.

Win ur yessufɣen ara TAMO, ad tesɛu dima TASEMDEST tetteḍseḍ, ad yili d aɛessaṣ, ad yessufeɣ LXEDMA ESOTÉRICO, acku d aqehwa, azɣal, tiɛessast, tiɣerfin n wul, tɛiwwa, teɣzi n tɣerfin, ad yili d amɣar n tqehwa n lɣella n umaḍal-agi yerna ad yeɣḍeb deg teẓɣelt.

Yettwaqqar belli deffir lmut, agdud n temperamento SATTVICO ad yenzeɣ ar ijenna neɣ TLAM moleculares d electrónicos anda ad yekkes seg dicha infinita uqbel ad YALI ar tɣremt taddit.

INICIADOS ssnent aṭas, s tiggawt iḍsan, belli agdud n temperamento RAYASICO YETTƐUD neɣ YETTAƔAL ar umaḍal-agi deg tegrawla yettɣimin deg ugaraw, yettɛawaden taggawt ad tger deg tɣremt taddit, maca ur yesɛi ara tɣerfin n wakken ad yekkes deg TLAM yemgarraden n tafaska.

Yal ILUMINADO yessen s laman belli deffir lmut wid n temperamento TAMOSICO ttagin ar UMADALEN-IJELLIDEN yellan deg uẓar n wakal deg ugensu n umaḍal agenna.

Yella d AƔBALU ad nessiɣ seg ugama-nneɣ agi tlata GUNAS, ma ɣur-nneɣ aɣilif aḥeqqani ad nger s ussid lxedma ESOTÉRICO.

BHAGAVAD GITA yeqqar: "Mi yettwali win yessnen belli GUNAS kan gan wid yettwaxedmen, yerna yessen WIN yugaren GUNAS, syin ad yessiweḍ ar UR".

Aṭas n wid yebɣan taknik i wakken ad essuffeɣen tlata GUNAS nerra-asen kan s ad yessuffeɣ EGO LUNAR ad yessen ad yessuffeɣ s ussid tlata GUNAS.

Win yeqqimen d agdal yerna ur yettwagdal s GUNAS, win yesɛan belli GUNAS kan tteddunt, yerna yeqqim d aɣdar war taggada, acku YESSEFƔED EGO LUNAR.

Win yeḥusseḍ yiman-is deg lferḥ neɣ deg iɣsan, win yellan deg ugama-s; win yettaken azal deg tqedaḥt n wakal ar azru neɣ ar taɛyunt n wureɣ; win yettḥarar deg wayen iɛezzan d wayen ur iɛezzan ara, deg tɣerrut neɣ deg lemtel, deg cciɛ neɣ deg uɛiban, deg umeddakkel neɣ deg uɛdaw yerna win yessufɣen seg yal tigawt taddit d taɣzart, acku yessufeɣ Tlata GUNAS yerna YESSEFƔED EGO LUNAR.

Win ur yesɛi ara taɛessast, win yessen tagrest n TƐAFFANT deg akk ṛebɛin d tesɛin n tazwara SUBCONSCIENTES n ulma, yessufeɣ Tlata GUNAS yerna yesseffeɣ EGO LUNAR.

"Akal, aman, tmes, aḍu, amkan, ulma, adɣar d EGO, gan ṣamaniya n tqitɛat yellan deg-s PRAKRITI-iw". Akken yettwaru, agi d awal n win d-yettubarken.

"Mi d-yeffeɣ WASS AMEQQRANT AKUSMİK, ad llan akk iɣersiwen d-ttasen seg INMANIFESTADA PRAKRITI; yerna deg tɣemza, ad d-ɣellin deg INMANIFESTADA".

Deffir INMANIFESTADA PRAKRITI yella INMANIFESTADO ABSOLUTO. Ilaq ad tger lwel ar INMANIFESTADA uqbel ad nessebzeg deg Uqerdus n INMANIFESTADO ABSOLUTO.

TAKMAM TAZDEG TAƔERFANT n UMAḌAL d ayen-nni yettusemman TMIRIT. Nettat d ISIS, win ur yesɛin kra n win ad yeftek uɣilif; deg uẓiẓẓef n YILES ad t-neɛbed.

Akk d DININ TIMEQQRANIN ttagin ar YEMMAT IKUSMİK; nettat d ADONÍA, INSOBERTA, REA, CIBELES, TONANTZÍN, atg., atg., atg.

ADEBBAD n TZEƔT YEMMAT yessen ad yessuter; tiktabin tiqdusin qqarren: Sutret yerna ad t-tekksem, sebbliret yerna ad t-teldi.

Deg UQERDUS AMEQQRANT n YEMMAT TAƔERFANT yettwaṣubben igemmaḍ. VIRGO yeqqarben UƐEBBUḌ.

Virgo yella yettuselkem aṭas ar iɛebbaḍen d deg ulɣu meqqren ar Páncreas d ISLOTES n LARGEHANS yettberriken INSULINA yugaren azal i tagrawla n wukkren.

Tazmrawin yettalin seg wakal, mi tewweḍ ar uɛebbuḍ, ttagent s hormoninos adrenales ad tent-heggent d ad tent-zedgent i wakken ad alin ar wul.

Deg uzeffan agi n VIRGO (TZEƔT TAƐESSRIT), nekkni, nɛellem uḍar deg weɛebbuḍ akken ad nerr iɛebbaḍen tizenzalin, s yiswi n tazmrawin yettalin seg wakal, ad ttagint deg uɛebbuḍ s hormoninos adrenales.

Win yettɣaren GNÓSTICO ilaq ad yefhem ugaren azal n tala agi yettwassen s aɛebbuḍ yerna ad yekkes s uɣlif seg lxilaf n laɛbuḍ.

INELMADEN n umeɣṛad BUDHA kan yeqqimen d lqut yellan d aḥeqqani deg wass.

ABRAHAM, IA-SAC, IA-CAB, IO-SEP, llan daymen d iɛabdden n TAƔAṬ TAQDUSET, IO, neɣ n DIOSA EGIPCIA IS-IS; ama d Musi, neɣ ad nerr AΣERBI yeddlen tiselmadin n Musi, ttagin ASESSEG n TAƔAṬ d teḥbibt yerna idim-is ad yeɣli ɣef uqerreɛ n akk, i uxatar n warrac-nsen.
