## AGLAM

### SƔUR USTAÐ GARGHA KUICHINES

Tella tsegniwt tusnakt neɣ tusnakt yettagwaden aṭas n leɣraya i uselmed-is, yid-s itsegniwen n yal tallit ugin-d ɣef yedrimen n usnerni. Am wakken i s-seǧǧlen i wemdanen imeqqranen am wakken d Hitler deg Alman i yesɣerben itsegniwen yettwassenen akken ad ten-yawi deg wemger-is.

Nekni Igniẓiyen netbeɛɛed i ulmud-agi, acku yid-s amdan d amaruz ɣef wayen ara d-yernu, d win ur nezmir ɣef yizmawen i yettaken tamawt ɣef tmerriwin yemxallafen d tɛedda n yetran, nekni nessen tsegniwt iɣ-yesɣaren amek ara nweḍ ɣer yetran akken ad neǧǧ aya ara d-inin imusnawen n tsegniwt tusnakt. Akken ad nbeddel tfekkiwin tignawin i d-nettlalem yid-sent, s tfekkiwin tifyiziyin neɣ tifenṭastin s yettwazalen si tẓur-nneɣ neɣ si tẓur-nneɣ.

Aɛdaw iɣ-yerren ad nɣewweɣ tamusni illu d Yiw-nneɣ Asetan neɣ Aqerru n leǧnun i yettwaḥakmen ɣef tfekka-nneɣ. Amsak iṣeḥḥan aṭas akken ad nqelleɛ Amɣar n Umellal am wakken i t-semman wid n uẓelmaḍ d ɣef usnerni i yettbeddu ɣur-nneɣ Igniẓiyen mi nekcem ɣer yinigen neɣ Lumisiales, s uselmed i yettwaḥakmen d yettwasɣaren sɣur Avatar n Acuario "Samael Aun Weor".

Deg tallit i deg yefka Ubeddiw Ɛisa uselmad-is i wid yettettbinen iberdan yenna: "Nekk d abrid, nekk d tideț, nekk d tudert, llan deg wussan-nni uɣerrabu ameqqran isem-is Simón uɣerrabu, iččuṛ d leɣder d yedrimen i yettbeɛɛaɣ i yinelmaden-is: "Ma yewweḍ Ɛisa ɣer baba-s s wayen yexdem, nekk Simón, ad awḍeɣ ɣer baba deg wayen ara xedmeɣ, d wayen yexdem d amenzu n uẓelmaḍ, yettbeɛɛed seg win yettḥemmel, tazzla-agi tella deg-neɣ mi nettaɣ wid yettettbinen iberdan ur zmiren ad sɛun tamusni deg wiberdan.

Wid i d-yuɣalen ɣer tmusni-yagi tikkelt tamezwarut, neqqar-asen-d iḥwaǧ ستة أشهر n leɣraya n umenzu, aɣɣar aɣɣar i warrawen n Samael Aun Weor d usnerni i tmusni-nsen, din din ma yefka-d tamawt belli yesɛa tamusni, d yebɣa tudert yugaren, din din ad yekkec ɣer Lumisiales s usnerni n uɣerri ameqqran.

Mi yetheggi win yettettbinen abrid i usnerni usnerni, tazzla tamezwarut ara yezzel d Amɣar n Umellal neɣ ad yemager Asetan-is win i d-yellan d aqerru-nneɣ d Uselmad-nneɣ azal n aṭas n leqrun. Ttmektiɣ-d llan deg useggas n 1949, yiwen seg yinelmaden n Hierofante n Tmesɣelt Tamaẓẓugt Aun Weor, d deffir n ugar n rebɛa waguren n taṣeffiwt tazedɣant, yettwaɣzen ɣer tazzla n Uɛessas ass n 27 yulyu. Yella win yebdan deg umkan d iɣawsen, netta yella deg uneggaru, maca ur yessen ara, mi yessawel i Uɛessas n Umellal, ur yessɛedda ara aṭas, iḥuss-d win yebdan belli tikkelt tamezwarut yella d asemmiḍ n lmut, tallit i deg yella d tifaw yella yettɣummu, yettnerni daɣen asemmiḍ d riḥa teṭṭuq, yessegraw-it d lxuf, yesɛa leḥyaǧ ad yeɛreq, maca taqvirt n Lmasiḥ i yessegraw deg tfekka-s s usnerni n ubeddel n tɣawsiwin, Arcano A. Z. F., yefka-as azref akken ad yeqqim deg umkan-nni ur yesɛi ara. Iwala din din tella-d ɣur-s tserɣint s lekmal n teɣmert, teẓẓiṭ, yesɛa iqcicen deg udem-is i d-ittfawen d yettẓallagen mi yettnuzzu, seg uɣrum d imi am teɣyult, yenna-as: A wi yebɣan ad iyi-teǧǧeḍ? Akka ara tesɣurfuḍ aṭas ara iyi-txedmeḍ? Teqqleḍ-iyi i wergaz-nni ur tessen ara? D netta yenna-as iččuṛ d lxuf belli ad t-yeǧǧeḍ, tserɣint-nni tuɣal ɣur-s d tazzla, win yebdan itettbenni-it, maca ur yezmir ara ad d-yini ɣef ubeddel-is s lexder, maca yettmekti-d belli d Chela n Lmasiḥ d itettbenni-it s yisem n Lmasiḥ akka ara d-yeqqel ɣer deffir, yal tikkelt i deg ara t-tuɣal, itettɛawan seg Lmasiḥ d Imɣaren-is yettḥemmel d yenna-as: ur tezmireḍ ara ɣur-i, ad k-ɣelbeɣ d tserɣint-nni teqqel yettḍegger tiɛaqib n yal ṣṣenf. Tazzla n unelmad-nni tella d tameqqrant, acku Hierofante yeffeɣ ɣer temdint nniḍen d aya yettagwad ugar n tlata wussan, maca mi yuɣal yenna-as d netta yenna-as: Ad k-iɛawen Rebbi, teffeɣ-d akken ilaq seg tazzla tamezwarut d Uɛessas n Umellal, ayagi d Asetan-ik, iɣef texdem d t-tɛeddayeḍ azal n aṭas n leqrun, d amek ara t-tɛeddayeɣ?, Yessuter unelmad-nni d lexder d Uselmad yenna-as, netta iɣaḍ-it ɣef tmeɣṛa-nnek, itettbanayen seg lebɣi-nneɣ, tmeɣṛa taẓẓayt, zzna, leḥwanut d tudert teṣṣeṭṭet, ayagi d amazaɣar i yellan deg lǧameɛ iɣef iɣ-d-yeḥder Lmasiḥ, ttaɣen deg lǧameɛ-nneɣ, tura terna-k-d ɣef uɣyul n lebɣi ad tessufeɣ seg-k amazaɣar-agi i k-yeṭṭfen d aklayen n Asetan, tura terna-k-d ad ten-ɣelbeḍ yal waḥed deg-sen ma tebɣiḍ ad tekkeḍ seg udir d tebdiḍ abrid amellal.

S warraɣ-yagi nezmer ad nẓur yemma itran yakk, d aɣɣar ɣer lǧameɛ n wul n yetran, s tuttra d axeddim d lmalayket n yetran, akken ur neɣli ara d amaruzen n tmara, maca tikkelt tamezwarut iɣeq-ad ad nessufeɣ imezdaɣ seg lǧameɛ-nneɣ, ilaq ad nelmed amek ara nberreẓ deg umkan-nneɣ. Akken ad yettɛeddi win yettettbinen iberdan ɣer sser-nsen n Lumisiales; din din itettlamad akken ad iḥemmel d yeqdec ɣer Rebbi ɣef kullec d ɣer uneggaru, itettmelal yid-sen d deffir itettfehem belli kullec n lɣella n uṭan yella itettqarab itettɛawin yid-s d itettfehem ayen ur yettwafaq ara. JACHIN d BOAZ n umkan ilaq akken ad yessufeɣ deg umkan-is d awḍen ɣer tallit i deg yettlamad akken ad yefk سبعة خطوات iḥeq-it akken ad yexdem deg uɣerru n Rebbi d deg udem n tiɛeccawt i yettbarkiten.

S uselmed n usɣerri nelmad amek ara nɣen azger ara neḍfer deffir ad nṣeggem s tallit tserɣint n uɣenjaṛ uffir. Akka ara nekkeɣ seg Uselmad n Tallit akken ad nedder tudert ur yettfukku d ččuṛen d sɛada tabeddit.
