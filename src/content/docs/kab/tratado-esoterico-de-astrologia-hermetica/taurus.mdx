Imi d Taurus ay d azamul amezwaru it-teṭṭfen Aɣurru amazday, iṭij-nneɣ ad yeflali anda tetteddu awal, tanekkra, yessefk ad naf mliḥ awalen n Yuḥenna, mi d-yenna: "Deg tazwara yella Uwal, Uwal yella ɣer Sidi Ṛebbi, Uwal d Sidi Ṛebbi, yis-s ay d-texleq kullac, yellan mazal ur d-ixleq."

Llan sebɛa tdersiwin n umaḍal, sebɛa akusanin itwaxleqen s tazmert n Uwal, s uẓawan, s ṣṣut.

Akus amezwaru yettnerrum gar Tafat ur-d-yufi lqaɛa n ABSOLUTU.

Taderst tis snat n umaḍal tettwabna si umaḍalen n tallast ur-iḥedden.

Taderst tis tlata n umaḍalen d tajmaɛt n yiṭij-nneɣ n tallast yettwazerrfen.

Taderst tis rebɛa n umaḍal d Iṭij it-yezzernun s yis-i s wamuren.

Taderst tis semmus n umaḍalen tettwabna si ifilalen n tallast n teɣmert.

Taderst tis sḍis n umaḍal d tagnit yis-i, d sebɛa wamuren n iɣezran yezdeg deg-sen imdanen ur-neḥsab.

Taderst tis sebɛa n umaḍalen tettwabna si sebɛa tissistrin neɣ d Umaḍalen n JAHANNAM n ugellid MINERALUM yettwerrumen seddaw teẓṛut n Tagnit.

Uẓawan, Awal, it-yefka LUΓUS deg sebɛa n tesnawtin, iṭij-nneɣ isemmaḍ.

Taderst tamezwarut n umaḍalen, taseɣrut DU. Taderst tis snat n umaḍalen, taseɣrut SI. Taderst tis tlata n umaḍalen, taseɣrut LA. Taderst tis rebɛa n umaḍalen, taseɣrut SOL. Taderst tis semmus n umaḍalen, taseɣrut FA. Taderst tis sḍis n umaḍalen, taseɣrut MI. Taderst tis sebɛa n umaḍalen, taseɣrut RE. Sakin kullac iteddu ɣer ABSOLUTU s taseɣrut DU.

Bla uẓawan, bla Awal, bla awal ameqqran, ad yuɛer ad d-yili sebɛa Ikusanin.

DU-RE-MI-FA-SOL-LA-SI. SI-LA-SOL-FA-MI-RE-DU. Sebɛa taseɣruwtin n teskala tameqqrant n Uwal amazday, it-tezzenznen deg kullac it-d-texleq, acku deg tazwara yella Uwal.

Taderst tamezwarut n umaḍalen itettwaḥkem s ccal kan, ɣer ccariɛa tameqqrant. Taderst tis snat n umaḍalen tettwaḥkem s tlata tcaliɛin. Taderst tis tlata n umaḍalen tettwaḥkem s sḍis tcaliɛin. Taderst tis rebɛa n umaḍalen tettwaḥkem s mraw snat tcaliɛin. Taderst tis semmus n umaḍalen tettwaḥkem s mraw rebɛa tcaliɛin. Taderst tis sḍis n umaḍalen tettwaḥkem s mraw tmanya tcaliɛin. Taderst tis sebɛa n umaḍalen tettwaḥkem s tesaɛ tmandit tcaliɛin.

Mi d-nettmeslay ɣef awal, nettmeslay daɣen ɣef ṣṣut n uẓawan, n Iṛeqṣuten, n Tmes yis-i tlata n tzeɣwin n MAHAVAN d CUȚAVAN it-isemmḍen Iṭij-nneɣ deg umugru-ines.

Iṣeḥḥayen d Iseḥḥayen n Ṣuṭran kan it-nettmeslayen ɣef MICROKOSMUS, d MAKROKOSMUS, kan it-qaren snat tdersiwin n umaḍal, mi d-yeṛṛiɣent lḥala llan sebɛa Ikusanin, sebɛa tdersiwin n umaḍal itettwaḥkemen s Uwal, s uẓawan, s FIAT Uzzal d SPERMATIKU deg taswiɛt tamezwarut.

Kullun seg sebɛa Ikusanin d amuddir it-yeṭṭsen, it-yeḥussu, yet-tedder.

Si tama-nniḍen Tesseḥlit, neɣd sers-ent araɣad, d ugellid yellan deg wadda. Ur-nezmir ara ad nali, mi ur-nellaḍ ara. Ugar ad ners sakin ad nali.

Ma nebɣa ad nessen Akus, ilaq ad nessen snat it-izdin, yellan garas-ent.
Am umezwaru, d win yellan seddaw-s, acku garasen ay d-itekken taggara.

Amal: Deg wakud-a anda Imsensiwin țnaɣen i tameḥrest n tilalt, xersum taɣawsa ur-tbedd-ara deg umaḍal.

Lexlaq-nni n sebɛa Ikusanin yuɛer ad tt-yili, mara ur-yilli Uwal, mara ur-yilli uẓawan.

Inelmaden-nneɣ n GNOSTIKU ur-yeɛqel ara ay ɣer yella tlata n tezmert it-wanɣen BABA-MESS-ṚEBBI-RUḤ QEDDES. Tezmert-a texleq SAGRADA TRIAMAZIKAMNO.

Aya d AFIRMACION SAGRADA, NEGACION SAGRADA, REKONSILIACION SAGRADA; ṚEBBI UQEDDES, AGAZU UQEDDES, AMDAN UQEDDES.

Deg trisiti, aya tlata n idisan PUZITIVU-NEGATIVU-NEUTRU. Mara ur-yilli trisiti-a, ur-yettili lexlaq.

Deg tussna ESOTERIKA GNOSTIKA, tlata tezmert ur-ittekken yis-i tt-wassen: SURP-OTHEOS; SURP-SKIROS; SURP-ATHANATOS. TEZMERT TAMSIFT, AFFIRMATIVA, PUZITIVA. TEZMERT NEGATIVA, TEZMERT N UNGAL, TEZMERT N ṚṚEḤMA. TEZMERT AREKONSILIADUR, TEZMERT IT-SSELHAYEN, TEZMERT IT-ISFEQQAƐAN.

Tlata tezmert-a deg RAJU N LEXLAQ tban am tlata ṛṛay, tlata tɛeqqal, tlata taḥdaynin. Kullun si tlata tezmert-a tḥerrez Yis-i kan kullac. Gar-asent kan kullec it-yezmer, ɣers-i.

D lmeɛna tameqqrant ad neẓṛ tlata tezmert ɣer umger: tneḍfer gar-asent, teddunt sakin ad d-uɣalent ad ilint trinidades ideggen umaḍalen, lexlaq-i-ijdiṭ.

Deg ABSOLUTU, tlata tezmert-a d LUΓUS WAḤDAN, LEJNICU N ṢṢUT deg taḥdaynit tameqqrant n tudert tixilast deg umger-is.

ANSA LEXLAQ N SAGRADA TRIAMAZIKAMNO KOSMIKA KOMUN it-yebdan s umenzi n Uwal acku deg tazwara yella Uwal, Uwal yella ɣer Sidi Ṛebbi, Uwal d Sidi Ṛebbi. Yis-s ay d-texleq kullac, yellan mazal ur d-ixleq.

Atifaq gar ccariɛa n SAGRADA N HEPTAPARAPARSHINOKH (CCARIƐA N SEBƐA), Sebɛa tmegratin deg CAUS i lebni n tallast-a taẓrant.

Atifaq gar ccariɛa n SAGRADA DE TRIAMAZIKAMNO (CCARIƐA N TLATA), iluhimen ttwaḍfaren ɣer tlata trebiɛ deg tmegrat ad-yeɣnen atifaq gar LITURJIA N Tmes.

Lexdem n wugri n PRAKRITI, neɣd CAUS, ɣer TAYEMMAT KOSMIKA, Aɛebbuḍ AMEQQRAN, d leqdic i yellan teɣmert-is.

Deg tmegrat kullec d tlata n trebbaɛ am-i-d: Amezwaru, d ASSEDDAD. Tis snat, d TADUKAL. Tis tlata: d tarbaɛt n iluhimen.

Ma tella deg ubal kan iluhimen d INDIΓENUS, sak ad țuḍfaren ad yilin deg usaf s tayri di ṣṣifa TAJDIDT, Tametat d Trinitat atifaq gar SAGRADA TRIAMAZIKAMNO KOSMIKA KOMUN.

Arrawan n tmes țacenwi d amezi n Uwal yugri aɛebbuḍ ameqqran n CAUS tewlel tilast.

Am yijeǧǧigen ičuččen i tbanen, meɛna waɛra, imeslayen ibanen, meɛna ugrien yellan deg umdan ur-yexdim ara am-i-d-yenna.

Meɛna am yijeǧǧigen ičuččen i tbanen, meɛna waɛra, imeslayen ibanen, ugrien yellan deg umdan yexdem ara am-i-d-yenna.

Yella cɣel ad nselkeɣ taɛkurt n Uwal, ilaq ad nemmeslay war ugur, di ṣṣifa TESSENT d ugani. Ilaq aɣes ad Nessen taɣuri n Uwal.

Yella leḥsab deg awalen d leḥkem s Uwal d aḍris. Ulac win yellan tezmert ad iḥkem ɣef win, iɣir ad teḥkim deg tmetti.

Awalen n tɣellawt ur-ttḥebsen ara sakin, i wakken ad d-uɣalen ɣef-neɣ-id am teɣlujt. Awalen n lekdeb, iɛiban, ad d-uɣalen i win it-id-yennan it-tuɣalen iɣirden.

Deg tikliwin iɛeddan, mi mazal ulac aṭas, ur-tezmir ara ad temmeslay di ṣṣifa iẓuran.
