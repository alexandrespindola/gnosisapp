20 YENNAR AR 19 FUṚAR

Anamek yeffren n AKWARYUM d AMAẒEL. AKWARYUM, azamul n bab n waman, d azamul n tezrigi aẓayan.

Llan kuẓ n telmas n TUSNA neɣ N TUSNA TUFFER. Ilaq a neẓer d acu-tent kuẓ n telmas-a n TUSNA.

AMENZU: VAJNA-VIDYA; tussna i yettwakseb s kra n tezmert yeffren i d-yettnernin deg udmawen-nneɣ n zdaxel, s kra n Tameɣriwin Tiseḥḥariyin.

WIS SIN: MAHA-VIDYA TAQBALIST. TUSNA n TEQBALA s akk ineffuten-ines, tusnakt-is, izamulen-is d teẓẓalit-is, tzemmer ad tili TANGELLUST neɣ DIBULIST, akk yettusemma ɣef umdan i tt-iseqdacen.

WIS KRAḌ: GUPTA-VIDYA; tussna n MANTRAM, Sseḥḥar n Awal; tettubeddel ɣef tezmert tumerkant n ṣṣut, deg tussna n TILELLI.

WIS KUẒ: ATMA-VIDYA neɣ TUSNA TAǦIḌANT N WAẒAR n SER, n ATMAN, n MUNADA AMEQQWREN.

Akk tilmas-a n TUSNA, ala WIS KUẒ, aẓar n akk tussniwin yeffren. Seg akk tilmas-a n tussna, ala WIS KUẒ, i d-yettasen TEQBALA, QUIROMANCIA, ASTROLOJÍA, FISIOLOGÍA TUFFER, CARTOMANCIA TAẒRAYANT, atg., atg.

Seg akk tilmas-a n tussna, seg akk YIẒEṚWAN TUKRIḌANIN-A, tussna tufa-d kra n tsecret, maca ANAMEK N UMECRAḌ YESNULFAREN, ULAC D UHIPNUTISM s tezmer ad ttwakseb s tẓuriwin-a.

AKTAB ASTROLOJÍGAN D HERMÉTICO ESOTÉRICO YELLA, ulac fell-as d ASTROLOJÍA n temlilit i d-yettwabedren deg IPÉRIÚDIKUS. Deg udlis-a nesselmed TUSNA n ATMA-VIDYA.

Ayen yellan d ameqqran d ATMA-VIDYA, nettat teṭṭafar akk deg udmawen-ines ISELḤANIN u tezmer ad tseqdec deg-sen kra n tikkelt; maca tettuseqdec kan deg tira-ines tisentetikin yeffsin seg akk tkerḍiḍin.

Tawwurt n WARƔI n Tussna tezmer ad tbeddel ɣer tewwurt tewseɛ d ubrid iwseɛ i yettarew ɣer ulac, tawwurt n tẓuriwin tiseḥḥariyin yettwaxedmen i lɣerḍ n yiman.

Nella deg TAMEẒLA N KALI-YUGA, tamezla n UZZAL, TAMEẒLA TUTTIĜ, u akk inelmaden n TUKRIḌANIYYA ttwaheggan ad ḍallen deg ubrid aberkan. Iḥemmel ad iẓer anaw ameqqran yellan d adrar deg "warrac" ɣef TUKRIḌANIYYA d usnulfu s tmuqran s wayes umiren ad awḍen ɣer TEWWURT d ad ɛeddin UMBRAL n TUMERT war AGUL AMEQQWREN.

Yuɛer a negrer ATMA-VIDYA war KUẒ IFACTURES n TEẒREGT N TUMERT.

Yuɛer ATMA-VIDYA war ma n-awḍen ɣer TLALLIT TIS SNNAT. Yuɛer ATMA-VIDYA war tamettant n YO PLURALIZADO. Yuɛer ATMA-VIDYA war AGUL ɣef talsa.

ULAC d Agitas n TEẒREGT ay d-yettakken ATMA-VIDYA. ULAC d Agitas n TINVOLUCIÓN ay d-yettakken ATMA-VIDYA. Ala ɣef ulsas n TRÉMENDAS d ESPANTOSAS TEẒREGT TINEZRARIN, n-awaḍ ɣer ATMA-VIDYA.

UBRID n TEẒREGT n tumert d SENDA DEL FILO DE LA NAVAJA; senda-a tella TERRIBLEMENTE TAƐSIRT; senda-a tella tettef telqeyt deg zdaxel u berra.

Ad neɣer tura deg yixef-a yal yiwen seg KUẒ IFACTURES n TEẒREGT N TUMERT deg talɣa teṭṭfen d tgezzumt, iwakken inelmaden INUSTIKUS umiren ad teggren ɣer lɣerḍ.

Sten, ihi, imeqqranen-nneɣ aṭas n laɛnaya deg uɣer n yal yiwen seg KUẒ IFACTURES n TEẒREGT N TUMERT, acku seg TILILA tameqqrant n yal yiwen seg KUẒ IFACTURES-A i yettɣim IMERDEG deg UXDIM-A.

### TILLAL

TILLAL TIS SNNAT d ugur akk yellan d aslullus. UƔAL TAQQAḌ AMEQDUŞ GAR IQBAIṚIYEN IQDIMEN, ilaq ad yili d ilemẓi, d aselḥi d useḥḥi iwakken ad yessikel PIEDRA FILOSOFAL. (ASLULLUS.)

Iγris i ttwaɛerzen sɣur IHIERUFANTIS IQBAIṚIYEN s-aɣ aɣer PIEDRA FILOSOFAL, s yiwen neɣ kra n TAQAQAḌ, akken yettban daɣen deg tamacahut n MINOTAURO CRETENSE.

Anamek iǧhedan ALKÍMICAN TTWAXEDMEN deg TAQAQAḌ i HÉRCULES yekksen-d ɣer Gerión, asikel-nni daɣen nufa-t deg targit n TAQAQAḌ IMEQDAREN n UTFI d wanda ttidiren ttidiren deg TIGZIRT n SICILIA d wanda Mercurio tekksen-iten-id.

ULAC akk TAQAQAḌ i d-yettaɣen aqqaɛ amellal neɣ amellal; kra deg-sen uɣalen d IZWIṚ akken yellan wi Gerión d akken yellan i ttwaɣedmen sɣur Sacerdote Israelita, acku PIEDRA FILOSOFAL, deg kra n teswiɛt Alkímican uɣalen d IZWIṚ u ayagi-a iẓer-it akk Alkímista.

Uɣal APIS ameɣlal, i ttwaɣuren aṭas deg TMEẒRAWIIN IQBAIṚIYEN, yella d ambeddal d umeksi n TLUṚWA. Uɣal APIS SIMBÓLICO yettwakkes-d ɣer ISIS, acku ɣef tidet yella yettusemma ɣer VACA TAQQAḌT, TMIMT TIḌEVINIT, ISIS, win ulac amettani i d-yessawen tserwelt.

Iwakken uɣal ad yili taɣla tameqqrant akken ad yettwaɛerzen ɣer tsegziwt-a, yella d amecṭuḥ ad yili d amqqaɛ u ad yili deg udem-ines neɣ deg taḍessa-ines, tilɣi tamellalt deg talɣa n tgemmert Taleɣmant.

Daɣen d tidet d akk tidet, d uɣal ameqqran aqqaḍ ad yettwaɛmerr s ddaw imɣi n uẓeṭṭa d ad yegmer ddaw iles-ines azamul n teskert tameqduṣt.

APIS yella d ASEMEL n LUNA, ɣef usirem n iqernuten-ines deg talɣa n tgemmert Taleɣmant, meqqar ɣer, ala Plenilunio, yiṭri-a yella yal yettili yiwen n tama tineɣɣimt i d-yettusemmer s uqaqaɛ n teẓruẓet d tayeḍ tettreqreq, yettwasemel s telɣi tamellalt.

APIS d MATERIA FILOSOFAL, ENS SEMINIS (SEMEN), tazwert-a temesɣaren, temeslalayen, VITRIOLO-a n ALKIMISTAS.

Deg zdaxel n ENS SEMINIS yella akk ENS VIRTUTIS n UTFI. Ilaq ad nebdel LUNA ɣer UTFI, anamek-ines, ad negmer IGMUḌIWIN UTFANIIN.

Agi d TUMRAWIIN N ISIS, TUMRAWIIN n UƔAL APIS. Mi deg uẓar n Iqbaiṛiyen n IFARAUNIS ttɣaren RUNA IS, ttwassen s snat wudmawen-ines. MASQULINO-FEMENINO, acku AWAUL AMEQDUŞ ISIS yettugezzem deg snat tgezzmin IS-IS; TI GEZZEMT tamezwarut d MASQULINA d tis snnat d FEMENINA.

UƔAL APIS d UƔAL n ISIS, PIEDRA FILOSOFAL. Argaz d tameṭṭut ilaq ad xedmen deg LABORATORIUM ORATORIUM-nsen s MATERIA FILOSOFAL, ad beddelen aLUNA ɣer UTFI.

D URGENTE ad yettwakseb azmer MÁGICO-a i d-yettwasemma KRIYA-SHAKTI neɣ n TAYRI d n UYOGA, azmer Mágico n IIRGAZEN ITFANIIN, azmer agejdan n UBDAL, war Generación u ayagi kan yettawi s MAITHUNA. (Ẓer yixef umnber ḥamiṭ.)

Ilaq ad yelmed ad yeggmer s tẓeḍdert aman n tudert gar sin ÁNFURAS n AKWARYUM, azamul n tezrigi n UMANAD.

D INDISPENSABLE ad yeggmer ELÍXIR AZWIṚ s ELÍXIR AMELLAL, ma yella i yebɣa ad yaweḍ ɣer TILLAL TIS SNNAT.

LUNA tesseml ISIS, TMIMT TIḌEVINIT, PRAKRITI ineffable u UƔAL APIS yettwasemell MATERIA FILOSOFAL, PIEDRA TAQQAḌT n ALKIMISTA.

Deg UƔAL APIS yettwasemel LUNA, ISIS, tazwert tamezwarut, PIEDRA FILOSOFAL, MAITHUNA.

AKWARYUM yettwamerr sɣur URANO u agni-agi ad yeqqen Tiɣellafin TILULLUSIN. Yuɛer ad t-yaweḍ ɣer TILLAL TIS SNNAT, ɣer ADEPTADO, ɣer AUTORREALIZACIÓN TAIMEZRAT, ma ulac netɣer TUMRAWIIN n ISIS, ma nettḥeqqer aɣur ɣer UƔAL APIS, ma ulac ad nelmed ad nzeggir ELÍXIR AZWIṚ s ELIXIR AMELLAL gar sin ÁNFURAS n AKWARYUM.

Deg terminología Cristiana netmeslay ɣef KUẒ IGMUḌIWIN imdanen. Amdan amezwaru d IGMUḌIWIN IQERQISNIN; wis snnat d IGMUḌIWIN IBIṚDANIIN; wis kraḍ d IGMUḌIWIN IṢEPIRITANIIN; wis kuẓ, ɣef terminología n anaw Cristiana Isuṭranit, d IGMUḌIWIN IḌEVINIIN.

Netmeslay deg uslil n Tiyusufit, ad nini d amdan amezwaru d IGMUḌIWIN IPHYSICUS, wis snnat d IGMUḌIWIN ASTRAL; wis kraḍ d IGMUḌIWIN IMENTAN; wis kuẓ d IGMUḌIWIN ICAUSAL neɣ IGMUḌIWIN N TAYRI TIḤEǦANIT.

Icekkaɛen-nneɣ ad ḍallen acku ur nebdar LINGAM SARIRA neɣ IGMUḌIWIN ḤIYAT, daɣen yettwasemma ḌUBEL ITIRIK. Ciertamente ulac neḥsab IGMUḌIWIN-a, ɣef uɣḍas iǧehden n ayagi yella kan d tgezzumt tasawaranit n IGMUḌIWIN IPHYSICUS, asqedmej agejdan ameqqran n akk ixeddimen iPHYSICUS, ikimikus, icalóricos, imbadalen, ipircepeṭtivos, atg.

ANIL IMEQDIṚ amenza d ur ILAL s ASTRAL, war MENTAL, war aṭas war IGMUḌIWIN ICAUSAL; igmuḍiwen-a kan umiren ad snernin deg tɣilt yecan n VULCANO. (ASLULLUS.)

IGMUḌIWIN ASTRAL ulac d IMPLEMENTU indispensaɣil i ANIL IMEQDIṚ; d ɣila, ɣila tameqqrant i umiren ad yefk kan kraḍ; meqqar akken, ANIL IMEQDIṚ yegmer IGMUḌIWIN AMULIKYULAR, IGMUḌIWIN N IRA i yettemcaban ɣer IGMUḌIWIN ASTRAL, maca n anaw ILUNAR, asemmaḍ, ifantasmal, ispietral.

ANIL IMEQDIṚ ulac-it IGMUḌIWIN IMENTAN, maca yegmer yiwen uɣerwaz intellectuel inilan, ILUNAR, yettemcaban aṭas ɣer IGMUḌIWIN IMENTAN, maca n tɣalt asemmaḍ d ufantasmal.

ANIL IMEQDIṚ ulac-it IGMUḌIWIN ICAUSAL neɣ IGMUḌIWIN N TAYRI tiḥeǧanit, maca yegmer ESENCIA, BUDHATA, EMBRIÓN DE ALMA ay umiren a ttwakal s IGMUḌIWIN ICAUSAL.

IGMUḌIWIN iṣeltalen ay leadbeater, Annie besant, steiner d waṭas ineffuten iɣeren deg ANIL IMEQDIṚ ilemẓi, d uɣerwazen ILUNARES.

Win i yebɣa ad yaweḍ ɣer TILLAL TIS SNNAT ilaq ad yegmer IGMUḌIWIN ITFANIIN, IGMUḌIWIN ASTRAL awray, IGMUḌIWIN IMENTAN awray, IGMUḌIWIN ICAUSAL aqqiḍ neɣ IGMUḌIWIN N TAYRI TIḤEǦANIT.

Yella kra umiren ad yegzem inelmaden INUSTIKUS: IGMUḌIWIN ASTRAL, MENTAL d ICAUSAL d imessifen d iɣsan, d wara ma n-ilal gar taɛebbuḍt tamezdat n TMIMT TIḌEVINIT, ssagden aɣrum i wesnerni d usnerni-nsen.

Llan sin n anawen n imessifen: amezwaru d imessifen i d-yettasen seg ADAM; wis snnat, d imessifen i ULAC d-yettasen seg ADAM. IGMUḌIWIN ITFANIIN d imessifen i ulac d-yettasen seg ADAM.

Iḥemmel ad neẓer d HIDRÓGENO ASLULLUS SI-12, yal yettwakel deg imessifen d iɣsan. IGMUḌIWIN iPHYSICUS d imessifen d iɣsan, u igmuḍiwen SOLARES daɣen d imessifen d iɣsan.

Aɣrum agejdan n IGMUḌIWIN iPHYSICUS d HIDRÓGENO ARBAƐIN U ḤAMIṬ.

Aɣrum agejdan n IGMUḌIWIN ASTRAL d HIDRÓGENO VENETICUATR.

Aɣrum INDISPENSABLE n IGMUḌIWIN IMENTAN d HIDRÓGENO ḌUZE.

Aɣrum ḤIYAT n IGMUḌIWIN ICAUSAL d HIDRÓGENO SEIS.

Akk IMASTURIS n LUSGA TMIMT, ANILIS, ARCÁNGELES, TRONUS, SERAFINES, VIRTUDES, atg., atg., atg., ttwalesan s IGMUḌIWIN ITFANIIN.

Ala wi yegmren IGMUḌIWIN ITFANIIN yegmer ENCARNADO ɣer SER. Ala wi yegmren SER yella D argaz n ẒIDET.

IGMUḌIWIN iPHYSICUS yettwaḥerrez s arbaɛin u ḥamiṭ agitas, IGMUḌIWIN ASTRAL yettwaḥerrez s veneticuatr agitas, IGMUḌIWIN IMENTAN yettwaḥerrez s ḍuze agitas; IGMUḌIWIN ICAUSAL yettɣim ɣef se agitas.

D URGENTE ad neḍru ɣer TɣILT YECAN n VULCANO (ASLULLUS), ad n-xdem s utfi d waman, aẓar n imeɣranen, iɣersiwen, irgazen d UḌIBDISIN; d urgent ad neḥdul ɣer tsegziwt tẓa iwakken ad negmer IGMUḌIWIN ITFANIIN d ad n-awi ɣer TILLAL TIS SNNAT.

Causa aɣuṛu ad neẓer d waṭas iḥessben iman-nsen d IMASTURIS d SANTUS, mazal ttwalesan s IGMUḌIWIN ILUNARES.

### TAMETTANT

Yelluḍ ɣef tserwelt i CONDE GABALIS mi yeqqar d isalamandras, gnomus, silfus, ninfas, ssagden ad zgaren d argaz iwakken ad uɣalen d wid ur yettmettaten.

D umeslub aɣḍas n CONDE GABALIS mi yeqqar d nekkni ssagden renunsiación s tserwelt ɣer tmettutin iwakken ad n-efk i lmara n SÍLFIDES D NINFAS.

IILMENTALIS n IILMENTUS, n imɣan, n tgerǧumin, iɣersiwen, ad ilin d irgazen n azekka war sseḥḥar ilezɣan i yettwasemmeln sɣur CONDE GABALIS.

D laẓda d waṭas MÉDIUMS n uspiritiṭism zgaren d IILMENTALIS d waṭas n umdan deg usenṭeq ttidiren d ÍNCUBUS, SÚBCUBUS d IILMENTALIS n akk anaw.

IIMEɣRANEN IZDEXLIANIN tettef akk aɣlan n imeɣyanen, kra n uselḥi, kra n wayen n dir, kra n ulac fellasen.

IDIVAS neɣ ANILIS, jamas d igemmaḍ i YIRGAZ. IDIVAS neɣ ANILIS d IIRGAZEN ITFANIIN TISIDET u ayagi d akk. IDIVAS neɣ ANILIS d SIN TIQAQAḌ ILULANIN.

I UYCINAS, sin n anawen iɛellalen n imaɣlaliben isurfanin d THIEN n tɣalt tasimanit d THI, THU neɣ imlalen.

Deg idesfilederus n KUEN-LUN, tamnaḍt tamesralit n tmurt neɣ MONTUS ILUNARES, targit tbeddel akk yiwen n umeɣruḍ amseggas d tumert yettwaḥerzen sɣur UḌIBDISIN.

Imeɣyanen-a IḌEVINIIN d IKU-HAN neɣ ILOANES UḌIBDISIN IMEQRANEN n imelyunen n imeɣyanen.

THI ttwalesan tizerwin tizwiṛanin d ttidiren deg kriptas neɣ igelmimen uzemri; tteɣɣisen seg sésamo, coriandro d tijeǧǧigin d tifeḥḥatin n umaɣuɣ n tudert; d SIN TIQAQAḌ ILULANIN, ttɣaren ALKIMIA, BUTÁNICA TUFFER, PIEDRA FILOSOFAL deg tikli n IMASTUR ZANONI d UMEẒZA-INES ARƔIQ, AMEQQWREN MEJNOUR.

Tigezmt tis kraḍ n imaɣlaliben isurfanin d ifabulosus SHEN neɣ SHAIN, ilulen daɣen deg umeɣruḍ SUBLUNAR, iwakken ad xedmen i uselḥi, ad ḍellen KARMA ANCESTRAL.

Tigezmt tis kuẓ n imaɣlaliben n imeɣranen izdexlianin ittwabedren sɣur UYCINAS, d TENEBROSUS MAHA-SHAN, imezɣiyen iǧehden n TSAHEṚA TIRƔI.

Imeɣyanen iǧehden d ur yettwaḍalen d imarqanen d TERRIBLUS MARUT neɣ TURAM; imeɣyanen ittwabedren sɣur RIG VEDA, legiunes n AANASMUSSIANUS; awal-agi yettwareqqi s j, akka: IJANASMUSSIANUS.

Ilegiunes-a ttɣiment seg teltmaẓrat d tlata taɣelmin, am wakken kra n iḥisaben isawalen-d ameqqran ɣer 823 neɣ 543 taɣelmin.

D laẓda AANASMUSSEN-a ttwaɣuren sɣur kra n imeslemen d iBRAHMANIS.

IJANASMUSSIANUS gmeren, akken nenna deg yixef umnber tẓa n udlis-a, snat talɣiwin timanen; ta TANGELLUST d tayeḍ DIBULIST.

D ubelḥi talɣa TASIMANIT, TANGELLUST, n yiwen n AANASMUSSIANO, jamas yettɣimi ad yesselmed yiwen n ikandidat ɣer TINIZIACIÓN, war s-yini amezwaru s TILILA TAMEQQRANT: "ĞARDE, NEKKNI NELLA ȚENTACIÓN AY UMIREN AD YUƔAL ɣer UNFIEL".

Iẓer aɣer talɣa TASIMANIT n yal MARUT neɣ TURAM, AANASMUSSIANO, d yegmer tayeḍ TALɣA TALUNAR, DIBULIST, TENEBROSA, s tezmer ad sderɣlen ɣer ikandidat ɣer TINIZIACIÓN.

S deffir akk sin n tiqaqaḍ ILULANIN teflalen sin wiberdan, wi iẓerman d wi siniistra. Wi iẓerman d wi yettaren iman-nsen ad MMTEN seg teswiɛt deg teswiɛt, wi DISUELVEN EL YO. Wi n SINIESTRA d ubrid aberkan, ubrid n wi deg uɣer ad MMTEN seg teswiɛt deg teswiɛt, deg uɣer ad DISOLVER EL YO, tiseḥḥinen gar IGMUḌIWIN ILUNARES. Wi irewlen s ubrid n ufus ajermani, yuɣal ɣer MARUT neɣ TURAM, anamek-ines, ɣer AANASMUSSIANUS.

Win i yebɣan ad yaweḍ ɣer TILELLI TANAGGURT, ilaq ad MMTEN deg teswiɛt deg teswiɛt. Ala MMTEN EL Mí MISMO ad nuɣal ɣer ANILIS PERFECTUS.

Llan kraḍ n telmas n TANTRISMO, AMELLAL, AGGAẒ D UGRIZ. MAITHUNA s eyaculación n ENS SEMINIS, yella AGGAẒ. MAITHUNA kra n tikkelt s eyaculación n ENS SEMINIS u kra n tikkelt war eyaculación, yella AGRIZ.

S MAITHUNA war eyaculación tlelli DEVI KUNDALINI seg ubrid medular i wesnerni n tezmert TIḌEVINIT d ad nuɣal ɣer ANILIS.

S MAITHUNA s EYACULACIÓN tamessast TÍGNEA N TZERMITN-NEƔ MAGIKIS, deg uɣer ad tlelli, telaɣ, yettraqqeq seg yeɣes COXÍGEO, ɣer iɣerbaten iṭumikanin n argaz, ad yuɣal ɣer AKERRUΣ N SATÁN.

MAITHUNA s eyaculación kra n tikkelt u war eyaculación kra n tikkelt, d kra ulac-as lɣerḍ, amurẓebus, aɣersan, ay kan ad yewḍen ɣer IGU LUNAR.

ITÁNTRICUS IGGADEN snernin ABOMINABLE ÓRGANO KUNDARTIGUADOR. Ilaq ad neẓer d ÓRGANO FATAL-a d AKERRUΣ n SATÁN.

Deg teswiɛin i yettḥeqqiren deg iṭ tneɣɣim n akk tidmiwin, ANIL IMEQDIṚ ilemẓi iɛezen asiti-ines aɣuṛan n tegnut i ssagden i tusnakt n tebtert u yebɣa ad yemmet; tella teswiɛt taɛrabit n kra n INDIVIDUUS IMEQDAREN ay yugren ad yefken i temlilt-a tameɣlalast tumant, abominable ÓRGANO KUNDARTIGUADOR.

Mi ANIL IMEQDIṚ yettu asiti-ines aɣuṛan n tegnut d ibɣa akk teɣaltin n umeɣruḍ-a, abominable ÓRGANO KUNDARTIGUADOR yettwaɛerzen; mebla aɣlaq tisuggasin yellan d diri n iǧhed aɣerni d ayagi ulac i yes umiren ad yettu, yuɣal yegmer deg xemsa icycles n tmasint.

Icylu amezwaru d n INTELECTU u yella deg CEREBRO; wis snnat d n IMUSYUNIS u yeggmer deg PLEXO SOLAR, ɣer taɣla n temɣart; wis kraḍ d n UMECRAḌ u yella deg tama TASAWARANIT N ESPINA DORSAL; wis kuẓ d n INSTINTO, u yella deg tama TASEMGEMTA N ESPINA DORSAL; wis xemsa d n ASLULLUS u yeggmer deg ilalen ASLULLUSIN.

Tisuggasin yellan d diri n abominable ÓRGANO KUNDARTIGUADOR yettwasemmel s illuɣmen d imelyunen n YOES n anaw aɣersan d unerẓay.

Deg ANIL IMEQDIṚ ulac yiwen n ammas n lamanḍu, war YO neɣ IGU yettɣimen.

Yal IDEA, yal imsila, yal isensa, yal ireb, yal YO ȚIREB taɣawsa-a, YO ȚIREB TAYEḌ TAɣAWSA-A, YO AMU, YO ULAC AMU, d yiwen n YO yettxallafen.

Akk YOES-a iǧehden d irqqaqan yettruqalen gar-asen, ṭṭeggalen ɣef isawalen, ulac yettwaqeqten gar-asen, war idriwen. Yal yiwen seg YOES-a iǧehden yettɣimi ɣef imeɣraden n teswiɛin n tudert d imeɣraden n imsilan.

Yal YO ilemẓi yegmer igezza-ines n tidmi, aɣḍas-ines, ulac tidmi TIDET deg ANIL IMEQDIṚ ilemẓi, aɣḍas-ines, ixeddimen-ines, igezza-ines, yettɣimen ɣef YO i yettwaḥerzen deg tswiɛin-a asiti.

Mi yiwen n YO ibda akk GNOSIS, iwer-d ttililli tasimanit ɣer UMECRA-NEƔ INUSTIKUS; lbayaɣ yettɣimi alama yiwen n YO i yettxallafen ɣer tɣeriwin-a, ad yexza azmer, d deffir nneẓra s taɣuṛu d asiti yettɣimi u yaweḍ ɣer tuɣalin d aɛdaw-neɣ.

YO i yettewer seg ass AMU TAIMEZDIT ɣer tameṭṭut, yettwexza sɣur YO tayeḍ i ulac yettɣimen ɣer tamacahut u tamettut tettergagi.

YO amellal yettɣimi iman-is ɣer tayeḍ d kra ttkkan yal d imlalayen s tayeḍ, maca ulac gar akk YOES-a kanun neɣ isiti.

Yal yiwen seg YOES-a yumen deg teswiɛt yellan d akk, maca ulac s tidet ugar n yiwen n tama tatimezrat n igerdan-nneɣ, meqqar yegmer usiti yellan d akken, d tidet, argaz iǧehden.

Ayen iḥemmlen d nekkni yefkan agitas ɣer YO n teswiɛt, meqqar teswiɛin deffir YO-a yettwexza sɣur YO tayeḍ. IGU LUNAR d tasumma n YOES i ssagden ad yettwaɛezzen deg talɣa taẓarant.

Ilaq ad neẓer d yal yiwen seg xemsa icycles n tmasint yegmer ilaqen-ines i nekkni ur yellan i nelweɛ.

Gar xemsa imassan n tmasint yella ixelleɛ seg urured.

Aɣlan issin aṭas aɣḍas, maca s tidet n tidet ammas IMEQDIṚ d amenzuy aṭas. D deffir, meqqar aṭas ugar n teɣzi, ttasen IIMMASSAN INSTINTIVO d UMECRAḌ neɣ MUTUR, i yessemalen ugar n teɣzi n twala. Igaǧhen n akk d AMMAS ASLULLUS, d yeɣmi deg umnber n teɣzi, AMMAS IMUSYUNAL.

Yella ixelleɛ ameqqran n teɣzi gar yal yiwen seg xemsa IIMMASSAN n tmasint.

Netɣer deg iman-nneɣ autu-observándonos, ad nẓer s tiṭ d UMECRAḌ iǧehden aɣḍas u d imsila iǧehden yiwen n umecraḍ u yal yiwen n aɣḍas.

IIMMASSAN MUTUR d INSTINTIVO d tlataɣraḍ imelyunen n tikkelt ugar n teɣzi ɣer AMMAS IMEQDIṚ. AMMAS IMUSYUNAL mi yettxeddem ɣer teɣzi i yellan-is iman-is, d tlataɣraḍ imelyunen n tikkelt ugar n teɣzi ɣer iimassan MUTUR d INSTINTIVO.

IIMMASSAN yegmren yal yiwen seg-sen TUERMT yettxallafen. Teɣzi n iimassan tesselmed aɣlan ameqqran n ifinuminus iǧehden i tussna tamaynut ur tzemmer ad tesselmed; ssagden ad nekkter teɣzi taɣuṛant n kra n iprususus isukulijigus, fisiulijigus d imentalis.

Yal AMMAS yettwagezzem deg sin idgan: positif d negatif; agezzem-a d iǧehden aɣer i AMMAS IMEQDIṚ d ɣer AMMAS INSTINTIVO.

Akk axeddim n AMMAS IMEQDIṚ yettwagezzem deg sin idgan: AFIRMACIÓN d NEGACIÓN; SÍ d ULAC, TESIS d ANTÍTESIS.

Deg AMMAS INSTINTIVO yella amennuɣ gar wayen iḥemmlen d wayen ur iḥemmlen; isensa iḥemmlen, isensa ur iḥemmlen u akk isensa-a yettwasmallen s xemsa imeẓẓuɣen: ad nẓer, ad nesler, ad nemɣer, ad nseɛɛa, ad nenal.

Deg AMMAS MUTUR neɣ n UMECRAḌ, yella amennuɣ gar UMECRAḌ d ARESTU.

Deg AMMAS IMUSYUNAL yella imusyunis iḥemmlen d ur iḥemmlen: tafruxt, tasamɣat, alɣuɣ, tilelli deg yiman, atg., d positif.

TIMUSYUNIS ur iḥemmlen, atayen akken alɣuṛu, iceggiten, tinqes, tinqes, aɣeddu, irqiqen, tugdi, d negatif akk.

Deg AMMAS ASLULLUS yella tareb d tirebt, taɛezza d taɛezza deg amennuɣ yettɣimen.

ANIL IMEQDIṚ yettwakal iɣerḍisen-ines ma yella d azalen, maca yella d umeslub ad yettwakal igerdan-ines iman-is.

Win i yebɣan ad DISOLVER EL YO PLURALIZADO, ilaq ad yettwakal igerdan-ines iman-is. ICEGGITEN d beddilen iɛessasen, ma nenqes iceggiten, temmet taɛessas, yettwakal aɣuṛu.

IRA d yettbeddel AƔUṚU; ma nekkes s IRA nettwaklel AƔUṚU, nettwahdem.
