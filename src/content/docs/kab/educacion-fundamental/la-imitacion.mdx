Yettubeddel dɣa, bla ccekk, belli tugdi teṭṭef lḥurriya n usnulfu. Imeẓriyen n wid yettidirin deg tegnit yellan d tiɛessast, yal wa deg-sen, d tamentilt n tugdi.

Yal aqcic yettaɣ-it tugdi, yettaṭṭaf yemma-s yeɛzizen, yeɛreq ɣur-es ad yesɛu laman. Argaz yettaɣ-it tugdi yettaṭṭaf tameṭṭut-is, yeḥusseɣ-tt iḥemmel-itt aṭas. Tameṭṭut yettaɣ-itt tugdi tettnadi ɣef urgaz-is d warraw-is, teḥusseɣ-ten iḥemmel-iten akter.

Si tama n tussna n nefsi, yesɛa azal aṭas, yerna d aneggaru ad nessen belli tugdi tetteḥubu aṭas deg ulibas n tayri.

Medden yellan ur sɛin ara azal urumi deg ulawen-nsen, medden meḥqur deg ulawen-nsen, tettnadin daymen deg barra kra ad kemmlen iman-nsen.

Medden meḥqur deg ulawen-nsen, ttidiren daymen s tneqqiḍin, daymen deg tiderɣal, deg teḥkayatin, deg leḥwayeǧ n lweḥc, atg.

Medden meḥqur deg ulawen-nsen ttidiren seg tugdi ɣer tugdi, akken i d-tlul, ttaṭṭafen argaz, tameṭṭut, imawlan, arraw, idles aqdim yellan d isserfen d iwumi, atg. atg. atg.

Yal argaz aqdim ameḥqur deg tussna n nefsi yettur dɣa s tugdi, yettugwad s ttawil ur neqdis ɣer idrimen, ar idles n tawacult, ar yelmemẓiyen, ar tira-s, atg. am akken yettnadi ɣef laman. Ayagi d ayen i nezmer ad t-neẓer s wallen nneɣ ma nennaɣ i lejdud.

Yal mi ara ttewqedren medden, ttekrezen deffir uleqziz yettaḥazen si lḥirma. Ttabaɛen idles, ama n tgelda, ama n tawacult, ama n tmurt, atg. atg. atg.

S tidet, yal idles d tamsalt n tuttra ur yesɛin ara sser, d ilmaɛna, ur yesɛin ara azal n tidet.

Medden meṛṛa sɛan tazzla s tuggda ad ḍefren ayen n wiyaḍ. Ayen-nni n uḍeffer d tadyant n tugdi.

Medden s tugdi ḍefren wid akk iṭṭfen. Ḍefren argaz, tameṭṭut, arraw, atmaten, imdukal i ten-yettḥazzen, atg. atg. atg.

TAMEẒRA d taggara n TUGDI. TAMEẒRA tḥettem akken ma tella LḤURIYA N USNULFU.

Deg yiɣerbazen, deg tikeltawin, deg tesdawiyin, iselmaden d tiselmadin ttaggin tumert ad slemden i inelmaden d tlemẓiyin ayen i wumi qqaren TAMEẒRA.

Deg tɣuriwin n tẓuri d usɣan, slemden i yinelmaden ad ḍefren, ad sɣanen tugniwin n tmeddit, ixxamen, idurar, iɣersiwen, atg. Ayagi mačči d usnulfu. Ayagi d AḌFEṚ, D TASSAWERT.

Usnulfu mačči d AḌFEṚ. Usnulfu mačči d TASSAWERT. Usnulfu d aseɣẓen, d assefraq s taɣriwt d tayri tmeddit i ɣ-yeɛjben, tafat n teɣri n tafat n yiṭij, tafat n ṣṣbeḥ s ttilas-is ur nettwattun ara, atg. atg.

Yella usnulfu n tidet deg uẓuri n Ccinwa d Japon n ZEN, deg uẓuri azawan d Semi-Azawan.

Ami ara yesseɣraṣ umazray n umazray n Ccinwa n CHAN d ZEN ur t-yetteɛni ara AḌFEṚ, tassawert. Imazrayen n Ccinwa d Japon: tteffriḥen s usnulfu d tuɣalin d usnulfu.

Imazrayen n ZEN d CHAN ur ttaḍefren ara, SNULFEN, d ayagi i d leqdic-nsen.

Imazrayen n CCINA d JAPON ur t-yetteɛni ara ad sɣanen neɣ ad ssiweḍen tassawert ɣef tameṭṭut tecbeḥ, tteffriḥen s usefraq n tecbeḥt-is azawan.

Imazrayen n CCINA d JAPON ur t-id-tettḥaz ara ad ḍefren tafat n teɣri n yiṭij, tteffriḥen s usefraq deg tecbeḥt tazawant ayen akk yeččan lḥal n tmeddit.

Aya i yesɛan azal mačči d AḌFEṚ, d aseɣẓen deg uberkan neɣ deg wumellal; aya i yesɛan azal d aseḥbiber n sser ameqqran n tecbeḥt d ussnen-is ad t-nsefraq, maca akken ad yesɛu, yessefk ur yelli ara tugdi, taṭṭuft ɣer lqwanin, ɣer idles, neɣ tugdi ɣer wayen ara yinin neɣ ɣer tqebḥat n uselmad.

D URGENT ad fehmen iselmaden d tiselmadin taɛkemt belli inelmaden d tlemẓiyin ad snernint tazmert tusnulfit.

S tidet, d ayen ur nezmir ara ad nessen, ad nslemden i yinelmaden ad ḍefren. Yugar ad ten-nslemden ad snulfun.

Amdan s tɣawsa d amuddir yettergen ur netfehhem ara, anda kan i yessen ad iḍfer.

Nettaḍfar albas n wiyaḍ, seg uḍfeṛ-agi ttfuɣent tazzlatin yemgaraden n taɣersiwt.

Nettaḍfar leḥrama, nettaḍfar ayen akk i d-tettasen, ayen daymen yettuttrun deg tallit, atg.

Yessefk iselmaden d tiselmadin n yiɣerbaz ad slemden i yinelmaden ad xtiren iman-nsen deg usentel ameqqran.

D URGENT ad ssnen yinelmaden tilelli n tidet, akken, war tugdi, ad zemren ad lemden ad xtiren iman-nsen, s tilelli.

Tixsi yellan d aqeddac n WAYEN ARA YININ, tixsi yettaḍefren, ɣef tugdi ad tḥettem idles, lqwanin, tidet, atg. Mačči d tixsi tusnulfit, mačči d tixsi tilellit.

Tixsi n medden d am wexxam yeqfel yerna yesɛa ṣṣbaɛ leɛqud, axxam anda ur yezmir ara ad yili kra isemlal, axxam anda ur yekcumeɣ ara tafat n yiṭij, axxam anda ur yeḥkem ara kan lmut d lemriḍ.

Ayen IṚUḥAN yezmer ad yili anda ur tili ara tugdi, anda ur tili ara TAMEẒRA, anda ur llin ara ttaṭṭufat ɣer tɣawsiwin, ɣer idrimen, ɣer medden, ɣer idles, ɣer tidet, atg.

Medden ttidiren d iqeddacen n tneqqiḍin, n tḥesdatin, n tidet n twacult, n leɛwayed, n tamat i ur yesɛin taggara ad rebḥen imukan, ad ɛalin, ad zegren ɣer tili n tesselliwt, ad ten-ḥussen, atg. atg.

D URGENT ad slemden ISELMADEN d TISELMADIN i inelmaden-nsen d tlemẓiyin taɛkemt belli ur ttaḍefren ara wayagi akk d leqwanin isserfen d iwumi n tɣawsiwin tiqdimin.

D URGENT ad lemden INELMADEN deg yiɣerbaz ad snulfun s tilelli ad xtiren s tilelli, ad ḥussen s tilelli.

Inelmaden d tlemẓiyin ttaɛebbaɛen ayen i yellan d ayen yelhan deg tudert-nsen deg yiɣerbaz akken ad ḥeṣṣlen AMAWAN meɛna ur yesɛin ara lawan akken ad xtiren deg wayagi akk.

Ɛecra neɣ ḥdac n yiseggasen deg yiɣerbaz ttidiren tudert n imuddar ur netfehhem ara, ttfuɣen seg yiɣerbaz s tfehmiwt ttergent, maca ttfuɣen seg yiɣerbaz ttmennin belli ttfhemen aṭas.

Tixsi n umdan ttidiret deg leqraɛ i d-yewwin gar tektiwin tiḥerqanin d yewwiɣzen.

Amdan ur yezmir ad ixtir s tidet tilellit acku yettur s TUGDI.

Amdan yesɛa TUGDI ɣer tudert, TUGDI ɣer lmut, TUGDI ɣer wayen ara yinin, ɣer wayen qqaren-t qqaren-t, ɣer teḥkayatin, ɣer aseqdec n leqdic, ɣer uḥettem n unectiɣ, ɣer win ara yerren tameṭṭut-is neɣ ad yuker tameṭṭut-is, atg., atg., atg.

Ma nennaɣ ad nternu d imezwura deg teɣzi n awal, yessefk ad nxeddem taɣri n leqdic-nni n IMUẒṚAN i ɣ-yettḍeggiren.

Mi nzemmer ad nessen leqdic-nni akk n IMUẒṚAN, mi nessefraq yal wa seg IMUẒṚAN, nettili d imfehmen, yerna d taggara tusridt, ttelul-d deg-neɣ deg umalas, tazmert ad nsnernu.

Ttaggin tumert iselmaden d tiselmadin yetmennin ɣalat belli inelmaden d tlemẓiyin ḥwaǧen ad ḌEFFREN akken ad lemden. Win IḌEFFREN ur yellemdu ara, win IḌEFFREN yettili d AMUDDAR, d ayagi kan.

Ur nettnadi ara ad NḌEFFER wayen ara yinin imasnen n taɣult, tussna, taḥesbiyt, tarikt, atg. AḌFEṚ, ASEḤFAẒ, TUTRRA am tbebbuṭin neɣ d tselɣaɣ, d leḥrem, yugar ad NEFHEM S TFEHMIWT ayen nɣur deg-s.

TAƐLIMT TISLAM d TUSSNA N TFEHMIWT, tussna i ɣ-yettaǧǧan ad naf ṛṛezq-nneɣ s medden, s tmeddit, s tɣawsiwin meṛṛa.

Tixsi i yessnen kan ad tḌEFFER d TAMEKNIKANIT, d tamsalt i tɣur, MAČČI tusnulfit, ur tezmir ara ad tesnernu, ur ttefehhem ara s tidet, tettuɣal-d kan d ayagi kan.

Iselmaden d tiselmadin ad ttɛebbiɛen s usteqsi n TFEHMIWT deg yal anelmad.

Inelmaden d tlemẓiyin ttaggin kan ad zegren aseggas d deffir… mi ttfuɣen seg yiɣerbaz, deg tudert tusridt, ttilin d imseɣlayen n leqdic n uɣerbaz neɣ d tiselɣaɣ n ixeddimen n warraw.

Ɛecra neɣ ḥdac n yiseggasen n tɣuri akken ad tfuɣen beddlen d imeḥduffen yettmeslayen, leḥwayeǧ i tɣurem ttwattunt ciṭṭuḥ, deg taggara ur yettɣimi ara kra deg tikti.

Ma xedmen inelmaden TFEHMIWT n leḥwayeǧ i tɣuren, ma ulac tɣuri-nsen tettaɣ kan ɣer AMAWAN, ɣer TAMEẒRA d USEḤFAẒ, ad tɣurɛen tayda. Ad tfuɣen seg yiɣerbaz s tussna TTEFHAMENT, UR TTWATTUN ARA, TIKEMMAMIN, i ur tettwagzam ara ɣer TIKTI UR YESƐIN ARA LḤAL.
