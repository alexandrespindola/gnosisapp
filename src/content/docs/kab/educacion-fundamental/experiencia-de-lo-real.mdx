Deg yiri n tewwurt yesɛan ccan n lǧameɛ n Delfus, yella tira tazerfant tettwakrez deg uẓru yeddren, i d-yeqqar: "UR TEẒRI ARA NEFS-IK". Ẓer iman-ik ad tẓreḍ aglim d Ṛebbi.

Tussna taneflit n tmeditasiyunt tesɛa am uẓru agejdan lemtel aqedsan-agi n Hiirufantat igrikiyen iqburen.

Ma nebɣa s tidet d ṣṣeḥ ad nesbedd lsas i tmeditasiyunt yellan d ṣṣeḥ, ilaq ad neɛqel iman-nneɣ deg yistraten akk n wallaɣ.

Asbeddi n lsas yellan d ṣṣeḥ n tmeditasiyunt d tidet, ad nili d ilelli seg tmuɣli, tumert, lxuf, leɛḍu, rreɣba n tezmerin tizdigin, uẓẓun n igmad, atg., atg., atg.

Yebna i medden akken ma llan d ur yelli kra n ccekk mara ad nesbedd AẒRU AGEJDAN n tmeditasiyunt allaɣ ad yeṭṭef iman-is d asusruɣ iqesḥen.

Seg tama n wemcabi i d-yeṭṭfen, d aɛeqli mara ad nebɣu ad nexdem AYEN YELLAN s tidet mbla ma nessen iman-nneɣ.

D uraren ad neɛqel deg uswir ILELL d deg yigen akk n wallaɣ, yal ugur akken it-id-iɣelli deg wallaɣ, yal aɣun, yal tamawt, yal lɛib aɣiman, atg.

Yebna i medden akken ma llan akken deg tmerriwt n tmeditasiyunt, ad ɛeddin deg telaɣt n wallaɣ deg umussu amekɛuz, akk lɛib aɣiman i ɣ-iẓerren, akk tafersa-nneɣ d leḥzan-nneɣ, timawin ur yeḥsaben, timsirin yemgaraden i d-yekkan ama seg umaḍal n berra, ama seg umaḍal n ddaxel, iɣunaten n yal tajumma, lemɣawla n yal asatal, tiɣellatin tiqdimin, leɛḍu, atg.

Win ibɣan s tidet ad yesbedd deg wallaɣ-is aẓru agejdan n tmeditasiyunt, ilaq ad yefk tamawt ifullen deg watanen-agi ufazen d wid ur nefɛi n tmuɣli-nneɣ d ad ten-yeɛqel deg uswir ilell mačči kan deg uswir ameskili kan, maca daɣen deg yigen akk uferallaɣen, d wallaɣen ur nettwal d wallaɣen ur nelliyen n wallaɣ. Ur ilaq ara ad nettu daɣen akken wallaɣ yesɛa aṭas n wusuiren.

Talɣa talemmast n watanen-agi akk d tidet, d tmusni n iman.

Yal akatuf deg telaɣt n wallaɣ yesɛa aseqdec d taggara. Mara ad yefnu udɣar n tugniwin, iɣunaten, lemɣawla, tamuɣli, timawin, atg., imiren allaɣ ad yeṭṭef iman-is d ususruɣ iqesḥen ILELLI seg yal tajumma n ixemmimen.

Inelmaden imezwura n tusnakt ufusenṭal gant ad sɛun tajriba n ULELLI IFERREREN. Aɣelluy n ULELLI ddaxel n wallaɣ-nneɣ yettarra ad nexdem tajriba, ad neḥsus, ad nedder amedyaz i ibeddilen, amedyaz-agi d AYEN YELLAN.

Ssemgired gar wallaɣ i yeṭṭfen iman-is d wallaɣ i yettwaṭṭfen s ṭṭerf.

Ssemgired gar wallaɣ i yellan d asusruɣ d wallaɣ i yettususruɣ s teɛkumt.

Deg tafat n kra n tuẓẓma taɣiwant ilaq ad neɛqel akken mara allaɣ yettwaṭṭef s teɛkumt, deg uẓar d deg yistraṭen nniḍen ur yeṭṭef ara iman-is d yettɛeggid akken ad yili d ilelli.

Seg tama n wemcabi ilaq ad neɛqel akken mara allaɣ yettususruɣ s teɛkumt, deg uẓar ur yelli ara d asusruɣ, yettɛeggiḍ d yettɛejjal s tɛkumt tameqqrant.

Tudert tususruɣ d taɣiwant n wallaɣ, tettas-d ɣur-neɣ am leɛnaya, am lferḥ, mara tfak telaɣt tuklalin n tudert-nneɣ deg telaɣt yessewhamen n uẓar.

Ala mara allaɣ yella d taɣiwant d ilell, ala mara allaɣ yettwaf deg ususruɣ yeɛzizen, yettas-d uɣelluy n ULELLI IFERREREN.

ULELLI ur yeshel ara ad t-nessefru. Ur yelli ara d aḥuddu neɣ d assefru, yal amedya i nettawi fell-as yezmer ad yeɣweḍ deg usatal amenzu.

ULELLI ur yezmir ad t-nessefru neɣ ad t-nessegzi s wawalen. Ayagi yella axater tutlayt tameɣnast tettwaxdem s timmad i ismawen n tɣawsiwin, ixemmimen d imussuten illan; ur teffeɣ ara ad tessegzi deg uswir yellan d ilell d uselkem, tiggura, tɣawsiwin d imussuten UR ILLAN.

Anezwi ad nessawal ɣef ULELLI deg tlisa n yiwet n tutlayt tettwexḍen i tugniwin n tmetti, s tidet d tɛeqlin, yella d ahbal d dɣal qessiḥ.

"ULELLI d UR YELLI, D AYEN YELLAN UR YELLI D ULELLI".

"TALɣA UR TEGMIR ARA SEG ULELLI, D ULELLI UR TEGMIR ARA SEG TALɣA".

"TALɣA D ULELLI D ULELLI D TALɣA, YELLA ɣEF ULELLI I TɣAWSIWIN LLAN".

"ULELLI D AYEN YELLAN SEWWIREN GER ATAS D UR TTƐARADEN ARA". ULELLI D AYEN YELLAN SSERWEN GER ATAS D TTƐANQAQEN.

"MARA IMUDAR N TUSSA KKAWEN TIZZERT, KKAWEN ALA TAMḌUT-IS I YELLAN, UR KKAWEN ARA TAMḌUT-IS ILELLAN".

"Yal AMUDAR IFERREREN yezmer ad yakku tamawt s yiwet n tikkelt i tamḍut illan d ILELLAN n kra n tɣawsa.

"ULELLI d awal kan i yettarra d tuttra TAYERMAN n tɣara UR YESƐIN ASEMḌU D UR YESƐIN ISEMḌU n yimudaren, d taɣara n ussekni n tɣalt n tukrist tufellit d tlelli".

Iselmaden d tisselmadin n tɣiwatin, tmezguniwin d tilas d tilas-tlmadent gant ad lemden d tɣerma talemmast i tusnakt-nneɣ taneflit d imiren ad slemden i inelmaden-nsen abrid i yettarran i tmerriwt n AYEN YELLAN.

Yella kan uraren ad naweḍ i TMERRIWT N AYEN YELLAN mara ixemmem yefna.

Aɣelluy n ULELLI yettarra-yaɣ ad nexdem tajriba n TAFAT TULZAZT n TIDETT ILZAZT.

TUSNA-NNEK TILLAN deg tidet ILELLAN, mbla taɣara d mbla ini, ILELLAN SEG TɣARA, d TIDETT TAɣIWANT, TUFFAɣT TAɣIWANT.

TMENT-IK yellan d tidet d ULELLI i ur ilaq ara ad ttwakal am ULELLI N WAR TɣAWSA MACA AM TMENT S TIMMAD MBLA AZAR, tettferrir, taɣiwant d tafersant, d TUDERT, d BUDDA TAɣIWANT YESƐAN TAɣIWANT.

TUDERT-IK TUKLALANT ILELLAN d TMENT TULZAZT D TAFERSSANT d ur ttwaḍernen ara. ASELKEN-NSEN SEG DHARMA-KAYA; TAGHALT N TURRI TAFULZAZT.

TUDERT-IK TUKLALANT TFERRIR, ILELLAN d ur ttwaḍernen seg tɣawsa tameqqrant N TURRI, UR TESƐI ARA ILLUL d UMETTAN d Tafat UR TETTBDIL AMITARA BUDDA.

Tussna-agi tqad. Ad tesɛu ULELLI N TMENT-IK AM TAGHALT N BUDDA d ad tt-tḥsebbeḍ am TUDERT-IK, gant ad tɛeddi deg WADDUD AɣIMAN N BUDDA.

Ḥafeẓ AMAẒ-IK mbla ma tḥessbeḍ deg TMEDITASIYUNT, tettu deg tmeditasiyunt, ur xmimeɣ ara akken tmeditasiyunt axater mara txemmem akken tmeditasiyunt, ixemmem-agi yeqad ad yewhem tmeditasiyunt. ALLAɣ-IK ILQ AD YILI D ILELLI akken ad tḥsuḍ AYEN YELLAN.
