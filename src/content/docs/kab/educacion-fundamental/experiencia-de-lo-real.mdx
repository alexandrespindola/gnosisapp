Deg umalu ussan d imeqqranen n lǧameɛ n Delfus, tella tira taqdimt yettwaẓẓan deg uẓru yeddren, tenna: "UR TEƐREB TIKELT-IK". ƐReb iman-ik ad tɛerbeḍ aglam d Rebbi.

Tussna taɛlayant n Tmedyazt tesɛa aẓru alemmas agejdan n yennayer aqdim n YIHIRUFANTEN IGRIKIYEN.

Ma yella s tideț d ṣṣeḥ, nebɣa ad nessers lsas i tmedyazt yellan d ṣṣeḥ, ilaq ad neɛqel iman-nneɣ deg yal astawa n wallaɣ.

Asers n lsas yellan d ṣṣeḥ n tmedyazt d tidet, ad tiliḍ tilelli seg tmeɣṛa, tilawt, tugdi, asekkir, uḍmaɛ n tezmerin ticeɛɛatin, aɣilif n tiggatin, atg., atg., atg.

Yettban i medden merra, bla ccekk, sakin ad nessers AẓRU ALEMMAS AGEJDAN n tmedyazt, allaɣ yeqqim dɣa deg tussda d usemmiḍ meqqren.

Seg tama n usnas yellan d ussid, d lɛib ma nebɣa ad nesɛu tajriba n Tidett bla ma neɛqel iman-nneɣ.

D lemɣawla ad neɛqel s umata deg yal tarwa n wallaɣ, yal ugur mi d-yeffeɣ deg wallaɣ, yal laman, yal tira, yal lɛib n tussna, atg.

Yettban i medden merra belli deg tmesriḍt n tmedyazt, ad ɛeddin deg tilqit n wallaɣ deg lweɛda s tuffɣa, yal lɛuyub n tussna i d-yettweɛqalqen, yal lfeṛḥ d usseɣnu-nneɣ, tira gar-asen, aṭas n uguren i d-yettasen seg umaḍal n berra, neɣ seg umaḍal n daxel, laman n yal ṣṣenf, ccehwa n yal ṣṣenf, tiɣmiwin timɛeqranin, asekkir, atg.

Win yebɣan s tideț ad yessers deg wallaɣ-is aẓru agejdan n tmedyazt, ilaq ad yefk tamawt tameqqrant deg azal-agi yellan d ṣṣeḥ d yettuglan seg uɛqil-nneɣ, ad ten-yefhem s umata ur kan deg weswir n tmusni kan, maca ula deg yal tarwa n wallaɣ-nniḍen, wallaɣ-nniḍen, wallaɣ-nniḍen n wallaɣ. Ur nettettu ara belli allaɣ yesɛa aṭas n yestawaten.

Almud ussid n yizmawen-agi merra d tidet, d tmusni n iman.

Tilqit merra deg tilqit n wallaɣ tesɛa amezwaru d taggara. Mi tfukk lweɛda n ṣṣurat, laman, ccehwa, tmeɣṛa, tira, atg., imiren allaɣ yeqqim deg tussda d usemmiḍ meqqren, ILI deg yal ṣṣenf n uxemmem.

Inelmaden imaynuten n usselmad n tussna yessefk ad sɛun tajriba n ILI YEẒRIWEN. Tuɣilt n ILI deg wallaɣ-nneɣ teqqen ad nesɛu tajriba, ad nḥuss, ad nedder aglam yettbeddilen, AgLAM-nni d tidett.

Fruq gar wallaɣ yeqqimen d wallaɣ yettwaqqnen s tqehhrt.

Fruq gar wallaɣ yellan deg usemmiḍ d wallaɣ yettwaqqnen s teɣdemt.

Deg tafat n yal tallalt, yessefk ad neɛqel belli mi allaɣ yettwaqqnen s teɣdemt, deg daxel d deg yestawaten-nniḍen ur yeqqim ara yeqqim dɣa ad iɛiwed ad yettwaserreḥ.

Seg tama n usnas ilaq ad neɛqel belli mi allaɣ yettwaqqnen s teɣdemt, deg daxel ur yeqqim ara deg usemmiḍ, yesserɣay dɣa ad yettɛeqqem s tɛeccebt.

Tussda d usemmiḍ yellan d ṣṣeḥ d ussid n wallaɣ, tusa-d ɣur-nneɣ am lxeṛ, am lfeṛḥ, mi tfukk tilqit yellan d timsegreft tameqqrant n tudert-nneɣ deg tilqit tamellalt n uɛqil.

Ala mi allaɣ yeqqim deg tussda d ussid, ala mi allaɣ yettwafeg deg usemmiḍ yellan d lezzat, d-tettaɣ tuɣilt n ILI YEẒRIWEN.

ILI ur yeshil ara ad yettwaɣzel. Ur yettwaqqen neɣ yettwabder, yal amaɣun i d-nettaɣ fell-as yezmer ad yesɣer deg ussan agejdan.

ILI ur yezmir ara ad yettwabder neɣ ad yettwaɛqel deg awal. Ayagi d ayen d-yuɣalen ɣer awal n ufgan yettwaɛeqlen s ṣṣeḥ, ilaq ad yettwasmeres s ussan, axemmem d isseɣnu i yellan; ur yettɛeqqal ara ad yettwaɛqel s ṣṣeḥ d ussid, asenned, tiggatin d isseɣnu UR YELLIN.

Ad nɛiwed ad nemmeslay ɣef ILI deg tallit n yiles yellan d aqessar s ṣṣurat n tuddert, s tidet, d lɛib dɣa yegger fell-as ṣṣeḥ.

«ILI d UR-YELLI, d YELLI UR D ILI ARA".

"TIRGIT UR TFERREQ ARA ƔEF ILI, D ILI UR YFERREQ ARA ƔEF TIRGIT".

"TIRGIT D ILI D ILI D TIRGIT, D AYEN ILI I TIGGA I YELLAN".

"ILI D YELLI D ISEMMEGGEG ULA ƔER GAR-ASEN UR TTEDDUN ARA". ILI D YELLI TTEKKAN DɣA TTESƐAFAREN.

"MI ILAWILEN N TUɣALIT TAWLEƐT ZERRAN TAɣAWSA, ZERRAN ALA TAFAT-IS YELLAN, UR ZERRAN ARA TAFAT-IS ILI".

"Yal ILAWI YETTWAẒAN YEZMER AD IZER S TIKELT TAFAT YELLAN D ILI N YAL TAɣAWSA.

"ILI D AWAL ASERSI YETTWAQNEƐN TABAƐA UR TESSƐI ARA ASAL NEɣ TIKELT N ILAWILEN, D TMATARAT N TUɣALIT N UḤETTƐEƔ D TIRRI".

Lmuɛellmin d Tmuɛellmin n Yisdunen, Tiɣiwan d Tiɣiwanin ilaq ad ɣren s umata Tussna-nneɣ Taɣewlant uqbel ad slemden i yinelmaden-nsen abrid i yetteddun ɣer tajriba n Tidett.

Yeshel kan ad taweḍ ɣer TAJRIBA N TIDETT mi yefukk uxemmem.

Tuɣilt n ILI teqqen ad nesɛu tajriba n Tafat Taṣṣeḥ n TIDETT TAZDIGT.

Tamusni-nni YELLAN deg ILI s tideț, bla talɣa d bla ini, ILI N TABAƐA, d TIDETT TAZDIGT, LXIṚ UMATAN.

Tamusni-ik ta tidet tabaɛat-is d ILI ur yettwaẓer ara am ILI N UR-YELLI maca am TAɣELT TIKELT S TIMAN-IS bla tarɣa, taẓṛiwt, tazellalt, d Lferḥ d TUSSA, d BUDDA UMATAN YEƐQLEN.

TUSSA-IK TIKELT ILI d TAɣELT YEẒRIWEN D LEZZAT ur ferriqen ara. TUDSA-NSEN n DHAṚMA-KAYA; ASTAWA N TAZDAYT YELLAN D ṢṢEḤ.

TUSSA-IK TIKELT TAẒRIWEN, ILI d ur yettferriqen ara seg UɣERBU AGEJDAN N ZERMEZMI, UR TESƐI ARA TALALIT NEɣ LMUT d tafat ur tettbeddil ara AMITARA BUDDA.

Tamusni-agi teqqen. Aseqqiɛ n ILI n TAɣELT-IK TIKELT am ASTAWA N BUDDA d amaɣun am TUSSA-IK TIKELT, ad temmed deg UṚUḤ UɣERBU n BUDDA.

Ḥrez TAɣELT-IK bla ma tesḍeqṛeɛ deg MEDYAZT, tettu belli teqqimed deg Medyazt, ur ttxemmem ara belli tetteqdimeḍ imi mi ttxemmimeḍ belli tetteqdimeḍ, axemmem-agi kan yeqqen ad yegzem tamedyazt. Allaɣ-ik ilaq ad yeqqim ILI akken ad tesɛuḍ tajriba n Tidett.
