Argaz ameqqran d tagrest i gellan deg yixef-is deg wanda llant tewser n imeṭṭawen, maca yesɛa, ttḥeqqa dɣa armi d ccwal ad yettuḥseb d agellid n tgama.

"Eẓẓer iman-ik" "Argaz, ẓer iman-ik". D ayen yuran deg yigider n ddheb aqdim yettwarun ɣef yiɣriben ur yettnernin ara n lǧameɛ n Delfus deg Ǧrika taqburt.

Argaz-nni, aɣersiw ameggag-nni iggulen sɣalet ad yettuḥseb d argaz, yesnulfa-d ugar n ulfan n tmaswaṛin imeqqranen yeqqaren akken yella yessen s teɣri ɣef tmaswaṛin, yessefk ad yesɛu snin n tɣuri d yiwelfen, maca deg wayen yeɛnan iman-is yettaɣ tafat ɣef waya, ɣas akken netta d tmaswaṛt ijemmlen ugar n wayen id yesnulfa.

Ulac argaz ur yeččuṛen ara d ixemmimen ixedɛen ɣef iman-is, aɛeqqa yugaren akk d akken ur yebɣi ara ad yeqqen fell-as akken d tmaswaṛt.

Tmaswaṛt n umdan ur tesɛi ara tilelli n tḥerrik, teḍfer kan s tuget n tmersin tiɣerfanin d tignitin tizegwaṛin.

Tḥerrikin, tiggawin, awalen, ixemmimen, imeɣbanen, iḥulfan, isemɣanen, n tmaswaṛt n umdan uɣalen-d sɣur tmersin tignitin dɣa sɣur tuget n sebbanin tiɣerfanin tibeṛṛaniyin yesɛan aɛzazen.

Aɣersiw ameggag d amejjay ameggal s tmekta d tudert, d takurt tudert, yesɛan tazzla tulmut akken yezmer ad YEXDEM, mi ara d tidet ur yezmir ad YEXDEM kra.

Xemmmet ɣef kra n tallit, ay amɣar aqari, d ajjay amaɣlan yettwaḥkem s tneknatin.
