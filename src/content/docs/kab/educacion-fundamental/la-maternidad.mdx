Tudert n wemdan tebda am taxellut tamectut, teḥkem-itt, akken i d ayen ilan, akud yesɛan tazzla tameqqrant n taxellutin yettidiren.

Tadist, tiddest, talalit, d nutni i d tarwaɛt tameɣnast i d-ttbeddu tudert n yal aqelqul.

D ayen yessewhamen ad nẓer belli tikeltin-nneɣ timenza n tmettant ilaq ad tent-nexḍu deg wayen yellan d ameẓyan aṭas, nbeddel, yal yiwen deg-neɣ, ar taxellut tamezduɣt i tettwaẓer kan s tmuɣli.

Nettbeddu ad nili s talɣa n taxellut ur yesɛin azal, yerna nettɛeggiḍ tudert s leɛmer, d imɣaren yerna yeččuren d leḥwayej.

Nekk d usmekti. Aṭas seg imɣaren ur ttidiren ara deg tura, aṭas seg imɣaren ttidiren kan s usmekti n wayen iɛeddan. Yal amɣar d ṣṣut d tallit kan. Yal amɣar d uẓar n wayen iɛeddan, d usmekti yemlal, d ayen yettkemmilen deg ijinen n tarwa-nneɣ.

Tadist n wemdan tebda s akuden yesɛan tazzla tameqqrant, maca deg tlufa yemgerraden n tudert, ttafen deg-sent tazzla tnaqes yerna tnaqes.

Aṭas iɣerrajen i d-yehwan ad smektin ɛlaqet n akud. Aqermuc ur yettidiren ara armi d kra n sswayeɛ deg unebdu, yettban am akken ur yidir ara, maca yidir ayen yidir wemdan deg tmanyin n yiseggasen, ayen iḍerrun d akken yidir s tazzla, wemdan yidir deg tmanyin n yiseggasen ayen yidir fell-aɣ deg imelyunen n yiseggasen.

Mi ara mlilen ijujeg iberqan ɣer tmellalt, tebda tiddest. Taxellut i d-ttbeddu tudert n wemdan, tesɛa ṛebɛin d tmanya n tɣawsiwin.

Tɣawsiwin tmettlen ɣer ijinen, meyya seg-sent neɣ kter cwiṭ d nutni i d ayen yellan d Taɣawsa.

Ijinen ẓẓayen aṭas i uselmed fell-asen, acku yal yiwen deg-sen yettwabna seg kra n tilikin i yettferfirren s tazzla ur yettwaqbel.

Aḍunya tameɣnast n ijinen tessileɣ ɣer tama yellan ger umaḍal yesɛan kraḍ n lemɛani d umaḍal n tmerniwt.

Deg ijinen i d-mlalen tumiwin n tamirit. NEKK N TUDERT seg imezwura-nneɣ, d win i d-yettasen ad yegzem tamellalt yettwagzem.

Deg tallit-a n Takunikt d tussna tatumit, ur yettili ara d ayen yugar mi ara d-nini belli aqejmur ilektru-magnati i yeǧǧa umezwaru i yeffeɣ nefs-is yettwagzem deg ijinen d tɣawsiwin n tmellalt i yettwagzem s tarwa-is.

Abrid n tudert yettwabna s iɣbalen n yeqracen n uɛudiw n lmut.

Deg ubrid n tmettant, ttilawin yemgerraden n tezmert tteddun deg tfekka n wemdan; yal tilawt n tezmert tesɛa amsal-is n umenṭel, yal tilawt n tezmert tettban deg wakud-is d ssaɛa-s.

Deg sin wagguren seg tadist, nesɛa umenṭel n ukukru, yerna deg ṛebɛa wagguren seg tadist yekkaw deg umenṭel tazmert i yettilin d ayen yesɛan ɛlaqa ṭṭiq s unagraw uḍḍis d unagraw ameslɣaɣ.

D ayen iɛejbnen leɛjeb n tussna n tlalit d tmettant n yal tɣawsiwin.

Aṭas seg imussnawen i d-qqaṛen yella unamek ṭṭiq ger tlalit n wemdan d tlalit n umaḍal deg tlemmast n wayla.

Deg tẓa wagguren yettlal-d wergaz, deg mraw yettbeddu umɣi s yal umenṭalen-is iɛejbnen d usnerni yemseḍrun d yellan s ṣṣeḥ n tiqejjirin.

Mi ara tmeḍḍel Tuffɣa tamezwarut n yal aqewqi deg sin neɣ tlata n yiseggasen, d amaɛun belli unagraw uɣuraj-aɣbal-a yettwakemmel s ṣṣeḥ.

Aṭas seg imussnawen i d-nnan belli tabɛit tesɛa tamuɣli yerna tamuɣli-a tefka talɣa yettidiren i yal ayen yellan, i yal ayen yellan, i yal ayen ara yilin.

Aṭas seg medden i yettaḍsen i tamuɣli yerna kra deg-sen ssawalen-as "TAMEJNUZT N UXXAM".

Deg tama n wawal TAMUƔLI yella waṭas n uɣbel yerna aṭas i yeffeɣen ger TAMUƔLI d TAƔUṚI.

Kra seg imussnawen qqaṛen yella sin n temuɣliwin. I tamezwarut ssawalen-as TAMUƔLI TAƔAWSAWT yerna i tis snat teqqnen ɣer ayen yellan d ayen yesɛan azal d yesɛan lḥerma yellan deg-neɣ.

Seg udmawen d tajriba, nezmer ad neɛqel belli yella daɣen umsektag n TSEM-TAMUƔLI TAƔAWSAWT TAWWAƐT TAMETTANT UR NESƐI TILAST d TAYESMANT.

Umsektag-a n TSEM-TAMUƔLI TAƔAWSAWT YETTWALI s ddaw TMENZAƐT TATAKALIT.

Tugniwin tilinawtin, ssinima tawwaɛt, timucuha tiraɛeɛtin s unamek d snat, iccaqaren iɛebban, atg., zemren ad ḥettmen ad twali s ddaw TSEM-TAMUƔLI TAƔAWSAWT.

Aweḥḥi n tɣawsa i neǧǧa ad nesselkem ar taggara yesɛan unamek belli tirga tilinawtin d tiwessaɛtin n yiḍ ttwassebbaben ɣer TSEM-TAMUƔLI TAƔAWSAWT.

TADDART ȚEFFAȚ TILELLIT teqqen ar wakken ad tili TSEM-TAMUƔLI TAƔAWSAWT.

D ayen ibanen deg allen belli TAMUƔLI TUDERT temgerrad s yiwen lemɛani ɣer ayen i ttwasawalen TAMUƔLI TAƔAWSAWT, TAYESMANT, UR NESƐI TILAST. TASEBTILAST.

Yal assileɣ yezmer ad yettwaẓer s talɣa YESSEMƐALAYEN NEFSAWEN d win i tesɛam leḥerma, maca TSEM-TAMUƔLI n umsektag aɣawsaw, ur nesɛi tilast, asebtilast, ur nesɛi tilast zemren ad aɣ-xdɛun mi ara ttwalin s umesliɣ d tugniwin iɛeggben, ijehlen, i ttwanḍafen.

Ma yella neɣwa TAƐESSIST TILELLIT, taddart-tilellit, n lqem, ilaq aɣ-netḥazay ara kan TAMUƔLI TUDERT, maca daɣen TAMUƔLI TAƔAWSAWT d TSEM-TAMUƔLI UR NESƐI TILAST, YETTWALI, TASEBTILAST, I TTWANDAFEN.

Ur ilaq ara ad nettu ara lebda ɛlaqa ṭṭiq i yellan ger SEXU d TAMUƔLI.

Seg tmeẓri tameqrant, ilaq ad nbeddel yal umsektag n tamuɣli taɣawsawt d yal talɣa n TSEM-TAMUƔLI d INFRA-TAMUƔLI YETTWALI, ar TAMUƔLI TUDERT, tadelsant.

TAMUƔLI TADELSANT d nettat deg iman-is yesɛan adali, bla nettat imessiɣ ur yezmir ad yesseɣleb telefun, arradyu, afaylu, atg.

TAMUƔLI n TMEṬṬUT deg uɣur yella umuɣ d ayen yesɛan azal i usnerni n wagu. Yettwudebber belli yal yemma tezmer s TAMUƔLI-s ad tessensel tidmi n wagu.

D ayen iɣelben belli tameṭṭut deg uɣur ara tẓer tugniwin iǧehden, imazagen iǧehden, yerna ad tessell tameẓẓuga tageldant d awalen imseḍrunen, akka tezmer ad tmenṭel ɣef tidmi n waqewqi i tettɛeyyic deg iɛedlen-is s uɣur.

Tameṭṭut deg uɣur ur ilaq ara ad tessaw aqraz, ad tesseɛwi, neɣ ad tẓer ayen irqeqen, ayen iɛewwiqen acku ayagi meṛṛa d ayen iḍuṛṛun i usnerni uɣur n waqewqi.

Ilaq ad nerr tagwilt i yal tumert d tɣeltamin n tmeṭṭut i ttɛeggben.

Aṭas seg irgazen ur zemren ara ad ttesriren d win ur nesɛi aɣaras s ṣṣeḥ, yettaɣen yerna sserɣayen tameṭṭut deg uɣur. Tiɣemmren n-a, tɣawsiwin i d-yessebba urgaz ur nesɛi tilqa, ttemɣirent ɣef wagu deg uɣur, ala ma yella s tfekka maca daɣen s tidmi.

Mi ara neɛqel tazmert n tamuɣli adalit, d unamek ad nini belli tameṭṭut deg uɣur, ur ilaq ara ad tẓer ayen irqeqen, ayen iɛewwiqen, ayen ur yettemsalayen, ayen yetteɣwaɛen, atg.

Tewweḍ-d ssaɛa i lḥukumat ad d-tɣawlent ad sseɣlent tlufa timeqqranin i yeqqnen ɣer tayemmat.

Yettban d ayen ur yemseḍrun ara deg tmetti i yebɣan ad tettwabder d tamasiḥit d tademuqratit, ur ttɛemmiden ad ḥaznen d ad ɛemmdin unamek n tayemmat. D ayen iɣelben mi ara nẓer alaaf n tlawin deg uɣur bla afus, twaǧǧen seg urgaz d tmetti, ttɛeggiden aqermud n uɣrum neɣ umaɣan d mi ara xedmen tikciwin timadduṛin tixsasin, i wakken ad zmirent ad ɛiicent s waqewqi i ttɛeyyicent ger iɛedlen-nsent.

Awalan-a ur yemseḍrunen n tmetti turagt, leɣder-a d ur taɛdilt n imḍebbren d wɣerfan d nutni i d-aɣ-d-yessekniwen s tewzelɣa belli mazal tademuqratit ur tella.

Ibniḍen s tɣuṛaf n tlalit mazal ur d-sɣelben ara allalen, acku ar ibniḍen-a zemren ad awḍen kan tlawin mi ara d-qeṛbent ɣer tlalit.

Țḥwajen yexxamen s tɣerma, timdinin n tẓegwa n ṣṣeḥ iǧehden s usafen d idɣaṛen i tlawin deg uɣur ifeqqṛen s ṣṣeḥ, tilkin d idɣaṛen n warrac n-a.

Ixxamen-a n tɣerma d amkan i ttɛicin deg-sen tlawin timeɣbuntin s ṣṣeḥ deg uɣur, ččuren d meṛṛa umektub, ijeǧǧigen, ameẓẓuga, urar, leǧehd, atg., ad sseɣlent s lekmal allalen n tayemmat.

Ilaq ad neɛqel belli tmetti n wemdan d tawacult tameqqrant yerna ulac allal agensas acku yal allal deg yiwet neɣ deg tayeḍ telɣa yettaɣ deg uɣur-is ar meṛṛa iɛeggalen n tmetti. D ayen ur yesɛin ara unamek ad tessenzel tlawin i ttɛeggben ɣef taɣara n ufeqri s ṣṣeḥ. D ccṛ ayen ara yettwaqqren, ad twasḍeyṛen neɣ ad ḥerzent deg usɣun n iɛeggben.

Deg tmetti-a i deg nettidir ur yezmir ara ad ilin warrac d ibekkaten, acku meṛṛa nukkni d imdanen yerna nesɛa iẓuran imseḍrunen.

Ilaq ad nessenṭeq tademuqratit n ṣṣeḥ, ma yella seg ṣṣeḥ ur neɣwiyi ara ad aɣ-yeɣḍeb Umunizmul.
