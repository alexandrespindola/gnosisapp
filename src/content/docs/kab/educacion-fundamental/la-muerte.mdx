Ila ilaq ad nefhem akken ilaq, deg yal amkan n wallaɣ, d acu-t lmut s timmad-is. Anagar s wakka i nezmer, s tidet, ad nefhem akken ilaq d acu-t tdukli.

Ad nwali tafekka n umdan neɣ yiwet n tmeṭṭut i neḥemmel deg usenduq, ur yettɛnidet ara nefhem asirem n lmut.

Tidet d ayen ur nessin ara deg yal lawan. Tidet ɣef lmut ur tzemmer ara ad tili d ayen yemxalafen.

Nekk (ego) yettmen dima, akken yella kan d aẓaṛ, aɣbel ɣef lmut, aɛhed yellan zdat, kra n win ara yeṭṭfen iɣef fell-aneɣ aɣerbaz yelhan d yal taɣara n tdukli deffir n uẓekka.

Nekk s timmad-iw ur yesɛi ara cwiṭ n rryiṣ ad yimmet. Nekk yebɣa ad ikemmel. Nekk yettagad aṭas lmut.

Tidet ur telli ara d amek teggulleḍ neɣ teccekkeḍ. Tidet ur tesɛi ara kra ara tt-yesdukklen akken teggulleḍ, neɣ akken teccekkeḍ. Tidet ur telli ara d ayen yeɛnan tiktiwin, tinaẓurin, arayanen, asenfel, azal, tirẓiwin, tiggura, tiktiwin-inaɣ, arayanen n uqeddem, atg. Tidet ɣef asirem n lmut ur telli ara d amek teqqenneɛ.

Tidet ɣef asirem n lmut tzemmer ad tettwassnen anagar s teɣri taqburt.

Yewɛer ad tessefsay teɣri n lmut-tidet i win ur tt-nessin ara.

Yal amedyaz yezmer ad yaru idlisen yelhan ɣef tayri, maca yewɛer ad tessefsay tidet ɣef tayri i yimdanen ur tt-nessin ara, akken ma yella nefka-d belli yewɛer ad tessefsay tidet ɣef lmut i yimdanen ur tt-yeddir ara.

Win yebɣan ad yessen tidet ɣef lmut ilaq ad yefrurex, ad yɛeddi fell-as s timmad-is, ad inadi akken ilaq, anagar akken i nezmer ad nerr-d azal aqdim n lmut.

Asmun d teɣri n tuget n yiseggasen i ɣ-yettunefken ad nefhem belli imdanen ur ttunfen ara ad fehmen azal aqdim n lmut; i yimdanen ayen kan i ten-yettunfen s tidet d ad kemmlen deg asirem yellan zdat.

Aṭas n yimdanen bɣan ad kemmlen s rrbeḥ, tbanit, tawacult, leɛwayed, tiktiwin, arrac, atg., u mi فهمن belli yal taɣara n tdukli tansayt d urbaɛ, ḥefyan, ur tḥebber ara, ur tḥemmel ara, selḥen-d, ttagaden, tteggen-d, ččuren-d s lxuf yellan ur yesɛi tmeddit.

Ur bɣan ara ad fehmen tiggura yettwanɣan, ur bɣan ara ad fehmen belli ayen akk yettkemmilin yettwaxdam deg lawan.

Ur bɣan ara ad fehmen tiggura yettwanɣan belli ayen akk yettkemmilin yettɣimi-d s lawan.

Ur bɣan ara ad fehmen tiggura yettwanɣan belli ayen akk yettkemmilin yettbeddil γer tmekkanizmat, d aɣlalaɛ, d aserɣi.

Ilaq, ilaq, ilaq, ad neqqim s ɣur-aneɣ ad nefhem azal aqdim n lmut, anagar akken ad yerwel lxuf n wakk ad nɣimi neɣ neffeɣ.

Mi neṭṭef iɣef akken ilaq i tmagit, nezmer ad nessider belli taɣara n wallaɣ tettwawzen s tɣara yettwassnen yebɣa ayen yettwassnen ad ikemmel zdat n uẓekka.

Taɣara n wallaɣ tettwawzen s tɣara yettwassnen, ur tzemmer ara ad tɛeddi fell-as ayen ur yettwassnen ara, ayen-tidet, ayen yellan d tidet.

Anagar mi neɣrem tasekkurt n lawan s tmeṭṭa n uqader, nezmer ad neɛeddi fell-as ayen ilan d ɣur-tmeddit, ur nesɛi ara lawan, ayen yellan d tidet.

Widak yebɣan ad kemmlen ttagaden lmut d leɛwayed-nsen d tinaẓurin-nsen ttunfen-d anagar i rrebbas.

Lmut s timmad-is ur tesɛi ara kra n tiggura yettwaqqaren, ayen yelhan aṭas, tbanit, ur yettwassenfay ara, maca taɣara n wallaɣ tettwawzen: deg wayen yettwassnen, yettzuxu anagar deg usekkil waḥid yellan yettali seg tɣara ɣer tkukki.

Mi nessider s tidet azal aqdim d ameqqran n lmut, nerr-d iman-neɣ s teɣri taqburt, belli tudert d lmut d ayen yellan s lekmal, d tdukli.

Lmut d aɣbalu n tudert. Abrid n tudert yettwaxdam s tukkin n yixef n lmut.

Tudert d tazmert tettwafaqqan d tettwafaqqan. Seg talalit ɣer lmut ttaggin gar tidet deg taɣessa n umdan yal taɣara n tezmart.

Anagar taɣara n tezmart i taɣessa n umdan ur tzemmer ara ad tɣum, d uzeṭṭa n lmut. Azeṭṭa-agi yesɛa adraf ilaqen deg uɣara n tmes. Taɣessa n umdan ur tzemmer ara ad tɣum adraf-agi.

Akken azeṭṭa yezmer ad yexleḍ deg ttejra, akken i uzeṭṭa n lmut mi yettali deg taɣessa n umdan, ad tt-yexleḍ.

Azeṭṭa n lmut yettbeddil anadi n lmut, gar wanadi n tlilit.

Azeṭṭa n lmut yettuɣal arayen n tmes i qerben aṭas d kra n tira yellan tesɛa tazmert, tettwawzen ad teḥseb tixsan deg tɣara n tewwura.

Azeṭṭa n lmut yettɣimi-d taɣessa n umdan ɣer tiggura yettwareklen.
