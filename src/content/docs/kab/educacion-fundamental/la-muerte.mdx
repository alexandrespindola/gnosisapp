Yella-d ilaq ad neɛqel mliḥ dɣa deg yal aḥric n wallaɣ, ayen yellan d lmut s timmad-is, anagar akka i nezmer, s tideț ad nefhem akken yessefk ayen yellan d tudert ur tettmettaten,

Ad nwali tfekka n wemdan n lɛaziz-nneɣ deg useندوق, ur yeɛni ara belli nefhem sser n lmut.

Tidet d ayen ur yettwassen ara yal imir. Tidet ɣef lmut ur tezmir ara ad tili d tamegda.

Nekk yettmenni yal wa, am akken yella d ṭbiɛa, ubezpieczenie ɣef lmut, d garansi zdat, kra n lḥekma ara yesseɣtiyen belli ad nesɛu lǧehd dɣa yal ṣṣenf n tudert ur tettmettaten s deffir n tmuqbert yeḍḥeqen.

Yiman-iw ur yesɛi ara cwiṭ sserḥ ad yemmet. Nekk yebɣa ad yeqqim. Nekk yettagad lmut.

Tidet ur telli d awal n umen dɣa ur tettelḥiq ara deg-s. Tidet ur tesɛi ara wala deg umeɛmen, wala deg win ur yumen. Tidet ur telli d awal n tiktiwin, tibeddiwin, रायيन, iḍrisen, iḍrisen n zdat, iḍrisen yettwabnan, iɣalen, tmeslayin, tmesleɣtin, atg. Tidet ɣef sser n lmut ur telli d tamegda.

Tidet ɣef sser n lmut nezmer kan ad nettwaɛqel sɣur tajriba tdewwelt.

Yewɛer ad nbeddel tajriba teḥqiqt n lmut i win ur tt-yessin ara.

Yal amedyaz yezmer ad yaru idlisen yelhan n tayri, maca yewɛer ad nbeddel tidet ɣef tayri i medden ur tt-neǧrib ara, akka am winna yettinin belli yewɛer ad nbeddel tidet ɣef lmut i medden ur tt-id-yessawed ara.

Win yebɣan ad yessen tidet ɣef lmut yessefk ad yemḥeq, ad yeǧreb iman-is, ad inadi akken yella d ilaq, anagar akka i nezmer ad naf aseɣẓan yeqqanen n lmut.

Aɛqal d tajriba n waṭas n yiseggasen ǧǧan-aɣ ad nefhem belli medden ur ten-yeḥric ara ad fehmen tideț aseɣẓan yeqqanen n lmut; medden, ayen kan ten-yeḥricen d ṣṣeḥ d ad qimen s deffir, dɣa ayagi kan.

Waṭas n medden bɣan ad qimen sɣur lmal, ccan, tawacult, liman, tiktiwin, arrac, atg., dɣa mi fehmen belli yal ṣṣenf n tudert teɣri ur tesɛi ara lmeɛna, tettɛeddi, tefsef, tesseḥriq, ihi, țḥussen iman-nsen ur sɛin ara ضمان, ur țțamen ara, ṭṭerḍaqen, țțagaden, ččuṛen d ugur d ameqran.

Ur bɣin ara ad fehmen imeskineɛen, ur bɣin ara ad ɛeqlen belli yal ayen yettkemmil iteddu deg wakud.

Ur bɣin ara ad fehmen imeskineɛen belli yal ayen yettkemmil yețɣimi deg wakud.

Ur bɣin ara ad fehmen imeskineɛen belli yal ayen yettkemmil yuɣal d amaẓẓal, d akud, d yir lḥal.

Yella-d ilaq, yella-d اللازم, yella-d waǧib, ad nili d aqqaɛen mliḥ n useɣẓan yeqqanen n lmut, anagar akka i yeṭṭef umagger ɣef tuget ad nefk ayen yellan.

Mi nettmuqul ɛlaḥsab ɣer medden, nezmer ad nɛqel belli wallaɣ yella ديما yettwaxxer deg wayen yettwassen dɣa yebɣa ayen yettwassen ad yeqqim s deffir n tmuqbert.

Wallaɣ yettwaxxren deg wayen yettwassen, ur yezmir ad yeǧreb ayen ur yettwassen ara, ayen yellan d ṣṣeḥ, ayen yellan d tidet.

Anagar mi neddem tiṭṭ n wakud sɣur lemɛayc yellan d ṣṣeḥ, i nezmer ad neǧreb ayen yettdumun, ayen ur nesɛi ara wakud, ayen yellan d ṣṣeḥ.

Win yebɣan ad yeqqim yettagad lmut, dɣa liman-nsen d tibeddiwin-nsen kan i ten-yettqeddimen d asurqi.

Lmut s timmad-is ur tesɛi ara wala deg wayen yeḍḥeqen, d ayen yelhan aṭas, d aqqaɛen, ur nezmir ara ad t-id-nesbedd, maca wallaɣ yettwaxxren: deg wayen yettwassen, yettḥerrek kan deg taɣwabt teǧhed yetteddun seg umeɛmen ɣer win ur yumen.

Mi nili d aqqaɛen ṣṣeḥ n useɣẓan yeqqanen d yeqqanen n lmut, ad naf i yiman-nneɣ sɣur tajriba tdewwelt, belli tudert d lmut ttqeddimen aḥric کاميل, uni-total.

Lmut d amaḍal n tudert. Aberrez n tudert yettwabna s izenqen n yiman n lmut.

Tudert d الطاقة tebṭi dɣa yettbeṭṭun. Seg tlalit ɣer lmut teddun deg tfekka n wemdan ṣṣenfat yettwexxren n الطاقة.

Ṣṣenf kan n الطاقة tfekka n wemdan ur tezmir ad tḥemmel, d tɣanimt n lmut. Taɣanimt-a tesɛa voltage الكهربائية yeqqaren aṭas. Tfekka n wemdan ur tezmir ara ad tḥemmel voltage am ta.

Akk akken tɣanimt tezmer ad teṛẓu ttejṛa, akka daɣen tɣanimt n lmut mi tedda deg tfekka n wemdan, tesseṛẓu-tt s tella.

Taɣanimt n lmut teqren afina lmut, s afina tlalit.

Taɣanimt n lmut teḍfer tugdiwin الكهربائية yeqqanen aṭas dɣa kra n tezwart tatrart tesɛa lǧehd, tessebṭi leǧnun deg tfekka yemmeqqran.

Taɣanimt n lmut tesseḥbas tfekka n wemdan ɣer iyallen-is imezwura.

EGO, nekk энергетические, yeqqim deg yeddarren-nneɣ lḥif.

Ayen yellan d tidet ɣef lmut, ayen yellan d tallit gar lmut d tgemmi d ayen ur yettelḥiq ara ɣer wakud dɣa kan sɣur tussna n lemɛayc nezmer ad neǧreb.

Imasṭren d Imasṭrat n Uɣerbaz, iɣerbazen d tesdawiyin, yessefk ad slemden i yinelmaden-nsen d tnelmadin-nsen aberrez yettqeddimen ɣer tajriba n ayen yellan d ṣṣeḥ, ayen yellan d tidet.
