Imazdaɣen d imseɣẓanen n idurar, iliziyen d tesdawiyin, ilaq ad ɣerren s tiddi tasneɣẓi n inqilaben i yeslemden Umussu Agnusan Agreɣlan.

Tasneɣẓi n inqilab ara yetteddun yemgarad akken ma tella ɣef wayen i yettwassen s yisem-a zik.

Bla ccekk, nezmer ad d-nini war akukru belli deg wass-aɣ n leqrun i nezdin, seg yiṭes ameqran n yal laṣel, werǧin tasneɣẓi tuɣal d tazɣiɣt am ass-a deg tallit-a n "Aɛṣi war ssebba" d igujilen n Rruc.

Tasneɣẓi yettaggwaden d uṭṭafar n talliyin-a timiranin armi i tesruɣ fell-as, yellaɣ-as kra i yuɣalen d lmeɛna-ines, d yal ddaw ireqqen ɣer laṣel-ines n tidet.

Deg talliyin-a n teɣḍawt tisensurelt d unɣal aɣris n wallaɣ, ur yettwasenqad ara kan akken ad yettwasmeɣẓi s ṣṣeḥ wawal tasneɣẓi, maca ula d tidet ur yettwasnen ara akk d tɣawsiwin tigejdanin n tsneɣẓi.

Wid yettḥesben s tuccḍa belli tasneɣẓi d tussna tamiranit n ussan-a ineggura, ttekksen, acku tasneɣẓi d tussna taqburt mliḥ i yesɛan laṣel-is deg idurar iqburen n yimsaruten Iqbayliyen.

Aṭṭaf n Snuv, aqerray amiran yernaɣ, aṭṭafar, yettuseḥsab fell-as d ṣɛib ad yesseɣzi ayen i yettwassen s yisem tasneɣẓi, acku ɣer tama n tallit-a tamiranit, iban belli tasneɣẓi ur tella werǧin ddaw yisem-ines, acku ɣer kra n ssebbat, tettwabḥeq fell-as belli tɛizzi i ussermed n ccwal n wudem asertan neɣ uɣerfan, dɣa ur ttunefki ara ad tbeddel deg tlibsat yemgaraden.

Seg zik, deg yidurar yemgaraden n usenẓag n tudert, tasneɣẓi tettili dima deg udur-ines, tettbeddil s tmusni deg tlibsat n telfalsaft.

Deg yiri n Ganges, deg Lhend Taqedsit n Vedas, seg yiḍes ameqran n leqrun, llan wudem n Yoga i yellan deg ṣṣeḥ, d tasneɣẓi tanagrawt n yefriwen ɛlayanen.

Sebɛa Yoga ttwaseɣzan dima d tarrayin, d ifadden, neɣ d irifen ifalsafen.

Deg umaḍal Aɛrab, aslemad Aqedsan n Sufis, deg tama n imetafizik, deg tama n uɣerfan, d tidet d tidet, deg udmawen ineglafen.

Deg Turuft Taqburt ternaɣ armi d aɣrem n uɣes deg waṭas n imenɣiyen, d lḥeqqaɛ tisenselmin, tiɣerfanin, tisertanin, atg. Ula d ar taggara n tasut iɛeddan, tasneɣẓi tebeddel deg lḥerraz n telfalsaft akken ur tettwaqabel ara.

Telfalsaft ɣas ma tella deg taggayatin-ines d taggayatin-ines, am tilikit, taggayt n tamusni, Ttiq, Tastitit, atg., D tidet deg iman-is, ddaw yal ccekk, tanegga n imaniban, tamusni timaɣlalit n yella, tafugent tamusnit n taɛeqqay n wudem.

Tuccḍa n waṭas n idurar ifalsafen tezzi ɣef tneɣẓi am kra n tɣẓiɣt n telfalsaft, am kra yeɛnan kan wudmawen tɣiɣaten, armi d tlalit n wemdan.

Aɣerdi mgaraden n yisenɣanen yeslaɣ-aɣ ad naweḍ ɣer unɣiɣ aɛeqli belli tussna n tsneɣẓi tella dima d taseddast deg wudem iɛedlen ɣer yal Aẓaṛ Uɣerfan. Yal aɣerdi mgaraden n yisenɣanen yesskanay-aɣ belli deg Tasekla Taqedsit teɣḍiɣt n tamurt yemgaraden d talliyin yemgaraden, llan yiɣerbazen yecban tussna Tasneɣẓi.

Igerdan imeqqranen deg uẓar n Agnusis yeslaɣ-aɣ ad naf taseddast-a yeccba yiwet n wudem Agnusis i yellan seg talliyin timenza n Ukristiyan d wawal n PHILOKALIA, yettusaɣtayen ar ass-a deg Tinezduɣt Tagmuḍant, ɛlaḥsab i uslemad n imɣaren.

Ddwa yal ccekk d war akk ukukru ad nɛeyyeɣ deg tuccḍa, nezmer ad d-nini s tiddi belli PHILOKALIA d ṣṣeḥ Tasneɣẓi Tanagrawt Tefreɣ.

Deg Yidiran Iqdimen n Yimsaruten n Yunan, Maṣer, Ṛuma, Lhend, Pars, Miksik, Piru, Asiri, Xaldiya, atg. atg. atg., Tasneɣẓi tella dima deg yidurar, ɣer tussna d unadi Awemman n tidet, ɣer tussna d din.

Deg talliyin tiqburiwin tasneɣẓi tettwaffen deg yidurar ireqqaqen n Tamezgunt Taqedsit, neɣ deg tilɣi tanegga deg Jeruglifs, neɣ deg tɣawsiwin tisensurelt, neɣ deg tacɛirt, neɣ deg tazzwart d armi d ccna yeccban idurar.

Uqbel tussna, telfalsaft, unadi d din ad ɛezlen akken ad virant s umusnaw, tasneɣẓi tella dima deg Yidiran Iqdimen n Yimsaruten.

Mi idurar Iniziaticin mmḍel axater n Kaliyuga, neɣ Tallit d Tarusiwin deg i mazal nettili, tasneɣẓi teɛdeyya gar tamsegzelt n Yiduraren Iɛezlanen d Yisudu-Iɛezlanen n Umaḍal Amiran d ɛlaḥsab gar Ugnusis Iɛezlan.

Amsellek d igdan imeqqranen deg uẓar, yeslaɣ-aɣ ad neɣret s tiddi tanegga belli irifen yemgaraden d tselmadin Tasneɣzi i yellan deg tmezwarut d i yellan deg ass-a, nezmer ad ten-nefrez deg snat n tuggayatin.

Tamezwarut.- Tselmadin ammi waṭas n imusnawen ten-ɣren. Tasneɣzi Tamiranit tella deg udmawen ɣer taggayt-a.

Tis snat.- Tselmadin i yeqqaren deg wemdan seg udem n Inqilab n Taɛeqqay.

Asegdaɣ-a ineggura d tidet d tselmadin n laṣel, d tiqburiwin kter, nutenti kan i yeslaɣen ad neɣret ɣer laṣel n tidet n Tsneɣẓi d tɣawsa-ines takkawit.

Mi nettakkin nefret deg wudem ireqqen d deg yal Iswiyen n Wallagh, acu tiddi yesɛa uslemad n wemdan seg wudem amaynu n Inqilab n Taɛeqqay, ad naɣret dɣa belli Tasneɣzi d uslemad n llsas, d yasenɣanen d tɣawsiwin yersen ɣer tbeddilt tarrat d taggara n UMESLULI.

D ilezma ad imazdaɣen d imseɣẓanen n idurar, iliziyen d tesdawiyin, ad frett s tiddi n tallit taqerreqṛant deg i nettidiren d tallit n tneɣruɣt Tasneɣzi deg tezzi Tlellit Tamaynut.

D ilezma ad neqqen "Aɣerbal Amaynu" deg ubrid n Inqilab n Taɛeqqay d wagi kan yettili s Tasneɣẓi n Inqilab n Useɣẓan agejdan.
