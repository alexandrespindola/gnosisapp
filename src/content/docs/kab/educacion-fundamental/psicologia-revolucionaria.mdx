Ilaẓen akked tilmeẓyin n uɣerbaz, asinaryu, d tesdawiyin, ilaq ad ɣren mliḥ tamusni n tussna n iman inqilabi i yesselmad umussu gnustik amazɣlan.

Tamusni n tussna n iman n unqilab yetteddun d tin yemgaraden s tmernit ɣef ayen akk yettwassen yakan s yisem-a.

Ulac ccekk, nezmer ad d-nini mbla akukru belli deg tallit n leqrun i ɣ-d-izwaren, seg iṭes ameqran n yal leɛmer, ulac tamusni n tussna n iman i d-yeɣlin ddel s wakka am wakken tura deg tallit-a n "Amḥaddi mbla ssebba" d iɛeggalen n ROCK.

Tamusni n tussna n iman iɛeṭṭalen d tin usuffeɣ n tallit-a tatrarat i d-tjemmeɛ lmeṣyaba tesruḥ lmeɛna-s, d yal tamsirt tusridt yellan d aẓar-is yellan d tidet.

Deg tallit-a n tuɣẓaṛt taɣuẓrant d tɛedda n uɣilif, ur teqqim ara kan d tin yeǧǧan ad yeqqel yessefk ad yettwasellek s tsellekta taẓẓiɣt ussan-a n tamusni n tussna n iman, maca i yettwagguggen s tidet ɣef tira-nni tiselmadin n tamusni n tussna n iman.

Wid i yeqqaren s tɣeltiyya belli tamusni n tussna n iman d tussna tatrarat n tallit-a taneggarut, ɛelṭen s tidet imi tamusni n tussna n iman d tussna taqburt yellan d aẓar-is deg uɣerbaz aqdim n tɣuṛaf n tɣuṛiwin.

Ɣer wanaw n SNOB, ɣer uqeccaf ultra-amur, ɣer umeɛṭel, yettuɛettel ad issellek ayen yettwassen am tamusni n tussna n iman imi ɣer tama n tallit-a tatrarat, d ayen ibanen belli tamusni n tussna n iman ur d-tban ara d yisem-is imi ɣef waṭas n tsebbiwin, tettwaɣil d tin yesskanayen ɣer tɣuṛiwin tiberkanin n wannaw aserti neɣ usɣan dɣa ɣef waya teẓra iman-is tettban s tɣuṛiwin yemgarraden.

Seg zman aqdim, deg tegnatin yemgarraden n uẓeṭṭa n tudert, tamusni n tussna n iman tesskanay dima adeg-is, tesseɣḍil iman-is s tɛeqliyya s tɣuṛi n telfalsaft.

Ɣer tama n Ganges, deg Lhend yesɛan lqeddasa n VEDAS, seg yiḍ ameqqran n leqrun, llant tɣuṛiwin n YOGA i yellan deg uzɣar, d tamusni n tussna n iman tagraɣlant, tessiɛ araḥ.

Sa tɣuṛiwin n YOGA tettwasellek dima am tɣuṛiwin, tixellisin, neɣ uddisen n telfalsaft.

Deg umaḍal aɛrabi, tiselmadin iɛeqqden n SUFIS, ɣer tama n tussna n ufella, ɣer tama n usɣan, d tiselmadin n lemer n tamusni n tussna n iman.

Deg Turuft taqburt tesserɣet ɣer wudem n yiɣes s waṭas n yimenɣiwen, tilɣuɣat iɣerrfanen, usɣanen, usertiwen atg. ar taggara n lqern yezrin, tamusni n tussna n iman tesseɣḍil iman-is s tɣuṛi n telfalsaft akken ad tɛeddi mbla ma yella wayen i tt-izerrɛen.

Telfalsaft ɣas ma yella fell-as waṭas n tɣuṛiwin d tɣuṛiwin tisɣunanin am tid n usikel, tamusni n tussna, tilɣuɣat, tussna taneggarut, atg., d tin n tidet i yiman-is, TUKKAWIYIN TUFRANIN, TAMUSNI TIZIWENT N UZɣAR, TIƔURIWIN TUKKAWIYIN N TTƐEQQA TUSELMAḌT.

Aɣeltiyya n waṭas n UƔERBAZEN N TLFALSFT tettili deg yella ɛaqel ɣef tamusni n tussna n iman am wayen yellan d waddayen ɣer TLFALSFT, am wayen yeqqnen kan ɣer tɣuṛiwin tizzegzawin d tin yellan d tifranin n ugama n wemdan.

Aɣuri ameɣrud n ddinat yettǧǧa ad tɣers aɣersa belli tussna n tamusni n tussna n iman tella dima yeqqnen s tmernit meqqren ɣer yal IBUDDANEN USɣANEN. Yal aɣuri ameɣrud n ddinat yettban belli deg tira iɛeqqden yellan d tiderɣal n tmura yemgarraden d tallit yemgarraden, llant tɣuṛiwin tizerfanin n tussna n tamusni n tussna n iman

Aseqsi ameqqran deg arqaq n GNOSTICISMO yettǧǧa ad naf ad naf tukkawit tayeffuḍen n imyura gnustik yemgarraden i d-yusan seg tallit tamezwarut n Tmesḥit i yettwassen s azwel n PHILOKALIA, i yettwaqecran ar ass-a deg TMEZZIT TAƐERABT, s ussarem ɣer usselmed n yirerban.

Ulac ccekk d mbla akukru ad nini s tezmert belli PHILOKALIA d tamusni n tussna n iman tagraɣlant.

Deg UƔERBAZEN AQDIMEN N TƔUṚAF n Tgelda tagraɣlant, Maṣer, Ṛuma, Lhend, Fares, Meksis, Peru, Asiriya, Caldea, atg. atg. atg., tamusni n tussna n iman tella dima yeqqnen ɣer telfalsaft, ɣer Uẓur aḥeqqani, ɣer tussna d ddin.

Deg tallit taqburt tamusni n tussna n iman tessaɣḍil iman-is s tɛeqliyya ger tikliwin tizemranin n tḥdayyin iɛeqqden, neɣ ger leɛjeb n ijerɣufiyen iberranen neɣ tlemziwin, neɣ deg umezgun, neɣ deg uɣelli d ar tama n uẓawaj amazɣlan n tmezgida.

Uqbel ad yemgerrad tussna, telfalsaft, uẓur d ddin akken ad tekkin s timmad-nsen, tamusni n tussna n iman teḥkem s tezmert deg yal UƔERBAZEN AQDIMEN N TƔUṚAF.

Mi tekmel Tasdawit seg ussan n KALIYUGA, neɣ TASEKKA TIBERKANT ideg neɣ deg-s ar ass-a, tamusni n tussna n iman tedder ger usemẓi n UƔERBAZEN ESOTÉRICAS d PSEUDO-ESOTERICAS n UMAḌAL ATRAṚ, d ussarem ɣer ESOTERISMO GNÓSTICO.

Aseqsi ameqqran d tira n uzɣar, tessefham s tsellekta tazzurant belli uddisen d tlɣuɣat n tamusni n tussna n iman yellan deg tallit iɛeddan d i yellan deg ass-a, zemren ad yettwagzem ɣer snat tɣuṛiwin.

Tamezwarut: Tlɣuɣat am akken i ten-iqaren waṭas n iɛeqliyen. Tamusni n tussna n iman tatrat teqqen s tidet ɣer tɣuṛi-a.

Tis snat: Tlɣuɣat i yeqqaren ɣef wemdan seg umaɣal n UNQILAB N TTƐEQQA.

Ayagi ay d tlɣuɣat n tidet, tid i yellan d tiqdimin, tanekma ay d-yeǧǧan ad neɛqel aẓar usrid n tamusni n tussna n iman d tneflit-is taɣurarant.

Mi neɛqel nekkni akk s umata d yal NNIBA n TTƐEQQA, acu-t azal meqqren yesɛa ɣuri-s aɣuri ɣef wemdan seg umaɣal amaynu n UNQILAB N TTƐEQQA, ad neɛqel tamusni n tussna n iman d aɣuri ɣef ibuddanen, tlɣuɣat d tiẓriwin yeqqnen s tmernit ɣer TUGA N UZɣAR d tin taggara n WEMDAN.

D ussan i ilaẓen akked tilmeẓyin n uɣerbaz, asinaryu, d tesdawiyin, ad fehmen s umata tallit taɣurarant ideg nedder d tɣeltiyya taɣurarant n teɣremt ideg tella tmurt taneggarut.

Ilaq ad neǧǧ "TULLIST TAMEJDAḌT" ad teddu ɣef ubrid n UNQILAB N TTƐEQQA d waya ay d-yeǧǧan kan ad yeqqel s TAMESNI N TUSSNA N IMAN TAḲILABT n USSELMED ABUDDAN.
