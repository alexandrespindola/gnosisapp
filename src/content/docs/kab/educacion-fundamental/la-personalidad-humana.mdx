Yella yiwen wergaz ilul, yettidir setta wagguren d semmus, yemmut. Meɛna anda yella uqbel n 1900, anda yezmer ad yili deg deffir n 1965? Tussna tunṣibt ur tessin acemma ɣef ayagi akk. D taggaẓt tamatu d akk yisteqsiyen ɣef tudert d lmut.

Aqli-t neqqar s tidet: "ARGZAZ YEMMUT AXATAR AKUD-IS YEKFA, UR YELLI AZEKKA I LMEYYET".

Ass yal ass d agelmam n wakud, ayyur yal ayyur d agelmam n wakud nniḍen, aseggas yal aseggas d agelmam n wakud nniḍen, akken yeglamen-a akk mgaraden deg wakkud d-yesdukkelen agelmam ameqran n tudert.

Akud d agerrez, tudert n UMAɣLAY ADNAN d taɣzut tettmeddu.

Tudert n UMAɣLAY ADNAN tettimɣur deg wakud-ines, tettlal deg wakud-ines, tettemmut deg wakud-ines, ur tezmer ara ad tili ur tessuffuɣ ara akud-ines.

Ayagi ɣef wakud d amur i yettwaxemmemen ɣur waṭas n wid yessnen. Ur yelli ccekk deg-s akud d TAZɣERT TASEKKALT.

Tasnakunt n EUCLIDES d tin yettwaqedcen kan ɣer umaḍal AKUD-UMESƐAD macca umaḍal yesɛa sebɛa tɣemmar, TAZɣERT TASEKKALT d AKUD.

Uɣal n ufgan yettfak akud i yettwaxemmemen d akud deg uɣrud asrid, ur yelli wayen yessakken ayagi ɣef tmuɣli-a, axatar TENDUZZI d TAZɣERT TISEMMAS.

Yal akud n tella tettili deg wakud tɛawad-d i lebda.

Lmut d TUDERT d sin yemgerraden i yemlalen. Tudert tefser i wergaz i yettmettaten meɛna tebda nniḍen. Akud yettfesser, akud yettbeddu, lmut tettili teddukkel s ussuffeɣ d AɛAWAD AYEZGAN.

Ayagi yebɣa ad yini ilaq a nerr, a nuɣal ɣer umaḍal-a mi nemmut, ad neɛawad adrar n tella, meqqar, UMAɣLAY UFGAN yettfesser s lmut, anwa neɣ d acu i yettraǧun?

Ilaq a nɛellem yal wa d tikelt tamezwarut d NEK i yettkemmilen deffir n lmut, d NEK i d-yettraǧun, d NEK i d-yuɣalen ɣer waddar n imeṭṭawen-a.

Ilaq aɣ-qqaren ur tɛattat ara Asadur n UƔAL s Asadur n TLELLI yettwesselmaden s TEOSOFÍA TANEGDATT.

Asadur i d-qqaren n TLELLI yebda deg unufus n KRISHNA i d DIN N INDOSTAN d anaw Védico, yettwaɣtas u yettwarqaq deg wessufɣen.

Deg unufus aṣeḥḥani n Krishna, kan IHERWAN, IMUDAR, wid i yesɛan TAɣUDT TAQUDSAN, d wid kan i yettlalen.

NEK YETTWAǦMAREN YETTRAǦU, yettɣimi macca ayagi mačči d TLELLI. Tikelt i tɣawsiwin, ɣlin maca ayagi mačči d TLELLI.

Lɣiba n UƔAL n tɣawsiwin, n yigiten, lɣiba n tɛawadt ur tɛedda ara, nezmer a tt-naf deg TUSNA TAQWRAFIT d tmezrayt taqdimt n INDOSTAN.

Aɛwud ayezgan n YASSATEN D IDAN n BRAHAMA, aɛwud ur yettwaɣtaɣen n KALPAS, atg, ttmgaraden deg wakkud iqedcen s tmusni TAQWRAFIT d Asadur n TLELLI ayezgan neɣ AƔAL AYEZGAN.

Gautama el BUDHA yesselmed s tmusni meqqren TADUKLI n UƔAL AYEZGAN d tɛeccact n tudert yettkemmilen, maca TADUKLI-ines tettwarqaq ɣur yettabaɛen-t.

Akk UƔAL YESƐA s tidet aṣuffeɣ n UMAɣLAY ADNAN amaynut, ayagi yettwassen deg seddis iseggasen imezwura n tmettant.

Tudert n twacult, tudert n ubrid d Uɣerbaz, ttefken UMAɣLAY ADNAN, tin aẓar-ines.

AMEDYA n wid meqqren d win yettwakfan i umaɣlay yelmeẓ.

Ileẓ yettlamad akken illa deg umedya, ula d ssut. Anaw yerqeqen n tudert, amedya ur nessin, lɛadat ttemɣallalen n wid meqqren, tefken i umaɣlay n ileẓ ayagi yerqiqen d win yettfakkiren d win yettwarqiqen n tikkelt i nedder.

Deg wussan-agi ineggura, ttagmat-tuɣalin tteggani, tɣellib aẓalim d ubezri, akken tella tidet ayagi yeṭṭafar taɣawsa tamsatit deg twaculin.

D waṭas ileẓran i yesɛan tikelt-a ad tteklen s usseqsi d uqell, tiggura d iseglafen n baba-s nniḍen neɣ n yemma-s nniḍen. D tidet, deg tɣawsa-nni UMAɣLAY n ileẓ yettimɣur deg umatar n uqelleb, aɣwrar d tikennawt.

Yella wawal yerqeqen yeqqar: "Mmi-s n wiyaḍ yettsuɣur deg akk ideggan". S tidet, deg ayagi ula d anwa i yettwafaɛen maca ayagi yezmer ad yettwaḥsab s uḍad d uɣal.

Tmeɣra gar baba-s d yemma-s i yisteqsiyen n ungal, imeṭṭawen d waɣɣaten n yemma-s yettɛeqben neɣ n urgaz yettwaḥqaren, yettwarzen u yettfakkiren, teǧǧa deg UMAɣLAY n ileẓ tikkin ur tettḥuquq ara n uqelleb d tuggdi ur yettettuɣal ara deg tudert akk.

Deg ixxamen i yerzan, tilawin i yettizmitten tteḥqiren tiserdasiwin-nsent mi ad ruḥent ɣer usalun i yettwaɣtasen neɣ ttedɣilen udem-nsent. Azmit n tilawin yeḥbes mi tettḍerr.

Ileẓ i yettwalin akk igiten-a, yettḍerr deg teɣzi ladɣa mi ad yili ɣer yemma-s yettizmitten u yettizmitten, neɣ ɣer tserdasit yettwareqqen d win yettwaḥqaren d tɣawsa tamɣart i UMAɣLAY N YILEẒ.

Segmi yettwales tilifizyun tettfesser tiddukla n twacult. Deg tikkelt nniḍen argaz yettuɣal-d seg ubrid u tettwaqbal ɣur tmeṭṭut-is s lferḥ meqqren. Ass-a tmeṭṭut ur tettuffuɣ ara ad teqqbal argaz-is deg tewwurt, axatar tella tetteɛqal tilifizyun.

Deg ixxamen ineggura baba-s, yemma-s, warraw, tullas, teqqaren ttaɣer-d taggaẓ deg udem n tilifizyun.

Tura argaz ur yezmer ara ad yawi yelli-s ɣef yisteqsiyen n wass, lxedma, atg, axatar ayagi yettqerreb ɣur-s ad yettwali tivizi n yeẓran, isekkiren n Al Capone, tezzi taneggarut n tazzla tamaynut, atg.

Ileẓran i d-yettwessnen deg uxxam anaw-a amaynut, d win yettxemmemen kan deg tlawin, tikeɛɛitin, ifurqanen n umezruy i yettḍerrun akk igiten n tlawin akken ttwalin-tt deg tilifizyun.

D tuggdi d aqeddec-a ifazen n tilifizyun ad yettwaqeddec s yiswi i yeɣḍen. Ma tella tmerrit tqeddec aqeddec-a deg anaw yesɛan azal ladɣa ad tlemmed tussniwin tigamanin, ladɣa ad tselmed taẓuri i d-qqaren n TAYEMMAT TUDERT, ladɣa ad tefke tiselmadin i ggelben i medden, imiren aqeddec-a yella-d d aseɣṛes i tmerrit, yezmer ad yettwaqeddec s tmusni i wakken ad yessnerni umaɣlay ufgan.

D wayen ur yettwasmen akken ad nssiweḍ UMAɣLAY YILEẒ s uẓawan yerqeqen, ur yettwameslay, yerqeq. D wayen yerqeqen akken ad nssiweḍ UMAɣLAY n ileẓran, s tullisin n imekkacen d iblisiyen, igiten n tili d uɛeqqer, adrar n ttagmat-tuɣalin, tazɣelt, atg.

Amulli n umeɣrud-a yezmer a ntwali deg Imeɛraden ur yettwassen ara, imeqqasen iqedcen, atg.

D ayen iɣebben akken ad ḥulfen yimmawraren arraw-nsen, ad ten-fken leɛtab, ad ten-ɛeqben s yiseɣlan iqquran d win ur yettwasmen. Amulli n teɣzi-a d aḥsis, d akrah, n tuffɣa n tayri, atg.

Deg taqeḍɛa nezmer a nwali ileẓran i d-yettwessen gar tiggura, iseglafen d iɣerfan, ad uɣalen d imdanen yerqeqen ččuren d imɣelben u ɛeqqren n wazal d uzmul.

D wayen yellan ɣer zdat akken ad neɛqel teḥwaǧ ad yettwazzel deg uxxam.

D ayen yessefk ad nessen akken ttusma d ttebɣest ilaq ad ttwaɣtassen gar sin yeggiten n tmeɣra n tidet.

BABA YESNUMAYED TTEBƔEST, TAYEMMAT YESNUMAYED TTUSMA. Baba yessenumed TUSNA. Tayemmat tettesmel TAYRI.

TUSNA D TAYRI, TTEBƔEST D TTUSMA ad ttwaɣtassen deg sin yeggiten n tmeɣra tazɣalt.

Ibaba d yemma n twacult ilaq ad ttwaɣtassen i zzagur n ixxamen.

D wayen yellan ɣer zdat, d wayen yellan ɣer teḥwaǧ, akken akk ibaba d yemma n twacult ad ɛeqlen teḥwaǧ akken ad zeɛen deg uɣal n ileẓ IRZAN AYEZGAN n UṚUḤ.

D ayen iɣebben akken ileẓran ineggura ur yesɛi ara lḥess n UZMUL, ayagi yezmer ɣer tullisin n imaɣazen d imekkacen d iblisiyen, tilifizyun, asaru, atg, tterqiqen uɣal n ileẓran.

Tasnudert TUGWIMT n TAZɣERT TANASTIT, deg anaw yellan ɣer zdat d win yettwaffren gar ƐIQEL d TNEQQA.

Deg kraḍ neɣ arbaɛ iseggasen imezwura n tudert, kan tettban deg ileẓ tazzla n TNEQQA, imiren ileẓ yellan ɣer zdat d win yerqeqen, ttusmen deg akk isekkiren-is Psicológicos.

Mi tebda ƐIQEL tettwamel uɣal yerqeq n ileẓ akk tazzla-nni n TNEQQA ttfesser u deg umkan-is ttbanen deg deffir, tazzla n ƐIQEL d win yellan ɣer zdat d win yellan ɣer zdat ufgan.

Akken neɣ tella ur nessettut ara ƐIQEL d TNEQQA, Ilaq a nsetti gar UMAɣLAY d TNEQQA.

Amdan yettlal-d s TNEQQA maca ur yettlal ara s UMAɣLAY, umazwaru ilaq a nexleq.

UMAɣLAY d TNEQQA ilaq ad ttwaɣtassen deg anaw yettwazzen d win yettwaɣtassen.

Deg taqeḍɛa nezmer a nesseffeq mi UMAɣLAY yettimɣur mi tella teɣzi ɣef tmerrit n TNEQQA, Amulli d win n UMAɣLAY.

Ameqqran d tuzzya n waṭas n iseggasen tefkan-aɣ akken a nesseɛqel mi TNEQQA tettimɣur gar-s u ur tetteɛqal deg tmerrit assnerni yettwazzen n UMAɣLAY, Amulli d win yellan deg uɣur ad yesɛu tazzla ur yessen ara, yerqeq deg uɣal maca ur yessin ara, ur yezmir ara.

Assnerni YETTWAGTASSEN n UMAɣLAY d TNEQQA yefka i wayen iqedcen u yelli-s.

Deg TNEQQA nella nessɛu akk acu yellan-d, deg UMAɣLAY akk acu yellan-aɣ-d.

Deg TNEQQA nessaɛu igiten-nneɣ i d-nesserked, deg UMAɣLAY nessaɛu amedya n wid meqqren, acu n lɣiba neslemmed deg uxxam, deg uɣerbaz, deg ubrid.

D wayen yellan ɣer zdat akken ileẓran ad uɣalen ad ttaɣen lqut i TNEQQA d lqut i UMAɣLAY.

TNEQQA tettwaffud s tussma, tayri ur tettḥuquq ara, uẓawan, ijeǧǧigen, tazzla, amaɣlal, atg.

UMAɣLAY ilaq ad yettwaffud s umedya yelhan n wid meqqren, s tselmedt n uɣerbaz, atg.

D wayen yettwaqtan akken ileẓran ad kecmen tmezwura deg seddis iseggasen, uqbel ad ɛeddin deg kinder.

Ileẓran ilaq ad lammden tizerɣatin timezwura ttezɣen, akken tselmedt ad yili i-sen, ḥlawen, ferḥen.

Taselmedt TAMEQQRANT tselmed seg kinder neɣ warda i ileẓran, ilaq ad tteddun deg anaw n UMAɣLAY UFGAN, yettwassnen akken d tɣemri, yettwazzel u yetteddu, akken umaɣlay n ileẓ yettimɣur deg anaw yettwazzen d win yettwazzen.

Isteqsi n tɣawsa n assnerni n UMAɣLAY n ileẓ d assnerni-ines, d win yesɛan tamerri yesɛan azal meqqren i BABA N TWACULT d IMASELMADEN N UƔERBAZ.

Tazzla n UMAɣLAY UFGAN tetteḥbes kan ɣef anaw n umatar Psicológico s wayes yettwaxleq u yettwaffud.

D wayen yettaɣen UMAɣLAY, TNEQQA, ƐIQEL neɣ NEK, yella gar imuddiren n PSICOLOGÍA tazzla.

Kra tteɛṭilen UMAɣLAY s TNEQQA, kra tteɛṭilen ƐIQEL neɣ NEK s TNEQQA.

Waṭas n Uɣerbazen Seudo-Esotéricas neɣ Seudo-Ocultistas yesɛan amkan deg udɣaɣen-nsen leɛyac ur nessin.

Ilaq a nɛellel akken mačči d UMAɣLAY i ilaq ad nesseḥbu.

D wayen yessefk a nessen akken neḥwaǧ ad nesseɣru ƐIQEL, NEK SWAYES NEK, NEK ad tt-nerr d aɣeqqal azɣal.

UMAɣLAY d aqerruy n wayen iqedcen, aqerruy i yuɣalen yessefk ad nexleq.

Deg umaḍal yella CALIGULAS, ATILAS, HITLERES, atg. Anaw n umaɣlay i d-yettili s win i yerqeqen yezmer ad yettbeddel mi ƐIQEL neɣ NEK yettesseḥbu akken yezga.

Ayagi n assḥubu n ƐIQEL neɣ NEK yettɛeqbel yettḍerr waṭas i Seudo-Esoteristas. Ayagi ttamnen akken ƐIQEL d AQDAS, ttamnen akken ƐIQEL neɣ NEK d AMDAN, MONADA AQDAS, atg.

D wayen yessefk, d wayen yellan ɣer zdat, d wayen yellan ɣer zdat, d wayen yerqeqen akken ad nesseɛqel akken ƐIQEL neɣ NEK ur yesɛi acu n wayen iqedsen.

ƐIQEL neɣ NEK d SATÁN n BIBLIA, asɣiɣ n tayunin, ɣellin, uɣal, aḥsis, aɛeqqer, concupiscencias, ttagmat-tuɣalin, tayunin n twacult, tɣemri, tamurt, atg.

Waṭas iqqaren deg anaw yerqeqen deg-neɣ yella NEK YELLAN ƔEF YUGAR neɣ AQDAS d NEK INEQQES.

YELLAN ƔEF YUGAR d INEQQES sin yeqɛacen n tɣawsa n yiwen. NEK YELLAN ƔEF YUGAR, NEK INEQQES, sin yeqɛacen n ƐIQEL yiwen.

AMDAN AQDAS, MONADA, INTIMO, ur yesɛi acu i yeqreb ɣer anaw n NEK. AMDAN d AMDAN d ayen akk. Lɛeql n AMDAN d AMDAN-is.

UMAɣLAY deg iman-is d aqerruy kan u ur yelli wayeḍ. ɣer UMAɣLAY yezmer ad yettban ƐIQEL neɣ AMDAN, akk yetteḥbis ɣer iman-nneɣ.

D WAYEN YEllAN ƔER ZDAT ad esseḥbu NEK, ƐIQEL, akken kan ad yettban ɣer tama n UMAɣLAY-nneɣ, TNEQQA PSICOLÓGICA n AMDAN-NNEƔ TIDET.

D wayen yettwaqtan akken IMELLALEN ad ɛeqlen yezmer teḥwaǧ ad ttwaɣtassen seg tɣemri n UMAɣLAY UFGAN.

Aggad yellan gar umaɣlay d TNEQQA, assnerni yettwazzen n TɣEMRI, TAMSEKRIT d TUZLIT, ETICA TUGWIMT, d wid yessnen taselmedt tameqqrant.
