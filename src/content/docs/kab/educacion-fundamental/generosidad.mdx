Ilaq ad nehḍar iman-nneɣ, ula d wiyaḍ ad aɣ-heḍren, maca ɣef tuccḍa n ddunit, ur llin wid iḥemmlen, wala wid ițwaḥemmlen.

Ayen umi qqaren leḥmala d ayen ur nessin ara medden, țɣelliten-t kan s lɛeqel, neɣ s lxuf.

Lemmer ad yeffeɣ i medden ad ḥemmlen, ula d wiyaḍ ad ḥemmlen, imiren ur yelli wamek ara yili nnfaq ɣef wudem n tmurt.

Aṭas n izewǧen, imi zemren ad ilin d iseɛdiyen, maca maɛna ur țtilin ara, ɣef ddemma n cceḥnat i d-ǧǧan yimaziɣen di tmusni.

Lemmer ad yili fell-asen lxiṛ, ad tettuɣen ayen i sen-yexsen ul, ad ɛicen deg tumert, yellan d tidet.

Tamusni tețneqqi leḥmala, tețțewwet-itt. Ayen i d-yeḍran zik, ayen ur ten-iɛǧiben ara, leɣyur n zik, ayagi akk yellan di tmusni, yețneqqi leḥmala.

Aṭas n tilawin i geẓẓben, zemrent ad ilint d tiseɛdiyin, lemmer ad yili fell-asent lxiṛ, ad ttut ayen iɛeddan, ad ɛicent deg wass-a, ad heḍrent ɣef wergaz-nsent.

Aṭas n yirgazen zemren ad ilin d iseɛdiyen d tlawin-nsen, lemmer ad yili fell-asen lxiṛ, ad semmḥen i tuccḍiwin n zik, ad ṭeggiren ayen i sen-yeqqaren ul, ayen i d-ǧǧan yimaziɣen di tmusni.

Ilaq, ilaq s tezla i izewǧen ad fehmen tiɣmert tameqqrant n lweqt-a.

Irgazen d tilawin ilaq ad țțuɣalen amzun d wid i zewǧen deg wass-a, ad ttut ayen iɛeddan, ad ɛicen deg wass-a s tumert.

Leḥmala d cceḥnat d tiɣawsiwin ur yezmir ara ad ilint ɣer lwaḥid. Di leḥmala ur yelli wamek ara tili cceḥna n kra n ṣṣenf. Leḥmala tețsemmiḥ i lebda.

Tella leḥmala deg wid iḥussen s lɣem n tidet ɣef lehmum n imdukal-nsen d iɛdawen-nsen. Tella leḥmala n tidet deg win ixedmen s wul-is ɣef lxiṛ n igellilen, n imeɣban, n wid yeḥwaǧen.

Tella leḥmala deg win iḥussen s timmuzɣa s tegnit ɣef ufellaḥ i yesrusuyen iman-is i wakken ad yesɣu aqqaɛ, ɣef umeddur i iɛedden, ɣef umeggad i yettḍelleɛ timẓidin, ɣef weydi igellil i iɛedden d wulli, i yemmuten s laẓ ɣer rrif n ubrid.

Mi nɛawan yiwen s wul-nneɣ, mi neḥrez ttejṛa s tegnit, mi nesseɣ tiwriqin n tẓegwa, mbla ma iḍleb-aɣ-d yiwen, imiren yella lxiṛ n tidet, timmuzɣa n tidet, leḥmala n tidet.

Maɛna ɣef tuccḍa n ddunit, ur llin wid yellan yesɛan lxiṛ n tidet. Medden țḥebbiṛen iman-nsen kan ɣef ayen i geṭṭfen, ɣef wayen bɣan, ɣef wayen zemren, ɣef ayen ssnen, ɣef wayen iɛeddan fell-asen, ɣef lferḥ-nsen, atg.

Di ddunit llan aṭas n medden yesɛan lxiṛ n ugaraw. Yella lxiṛ n ugaraw deg umeslaɛṛeq aɛeqli, deg ucceɛqiq n tedwilt i yesrufuyen aqiḍeṛ s tmuɣli yellan kan d win-is i wakken ad yaɣ tazmert, ccan, amkan, igella, atg. Ur ilaq ara ad nɣelli ger umcic d awtul.

Lxiṛ n tidet yettili mbla kra n temmuɣli, maca yeshel ad nɣelli ger lxiṛ n ugaraw yellan kan i iman-is n ugerruj n tsertit, n imaqqaren n leṣwar, n iɛeccaden i yettmennin tameṭṭut, atg.

Ilaq ad nili d wid yellan d wid yellan s wul. Lxiṛ n tidet mačči d win n tmusni, lxiṛ n tidet d abḥar n wul.

Lemmer ad yili fell-asen lxiṛ, ad ttutent akk cceḥnat i d-ǧǧan yimaziɣen di tmusni, akk ayen i sen-yexsen ul i yɛeddan, ad lemden ad ɛicen seg lweqt ɣer lweqt, dima feṛḥen, dima d wid yellan s wul, ččuṛen d tidet n tidet.

Maɛna Nekk d tmusni, yettidir deg yezri, ițmenni dima ad yuɣal ɣer yezri. Yezri yettaɣ medden, yettaɣ tumert, yettaɣ leḥmala.

Tamusni yeskeflen deg yezri, ur tezmir ad tefhem s lekmal tiɣmert tameqqrant n lweqt ideg nella.

Aṭas n medden i aɣ-d-yettarun, țnadint afud, țḍellɛen afud azedgan i wakken ad srewlent ul-nsen i iɛedden, maca drus-iten wid iḥebbiṛen ad afuden wid yeḥwaǧen.

Aṭas n medden i aɣ-d-yettarun i wakken ad aɣ-d-ḥkun ɣef ḥala yeɛfan ideg ɛecen, maca endar wid i yeqqerɛen taɣrumt-nsen i yes-s ara ɛicen, i wakken ad țbeṭṭun d wiyaḍ i yeḥwaǧen.

Ur bɣin ara ad fehmen medden belli deffir n yal taɣawsa, tella sebba, belli ma nbeddel sebba, ad nbeddel taɣawsa.

Nekk, Nekk-nneɣ aɛzizen, d tazmert i ɛeddan deg yimezwura-nneɣ, i d-yessawḍen kra n ssebba n zik, i d-yeǧǧan tiɣawsiwin n tura i yettɛawanen tudert-nneɣ.

Nessaɛa LXIṚ i wakken ad nbeddel ssebba, ad nbeddel tiɣawsiwin. Nessaɛa lxiṛ i wakken ad nqeddem lbabuṛ n tudert-nneɣ s tmusni.

Nessaɛa lxiṛ i wakken ad nbeddel tudert-nneɣ iman-nneɣ s ṣṣeḥ.

Lxiṛ yellan d win n ṣṣeḥ mačči d win n tmusni. Timmuzɣa n tidet d leḥmala n ṣṣeḥ, ur zemrent ara ad ilint d aḍris n lxuf.

Ilaq ad nefhem belli lxuf yettaɣ timmuzɣa, yettaɣ lxiṛ n wul, yettaɣ deg-nneɣ abḥar yelhan n LEḤMALA.

Lxuf d aẓar n yal tkeffut, d aɣbalu n yal nnfaq, d ssem-nni i yettemmuten i yeẓẓalen d i yemmuten.

Sselmaden d sselmadat n uɣerbaz, n ulɣu d tesdawiyin, ilaq ad fehmen tira n ad ɛeddin inelmaden-nsen d tnelmadin-nsen deg ubrid n lxiṛ n tidet, d tezmert d tidet n wul.

Medden iqdimen iɛeqlen, anida ad fehmen ayen yellan d ssem n lxuf, țkessben-t am tujut n ṣṣifa n daxel n wexxam. Aḍris n wayagi yella d tkeffut, d lɛecc, d taluft.

Sselmaden d sselmadat ilaq ad fehmen lweqt ideg nella, ḥala tanzit ideg nzeddeɣ, tira n ad refden tizziyin timaynutin ɣef lsas n tira tamaziɣt, i yezdin d tallit tabuṭumit i yebdan deg yimirat-a n lɣem d uɣilif ger tɛeggiḍt tameqqrant n umyag.

TAṚBIt tameqqrant tegzem ɣef tira tamaziɣt d tusnakt tamaziɣt, i yezdin d asfugel ajdid n tallit tajdidt.

Tazmert n timɛawant ad teffeɣ akk ɣef leɛtab iderren n timɛaddit yellan kan i iman-is. Ur yezmir ad yexdem umdan ma ur yelli llsas n lxiṛ n ṣṣeḥ.

Ilaq ad nefhem s lekmal, mačči kan deg weswir n umyag, maca ula deg iɣerbazen-nni n tmusni taɛeqlit d tmusni taduɛeqlit, ayen yellan d tuga n lxiṛ d lxuf n iman-is. Anida ma nedda s tmussni ɣef wayen yellan deg-nneɣ d lxuf n iman-is, teffeɣ-d deg wul-nneɣ ttejra n LEḤMALA n tidet d LXIR n ṣṣeḥ, i mačči d win n tmusni.
