Tamusni d leḥmala d snat n tgejda s wacu tettwabna tmadunit n tideț.

Deg yiwen uɣareb n tmazalt, yessefk ad nessers tamusni, deg uɣareb nniḍen ad nessers leḥmala.

Tamusni d leḥmala yessefk ad ɛedlent gar-asent. Tamusni mbla leḥmala d aɣbalu n tmesɣelt. Leḥmala mbla tamusni tzemmer aɣ-tewwi ɣer tilufa "LEḤMALA D ASARUG, LAKIN LEḤMALA I YESƐAN TAMUSNI".

Yelzem ad neɣer aṭas akken a nrebḥ tamusni, maca dɣa yesɣun fell-aɣ ad nessenqes deg iman-nneɣ TUDERT TANEẒRUFT.

Tamusni mbla TUDERT TANEẒRUFT yettwaṣubben ɣer wugur, teqqel d sebba n wayen iwumi qqaren BRIBONISMO.

TUDERT yettwaṣubben deg iman-nneɣ, maca mbla tamusni taɣarmant, dɣa tesseɣli-d Iɛeggazen ijehlen.

Aɛeggez ajihli yesɛa TUDERT TANEẒRUFT yettwaṣubben aṭas, maca imi ur yesɛi ara tamusni taɣarmant, ur yezmir ad yexdem acemma imi ur yeẓri amek ara yexdem.

AƐEGGEZ ajihli yesɛa tazmert n Uɣerbaz, maca ur yezmir ad yexdem imi ur yeẓri amek ara yexdem.

Tamusni taɣarmant mbla TUDERT TANEẒRUFT yettwaṣubben dɣa tesseɣli-d aɣwas taɣarmant, lefsad, zzux, atg., atg.

Deg umgaru amaḍlan wis sin, luluf n imusnawen ur yesɛin ara TANEẒRUFT, s yisem n tussna d tmetti, xedmen tiɣriwin țțagʷaden akken ad xedmen imjerrben imusnawen.

Ilaq a nesseɣli-d idles aɣarman yeqwan maca yettɛedlen s waṭas d TANEẒRUFT n tidet.

Nḥwağ ANEGLU AREWREWI d TESNIMAN AREWREWI ma nebɣa s tideț ad nessenger ID deg iman-nneɣ akken ad nessenqes TUDERT TANEẒRUFT deg iman-nneɣ.

D lḥif imi imdanen, ɣef ddemma n laɛɣil n LEḤMALA, sseqdacen AGERREZ deg wannar n tmesɣelt.

Inelmaden d inelmadat ḥwağen ad ɣren tussniwin, umezruy, tusnakt, atg., atg.

Yelzem ad nrebḥ tamusni n tiɣri, s wazal n uzayed i luɣet.

Adeɣri d yessefk. Adjemɛeɛ n tamusni tagensant d illa dɣa, maca tugdi ur d illa ara.

Aṭas n imdanen adjemɛen tamusni ɣef ddemma n tugdi; Țțaggʷaden tudert, lmut, laẓ, leḥqara, ayen ara d-nnin, atg., dɣa ɣef aya ɣren.

Yessefk ad neɣer ɣef ddemma n leḥmala i watmaten-nneɣ, s tɣerɣert n uzayed i luɣet, maca abadan ur yessefk ad neɣer ɣef ddemma n tugdi.

Deg tudert taddayt, nufa d akken wid akk iɣerran ɣef ddemma n tugdi, ad qqlen d BRIBONES ur țɛeṭṭilen.

Ilaq ad neṣseḥ ɣer iman-nneɣ akken ad nɛass iman-nneɣ dɣa ad naf deg iman-nneɣ akk usenqes n tugdi.

Ur ilaq ad nttu abadan deg tudert d akken tugdi tesɛa aṭas n lemɛani. Tikwal tugdi tettɛeddel d azal. Iɛsekriwen deg ugʷar dɣa ibanen d wid yesɛan azal, maca deg tidet ttaṭṭafen dɣa țțaɣen ɣef ddemma n tugdi. Win iḥemmlen ad ineɣ iman-is dɣa iban s tmuɣli tasensant d azawali aṭas, maca deg tidet d aḥeqqar yaggaden tudert.

Yal BRIBON deg tudert dɣa iban d azawali aṭas, maca deg ugʷer d aḥeqqar. I-bribon zemren ad sseqdacen tiɣri d tazmert deg wannar n tmesɣelt m'ara aggʷaden. Amedya; Castro Rúa; deg Kuba.

Nekkni abadan ur nekkas mgal asmed n tudert taddayt, neɣ mgal ṣubbeɛ n ugerrez, maca nekkes mgal laɛɣil n LEḤMALA.

Tamusni d ismuden n tudert dɣa qqelen d wid tmesɣelt m'ara laɛɣil n LEḤMALA.

EGO dɣa ițeṭṭef ismuden d tamusni taɣarmant m'ara tettili temɣer n wayen iwumi qqaren LEḤMALA.

EGO iqerreb deg ismuden d ugerrez m'ara ten-yesseqdec akken ad iqwa.

Nessenger EGO, YO, MI MISMO, ismuden d Ugerrez ad qqimen deg ufus n SER INTIMO dɣa yal aqrub dɣa iqqel d iɛezzel.

Yal anelmad yessefk ad iqeddem deg ubrid n tiɣri-s dɣa ad yeɣer ɣer ugʷer akk iselɣan i d-yettunḍen tiɣri-s.

Adeɣri, ugerrez, ur iḍerr abadan i yiwen, maca ur ilaq ad nqeṛreb deg ugerrez.

Nḥwağ ad neɣer akken ur nqeṛreb deg uɣwas. Win iqerren deg uɣwas d win ibɣan ad yeɣer iselɣan n tiɣri yemgarraden, win ibɣan ad iḍerr i wiyaḍ s ugerrez, win iṣerwen i uɣwas n wiyaḍ, atg., atg., atg.

Yelzem ad neɣer tiselɣatin tidaɣalen d tiselɣatin tinẓuraft akken ad nesɛu uɣwas iqesden.

D URGEN ad ners ɣer TESNERT taɣarmant d tesnert Taneẓruft ma nebɣa s tideț ad nesɛu uɣwas iqesden.

Yiselmaden d yiselmadat n Yikallisen, tizedɣatin, Timawasin, atg., yessefk ad ɣren ɣer ugʷer Tasniman-nneɣ Taɣarimant ma nebɣan s tideț ad seddun inelmaden-nsen deg ubrid n TREWREWT TA-AGEJDAN.

Yelzem ad srebḥen inelmaden TUDERT TANEẒRUFT, ad ssenqsen deg iman-nsen TUDERT TA-TIDEȚ, akken ad ffɣen seg Yikallisen qqlen d imdanen yesɛan tannant, ur qqlen ara d I-BRIBONES ijehlen.

Ur yelḥeq abadan Tamusni mbla Leḥmala. Ugerrez mbla Leḥmala dɣa iǧǧa-d I-BRIBONES kan.

Tamusni deg iman-is d Tameddis Taṭumt, aɣelluy Aṭumt yessefk kan ad t-id-qeblen imdanen iččuren d Leḥmala n tidet.
