Lخير dɣa شر ulac-iten. Ayen yelhan d ayen i ɣ-yefken lfayda, ayen diri d ayen ur aɣ-yefki ara lfayda. Lخير dɣa شر d amsedhu n rrɣaybat n yiman d tɣawsiwin n wallɣu.

Argaz i d-yesnulfan imyalen-a ixataren n Lخير d شر d Atlanti isem-is MAKARI KRONVERNKZYON d aɛeggal yettwassnen deg Tiddukla Tasnislit AKALDAN, tezga-d deg umenẓar Atlantik yeqqlen d idrim.

Ur yelli deg ul n umezwaru aqdim-nni sser belli ad yawi uray aqwa i yemdanen s usnulfu n snat n tefyirin-is.

Imezwura Atlanti درسن لعمق كل القوى التطورية، التطورية المحتملة والقوى المحايدة في الطبيعة ، ولكن هذا الحكيم العجوز خطرت له فكرة تعريف الأولين بمصطلحي الخير والشر. القوى من النوع التطوري دعاها جيدة والقوى من النوع المتخلف عمدها بمصطلح الشر. القوى المحايدة لم يعطها اسما.

D qewwa-nni i d-yettilin deg wemdan dɣa deg ugama, qewwa-nni ur tettbeddil ara d aqqa n tdukli d useɛdel.

Aṭas n leqrun seld mi tenɣer ATLANTIDA s POISEDONIS-ines yettwassnen, fell-as i d-yeḥka FLATO deg الجمهورية-is, tella deg tmadun taɣerfant TIKLYAMISHAYANA yiwen n leqdic aqdim i yexdem tuccḍa tarasant n usteɛmel n imyalen Lخير د شر s ccḥḥaḥ i wakken ad ibeddel fell-asen tikkelt. Argaz n leqdic-a d ARMANATOORA.

S tuɣalin n umezruy s leqrun ur yettḥesben, yesseddaɛ yimdanen s snat tefyirin-a d ten-yuɣalen d lsas n umusnawen-nsen. Ass-a snat tefyirin-a tettalqaḍ-itent deg lmakla.

Tura llan aṭas n IMSEGLAN i bɣan TIRRIT TAɣARANT maca ɣur ccqa n yiman-nsen d ddunit-a yerwan ulawen-nsen ttekken tḥuna deg Lخير د شر.

Tikkelt tettwabeddel ɣef tefyirin Lخير د شر ɣef wayi tekkelt akk IMSEGLAN IɣARANT d uɣalen.

Imyalen Lخير د شر tettwasqeddec-d dima i uṣewweq neɣ aɣḍem tuccḍiwin-nneɣ.

Win iṣewwiqen neɣ yeɣḍem ur yefhim ara. D lɛeqliya ad tfhim tnerki n qewwa tiṭṭawiyin maca mačči d lɛeqliya ad tent-tsewweq s yisem n yelhan. D lɛeqliya ad tfhim tudert n qewwa tilawt-nni maca d tbuqes ad tent-teɣḍem s yisem n dirin.

Yal qewwa tanqersist tezmer ad tuɣal d taɛzizt. Yal qewwa talawt tezmer ad teqqleb d teṭṭawt.

Deg tlemmast n tudert ur yettḥesben n tazmert deg waḍu taṭṭawi llan tudert ur yettḥesben n tazmert deg waḍu talawt.

Deg tlemmast n yal aqqa yella anaw n tazmert i ttetnerkin, tettilawen, d ttuqqlin dima.

Aṣewweq anaw n tazmert dɣa aɣḍem waḥed nniḍen, mačči d afham. Ayen yelhan d afham.

Tudert n TIQQIQA tella d tabaɛit deg tlemmast n yimdanen ɣef ssebba n tḥuna n wallɣu. Lɣaci ttken tḥuna deg tɛerrast Lخير د شر.

TASNEGLIT TANQERSIST n UMUSS N TNASTI tebded ɣef ussdeɛ n anaw n tazmert i ttetlin deg tfekka n wemdan dɣa deg ugama.

UMUSS N TNASTI yesɛa TIɣANT TANQERSIST ur yelli ara deg tikkelt n win uɣalen neɣ deg imyalen imseɣtiwen d imezwura n Lخير د شر.

Deg tlemmast n leɣbar n ussnan n tfekka n wemdan llan qewwa tiṭṭawiyin, tilawiyin, d tin ur ttbaddalen i ilaq ad ttwasdren dɣa ad ttwafehmen ɣer tlemmast.

Amyal n Lخير yeḥbes afham n tazmert tiṭṭawiyin ɣef ssebba n uṣewweq.

Amyal n شر yeḥbes afham n qewwa tilawiyin ɣef ssebba n weɣḍem.

Aṣewweq neɣ aɣḍem ur yettbeddel ara afham. Win yebɣan ad ifak ccal n yiman-is ur ilaq ara ad tent-iṣewweq neɣ ad tent-yeɣḍem. D ayen yeɛzelen ad nfehem tuccḍiwin-nneɣ.

Afham n ɣadb deg yal aɛla n wallɣu d ayen yesɛan azal i wakken ad yali deg-neɣ talwit d uselwef.

Afham n tɣersiwin ur yettḥesben n tcemmaɛ ihi ad yali deg-neɣ tdukkli d tɣersiwin.

Afham n lfehṣ deg yal aɛla n wallɣu d lemriɣ aseqqan i wakken ad yali deg-neɣ tilin taẓidanit taḥeqqanit.

Afham n tḥasda deg yal aɛla n wallɣu d ayen i wessɛan akken ad yali deg-neɣ tamsarit n lemɛawna d usserḥi s rreḥḥa n ussefti d usnerni.

Afham n zzux deg yal ccḥḥa d tseddaṛ d lsas i wakken ad yali deg-neɣ deg talɣa talalt d tseddart ajajig amaḍlan n telwit.

Afham n ayen d taɣawsa-nni n tuffɣa isem-is tqelles, mačči kan deg talɣat-is tiberkaniwin maca daɣen deg talɣat-is tseddartin, d ayen i wessɛan i wakken ad yali deg-neɣ tamsarit n uqeddec.

Afham n talɣiwin yemgaraden n tumert d tleḥḥeft tettbeddel s useɣti n tidyan n uɣerbaz am tneɣruft, ussefreɣ, tmeɣra, uccen, uggad n lmut, tawada n usnerni n YI, uggad n lmergaḍ, atg.

Imessugen n uɣerbaz, tseddarin d tseddarin n tesdawiyin tenhewin iɣarazen-nsen d taɣarazin-nsen akken ad snernin am akken YI yezmer ad isnerni, akken ad d-awin afuden yellan am akken YI yezmer ad yesɛu afuden, atg.

D ayen i wessɛan ad nfehem belli YI ur yessennerni ara, ur yelli dima d win yelhan yerna win itettemen afuden yetɣimi YI.

TASDAḤ TADUSANT tettali deg-neɣ kan s ulmergaḍ n YI. Afuden ttalin deg-neɣ deg talɣa talalt d tseddart mi nfehem ccal-nneɣ snislan mačči kan deg tlata tamatayant maca daɣen deg yal aɛla usnislan d udrim n wallɣu.

Ad nebɣu ad nsennerni d tbuqes, ad neɛru tizedgart d tḥasda, ad nettemmen afuden yetɣimi YI s ssam n tḥasda.

Nesɛa aḥwaǧ i lmut n YI mačči kan deg tlemmast tamatayant maca daɣen deg yal aɛla, idurar, igran d tiserdas n wallɣu. Mi nemmut s tlata, tettɣimi-d deg-neɣ kan TɣAWSA-nni i mazalen TELHA. TɣAWSA-nni i tettnerki n afuden, TɣAWSA-nni i d AZAR n SER-NNEɣ ANESBAɣUR, TɣAWSA-nni ur telli ara n tallit.

Kan s tufḥam ɣer ddu deg tudert ur yettḥesben n qewwa tiṭṭawiyin i d-yettilin deg-neɣ yiman-nneɣ da dɣa tura. Kan s tufḥam deg talɣa tadusant tulɣut yemgaraden n qewwa tilawiyin i yettilin deg-neɣ yiman-nneɣ deg tikelt, nezmer ad nsellew YI.

Imyalen Lخير د شر tettwasqeddec-d i uṣewweq d weɣḍem maca dima i tufḥam.

Yal ccal yesɛa aṭas n ccḥi, tiyita, tuddma d tqejjert. Afham ccal deg tlemmast tamatayant ur yettbeddel afham deg igran usnislan, udrim d uḍfar n wallɣu.
