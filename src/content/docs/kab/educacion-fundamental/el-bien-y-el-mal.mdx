Ulac lxir ulac ccer. Ayen yelhan d ayen i ɣ-isehlen, ayen n diri d ayen ur aɣ-isehlen ara. Lxiṛ d cceṛ d amek ara nsehel fell-awen, d rrɣay n lxater.

Argaz id-yesnulfaen awal ɣef lxiṛ d cceṛ d Atlanti qqaren-as Makari Kronvernkzyon yella d aɛeggal yettwabeynen deg Tdukla Taɣensart Akaldan, tezga-d deg yilel n Atlanti.

Ur yumin ara umezwaru aqdim ɣef ddaɛwessu ara yawi ɣer ddunit, s usnulfu n snat n awal-is.

Iɣensaren n Atlanti ɣurun lqayes ɣef yisallen n tezmert n temẓi, n uttuɣ d tid yeqqimen di tegrest, maca iɣerda aqdim yusa-d ɣer uɣilif ad yefreq gar tizmerin-agi s awal n lxiṛ d ccer. I tezmerin n temẓi qqaren-asent lxiṛ, i tezmerin n uttuɣ qqaren-asent ccer. I tezmerin yeqqimen di tegrest ur sent-yefki ara isem.

Tizmerin-agi ttilint deg wemdan d tegrest, tezmerin yeqqimen di tegrest d targa n lbenyan.

Aṭas n leqrun ɣef uɣelluy n Atlantida s Poisedonis yettwabeynen, iɣef yeḥder Platun deg taɣiwant-is, yella deg tɣerma tagmuḍant Tiklyamishayana yiwen n umɣar aqdim id-yeɣlen deg tuccḍa yettwabeynen n useqdec n awal n lxiṛ d cceṛ, yesseqdec-iten deg leɣlaḍ i wakken ad yesɣer fell-asen leɛqel. Isem n urgaz-agi d Armanatoora.

Mi d-tezger tmezruyt deg aṭas n leqrun, ddunit teɣleḍ s snat n awal-agi, yuɣal d llsas n wakk-n teɛquban. Ass-a snat n awal-agi tettamen-itent ula deg lḥella.

Tura llan aṭas n wid ibeddlen bɣan ad uɣalen s leɛqel, maca ɣef ddaɛwessu-nsen d ddunit-agi teɣli fell-asen, tɣenṣel tixmi-nsen gar lxiṛ d cceṛ.

Leɛqel yettwasɣer ɣef awal n lxiṛ d cceṛ, ɣef way-agi wakk-n iɣerda iɛqulen d aɛeqli.

Awal n lxiṛ d cceṛ yettuseqdacen i wakken ad nesseɣret neɣ ad nḥaqer tuccḍawin-nneɣ.

Win isseɣren neɣ iḥeqren ur ifhim. Ɛeqli ad nefhme amek ttemɣuyent tezmerin, maca ma nesseɣra-tent s awal n lxiṛ, ayagi mačči d aɛqel. Ɛeqli ad nefhme ayen iḍerrun i tezmerin n uttuɣ, maca ma nḥaqer-itent s awal n ccer, ayagi d lehbal.

Tezmert ur tettaǧǧa ara tzemmer ad tuɣal tezmert tḥettem. Tezmert n uttuɣ tzemmer ad teqqel tezmert n temẓi.

Deg gar yisallen ur tenfuffuḍen n tezmert di temẓi, llan yisallen ur tenfuffuḍen n tezmert di uttuɣ.

Deg gar yal yiwen n wemdan llan iɣerbazen n tezmert, ttemɣuyun, ttuɣun d ttbeddilen.

Ad nesseɣre yiwen n uɣerbaz n tezmert, ad nḥaqer wayeḍ, ayagi mačči d ufhimi. Ayen yesɛan azal d ufhimi.

Tadyant n tideț teqqel-d d ayen iweɛren gar medden ɣef ṣṣeḥ n uɣenṣul n tixmi. Medden tɣenṣelen gar lxiṛ d cceṛ.

Tasnijjya id yettbeddilen n Usud amagnu tessɣer ɣef uɣerbaz n tezmert i yettxeddmen deg tfekka n wemdan d tegrest.

Usud amagnu yesɛa taɣawsa id yettbeddilen ur tesɛi ara kra atekki ɣer leɛqel n wid iɛeqlen, neɣ ɣer wid yettḥettimen ɣef lxiṛ d cceṛ.

Deg gar tasnajjya d tɣuri n tfekka n wemdan llan tizmerin ttemɣuyun, ttuɣun d tezmert yeqqimen di tegrest, ilaq ad nettwaɣer nețțeɛqel fell-asent.

Awal n lxiṛ yeṭṭfeḍ amek ara nefhem tezmerin d temɣuyin ɣef usseɣri.

Awal n ccer yeṭṭfeḍ amek ara nefhem tezmerin n uttuɣ ɣef uḥaqri.

Ad nesseɣre neɣ ad nḥaqer ur yettɛeddi ara ɣer ufhimi. Win ibɣan ad yekkes tuccḍawin-is ur ilaq ara ad yesseɣre neɣ ad iḥaqer. Ilha ad nefhem tuccḍawin-nneɣ.

Ad nefhem ɣadb deg wakk-n inurar n tixmi d ayen iweɛren i wakken ad yali deg-nneɣ ṭṭman d lehna.

Ad nefhem aṭas n lemɛani n ṭṭmeɛ d ayen ilaqen i wakken ad yali deg-nneɣ lḥub d useḥluf.

Ad nefhem cceḥwa deg wakk-n inurar n tixmi d ṣṣifa iweɛren i wakken ad yali deg-nneɣ taɣda s ṣṣeḥ.

Ad nefhem tḥesdest deg wakk-n iraffen n tixmi d ayen ikfaen i wakken ad yali deg-nneɣ lqima n lemɛawna d lferḥ ɣef leεnaya d temɣuyt n wiyaḍ.

Ad nefhem zzux deg wakk-n lemɛani-is d lebṣayṣ id yettwasɣeren, ɣef wakk-n ad yali deg-nneɣ s ṣṣifa n taẓẓeyt n ddunit tameqqrant n tesmekta.

Ad nefhem ayen yellan d aɣbel qqaren-as tiɣzi, mačči kan deg ṣṣifat-is tin ifehhecen, maca ula deg ṣṣifat-is tudmimt, d ayen ilaqen i wakken ad yali deg-nneɣ lqima n temɣuyt.

Ad nefhem aṭas n ṣṣifat n cceḥwa d uɛeyyaɛ yerɣa ad negmer cceḥwa n uẓar n uẓu, am tmeɣriwin, tisekrin, tamawt n laṣel, aɣyul n aksum, agwad n lmut, aray ad yeqqim nekk, agwad n umeḥḥu, atg.

Aselmaden n uɣerbaz, n uɣerbaz amenzu d tesdawit nsen-itnadi deg inelmaden-nsen, inelmaden i d-yussen yelha ad ttwaliyen, amzun nekk yezmer ad yelhu, ad afen ṣṣifat yelhan, amzun nekk yezmer ad yesɛu ṣṣifat yelhan, atg.

Ilaq ad nefhem belli nekk ur yelhi ara, ur yettili ara d ameqqran, win itettmen ṣṣifat yelhan yesnerni nekk.

Tawuri takemmelant kan tettlal deg-nneɣ s ussefser n nekk. Ṣṣifat yelhan ttettlal deg-nneɣ s ṣṣifa taɣlayant, ass mi nfhem tuccḍawin-nneɣ mačči kan deg unnar n tmusni, maca ula deg wakk-n iraffen ibedlen, d tin ur nettwali ara n tixmi.

Ad nebɣu ad neɣleq d lehbal, ad nettmen taɣda d taɣda, ad nettmen ṣṣifat yelhan teɛni ad nesnerni nekk s ssem n tḥesdest.

Nettwazma ad yemmet nekk mačči kan deg unnar n tmusni, maca ula deg wakk-n isekkilen, tamiwin, inurar n tixmi. Mi nemmut s ṣṣeḥ ad yeqqim deg-nneɣ ayen yellan d akemmelan. Ayen yellan yeččuṛ s ṣṣifat yelhan, ayen yellan d ṛṛuḥ n Ser-nneɣ, ayen yellan ur yelli ara d ayen n zman.

Kan ma nefhem ayen yellan d aqqaɛ n yisallen d yimɣan n tezmert i d-yettilin deg-nneɣ d tura. Kan mi nefhem s ṣṣifa takemmelant ṣṣifat ibedlen n tezmerin n uɣelluy i d-yettilin deg-nneɣ deg lweqt ɣer wayeḍ, nezmer ad nessefsir nekk.

Awal n lxiṛ d ccer yettuseqdacen i wakken ad nesseɣre, ad nḥaqer, maca ur yettɛeddi ara ɣer ufhimi.

Yal tuccḍa tesɛa aṭas n lemɛani, ussan, tɣuṛma d tunɣilin. Ad nefhem tuccḍa deg unnar n tmusni ur teɛni ara neffehm-itt deg wakk-n iraffen yettbeddilen, n tixmi.

Yal tuccḍa tzemmer ad teddu si unnar n tmusni ad tkemmel deg iraffen-nniḍen n tixmi.

Aɣadb yettales s tɣessa n uɣeṭṭaf. Aṭas itettmen ad ur ttettmen ara, llan wid ur ttettmen ara idrimen maca ttettmen tazmert, ṣṣifat yelhan, tayri d lferḥ da neɣ ɣer deffir n lmut, atg.

Aṭas n yirgazen d tlawin ṭṭfen-iten leɛqel d cceḥwa ɣer wid ibedlen deg ṣṣifa n ṣṣifa, akken ma llan ḥemmlen tinit, aẓar-nsen yettxattaɛ fell-asen, cceḥwa yettales deg tmuski n teqdimt.

Aṭas n wid yesḥassen yesḥassen ɣer wid yellan d iɣudan, ttxellṣen timekkizin fell-asen i wakken ad uɣalen d iɣudan.

Aṭas n wid yesḥassen yesḥassen ɣer wid iqeddcen i medden ɣef way-agi bɣan ad uɣalen d imeqqranen, ttaɣen-d gar wid yesḥassen fell-asen, ttaǧǧan akk-n tilufa-nsen i tɣawsiwin-nsen.

Llan wid yesɛan zzux ɣef tɣella-nsen, idrimen-nsen, tawaɣuri-nsen, llan wid yesɛan zzux ɣef ṣṣifa-nsen tudmint.

Diyujin yesɛa zzux deg tmurt ideg yettnaḍ deg wasmi yusa-d ɣer taddart n Suqraṭ, yenna: "Tɛebbi zzux-ik Suqraṭ, tɛebbi zzux-ik". "Ih, Diyujin, s zzux-ik tɛebbi zzux-iw". D awal n Suqraṭ.

Tilawin iɛebban ddunit ttemɛawadent tcekkilent iman-nsent, syesqesqiyent-d iman-nsent s wakk-n i zemrent i wakken ad d-tɛebbi tḥesdest n tilawin-nniḍen, maca tawaɣuri-nni daɣen ttezzi s teɣzi n ddunit tameɛzuzt.

Yeqqar-d lxabar belli Arisṭibu afiluzuf agrigi yebɣa ad yesken i medden ɣef tmuski-is d temekkest-is, yels-d tmeqqen taqdimt yeččuṛen d tiɣermiwin, yeṭṭef deg ufus-is ayeffus aɣrab n tilifizuft, iruḥ ɣer yiberdan n Atina. Ass mi t-yeẓra Suqraṭ yenna-as: "Tettwali-nneɣ tawaɣuri-ik seg tɣermiwin n tmeqqent-ik, a Arisṭibu".

Aṭas i yellan deg tɣellabt ɣef tɣawsa n tiɣzi, maca llan medden xedmen aṭas i wakken ad sɛun tudert, maca sɣuṛen i wakken ad ɣren, ad issinen iman-nsen i wakken ad ssufsugen nekk.

Aṭas i yerran cceḥwa d uɛeyyaɛ, maca ɣef ddaɛwessu-nsen yesker-d tisekri d tɣawsiwin-nniḍen.

Yal tuccḍa tesɛa aṭas n udem d temɣuy, i tettili, tettwaɣer deg ṣṣifa id yettwasɣeren si llsas aneggaru n tesnijjya ɣer llsas aneɣrus.

Deg taɣara tameẓẓaɣt n yisli daɣen tettuffeɣ tɛedda.

Tɛedda daɣen tlezzi deg uɣud, deg uɛissi deg tezdeg.

Ulac lxir d ccer, awal-agi yettuseqdac kan i wakken ad yettnadi deg tidett, ad d-nɛeddi tiɣri taqqaɛt n tuccḍawin-nneɣ.
