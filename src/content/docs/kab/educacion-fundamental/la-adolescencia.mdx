Tura d-yewweḍ lweqt ad neǧǧ deg yiwen n wuḍan sser d tsekrifa n lekdeb, d tirrugza yeɛnan amennuɣ n tmeṭṭut.

Ilaq ad nefhem s tselɛayt amennuɣ n tmeṭṭut deg ilemẓiyen n snat n tewsitin.

Deg ɛecṛa d ukkuẓ n yiseggasen n lɛemr, tettban deg tfekka n umdan tameṭṭut, tezmert-nni i tettazzal s ddaw n uẓeṭṭa n wesɣan-aɣersan.

Aḍris-a yessegrew tafekka n umdan, isbeddal taɣect deg urgaz, yessileɣ taseddart deg tmeṭṭut.

Tafekka n umdan d afabriq n ṣṣeḥ i yesbeddalen ayen ɣerssen ɣer lemɛani ilemẓiyen.

Lqut i nesserɣay ɣer uɛebbuḍ ɣer wuqerruy yettɛeddi deg waṭas n beddlen d useffi i tɛeddi ldiwan deg ayen i d-yettwaẓerren s yisem n Ens.-Seminis (Lqima n Semen)

Amedlu-nni ilemẓiyen, yettaẓẓẓuɣen, yeshelen, i lmenni, yesɛa deg iman-is, deg lqima, akk iɣerɣazen n tudert.

Gnosticisme yeɛteref deg lmenni CAOS s wacu i d-tettban tudert s lehna.

Imɣuzen ilemẓiyen n teɣri i yellan am Paracelse, Sendivogius, Nicolas Flamel, Raymundo Lulio, atg., ɣuren lehna d tameqqrant ENS-SEMINIS neɣ mercurio n tfalsafa tussist.

VITRIOLO-agi, d azamul n ṣṣeḥ ilemẓiyen i yettwaẓerren s tselɛayt, i ddaw n waman deg ɣerɣazen yessemlalen.

Deg mercurio-agi n tmusni taqburt, deg lmenni-agi, uffir deg ṣṣeḥ akk iɣerɣazen n tilin.

D ayen yeɣḍen aṭas ilemẓiyen ɣef tɛecfu n uqader ad ilin ɣer vicio n usteɛmal neɣ ad ṭṭfen deg ubrid infra-sexuel n homo-sexualisme.

I warrac d ilemẓiyen ad fken isalan s tmusni ɣef waṭas n thema ad ten-nesseddu ɣef ubrid n waddal i yessuffuɣen tudert s lemḥayen, maca maɛjeb, mi tettban tezmert tameṭṭut i yettbeddun ilemẓiyen, akk aẓnuzu n twaculin am uselmad n uɣerbaz, d isalen deg lɣerd n ṣṣeḥ d tmusni yeqqnen, ad d-senheyyen ad ssusmen s leɛnaya.

Uffir n leɛnaya d leɛyub d ulfan n tlawin. Ssusmen ɣef amennuɣ n tmeṭṭut d leɛyub. Hemmlen ɣef umennuɣ n tmeṭṭut d leɛyub daɣen.

Ma ssusmen aẓnuzu n twaculin d uselmad, ahemmlen i tenɣent tameṭṭut d ilemẓiyen i ur neẓri.

Ma ilemẓi ur yezmir ad yesteqsi aẓnuzu n twaculin neɣ aselmad, ad yesteqsi iɣarimen n uɣerbaz i yellan yettwaḍerrun ɣef ubrid itteddun. Nnati ur yettarra aṭas n lweqt d ilemẓiyen ilemẓiyen itteddun leɛyub ad ilin ɣer vicio n usteɛmal neɣ ad ṭṭfen deg ubrid n homo-sexualisme.

Vicio n usteɛmal ad yeqqen akk leɛnaya n uẓeṭṭa. Ilaq ad neẓer illa illaɛlala ilemẓiyen ger lmenni d uẓeṭṭa. Ilaq ad nesɣerɣer lmenni. Ilaq ad nemel tudert n uẓeṭṭa.

Uẓeṭṭa yemlal transmutando la energía sexual, sublimizándola, convirtiéndola en potencia cerebral.

Deg tewsit-agi yeqqen lmenni cerebrizado d uẓeṭṭa seminizado.

Tussna Gnóstica tessɣar ɣef tɛecfu endocrinología d yessalen imezɣan d usistema i transmutar iɣallen tameṭṭut, maca amennuɣ-a ur yettḍfar ara ddaw n udlis-a.

Ma yebɣa ameqqran isalen ɣef Gnosticisme ila ad yesɣar idlisen-nneɣ Gnósticos d ad yekcem ɣer usɣar-nneɣ.

Ilemẓiyen ila ad sublimar iɣallen tameṭṭut cultivando el sentido estético, al aprendiendo la música, la escultura, la pintura, realizando excursiones a las altas montañas, etc.

Acḥal n iɣerɣazen i iɣerɣzen ad ilin yelhan ad d-ẓuren!

Acḥal n iẓeṭṭa ad d-kecmen! Akk ɣef tɛecfu n teɣri n liser deg lweqt i d-yewweḍ.

Vicio n usteɛmal, am deg ilemẓiyen am deg tullas i tuɣal d teẓẓiḥen ɣer uẓɣal i yessifen afus.

Manicomios d ufullil n yirgazen d tlawin i yeqqnen uẓeṭṭa-nsen deg vicio-aɣi n usteɛmal. Ad d-uɣalen deg imezdaɣ d manicomio.

Vicio del homo-sexualisme yesɛa tẓuran iderrsen n tgellidt-a i d-yeqqnen d perversa.

Yettwaẓerren deg tmurt am Inglaterra i yettwasellen d kultos d súper-civilizados, ad ilin ddaw n usser cinemas i yessenɣnen fellasafilmen n tewsit homo-sexual.

Yettwaẓerren dayen deg Inglaterra anda d-heggan kra n lḥezban i ten-nesseddu amenzay n uqader n tewsit homosexual.

Deg imeqqranen n ddunit yella tura prostíbulos d clubes n tewsit homo-sexual.

Tadukla tenebrosa n yixebbaten n tmeṭṭut, ass-a deg wass yesɛa iseddayen yettwaḍerrun i yettwaẓerren deg terwa-nsen yettwaḍerrun.

I waṭas n imeqqranen ad ten-yeɣḍen aṭas ay-a n "terwa yettwaḍerrun" maca ur ilaq ad nettu deg akk lweqt n umezruy llan waṭas n terwatin n leɛyub.

Tadukla morbosa n yixebbaten n tmeṭṭut, d ddaw n ulɣu-nni n leɛyub.

Ixebbaten n tmeṭṭut ad ilin ɣer tɛecfu n tɛecfu n tmura ɣer tmurt n uɣerbaz.

Mi ara ad yerzef homo-sexual ɣer lḥebs, ad d-ili ɣer uɣerbaz ɣef tɛecfu n yirgazen n tmurt n leɛyub.

Ma yettwaḍerrun ad d-yekker tɛecfu, ad yelhu ɣer uselmed n leɛnaya n akk idmaren siniestros n Tadukla n leɛyub.

Idmaren tenebrosos n homo-sexualisme ad ttwaẓerren ger-asen s unifor i yettwaẓerren.

Yettwaẓerren ad d-ilhu ameqqran ad yesɛu unifor, maca akka i tettili. Unifor n homo-sexuales yettḍfar i akk moda i yettbeddun. Imɣuzen ad d-begnen yal moda tajdiṭ. Mi ara ad tuɣal moda d tazdaɣt, ad d-begnen tayeḍ. Deg tewsit-agi unifor n tadukla n leɛyub yettbeddal yal tikkelt.

Akk imeqqranen n ddunit, ass-a deg wass yesɛa imelyunen n homo-sexuales.

Vicio del homo-sexualisme ad yebdu aseyreɣ-is deg ilemẓiyen.

Waṭas n uɣerbaz n yilemẓiyen n urgazen d teflukin d prostíbulos n tewsit homo-sexual.

Imelyunen n teflukin ad d-kecmen s ubrid tenebroso n yixebbaten n urgaz.

Imelyunen n yilemẓiyen n tewsit taqcict d homo-sexuales. Tadukla n leɛyub ger homo-sexualisme taqcict d tezmert, am tadukla n leɛyub ger tewsit tazewwert.

Yettafeg ad neǧǧ deg wadda n uḥric i uqeɛɛen d ad nheggi i ilemẓiyen n snat n tewsitin, akk iɣerɣazen tameṭṭut.

Akka ad ilin iɣerɣazen ilemẓiyen deg ubrid n REGENERACIÓN.
