Yewweḍ-d lawan ad neǧǧ s lekmal lḥecma yerɣan akk-d tmuɣliwin ur nettban fellasen ɣef wugur n tdukli-nnsen.

Ilaq ad nefhem s tiddi d ṣṣeḥ ugur n tdukli-nnsen n ilemẓiyen n snat n tewsitin.

Deg leɛmeṛ n ṛebɛṭṭac iseggasen, teffeɣ-d deg tfekka n ulemẓi tazmert n tdukli-nnsen, i tetteddu s teḥḥerci deg wuggug n unagraw asiman-asɣan.

Anaw-a umgarad n tezmert yettbeddil tafekka n wemdan, yesseɣɣay taɣect deg urgaz, yebda lxedma n tɣerrat deg tmeṭṭut.

Tafekka n wemdan d aẓruẓ ameqqran i yettbeddilen iɣunab s tɣawsiwin timaynutin i tudert.

Tettɛeddi tmekla i nettɛeddi ɣer iɛebbuḍ deg waṭas n wbeddilat d usḥenniɛ armi d taggara deg tɣawsa-nni tazelmaẓt, talilt i d-yenna Paracelse s yisem n Ens.-Seminis (Tameddent n Uẓu).

Aglaz-nni asilan, yelhan, yeshel ad yettbeddel, aẓu-nni, yesɛa deg-s s taɣewsa akk i d-yeffɣen seg tudert.

Agnustisme yettɛerref deg uẓu CAOS i d-yeffɣen s teḥḥerci tudert.

Iɛeqqayen iqdimen iqdimen am Paracelse, Sendivogius, Nicolas Flamel, Raymundo Lulio, atg., ɣran s leqder aɛlayan ENS-SEMINIS neɣ lmerkuri n tsekla taɣerfant tuffirt.

VITRIOL-a, d aksum n tezmert ṣaḥ ibdaɣ s leɛqel, s tellaɛraft deg tlemmast n tɣerrafin tizemra.

Deg lmerkuri-a n tmusni taqburt, deg uẓu-a, uffɣen-d ṣaḥ akk imkanat n tilin.

D lɛib ma waṭas n ilemẓiyen ɣef laɣlaq n tamusni n tmusni n ddunit d tezmert n tdukli-nnsen ɣer lɣiba n tmesxer neɣ ad tɛeddin s tɣerɣer ɣer ubrid n tilawt n waɛrac.

I warrac d ilemẓiyen ttefken-asen tamusni taɣersawit ɣef waṭas n usentel d yettɛeddun-ten deg ubrid n tdukkli i yettaɣen tudert s ccwal, maca ɣer lɛerc i d-yettasen tdukli-nnsen i d-yessukksen ilemẓiyen, imawlan n twacult d yiselmaden n uɣerbaz, yellan ɣef tɣarast n udabu d teẓri yelhan d tmusni tamerreẓt, ad ssusmen s tɣerma.

Llan ssusmat iɣermanen d llan awalen imseɛɛacen. Ssusem ɣef wugur n tdukli-nnsen d aɣrum. Awal s lɣella ɣef wugur n tdukli-nnsen daɣen d aɣrum.

Ma ssusmen imawlan d yiselmaden, ad awalen iɛibanen n tdukli-nnsen d imeɣban ad ilin ilemẓiyen ur nessin.

Ma ulemẓi ur yezmir ad isteqsi imawlan neɣ iselmaden, ad isteqsi imdukkal-is n uɣerbaz i yellan yakan uɣnen deg ubrid ur yelhan. Taggara ur ttaɛqal ɣef waṭas n wakud d ulemẓi amaynu, yettḍfar iɛeqqaben ur ṣṣeḥ ad yettsebbi deg lɣiba n tmesxer neɣ ad tɛeddi deg ubrid n tilawt n waɛrac.

Lɣiba n tmesxer tezzezzer s lekmal taɣewsa n uɣezdis. Ilaq ad nessin yella aṭas n tdukli gar uẓu d uɣezdis. Ilaq ad nessɣerdis uẓu. Ilaq ad nessaẓu uɣezdis.

Uɣezdis yessaẓu yettbeddil tazmert n tdukli-nnsen, yettɛelleɛ-itt, yesseɣɣay-itt ɣer tezmert n uɣezdis.

Deg tɣarast-a yeqqim uẓu yettɣerdis d uɣezdis yessaẓu.

Tusnakt n gnusis yettɛerraf s tlelliɣ tazmara d yesselmaden ttaɣalen d unagrawen i wakken ad ibeddelen tizmar n tdukli-nnsen, maca usentel-a ur yettɛedda deg tlemmast n yedlis-a.

Ma yebɣa aqra tamusni ɣef gnusis ilaq ad iɣer idlisen-nneɣ n gnusis ad ikecɛ amdan deg ɣur-nneɣ.

Ilemẓiyen ilaq ad sɛellɛen tizmar n tdukli-nnsen ad qaren taɣewsa tazzayt, ad lemden tacennayt, tẓuri n udlis, tiẓuri, ad xedmen ussan ɣer idurar ɛlayanen, atg.

Acḥal n wudmawen i yezmer ad ilin yesnɛen ad tṛuḥen!

Acḥal n uɣezdis i yettṛuḥen! Akk ɣef laɣlaq n taɣɣat n tɣerɣer deg lawan iṣeḥan.

Lɣiba n tmesxer, deg ilemẓiyen d tlennawtin, tuɣal teqqen akken ad teɣḍes afus.

Ixxamen n ireqqasen ččuren d irgazen d tilawin i yewwin uɣezdis-nsen deg lɣiba d tmesxer. Aqdar n yemsaxaren d axxam n ireqqasen.

Lɣiba n tilawt n waɛrac tezzerzer iẓuran n race-a yekfan d yettbeddilen.

Yettɣil d ur yettban deg tmura am Linglizi i yettɛeyyun iɛellalen d imayluten, llan s tilelli idlisen i deg ttbanen ssura n tilawt n waɛrac.

Yettɣil d ur yettban d ṣaḥ deg Linglizi i deg xeddmen ɣef wakken ad sɣerken s tɣerma aɣaren n waɛrac.

Deg temdinin timqranin n ddunit llan tura ixxamen n tɣewɣatin d yixxamen n waɛrac.

Tadukli tarqiq n iɛdawen n tmeṭṭut, ass-a deg-s ad teɣḍes s temlilit-nsen tesseddiḥen.

Aṭas n iɣerramen ad yeqqen-iten waṭas ayagi n "tmlilit tesseddiḥen" maca ur ilaq ad nettu deg akk wakuden n umezruy llan yal tameɣra n tɣerma.

Tadukli tarqiq n iɛdawen n tmeṭṭut, war ccek d tameɣra n tɣerma.

Iɛdawen n tmeṭṭut ttɣumun deg yal lawan neɣ akken deg imkanen iferḍanen deg tlemmast n tɣerma taɣersawit.

Mi yerẓa waɛrac ɣer lḥebs, sserḥen-as-d sɣaḍ afus ɣef taɣessa iṣeḥan n urgazen iferḍanen n tadukli n tɣerma.

Ma yeɣli umessas deg ddunit, sserḥen-as-d sɣaḍ afus d yiɣerfan deg uẓar n akk iɛedwafen d tadukli n tɣerma.

Iɛeggalen iberkanen n tilawt n waɛrac ttɛerrefen gar-asen ɣef tserɣint i ttɛeyyun.

Yessedhiḥ ad nessin imerwaten ttɛeyyun tserɣint, maca akken. Tserɣint n waɛrac tettɛeddi ɣer yal tɣara i d-yettbanen. Imerwaten ttbeddilen yal tɣara tamaynut. Mi tuɣal tɣara d tagtama, nettawi-d ad beddlen tayeḍ. Deg tɣara-a tserɣint n tadukli n tɣerma tettḥadra yal ass.

Akk temdinin timqranin n ddunit, ass-a deg-sen llan imelyan n waɛrac.

Lɣiba n tilawt n waɛrac yessukksen ameddu deg ilemẓi.

Waṭas n uɣerbazen n ilemẓiyen irgazen d tilawin d ixxamen n tɣewɣatin n waɛrac.

Imelyan n tlennawtin ttɛeddayant s lebda deg ubrid aberkan n iɛdawen n urgaz.

Imelyan n ilemẓiyen n tewsit tamettant d waɛrac. Tadukli n tɣerma gar tilawt n waɛrac tamettant teqqen, am tadukli n tɣerma deg tewsit tazikrt.

D tezɣent ad neǧǧ s lekmal d s taggara lḥecma yerɣan d ad neqqar i ilemẓiyen n snat n tewsitin, s teɣẓer akk iɣerfan n tdukli-nnsen.

Akk ad izmiren ad ttɛeddin tizzwarin timaynutin deg ubrid n TIJJIḌA.
