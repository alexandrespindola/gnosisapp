Deg-gmaḍal llan waṭas n wid yettmeslayen s uẓawan i d-ițeṭṭfen ulawen, maca drus kan wid yessnen ad sselen.

Ad tesselḥiḍ d ayen iweɛṛen aṭas, drus kan wid yessenen amek ara sselḥin s tidet.

ASMI I D-YETTMESTLAY USELMAD, tselmadt, win yettmeslayen deg umḍiq, amḍiq n tmeẓla yettban-d d win iḥerzen aṭas, am akken ttḍefren deg yal taɣect n win yettmeslayen, kullec yettak-d tamuɣli belli țṣeḥḥan, belli llan deg uḥraz, maca deg ul n yal yiwen, yella umaru i yesseɣṛaden yal taɣect n win yettmeslayen.

UMARU-YA D NEKK, NEKK S TIMMAD-IW, NEKK S TIMMAD-IS. Aɣbalu n umaṛu-ya d ad yesseɣṛed s lɛib, ad yesseɣṛed s tmuɣli n diri taɣawsa i d-iqqar win yettmeslayen.

NEKK YESSEƔRAD ƔEF USSUFEƔ N WID IQUBLEN, wid iqebblen zik, lxuf, zzux, tmeɣṛiwin, ixemmimen, tira, atg., atg., atg.
