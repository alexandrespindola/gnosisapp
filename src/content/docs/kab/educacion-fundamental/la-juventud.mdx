Tameẓlit tebḍa ɣef snat n talliyin, yal ta tesɛa sebɛa iseggasen. Tallit tamezwarut tebda deg umur n 21 iseggasen, tennekmal deg 28. Tallit tis snat tebda deg umur n 28 tefser deg umur n 35.

Isurifen n tmeẓlit ttufraren deg uxxam, aɣerbaz d webrid. Tameẓlit tebna ɣef tigertilt n Usselmed Aseklan teffeɣ-d d win yesnernayen, d win yesnernayen tilelli.

Tameẓlit tebna ɣef isurifen ikeddaben, ɣef waya d aɣaras ikeddaben.

Aṭas n yirgazen sseqdacen amur amenzu n tudert i wakken ad ssinɣen amur nniḍen.

Ilmeẓyen s unamek ikeddaben n urgaz ikeddaben, ttagren deg ufus n tlawin tiznatin.

Tirrugza n tmeẓlit d tira ttwaḍefren ɣef tuɣmert ad t-ixelsen s rrbeḥ ɣlayen ɣef tlata iseggasen.

Bla Usselmed Aseklan tameẓlit teffeɣ-d d tameɣlalt n dayem: d tissit n tuccḍa, aman n tẓalli d lḥirra tayeffust.

Ayen akk ara yili urgaz deg tudert-is yella d imsedreq deg wussan imenza n tlata iseggasen n lwejud-ines.

Seg yikersen imeqqranen akk n ufgan i nettaẓẓar fell-asen, ama deg talliyin iɛeddan ama deg tallit-nneɣ, aṭas seg-sen i yebdan uqbel tlata iseggasen.

Argaz i yewweḍ tlata iseggasen yezmer ad yizmer iman-is am akken yeffeɣ-d seg nnfaq ameqqran i deg yezra aṭas n imdukkal-is ttagren wa deffir wa.

Deg umur n tlata iseggasen irgazen d tlawin xesren akk tissirt-nsen d tissirt-nsen yerna ma fellxen deg yikersen-nsen imenza, ttṛeqqen d zzelɣen aɣaras.

Tilmiwin n umɣar ttazzalen deffir tilmiwin n tmeẓlit. Bla Usselmed Aseklan tmedyazt n tuɣmert d aɣbalu.

Tameẓlit d win yettarran. Tissirt d yelli n tmeẓlit, maca d ikeddaben, ur tettɣimi.

Tameẓlit tesɛa Tasekla tuder d Lḥekma d tuder. Dɣa ur tettili ara deg tudert ilmeẓyen n Lḥekma tuder d Tasekla tuder.
