Tameṭṭut d ilemẓi tettwabḍa ɣef sin n talliyin n sebɛa iseggwasen yal yiwet. Tallit tamezwarut tebda deg 21 iseggwasen n lɛemr yerna tekfu deg 28. Tallit tis snat tebda deg 28 yerna tefser deg 35.

Isurifen n tlemmẓa ttilin deg uxxam, aɣerbaz d ubrid. Tlemmẓi yettwarfedn ɣef lsas n ULMAD AMENZUY tettban s tideț d tin yessalayen yerna s tezɣent d tin yessnerfayen.

Tlemmẓi yettwarfedn ɣef lsas ur ṣṣeḥan d aɣaras iɣelben, d ayen i d-yettasen s tmuɣli.

Aṭas n yirgazen i seqqeden amur amezwaru n tudert deg yuɣalen amur yeqqimen d tin ur tban ara.

Ilemẓiyen ɣef ddemma n tmuɣli yellan d tikerkas ɣef wergaz n lekdeb, ttaɣen tilawin n tumert.

Asugget n tlemmẓi d isenfaren yettwagren mgal tɣerrfet yettwacellxen s rrbaḥ ayɣezfan ɣef tlata iseggwasen d ussan.

Mbla ULMAD AMENZUY tlemmẓi tuɣal d aseṛxu ameddal: d tbuɣda n tuccḍa, d ccṛab d wul yeɣlin ɣer lewḥuc.

Ayen akk ara yili d argaz deg tudert-is yella deg umḍiq n tezmer deg wussan amezwaru n tlata iseggwasen n tili.

Seg yisenfar imeqqranen n wemdan i neẓra, ama deg talliyin yezrin neɣ deg tin-nneɣ, aṭas seg-sen i yettwabdan uqbel tlata iseggwasen.

Argaz i yuɣalen ɣer tlata iseggwasen yettḥussu s wudem am tin yeffɣen seg tnegluft tameqqrant deg yuɣal yuẓra aṭas n yimdukkal-is ttaɣen deg wayeḍ.

Deg tlata iseggwasen irgazen d tilawin xesren akk tazzla-nsen d wurfan-nsen yerna ma ɣelben deg yisenfaren-nsen imezwura, ččuṛen d uḥezzen yerna gren iman.

Tideț n tmezɣent tettbeddel i tideț n tlemmẓi. Mbla Ulemd Amenzuy tyerda n tɣerrfet tuɣal d aɣbel.

Tlemmẓi d tin yeffɣen. Tecbaylit d ayla n tlemmẓi, maca d tumert kan, ur tettɣimi.

Tlemmẓi tesɛa tmusni tudert d usbeddi d adabu yeqqel. Deg tudert ur yeffrin ara yilemẓiyen n Usbeddi yesɛan tazmert d Tmusni tudert.

Mbla ULMAD AMENZUY ilemẓiyen tuɣalen d tazzla, aseṛxu, idreɣlen, yessasaten, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen, imeɣṛuḍen.

Tilemẓi d Tafukt n unebdu i tettɣimi s teɣɣaw. Ilemẓiyen ḥemmlen ad xessen azal n tilin n tlemẓi.

Iɣerfan xulfen axxam i yilemẓiyen yerna wwin-ten ɣer imenɣi.

Imezdaɣ ẓẓayen zemren ad beddlen yerna ad beddlen ddunit ma ttwareffden s ubrid n ULMAD AMENZUY.

Deg tlemẓi ččuṛen s tikerkas i neɣwi kan ɣer uɣbel.

NEKK tesseqdac tafat n tlemmẓi i yesnernin d yessnernin adabu-ines.

NEKK tebɣa tiserwal, tazzla ɣef yal azal ula ma tɣerfet d tin ur nesɛi ara azal.

Imezdaɣ ẓẓayen ḥemmlen ad gren iman deg tfekka, ccṛab d tissas n yal assaɣ.

Ur bɣin ara ad faqen ilemẓiyen d akken yella d aklan n tissas d ayen i tbeddlen maca ur telli ara d irgazen tideț.

Ulac tissas i yettɣimin akken iṣḥa. Tuẓẓya n tissas d aɣbel i yessɣiren anagar ILEFMAN IMUSNAWEN. Ameɛna iqeddes n tutlayt taspenyulit Jorge Manrique, yenna:

"Acḥal qrib yeffeɣ tissas,
Amek uqbel ma nessalay,
Yessawaɣ,
Amek ɣer wul-nneɣ
Yal tallit yezrin tella tesɛa azal."

Aristóteles yettmeslay-d ɣef tissas yenna: "Mi d-yella lḥal n uɣawas tissas irgazen ur llin ara d asertiyen ur yugiten."

ILEFMAN IMUSNAWEN ferḥen s uɣawas tissas. Federico el Grande ur yesɛi ara uguren ad yenna s ṣṣeḥ: "TISSAS D AYEN YESƐAN AZAL AKTEṚ DEG TUDERT-A."

Aɣbel akteṛ i d-yettɣimin d win yettwaɣɣalen ɣef ddemma n tissas tameqqrant akteṛ.

Ilemẓiyen ẓẓayen kkan am tuga ur yetteɣlayen ara. NEKK ẓẓayan yemmal tissas deg yal azel.

Aẓẓayan ASERTIL yeẓẓẓaḍ Izewwagen neɣ yebɣa ad ten-yesserɣed. D aɣbel ad yesserɣed Izewwagen s uɣawas ad yessissin tissas n ddunit akk.

D ayen ur nesseɣed ara ad yekfu azal n tlemmẓi yerna deg zewwagen, irgazen n ṣṣwab ɛellben d arrac.

Aṭas n yirgazen zewwagen acku ɛeyyawn, aṭas n tilawin zewwagen ɣef ddemma n tumert yerna agal n yisenfaren ur nessɛi ara azal yella d unegmu.

Yal argaz yessen ad yeḥmel s ṣṣeḥ d wul akk i tlawt i yerra.

Ilzem fell-aɣ ad nezewweg deg tlemmẓi ma nebɣa ad nesɛu tɣerrfet tin ur nessɛi ara azal.

I yal assaɣ yella wakud deg tudert. Ad yezewweg yilemẓi d ayen yettwassen, maca ad yezewweg uɣerfan d tumert.

Ilemẓiyen ilzem ad zewwagen yerna ad snernin uxxam-nsen. Ur ilzem ad nessu adaru n lɣira yettuɣal uxxam.

Salomón yenna: "Lɣira d tin ur teqqen ara am leqber; tiẓẓalin-is d tiẓẓalin n tafat."

Tarwa n ILEFMAN IMUSNAWEN țɣiren am ayḍan. Lɣira tella d LEWḤUC.

Argaz i yellan d tin yeɣri ɣer tlawt ur yessin ara ɣer win yeṭṭfen. Aḥsen ad ur yegri ara i yal win yessen d acu n tlawt i nesɛa.

Tɛeqqa tisseɣt n tlawt teɣṛi tettɣimi d tin iɣelben akteṛ n yeɣlan n ayḍan yeffɣen.

D lekdeb ad iniɣ anda tella lɣira tella leḥmala. Lɣira ur tettlal ara seg leḥmala, leḥmala d lɣira ur zemren ara ad gden. Asurif n lɣira yella deg ugur.

NEKK yettara lɣira s azal n waṭas n assaɣen. NEKK yettagwad ad ixesseṛ win iḥemmel.

Win yebɣan ad yesselɣu NEKK ilzem fell-as ad yettwaha ad ixesseṛ win iḥemmel akteṛ.

Deg umeslaq nezmer ad nebḍen uqbel waṭas n iseggwasen n uskar, yal yiwen yeddan ɣer tɣerrfet tuɣal d argaz yeɣṛan.

Yal argaz yella d tin yefsan.

Argaz d tlawt ilzem ad ilin ttegdin s wudem iɣelben d leḥmala, ur telli ara d agur d lɣira.

Zdat N LEGAN IMEQQREN argaz ilzem ad yerr tajawt ɣef tikelt-is d tlawt ɣef tin-is. Argaz ur yezmir ara ad yerr tajawt ɣef tikelt n tlawt d tlawt ur tezemmer ara ad terr tajawt ɣef tikelt n urgaz-is. Erret tajawt yal yiwen ɣef tikelt-is yerna ɣwerret lɣira.

Aɣbel amenzuy n tlemmẓi d Izewwagen.

Tasebtit ẓẓayen yesɛan waṭas n yilellan teqqim d timɣart "acku am ugaren am wid yeqqimen ttaɣen uguren ɣur-s.

Ilezm ad issinen tilawin ad ḥerzent ilellan-nsent ma bɣant ad zewwgent.

Ilezm ad ur neɣleb LEḤMALA d TAẒLA. Ilemẓiyen iḥemmlen d tiḥeqqaniyin, ur ssnen ad qetlen gar leḥmala d tazzla.

D ayen yettwazzelen ad nessen d akken TAẒLA d ssem i yesɣelben ul.

Yal argaz yellan d tazla d yal tlawt yellan d tazla zemren ula ad ggin s imeṭṭawen n idammen d akken llan s ṣṣeḥ ḥemmlen.

Uqbel ma tserɣ tazzla tilefman, tberna n tɣimit tuɣal ɣer wadis.

Uɣelben n waṭas n Izewwagen yella ɣef ddemma n zewwgen ɣef tazzla tilefman, maca ur telli ara ɣef LEḤMALA.

Amgired ameɣqar i neggan deg tlemmẓi d Izewwagen d deg Iɣerbazen, Ikuluten d Tseddawiyin ilzem ad heggin ilemẓiyen d tiḥeqqaniyin i amgired-a yesɛan azal.

D ayen iɣelben d akken aṭas n yilemẓiyen d tiḥeqqaniyin zewwagen ɣef ddemma n uɣawas n tselwit neɣ d tmuɣliyin timetti kan.

Mi d-yella lḥal Izewwagen yettili ɣef tazzla tilefman neɣ ɣef tmuɣliyin timetti neɣ aɣawas n tselwit, agal yella d uɣelben.

Aṭas n yergazen iɣelben deg Izewwagen ɣef ddemma n ur yettgemmar ara akkar.

Tlawt i zewwgen d yilemẓi yeɣran, yeɣran, yeɣran, tuɣal d tameɣtut n win yessawalen.

Ilemẓi i zewwgen d tlawt teɣṛi, teɣṛi, teɣṛi, d ayen yebɣan ad yessen ad yekker tudert-is deg uzzal.

I d-yella lḥal leḥmala tideț gar sin n imezdaɣ, d ayen yettwazzelen ur telli ara tazzla tilefman, d ayen yesɛan azal ad nselɣu NEKK n lɣira, d ayen yettwazzelen ad neṣɣu azzal, d ayen yettwazzelen ur nessutur kra.

NEKK yesɣaren uxxamen, NEKK MI yessuɣen ur teqqen. Ma ilemẓiyen d tiḥeqqaniyin ɣran ULMAD-NNEƔ AMENZUY yerna gren iman ad selɣun NEKK, d ayen yebɣan deg wudem-nsen akken ad awin ɣer ubrid n IZEWWAĞEN IMENNƔIYEN.

Anagar ma nselɣ NEKK, ara yili tumert s ṣṣeḥ deg uxxamen. I ilemẓiyen d tiḥeqqaniyin i bɣan ad ilin d tin yellan deg Izewwagen nesɛa asefru ad ɣren ULMAD-NNEƔ AMENZUY yerna ad selɣun NEKK.

Aṭas n Yimawlan n twaculin tɣiren i yelli-tsent s tuggdi yerna ur bɣin ara ad ilin i tlatin-a deg ilellan. Aɣawas-a yella d tumert n meyya deg meyya acku tiḥeqqaniyin ilzem ad sɛunt ilellan d zewwagen.

Agal n ur nfaq d ilellan i yettḍerren, deg ubrid, ɣef ugur ɣer yeɣlan deg ufus n usɣellef i yessen ad yegzel.

Tiḥeqqaniyin ilzem ad sɛunt azref i yal win yesɛan allal-is, maca ɣef ddemma n yella ur sselɣunt ara NEKK, d ayen iḥsen ad ur nttekki ara lukan kan yesɛan allal.

Ilemẓiyen d tiḥeqqaniyin ilzem ad sɛun azref ad aggen tizwiyin deg uxxam. Ususru imennugur ur yessawaɣ ara ɣer yiwen yerna Tlemmẓi tebɣa ad tesɛu ususru.

Ayen yessawaɣen ɣer tlemmẓi d ccṛab, tasegdef, tifekka, timraradin, tizdiyin, iberḍaden, ikabaret, atg.

Tizwiyin n twaculin, tizzlatin iwulɣen, uẓawan uɣalen, imezwura ɣer yigarren, atg, ur zemren ara ad sawɣen ɣer yiwen.

Aɣezfan yesɣaren leḥmala. Aṭas n yilemẓiyen xessen assaɣ ad beddlen Izewwagen d tilawin ibawen ɣef ddemma n ugur n tselwit, ɣer yessalayen n iḍelli ɣer uɣbel n uzekka.

Agur ɣer tudert, ɣer ljuɛ, ɣer leɛqeb d yisenfaren n aɣezfan tuɣalen d asurif amenzuy n yal uɣḍel azewwağ.

Aṭas n yilemẓiyen yeɣwan ad ur beddlen izewwaǧat ar ma sɛan azal-a n yedrimen, uxxam-nsen, takerrust teɣzif d uɣuṛma n leḥram amek akken yal ayen yella d tumert.

D ayen iɣelben d akken aklass-a n yirgazen xessen assaɣen imlellen n Izewwaǧat ɣef ddemma n ugur ɣer tudert, ɣer tmettant, ɣer win ara yinin, atg.

Aklass-a n yirgazen qimen waḥḥed-sen i yal tudert-nsen neɣ zewwagen aṭas kan, mi ur yekkan ara wakud-nsen ad nferfen twacult yerna ad snernin arrac-nsen.

S ṣṣeḥ yal ayen yettḥwaj argaz ad yeḥrez tlawt-is d arrac-is yella yesɛa afgan neɣ aɣara ɛadilen, d ayen.

Aṭas n tiḥeqqaniyin qimen waḥḥed-nsent ɣef ddemma n beddlen argaz. Tilawin iḥesben, yesɛan azal, tuɣalen qimen waḥḥed-nsent neɣ xessen s ṣṣeḥ deg Izewwagen.

D ayen iḥsen ad faqen tiḥeqqaniyin d akken yal argaz yefsen ɣer tlawt yesɛan azal, iḥesben, yesɛan NEKK.

Aṭas n tilawin ẓẓayen bɣant ad ṭṭfen argaz tessentrem wudem-nsent s wudem iɣezfan, ggin tizzlin i tmettun, ggin tizzlin ɣef uɣurar, ggin tizzlin i tizzlin d tiggad, tilawin-a ur faqent ara s tmuɣli taɣezfant n urgaz.

Argaz s uẓar yeɣṛa ɣer temqelliwlin tessentrem yerna yesnerni tecbaylit yal waḥḥed yellan d uẓar d teɣzi n tmuɣli.

Argaz yebɣa ad yuẓra deg tlawt tizḍet, asɣellef, leḥmala tideț d ur ttagwad, teɣzi n uẓar.

Tiḥeqqaniyin i bɣant ad zewwgent ilzem ad faqent s tmuɣli taɣezfant n uẓar n tkerfa.
