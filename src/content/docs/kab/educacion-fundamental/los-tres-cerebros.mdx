Tasuqilt taɣellavt n tallit tamaynut tesskanay-d belli takrura tagmant n ULLUγ ANESLAK yettusemman d amdan s uɣalat, tettili-d deg umgarad n tlata yimukan neɣ tlata yefker.

Afekru amezwaru yezga deg ukessar n uqerru. Afekru wis sin yeɛnan s usrid tgecrirt n umagur s tmerrizt-is talemmast d yiferrawen-is akk n idimazen. Afekru wis tlata ur yezgi deg umkan yettwafhamen neɣ d addal yemgaraden. S tidet afekru wis tlata yettwabna seg yireṭṭalen n idimazen imsemlalen d akken umum seg akk iɣerbazen n idimazen imaẓẓalen n takrura tamentut.

Afekru amezwaru d ammas n umyaru. Afekru wis sin d ammas n uḥerrek, yettusemman s tuget ammas n umyaru. Afekru wis tlata d ammas n yisragen.

Yettubeggen s lekmal deg tɣuri belli yal azɣel n ufecku amgaru yettarra-d aserseg uzɣen n tezmert taɣellavt. Ihi d ayen ilan s tikelt akken ad neqqar mbla tugdi belli ixxamen n iḥeqqanen d imeqber n lmuten iɣellaven.

Uraren imseɣzanen d imsewfanen d ifayden i ufecku umyaru, maca azɣel n wurar yettarra-d aserseg uzɣen n tezmert n uḥerrek dɣa agmuḍ yettili-d d ayen yemxalbafen. Ur d ayen ibehlen akken ad neqqar belli llan lmuten n ufecku umyaru. Lmuten-agi ttwassen am imuḍan n Hemiplejia, Paraplejia, Parálisis progresiva, atg.

Tiremt tamezdayt, tamsegda, asufeɣ, uẓawan yeflalen, d ayen ilaḥḥun akken ad neṣṣawef ammas n yisragen, maca azɣel n ufecku-agi yettarra-d aserseg ur ilaḥḥun d tzerzut n tezmert n yisragen. Azɣelen n ufecku n yisragen imawlan n "tallit tamaynut", ifennanen n Rock, Ifennanen-usuḍen yessuksenen n tẓuri tamirant, imḍebberen iḥeqqanen n tussna, atg., atg.

Ɣas ma yettban d ayen ur nnumen, lmut s tidet tettili-d s tlata yixfi deg yal amdan. Yettubeggen yakan ɣer taggara belli yal uḍan ila d llsas-is deg yiwen seg tlata yefker.

Tala tameqqrant tesers s tẓeḍt deg yal yiwen seg tlata yefker n ulluɣ aneslak, d aqeṛṛuy yettwafhamen n TURMININ IBUDDEN. Akrez aqeṛṛuy-agi yettarra-d s tidet ad tiddi tudert, azɣel aqeṛṛuy-agi yettarra-d lmut.

Idles aqdim i d-yewḍen ɣur-nneɣ seg yiḍ ameqqran n leqrun, qaren-d belli aḥric n tudert tamentut deg Uruppa Taqburt MU, yezgan deg Ugaraw Amalu, yettili-d gar Tneṭṭi d Tẓat n Leqrun.

S tlalit n leqrun deg akk leɛmaṛ, asseqdec uslig n tlata yefker yessaẓ yettiddi tudert cwiṭ cwiṭ.

Deg tmurt yessuksenen n KEM... dinna deg Maṣer Taqburt n Ifarawen aḥric n tudert tamentut yewweḍ kan ɣer meyya tẓat n yiseggasen.

Tura deg tallit-agi tamirant n lbenzin d cciluyd, deg tallit-agi n umawal d iɛeryanen n Rock, aḥric n tudert tamentut ɣer kra n tuddsa n Surens, d kan ḥemcin n yiseggasen.

Massawen Marksisten-Leninisten n UNIÓN SOVIÉTICA, imceɛɛen d imsakaren am zik, dinna deg yettqeddimen qaren-d belli ufɣen-d d amserweɛ yettwafhamen akken ad tiddi tudert maca agellid Kruschev ur mazal ara ila ɣer ḥemcin n yiseggasen dɣa ila ad yessuter tazzert i uḍar akken ad yessali wayeḍ.

Deg ammas n ASIA tella tamaɛit tadinselt i yettwabnan seg imɣaren ur mazal ara ttumun tulmut-nsen. Aḥric n tudert n imɣaren-agi yettili-d gar ṛebɛa tmeṭṭatin d ḥemca tmeṭṭatin n yiseggasen.

Akk Lsir n tudert tiddi n Irhebaniyen-agi i Asiwa yezga deg useqdec ilulen n tlata yefker.

Tuɣmast imsewfanet d tamseɣzanet n tlata yefker tettarra-d akrez n TURMININ IBUDDEN dɣa am dɣa talalit, tiddi n tudert.

Tella Tala tagreɣlant yettwassen am "ASEMDE N TURMININ N TƔELWIN TUGET". Irhebanen n Umiḥrab-agi ssinen akken ad ssexdmen Tala-agi s useqdec n tlata yefker.

Tasnilɣa ur yellan d tallit tettarra-d inelmaden d inelmadtiyen ɣer azɣel n ufecku amgaru i d-yerra yakan Psiquiatría.

Aṣṣawef ilulen n tlata yefker d TASEQQAFTA TAZWERANT. Deg yeqdiren iqburen n uɣrib n Babylone, Grèce, Inde, Perse, Maṣer, atg., inelmaden d inelmadtiyen ttarran-d tasuqilt tqerḍaɛt usrid, i tlata yefker-nsen s uqeṛṛuy, taẓeṭṭit, uẓawan, atg., yettwabnanen s tewzel.

Tidaratin n tallit taqburt llant d tamaɛit seg uɣerb. Adɣar n uɣalad, tazdayt, taddart tameqqrant, yettwabnanen s tewzel s umeskan yettwafhamen, uẓawan, aselmed uɣiman, atg. Uɣden akken ad rren tasuqilt i tlata yefker n yal amdan.

Dɣa inelmaden ur yellan d izɣalen n ufecku amgaru dɣa ssinen akken ad ssexdmen s tewzel d s turmin tlata yefker-nsen.

Tiẓeṭṭiyin n uɣrib n Eleusis deg Grèce, adɣar n uɣalad deg Babylone, tayri deg Grèce llan zik s tuget yettwasqedcen akken ad srewren tamussni i yinelmaden d inelmadtiyen.

Tura deg tallit-agi yettwaẓenqen n Rock, inelmaden d inelmadtiyen yettwaxlaɛen d yettwajerḍen ttidiren deg ubrid ameqran n azɣel amgerrad.

Tura ur llin ara tidusiyin yessufɣen tidusiyin yellan s tidet i uṣṣawef amseɣzan n tlata yefker.

Iselmaden d tiselmadtiyin n uɣerb, iɣerbazen d taseddawiyin, kan ttidiren ɣer tmust-usuḍen n inelmaden yettwabren yettraǧun s tḥerci tizi n tuffɣa seg uɣerbaz.

D ayen ilaḥḥun, d ayen yelzem akken ad nessen akken ad nesdukkel tiɣri, aḥerrek d taglaɛt s uɣerbal akken ad nawi tasuqilt tqerḍaɛt ɣer tlata yefker n inelmaden.

Agmuḍ d ayen ibehlen akken ad nerreḥ i yiwen ufecku. Afekru amezwaru ur yelli d yiwen kan n tmust. Agmuḍ d ayen imekkawen akken ad neɣzel ufecku amgaru n inelmaden d inelmadtiyen.

TASEQQAFTA TAZWERANT ila ad tessiweḍ inelmaden deg ubrid n tunuft tamseɣzanet.

Tasuqilt taɣellavt tesselmed s usrid belli tlata yefker d tlata n wanawen n imsedduklalen imanɣalen yemgaraden s lekmal. Tlata wanawen-agi n imsedduklalen srewren aṭas n wanawen n uɣellel seg tudert.

Aya yettarra-aɣ-d s tidet tlata n tɣerma yemgaraden ur ttudumen ara kra deg tegma-nsent neɣ deg tmugar-nsent.

Tasuqilt taɣellavt n tallit tamaynut tesselmed belli deg yal amdan llan tlata wanawen imgaraden n imsedduklalen. S tamaɛit seg tneflit tasufanen nerreḥ aɣalad, s tamaɛit tayeḍ nerreḥ kra yemgaraden yefhamen dɣa ɣer tamaɛit tis tlata nxeddem kra yemxalafen s lekmal.

Deg tizi n twaɣit tameqqrant, ahat aserzeg n amdan ilulen neɣ kra n tmekkest tameqqrant tatut, tɣerma tasraganett tewweḍ ɣer talqemt maca tɣerma taɣellavt tseqsa aɣerbal n akk twaɣit-agi dɣa tɣerma n uḥerrek kan terraḥ ad trewel seg uɣerbal.

Tlata tɣerma-agi yemgaraden yemxalafen d waṭas n tikelt ar taɣlent ttilin yettwaṣṣafen s tewzel d yettwassellmen s tɣura d tusnakt yettwafhamen deg akk iɣerbazen, iɣerbazen d taseddawiyin.

Seg tɣuri tasufanent agmuḍ d ayen ibehlen akken ad neṣṣawef kan tɣerma taɣellavt. Amdan ila tlata n tɣerma yellaḥḥun s tḥerci TASEQQAFTA TAZWERANT.
