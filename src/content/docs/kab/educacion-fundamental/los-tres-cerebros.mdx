Tasiknfrut taqbaylit tatrart terna-d belli amesniw amagnu n Uɣersiw Alemmas yesɛan s tkerri tsemkelt neɣ s tkerri iqefflalen.

Aqeffal amezwaru yezga deg uɣrab n uqerruy. Aqeffal wis sin yezga deg uɣesmar d ameqran akked tiddi-ines tameqqrant d yiferdisen-is n wudem. Aqeffal wis tlata ur yezgi deg umkan yettwafen neɣ deg uɣaras yettwasen. Ț-țideț belli aqeffal wis tlata yebḍa ɣef yisuyas n uqerruy d uẓar d yal leṣnaf n yimukan n uqerruy n ufgan.

Aqeffal amezwaru d ammas n umyir. Aqeffal wis sin d ammas n tennɣelt, yettwassnen daɣen s ammas amesnerni. Aqeffal wis tlata d ammas n yirẓan.

Ț-țideț, s tilawt, belli yal axdam n uqeffal n umyir yesnerni tameqqrant n tezmert n uɣerbaz. Ț-țidet, tura, ad d-nini mbla ugur belli idɣaɣen d timqebburt ɣef wid yemmuten n uɣerbaz.

Urawen yezdin d wid iqwanen d wid iwulmen i uqeffal amesnerni, maca axdam-nsen yessedhu tezmert tameqqrant n tennɣelt dɣa agaz-ines d asdurus. Ur yettḍur ara ma ad d-nini belli llan wid yemmuten n uqeffal amesnerni. Wid yemmuten-nni ttwasnen s wussan n Uɣelluy, Uɣelluy azedgan, Uɣelluy ad yettnernin, atg.

Tazmert n udmawen, tamaziɣt, tukksa, taẓuri tameqqrant, lant lḥaja i wakken ad tnernin ammas n yirẓan, maca axdam-is yessedhu axsim n lɛib d usnerni n tezmert n yirẓan. Xedmen uqeffal n yirẓan wid iqwanen n "tidfi n ussan", wid yettḥemmiḍen i Rruk, Iɣerfan-Uẓuren n taẓuri tatrart, Wid yerɣan n taḥuski, atg, atg.

Ɣas ma yella yettban d ayen ur nettwamen ara, tamettant tella tettnerni s tlata yebḍan deg yal yiwen. Tettwaḍeggeṛ teɣmert n uɣaras belli yal aɣelluy yesɛa lɣerḍ-is deg yal yiwen seg tlata iqefflalen.

Uɣaras ameqran yella yessers s tmusni deg yal yiwen seg tlata iqefflalen n uɣersiw alemmas, tamɣart n TILELLI TIGMATIN. Ad nḥasbeɣ tamɣart-nni yesɛa lmeɛna ad nesnerni tudert, ad nesned tamɣart-nni yesɛa lmeɛna tamettant.

Aṭas n leɛwayed i d-yusan ɣur-nneɣ seg yiḍ yugaren n tseqqelt, arɣalen belli aseɣẓan n tudert n ufgan deg Uɣerbaz Aqdim MU, yezga-d deg ugaraw n Tafsut, yebḍa gar Tizwar d Ṭṭam Tseqqelt.

Akken id ttilin tseqqeltin deg yal leɛmer axdam s tuccḍa n tlata iqefflalen yeqẓer tudert kra kra.

Deg tmurt n KEM yezga-d... deg Mezda-d taqburt n Firaɛun aseɣẓan n tudert n ufgan yewweḍ ɣer miya' d ṛebɛin iseggasen.

Tura deg ussan-a ijdiden n lgaz n taselulit, deg tallit-a n tidfi d iɣerfan n Rruk, aseɣẓan n tudert n ufgan, ɣef leḥsab n kra n tmesdurar n Tseddurt, d ḥaca ɛecrin iseggasen.

Igellilen Marxistes-Leninistes n UNIÓN SOVIÉTICA, d imesluben d ikeddaben akken ma llan, dinna qaren belli snernan tiyitaḍ ijdiden i wakken ad snernin tudert maca amɣar Kruschev mazal ur yesɛa ɛecrin iseggasen d yelzem ad yessuter afus i wakken ad yesnerni afus nnḍen.

Deg ammas n ASIA tella tɣiwant taɣiwanit tettwabeddel s imɣaren ur ttumun ara ilemẓi-nsen. Aseɣẓan n tudert n imɣaren-nni yebḍa gar ṛebɛa tmeyyar d ḥamset tmeyyar iseggasen.

Aḍris ameqran n tudert tuzzilt n Imunken Asia-nni yezga-d deg uɣaras n tmusni n tlata iqefflalen.

Axeddim yezdin d win iwulmen n tlata iqefflalen yesɛa lmeɛna n usnerni n TILELLI TIGMATIN dɣa akken illan, asentel n tudert.

Tella Uɣaras n Uɣilas tettwasen s "TAWEN N UDMEN N AṬAS N TIƔEZZIWIN". Imunken n Uɣarbaz-nni ssnen ad xedmen Uɣaras-nni s uɣaras n tlata iqefflalen.

Taswir tayettant tewwi arraw d arrawn ɣer axdam n uqeffal n umyir agaz-ines tella tettwasen ɣur Psikiyatriya.

Tenneɣri tamusni n tlata iqefflalen d TENNULINT TISNILANT. Deg uɣerbazat iqburen n tmesdurar n Babilonia, Grecia, India, Persia, Egipto, atg, arraw d arrawn ttatfen tidet tella deg uɣaras azedgan, i tlata iqefflalen-nsen s leɛwayed, adɣaɣ, taẓuri, atg, yettwazedden s tmusni.

Teqcerḍaten n tallitat iqburen zgan-d deg uɣerbaz. Aɣebbar, taḍsa, asdurus, zdin akked tkerri taẓewwalt, taẓuri, tenneɣri taɣerfant, atg. Xeddmen akken ad snernin tlata iqefflalen n yal yiwen.

Din din arraw ur xeddmen ara uqeffal n umyir d ssnen ad xedmen s tmusni d teɣdemt tlata iqefflalen-nsen.

Adɣaɣen n tmesdurar n Eleusis deg Grecia, taqcerḍa deg Babilonia, taẓuri deg Grecia zgan-d xeddmen akken ad awin tameɣra i arraw d arrawn.

Tura deg tallitat tegrudwin n Rruk, arraw d arrawn yettwaḥerqen d wid yettwabḍen ɣef uɣaras ugerrum n axdam alemmas.

Tura ur llin ara uɣarbazat id yessnernin i wakken ad snernin s teɣdemt tlata iqefflalen.

Iɣerfan d tiɣerfan n uɣerbazat, lecyax d tedwilat, d ḥaca yettwalesen i tɣerma ur nettwamen n arraw imḍullen i yettrajun s teɣdemt tikkelt n tuffɣa seg uɣerbaz.

D taɣawsa taɣdemt, d taɣawsa id yelzem ad nessen ad nzeddeɣ aɣerbaz, tennɣelt d leḥnana akken ad awin tidet tella deg tlata iqefflalen n arraw.

Yettban d ayen yesdurusen ad snernin aqeffal yellan waḥdu-ines. Aqeffal amezwaru ur yelli waḥdu-ines deg tmusni. Yettban d aɣilif ad xedmen uqeffal n umyir n arraw d arrawn.

TENNULINT TISNILANT ad tewwi arraw ɣef uɣaras n tenneɣri talɣawant.

Tasiknfrut tayettant terna-d s teɣdemt belli tlata iqefflalen yesɛan tlata leṣnaf n tzedda yellan waḥdu-sen d wid yettwabḍen. Leṣnaf-a n tzedda ad d-awin leṣnaf n tenneɣriwin n tallit.

Ayagi ad d-yefk tlata tiswaɛtin yettwabḍen ur yesɛan acemma yezdin deg tidett-nsen neɣ deg usnerni-nsen.

Tasiknfrut taqbaylit tatrart tessawaɛ belli deg yal yiwen llan tlata iswiriyen tiɣawasin yettwabḍen. Akked udmawen seg teswiɣi tiɣawasin nettwamen taɣawsa, akked udmawen nnḍen nettwamen ayen yettwabḍen akken iwulmen dɣa s ɣur tameɣra ad nxeddem ayen yettwabḍen.

Deg tikkelt n uɣebbar ameqran, talɣa n ussedhu n waɣrib neɣ yal aɣebbar nnḍen yellan deg uɣerbaz, tiswaɛi tiɣawasin tewweḍ ɣer tumast mi akken tiswaɛi n uɣerbaz teqqaṛ ɣef ayen yettḍurun dɣa tiswaɛi n tennɣelt d ḥaca tettwam akken ad tgerrez seg uɣaras-nni.

Tlata tiswaɛtin-a yellan yettwabḍen d aṭas n tikkeltin armi d wid ireqqen ad tnernin s tmusni d tidet s uɣarasen d tussniwin tiẓewwalin deg yal uɣerbaz, lecyax d tedwilat.

Seg udmawen n tɣiwsa, yettban d ayen yesdurusen ad tnernin ḥaca tiswaɛi n uɣerbaz. Argaz yesɛa tlata tiswaɛtin yellan yelzem d taɣdemt TENNULINT TISNILANT.
