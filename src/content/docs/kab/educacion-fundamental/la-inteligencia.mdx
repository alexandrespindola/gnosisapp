Nesmers-d-nettwali dakken aṭas n yiselmaden d tiselmadin n Umezruy Alemmas deg umalu n umaḍal tteẓẓuḍun s BUDDHA, Confucio, Mahoma, Hermes, Quetzacoal, Musa, Krishna, atg.

Berra n kra n ccekk, nufa daɣen s tuget, s tkerkas, s tuffɣa, s tmeẓẓuɣt i d-tettunefken seg yiselmaden d tiselmadin mgal ddinaat tiqdimin, mgal yillu, mgal tasnariyya, atg. Akken ma yella d ayen-nni d laṣel n tuɣmert.

Deg uɣerbaz, deg ulis d tesdawiyin, ilaq ad-nheḍren ɣef umawal adini, s leḥḥrema nniḍen, s unamek εlayen n tqaderɛet, s tmusni n tidett ara d-yesnulfuyen.

Talɣiwin tidiniyin ḥerzent azal adawi d ttidurint s usuddu γef leḥwayeǧ isikilujiyen d tidmesrayin n yal agdud, n yal taγemsa.

Ddinat akk sɛant usunen yellan d imecbaren, azal yellan d imecbaren d wid ttidurent ala ad-mgaradent deg talɣa.

Ur yelli d ameɣlal mi ara yesḍsu uqristyan s ddin n Buddha neɣ s ddin Uεebri neɣ Uhendusi acku ddinaat akk ttilint γef isunen yellan d imecbaren.

Tkerkas n waṭas n yiεeqliyen mgal ddinaat d yesbedden-nsen tettwabnan γef ssem AMARXISTI ay yessukksen deg akud-a i wallen yellan d ḍεifen akk.

Iselmaden d tiselmadin n uɣerbaz, n ulis d tesdawiyin ilaq ad-fken tiγri i yinelmaden d tnelmadin-nsen deg ubrid n leḥḥrema n tidett i watmaten-nneɣ.

Yella-d deg allen merra d imeggir d ilem deg udem n ttheoryt n kra n telmest, ara yesḍsun s temqerin tidiniyin, s temheḍritin, s uγerbazen neɣ s tmezdayin turufanin.

Mi ara ḥerren tiɣriwin n tεerbit, inelmaden ilaq ad-afen wid i d-yusan seg ddinaat akk, seg uɣerbazen, seg temheḍritin, ur yelli d ameɣlal win ur nessin ad-yeḥrez tserwit ilaqen deg temqert.

Mi ara ḥerren tiɣriwin deffir mraw neɣ mraw-d-semmus iseggasen n tεarbi, ilmeẓyen d telmeẓyin ttufant-d d imugren d iden am wiyyaḍ n yemdanen, ččuren d tsemɣer d tqella n tmusni am wass amezwaru mi kecmen ɣer uɣerbaz.

D asuggas inelmaden, gar kra n γerwiɣen, ad-snernin ammas anamur acku kullec mačči d aεeqli. Ilaq ad-nelmed ad-nḥus ttawilat tunṣibin n tudert, tezmert n ttejra waḥdes, uḍebsi n yefri deg tẓegwa, tasenfuniyya n uẓawan d yiniten n tuɣẓi meqqren.

Ilaq daɣen ad-nḥus d ad-neqḍεε s tmusni merra γerwiɣen imeggaraden n tudert, am usuddu aεeqqi d ameqqran n tallit-a deg-netteddren, iberdan ččuren d yemma s tuɣmas ay d-yusan d yefruc meẓẓiyen yettwaḥwaǧen kra n uɣrum, ibudaε yellan d yebεaden ideg ddren imelyunen n twaculin imeḥwaǧen, iberdan yellan d imeqqaren ideg ddren imelyunen n tkerrusin yessegdawen s wazalen i ten yeḍuren.

Anelmad ay d-yusan seg tiɣriwin ilaq ad-yeεreḍ ur yelli d iman-is kan d tγεewwaren-is i yellan yakan, maca d iman n yemdanen akk d tuget n tγεewwaren n tmetti tanemdayt.

Ayen yugaren akken ma yella d akken anelmad ay d-yettalin tiɣriwin, ula ma yesεa tayri taεeqlit, ur yesεi tmusni, taεeyyari-ines ttedduɣ, yettwahyyeε s tmusni i uεraq γer tudert.

Tewweḍ-d tesεet ad-naf d ad-neqḍεε d acu ayen i wumi qqaren TMUSNI. Amawal, talɣut, ttilin d ilalen i useɣti s sser TMUSNI.

Wer tmusni ur yelli d aseɣti ameɣlal neɣ tafat n tidett d iquran deg tudert ad-naf imdanen yellan d imeɣlalen n tidett.

Ayen i yellan d aεlayan deg tudert mačči d kan ad-nessen awal TMUSNI, maca ad-nesεu deg iman-nneɣ anamek-is ameɣlal.

Aṭas ay yeεemmeden s tmusni, ulac aḍensu ara yeεemmed s tmusni d Carlos Marx yettwalit iman-is d ameɣlal tuget, yura farsa-ines tamaterialt ay yeɣlin γef umaḍal aḍqan n wazalen tidurent, asedrem n imelyunen n yiqessisen n ddinaat yemgarraden, aɣerrabu n tsufgin, Budistas, Cristianas, atg., tmeḥreq n temqerin tuget, aεraq n imelyunen d yimelyunen n yemdanen, atg. atg. atg.

Kra yellan yeεreḍ s tmusni, ayen iεerrben ad-yili s tidett.

Mačči s tayri n tirawin tunṣibin, s tussna tuget, s tγarɣert tuget, s γerwiɣen tuget i useɣṛeb yimdanen, s idrimen tuget i yuɣen iḥekkamen d yilisyen; atg. amek ara ttaweḍen γer ayen i wumi qqaren TMUSNI.

Mačči s MAS-a, amek ara ttaweḍen ad-sεun TMUSNI. Tḍulun deg udem win i yefranen dakken tmusni tezemmar ad-ttwaɣden s tira n MAS.

D asuggas ad-neqḍεε s lqayda deg akken ma yella d tergiwin n walli s-ddaw d war tqellbi, ayen ay d ttreḍḍun tira n MAS, acku deg tɣerɣert yettufra lεeziz EGO, YO, MI MISMO, ay yebɣan d yettraǧu دايمن MAS d MAS i usemɣer d useɣzez.

Mefistófeles-a ay neḍfer deg-nneɣ, aSSATAN-a, AYI, yeqqar: AYI sɛiɣ MAS idrimen, tazmert tuget, tmusni tuget n winna, aẓayer tuget, taɣawsa tuget, atg. atg. atg.

Win ay yebɣan s tidett ad-yeqḍεε d acu ay d TMUSNI, ilaq ad-ilmed ad-ittḥus, ilaq ad-yidir d ad-yeεreḍ s temlilit taɣerfant.

Akk ay leɣɣun medden gar tmeqqbert yeqqefen n yettwanfu taγurart, talɣut taεeqlit, tγarɣar n tudert, ttettwaseḍren daɣen deg termin n MAS d MAS. Deg wamek ur ttaweḍen ara ad-ssnen anamek ameɣlal n kullec ay leɣɣun.

Aṭas ay qqaren kitab deffir-s ttaɣen-t deg taγurart yesεan tafat s tayri i lemɣi n talɣut tuget, maca mi ara tent-id-ssenṭḍen i yettwanefken s tira deg kitab ay qran, ttwaẓuren ad-ssnen anamek ameɣlal n tɣuri, maca AYI yebɣa talɣut tuget, kitabat tuget ula mi ur yesεi adir n tɣuri n yiwen seg-sen.

Tmusni ur tettwaggad s talɣut tunṣibt tuget, wala s tγarɣart tuget, wala s idrimen tuget, wala s aẓayer tuget, tmusni tezemmar ad-teɣrem deg-nneɣ mi ara neqḍεε merra tira n AYI, mi ara neqḍεε s lqayda merra imesegdiwen isikilujiyen n MAS.

D asuggas ad-neqḍεε dakken wallen d ammas ameqqran n MAS. S tidett MAS-a d AYI isikilujiyen-nneɣ i d-yessutren d wallen d aẓar-ines ameqqran.

Win ay yebɣan ad-yili d ameɣlal n tidett, ilaq ad-yeεreḍ ad-yemmet mačči kan deg uswir aεeqli yettwaɣren, maca daɣen deg akken ma yella d teren n walli s-ddaw d war tqellbi n wallen.

Mi ara yemmet AYI, mi ara yemɣer Y0 s wudem uslig acu kan ay yeqqimen deg-nneɣ d SER untiɣ, SER n tidett, tmusni tlegit ḥwaǧen s lqayda d dεessas.

Medden ttamnen dakken wallen d aseɣti, tḍulun. AYI mačči d amseɣti d wallen d aẓar ameqqran n AYI.

Tmusni d aseɣti acku nettat n SER, d taγewsa n SER. Ur ilaq ara ad-nseḥlufen wallen s TMUSNI.

Tḍulun deg PLANU د wudem ameqqran win i yefranen dakken TMUSNI d γerwiγ i tezemmar ad-ttwaɣden am lehrir n tsebḥit NEǦ γerwiɣ tezemmar ad-ttwaɣden amek yuɣen titulat n teɣdemt neɣ yesεan taktabt meqqren.

Ilaq ad-neqḍεε s lqayda merra tira n wallen, merra tγerran, MAS isikilujiyen-a ay leɣɣun, atg. Akken kan ara d-teɣrem deg-nneɣ deg wudem alami d uslig tmes yeqqaren n TMUSNI.

Am Mefistófeles ay neḍfer deg-nneɣ yettwasedrir, tmes n tmusni tisnulfayt tettwaḍher uḍḍis uḍḍis deg-nneɣ, armi ad-tewteγ s uḥeqqani.

SER n tidett-nneɣ d TAḤERFIT d seg TAḤERFIT-a ay d-tlul TMUSNI taɣerfant d tlegit ur yelli n tallit.
