Ila d umurti, ila d ayen ilaqen ad nesnerni tamuɣli taɣbalut acku nettat ay tessawaḍen i umdan tilelli taḥeqqanit deg tudert. Bla tamuɣli, d amerkanti ad taweḍḍeḍ ɣer tezmert taḥeqqanit n tɣuri tameqqrant.

Iselmaden d tiselmadin n yiɣerbazen, n tmezduɣa d tesdawiyin ilaq ad wwten iberdan i yinelmaden d tnelmadin nsen s ubrid n tamuɣli taɣbalut tamsudut.

Deg yixef-nneɣ yezrin, nɣer akken iwata ɣef usenqes n ususru ma yella nebɣa ad nkemmel akk idgan n ddɣel, ama d wid n ddin, d wid n leḥḥuba, atg. ilaq ad nessefru ayen i d ususru, acku kan ma nessefru akken iwata d akken irẓan usenqes n ususru, ad naweḍ ɣer ddɣel n yal anaw.

Ddɣel yessexrab tizwitinin, ddɣel yessexrab tuddukel, ddɣel yesseɣlay imenɣi n ddin, tirniwin timaɣlalit, timenɣiwtin d leɛtab n yal anaw.

Ususru s akk idgan-is d-yettɣumm deg yiglamen yemmugaren. Yella ususru deg win yuɣen awal ɣef tilin n yiɣersiwen ifazen. Mahatmas neɣ Gurús, yebɣa daɣen ad yuɣal d aqeddac. Yella ususru deg win iḥettmen ad iẓedɣ i wiyad n filantropin. Yella ususru deg yal yiwen yettmen iɣersiwen acku yuɣ awal, acku deg tɣezrant-is llan yidaten ɣef tilin n yimdanen yeqqedsen yellan d ifazen.

Tebɣest ad yuɣal d aqeddac, tebɣest ad yuɣal d amejrud, tebɣest ad yuɣal d ameqqran yesɛa ususru d aslal.

Iqeddacen s iɣersiwen-nsen ẓerren aṭas n leqher. Yettasen-d ɣer umekti-nneɣ wamek yella yiwen wergaz iḥseb iman-is d aqeddac aṭas.

Deg temlilit, yiwen umedyaz illuẓen d umecṭuḥ yekkat tawwurt-is akken ad yefk deg yifassen-is iseɣlen ifazen s timmad-is i wqeddac n tullist-nneɣ. Amedyaz yettraju kan yiwet n teẓɣelt akken ad yaɣ imensa i tfekka-s yellan d ayen izedɣen d ayen iɣewlen.

Amedyaz yettxemmim kullec siwa taɛkkumt. Imeqqer-it leɛǧeb mi aqeddac s allen yefkan d leɛtab d ukebbac iqqucen yessekker tawwurt yeqqar i umedyaz umḥeqqer: "ffeɣ syagi ameẓẓuɣ, ɣezzif, ɣezzif... ur sseḥmilenɣiɣ i kra n wayagi, ttqeẓziɣ aɣurru... ur sseḥmilenɣiɣ i tiɛezriwin n ddunit, tudert-agi d tilemt... ttekkiɣ ubrid n ddullu d ṣṣut". Amedyaz umḥeqqer yella yettraju kan taẓɣelt sdat-is i yuɣ ɛekkumt n uqeddac, awal yettjerḥen, iḥebbis, d s wul-is yeqqerḥen d tili-s tuɣal d iqermuden yeffeɣ iiberdan n temdint s tizi... s tizi... s tizi.

Tiremt tajdiṭ ilaq ad tenker ɣef aslal n tamuɣli taḥeqqanit acku nettat d taɣbalut s lekmal.

Amekti d umekti mačči d taɣbalut. Amekti d aẓekka n uzekka. Amekti d umekti d lmut.

Tamuɣli taḥeqqanit d aggag aɣelsan n tilelli s lekmal.

Imektayen n umekti ur zemren ad aɣ-d-awi tilelli taḥeqqanit acku n uzekka yerna d ayen yemmuten.

Tamuɣli mačči d ayen n uzekka neɣ n uɛteddi. Tamuɣli tettili i akud i nella nettidir da d tura. Amekti yettara-d daymen tamusni n uɛteddi.

Ilaq ad neɣer tussna, taflsafat, asaru d ddin, maca ur ilaq ad nttkel ɣef tebɣest n umekti acku nettat mačči d taɣbalut.

D ayen ur nsensi ara ad neffeḍ iɣbula deg uẓekka n umekti. D ayen d aɛqal ad neffeḍ deg uẓekka n uzekka iɣbula i ilaq ad nessefru.

Nukni ur nezmir ad nesseɣlin ɣef leqraya, ɣef tmusni, ɣef tussna, maca yettili d amerkanti ad nffeḍ tiɣersiwin ttidirent n tmusni deg uẓekka yellan d ayen yeqqerḥen n umekti.

Yettili d ayen yelzem ad neɣer, yettili d ayen yelzem ad neɣer, yettili d ayen yelzem ad neɣer, maca ilaq ad nemmedet akken iwata akken ad nessefru deg akk iswiren n tɣezrant.

Argaz yellan d awḥid s tidet d ayen yellan d amasfer s tmessi d yesɛa tɣezrant tatat.

Ayen yellan d azaɣur deg tudert mačči d ayen i neɛzel deg uẓekka n umekti, maca d ayen i nessefra mačči kan deg uswir aɣelsan maca daɣen deg yigran iḍebsanen iḍebsanen n tɣezrant.

Tussna, tussna, ilaq ad tuɣal d tamuɣli deg tizi. Mi tussna, mi leqraya tuɣal d tamuɣli taɣbalut taḥeqqanit nezmer ad nessefru imiren akk tɣawsiwin deg tizi acku tamuɣli tettili d tamaḍlant, d taɣbalut.

Deg wergaz awḥid ulac idgan deg tɣezrant acku akk idgan n tɣezrant ttalen-d ɣer umekti. Nekki yellan d amezdaɣ i nekki d amekti yuɣalen.

Tiɣriwin n tudert ilaq ad tuɣalent d tamuɣli taḥeqqanit.

Mi tiɣriwin ur tuɣalent ara d tamuɣli, mi tiɣriwin ttaɣent deg umekti sseɣrayen aqermud n uẓekka ɣef wacu yessenɛat iɣil aḥeqqani d Luciferica n leḥsab.

Ilaq ad neẓer belli leḥsab aɣersiw yellan d amellal s lekmal si yal taɣiwant d kan d asaru n umekti, taɛeẓẓuga tazekka yessenɛaten ɣef tlawḥt talɣant.

Argaz awḥid yesɛa tɣezrant d ilelli n tiɣriwin acku ayagi yuɣal d tussna, yuɣal d tamuɣli taɣbalut.

Lmut d tudert ttilint d iqqnen s tidet. Kan mi yemmut uglim ad tlul tuga, kan mi yemmut iɣri ad tlul tamuɣli. Ayagi d usenqes n tɣara taḥeqqanit.

Argaz yellan d imsedda yesɛa amekti yeččuṛen d tiɣriwin.

Ayagi yessenqed ddellu-s n tamuɣli taɣbalut acku mi tiɣriwin tettusfar s lekmal deg akk iswiren n tɣezrant ttkemmelent ad ilint am tiɣriwin tlulin am tamuɣli.

Ilaq uqbel ad nesseɣri, maca ur ilaq ad nebqa deg uɣerbal n tiɣri acku imiren tɣezrant tettemɣur tettili d ayen iɛetben. Ilaq ad idir tudert s taɣẓint yerna ad nesseɣyar akk tiɣriwin d tamuɣli taɣbalut taḥeqqanit.

Wid yettḥussun sɣellet belli akken ad nili d imasferen d awḥid ad nezdi abrid, ad nuɣal d imnuddiyen, ad nedder deg tixxamin isebɣan yerna ad nessenfali tacḍaḍt deg umkan n usaru aqeẓẓay, d ayen yessḍeggren s lekmal.

Aṭas n yimḍebbren, aṭas n yimezdaɣen ilelliyen, aṭas n yimnuddiyen, sɛan tɣezranin yemɣurten d ayen iɛetben.

D amerkanti ad neɣer si ddunit yerna ad nedder am yimḍebbren ma amekti yeččuṛ d tiɣriwin yettarun iɣerrimen n teɣzi ilellit.

D amerkanti ad nedder am yimezdaɣen nɛettem ad nawi tudert n yiqeddacen ma amekti yeččuṛen d tira ur tettwassefra akken iwata, ur tettuɣal d tussna deg yimukan, ikufi d temnadin iḍebsanen n tɣezrant.

Wid yettɣayyaren tiɣbula tiɣelsanin d tamuɣli taɣbalut taḥeqqanit, wid yettɣayyaren tiɣriwin n tudert d tamuɣli taḥeqqanit n ugur ur sɛan kra deg umekti, ddren deg tizi ɣer tizi yeččuṛen d tɣara taḥeqqanit, uɣalen d imasferen d awḥid ɣas ma ddren deg temziwin teqreb d deg dɣel n tudert n temdint.

Iẓiẓwan imeẓyanen uqbel n umɣar sɛan tamuɣli d tẓuri taḥeqqanit acku kan tessenqed s tenmeggart-nsen TESSENSIT tidirent n tudert deg ddellu s lekmal n Nekki Aɣelsan.

Ilaq ad nessewleɣ tagerzmat yexsen deg wul-nneɣ d deg tɣezrant-nneɣ. Ilaq ad nessewleɣ tamuɣli ma yella nebɣa ad nili d isɛaden.

Tiɣriwin d leqraya yettɣayyaren d tamuɣli n ugur ur tessenqed ara deg uẓekka n umekti yerna imiren, nttili d imasferen, d awḥid, d ilelliyen, d isɛaden.

Ameddit n ugur ɣef tiɣriwin d tussniwin yuɣalen, asusru aɣbalan, taflsafat tazekka yettɣayyaren, tettɣayyaren kullec d tamuɣli taɣbalut taḥeqqanit. Ayagi d ubrid n tamaraḥut taḥeqqanit tlul-d seg tmusni d tayri.
