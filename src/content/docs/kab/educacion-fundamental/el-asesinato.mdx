Tettut daxel, d ayen ur yettusukkulen ara, d amcum d yugaren akk ayen yellan deg umaḍal.

Amedya n tmettant yellan d amcum, d win yeffsen tudert n wid yellan d ameddakel-nneɣ.

Yella win yessexlaɛen, d azettaɣ yessexlaɛen, d win yesseḥṛeqen tidet n teɣmar meɛna yella win yugaren ayagi s alaf n tikkal, win yesseḥṛeqen atmaten-is.

Ur nettet s ufus n lmitrayuz, n ṣṣutmat, n tmeddaɛ, n lbisṭulett neɣ n tbummatin ṭumiyyin, meɛna nettet daɣen s wallen yessnuzzunen ul, allen yellan d uɣḍan, allen yellan d aḍar, allen yellan d tikti; neɣ nettet s leḥsab n tikli ur nesɛi azal, s tikli taberkant, neɣ s uɣḍas, neɣ s awal yeṭṭaɛnan.

Amaḍal yettwajjel s wid yessnen imawlan-nsen, wid ur yesɛin azal i yemma-tsen d baba-tsen, yesseḥṛeqen imawlan-nsen, ama s wallen-nsen, ama s awal-nsen, ama s tikli-nsen yellan d uɣḍan.

Amaḍal yettwajjel s irgazen yessenfen tilawin-nsen, d tilawin yessenfen irgazen-nsen.

Iban-d belli deg umaḍal-a yellan d amcum, amdan yettett ayen yebɣa.

Ur yettidir bnadem s weɣrum kan meɛna yettidir s wayen yellan deg iman-is.

Aṭas n yirgazen ttwakken ad ddren mliḥ ma yella tilawin-nsen ur ten-ḥarzen ara.

Aṭas n tilawin ttwakken ad ddrent mliḥ ma yella irgazen-nsen ur tent-ḥarzen ara.

Aṭas n yimawlan n twacult ttwakken ad ddren mliḥ ma yella arraw-nsen ur ten-ḥarzen ara.

Taweṭṭant tessawḍen win neḥmal ɣer iẓekwan tebda-d s awal yessexlaɛen, allen yeṭṭaɛnan, tikli ur nesɛi azal, atg.

Tmetti-a tefna teɣli tettwajjel s imettawen ur neḥsib ara yettarran iman-nsen d imseddasen.

Lḥebs yettwajjel s imettawen meɛna amcum yugaren akk yettarra iman-is d amsedda yesɛa tilelli.

Ulac amedya n tmettant yellan s tsebbit. S tmettant n wayeḍ ur nella ara ad nesegger s uɣurfu deg tudert.

Imenɣiyen ur seggran uɣurfu. S tbummatin n temdinin ur yesɛin ara amdan, d tmettant n mraw n yimelyunen n medden ur seggran acemma.

Amenɣi yella d ayen yeqqerḥen, d ayen yettazzalen, d amcum, d ayen yettwakren. Mraw n yimelyunen n tmekkanin yellan d amdan iṭṭsen, ur ḥess ara, ṭulpin, yettunfufen ɣer imenɣi akken ad sneḥṛeqen mraw n yimelyunen n tmekkanin n medden ur ḥess ara.

Aṭas n tikkal yeṭṭef-d tmeɣra ɣef tegnit deg akuzmus, neɣ ad yili wemkan amcum n yetran deg igenni, akken ad uneffen mraw n yimelyunen n yergazen ɣer imenɣi.

Tmekkanin n medden ur yesɛin ara lɛeqliya, tmettlen deg wudem amcum mi yella amedya n wallalen yukuzmiyen yettɛawwen fell-asen deg tuffra.

Ma yella medden ssufɣen-d lɛeqliya, ma yella seg uɣerbaz leɛqal n yinelmaden yettwarebbi akken ilaq, ad asen-yeqqar ɣef lɛeqliya n wayen yellan d afennu d imenɣi, ad ttunɣalen wayen yettɛawaden, ulac win ara yettunfufun ɣer imenɣi d wallalen yellan d amcum deg akuzmus ad ttwasxedmen deg wudem wayeḍ.

Amenɣi yessuɣur d aɣersiw, d tudert deg lemɣarat, d leḥsab n leḥwaj deg uɣersiw yellan d amcum, d taɣwalt, d tseft, d lḥerb, d taɣuri n idammen, yella d ayen ur nemyussan ara s tmeddurt talsayt.

Akk irgazen deg imenɣi llan d at-tekkeɣ, d at-takut d yihiruten yettwaččan s leḥsab llan d at-tekkeɣ yugaren akk, d at-takut yugaren akk.

Win yesseḥṛeqen iman-is yettarra iman-is d aḥerfiq meɛna yella d at-tekkeɣ acku yeggwad tudert.

Aḥerfiq deg ul-is yella d win yesseḥṛeqen iman-is deg yiwet n teswiɛt n tugdi yesɛa-d leḥbul yettruḥun.

Leḥbul n win yesseḥṛeqen iman-is yettuɣal-d d azal n uḥerfiq.

Ma yella neẓra tikli n yeserdas deg imenɣi, amek yettmettil, allen-is, awal-is, adalen-is deg imenɣi, nzer belli yella d at-tekkeɣ meṛṛa.

Iselmaden n uɣerbaz, n tesdawit, ilaq ad slemden inelmaden-nsen tidet ɣef imenɣi. Ilaq ad awḍen inelmaden-nsen akken ad ssinen tideț.

Ma yella medden ḥussen s tideț-a yellan d tamcum, ma yella iselmaden ssinen inelmaden-nsen, ulac yiwen n umdan ara yettunfufun ɣer temmettant.

Aɣri yellan d uɣurfu ilaq ad yettunefk tura ɣer akk uɣerbazen, tesdawiyin acku seg uɣerbaz i ilaq ad nxeddem ɣef telwit.

Yella d wazib ad ilin leǧyal id-yettasen ḥussen ɣef wayen yellan d temẓayt d wayen yellan d imenɣi.

Deg uɣerbazen, tesdawiyin, ilaq ad ttwassen afennuyen d imenɣiyen s wayen akk yellan deg-sen.

Ilaq ad fehmen leǧyal id-yettasen belli at-uqdim s tikti-nsen yellan d timeqqranin, yettwexxiren dima at-meẓẓi akken ad ten-awin d izgaren ɣer temmettant.

At-meẓẓi ur ilaq ad ttwaɣuṛen ara s uɣri n yimenɣiyen, neɣ s tsebbiwin n at-uqdim, acku s tsebbit ara yaɣen tsebbit wayeḍ, d rayi ara yaɣen rayi wayeḍ, meɛna ulac s yirayen neɣ s tsebbiwin ara yillin d tidet ɣef imenɣi.

At-uqdim sɛan alaf n tsebbiwin akken ad ddukkulen imenɣi akken ad awin at-meẓẓi ɣer temmettant.

Ayen yellan d aḥeqqi mačči d tsebbiwin ɣef imenɣi meɛna ad nessin tideț n wayen yellan d imenɣi.

Ur nḥemmel ara afennu neɣ lḥal, kan nbedd ad nessin tidet ɣef imenɣi, syin ad nefṛeḥ nessaramnessen ad nessissen.

Yella d ayen ur yettuɣal ara ad nessin tideț ur nettet ara, ma yella nefseg afḍis ameqqran.

Ala afḍis ameqqran yessawḍen ad nessin tideț ɣef imenɣi.

Iselmaden ur ilaq ara ad fken kan isefka deg lɛeqliya i yinelmaden-nsen. Iselmaden ilaq ad ssinen inelmaden-nsen ad sqedcen lɛeqliya-nsen, akken ad ssinen TIDET.

Tarwa-a tefna teɣli ur tettel ara ala kan ad tettent. Ayagi kan ad tɣellin s ddula deg tarwa n bnadem teɣli.

S tilivizyu d s sinima, igensas n leḥram ttunfufen tikti-nsen yellan d amcum.

Iɛeggalen n tlalit tamaynut tettwaččan mkul ass s tilivizyu d tulmistin n warrac d s sinima d tesseɣlin, d asefru, atg., iččan taɣeltut yellan d asɣar n tmettant d lmizaɛ, n tmeɣra yellan d amcum, atg.

Ur tezmir ara ad tessewjed tilivizyu ur ttfaq ara fell-asen deg awal yettwajjel s aḍar, lmizaɛ d tmara.

Ulac wayen xedmen tuddsa n tmurt mgal uɣri n leḥram.

Lɛeqliya n warrac d wid yellan d ilemẓiyen yettwaččan s igensas n leḥram, s webrid n leḥram.

Tura tefseɣ tikti n tmettant, tefseɣ s tuɣaltin n leḥwayeǧ.

Uɣelnun n yal ass tettunfaq deg tilivizyu n uɣerbaz, deg sinima, deg tulmistin, deg tesseɣlin.

Yella d wazib ad rren iselmaden akken ad d-ssufɣen taɣuɣa deg igenni d lqaɛa akken ad beddlen tiweṭṭant-a deg lɛeqliya.

Yella d wazib ad sersren iselmaden n uɣerbaz d tesdawit taɣuɣa d ad ssutren i tuddsa n umaḍal ad gdel ɣef sinima d tilivizyu, atg.

Leḥram yettizdi deg waṭas acku yesɛa temẓayin n idammen d adalen ara yettawḍen ass-a ur tezmir ara ad teɣri deg iberdan s tilelli bla akukru ad ttwattlen.

Radio d sinima, d tilivizyu d tesseɣlin n idammen, fken aṭas i uɣri n tmettant, rran-t d ayen yellan d lɛali deg lɛeqliya yellan d timeẓẓyanin teɣlin, ur yesɛi yiwen ul akken ad yefk aɣbel neɣ ad yeɣṛes s taɣwalt i wayeḍ.

A teḥreṣ seg waṭas n uɣri n tmettant, lɛeqliya yellan d timeẓẓyanin nemyussan aṭas s leḥram d tura ar ttfaraḥen akken ad ettlen akken ad lḥeqen ɣef wayen ẓran deg sinima neɣ deg tilivizyu.

Iselmaden yellan d leɛqel n medden yella fell-asen deg wayen iɛnan ixeddimen-nsen ad nnaɣen ɣef leǧyal id-yettasen akken ad ssutren i tuddsa n umaḍal ad gdel ɣef temẓayin n idammen, d asɣurfu ɣef akk tulɣa n leḥwayeǧ fell-asen aɣrawal n yettwaṭṭlen, atg.

Ameɣruy n yiselmaden ilaq ad iffeɣ ad yaɣ turri d buksur.

Amedya n uṛeffas n lberr yella d amedya n at-tekkeɣ d amcum. Uṛeffas n lberr yebɣa akk lɛali fell-as d yetett akken ad ifṛeḥ lǧemɛ.

Amedya n ubuksur yella d win amcum n tmettant, deg wudem-is d aziz, d win yeṭṭaɛnen d win yettetten akken ad ifṛeḥ lǧemɛ.

Amedya-a n temẓayin n idammen d adalen deg timit yettunɣalen deg uɣerbaz, deg lɛeqliya, yettunfufen ɣef webrid n leḥram. Ma nebɣa tideț akken ad nnaɣ ɣef telwit n umaḍal, ilaqa ad nbeddu amennu deg uɣurfu mgal temẓayin n idammen.

Mi yella deg ul n bnadem tikta yellan d amcum yella imenɣi ur yezmir ad iffeɣ.

Deg ul n bnadem yella tikta yellan d amcum ad yili imenɣi, tikta-nni llan d aḍar.

Deg ul n bnadem yella tikta yellan d amcum ad yili imenɣi, tikta-nni llan d aḍar, d tukerda, d tḥerfin deg wayen akk yellan, d asɣurfu, d lxiṭaf, d taɣultin n leḥram, d tikta n yimenɣiyen yettunfufen deg tilivizyu, radio, sinima, atg.

Uɣri ɣef telwit, arawasen n tuba n telwit ttuɣalen d igugalen mi yella deg ul n bnadem tikta yellan deg laman yessuffuɣen imenɣi.

Tura aṭas n imettawen sɛan arawasen n tuba n telwit.
