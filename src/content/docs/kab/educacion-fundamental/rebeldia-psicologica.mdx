Wid it-teṭṭfen lweqt-nsen i usikel ɣef akk tmura n ddunit akken ad درسen s telqayt akk iẓuran n yemdanen, zemren ad walin s yiman-nsen belli ṣṣifa n "ɣers-a ameqqran" isem-ines n tṣeḥḥaḥt d amdan, teqqim tazedɣant, ama di Turuft taqdimt neɣ di Tefriqt yedlem lɣella n tdukli, di tmurt yeqdacen n Vedas neɣ di Tgelda Tarrayt, di Ustriya neɣ di Ccinwa.

Ayen-a ibanen, tidet-a tameqqrant tessewham yal amdan y درسen, yezmer ad yettwassen mliḥ ma yella umusakel yettṛuḥ ɣer leɣṛus, lekalij, d tesdawiyin.

Nessaweḍ ɣer tallit n tayyuga n timetti. Tura kullec yettwaxdam s tɣerɣert tazedɣant d tmeqqrant. Tiggura n tdayunin, tkerrusin, tɣawsiwin ɣlayen, atg., atg., atg.

Uɣur ma iban-d d ayen ifesssen, d tidet tameqqrant belli leɣṛus tilmestin, tesdawiyin, atg., uɣalent d tigezmatin timusklanit i tayyuga n timetti.

Di tallit-a n tayyuga n timetti, celɣi n tudert d amqran i usukkel n امنية tusrift. Lɣella tettgalla ɣef kullec tettnadi ɣef usukkel.

Asirem imesskan di tallit-a n tayyuga n timetti, yettili d ayen yeṭṭfen ulma d aɣerɣar acku atil-a n usselmed ameqqran yebna ɣef talɣiwin iɛeqlen kan.

"Amur amaynut" yettidir s teɣzalt tameqqrant akken. Ma yella yiwen yebɣa ad yeqqeɛ ɣef wiyaḍ, kullec ad t-yessenṭeq, kullec ad t-yemqer, ad yettwaɣebbal, ad yettwexxer leɣṛu.

Assirem i usukkel n idrimen i tudert d usseḍsi, leɣlaɣa i usukkel n wayen yelhan di tudert, usukkel n امنية, assirem adɣali n aṭas n tɣawsiwin akken ad ibeggen i wiyaḍ, atg., aḍfar i asirem azedɣan, n tili d ddaɛwa.

Iban-d di tmurt belli lɣella tessɣer abrid d tuɣsaḍt.

Di tallit-a n lɣella d usukkel n امنية, medden ttɣimin deg ibuda-nsen, deg izuɣar-nsen, deg uẓru-nsen, di temdaɣ iɣilen yezmer ad ili yesserf, tawaɣit adɣali yuggaden ad ffɣen syen, tuggda n tudert, tuggda n tilawtin timaynutin, n tillɣat timaynutin, atg., atg., atg.

Kullec ayen-a yettwarwaḥen usselmed ameqqran yebna ɣef lɣella d usukkel n امنية, medden uɣalen d iɣelben, tuggda d tameqqrant ɣef tallit-nsen.

Medden, tuggda tameqqrant ɣef kullec, tuggda n tuffɣa ɣef lɛaddat tiqdimin yettwasbedden, ad qqɛen ɣef medden wiyaḍ, ad ttaɣen tamawt s talɣa tamellayt, ad yettwaɣen akk tizwara n tmetti taneggarut, atg.

S tesɛet, ttiddiren di ddunit aṭas n yemdanen isehwanen d imawalen, s tidet bɣan ad ẓren ɣer lebda akk tawaɣit n usirem, maca di aṭas nnɣer tella i tmara tarwa n tikerkas d tageldit.

Llan snat n telɣiwin n TAGELDIT i mazal ttkawenten. Amezwaru: Tageldit Tamusklanit d taɣzayant. Aɣareb: Tageldit Tamusklanit d taɣzarant n TIZMERT.

Amud amezwaru n Tageldit d Amessaknan d amurran d uḍebbar. Amud aɣareb n Tageldit d AMESLLAY.

Di umud amezwaru n Tageldit Tamusklanit nufa AMESLIL i yewwin iḍebsiyen iqdimen d iɣerrumen idurar iqdimen akken ur ttṛuḥun, amud adɣali, Amesllay n idammen d tɣurma, ameqqran n iḍebsiyen d tɣerɣert n Imaratan, amdan n ufuɣ ɣef ubaba, Amessal i yettwaɣen yesswaẓen ɣer tebrijt akk wid ur yettwamnen tili-nsen, tisinasin-nsen.

Di umud aɣareb n Tageldit Tamusklanit nufa BUDDHA, JESUS, HERMES, Amessaknan, AMAGER TIZMERT, AMESLLAY, iqqwanen Imeqqranen n Tɣerɣert N TMENTEGT, atg., atg., atg.

Wid kan yessiwḍen ɣer tuddar usirem aɣerɣar n tɣella tameqqrant deg ugellid n tɣaltin, ad alin, ad ggren ɣer tɣella n tɣaltin, ad qqɛen, atg., ur yettilin i lebda, d Imejhalen s ṣṣifa, d iɣerbazen, d iɣerbazen, d iɣerbazen iɛeqlen ɣer tmerwazt.

Yettwassen ɣer lebda belli di win yettidir ur tella TMELLULI n tɣawsa d tayri, uɣur ma nɣer aṭas n usselmed, tudert tella d tameɣlal, d tallit, d tasennuqent s aṭas n tɣalla n kullec.

Ḍeffir n kullec d ur tuggda n tɣella, nezmer ad nessenṭeq akken yelha belli war aselmed IMELLULI, tudert tella d tmeɣlal, d tameɣlal, d uḍebbar.

Amɣar AMEQQRAN ila EGO ILELLI yeqqnen s tɣawsiwin tinurfan i tenḥiren s USSELMED IQLAL.

I PLURALIZED i yal yiwen nnɣer, d tikkelt tazedɣant n akk izuɣran d tilawat nnɣer.

ASELMED AGELLID ilaq ad yesselmed ɣer tarwa tamaynuten DIDACTICA NNɣer i TISELMET N I.

Kan s tusɣilt n tilawtin i s umata ttkawent Ego (I) nezmer ad nessetbeddi nnɣer d ammas i ibawen n tmenteg tameɣlent, uɣur ad nɣer IMELLULEN.

Akk tallit yella nnɣer I PLURALIZED, ur nețḍebbi kan tudert i yiman-nneɣ maca nețḍebbi tudert i wiyaḍ.

D acu yeɣzen nessiwḍen ufran d neqqim d aɣerɣad, ma ntesselɣal izuɣran? D acu yeɣzen nẓer di tmenteg nnɣer aṭas n tɣawsiwin, ma mazal neqqim d iɣelben? D acu yeɣzen tizeltin timesskanin d tiɣerma ma nesɣen i usɣal n iɣersiwen-nneɣ?

Ur yeffiɣ usiwḍ, ad nɣer imurasen, ad nɣer, ma di tɣawsa n tudert n wussan nesɣen i yiman-nneɣ yiman-nneɣ.

Celɣi n usselmed ur ilaq ad yeqqim kan ad yessiwḍ yal aseggas isukalen n leɣṛu, amud amaynut n iɣerbazen, amud amaynut iɣerɣazen i ur yessinen ad ḥermen Din n usmili, atg.

Celɣi azedɣan n USELMED AGELLID ilaq ad yessiwḍ yemma d yemma n IMELLULEN uɣalen s tmenteg d tizmert.

Yemma n uɣerɣad i tɣurawtin n leɣṛus, lekalij d tesdawiyin, kullec ttaɣen tamawt, wa ur yesḍegger TIZMERT IMELLULI n TIBBIRA.

Kullec yezmer ad yeẓer d ad yessiweḍ tazdayin, tifratin, timsal, d wa yeqqim d amessi, maca ay-a ur yeqqim ad nɣer TIZMERT.

TIZMERT ur tezemr ad teqqim d afud yezmer, TIZMERT ur tezemr ad teqqim d tikkelt n tɣawsa tiɣerbazin, TIZMERT ur d tazmert i usiwḍ s yiwennit s awal iẓidan ɣef kullec. TIZMERT ur d awal n tmenteg. TIZMERT d tazmert i usiwḍ akken yelha ESSENCIA, TIDET, acu yeqqimen.

USELMED AGELLID d tẓuri i tenḥiren tazmert-a nnɣer d wiyaḍ.

USELMED AGELLID yettɣawsi yal AMDAN ad yẓer TAZMART i ffɣen s tikkelt n usukkel d COMPRENSION IMELLULI N YIMAN-IS.

Tallit ur yettidir nnɣer AUTO-CONOCIMIENTO, uɣur AUTO-EXPRESION teqqim d AUTO-AFIRMACION EGOISTA D DESTRUCTIVA.

USELMED AGELLID yettidir kan ad yesḍegger yal amdan TAZMERT ad yefhem yiman-is di akk tuddar n usirem d ur yeqqim kan i tikkelt n AUTO-EXPRESION tqlal N I PLURALIZED.
