Ur yezmir ad n-ttut belli yella d-wen degneɣ yiwen umḍiq tusnakt... War ccekk, amḍiq-a ur yezmir ad yili deg yizri, neɣ daɣen deg ussan ay d-iteddun...

Win yebɣan ad yaf amḍiq-a yeffren, ilaq ad t-yqelleb dagi akked tura, deg iman-is, deg yiwet n teswiɛt, ur yella ula d snat n teswiɛin ar zdat, neɣ daɣen ar deffir... Sin yiɣsan-nni Agʷlaw akked Uzɣar n Ssalib, ttilin deg umḍiq-a...

Nettfaɛ ad nili seg teswiɛt ar teswiɛt zdat sin Iberdan: Azɣar akked Agʷlaw... D ayen ibanen belli Azɣar d ayen "yellan d amesɣar", deg-s iteddu "Si Muḥend akked medden", "Si Fulan akked win yellan iteddu-d", "Si Fulan akked ddunit meṛṛa"...

D ayen yettbanen belli Agʷlaw yemgarad; d abrid n wid yellan d imḥerḍan ifehmen, win n wid yellan d iruqqaṣen... Melmi ara d-n-mmekti iman-nneɣ, melmi ara nexdem ɣef iman-nneɣ, melmi ur nettekki ara deg yakk uguren akked lehmum n tudert, s tideț nteddu deg Ubrid Agʷlaw...

S tideț ur yelli ara d ayen isehlen ad n-nekkkes tikta timegṛaḍin; ad neggan yakk takti s ddunit-nneɣ; uguren n yal aɣawas, tizi, idewwan, iḥricen, tilifun, waman, tafat, atg., atg., atg. Wid ur nesɛi ara lxedma, wid yellan, ɣef sebba neɣ tayeḍ, ǧǧan lxedma-nsen, s tideț ttɛeggiḍen ɣef ddemma n yedrimen, ad ttun lecɣal-nsen, ur ttaɣen ara fell-asen, neɣ daɣen ad tekkin deg wugur-nsen, d ayen iweɛṛen aṭas.

Wid ittɛeggiḍen, wid ittrun, wid yellan d imeɣban n kra n lxiyana, n ucelleɛ deg tudert, n ur nḥenni ara, n lehduṛ d lekdeb neɣ daɣen kra n wumḍiq, s tideț ttuɣen iman-nsen, Ser-nsen yellan d aqreb, tekkin deg tmuɣli-nsen meṛṛa...

Lxedma ɣef iman, d ayen yellan d agejdan deg Ubrid Agʷlaw. Ulac win yezmer ad yeddu deg Ubrid n Tḥerri-t tameqrant, ma yella ur yexdim ara ɣef iman-is... Lxedma iɣef d-nettmeslay, d lxedma tusnakt; tettḥaz deg kra n ubeddel n teswiɛt deg i nella. Ilaq a nemmel ad neɛyi seg teswiɛt ar teswiɛt...

Amedya, yiwen umdan yellan ittergigi ɣef kra n wugur n tayri, d idrimen neɣ daɣen d aserti, yettban belli yettu iman-is... Umdan-a ma yella yeḥbes kra n teswiɛt, ma yella yezra lḥala yeqelleb ad d-immekti iman-is dɣa ad iɛeyyeq ad yemmeslay ɣef lmeɛna n tikta-ines... Ma yella ixemmem kra, ma yella ixemmem belli kullec iteddu; belli tudert d taɣuṛi, ttṣuffuɣ dɣa lmut tettuɣal d iɣed deg yakk imesɣaren n ddunit...

Ma yella yefhem belli ugur-is deg usuk ur yelli ara d "taftilt n yicwi", d times tilelli i yettṛuḥun s tɣawla, ad yeqqel ad yezre s umengar belli kullec yebeddel... Ad nbeddel tikta tigmaman yezmer s wudem yesseḥsan akked Tmuɣli tazedgant n Ser...

D ayen yettbanen belli medden tteggen s tɣawla zdat lḥalat n tudert... Ay imeskinen!; Țeddun ad uɣalen d imeɣban. Melmi ara ten-seḥmaden ad ttmeǧǧiden; melmi ara ten-rehhen, ttɛeggiḍen. Țekkaten ma yella tekkan-ten; jerḥen ma yella jerḥen-ten; ur llin ara d ilelliyen; imezdaɣ-nsen sɛan tazmert ad ten-yessalin seg lfeṛḥ ar leḥzan, seg usirem ar tuggdi.

Yal yiwen seg wid iteddun deg Ubrid Azɣar, yettuɣal d am asuraru, anda yal yiwen seg imezdaɣ-is iteddu iɣef yebɣa... Win yemmlen ad yebeddel tarwa tigmaman, s tideț yekcem deg "Ubrid Agʷlaw". Ayagi yesskan-d abeddel agejdan deg "Astar n Ser" aḍris yessewhamen n "Tḥerri Tusnakt".
