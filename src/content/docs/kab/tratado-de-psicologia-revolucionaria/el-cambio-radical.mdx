Aql-i yella umdan iteddu deg tuccḍa n laman belli d yiwen, d anagar, d win yiman-is, d ayen ibanen belli beddel ameqqran ad yili d ayen akteṛ n wayen yemmal ur yezmir ad yili.

Tidett n tidet n wayen yella lxedma tazɣart tekkat s usenqad amesɣar n yiman-is, tebgen-aɣ-d azal meqqren n yiferṭiyen yesɣilen, d lxyuḍ neɣ d tgemmiwin ur nebɣi ara ayen yelzem ad nkes, ad negger seg zdaxel nneɣ.

Ayen ur yettḍeggiṛ ara deg wudem yeqqen ur yezmir ad yili umbeddel n tuccḍiyin ur neẓri ara, yelzem ad nsenqed zdat wayen nebɣa ad neɛzel seg tmental nneɣ.

Lexedma n wanaw-agi mačči d axesṛani meɛna d zdaxlani, wid-nni id-yeqqaren belli kra n udlis n lebṣaḥ neɣ d usenqad n tmezɣent axesṛani akessas ad ten-yawi ɣer usuksess, ad ilin deg tidett d wid yetɣelben.

Tidett tgemmeḍt d taneggarut n wayen yella lxedma tazɣart tebda s tmuɣli tjemmɛen ɣer usenqad yemmalen n yiman-is, d sebba akteṛ n wayen yettaǧǧan ad ibeggen belli ayagi yettaxeṛ ameɣlal anamas meqqren n yal yiwen seg-neɣ.

S awal ṣfiḥ yerna ur nteg- ara asfugel, nessenṭeq s usenqad meqqren ayagi: Ulac amdan ara izmir ad ixdem lxedma-agi deg wemkan nneɣ.

Ur yezmir ad yili umbeddel deg tmental nneɣ, war usenqad udmawan n wayen akk yellan d aggay n yiferṭiyen yelzem ad nebbib yid-neɣ deg zdaxel.

Awi-d ayen yellan d aɣil n azal n umɣal n tuccḍiyin, teggugga n lazem n usɣuṛ d usenqad udmawan n yides, yettak deg tidett d awal n tuṭṭfa neɣ d tuffɣa, d tuffɣa seg yiman-is, d akken ad iɣuṛṛ yiman-is.

Anagar s ameɣlal amesɣar n usenqad ɣilen n yiman-is, war tuṭṭfiwin n kra n wanaw ad nezmer ad nebgen deg tidett belli mačči d "yiwen" meɛna d "aṭas".

Aɣal n tegrayit n YO yerna beggen-itt s usenqad amesɣar, d sin n wawalen yemxallafen.

Kra n umdan yezmer ad yeɣal s Tameddurt n waṭas n Yo war ma yebggen-itt ara maḍi; ayagi anagar yezmer ad yili s uɣuṛṛu n usenqad n yiman-is s lemɛun.

Seggerlex lxedma n usenqad zdaxel, ssutur tuṭṭfiwin, d ayen iressan ur yesɣaɣ ara n uneggar.

Aql-i yella umdan yettaṭṭaf awal n laman belli d yiwen akk d netta amezwaru, ur yezmir ad ibeddel yerna, d ayen ibanen belli tmetti n lxedma-agi s tidett ad taweḍ ɣer umbeddel s ussan deg tudert-nneɣ zdaxel.

Beddel ameqqran d tawaɣit teqqen yesɣilen n wawal yettɛeddin mliḥ mi ur nexdim ara ɣef yiman-is.

Asenqad amezwaru n umbeddel ameqqran yeqqim yetɣummi mi umdan ikemmel deg laman belli d yiwen.

Wid-nni id-yeqqaren belli ṭeggiṛen Tameddurt n waṭas n Yo, bengen deg wudem yeqqen belli ur ssenqden ara yiman-nsen s ṣṣeḥ.

Asenqad ameqqran n yiman-is war tuṭṭfiwin n kra n wanaw, yettarra-aɣ-d ad neɛqel yiman-nneɣ s yiman-nneɣ d tidett taḥerract belli mačči d "yiwen" meɛna d "aṭas".

Deg umaḍal n tmuɣliwin tiɣarimin, d tɣuṛiwin yemxallafen tiẓreɣẓeɣriyin neɣ tiẓreɣẓeɣriyin-ɣer-ufella, ttawḍent d agemmaḍ ad rwel seg yiman-nsen...

Deg wudem yeqqen awal n laman belli d yiwen d netta amezwaru, yettarra-d ad iɛeqqel usenqad n yiman-is...

Kra n umdan yezmer ad yini: "ẓriɣ mačči d yiwen meɛna d aṭas, Gnosis yesselmed-iyi-d". Awal-agi ɣas ma yella ṣfiḥ mliḥ ma ulac tajṛiba yemmalen ɣef usenqad-agi, deg wudem yeqqen awal-agi ad yili d ayen axesṛani d akessas.

Bgegen, tajṛiba, yerna fehmen d ayen yeqqnen, anagar akk-agi d ayen iṛṣan ad nexdem s tmentilt ad naweḍ ɣer umbeddel ameqqran.

Ssenṭeq d taɣawsa yerna fehmen d taɣawsa-nniḍen. Mi kra n umdan yeqqar: "fhemeɣ mačči d yiwen meɛna d aṭas", ma afehmi-ines d tidett war awal yesɣilen n tmeslayt tbeddel, ayagi yemmal, yettak usenqad, yettḥasab, tabeggent yemmalen n Tameddurt n waṭas n Yo.

Aɣṛa d ufehem yemxallafen. Amezwaru seg wanwa d seg lmexla, wis sin seg wul.

Aɣṛa ur nḍerru ara n Tameddurt n waṭas n Yo seg kra yellan; s tumert deg lawan-agi deg ayen nettidir, aɣṛa yella yecbaṛ akteṛ n ufehem, acku aɣersiw amellal yetɣelben iwumi semman d umdan, yesnerna anagar tama n aɣṛa yerna yetṭeggiṛ s tumert tama yeqqnen n Ussileɣ.

Aẓer n Tameddurt n waṭas n Yo yerna fehmen-itt, d ayen yeqqnen i kra n umbeddel ameqqran n tidett.

Mi kra n umdan yebda ad isenqed yiman-is s ṣṣeḥ, seg leǧwahi n wayen mačči d yiwen meɛna d aṭas, deg wudem yeqqen yebda lxedma n ṣṣeḥ ɣef tneglit-is zdaxel.
