Melmi ara yeqqim umdan yeؤؤmen s tɣaltit yesla iman-is Yiwen, Yiwen kan, Ur yettwabḍa, iban d akken beddel ara ikemmel ad yeqqim d ayen ur yezmir. Ayen i d-yebda lxedma n lbaṭen s ulɣuẓ ɣef iman-is s ṣṣeḥ, yeskan-aɣ-d aṭas n lecɣal iɣuṛfan, Iɣefan neɣ iderman ur neṣɛi ara, ilaq ad ten-nekkkes, ad ten-nessufeɣ seg wul-nneɣ.

Bla ccekk, ur yezmir ara ad nekkes tɣaltin ur nessin ara; ilaq ad nalɣuẓ zik ɣef ayen nebɣa ad neffeɣ seg iman-nneɣ. Axxam n lxedma-agi mačči deg berra, meɛna deg daxel, wid yeṭṭfen belli kra n udlis n tezdayt neɣ aswir n tissas n berra d wudem ad ten-yessiwḍen ɣer lfayda, ad ɣlin deg tɣaltit.

Akk-nni yesɛan lmeɛna d ṣṣeḥ belli lxedma n daxel tebda s wallaɣ n tektiwin deg ulɣuẓ deg iman-is s lekmal, d sebba tameqqrant akken ad nban belli yettwaleq amennuɣ d amaynut i yal yiwen seg-neɣ. Ad nmeseɣti ṣṣeḥ d bla ccekk, ad nini s ṣṣeḥ akka: Ulac umdan yezmer ad yexdem lxedma-agi fell-aɣ.

Ulac abeddel deg iman-nneɣ bla ulɣuẓ d usrid ɣef ayen akk i d-ǧemɛen lecɣal-nni yeṭṭfen deg tɣawsiwin-nneɣ i nerrez deg daxel-nneɣ. Ad neqbel aṭas n tɣaltin, ad neqqar ur ilaq ara ad nelmed d ad nalɣuẓ fell-asent srid, d lḥal d uɣurru neɣ d tazzla, d turewla ɣef iman-is, d tɣaltit ɣef yiman-is.

Anagar s amennuɣ n ulɣuẓ s tẓuri ɣef iman-is, bla turewla n kra n ṣṣenf, ad nezmer ad neẓri ṣṣeḥ belli ur nelli "Yiwen" meɛna "Aṭas". Ad neqbel aṭas n YES, ad nbanent s ulɣuẓ s ṣṣeḥ d sin n lecɣal yemgarraden.

Wellaḥ yezmer ad iqbel Tamusni n aṭas n Iɣefan bla ma yella yeẓra-ten; ayagi anagar ma yeẓra iman-is s leɛqel. Ad nerwel lxedma n ulɣuẓ n daxel, ad nqelleb ɣef uɣurru, d laman ur yettwaḍuq ara n lexyab. Melmi ara yeqqim umdan yeṭṭef deg tɣaltit belli daymen d netta kan, ur yezmir ara ad ibeddel, iban d akken aswir n lxedma-agi d ṣṣeḥ ad nessufeɣ abeddel s wudem deg tudert-nneɣ n daxel.

Abeddel ara ikemmel d tagella yesɛan laman yettruḥu melmi ur nexdim ara fell-as. Axxam amezwaru n ubeddel ara ikemmel yettuffer melmi umdan yeqqim yeؤؤmen belli d Yiwen. Wid i ur yeqbilen Tamusni n aṭas n Iɣefan sselken belli ur ẓrin ara yiman-nsen s lḥerma.

Ulɣuẓ s lḥerma ɣef iman-is bla turewla n kra n ṣṣenf yettajeɛ-aɣ ad nẓer s yiman-nneɣ lḥeqq n ṣṣeḥ belli ur nelli "Yiwen" meɛna "Aṭas". Deg umaḍal n tmuɣliwin n tɣawsiwin-nneɣ, aṭas n tmuɣliwin ɣelbeniwin-is neɣ tferqinin-is daymen ttaxren-aɣ ɣef iman-nneɣ... Bla ccekk, tɣaltit belli daymen d netta kan ttaxer-aɣ ɣef ulɣuẓ-nneɣ...

Wellaḥ yezmer ad yini: "ẓriɣ ur nelli Yiwen meɛna Aṭas, Gnosis tesselmed-iyi-t". Awal-agi, ɣas ad yili d ṣṣeḥ, bla ma tella t تجربة tesɛan tmeddit ɣef uswir-agi n Tamusni, s lekmal awal-agi ad yeqqim d ayen n berra d wudem. Ad nessken, ad n تجربة d ad neɛqel d ayen yesɛan azal; akka kan nezmer ad nexdem s leɛqel akken ad nessufeɣ abeddel ara ikemmel.

Ad nini d ayen, ad neɛqel d ayen nniḍen. Melmi wellaḥ yeqqar: "Fehmeɣ ur nelli Yiwen meɛna Aṭas", ma tefhem-it s ṣṣeḥ d mačči kan awal ur yesɛi ara lmeɛna n umeslay amɣawwa, ayagi yeskan, yettaǧa, yesserɣay, lḥeqq n ṣṣeḥ n Tamusni n aṭas n Iɣefan. Tmusni d Tafhemt d yemgarraden. Amezwaru seg-sen seg uqerruy, wis sin seg wul.

Kan tmusni n Tamusni n aṭas n Iɣefan ur تسمن. Lmeɛna, deg lawan-agi i nezmer, tmusni terbaḥ tafhemt, acku الحيوان aqerruy umi qqaren umdan yeǧǧa kan ad yellaḥ kan tmusni ur yettɣimi ara ɣer deffir deg lweqt n Ser. Ad nessin Tamusni n aṭas n Iɣefan d ad neɛqel-itt d ayen yesɛan azal i yal abeddel ara ikemmel d ṣṣeḥ.

Melmi umdan ibda ad yeẓẓer iman-is deg teẓẓegga n belli ur yelli Yiwen meɛna Aṭas, s lekmal yebda lxedma n ṣṣeḥ ɣef tɣawsiwin-is n daxel.
