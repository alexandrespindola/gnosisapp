Asegmi amezwaru n iman n wegdud-nneɣ d win ur yettɛeṭṭilen ara ma yella d ilaq ad naf akk talliyin tignawin idenneyen.

Bla ccek, talliyin n daxel idenneyen zemrent ad ttuseggwent s tikkwatin iduzen.

Akken yella tudert n daxel d amɣenay yesɣaray imeɣriyen n berra, ilaq-aɣ s tɣawsa tameqrant ur yettɛeṭṭilen ara ad nekkes seg tmuski-nneɣ talliyin tignawin itɣeltiyen.

Ad nseggem talliyin tignawin idenneyen d tɣawsa yessefk ad yili ma nebaɣ ad nebeddel s lqayda tabadut n kra n imeɣriyen ur neḥemmel ara.

Ad nbeddel arbib-nneɣ akked kra n imeɣriyen, yezmer ma nekkes seg daxel-nneɣ kra n talliyin tignawin ur nesɛi lmeɛna.

Tawennaṭin n berra tmesɣariyin, zemrent ad uɣalent d tibenɣasin ula d tid yebnuyen s useggem uffir n talliyin n daxel itɣeltiyen.

Yal wa yezmer ad ybeddel tagma n imeɣriyen ur neɛjib ara iten-id-yettilin, mi ara yeṣfu seg ul. Win ur yesseggem ara talliyin tignawin ur nesɛi lmeɛna, yettamen s iman-is yettɣimi d awezɣi n tiktawin.

Ad nesseɣzu deg uxxam-nneɣ n daxel yettwaɣzen d tɣawsa tameqrant, mi nebaɣ ad nbeddel tikli n tudert tmeɣbunt.

Lɣaci tettɛeyyi ɣef yal tɣawsa, tettɛeqqa, tettru, tettɛeyyi, tebɣa ad tbeddel tudert, ad teffeɣ seg lexyab id ttafen, meɛna s tɣara ur xeddmen ara ɣef yiman-nsen.

Ur bɣan ara ad fehmen lɣaci belli tudert n daxel tesɣaray tikkawtin n berra, ma yella ttid n leɛtab d sebba n talliyin n daxel ur nesɛi lmeɛna.

Ayen yellan deg berra d ttawil kan n wayen yellan deg daxel, win yettbeddilen deg daxel ad yesnulfa addad ajdid n tɣawsiwin.

Imeɣriyen n berra ur yezmir ad ilin d imeqranen am akken yella d tikli n urar zdat-sen.

Teqqimeḍ tessusem zdat win yettɛeyyiwen? Teqbaled s tumert tikkawtin ur neɛjib ara n watmaten-ik?

Amek urareḍ zdat tixṣim tewwet tasa? Tjeǧǧeḍ-ik ad teddukkel tiser n tismin? Tenɣeḍ? Tellaḍ deg lḥebs?

Sbitarat, leqbur neɣ imukan n leqyada, lḥebs, ččaṛen d wid yettɣelṭen s ṣṣeḥḥa uraren s tɣara ur nesɛi lmeɛna zdat imeɣriyen n berra.

Ayla yifen akk i yezmer urgaz ad yesseqdec deg tudert, d addad agnawan iduzan.

Yal wa yezmer ad yessebser iɣersiwen u ad ifek trajit, s talliyin n daxel iduzen.

Talliyin n daxel itɣeltiyen ttɣimin-aɣ d awezɣi ur nesɛi ara amkan n tḥerzi zdat tissas n wemdan.

Lemdet amek ara ttameldeḍ zdat imeɣriyen ur neɛjib ara n tudert tunṣibt s tikli n daxel iduzen...

Ur tetteffgeḍ ara s kra n umeɣri; mmektit-d belli kullec yezger; lemdet ad twalit tudert am yiwen usaru u ad taɣem tafuyt...

Ur ttut ara belli imeɣriyen ur nesɛi ara azal yezmer ad awen-yawin ɣer lmeɣbuna, ma ur tekkesem ara seg tmuski-nwen talliyin n daxel idenneyen.

Yal ameɣri n berra yeḥwaǧ, bla ccek, tikṛaṭ iduzen; yeɛni, addad agnawan yemɛaddan.
