Tazwara tasɣart n tikli i diri deg iman d ayen ur yettɛeṭṭilen ara akken ad naf arraten idelsan ur yesseḥḥan ara. War ccekk, arraten n daxel ur yesseḥḥan ara zemren ad ttusemmaḥen s tikliwin yellan d tid yesseḥḥan.

Akken tezmert i tudert n daxel d imɣez i d-yettjebdun idrisen n berra, nessarem s tɣawla tameqqrant ur nɛeṭṭilen ara akken ad nekkes seg tnefsit-nneɣ arraten idelsan imekden. Asseḥḥu n arraten idelsan ur yesseḥḥan ara d ayen ilaqen mi ara nebɣu ad nbeddel s tigawt taberkant tabaɛet n kra n idrisen ur neɛǧib ara.

Ad nbeddel taɣara-nneɣ akked kra n idrisen yemmalen-d ma nekkes-d seg daxel-nneɣ kra n arraten idelsan ur yesɛin ara lmeɛna. Tignatin n berra yemmalen-d tixessaṛin zemrent ad uɣalent d tid ur yesɣersayen ara ula d tid yebnan s usemmaḥ n arraten n daxel imekden.

Yal wa yezmer ad ibeddel tabaɛet n idrisen ur neɛǧib ara i d-yekkan fell-aneɣ mi ara yessebɣer iman-is n daxel. Win ur yesseḥḥun ara arraten idelsan ur yesɛin ara lmeɛna, yettamen iman-is d aḥerfi, yettuɣal d aɣellaq n tignatin.

Ad nerr asudd deg uxxam-nneɣ n daxel d ayen ilaqen mi ara nebɣu ad nbeddel tikli n tmeddurt-nneɣ n teɣḍeb. Lɣaci ttmesslayen ɣef yal wa, ttmɛeṭṭaben, ttrun, ttnarnaɣen, bɣan ad bedlen tudert-nsen, ad ffɣen seg tɣawsa ur neɛǧib ara deg ula d-llan, maɛandi ur xedmen ara ɣef iman-nsen.

Ur bɣan ara ad fehmen lɣaci belli tudert n daxel tettejbed tignatin n berra d ma tigi d tid yessektafen d tabaɛet n arraten n daxel ur yesɛin ara lmeɛna. Ayen yellan d azewweṛ d awal kan n ayen yellan d daxel; win ibeddlen daxel-is yettarra-d asudd ajdid n umahil.

Idrisen n berra ur ttnagwin ara ad ilin d imeqqranen am wamek i nessekkay-d ɣur-sen. Teqqimeḍ d asemmaḍ zdat win i k-yeḍḥan? Teqbleḍ s usefrah tignatin ur neɛǧib ara n warraw n jins-ik? Amek teskayeḍ-d zdat tixessaṛt n win tḥemmelḍ? Teffeɣ-d s ssem n tmeẓẓugẓut? Teggaraḍ? Telliḍ deg lḥebs?

Ixxamen n leɛyada, imeqqabren, lḥebus, ččuren d wid yesɣersayen yesḥeqqen s tikli ur yesɛin ara lmeɛna zdat idrisen n berra. Acal ameqqran win yezmer umdan ad yesseqdec deg tudert d arrati idelsan yesseḥḥan.

Yal wa yezmer ad yessemruǧ aydaɣ d ad yesɣerɣer imseɣlan s arraten n daxel yellan d tid yessaɛan leqder. Arraten n daxel ur yesseḥḥan ara ttarran-aneɣ d aɣellaq n tixessaṛt n wemdan. Lemdet ad teẓrem zdat tignatin ur neɛǧib ara n tudert taɣersant s tɣara n daxel yellan d tid yessaɛan leqder...

Ur ttmettaren ara akked yelliḍ ayen, d-ttmektiḍ belli yal wa yezger; lemdet ad teẓrem tudert am film d a d-tetteqebleḍ lexyar... Ur ttettuḍ belli tignatin ur yesɛin ara azal zemrent ad kkrsent-ken ɣer teɣḍeb ma ur tekkesḍ ara seg tnefsit-nwen arraten n daxel ur yesɛin ara lmeɛna.

Yal idris n berra yessaram, war ccekk, adlis yessaɛan leqder; yeɛni, arrati idelsan yessaɛan leqder.
