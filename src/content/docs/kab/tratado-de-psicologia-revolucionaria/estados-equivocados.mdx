War d-iban, deg leqdic n tmusni n yiman, ilaq daymen ad yili tifrat gar wayen yellan deg tudert n yal ass d wayen yellan deg ul.

Ilaq fell-aɣ ad nẓer anda nella deg yal lḥal, ama deg wayen yellan deg ul-nneɣ, ama deg wayen i d-yeḍrun deg barra.

Tudert s yiman-is d amḍiq anda ttilint tɣawsiwin deg uzemz d wanda yella wemkan...

Yenna yiwen: "Tudert d ssnesla n leɛtab yessegrew umdan deg teṛwiḥt-is..."

Yal wa yeḥrek ad ixemmem akken i yebɣa; nekk ɣileɣ belli iḍuluyen n tferḥa n kra n yewwi, ad ten-ḍefren leḥzen d lemṛaṛa...

Yal tɣawsa tesɛa tɛemmt-is yettwabeqen fell-as, yerna lḥalat n ul ttilin-d s ṣṣenf nniḍen; ayagi d tidet ur yettuwexxer...

S tidet, axeddim deg ul ɣef yiman d ayen yeqqnen ɣer lḥalat n tmusni yemgaraden...

Ulac win ara yesgunfuyen belli deg ul-nneɣ neɣleb aṭas n tuccḍiwin yerna ttilint lḥalat urǧin...

Ma nebɣa s tidet ad nebdel, ilaq fell-aɣ s teɣɣat d lebda ad nbeddel seg ṣṣaṣ ayagi d lḥalat urǧin n tmusni...

Asbeddel amcum n lḥalat urǧin, yeggra-d ibeddilen imcum seg ṣṣaṣ deg ugaraw n tudert n yal ass...

Mi ara ixeddem yiwen s ṣṣeḥ ɣef lḥalat urǧin, s tidet tɣawsiwin ur neɛjib ara n tudert, ur zemrent ara ad ḍurrunt fell-as s lfeṭla...

Neqqaqay-d kra kan i zemren ad t-yefhem win i t-izedɣen, yeḥḥuss s tidet deg ugaraw n wayen yellan...

Win ur nexdem ɣef yiman-is daymen d ameɣbun n tewtilin; am uẓɣar ameẓẓyan gar waman yettberriḥen n yilel...

Tɣawsiwin beddlent daymen deg taggayt-is yemgaraden; ttulin-d wa deffir wa deg wagguden, d ttɛawinat...

S tidet ttilint tɣawsiwin yelhan d tɣawsiwin diri-t; kra n tɣawsiwin ad ilint yelhan neɣ diri-tent ugar n tiyaḍ...

Asbeddel n kra n tɣawsiwin yezmer; Asenqas n igemmaḍ, asbeddel n tewtilin, atg, yella s tidet deg uṭṭun n tzemmar.

Meɛna ttilint tewtilin s tidet ur zemrent ad țbeddlent; deg leḥwayeǧ-agi yellan gar tlisa ilaq ad tent-neqbel s tmusni, ɣas akken kra seg-sent ḍerrent s waṭas yerna tzemrent ad ḍurrunt...

War ccekk, leɛtab yettruḥ mi ur nettɛawez ara iman-nneɣ ɣer ugur i d-yeffɣen...

Ilaq ad nḥes tudert am akud yettḍerrun n lḥalat n ul; tamacahut taḥeqqanit n tudert-nneɣ n yiman d wayen i d-yessegrawen seg akk lḥalat-agi...

Mi ara d-nelqeḍ akk tudert-nneɣ n yiman, nezmer ad neḥqe-tt s yiman-nneɣ deg talɣa tusridt, belli aṭas n tewtilin ur neɛjib ara yezmer ad ilint s lḥalat n ul urǧin...

Aleksander Ameqran ɣas akken yella daymen d aquran s ṣṣifa, yefka iman-is ɣer lexyub yessawḍen ɣer lmut-is s zzux...

Francisco I yemmut s sebba n zzna yemgaraden yerna yeccḍen, i d-tettmektin mazal amezruy...

Mi yettwanɣa Marat sɣur tmeṭṭut tucḍan, yella yemmut seg zzux d ẓẓel, yumen s yiman-is d aḥeqqi seg ṣṣaṣ...

Tlawin n ufrag n Iqeddacen s tidet sselkent akk tudert n uḥejjam n tɣawsiwin iɛeddan isem-is LUIS XV.

Aṭas n medden yettmettaten ɣef rreḥba, wurrif neɣ ẓẓel, ayagi t-ẓran akken iwata Yesɣunen...

Akkenni ara t-tɛawed tebɣest-nneɣ lebda deg tnegluft ur nesɛi lmeɛna, nttalas ad nuɣal d imeɣban ɣer tmanaẓt neɣ ɣer uẓekka...

Otelo s sebba n ẓẓel yuɣal d uneggaray yerna lḥebs ččuṛ d tuccḍiwin s wul d neyya...
