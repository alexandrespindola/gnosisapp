Imi d akken win iɛlayan d win wadda d snat n tɣessa n yiwet n tɣawsa, ur yettḍur ara ad nesbedd amgired-a: "NEK AɛLAYAN, NEK WADDA" d snat n wudmawen n yiman aglamisan d yemgerraden.

Win umi qqaren "IMAN ILAHI" neɣ "IMAN AɛLAYAN", "ALTER EGO" neɣ kra n wayen icuban, d tteḥḥila n "MI MISMO", d tarrayt n TƐEJJEB IMAN.

Mi yebɣa IMAN ad ikemmel da d dinna, yetteɛjeb iman s tikti yelan n IMAN ILAHI AMETTANT...

Ula d yiwen seg-neɣ ur yesɛi "iman" n tideț, amagnu, ur yettbeddil, anebdad, ur yettwakal, atg., atg., atg.

Ula d yiwen seg-neɣ ur yesɛi s tideț Tdukli n Tɣessa n tidet; Ɛlaḥsab-nneɣ ur nesɛi ula d tmedyazt tunṣibt.

Ego ɣas akken ittekemmel seld n tsertit, yesɛa daɣen tazwara d taggara.

Ego, IMAN, ur yelli ara d tɣawsa tinemlayt, tamegdayt, tamegdayt. Bla ccekk, IMAN d "IMANEN".

Di Tibit Utriman, "IMANEN" ttwagdelen "TDUKLIYIN TIKSANIN" neɣ kan "Azalen" ama d izuranen neɣ d irsalen.

Ma nemmel belli yal "Iman" d amdan yemgerraden, nezmer ad d-nini s tsebɣet aya: "Deg yal amdan i yettidiren deg umaḍal, llan aṭas n medden".

Bla ccekk deg yal yiwen seg-neɣ ttidiren aṭas n medden yemgerraden, kra yelhan, kra yettḍurren...

Yal yiwen seg Imanen-agi, yal yiwen seg medden-agi ittaɛra i tzelɣa, yebɣa ad yili d win iḥerzen, ixdem fell-aɣ deg wallaɣ amagnu neɣ deg imerkez n usirem d tuzzya yal mi yezmer, mi ara t-yefser wayeḍ...

Tasenɣart n waṭas n Imanen ttwalem di Tibit Utriman sɣur wid yeẓran tidet, sɣur wid iwalan tidet...

Yal yiwen seg lɛibad-nneɣ iseklan yettwammet deg Iman-agi neɣ deg wayeḍ. Akken yesɛan gar-aneɣ imɣuren d imellalen d waṭas n lɛibad, s useɣti ttidiren aṭas n medden deg daxel-nneɣ.

Deg yesteqsiyen iseklan nezmer ad nebder s jelas belli wid yesɛan aɣbel ad d-sseɣḍen, wid yettferriḥen iman-nsen d wid yettmeslayen ɣef tidet ur ttjaḥḥiden ɣef lqima n Ego iḥemmlen.

Bla ccekk medden-agi keren mgal tasenɣart n aṭas n "Imanen".

Mi yebɣa yiwen s tidet ad yemmezzi iman-is, ilaq ad yettwali iman-is yerna ad yeɛreḍ ad yessen "Imanen" yemgerraden yellan deg daxel n tɣessa.

Ma kra seg wid iɣerren ur yefhim ara mazal tasenɣart-agi n aṭas n "Imanen", d sebba kan n leɛyaḍ deg temsalt n Wali Iman.

Akken ittemmeslay yiwen iman-is deg Walli Iman Utraray, ad yaf iman-is aṭas n medden, aṭas n "Imanen", ittidiren deg daxel n tɣessa-nneɣ.

Wid yettnejjiden tasenɣart n waṭas n Imanen, wid iɛebden IMAN ILAHI, s lmut ur ttwallin iman-nsen s tɛeqqal. Ad nemmeslay tikkelt-agi deg uɣrib n Sukrat, ad d-nini belli medden-agi ur ẓran ara kan, maca daɣen ur ẓran ara belli ur ẓran ara.

S tidet ur nezmir ara ad nemmezzi iman-nneɣ, war Walli Iman s tɛeqqal d tlemmast.

Akken ad yili umdan yesḥisseb iman-is d yiwen, jelas belli yal asenfel utraray ad yili d tɣawsa iweɛren.
