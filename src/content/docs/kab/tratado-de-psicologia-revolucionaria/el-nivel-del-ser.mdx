Anwa neɣ? Anda d-nekka? ɣer anda ara nṛuḥ? Acuɣer i nedder? Acuɣer i nella ddunit?...

Ur yelli kra n ccek belli "Aɣersiw Aɣalen" ameskud yettusemman argaz, mačči kan ur yessin, meɛna ula d akken ur yessin ara ur yessin... Ayen yugaren akk d tagnit-a i deg nzedɣen, ur nessen sser n tmeɣṛanin-nneɣ, meɛna nettuɣal yella fell-aɣ nessin kullec...

Tawi-d "Amestuɣ Aɣalen", yiwen seg wid yettarran iman-nsen d imeqqranen di tudert, tawi-t ɣer tlemmast n uneẓruf n Sahara, ǧǧ-it dinna beɛda ɣef yal oasis, tessermeḍ-t seg uɣerrabu igenni ayen yeḍṛun... Tira ad mmeslayent s iman-nsent; "Aɣersiw-mmi Aɣalen" ɣas yettarra iman-is d aqewwi yeqqar d aqcic, deg ul-is d adɣaɣ uḥric...

"Aɣersiw Aɣalen" d abuhal s timmad-is; yettxemmim ɣef yiman-is ayen yelhan; yumen yezmer ad yelḥeq ayen yebɣa s Kindergarten, Imesfuḍen n Tnezduɣt, Timelmadin Tilemmastin, Timelmadin Tineggura, Baccalaureate, Tasdawit, isem yelhan n baba-s, atg., atg., atg. Aḥzen, deffir n tira d lɛeqliya yelhan, izwilen d yedrimen, nessen belli kra n lehlak deg uɛebbuḍ ad yeɣzen fell-aɣ, deg ul-nneɣ mazal nella d imeɣban d imeskinen...

Yekfa ad neɣṛe Amezruy Alemad iwakken ad nessen nella d ibarbaren n zik, anagar deg umesbadel n leḥwala nerra ur neɛdil ara... Asgass-a wis XX akk ayen yeččuṛen-t, imenɣan, tlawin tiserɣidin, tameslut n yal tamurt, tilas tiseksiyin, ttay, ccṛab, ẓẓlem ur nelli d wagi-nni, lɛehya tameɣṛanit, leḥwaǧ yettuɣalen d iɣersiwen, atg., atg., atg., d lemri i deg ilaq ad nzer iman-nneɣ; ulac sebba i deg ilaq ad nezux nezṛa nessawḍeɣ ɣer tarrayt yugaren deg unekcum...

Ad txemmimeḍ lweqt yesɛani anekcum d amcum, aḥzen "imeẓlanen imɣalen" mazal tenṭḍen deg "Aɣris n Tnezɣert"... Deg yisebḍar imseberkan n "Amezruy Aberkan" nufa kul waqt iɣuṛaf uḥric, leḥɣa, imenɣan, atg. Meɛna imezdaɣ-nneɣ "Imesbedden ɣef ufus" mazal umnen ayen yellan d imenɣi d ayen yettwaggaden kan, taɛkemt i d-yeffɣen ur yesɛi ara tamsalt akked "Taseddaṛt Tatrart"-nsen...

War ccek ayen yelhan d amek yella yal yiwen; kra n medden ad ilin d imaɣlaḍen, wiyaḍ ad qqimen, wiyaḍ d imdanen, wiyaḍ d imeɣban; seg kullec yella di tudert... Tigrumma d agem n medden; ayen yellan d yiwen, d tigrumma, d aɣella, atg. Tigrumma d asteɛṛef n yiwen; ur yettili wassenɣel n tigrumma, n leǧnas, ma yiwen, ma yal yiwen, ur yesseɣfel ara...

Ulac win yezmer ad yenkeṛ yella uɣrudan imettiyen yemgaraden; llan medden n lkanisa d medden n tserɣida; n tnezzut d tellaɣt, atg., atg., atg. Akken daɣen llan uɣrudan yemgaraden n Lwejd. Ayen yellan deg ul-nneɣ, d aεzizen neɣ d abxiε, d aɛzizen neɣ d imesxaṛ, d imjuhden neɣ d imɛeṭṭlen, d izedɣanen neɣ d imeqqren, yessawḍen-d tignatin yemgaraden n tudert...

Yiwen ameqṛan ad yessawḍ-d yal waqt tignatin, idrimen d ula d tmeɣṛanin n tserɣida deg ara yettuɣal... Yiwen amaɣlaḍ ad yessawḍ-d imaɣlaḍen ad yuɣal yal waqt ɣer tɛeqqin, d ayagi ɛellan... Acu ara d-yessawḍ usebɣaɣ, d umesxaṛ? Acḥal n aɣwan, lḥebs, lehlak?

Meɛna medden iḍeqqen, yeqwan seg leɛtab, bɣan ad sseɣlen, ad sseɣlen asebṭar n umezruy-nsen... A medden imeɣban! Bɣan ad seɣlen ur ssin ara amek; ur ssin ara aɣarass; ttwaḥeqren deg ubrid ur yesɛi ara abrid... Ayen i sen-yeḍṛan iḍelli ad sen-yeḍṛu ass-a ad sen-yeḍṛu azekka; ɣɣaren kul waqt tilɣa yellan d imcumen ur lemden ara tira n tudert ulac wazal...

Kullec yettaɣ deg tudert-nsen; qqaren ayen qqaren, xeddmen ayen xeddmen, ḥeznen ɣef ayen iḥeznen... Tagarayt-a yesselḥayen n idrimen, tullisin d tmeɣṛanin, ad teqqim akka ɣas nezzel deg ul-nneɣ aɣbalu iqessiḥen n ɣḍab, Lbuɣes, Tserɣida, Tusmin, Ccerf, Tεeggez, Lmakla ur nelli d yagi-nni, atg., atg., atg.

D acu-t uɣrud imseggem-nneɣ?, neɣ ayen yelhan: D acu-t Uɣrud n Lwejd-nneɣ? ɣas Uɣrud n Lwejd ur yesseɣfel ara, ad teqqim tagarayt n kul imeskinen-nneɣ, tignatin, lehlak d ttlaɛit... Kullec, tignatin, i d-yettaɣen beṛṛa fell-aɣ, deg umezdaɣ n ddunit-a, d agris kan n wayen yellan deg ul-nneɣ.

S tidett nezmer ad nmeslay s tseddaṛt belli "ayen yellan beṛṛa d agris n wayen yellan deg ul". Asmi ara yesseɣfel yiwen deg ul-is akk taɣellist-a d taɣellist, ayen yellan beṛṛa, tignatin, tudert, ttbeddilen daɣen.

Sermi ttmuqulleɣ deg lweqt-a (Aseggas 1974), yiwen n ugraw n medden i yellan ɣer tellaɣt. Dagi di Mexico medden-a ttwasemman s yisem yessewham "PARACAIDISTAS". D imezdaɣ n colonia campestre Churubusco, qerben ɣer uxxam-iw, s sebba ayagi zemreɣ ad ten-ɛasseɣ...

Ad yili d imeɣban ulac fell-asen tawaɣit, meɛna ayen yeqqsen ulac dinna, anagar deg Uɣrud n Lwejd-nsen... Yal ass ttaɣen iman-nsen, ttamɣalen, tsekkafayen, uɣalen d imeqqaren n imdukkal-nsen, zedɣen s tidett deg tɣerɣuriyin i deg umesbadel n tayri yesseɣyayen aẓẓal...

Aṭas n tikelt xemmmeɣ ma yiwen seg wid, ad yefsi seg ul-is aẓẓal, ɣḍab, tserɣida, tumlilt, lḥeqq, ẓẓlem, d umesxaṛ, d tismin, d tayri i yiman-is, d ccerf, atg., atg., atg., ad yegmeḍ i medden, ad yesdukel s tarwa n ẓẓeḥma n tɣawsiwin n tɣerma, s medden igerrzen, s medden imseḍfanen; tadukli-agi tajdiṭ ad tili taggara i useɣfel amazun d imetti...

Ad yili d ussan i deg ad yawi medden, ad yeǧǧ "cochera", "cloaca" taberkant... Akken daɣen, ma nebɣa aɣellist, ayen s wazal-is ilaq ad nfehmeḍ belli yal yiwen seg-neɣ (ma d amellal neɣ d aberkan, d azeggaɣ neɣ d azizaɣ, d amejhal neɣ d amɣal, atg.), yella di "Uɣrud n Lwejd".

D acu-t Uɣrud n Lwejd-nneɣ? Tessexemmmeḍ ayen deg yiwen n lweqt fell-as? Ur yezmir ad nɛeddi ɣer uɣrud ma nettusemɣel tagnit i deg nella.
