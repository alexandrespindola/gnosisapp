Anwa ay nennu? Anda d-nekka? ɣer anda ara nruḥ? Acuɣer ara nɛic? Acuɣer ntedder?...

Bla ccekk, "Aɣersiw Amekkay" ameɣbun, i semman i wemdan s tuccḍa, ur yessin kan ara, maca ula d yessin ur yessin ara...

Acu yugaren akk d lḥala-nni iweɛren i deg nella, ur nessen ara sser n tagedyazt-nneɣ merra, maca nerra d ṣṣeḥ nessen kullec...

awi-d "Amderyal Aqli" (Mammifère Rationnel), yiwen seg wid deg tudert țseḍhin iman-nsen d wid yesɛan azal, s axxam n ṣṣeḥra n Sahara, ǧǧ-it dinna beɛda ɣef yal tanzuft, teẓreḍ-t seg uɣerrabu deg igenni d acu ara yeḍrun...

Tidyanin ad mmeslayent iman-nsent; "Ameddan Amekkay" ɣas yesɛedda iman-is d aqewwaɛ urra iman-is d argaz, deg uzɣar yețban d adɛif aṭas...

"Aɣersiw Aqli" d amehbul deg meyya yimelyunen; yețxemmim ɣef iman-is d ayen yelhan; yumen yezmer ad yelḥu akken yelha s Kindergarden, Tiktawin n Tmeddurt Tamaneɣt, Ibderriyen, Isinanen, Asdawan, Isem n baba-s yelhan, atg., atg., atg.

A tawaɣit, deffir n tira-nni merra d lecɣal ilhan, azalen d idrimen, nessen akken yesseɣlay-aɣ aqerruy kan n tidet-nneɣ yessufeɣ-aɣ-d leḥzen d akken mazal nella d imeɣban d yeqquranen...

Yeqqen ad nɣer Amezruy Alemɣi (Histoire Universelle) ad nessen akken nella d irqiqen imezwura d akken deg wadeg n ma nnerni nebɣu nexreb...

Lqern-agi wis XX akk d ccbaḥa-ines, imenɣiyen, tlawin imeɣyalen, taɣewsa n ṣṣiyasa deg ddunit, asemɣer n ṣṣiyasa n tewtemt, ddrugga, ccṛab, assexreb iggarren, leɛyub izewwqen, tiɣawsiwin yettḍuṛun, atg., atg., atg., d lemri i deg ilaq ad nezzer iman-nneɣ; ulac ssebba n ṣṣeḥ i deg ara nezmer ad neḍfer akken newweḍ ɣer tarrayt taneggarut n tuẓẓma...

Axxamen akken akud yesɛa anamek d anerni d ayen ur nessɛi ara lmeɛna, aɣriben ilawen mazal ttnuzun deg "Asmekti n Tiddi n Umaḍal"...

Deg yal asebter aberkan n "Umezruy Aberkan" nufa-d yakan imerwalen issexraɛen, tmuɣliwin, imenɣiyen, atg.

Maca imezdaɣ-nneɣ "Iṣṣubren-tmeddura" mazal umnen akken ayen-nni n imenɣi d ayen yellan deg daxel, asibda iɛeddan ur yesɛi ara leqdic akk d "Tmeddurt Tamaneɣt"...

S tidett ayen yeqqnen d amek ara yili yal yiwen; kra ad ilin d irqiqen, kra ad ilin d wid ur yeswennuyen ara, wid-nni ad ilin d wid yeqqaren d wid-nni ad ilin d imjerben; kullec yella deg tudert...

Lɣaci d jemɛa n medden; ayen yella deg yiwen yella deg lɣaci, d adabu, atg.

Lɣaci d tiddi n yiwen; ur yezmir ad yili asnerni n lɣaci, n leǧnas, ma yiwen, ma yal yiwen, ur yettnerni ara...

Ulac win ara yenkuren yella gar-asen iẓragen imaɣalen; llan medden n ǧǧameɛ d yeqquranen n axxam n tlawin imeɣyalen; n tjarat d tmazirt, atg., atg., atg.

Ula daɣen yella gar-asen Iẓragen n Yili. Acu yellan deg daxel-nneɣ, d imeqranen neɣ d imezẓiyen, d wid yettaken neɣ d wid yettṭfen, d wid yesɛan lɛeẓ neɣ d wid yettṣillin, d wid yesɛan ccḥama neɣ d wid yesɛan ccehwa, yessawel-d lecɣal yemgaraden n tudert...

Win yesɛan ccehwa ad d-yessawel dima ɣef ṣṣur, aɣewwanen d ula d tagedyazt n ccehwa deg ara yettɣummu...

Win yessenwen ccṛab ad d-yessawel ɣef wid yessenwen ccṛab ad yettɣummu dima deg ibarat d yeqquranen, ayagi iban...

Acu ara d-yessawel ɣef win yettaken riba, win yexḍan? Acḥal n uguren, lḥebs, tawaɣit?

Maca lɣaci iqquṛen, yewɛen seg usuffer, bɣan ad beddlen, ad zwaren asebter n umezruy-nsen...

A lɣaci ameɣbun! Bɣan ad beddlen ur ssinen amek; ur ssinen ara ttawil; ttnuzun deg yiwen iberdan ur yesɛi ara taggara...

Acu yedṛan yid-sen iḍelli ad yedṛu yid-sen ass-a d ad yedṛu yid-sen azekka; ttaɣen dima tuccḍiwin-nsen, ur lemden ara leqdic n tudert neɣ s yifadden.

Kullec yettwaɛawed deg tudert-nsen; qaren tidyanin yellan, xeddmen tidyanin yellan, ttnadin tidyanin yellan...

Aɛawed-agi yeqqaren n aɣewwanen, ur yezmir ara ad iɛeddi ɣas ur yezmir ara ad yekkes deg daxel-nneɣ iferdisen ur nesseɛḍay ara n zzɛaf, ṭṭmeɛ, ccehwa, taseddaɛt, ṣṣura, taɣewsa, taɣewsa, atg., atg., atg.

Acu-t azal-nneɣ ilan? neɣ yelha ad nini: acu-t Azal-nneɣ n Yili?

Ma Azal n Yili ur yebeddel ara s uẓar, ad yaɛawed yakan tawaɣit-nneɣ merra, aɣewwanen, tawaɣit d tawaɣit...

Kullec, lecɣal merra, i yedṛun beṛṛa-nneɣ, deg udiwan n ddunit-agi, d akken yella d tiddin n acu i nella deg daxel.

S ṣṣeḥ nezmer ad nini s tsellet akken "ayen beṛṛa d tiddin n ayen yellan deg daxel".

Mi ara yebeddel yiwen deg daxel d beddel-agi d uẓar, ayen beṛṛa, lecɣal, tudert, ad beddlen daɣen.

Lliɣ ttẓerreɣ deg akud-agi, (Aseggas 1974), yiwen n tegrewla n lɣaci i yeṭṭfen araz n wiyaḍ. Dagi deg Meksik lɣaci-agi yettwassaman s "PARACAIDISTAS".

D imezdaɣ n teɣremt n kampestri Churubusco, zgan ɣer taddart-iw, d ayen-agi i d ssebba i deg zmereɣ ad țekkeɣ fell-asen...

Ameddakul ur yezmir ad yili d tawaɣit, maca ayen iweɛren mačči dinna i yella, maca deg Azal n Yili...

Yal ass ttaɣen gar-asen, sewwenen ccṛab, qqaren gar-asen awalen n ddɣel, uɣalen d ineqqasen n imdukkal-nsen n tawaɣit, zgan s tidet deg texxamin iceḍbanin deg daxel n yeqquranen deg wadeg n tayri d lkerh i yellan...

Aṭas n tikkal xemmmeɣ ma yiwen seg wid-nni, ad yekkes seg daxel-is lkerh, zzɛaf, ccehwa, assexreb, anezwa, assexreb, lɛib, lxilaf, taseddaɛt, tayri iman, ṣṣura, atg., atg., atg., ad ɛeǧben i wiyaḍ, ad tdukkel s Lqanun n Tɣellisin Tameddaytin akk d lɣaci iṣeffan, yesɛan iman; tdukliwin-agi timeddura ad ilint d tid ibanen ɣef beddel ahrawi d ameddan...

Ad yili d asatal-nni ara yesnernin i win yellan, ad yeǧǧ "tamcarṭawt", "tilemt" iceḍban...

Akka, ma nebɣa s tidett beddel iḍefṛen, ayen ilaq ad neɛqel d acu yiwen-yiwen deg-nneɣ (ama d amellal neɣ d aberkan, d awraɣ neɣ d azeggaɣ, d aɣrib neɣ d illaw, atg.), yella deg "Azal n Yili".

Acu-t Azal-nneɣ n Yili? Ma txemmemem yakan ɣef wayagi? Ur yezmir ara ad nɛeddi ɣer wazal-nni ma ur nessin ara lḥala i deg nella.
