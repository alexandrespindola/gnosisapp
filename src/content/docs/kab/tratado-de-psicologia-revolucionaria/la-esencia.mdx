Ayen i yeǧǧan yal aqcic i d-ilulen d ameqqran d uceɛɛan d ẓẓaher-is; ayagi yettili d tidet-is... Anerni n ẓẓaher deg yal lxelq, s tidet, d ameẓyan, d amezwaru kan...

Teffɣal n tfekka n wemdan tettnerni tettnernay akken i d-tennaẓent leqwanin tibelɣiyin n telmest, maca aken i d-tewwet, tella d ameẓyan i ẓẓaher... Ur yesɛi ara ccekk, ẓẓaher yezmer ad yernu weḥd-s mbla ɛiwen, deg ubrid ufiḥal...

Acuɣer ur n-neqqar ara s terangugt belli anerni ara d-yekken seg ẓẓaher, d ayen ara yilin deg tlata, rebɛa neɣ xemsa n yiseggasen imezwura n tudert, yeɛni deg tarrayt tamezwarut n tudert... Lɣaci ttɣilen belli anerni d unernay n ẓẓaher, yella d win yettnernin, akken yella deg ufus n leɛnaya, maca lɣustikizm agraɣlan yesɛellem-aɣ belli ayagi ulac...

Akken ad yernu ẓẓaher s nnig, ilaq ad yili wayen yugaren, ilaq ad nexdem ayen ijdiden. Bɣiɣ ad d-mmeslayeɣ s turḍelt ɣef lxedma ɣef yiman-ik. Anerni n ẓẓaher yella kan ma neddukel deg lxedma d lqella n uɣil...

Ilaq ad nefhem belli lxedma-agi ur tettmeslay ara ɣef tmesleɣlin, ibanka, tḥeddin, tɛemmar, tḍebsireɣ n telɣuɣa n wakal neɣ ɣef temsal n ufis... Lxedma-agi d win n yal yiwen i yellan yesɛa tawsit; tella d taysikit...

Nekwni akk نعرف belli nella deg yiman-nneɣ ayen i wumi neqqar EGO, NEK, YIMAN-IW, YIMAN-IS... Aɣil-aɣ, ẓẓaher yella deg tɣemmar, deg tlemmast, deg EGO, ayagi d ayen yettruḥen. Ad nɣell lYU n teskiɣit, ad nqeṛṛeḥ iɣerfan-is iḍuṛṛen, d ayen iǧehden, ur nezmir ad nɛetteq, ur nezmir ad nsenḍeq... akka i yella leɛnaya n lxedma ɣef yiman-ik. Ur nezmir ara ad nserḥeṛ ẓẓaher mbla ma nqeṛṛeḥ uqbel lYU n teskiɣit...

Deg ẓẓaher tella d din, lBUDDA, tmusni, tiḥerract n wawaṭ-nneɣ i yellan deg igenwan d akk ayen i nesḥeq ad nexdem i iman-nneɣ. Ulac win i yzemren ad yeɣɣel lYU n teskiɣit mbla ad isseḥyu uqbel iɣerfan ur nḥeemmel i nella deg yiman-nneɣ...

Nesḥeq ad nerr i wakal lweḥc n zzman-a: tismin i d-tuɣal d tidet i d-yettarran ad nexdem; tarrayt n uḥerri i d-yerran tudert d ayen yetṭuɣuren: tarrayt n uṣeḥyu: tarrayt n lkadba i d-yettawin tameɣra: tiḥercawin: tarrayt n zzina i yellan ur yettɛaǧab ara: ugar d ugar...

Akken tettɛeddi temsal-a ɣer wakal, ẓẓaher ula d netta ad yekker, ad yernu, ad yettnernay s terwa... War ccekk, mi yemmut lYU n teskiɣit, ad yeẓẓuɣeṛ ẓẓaher deg-neɣ...

Ẓẓaher azedgan i yettawi ɣur-neɣ ṣṣifa; seg ṣṣifa-agi i d-yekka lferḥ d leḥmala n tidet... Ẓẓaher yesɛa tarrayin n tezdeg d iẓuran n tezmert... Mi "Nemmut deg Yiman-Nneɣ", mi nɣell lYU n teskiɣit, ad nesɛu ṣṣifat d tezmert n ẓẓaher...
