Ila ad nebɣu aɣuṛṛu n tidet, ad neffeɣ seg tmuɣli-yagi iɣuṛṛun, seg tudert-agi iqeddcen kan, yetɛeyyun...

Ayen yessefk ad neɛqel s tuẓẓma tameqqrant, yal yiwen deg-nneɣ, ama d aṛumawi, ama d axeddam, ama d ameṛkanti, ama d ameɣlal, ama d ameṛkanti, ama d aḍellaɛ, yella deg yiwen n Weswir n Tudert...

Asewir n Tudert n usekkran yemxalaf ɣef win n win ur yeswa ara, win n tmeṭṭut-nni yezzenzen iman-is yemxalaf ɣef win n teqcict. Ayen i d-neqqar d ayen ur yettujaddalen, ur yettuseqsan...

Mi newweḍ ɣer tama-yagi n yixef-nneɣ, ur nxesser acemma ma nexdem-d taswiɛt ɣef tserriḥt yellan teɣzi seg wadda ɣer yeffus, d tin yellan d tmezwarut d useɣzu mliḥ...

Deg tlemmast n uswir-agi, d ayen ur nettujaddalen, ad naf iman-nneɣ; deg uswir-nni d wadda ad ilin medden yugar-aɣ lɣaci; deg uswir-nni d yeffus ad ilin medden yugar-aɣ lɣaci...

Deg tin yellan d taɣzelant-agi, deg tserriḥt-agi d taɛjebt, d ayen ibanen ad naf akk iswiren n Tudert... yal amdan yemxalaf ɣef wayeḍ, ur yelli win i zemren ad t-yessenqed...

D ayen ur yettujaddalen, ur netmeslay ɣef udmawen-nni iqebḥen neɣ icebḥen, ur d ameslay ɣef laṣel n leɛmeṛ. Llan medden d imeẓyanen d imeqqranen, imɣaren iqerben ad mmten d warrac i d-ilulen kan...

Ameslay ɣef lweqt d useggas; ayen n tlalit, ad yemɣuṛ, ad yennecli, ad izewweǧ, ad d-yesɛu dderya, ad yeɣwer, ad yemmet, d ayen yellan d useɣzu kan...

Deg "Tserriḥt Taɛjebt", deg tin yellan d tmezwarut, ameslay ɣef lweqt ur yettak ara. Deg uswir-agi n tserriḥt, ad naf kan "Iswiren n Tudert"...

Asirem n lɣaci ur yesɛi ara lmeɛna; ḥesben belli s lweqt, leḥwayeǧ ad ḥlawent; akka i ḥesben jeddi-nneɣ d jeddi-nnejdeddi-nneɣ; tidet tessaweḍ-d ad tessekned taneggarut...

"Asewir n Tudert" d ayen yesɛan azal d tin yellan d tmezwarut; nella deg yiwen n uswir maca nezmer ad naik s uswir-nni wayeḍ...

"Tserriḥt Taɛjebt" i ɣef netmeslay d tin yeɛnan "Iswiren n Tudert" yemxalafen, s tideț, ur tesɛi ara walaɣ d lweqt usrid...

"Asewir n Tudert" yellan d agʷer, yella nnig-nneɣ, yal tikelt...

Ur yelli deg uzekka yellan d useɣzu, maca dagi d tura; deg ulawen-nneɣ iman-nneɣ; deg tmezwarut...

Iban yerna yal yiwen yezmer ad t-yefhem, sin yezwiren—Useɣzu d Tmezwarut—tettemlili deg uqelleb-nneɣ, yerna xedmen Tserriḥt...

Tudert tezdeɣ, tetteddu deg yezwir n Tudert. Ad ilul ad yemmet deg lweqt-ines usrid; yettemmettat; ur yelli uzekka i tudert n win yemmuten; ur d Tudert...

Iswiren n Tudert; Tudert iman-is, ur d lweqt, ur tesɛi ara walaɣ d Yezwir Useɣzu; tella deg ulawen-nneɣ iman-nneɣ. Tura, deg Tmezwarut...

Ad iban ma nessufeɣ Tudert-nneɣ ɣer berra...

Ur d ayen yellan d wadda ma nerra ayagi: Izwal, taɣultin, tirrin, atg., deg amaḍal yellan d agensan, s wudem ur yellan d agerram ad d-teffenɣen taɛzizt n tidet, ad d-rren azal n Tudert, ad ɛeddin ɣer uswir nnig "Iswiren n Tudert"...
