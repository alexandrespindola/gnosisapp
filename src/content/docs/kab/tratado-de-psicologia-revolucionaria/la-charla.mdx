Ilaz s tehri ur d-iɛeggiḍ, ur yelli wass-is, yeḥwaǧ ad nwali awal n daxel akked umkan d exact anda i d-yekka.

Bla ccekk, awal n daxel yellan d aɣalat d "Sebba n Sebba" n waṭas n waddaden iḥulfan ur nettwafeq ara akked wid ur nseɛɛuq ara deg wass-a akk-d asirem.

S uɛeqli, awal-agi ur nesɛi lmeɛna, ur yesɛi azal, d awal anda ur tettwaḍefret ara lmeɛna, akk-d yal awal yeḍurren, yeɛjeben, d laɛmeq, yeqqaren deg umaḍal n berra, yesɛa aẓar-is deg umeslay n daxel yellan d aɣalat.

Nettwassen belli yella deg tmusni taddayrant (ésotérique) n tussna taddayrant (Gnose) n tissusmi n daxel; ayagi ttwasnen sɣur inelmaden nneɣ n "Taqcict Tis Tlata".

Ur yeɛdi ara ma neqqar s ṣṣeḥ belli tissusmi n daxel ilaq ad tettwakkes i kra yellan d exact yerna yettwassen.

Mi ara yeqqers uswir n umyaru (penser) s lebɣi deg urefnu addayan (méditation) a d-iɛeddi tissusmi n daxel; maca mačči d ayen i nebɣa ad nselḥu deg yixef-agi.

"Ad tessufeɣ tamgeṭ" neɣ "ad t-tesserɣeḍ" bac a d-iɛeddi s ṣṣeḥ tissusmi n daxel, ur d ayen nebɣa ad nselḥu tura deg yefyar-agi.

Ad txedmeḍ tissusmi n daxel i nebɣa ad nemmeslay fell-as, ur tɛni ara ad tɛeqqleḍ kra ad yekcem deg tamgeṭ.

S ṣṣeḥ, nemmeslay tura ɣef uɣerbaz n tissusmi n daxel yemxalafen aṭas. Mačči d kra yellan d amatar.

Nebɣa ad nxedmeḍ tissusmi n daxel deg wanda yella kra yellan yakan deg tamgeṭ, amdan, wayen yeḍran, lameṛ nnek neɣ n wiyaḍ, ayen i ɣ-d-qqaren, ayen i yexdem fulan, atg., maca mbla ma nettaney fell-as s yiles n daxel, mbla lemɛawna iḍuman (intime).

Ad tlmed ad tessusem mačči kan s yiles n berra, maca daɣen, ugar, s yiles ufsin, addayan, yezmer ad yili d ayen isehlen, d ayen ifazen.

Aṭas i yettusussumen deg berra, maca s yiles n daxel-nsen ttkessen tudert i umeddukel-nsen. Awal n daxel d aqqaɛ d amesmer, yefka-d ccwal deg daxel.

Ma twalaḍ awal n daxel yellan d aɣalat a t-tẓred yețwaxdem s tideț n uḥeẓẓi, neɣ s tideț yeṭṭfen gar-asen s tarrayt ur teṣṣeḥ ara, neɣ kra i d-nerna neɣ i nettu.

Ayaɣes, tudert-nneɣ taḥulfant (émotionnelle) tella kan ɣef "tumert-nneɣ".

I usfari n ccedda-agi, nettwareɛ kan iman-nneɣ, ɣer "ɣuḍu" nneɣ aɛaziz, neḥmal akk-d wid ur nettwareɛ ara yid-neɣ.

Nettwareɛ aṭas iman-nneɣ, nekkni d narcisistes deg meyya n meyya, ayagi d ayen ur nezmir ad nesseɣleb, ad nɛerreq.

Mi ara naṣar ad nili nɛemmer deg "tumert-nneɣ", yal tunṭaḍa n Wayes-i, tettwaxdem ugar n wayen ur nezmir.

Ilaq a nelmed ad nwali tamuɣli n wiyaḍ. Ilaq s tehri ad nessen ad nezger ɣer waddad n wiyaḍ.

"Akka daɣen, ayen akk i tebɣam ad t-xedmen yimdanen yid-wen, xedmet-t ula d kunwi yid-sen". (Mateyu: VII, 12)

Ayen yesɛan azal s ṣṣeḥ deg leqraya-agi d tarrayt anda imdanen ttamnen deg daxel d wayen ur nettwali ara gar-asen.

Ayaɣes ula ma nella d imɣurbanen, ḥett deg ṣṣeḥ s tikwal, ulac ccekk belli d ayen ur nettwali ara d daxel nettaɣ aṭas gar-aneɣ.

Imdanen yettwassenen d imɣurbanen, ttaɣen yal ass amekcem nnsen ɣer tɣemmar ufsin n iman-nsen, bac ad xedmen yid-sen, ayen akk i yeɛjeb-asen. (Ttawẓaṭ, tmesxir, tiraɣ)
