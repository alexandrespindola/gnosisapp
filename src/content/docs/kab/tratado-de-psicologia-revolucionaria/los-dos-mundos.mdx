Ad xemmimen d win yettwaxemmim fell-as d sin n yixefen yemgaraden, maca yal yiwen deg-sen yeddem tmuɣli.

Deg uxemmem, tmuɣli tettwazzen ɣer berra, ɣer umaḍal n berra, s ttaqen n wallen.

Deg uxemmem ɣef yiman-is, tmuɣli tettwazzen ɣer daxel, i wagi, tẓuṛiwin n unagraw n berra ur s-ttawḍent ara, d ayen i yesḥussen d ayen yesɣarayen i umezwaru, axemmem ɣef tlufa tusnaktin i d-yellan deg daxel.

Tasnarrayt n tmassant tunṣibt deg tama-s taṣṣura, d ayen yettwabderren. Tasnarrayt n lxedma ɣef yiman-is, d ad xemmimen ɣef yiman-is, d ayen i yettwabderren ɣef yiman-is.

Ur yesḥaqq ara ma yella d ayen yeḍran ger snat n tasnarrayin-agi i d-yezgan fell-aɣ, ad aɣ-awint ɣer temsal yemgaraden s lekmal.

Yezmer ad yiwen yeqqim d aqdim ger tmesɣayt n tmassant tunṣibt, ad iɣar ɣef tilufa n berra, ad ixemmem ɣef tzelmaḍ, iṭumaten, tiẓunin, itij, itran, ikukraḍ, atg, war ma yeḍra deg daxel-is kra n beddel aqbayli.

Taɣessa n tmusni i ibedlen daxel n yiwen, ur tezmir ara ad tettwaɛreḍ s axemmem n berra.

Tamusni taṣṣura i yezmer ad d-teffeɣ deg-neɣ kra n beddel deg daxel yesɛa aẓaṛ deg uxemmem ɣef yiman-is s ttawil.

D ayen yebɣan i inelmaden nneɣ inusnaktin ad xemmimen ɣef yiman-nsen d acu i d tewser a xemmimen ɣef yiman-nsen dɣa d acuɣer.

Axemmem d abrid i beddel n talɣiwin timessaranin n umaḍal. Ad xemmimen deg daxel d abrid i beddel deg daxel.

Akk daxel n wayagi, nezmer d ilaq ad nini s ṣṣut ɛlayen, llan snat n tɣessawin n tmusni, tin n berra d tin n daxel, dɣa ma ur nesɛi deg yiman-nneɣ ammas amagnati i yezmer ad yefreq timɣarin n tmusni, axlaḍ-agi n snat n tasnarrayin, neɣ tin n wudem n yixefen yezmer ad aɣ-yawi ɣer tmerziwt.

Asɣanen iɛlayen iɛudden ɣer tin yeffren i yeṣfan, yesɛan tazdayt d tmassant deg tlemmast, tteɛladen ɣer wanda i nettwabderren, maca ttwamnen sɣur aṭas n wid yebɣan ad ilin d tmusni deg daxel.

Nettwaxlaḍ-d zdat snat n umaḍalen, win n berra d win n daxel. Amezwaru deg-sen yettwazgar sɣur tẓuṛiwin n unagraw n berra, wis sin yezmer ad yettwassen kan sɣur tẓuṛiwin n uxemmem n daxel.

Tixmamin, ttawilat, iɣuṛfan, iṭṭfen, asirem, aḥebbus, atg, llan deg daxel, ur ttwalin ara s tẓuṛiwin timzeggẓanin, maca ɣur-neɣ llan s tideṭ akken yella umeslay n umẓan neɣ lkuṛsi deg tmazirt.

S tideṭ nettwidir deg umaḍal-nneɣ n daxel akken deg win n berra, ayagi ur yettḥeggi ara, ur yettwagren ara.

Deg umaḍalen-nneɣ n daxel, deg umaḍal-nneɣ n tuffra, nettnadi, nettḍurru, nettḥasi, nettmuḥ, nettɛeyyab, nettfaraḥ, nettwakal, nettwakafaw, atg, atg, atg.

Ur yelli ccek belli snat n umaḍalen n daxel d win n berra ttwaẓaran s ttawil. Umaḍal n berra d ayen yettwabderren. Umaḍal n daxel d ayen i ixemmimen fell-as deg yiman-is dɣa daxel-is, dagi dɣa tura.

Win yebɣan s tideṭ ad yissin "Umaḍalen n daxel" n umaḍal neɣ n umazzag anafuk neɣ n tmazzag i deg nettwidir, ilaq ad yissin uqbel umaḍal-is aqerrib, tudert-is deg daxel, yellan d yiman-is, "Umaḍalen n daxel" yellan deg-s.

"Argaz, ẓer iman-ik ad tessined amaḍal d yillu".

Akken i yella wa iqelleb "Umaḍal n daxel" i wumi qqaren "Yiman-is", akken i yessaɛref yettidir deg snat n umaḍalen, deg snat n tidett, deg snat n timḥitin, win n berra d win n daxel.

Akk i yella win i t-yessawalen ad yelmed ad ileḥḥu deg "Umaḍal n berra", i wakken ur yeɣḍer ara deg uẓru, ad yemmerzeg deg ibṛidan n taddart, ad yextar imdukkal-is, ur yettɛawan ara d wid yeṭṭefren, ur yeṭṭeɣ ara asemmam, atg, akken daɣen s lxedma tasanaktin ɣef yiman-is, nettlemmid ad ileḥḥu deg "Umaḍal n daxel" i wumi i nezmer ad naɣer sɣur uxemmem ɣef yiman-is.

S tideṭ tẓuṛiwin n uxemmem ɣef yiman-is ttentwaḍeɛtent deg tderɣal n talsa idewlen n tallit-agi tberkant i deg nettwidir.

Akken i nettdumu deg uxemmem ɣef yiman-nneɣ, tẓuṛiwin n uxemmem ɣef daxel ad ttnernin s tuẓẓma.
