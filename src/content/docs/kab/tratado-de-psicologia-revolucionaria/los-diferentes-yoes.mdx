Amcic yettxemmim yesseɣleḍ s yisem n wemdan, ur yesɛi tilawt tusligt.

Bla ccekk, tamesxilt-agi n teɣḍert deg wemdan, d ssebba n waṭas n wuguren d lemriṛ.

Tiddi tajeṛmanit d tdukli tlemmat yerna tettwaxdem am yiwet, ɣas ma teṭṭef-itt uhellec.

Lameɛna, tudert n wemdan deg ul, ur tella d tdukli tusligt.

Ayen i yugaren ayagi akk, ɣas ma d-nnan kra n yiɣerbazen n tewsit n tsufɣa-tugniwin d ukris-tugniwin, d laɣlaq n tikti deg ul n yal yiwen.

S tideț, deg waddad-agi, ulac axeddim ilemman am yiwet, deg tudert n ul n medden.

Amdan, ɣef wayen yellan deg ul-is, d tamuget n tikti, d jemɛa n "nekni".

Imqellaɛen i yesselmden deg lweqt-agi aberkan, ttaɛbaden "nekni", sselɣayen-t, srusun-t ɣef umezdaɣ, ssawalen-as "nekni nniḍen", "nekni yeflalen", "nekni araken", atg., atg., atg.

Ur bɣin ara ad fehmen "Imeẓẓuɣen" n leɛmer-agi aberkan deg i nzedɣen, belli "nekni yeflalen" neɣ "nekni yeddanen", d sin iḥricen n wagi i yettemɣuren...

Amdan ur yesɛi s tideț "nekni yeqqimen" meɛna aṭas n "nekni" yemgaraden iderfanen d tmuɣlin.

Aɣersiw amekla yettuxemmim yesseɣleḍ s yisem n wemdan, icuba axxam yeccuɛren anda anwa i yesɛan axxam, llan aṭas n yiqeddacen i bɣan ad amren yerna ad xedmen ayen i sen-yehwa...

Aɣalat ameqran n Tsufɣa-tugniwin d Ukris-tugniwin uḍim, d tmuɣli belli wiyaḍ sɛan neɣ yesɛa yiwen "nekni yeqqimen d ureffu" bla tazwara d taggara...

Ma yella wid i yettxemmimen akka uɣalen ɣer tmusni ɣas ma d imir kan, zemren ad gren allen ɣef iman-nsen, belli amdan yettxemmim ur yettbeddil ara...

Amcic amekla seg tama n tikti, yettbeddil yal ass...

Ad ttxemmimeḍ belli ma yella yiwen isem-is Sliman yella d Sliman, icuba ɛuhda ur teɛjib ara...

Amahil-nni i weqqaren Sliman, yesɛa deg iman-is "nekni" nniḍen, tinefsatin nniḍen, i d-yettbanen ɣef tɣelt-is deg tallitin yemgaraden, ɣas ma ur yeɛjib Sliman tira, "nekni" nniḍen deg-s —ad as-neqqar Ḥmed— teɛǧeb-it tira, akka akka...

Ulac yiwen amdan i yettbeddil deg tallit lemmat, ur ilaḥḥiq ad iliɣ d amussnaw i wakken ad fehmeɣ ibeddalen d tmesɣeltin ur ṣehḥant n yal yiwen...

Ad ttmuɣuleḍ belli yiwen yesɛa "nekni yeqqimen d ureffu" icuba ɣef uḥeqqi ɣer wemdan nniḍen d iman-is...

Deg yal amdan zedɣen aṭas n medden, aṭas n "nekni", ayagi zemren ad gren allen fell-as iman-nsen, yal yiwen yeqqimen d araken...
