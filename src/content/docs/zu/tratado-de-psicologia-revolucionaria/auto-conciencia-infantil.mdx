Sitshelwe ngobuhlakani ukuthi sinamaphesenti angamashumi ayisishiyagalolunye nesikhombisa OMqondo Ongaphansi nokuthi sinamaphesenti amathathu OMqondo Ophezulu.

Uma sikhuluma ngokusobala nangokungagwegwesi, sizothi amaphesenti angamashumi ayisishiyagalolunye nesikhombisa weSiqu esisiphethe ngaphakathi, itholakala iboshiwe, ifakwe, ifakwe ngaphakathi koBuntu ngabunye obakha i-"Mina ngokwami".

Ngokusobala iSiqu noma UMqondo Ophezulu ofakwe phakathi koBuntu ngabunye, uyacutshungulwa ngoba imvelo yawo iyawuvumela.

Noma yibuphi uBuntu obuqediwe bukhulula iphesenti elithile loMqondo Ophezulu, ukukhululwa noma inkululeko yeSiqu noma yoMqondo Ophezulu, bekungeke kwenzeke ngaphandle kokuqedwa koBuntu ngabunye.

Uma kunenani elikhulu loBuntu obuqedwayo, kuba nokuzazi okukhulu. Uma kunenani elincane loBuntu obuqedwayo, iba ncane iphesenti loMqondo Ophezulu ovukile.

Ukuvuka koMqondo Ophezulu kungenzeka kuphela ngokuqedwa kobuMina, ngokufa kuye ngokwakhe, lapha manje.

Ngokungangabazeki uma iSiqu noma UMqondo Ophezulu ufakwe phakathi koBuntu ngabunye esibuthwele ngaphakathi, ulalele, usesimeni somqondo ongaphansi.

Kuphuthuma ukuguqula umqondo ongaphansi ube umqondo ophezulu futhi lokhu kungenzeka kuphela ngokuqeda oBuntu; ngokufa kuye ngokwakhe.

Akunakwenzeka ukuvuka ngaphandle kokuthi ufe kuqala kuwe ngokwakho. Labo abazama ukuvuka kuqala ukuze bafe kamuva, abanalo ulwazi lwangempela lwalokho abakushoyo, bahamba ngokuqinile endleleni yamaphutha.

Izingane ezisanda kuzalwa ziyamangalisa, zijabulela ukuzazi ngokugcwele; zivuke ngokuphelele.

Ngaphakathi komzimba wengane esanda kuzalwa iSiqu siphinde sahlanganiswa futhi lokho kunikeza isidalwa ubuhle baso.

Asisho ukuthi amaphesenti ayikhulu eSiqu noma oMqondo Ophezulu aphinde ahlanganiswe enganeni esanda kuzalwa, kodwa amaphesenti amathathu akhululekile ngokuvamile angaboshiwe phakathi koBuntu.

Noma kunjalo, lelo phesenti leSiqu samahhala elihlanganiswe kabusha phakathi komzimba wezingane ezisanda kuzalwa, libanikeza ukuzazi ngokugcwele, ukucaca, njll.

Abantu abadala babheka ingane esanda kuzalwa ngesihawu, bacabanga ukuthi isidalwa asizazi, kodwa bayaphambuka ngeshwa.

Ingane esanda kuzalwa ibona umuntu omdala njengoba enjalo ngempela; akazi, unonya, unenkohlakalo, njll.

OBuntu bengane esanda kuzalwa beza bebuye, bazungeze ubhede, bangathanda ukungena emzimbeni omusha, kodwa ngenxa yokuthi ingane esanda kuzalwa ayikakhi ubuntu, noma yimuphi umzamo woBuntu wokungena emzimbeni omusha, uba yinto engaphezu kokungenzeki.

Ngezinye izikhathi izidalwa ziyethuka lapho zibona lezo zipoki noma oBuntu abasondela ebhedeni lazo bese ziyakhala, ziyakhala, kodwa abantu abadala abakuqondi lokhu futhi bacabange ukuthi ingane iyagula noma ilambile noma yomile; bungenjalo ubuyanga babantu abadala.

Njengoba ubuntu obusha buqala ukwakheka, oBuntu abavela ezimpilweni zangaphambilini, baya bengena kancane kancane emzimbeni omusha.

Lapho sekuhlanganiswe wonke oBuntu, sivele emhlabeni ngobubi obubi bangaphakathi obusikhombisa; bese sihambahamba njengabantu abalele yonke indawo; ngaso sonke isikhathi singazi, ngaso sonke isikhathi sinenkohlakalo.

Lapho sifa, izinto ezintathu ziya ethuneni: 1) Umzimba womuntu. 2) Ingemuva elibalulekile lezinto eziphilayo. 3) Ubuntu.

Ingemuva elibalulekile, njengesipoki, liyaqhubeka liwohloka kancane kancane, phambi kwethuna njengoba nomzimba womuntu uwohloka.
