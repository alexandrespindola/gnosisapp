Nokho, ngeke sikwazi ukuzisebenzela ngenhloso yokuchitha "uMina" othile ngaphandle kokuthi siqale simqaphele.

Ukuziqaphela kwenza umsebe wokukhanya ungene ngaphakathi kithi.

Noma yimuphi "uMina" uziveza ekhanda ngendlela ethile, enhliziyweni ngenye indlela, nasebulileni ngenye indlela.

Sidinga ukubheka "uMina" esithola ebambekile ngesikhathi esithile, kuyadingeka ukumbona kulezi zikhungo ezintathu zomzimba wethu ngamunye.

Ngokuphathelene nabanye abantu uma siqaphile futhi sihlale siphapheme njengomqaphi ngenkathi yempi, siyazithola.

Uyayikhumbula yini ukuthi ukuziqhenya kwakho kwalinyazwa ngasiphi isikhathi? Ukuzazisa kwakho? Yini eyakucasula kakhulu ngalolo suku? Kungani ube nalo lelo dumo? Yini imbangela yayo eyimfihlo? Funda lokhu, qaphela ikhanda lakho, inhliziyo yakho nobulili bakho...

Impilo ewusizo yisikole esimangalisayo; ekuxhumaneni singathola labo "Mina" esibathwele ngaphakathi kwethu.

Kudingeka sizazi thina ngaphambi kokuba sikwazi ukwazi abanye. Kubalulekile ngokuphuthumayo ukufunda ukubona umbono womunye umuntu.

Uma sizibeka ezikhundleni zabanye, sithola ukuthi amaphutha engqondo esiwabeka kwabanye, sinazo ngaphakathi kakhulu.

Ukuthanda umakhelwane wakho kubalulekile, kodwa umuntu akakwazi ukuthanda abanye uma engafundi kuqala ukuzibeka esimweni somunye umuntu emsebenzini we-esoteric.

Ubunzima buzoqhubeka bukhona ebusweni bomhlaba, inqobo nje uma singazange sifunde ukuzibeka endaweni yabanye.

Kodwa uma umuntu engenaso isibindi sokuzibona yena, angazibeka kanjani endaweni yabanye?

Kungani kufanele sibone kuphela ingxenye embi yabanye abantu?

Ukungathandi okuzenzakalelayo ngomunye umuntu esihlangana naye okokuqala, kubonisa ukuthi asikwazi ukuzibeka endaweni yomakhelwane wethu, ukuthi asimthandi umakhelwane wethu, ukuthi sinokuqonda okuconsa kakhulu.

Siyamzonda kakhulu umuntu othile? Ngani? Mhlawumbe uyaphuza? Asizihlolisise... Siyaqiniseka ngobuhle bethu? Siyaqiniseka ukuthi asithwele yini ngaphakathi kwethu "uMina" wokudakwa?

Ingabe uqinisekile ukuthi ngaphakathi kwakho akukho nkanuko nokuziphatha okubi okubonayo kulowo wesifazane?

Kungcono ukuthi uzibheke ngasese futhi ngokuzindla okujulile uthathe indawo yalowo wesifazane omzondayo.

Kubalulekile ngokuphuthumayo ukwazisa umsebenzi we-esoteric we-Gnostic, kubalulekile ukuwuqonda futhi uyuqonde uma ngempela silangazelela ushintsho olukhulu.

Kuyadingeka ukwazi ukuthanda abanye bethu, ukutadisha i-Gnosis futhi sisebenzise le mfundiso kubo bonke abantu, ngaphandle kwalokho sizoziwela ebugandeni.

Uma umuntu ezinikela emsebenzini we-esoteric kuye, kodwa enganikezi abanye imfundiso, intuthuko yakhe eseduze iba nzima kakhulu ngenxa yokuntuleka kothando ngomakhelwane.

"Onikelayo, uyamukela futhi lapho enikeza kakhulu, kulapho ezokwamukela khona kakhulu, kodwa kulowo onganikezeli lutho ngisho nalokho anakho kuzothathwa kuye". Lowo nguMthetho.
