Η άσκηση της αληθινής ΕΞΟΥΣΙΑΣ μπορεί να ασκηθεί μόνο από εκείνους που κατέχουν το συνειδητό ΟΝ.

Εκείνοι που δεν κατέχουν ακόμη το ΣΥΝΕΙΔΗΤΟ ΟΝ, εκείνοι που δεν είναι ακόμη ΑΥΤΟ-ΣΥΝΕΙΔΗΤΟΙ, συνήθως ΚΑΤΑΧΡΩΝΤΑΙ ΤΗΝ ΕΞΟΥΣΙΑ και προκαλούν μεγάλη ζημιά.

Οι ΔΑΣΚΑΛΟΙ πρέπει να μάθουν να διατάζουν και οι μαθητές πρέπει να μάθουν να υπακούν.

Εκείνοι οι ΨΥΧΟΛΟΓΟΙ που τάσσονται κατά της υπακοής κάνουν λάθος, γιατί κανείς δεν μπορεί να διατάξει συνειδητά αν πρώτα δεν έχει μάθει να υπακούει.

Πρέπει να ξέρουμε πώς να διατάζουμε ΣΥΝΕΙΔΗΤΑ και πρέπει να ξέρουμε πώς να υπακούμε συνειδητά.
