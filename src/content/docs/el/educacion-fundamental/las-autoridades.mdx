Εκείνοι που δεν κατέχουν ακόμη το ΣΥΝΕΙΔΗΤΟ ΟΝ, εκείνοι που δεν είναι ακόμη ΑΥΤΟ-ΣΥΝΕΙΔΗΤΟΙ, συνήθως ΚΑΤΑΧΡΩΝΤΑΙ ΤΗΝ ΕΞΟΥΣΙΑ και προκαλούν μεγάλη ζημιά.

Οι ΔΑΣΚΑΛΟΙ πρέπει να μάθουν να διατάζουν και οι μαθητές πρέπει να μάθουν να υπακούν.

Εκείνοι οι ΨΥΧΟΛΟΓΟΙ που τάσσονται κατά της υπακοής είναι στην πραγματικότητα πολύ λανθασμένοι, επειδή κανείς δεν μπορεί να διατάξει συνειδητά αν δεν έχει μάθει πρώτα να υπακούει.

Πρέπει να ξέρει κανείς να διατάζει ΣΥΝΕΙΔΗΤΑ και πρέπει να ξέρει κανείς να υπακούει συνειδητά.
