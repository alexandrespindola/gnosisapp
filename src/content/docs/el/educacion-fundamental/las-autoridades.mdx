Η άσκηση της αληθινής ΕΞΟΥΣΙΑΣ μπορεί να ασκηθεί μόνο από εκείνους που κατέχουν το συνειδητό ΟΝ.

Εκείνοι που δεν κατέχουν ακόμα το ΣΥΝΕΙΔΗΤΟ ΟΝ, εκείνοι που δεν είναι ακόμα ΑΥΤΟ-ΣΥΝΕΙΔΗΤΟΙ, συνήθως ΚΑΤΑΧΡΩΝΤΑΙ ΤΗΝ ΕΞΟΥΣΙΑ και προκαλούν πολλή ζημιά.

Οι ΔΑΣΚΑΛΟΙ πρέπει να μάθουν να δίνουν εντολές και οι μαθητές πρέπει να μάθουν να υπακούουν.

Εκείνοι οι ΨΥΧΟΛΟΓΟΙ που τάσσονται κατά της υπακοής είναι στην πραγματικότητα πολύ λανθασμένοι, επειδή κανείς δεν μπορεί να δώσει εντολές συνειδητά εάν δεν έχει μάθει πρώτα να υπακούει.

Πρέπει να ξέρουμε να δίνουμε εντολές ΣΥΝΕΙΔΗΤΑ και πρέπει να ξέρουμε να υπακούμε συνειδητά.
