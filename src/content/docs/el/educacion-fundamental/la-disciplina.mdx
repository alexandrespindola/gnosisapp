Οι δάσκαλοι σχολείων, κολλεγίων και πανεπιστημίων δίνουν μεγάλη σημασία στην πειθαρχία και εμείς πρέπει να τη μελετήσουμε προσεκτικά σε αυτό το κεφάλαιο.

Όλοι όσοι έχουμε περάσει από σχολεία, κολέγια, πανεπιστήμια κ.λπ. γνωρίζουμε πολύ καλά τι είναι οι πειθαρχίες, οι κανόνες, οι βέργες, οι επιπλήξεις κ.λπ., κ.λπ., κ.λπ.

Πειθαρχία είναι αυτό που ονομάζεται ΚΑΛΛΙΕΡΓΕΙΑ ΤΗΣ ΑΝΤΙΣΤΑΣΗΣ. Στους δασκάλους αρέσει να καλλιεργούν την ΑΝΤΙΣΤΑΣΗ.

Μας διδάσκουν να αντιστεκόμαστε, να υψώνουμε κάτι ενάντια σε κάτι άλλο. Μας διδάσκουν να αντιστεκόμαστε στους πειρασμούς της σάρκας και μαστιγωνόμαστε και κάνουμε μετάνοιες για να αντισταθούμε.

Μας διδάσκουν να ΑΝΤΙΣΤΕΚΟΜΑΣΤΕ στους πειρασμούς που φέρνει η τεμπελιά, τους πειρασμούς να μην μελετάμε, να μην πηγαίνουμε στο σχολείο, να παίζουμε, να γελάμε, να κοροϊδεύουμε τους δασκάλους, να παραβιάζουμε τους κανονισμούς κ.λπ., κ.λπ., κ.λπ.

Οι δάσκαλοι έχουν την εσφαλμένη αντίληψη ότι μέσω της πειθαρχίας μπορούμε να κατανοήσουμε την ανάγκη να σεβόμαστε την τάξη του σχολείου, την ανάγκη να μελετάμε, να διατηρούμε την ψυχραιμία μας μπροστά στους δασκάλους, να συμπεριφερόμαστε καλά στους συμμαθητές μας κ.λπ., κ.λπ., κ.λπ.

Υπάρχει στους ανθρώπους η εσφαλμένη αντίληψη ότι όσο περισσότερο αντιστεκόμαστε, όσο περισσότερο απορρίπτουμε, τόσο περισσότερο γινόμαστε κατανοητοί, ελεύθεροι, πλήρεις, νικητές.

Δεν θέλουν να συνειδητοποιήσουν ότι όσο περισσότερο αγωνιζόμαστε ενάντια σε κάτι, όσο περισσότερο αντιστεκόμαστε, όσο περισσότερο το απορρίπτουμε, τόσο λιγότερη είναι η ΚΑΤΑΝΟΗΣΗ.

Πολλοί είναι αυτοί που πειθαρχούν τον εαυτό τους ενάντια στη ζήλια και τελικά πιστεύουν σταθερά ότι την έχουν ήδη εξαλείψει, αλλά επειδή δεν την κατάλαβαν, είναι σαφές ότι αυτή εμφανίζεται ξανά στη σκηνή ακριβώς όταν την πιστεύαμε ήδη καλά νεκρή.

Μόνο με πλήρη απουσία πειθαρχιών, μόνο σε αυθεντική ελευθερία, αναδύεται στο νου η φλογερή φλόγα της ΚΑΤΑΝΟΗΣΗΣ.

Η ΔΗΜΙΟΥΡΓΙΚΗ ΕΛΕΥΘΕΡΙΑ δεν μπορεί ποτέ να υπάρξει σε ένα ΣΚΕΛΕΤΟ. Χρειαζόμαστε ελευθερία για να ΚΑΤΑΝΟΗΣΟΥΜΕ τις ΨΥΧΟΛΟΓΙΚΕΣ μας ελλείψεις με ΟΛΟΚΛΗΡΩΜΕΝΟ τρόπο.

Χρειαζόμαστε ΕΠΕΙΓΟΝΤΩΣ να γκρεμίσουμε τείχη και να σπάσουμε ατσάλινες αλυσίδες, για να είμαστε ελεύθεροι.

Πρέπει να βιώσουμε οι ίδιοι όλα όσα οι Δάσκαλοί μας στο Σχολείο και οι Γονείς μας μας έχουν πει ότι είναι καλά και χρήσιμα. Δεν αρκεί να μάθουμε απέξω και να μιμούμαστε. Πρέπει να κατανοήσουμε.

Όλη η προσπάθεια των Δασκάλων πρέπει να κατευθυνθεί στη συνείδηση των μαθητών. Πρέπει να προσπαθήσουν να μπουν στο δρόμο της ΚΑΤΑΝΟΗΣΗΣ.

Δεν αρκεί να πούμε στους μαθητές ότι πρέπει να είναι αυτό ή εκείνο, είναι απαραίτητο οι μαθητές να μάθουν να είναι ελεύθεροι ώστε να μπορούν οι ίδιοι να εξετάσουν, να μελετήσουν, να αναλύσουν όλες τις αξίες, όλα τα πράγματα που οι άνθρωποι έχουν πει ότι είναι ωφέλιμα, χρήσιμα, ευγενή και όχι απλώς να τα αποδεχτούν και να τα μιμηθούν.

Οι άνθρωποι δεν θέλουν να ανακαλύψουν μόνοι τους, έχουν κλειστά, ανόητα μυαλά, μυαλά που δεν θέλουν να ερευνήσουν, μηχανιστικά μυαλά που ποτέ δεν ερευνούν και που μόνο ΜΙΜΟΥΝΤΑΙ.

Είναι απαραίτητο, είναι επείγον, είναι απαραίτητο οι μαθητές από την πιο τρυφερή τους ηλικία μέχρι τη στιγμή που εγκαταλείπουν τις ΑΙΘΟΥΣΕΣ να απολαμβάνουν πραγματική ελευθερία για να ανακαλύψουν μόνοι τους, να ερευνήσουν, να κατανοήσουν και να μην περιορίζονται από τα άθλια τείχη των απαγορεύσεων, των επιπλήξεων και των πειθαρχιών.

Εάν στους μαθητές τους λένε τι πρέπει και τι δεν πρέπει να κάνουν και δεν τους επιτρέπεται να ΚΑΤΑΝΟΗΣΟΥΝ και να βιώσουν, ΠΟΥ λοιπόν είναι η ευφυΐα τους; ΠΟΙΑ είναι η ευκαιρία που έχει δοθεί στην ευφυΐα;

Τι χρησιμεύει λοιπόν να δίνουμε εξετάσεις, να ντυνόμαστε πολύ καλά, να έχουμε πολλούς φίλους αν δεν είμαστε έξυπνοι;

Η ευφυΐα έρχεται μόνο σε εμάς όταν είμαστε αληθινά ελεύθεροι να ερευνήσουμε μόνοι μας, να κατανοήσουμε, να αναλύσουμε χωρίς το φόβο της επίπληξης και χωρίς τη βέργα των Πειθαρχιών.

Οι φοβισμένοι, τρομαγμένοι, υποταγμένοι σε τρομερές πειθαρχίες μαθητές δεν θα μπορέσουν ποτέ να ΜΑΘΟΥΝ. Δεν θα μπορέσουν ποτέ να είναι έξυπνοι.

Σήμερα, στους Γονείς και στους Δασκάλους, το μόνο που τους ενδιαφέρει είναι οι μαθητές να κάνουν μια καριέρα, να γίνουν γιατροί, δικηγόροι, μηχανικοί, υπάλληλοι γραφείου, δηλαδή ζωντανοί αυτόματοι και μετά να παντρευτούν και να γίνουν επίσης ΜΗΧΑΝΕΣ ΚΑΤΑΣΚΕΥΗΣ ΜΩΡΩΝ και αυτό είναι όλο.

Όταν τα παιδιά θέλουν να κάνουν κάτι καινούργιο, κάτι διαφορετικό, όταν αισθάνονται την ανάγκη να βγουν από αυτό το σκελετό, τις προκαταλήψεις, τις ξεπερασμένες συνήθειες, τις πειθαρχίες, τις οικογενειακές ή εθνικές παραδόσεις κ.λπ., τότε οι γονείς σφίγγουν περισσότερο τις αλυσίδες της φυλακής και λένε στο αγόρι ή στο κορίτσι: Μην το κάνεις αυτό! Δεν είμαστε διατεθειμένοι να σε υποστηρίξουμε σε αυτό, αυτά τα πράγματα είναι τρέλες κ.λπ., κ.λπ., κ.λπ.

Η ΒΑΣΙΚΗ ΕΚΠΑΙΔΕΥΣΗ διδάσκει να συμφιλιώνουμε την ΤΑΞΗ με την ΕΛΕΥΘΕΡΙΑ.

Η ΕΛΕΥΘΕΡΙΑ ΚΑΙ Η ΤΑΞΗ σοφά συνδυασμένες αποτελούν τη ΒΑΣΗ της ΒΑΣΙΚΗΣ ΕΚΠΑΙΔΕΥΣΗΣ.

Οι ΜΑΘΗΤΕΣ πρέπει να απολαμβάνουν τέλεια ελευθερία για να ανακαλύψουν μόνοι τους, να ΕΡΕΥΝΗΣΟΥΝ, να ΑΝΑΚΑΛΥΨΟΥΝ τι πραγματικά, τι αληθινά είναι στον ΕΑΥΤΟ ΤΟΥΣ και τι μπορούν να κάνουν στη ζωή.

Η ΠΕΙΘΑΡΧΙΑ καταστρέφει την ανθρώπινη ΕΥΑΙΣΘΗΣΙΑ και αυτό έχει ήδη αποδειχθεί πλήρως από την ΠΑΡΑΤΗΡΗΣΗ και την ΕΜΠΕΙΡΙΑ.

Λόγω τόσων πειθαρχιών και κανονισμών, οι άνθρωποι αυτής της εποχής έχουν χάσει εντελώς την ΕΥΑΙΣΘΗΣΙΑ και έχουν γίνει σκληροί και αδίστακτοι.

Για να είμαστε αληθινά ελεύθεροι, πρέπει να είμαστε πολύ ευαίσθητοι και ανθρωπιστές.

Αλλά δυστυχώς δεν τους διδάσκουν να ΚΑΤΑΝΟΗΣΟΥΝ ΠΡΑΓΜΑΤΙΚΑ τι είναι η ΣΥΝΕΙΔΗΤΗ ΠΡΟΣΟΧΗ.

Η δημιουργική ενέργεια είναι ο πιο λεπτός τύπος δύναμης που κατασκευάζεται από την ΟΡΓΑΝΙΚΗ ΜΗΧΑΝΗ.

Εμείς τρώμε και πίνουμε και όλες οι διαδικασίες της πέψης είναι ουσιαστικά διαδικασίες λεπτομέρειας κατά τις οποίες οι χονδροειδείς ύλες μετατρέπονται σε χρήσιμες ύλες και δυνάμεις.

Η δημιουργική ενέργεια είναι: ο τύπος ΥΛΗΣ και ΔΥΝΑΜΗΣ πιο λεπτός που επεξεργάζεται ο οργανισμός.

Εάν ξέρουμε να δίνουμε ΣΥΝΕΙΔΗΤΗ ΠΡΟΣΟΧΗ μπορούμε να εξοικονομήσουμε δημιουργική ενέργεια.

Παρατηρήστε προσεκτικά έναν ΚΙΝΗΜΑΤΟΓΡΑΦΙΣΤΗ. Βρίσκεται κοιμισμένος, τα αγνοεί όλα, αγνοεί τον εαυτό του, είναι κενός, μοιάζει με υπνοβάτη, ονειρεύεται την ταινία που βλέπει, με τον ήρωα της ταινίας.

Όταν ο ΜΑΘΗΤΗΣ κάνει το λάθος να ΤΑΥΤΙΣΤΕΙ με τους ανθρώπους, τα πράγματα, τις ιδέες, όταν κάνει το λάθος να ξεχάσει τον εαυτό του, τότε πέφτει στη γοητεία και στον ύπνο.

Η πειθαρχία δεν διδάσκει στους μαθητές να δίνουν ΣΥΝΕΙΔΗΤΗ ΠΡΟΣΟΧΗ. Η πειθαρχία είναι μια πραγματική φυλακή για το νου.

Οι μαθητές πρέπει να μάθουν να χειρίζονται την ΣΥΝΕΙΔΗΤΗ ΠΡΟΣΟΧΗ από τα ίδια θρανία του σχολείου ώστε αργότερα στην πράξη της ζωής, έξω από το σχολείο, να μην κάνουν το λάθος να ξεχάσουν τον εαυτό τους.

Ο άνθρωπος που ξεχνά τον εαυτό του μπροστά σε έναν υβριστή, ταυτίζεται μαζί του, γοητεύεται, πέφτει στον ύπνο της ασυνειδησίας και τότε πληγώνει ή σκοτώνει και πηγαίνει αναπόφευκτα στη φυλακή.

Εκείνος που δεν αφήνεται να ΓΟΗΤΕΥΤΕΙ με τον υβριστή, εκείνος που δεν ταυτίζεται μαζί του, εκείνος που δεν ξεχνά τον εαυτό του, εκείνος που ξέρει να δίνει ΣΥΝΕΙΔΗΤΗ ΠΡΟΣΟΧΗ, θα ήταν ανίκανος να δώσει αξία στα λόγια του υβριστή, ή να τον πληγώσει ή να τον σκοτώσει.

Όλα τα λάθη που κάνει ο άνθρωπος στη ζωή οφείλονται στο ότι ξεχνά τον εαυτό του, ταυτίζεται, γοητεύεται και πέφτει στον ύπνο.

Καλύτερα θα ήταν για τη νεολαία, για όλους τους μαθητές, να τους διδάσκονταν την ΑΦΥΠΝΙΣΗ της ΣΥΝΕΙΔΗΣΗΣ αντί να τους υποδουλώνουν με τόσες παράλογες πειθαρχίες.
