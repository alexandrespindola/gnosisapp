Στη ζωή το μόνο σημαντικό είναι η ριζική, ολική και οριστική αλλαγή· τα υπόλοιπα ειλικρινά δεν έχουν την παραμικρή σημασία.

Ο διαλογισμός είναι θεμελιώδης όταν ειλικρινά θέλουμε μια τέτοια αλλαγή.

Σε καμία περίπτωση δεν επιθυμούμε έναν διαλογισμό ασήμαντο, επιφανειακό και μάταιο.

Πρέπει να σοβαρευτούμε και να αφήσουμε στην άκρη τόσες ανοησίες που αφθονούν εκεί έξω στον ψευδοεσωτερισμό και τον φθηνό ψευδο-αποκρυφισμό.

Πρέπει να ξέρουμε να είμαστε σοβαροί, πρέπει να ξέρουμε να αλλάξουμε αν πραγματικά δεν θέλουμε να αποτύχουμε στην εσωτερική εργασία.

Όποιος δεν ξέρει να διαλογίζεται, ο επιφανειακός, ο αμαθής, ποτέ δεν θα μπορέσει να διαλύσει το Εγώ· θα είναι πάντα ένα ανίσχυρο ξύλο στην άγρια θάλασσα της ζωής.

Ένα ελάττωμα που ανακαλύπτεται στο πεδίο της πρακτικής ζωής, πρέπει να γίνει κατανοητό σε βάθος μέσω της τεχνικής του διαλογισμού.

Το διδακτικό υλικό για τον διαλογισμό βρίσκεται ακριβώς στα διάφορα γεγονότα ή καθημερινές περιστάσεις της πρακτικής ζωής, αυτό είναι αναμφισβήτητο.

Οι άνθρωποι πάντα διαμαρτύρονται για τα δυσάρεστα γεγονότα, ποτέ δεν ξέρουν να δουν τη χρησιμότητα αυτών των γεγονότων.

Εμείς αντί να διαμαρτυρόμαστε για τις δυσάρεστες περιστάσεις, πρέπει να εξάγουμε από αυτές, μέσω του διαλογισμού, τα χρήσιμα στοιχεία για την ψυχική μας ανάπτυξη.

Ο ενδελεχής διαλογισμός πάνω σε μια τέτοια ευχάριστη ή δυσάρεστη περίσταση, μας επιτρέπει να νιώσουμε μέσα μας τη γεύση, το αποτέλεσμα.

Είναι απαραίτητο να κάνουμε μια πλήρη ψυχολογική διάκριση μεταξύ της γεύσης εργασίας και της γεύσης ζωής.

Σε κάθε περίπτωση, για να νιώσουμε μέσα μας τη γεύση εργασίας, απαιτείται ολική αναστροφή της στάσης με την οποία κανονικά αντιμετωπίζουμε τις περιστάσεις της ύπαρξης.

Κανείς δεν θα μπορούσε να απολαύσει τη γεύση εργασίας όσο διαπράττει το λάθος να ταυτίζεται με τα διάφορα γεγονότα.

Βεβαίως, η ταύτιση εμποδίζει την ορθή ψυχολογική εκτίμηση των γεγονότων.

Όταν κάποιος ταυτίζεται με ένα τέτοιο γεγονός, σε καμία περίπτωση δεν καταφέρνει να εξάγει από αυτό τα χρήσιμα στοιχεία για την αυτο-ανακάλυψη και την εσωτερική ανάπτυξη της συνείδησης.

Ο εργάτης Εσωτεριστής που επιστρέφει στην ταύτιση αφού έχασε την επαγρύπνηση, ξαναβιώνει τη γεύση ζωής αντί για τη γεύση εργασίας.

Αυτό δείχνει ότι η ψυχολογική στάση που είχε αντιστραφεί πριν, έχει επιστρέψει στην κατάσταση της ταύτισης.

Οποιαδήποτε δυσάρεστη περίσταση πρέπει να ανακατασκευαστεί μέσω της συνειδητής φαντασίας μέσω της τεχνικής του διαλογισμού.

Η ανακατασκευή οποιασδήποτε σκηνής μας επιτρέπει να επαληθεύσουμε μόνοι μας και άμεσα την παρέμβαση διαφόρων εγώ που συμμετέχουν σε αυτήν.

Παραδείγματα: Μια σκηνή ερωτικής ζήλιας· σε αυτήν παρεμβαίνουν εγώ οργής, ζήλιας και ακόμη και μίσους.

Η κατανόηση κάθε ενός από αυτά τα εγώ, καθενός από αυτούς τους παράγοντες, συνεπάγεται στην πραγματικότητα βαθύ στοχασμό, συγκέντρωση, διαλογισμό.

Η έντονη τάση να κατηγορούμε τους άλλους είναι εμπόδιο, παρεμπόδιση για την κατανόηση των δικών μας λαθών.

Δυστυχώς, είναι πολύ δύσκολο να καταστρέψουμε μέσα μας την τάση να κατηγορούμε τους άλλους.

Στο όνομα της αλήθειας πρέπει να πούμε ότι εμείς είμαστε οι μόνοι ένοχοι για τις διάφορες δυσάρεστες περιστάσεις της ζωής.

Τα διάφορα ευχάριστα ή δυσάρεστα γεγονότα υπάρχουν με εμάς ή χωρίς εμάς και επαναλαμβάνονται μηχανικά συνεχώς.

Ξεκινώντας από αυτή την αρχή, κανένα πρόβλημα δεν μπορεί να έχει μια οριστική λύση.

Τα προβλήματα είναι της ζωής και αν υπήρχε μια οριστική λύση η ζωή δεν θα ήταν ζωή αλλά θάνατος.

Επομένως, μπορεί να υπάρξει τροποποίηση των περιστάσεων και των προβλημάτων, αλλά ποτέ δεν θα σταματήσουν να επαναλαμβάνονται και ποτέ δεν θα έχουν μια οριστική λύση.

Η ζωή είναι ένας τροχός που γυρίζει μηχανικά με όλες τις ευχάριστες και δυσάρεστες περιστάσεις, πάντα επαναλαμβανόμενος.

Δεν μπορούμε να σταματήσουμε τον τροχό, οι καλές ή κακές περιστάσεις επεξεργάζονται πάντα μηχανικά, μπορούμε μόνο να αλλάξουμε τη στάση μας απέναντι στα γεγονότα της ζωής.

Καθώς μαθαίνουμε να εξάγουμε το υλικό για τον διαλογισμό μέσα από τις ίδιες τις περιστάσεις της ύπαρξης, θα ανακαλύπτουμε τον εαυτό μας.

Σε οποιαδήποτε ευχάριστη ή δυσάρεστη περίσταση υπάρχουν διάφορα εγώ που πρέπει να γίνουν κατανοητά πλήρως με την τεχνική του διαλογισμού.

Αυτό σημαίνει ότι οποιαδήποτε ομάδα εγώ που παρεμβαίνει σε ένα τέτοιο δράμα, κωμωδία ή τραγωδία της πρακτικής ζωής, αφού κατανοηθεί πλήρως θα πρέπει να εξαλειφθεί μέσω της δύναμης της Θείας Μητέρας Κουνταλίνη.

Καθώς κάνουμε χρήση της αίσθησης της ψυχολογικής παρατήρησης, αυτή θα αναπτύσσεται επίσης θαυμάσια. Τότε θα μπορούμε να αντιληφθούμε εσωτερικά όχι μόνο τα εγώ πριν εργαστούμε πάνω τους, αλλά και κατά τη διάρκεια όλης της εργασίας.

Όταν αυτά τα εγώ αποκεφαλίζονται και αποσυντίθενται, νιώθουμε μεγάλη ανακούφιση, μεγάλη ευδαιμονία.
