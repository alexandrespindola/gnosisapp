Ένα άτομο είναι αυτό που είναι η ζωή του. Αυτό που συνεχίζεται πέρα από τον θάνατο, είναι η ζωή. Αυτή είναι η σημασία του βιβλίου της ζωής που ανοίγει με τον θάνατο.

Κοιτάζοντας αυτό το θέμα από μια αυστηρά ψυχολογική σκοπιά, μια οποιαδήποτε ημέρα της ζωής μας, είναι πραγματικά ένα μικρό αντίγραφο του συνόλου της ζωής.

Από όλα αυτά μπορούμε να συμπεράνουμε τα εξής: Εάν ένας άνθρωπος δεν εργαστεί πάνω στον εαυτό του σήμερα, δεν θα αλλάξει ποτέ.

Όταν ισχυρίζεται ότι θέλει να εργαστεί πάνω στον εαυτό του, και δεν εργάζεται σήμερα αναβάλλοντας για αύριο, μια τέτοια δήλωση θα είναι ένα απλό σχέδιο και τίποτα περισσότερο, γιατί σε αυτό το σήμερα βρίσκεται το αντίγραφο ολόκληρης της ζωής μας.

Υπάρχει εκεί έξω ένα κοινότοπο ρητό που λέει: "Μην αφήνετε για αύριο αυτό που μπορείτε να κάνετε σήμερα".

Εάν ένας άνθρωπος λέει: "Θα εργαστώ πάνω στον εαυτό μου, αύριο", δεν θα εργαστεί ποτέ πάνω στον εαυτό του, γιατί πάντα θα υπάρχει ένα αύριο.

Αυτό είναι πολύ παρόμοιο με μια συγκεκριμένη ειδοποίηση, ανακοίνωση ή πινακίδα που ορισμένοι έμποροι βάζουν στα καταστήματά τους: "ΣΗΜΕΡΑ ΔΕΝ ΔΑΝΕΙΖΩ, ΑΥΡΙΟ ΝΑΙ".

Όταν κάποιος που έχει ανάγκη φτάνει για να ζητήσει πίστωση, πέφτει πάνω στην τρομερή ειδοποίηση, και αν επιστρέψει την άλλη μέρα, βρίσκει ξανά την άτυχη ανακοίνωση ή πινακίδα.

Αυτό είναι που ονομάζεται στην ψυχολογία η "ασθένεια του αύριο". Όσο ένας άνθρωπος λέει "αύριο", δεν θα αλλάξει ποτέ.

Χρειαζόμαστε με επείγουσα, ανυπέρβλητη ανάγκη, να εργαστούμε πάνω στον εαυτό μας σήμερα, να μην ονειρευόμαστε τεμπέλικα ένα μέλλον ή μια εξαιρετική ευκαιρία.

Αυτοί που λένε: "Θα πάω πρώτα να κάνω αυτό ή εκείνο και μετά θα εργαστώ". Δεν θα εργαστούν ποτέ πάνω στον εαυτό τους, αυτοί είναι οι κάτοικοι της γης που αναφέρονται στις Αγίες Γραφές.

Γνώρισα έναν ισχυρό γαιοκτήμονα που έλεγε: "Πρέπει πρώτα να στρογγυλοκαθίσω και μετά να εργαστώ πάνω στον Εαυτό Μου".

Όταν αρρώστησε θανάσιμα τον επισκέφθηκα, τότε του έκανα την εξής ερώτηση: "Εξακολουθείτε να θέλετε να στρογγυλοκαθίσετε;"

"Λυπάμαι πραγματικά που έχασα τον χρόνο μου", μου απάντησε. Μέρες αργότερα πέθανε, αφού αναγνώρισε το λάθος του.

Εκείνος ο άνθρωπος είχε πολλές γαίες, αλλά ήθελε να ιδιοποιηθεί τις γειτονικές ιδιοκτησίες, να "στρογγυλοκαθίσει", έτσι ώστε η περιουσία του να οριοθετείται ακριβώς από τέσσερις δρόμους.

"Αρκετή για κάθε μέρα είναι η έγνοιά της!", είπε ο Μέγας KABIR ΙΗΣΟΥΣ. Να αυτοπαρατηρούμαστε σήμερα, όσον αφορά την ημέρα που επαναλαμβάνεται πάντα, μικρογραφία ολόκληρης της ζωής μας.

Όταν ένας άνθρωπος αρχίζει να εργάζεται πάνω στον εαυτό του, σήμερα, όταν παρατηρεί τις δυσαρέσκειές του και τις λύπες του, βαδίζει στον δρόμο της επιτυχίας.

Δεν θα ήταν δυνατό να εξαλείψουμε αυτό που δεν γνωρίζουμε. Πρέπει πρώτα να παρατηρήσουμε τα δικά μας λάθη.

Πρέπει όχι μόνο να γνωρίζουμε την ημέρα μας, αλλά και τη σχέση με αυτήν. Υπάρχει μια συγκεκριμένη συνηθισμένη ημέρα που κάθε άτομο βιώνει άμεσα, εκτός από τα ασυνήθιστα, ασυνήθιστα γεγονότα.

Είναι ενδιαφέρον να παρατηρήσουμε την καθημερινή επανάληψη, την επανάληψη λέξεων και γεγονότων, για κάθε άτομο, κ.λπ.

Αυτή η επανάληψη ή επανάληψη γεγονότων και λέξεων, αξίζει να μελετηθεί, μας οδηγεί στην αυτογνωσία.
