Δεν θα μπορούσαμε με κανέναν τρόπο να αρνηθούμε τον Νόμο της Επανάληψης που επεξεργάζεται σε κάθε στιγμή της ζωής μας.

Σίγουρα σε κάθε ημέρα της ύπαρξής μας, υπάρχει επανάληψη γεγονότων, καταστάσεων συνείδησης, λέξεων, επιθυμιών, σκέψεων, βουλήσεων, κ.λπ.

Είναι προφανές ότι όταν κάποιος δεν αυτο-παρατηρείται, δεν μπορεί να συνειδητοποιήσει αυτή την αδιάκοπη καθημερινή επανάληψη.

Είναι φανερό ότι όποιος δεν αισθάνεται κανένα ενδιαφέρον να παρατηρήσει τον εαυτό του, ούτε επιθυμεί να εργαστεί για να επιτύχει έναν πραγματικό ριζικό μετασχηματισμό.

Επιπλέον, υπάρχουν άνθρωποι που θέλουν να μεταμορφωθούν χωρίς να εργαστούν πάνω στους εαυτούς τους.

Δεν αρνούμαστε το γεγονός ότι ο καθένας έχει δικαίωμα στην πραγματική ευτυχία του πνεύματος, αλλά είναι επίσης αλήθεια ότι η ευτυχία θα ήταν κάτι περισσότερο από αδύνατη αν δεν εργαζόμασταν πάνω στους εαυτούς μας.

Κάποιος μπορεί να αλλάξει εσωτερικά, όταν πραγματικά καταφέρει να τροποποιήσει τις αντιδράσεις του στα διάφορα γεγονότα που του συμβαίνουν καθημερινά.

Ωστόσο, δεν θα μπορούσαμε να τροποποιήσουμε τον τρόπο που αντιδρούμε στα γεγονότα της πρακτικής ζωής, αν δεν εργαζόμασταν σοβαρά πάνω στους εαυτούς μας.

Πρέπει να αλλάξουμε τον τρόπο που σκεφτόμαστε, να είμαστε λιγότερο αμελείς, να γίνουμε πιο σοβαροί και να πάρουμε τη ζωή με διαφορετικό τρόπο, στην πραγματική και πρακτική της έννοια.

Ωστόσο, αν συνεχίσουμε έτσι όπως είμαστε, συμπεριφερόμενοι με τον ίδιο τρόπο κάθε μέρα, επαναλαμβάνοντας τα ίδια λάθη, με την ίδια αμέλεια όπως πάντα, οποιαδήποτε πιθανότητα αλλαγής θα εξαλειφθεί στην πραγματικότητα.

Αν κάποιος θέλει πραγματικά να γνωρίσει τον εαυτό του, πρέπει να αρχίσει παρατηρώντας τη δική του συμπεριφορά, μπροστά στα γεγονότα οποιασδήποτε ημέρας της ζωής.

Δεν θέλουμε να πούμε με αυτό ότι δεν πρέπει κάποιος να παρατηρεί τον εαυτό του καθημερινά, απλώς θέλουμε να επιβεβαιώσουμε ότι πρέπει να αρχίσει παρατηρώντας μια πρώτη ημέρα.

Σε όλα πρέπει να υπάρχει μια αρχή, και το να αρχίσουμε να παρατηρούμε τη συμπεριφορά μας σε οποιαδήποτε ημέρα της ζωής μας, είναι μια καλή αρχή.

Το να παρατηρούμε τις μηχανικές μας αντιδράσεις σε όλες αυτές τις μικρές λεπτομέρειες της κρεβατοκάμαρας, του σπιτιού, της τραπεζαρίας, του σπιτιού, του δρόμου, της δουλειάς, κ.λπ., κ.λπ., κ.λπ., αυτό που λέει, αισθάνεται και σκέφτεται κάποιος, είναι σίγουρα το πιο ενδεδειγμένο.

Το σημαντικό είναι να δούμε μετά πώς ή με ποιον τρόπο μπορεί κάποιος να αλλάξει αυτές τις αντιδράσεις. Ωστόσο, αν πιστεύουμε ότι είμαστε καλοί άνθρωποι, ότι ποτέ δεν συμπεριφερόμαστε ασυνείδητα και λανθασμένα, δεν θα αλλάξουμε ποτέ.

Πρώτα απ 'όλα πρέπει να κατανοήσουμε ότι είμαστε άνθρωποι-μηχανές, απλές μαριονέτες που ελέγχονται από μυστικούς πράκτορες, από κρυμμένα Εγώ.

Μέσα στην προσωπικότητά μας ζουν πολλοί άνθρωποι, ποτέ δεν είμαστε ίδιοι. Μερικές φορές εκδηλώνεται μέσα μας ένα άτομο μικρόψυχο, άλλες φορές ένα άτομο ευερέθιστο, σε οποιαδήποτε άλλη στιγμή ένα άτομο υπέροχο, καλοπροαίρετο, αργότερα ένα άτομο σκανδαλώδες ή συκοφάντης, μετά ένας άγιος, μετά ένας απατεώνας, κ.λπ.

Έχουμε ανθρώπους κάθε είδους μέσα στον καθένα μας, Εγώ κάθε είδους. Η προσωπικότητά μας δεν είναι παρά μια μαριονέτα, μια κούκλα που μιλάει, κάτι μηχανικό.

Ας αρχίσουμε να συμπεριφερόμαστε συνειδητά για ένα μικρό μέρος της ημέρας. Πρέπει να σταματήσουμε να είμαστε απλές μηχανές, έστω και για λίγα σύντομα λεπτά καθημερινά, αυτό θα επηρεάσει αποφασιστικά την ύπαρξή μας.

Όταν Αυτο-Παρατηρούμε και δεν κάνουμε αυτό που θέλει το ένα ή το άλλο Εγώ, είναι σαφές ότι αρχίζουμε να σταματάμε να είμαστε μηχανές.

Μια μόνο στιγμή, στην οποία είμαστε αρκετά συνειδητοί, ώστε να σταματήσουμε να είμαστε μηχανή, αν γίνει εθελοντικά, συνήθως τροποποιεί ριζικά πολλές δυσάρεστες περιστάσεις.

Δυστυχώς ζούμε καθημερινά μια μηχανιστική, ρουτινιάρικη, παράλογη ζωή. Επαναλαμβάνουμε γεγονότα, οι συνήθειές μας είναι οι ίδιες, ποτέ δεν θελήσαμε να τις τροποποιήσουμε, είναι η μηχανική ράγα όπου κυκλοφορεί το τρένο της άθλιας ύπαρξής μας, ωστόσο, σκεφτόμαστε για τον εαυτό μας το καλύτερο...
