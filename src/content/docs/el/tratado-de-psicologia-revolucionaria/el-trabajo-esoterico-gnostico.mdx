Είναι επείγον να μελετήσετε τη Γνώση και να χρησιμοποιήσετε τις πρακτικές ιδέες που δίνουμε σε αυτό το έργο για να εργαστείτε σοβαρά πάνω στον εαυτό σας.

Ωστόσο, δεν θα μπορούσαμε να εργαστούμε πάνω στον εαυτό μας με την πρόθεση να διαλύσουμε ένα συγκεκριμένο "Εγώ" χωρίς να το έχουμε παρατηρήσει προηγουμένως.

Η παρατήρηση του εαυτού επιτρέπει σε μια ακτίνα φωτός να εισχωρήσει μέσα μας.

Οποιοδήποτε "Εγώ" εκφράζεται στο κεφάλι με έναν τρόπο, στην καρδιά με έναν άλλο τρόπο και στο σεξ με έναν άλλο τρόπο.

Πρέπει να παρατηρήσουμε το "Εγώ" που σε μια δεδομένη στιγμή βρίσκουμε παγιδευμένο, είναι επείγον να το δούμε σε καθένα από αυτά τα τρία κέντρα του οργανισμού μας.

Σε σχέση με άλλους ανθρώπους, εάν είμαστε σε εγρήγορση και άγρυπνοι σαν φρουρός σε καιρό πολέμου, αυτο-ανακαλυπτόμαστε.

Θυμάστε πότε πληγώθηκε η ματαιοδοξία σας; Η υπερηφάνειά σας; Τι σας ενόχλησε περισσότερο μέσα στην ημέρα; Γιατί είχατε αυτή την ενόχληση; Ποια είναι η μυστική της αιτία; Μελετήστε αυτό, παρατηρήστε το κεφάλι, την καρδιά και το σεξ σας...

Η πρακτική ζωή είναι ένα υπέροχο σχολείο. Στην αλληλεπίδραση μπορούμε να ανακαλύψουμε αυτά τα "Εγώ" που κουβαλάμε μέσα μας.

Οποιαδήποτε ενόχληση, οποιοδήποτε περιστατικό, μπορεί να μας οδηγήσει μέσω της οικείας αυτοπαρατήρησης, στην ανακάλυψη ενός "Εγώ", είτε αυτό είναι ιδιοτέλεια, φθόνος, ζήλια, οργή, απληστία, υποψία, συκοφαντία, λαγνεία, κ.λπ., κ.λπ., κ.λπ.

Πρέπει να γνωρίσουμε τον εαυτό μας πριν μπορέσουμε να γνωρίσουμε τους άλλους. Είναι επείγον να μάθουμε να βλέπουμε την άποψη των άλλων.

Εάν μπούμε στη θέση των άλλων, ανακαλύπτουμε ότι τα ψυχολογικά ελαττώματα που καταλογίζουμε στους άλλους, τα έχουμε σε περίσσεια μέσα μας.

Το να αγαπάμε τον πλησίον είναι απαραίτητο, αλλά δεν μπορεί κανείς να αγαπήσει τους άλλους αν πρώτα δεν μάθει να μπαίνει στη θέση ενός άλλου ατόμου στην εσωτερική εργασία.

Η σκληρότητα θα συνεχίσει να υπάρχει πάνω στη γη, όσο δεν έχουμε μάθει να μπαίνουμε στη θέση των άλλων.

Αλλά αν κάποιος δεν έχει το θάρρος να δει τον εαυτό του, πώς θα μπορούσε να μπει στη θέση των άλλων;

Γιατί να βλέπουμε αποκλειστικά την κακή πλευρά των άλλων ανθρώπων;

Η μηχανική αντιπάθεια προς ένα άλλο άτομο που γνωρίζουμε για πρώτη φορά, δείχνει ότι δεν ξέρουμε να μπούμε στη θέση του πλησίον, ότι δεν αγαπάμε τον πλησίον, ότι έχουμε τη συνείδηση πολύ κοιμισμένη.

Μας είναι πολύ αντιπαθητικό ένα συγκεκριμένο άτομο; Για ποιο λόγο; Ίσως πίνει; Ας παρατηρήσουμε τον εαυτό μας... Είμαστε σίγουροι για την αρετή μας; Είμαστε σίγουροι ότι δεν κουβαλάμε μέσα μας το "Εγώ" της μέθης;

Θα ήταν καλύτερα, όταν βλέπουμε έναν μεθυσμένο να κάνει αστεία, να λέγαμε: "Αυτός είμαι εγώ, τι αστεία κάνω".

Είστε μια έντιμη και ενάρετη γυναίκα και γι' αυτό δεν συμπαθείτε μια συγκεκριμένη κυρία. Νιώθετε αντιπάθεια για εκείνη. Γιατί; Νιώθετε πολύ σίγουρη για τον εαυτό σας; Πιστεύετε ότι μέσα σας δεν έχετε το "Εγώ" της λαγνείας; Νομίζετε ότι εκείνη η κυρία που έχει δυσφημιστεί από τα σκάνδαλα και τις ακολασίες της είναι διεστραμμένη; Είστε σίγουρη ότι μέσα σας δεν υπάρχει η ακολασία και η διαστροφή που βλέπετε σε αυτή τη γυναίκα;

Θα ήταν καλύτερα να παρατηρούσατε τον εαυτό σας οικεία και σε βαθύ διαλογισμό να καταλάβατε τη θέση εκείνης της γυναίκας που απεχθάνεστε.

Είναι επείγον να αξιολογήσουμε την εσωτερική Γνωστική εργασία, είναι απαραίτητο να την κατανοήσουμε και να την εκτιμήσουμε αν πραγματικά επιθυμούμε μια ριζική αλλαγή.

Καθίσταται απαραίτητο να ξέρουμε να αγαπάμε τους συνανθρώπους μας, να μελετήσουμε τη Γνώση και να μεταφέρουμε αυτή τη διδασκαλία σε όλους τους ανθρώπους, διαφορετικά θα πέσουμε στον εγωισμό.

Εάν κάποιος αφοσιωθεί στην εσωτερική εργασία πάνω στον εαυτό του, αλλά δεν δίνει τη διδασκαλία στους άλλους, η εσωτερική του πρόοδος καθίσταται πολύ δύσκολη λόγω έλλειψης αγάπης προς τον πλησίον.

"Αυτός που δίνει, λαμβάνει και όσο περισσότερο δίνει, τόσο περισσότερο θα λάβει, αλλά από αυτόν που δεν δίνει τίποτα, ακόμη και αυτό που έχει θα του αφαιρεθεί". Αυτός είναι ο Νόμος.
