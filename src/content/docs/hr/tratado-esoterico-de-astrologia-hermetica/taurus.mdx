Peti red svjetova sastoji se od svih planeta Sunčevog sustava.

Šesti red svjetova je sama Zemlja, sa svojih sedam dimenzija i područja naseljenih beskonačnim bićima.

Sedmi red svjetova sastoji se od tih sedam koncentričnih sfera ili PAKLENIH SVIJETOVA KRALJEVSTVA MINERALA URONJENOG ispod Zemljine kore.

Glazba, GLAGOL, postavljen od LOGOSA u sedam glazbenih oktava, održava Svemir čvrsto u njegovom hodu.

Prvi red svjetova, nota DO. Drugi red svjetova, nota SI. Treći red svjetova, nota LA. Četvrti red svjetova, nota SOL. Peti red svjetova, nota FA. Šesti red svjetova, nota MI. Sedmi red svjetova, nota RE. Poslije se sve vraća u APSOLUT s notom DO.

DO-RE-MI-FA-SOL-LA-SI. SI-LA-SOL-FA-MI-RE-DO. Sedam nota VELIKE LJESTVICE KREATIVNOG GLAGOLA, odjekuju u svemu stvorenom, jer u početku bijaše GLAGOL.
